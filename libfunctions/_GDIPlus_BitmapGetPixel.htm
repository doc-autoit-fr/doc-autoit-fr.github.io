_GDIPlus_BitmapGetPixel

Gets the color of a specified pixel in this bitmapObtient la couleur d'un pixel spécifié dans un bitmap

#include <GDIPlus.au3>
_GDIPlus_BitmapGetPixel ( $hBitmap, $iX, $iY )

ParametersParamètres

$hBitmap Pointer to the Bitmap objectHandle de l'objet Bitmap
$iX The X coordinate of the pixelLa coordonnée X du pixel
$iY The Y coordinate of the pixelLa coordonnée Y du pixel

Return ValueValeur de retour

Success:Succès : the pixel color of the bitmap.Retourne la couleur du pixel du bitmap.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

ExampleExemple

#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $iWidth = 150, $iHeight = 150
    Local $iColor = 0
    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, $iWidth, $iHeight) ;create a GDI bitmap by capturing an area on desktopCrée un bitmap GDI en capturant une zone du bureau
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;convert GDI to GDI+ bitmapCrée un bitmap GDI en capturant une région du bureau
    _WinAPI_DeleteObject($hHBmp) ;release GDI bitmap resource because not needed anymoreLibére la ressource du bitmap GDI car n'est plus nécessaire

    For $iY = 0 To $iHeight - 1
        For $iX = 0 To $iWidth - 1
            $iColor = _GDIPlus_BitmapGetPixel($hBitmap, $iX, $iY) ;get current pixel colorObtient la couleur de pixel courant
            _GDIPlus_BitmapSetPixel($hBitmap, $iX, $iY, BitXOR(0x00FFFFFF, $iColor)) ;invert RGB pixel color onlyXOR avec la couleur du pixel
        Next
    Next
    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;create a test GUIcréer une interface de test
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap, 0, 0) ;copy negative bitmap to graphics object (GUI)Copie le bitmap négatif dans l'objet graphique (GUI)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
