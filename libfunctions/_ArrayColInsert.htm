_ArrayColInsert

Inserts a specified column into a 1D or 2D arrayInsère une colonne spécifiée dans un tableau 1D ou 2D

#include <Array.au3>
_ArrayColInsert ( ByRef $aArray, $iColumn )

ParametersParamètres

$aArray Array to modifyTableau à transposer
$iColumn Column to insert - if array is 1D it is automatically converted to 2DColonne à insérer - si le tableau est 1D, il est automatiquement converti en 2D

Return ValueValeur de retour

Success:Succès : the number of columns remaining.Retourne le nombre de colonnes restantes.
Failure:Échec : -1 and sets the @error flag to non-zero.-1 et met @error <> 0.
@error:@error : 1 - $aArray is not an array1 - $aArray n'est pas un tableau
2 - $aArray is not a 1D or 2D array2 - $aArray n'est pas un tableau 1D ou 2D
3 - $iColumn is less than 0 or greater than upper array bound plus 13 - $iColumn est inférieure à 0 ou supérieure à la limite supérieure plus 1

ExampleExemple

#include <Array.au3>

Local $aArray[4] = [0, 1, 2, 3]
_ArrayDisplay($aArray, "Original")

_ArrayColInsert($aArray, 0)
; Now a 2D arrayMaintenant un tableau 2D

_ArrayDisplay($aArray, "Col 1 inserted")

_ArrayColInsert($aArray, 3)
_ArrayDisplay($aArray, "Col 3 inserted")
