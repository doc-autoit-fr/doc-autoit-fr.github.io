_WinAPI_IsWow64Process

Determines whether the specified process is running under WOW64

#include <WinAPIProc.au3>
_WinAPI_IsWow64Process ( [$iPID = 0] )

ParametersParamètres

$iPID [optional][optionnel] The PID of the process. Default (0) is the current process.

Return ValueValeur de retour

Success:Succès : True - The process is running under WOW64.
False - Otherwise.
Failure:Échec : Sets the @error flag to non-zero, call _WinAPI_GetLastError() to get extended error information.
