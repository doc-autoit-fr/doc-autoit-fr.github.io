_GDIPlus_ImageGetPixelFormat

Returns pixel format of an image: Bits per pixel, Alpha channels, RGB, Grayscale, Indexed etc

#include <GDIPlus.au3>
_GDIPlus_ImageGetPixelFormat ( $hImage )

ParametersParamètres

$hImage Handle to an image objectHandle de l'objet image

Return ValueValeur de retour

Success:Succès : an array with the following format:Retourne un tableau au format suivant :
    [0] - Integer of pixel format constant
    [1] - String of pixel format
Failure:Échec : sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).
@error:@error : 10 - Invalid pixel format.10 - Le format de pixel est invalide.
11 - Invalid image handle.11 - Handle d'image invalide.
12 - Unknown pixel format.12 - Le format de pixel est inconnu.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPI.au3>
#include <WindowsConstants.au3>

Global $g_idMemo

Example()

Func Example()
    Local $hBitmap, $hImage, $aRet

    ; Create GUICrée la GUI
    GUICreate("GDI+", 600, 400)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Initialize GDI+ libraryInitialise GDI+ Librairie
    _GDIPlus_Startup()

    ; Capture 32 bit bitmapCapture un bitmap 32 bits
    $hBitmap = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; Show pixel format for screen captureAffiche le format de pixel pour la capture d'écran
    $aRet = _GDIPlus_ImageGetPixelFormat($hImage)
    MemoWrite("Image pixel format of screen capture: " & $aRet[1]);
    MemoWrite("Image pixel format constant: " & $aRet[0]);
    MemoWrite();

    ; Save screen capture bitmap to fileEnregistre le bitmap de la capture d'écran dans un fichier
    _GDIPlus_ImageSaveToFile($hImage, @MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Clean up resourcesLibère les ressources
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; Load screen capture bitmap from fileCharge le bitmap de la capture d'écran à partir du fichier
    $hImage = _GDIPlus_ImageLoadFromFile(@MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Show pixel format for saved fileAffiche le format de pixel du fichier sauvegardé
    $aRet = _GDIPlus_ImageGetPixelFormat($hImage)
    MemoWrite("Image pixel format of saved file: " & $aRet[1]);
    MemoWrite("Image pixel format constant: " & $aRet[0]);

    ; Clean up resourcesLibère les ressources
    _GDIPlus_ImageDispose($hImage)

    ; Shut down GDI+ libraryFerme la librairie GDI+
    _GDIPlus_Shutdown()

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Write a line to the memo controlÉcrit une ligne dans le contrôle mémo
Func MemoWrite($sMessage = '')
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
