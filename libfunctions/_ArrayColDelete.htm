_ArrayColDelete

Deletes a specified column from a 2D arraySupprime une colonne spécifiée d'un tableau 2D

#include <Array.au3>
_ArrayColDelete ( ByRef $aArray, $iColumn [, $bConvert = False] )

ParametersParamètres

$aArray Array to modifyTableau à transposer
$iColumn Column to deleteColonne à supprimer
$bConvert [optional] If True then if only one column remains the array is converted to 1DSi True, alors si une seule colonne reste, le tableau est converti en 1D

Return ValueValeur de retour

Success: the number of columns remaining.Retourne le nombre de colonnes restantes.
Failure: -1 and sets the @error flag to non-zero.-1 et met @error <> 0.
@error:@error : 1 - $aArray is not an array1 - $aArray n'est pas un tableau
2 - $aArray is not a 2D array2 - $aArray n'est pas un tableau 2D
3 - $iColumn is outside array bounds3 - $iColumn est en dehors des limites du tableau

ExampleExemple

#include <Array.au3>

Local $aArray[4][3]
For $i = 0 To 3
    For $j = 0 To 2
        $aArray[$i][$j] = $i & $j
    Next
Next
_ArrayDisplay($aArray, "Original")

_ArrayColDelete($aArray, 2)
_ArrayDisplay($aArray, "Col 2 Deleted")

; Copy 2 col arrayCopie 2 colonnes du tableau
Local $aArray_2Col = $aArray

; Default 1 column left as 2D arrayPar défaut 1 colonne à gauche comme tableau 2D
_ArrayColDelete($aArray, 0)
_ArrayDisplay($aArray, "Col 0 del - 2D array")

; Convert 1 column array to 1DConvertit une colonne de tableau en 1D
_ArrayColDelete($aArray_2Col, 0, True)
_ArrayDisplay($aArray_2Col, "Col 0 del - 1D array")
