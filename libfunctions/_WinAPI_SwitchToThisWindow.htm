_WinAPI_SwitchToThisWindow

Switches the focus to a specified window and bring it to the foreground

#include <WinAPISys.au3>
_WinAPI_SwitchToThisWindow ( $hWnd [, $bAltTab = False] )

ParametersParamètre

$hWnd Handle to the window being switched to.
$bAltTab [optional][optional] False - i, v, x ... Specifies whether switches to using the Alt/Ctl+Tab key sequence, valid values:
    True - The window is being switched to using the Alt/Ctl+Tab key sequence.
    False - Otherwise (Default).

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.
