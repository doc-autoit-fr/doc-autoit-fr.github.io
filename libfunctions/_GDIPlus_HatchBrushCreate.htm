_GDIPlus_HatchBrushCreate

Creates a HatchBrush object based on a hatch style, a foreground color, and a background colorCrée un objet HatchBrush (pinceau à hachures) à partir d'un style de hachure, d'une couleur de peinture, et d'une couleur de fond

#include <GDIPlus.au3>
_GDIPlus_HatchBrushCreate ( [$iHatchStyle = 0 [, $iARGBForeground = 0xFFFFFFFF [, $iARGBBackground = 0xFFFFFFFF]]] )

ParametersParamètres

$iHatchStyle [optional][optionnel] Pattern of hatch lines that will be used, see remarks.Motif de hachures à utiliser, voir Remarques. Default = $GDIP_HATCHSTYLE_HORIZONTAL (0).Par défaut = $GDIP_HATCHSTYLE_HORIZONTAL (0).
$iARGBForeground [optional][optionnel] Alpha, Red, Green and Blue components of the hatch linesComposantes Alpha, Rouge, Vert et Bleu des lignes de hachures
$iARGBBackground [optional][optionnel] Alpha, Red, Green and Blue components of the hatch backgroundComposantes Alpha, Rouge, Vert et Bleu du fond

Return ValueValeur de retour

Success:Succès : a pointer to a new HatchBrush object.Retourne le handle d'un nouvel objet HatchBrush.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

The pattern constants are declared in GDIPlusConstants.au3, those that start with $GDIP_HATCHSTYLE_*.Les constantes des motifs de hachures sont déclarées dans GDIPlusConstants.au3, celles qui commencent par $GDIP_HATCHSTYLE_*.
When you are done with the HatchBrush object, call _GDIPlus_BrushDispose() to release the resources.Lorsque vous en avez terminé avec l'objet HatchBrush, appelez _GDIPlus_BrushDispose() pour libérer les ressources.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hBrush, $iX, $iY

    ; Create GUICrée la GUI
    Local $hGUI = GUICreate("GDI+", 810, 610)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    Local $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;Create a graphics object from a window handleCrée une GUI

    For $i = 0 To 52
        $iX = 10 + Mod($i, 10) * 80
        $iY = 30 + Floor($i / 10) * 100
        _GDIPlus_GraphicsDrawString($hGraphic, $i, $iX, $iY - 16)

        $hBrush = _GDIPlus_HatchBrushCreate($i, 0xFF00FF00, 0xFF0000FF)
        _GDIPlus_GraphicsFillRect($hGraphic, $iX, $iY, 70, 70, $hBrush)

        _GDIPlus_BrushDispose($hBrush)
    Next

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
