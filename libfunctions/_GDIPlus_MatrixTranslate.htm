_GDIPlus_MatrixTranslate

Updates a matrix with the product of itself and a translation matrixRemplace les coefficients d'une matrice par les coefficients du produit d'elle-même par une matrice de translation

#include <GDIPlus.au3>
_GDIPlus_MatrixTranslate ( $hMatrix, $fOffsetX, $fOffsetY [, $bAppend = False] )

ParametersParamètres

$hMatrix Handle to a Matrix objectHandle de l'objet Matrix
$fOffsetX Amount of pixels to add along the x-axisNombre de pixels à ajouter le long de l'axe x
$fOffsetY Amount of pixels to add along the y-axisNombre de pixels à ajouter le long de l'axe y
$bAppend [optional][optionnel] Specifies the order of the multiplication:Spécifie l'ordre de la multiplication:
    True - Specifies that the translation matrix is on the left
    False - Specifies that the translation matrix is on the right

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

; Create GUICrée la GUI
Local $hWnd = GUICreate("GDI+ Example (" & @ScriptName & ")", 400, 300)
GUISetState(@SW_SHOW)

; Start GDI+Lance GDI+
_GDIPlus_Startup()
Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hWnd)

Local $hMatrix = _GDIPlus_MatrixCreate()
; Move the matrix half the width and helf the height towards the centerMatrice de translation de la moitié de la largeur de la fenêtre et de la moitié de la hauteur
_GDIPlus_MatrixTranslate($hMatrix, 200, 150)
_GDIPlus_MatrixRotate($hMatrix, 45)
_GDIPlus_GraphicsSetTransform($hGraphics, $hMatrix)

Local $hPen = _GDIPlus_PenCreate(0xFF00FF00, 10)

_GDIPlus_GraphicsClear($hGraphics)
; Draw around upper left corner of the GUI, but since we have translated the matrix the object will appear at the center of GUIDessine autour du coin supérieur gauche de la gui, mais depuis que nous avons translaté la matrice l'objet apparaît au centre de la gui
_GDIPlus_GraphicsDrawRect($hGraphics, -50, -50, 100, 100, $hPen)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE

; Clean up resourcesLibère les ressources
_GDIPlus_PenDispose($hPen)
_GDIPlus_MatrixDispose($hMatrix)
_GDIPlus_GraphicsDispose($hGraphics)
_GDIPlus_Shutdown()
