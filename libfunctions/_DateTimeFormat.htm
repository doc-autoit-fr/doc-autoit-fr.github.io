_DateTimeFormat

Returns the date in the PC's regional settings formatRetourne la date dans le format des paramètres régionaux du PC

#include <Date.au3>
_DateTimeFormat ( $sDate, $sType )

ParametersParamètres

$sDate Input date in the format "YYYY/MM/DD[ HH:MM:SS]"Date à saisir dans le format « AAAA/MM/JJ[ HH:MM:SS] ».
$sType one the following:Un des suivants :
0 - Display a date and/or time.0 - Affiche une date et/ou une heure. If there is a date part, display it as a short date.Si c'est une partie de date, l'afficher comme une date courte.
If there is a time part, display it as a long time.Si c'est une partie d'une heure, l'afficher comme une heure longue. If present, both parts are displayed.Si présent, les deux parties sont affichées.
1 - Display a date using the long date format specified in your computer's regional settings.1 - Affiche une date en utilisant la date au format long spécifié dans les paramètres régionaux de votre ordinateur.
2 - Display a date using the short date format specified in your computer's regional settings.2 - Affiche une date en utilisant la date au format court spécifié dans les paramètres régionaux de votre ordinateur.
3 - Display a time using the time format specified in your computer's regional settings.3 - Affiche une heure en utilisant l'heure au format spécifié dans les paramètres régionaux de votre ordinateur.
4 - Display a time using the 24-hour format (hh:mm).4 - Affiche une heure en utilisant le format 24 heures (hh:mm).
5 - Display a time using the 24-hour format (hh:mm:ss).5 - Affiche une heure en utilisant le format 24 heures (hh:mm:ss).

Return ValueValeur de retour

Success:Succès : the date in proper format.la date dans le format approprié.
Failure:Échec : 0 and set the @error flag to non-zero.0 et met @error <> 0.
@error:@error : 1 - Invalid $sDate1 - $sDate non valide
2 - Invalid $sType2 - $sType non valide

RemarksRemarques

See _DateTimeSplit() for other possible variations of the input date format.Voir _DateTimeSplit() pour d'autres variations possibles du format de la date à saisir.

ExampleExemple

#include <Date.au3>
#include <MsgBoxConstants.au3>

; Show current date/time in the pc's formatAffiche la date/heure actuelle dans le format du PC
