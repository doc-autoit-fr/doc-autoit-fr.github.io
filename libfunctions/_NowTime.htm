_NowTime

Returns the current Time in the requested formatRetourne l'heure actuelle dans le format demandé

#include <Date.au3>
_NowTime ( [$sType = 3] )

ParametersParamètres

$sType [optional][optionnel][optionnel]
    3 = (default) Display a time using the time format specified in your computer's regional settings.    3 = (par défaut) Affiche une heure en utilisant le format d'heure spécifié dans les paramètres régionaux de l'ordinateur.
    4 - Display a time using the 24-hour format (hh:mm).    4 - Affiche une heure en utilisant le format 24 heures (hh:mm).
    5 - Display a time using the 24-hour format (hh:mm:ss).    5 - Affiche une heure en utilisant le format 24 heures (hh:mm:ss).

Return ValueValeur de retour

Returns the current TIME according to the time format used.Retourne l'heure actuelle selon le format d'heure utilisé.

RemarksRemarques

If your computer's regional settings is not accessible it returns the time in the "h:mm:ss tt" format.Si les paramètres régionaux de l'ordinateur ne sont pas accessibles, elle retourne l'heure dans le format « h:mm:ss tt ».
