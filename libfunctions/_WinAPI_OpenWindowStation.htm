_WinAPI_OpenWindowStation

Opens the specified window station

#include <WinAPISys.au3>
_WinAPI_OpenWindowStation ( $sName [, $iAccess = 0 [, $bInherit = False]] )

ParametersParamètres

$sName The name of the window station to be opened. Window station names are case-insensitive. This window
station must belong to the current session.
$iAccess [optional][optionnel][optionnel] The access to the window station. This parameter can be one or more of the following values.
$WINSTA_ALL_ACCESS
$WINSTA_ACCESSCLIPBOARD
$WINSTA_ACCESSGLOBALATOMS
$WINSTA_CREATEDESKTOP
$WINSTA_ENUMDESKTOPS
$WINSTA_ENUMERATE
$WINSTA_EXITWINDOWS
$WINSTA_READATTRIBUTES
$WINSTA_READSCREEN
$WINSTA_WRITEATTRIBUTES
$bInherit [optional][optionnel][optionnel] Specifies whether inherites the handle by a processes, valid values:
    True - The processes created by this process will inherit the handle.
    False - The processes do not inherit this handle (Default).

Return ValueValeur de retour

Success:Succès : Handle to the specified window station.
Failure:Échec : 0, call _WinAPI_GetLastError() to get extended error information.

RemarksRemarques

After you are done with the handle, you must call _WinAPI_CloseWindowStation() to free the handle.
