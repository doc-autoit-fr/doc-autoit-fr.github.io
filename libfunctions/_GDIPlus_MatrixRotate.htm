_GDIPlus_MatrixRotate

Updates a matrix with the product of itself and a rotation matrixRemplace les coefficients d'une matrice par les coefficients du produit d'elle-même avec une matrice de rotation

#include <GDIPlus.au3>
_GDIPlus_MatrixRotate ( $hMatrix, $fAngle [, $bAppend = False] )

ParametersParamètres

$hMatrix Handle to a Matrix objectHandle de l'objet Matrix
$fAngle The angle of rotation in degrees.L'angle de rotation en degrés. Positive values specify clockwise rotation.Les valeurs positives indiquent une rotation dans le sens horaire.
$bAppend [optional][optionnel] Specifies the order of the multiplication:Spécifie l'ordre de la multiplication:
    True - Specifies that the rotation matrix is on the left
    False - Specifies that the rotation matrix is on the right

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

;                  $hImage - Handle to an Image object to be inserted$hImage - Handle de l'objet Image à insérer

Func DrawInsert($hGraphic, $hImage2, $iX, $iY, $fAngle, $iWidth, $iHeight, $iARGB = 0xFF000000, $iPenWidth = 1)
    Local $hMatrix, $hPen2

    ;Rotation MatrixMatrice de rotation
    $hMatrix = _GDIPlus_MatrixCreate()
    _GDIPlus_MatrixRotate($hMatrix, $fAngle, "FalseRetourne False")
    _GDIPlus_GraphicsSetTransform($hGraphic, $hMatrix)

    _GDIPlus_GraphicsDrawImage($hGraphic, $hImage2, $iX, $iY)

    ;get pen + colorObtient le crayon + la couleur
    $hPen2 = _GDIPlus_PenCreate($iARGB, $iPenWidth)

    ; Draw a frame around the inserted imageDessine un cadre autour de l'image insérée
    _GDIPlus_GraphicsDrawRect($hGraphic, $iX, $iY, $iWidth, $iHeight, $hPen2)

    ; Clean up resourcesLibère les ressources
    _GDIPlus_MatrixDispose($hMatrix)
    _GDIPlus_PenDispose($hPen2)
    Return 1
EndFunc   ;==>DrawInsert
