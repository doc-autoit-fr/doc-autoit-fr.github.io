_WinAPI_EnumUILanguages

Enumerates the user interface languages that are available on the operating system

#include <WinAPILocale.au3>
_WinAPI_EnumUILanguages ( [$iFlag = 0] )

ParametersParamètres

$iFlag [optional][optionnel] The flag identifying the language format. This parameter must be 0 or one of the following values Windows Vista or later:
    $MUI_LANGUAGE_ID
    $MUI_LANGUAGE_NAME

Return ValueValeur de retour

Success:Succès : The array of the UI language identifiers or language names.
Failure:Échec : Sets the @error flag to non-zero.Retourne @error <> 0
