_DebugReport

Writes to a debugging sessionÉcrit dans une session de débogage

#include <Debug.au3>
_DebugReport ( $sData [, $bLastError = False [, $bExit = False]] )

ParametersParamètres

$sData Data to be reported
$bLastError [optional][optionnel][optionnel] True if _WinAPI_GetLastErrorMessage() must be append to the data being reported
$bExit [optional][optionnel][optionnel] True if the script must be terminatedTrue si le script doit être terminé

Return ValueValeur de retour

Success:Succès : 1 (see remarks).
Failure:Échec : 0.Retourne 0.

RemarksRemarques

If no _DebugSetup() have been issued the function always returns.
@error of the caller is preserved.
@extended can be set to Windows API _WinNet_GetLastError() if $bLastError = True.
Otherwise it is preserved.
