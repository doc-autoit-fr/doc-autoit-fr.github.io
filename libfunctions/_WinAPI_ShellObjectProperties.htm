_WinAPI_ShellObjectProperties

Invokes the Properties context menu command on a Shell object

#include <WinAPIShellEx.au3>
_WinAPI_ShellObjectProperties ( $sFilePath [, $iType = 2 [, $sProperty = '' [, $hParent = 0]]] )

ParametersParamètres

$sFilePath The object name.
$iType [optional][optionnel][optionnel] The value that specifies the type of object.
$SHOP_PRINTERNAME
$SHOP_FILEPATH (Default)
$SHOP_VOLUMEGUID
$sProperty [optional][optionnel][optionnel] The name of the property sheet page to be opened initially.
$hParent [optional][optionnel][optionnel] Handle of the parent window of the dialog box.

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False.Retourne False.

See AlsoVoir aussi

Search SHObjectProperties in MSDN Library.

ExampleExemple

#include <WinAPIShellEx.au3>

Opt('WinTitleMatchMode', 4)

If Not _WinAPI_ShellObjectProperties(@ScriptFullPath) Then Exit

Local $hWnd = WinWaitActive(@ScriptName, '', 3)
If Not $hWnd Then
    Exit
EndIf

While WinExists($hWnd)
    Sleep(100)
WEnd
