_WinAPI_GetClipboardSequenceNumber

Retrieves the clipboard sequence number for the current window stationRécupère le numéro de séquence du presse-papiers pour la station de fenêtre actuelle

#include <WinAPISys.au3>
_WinAPI_GetClipboardSequenceNumber ( )

Return ValueValeur de retour

Success:Succès : The clipboard sequence number.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

The calling process must have WINSTA_READATTRIBUTES access to the window station.
This function requires Windows Vista or later.
