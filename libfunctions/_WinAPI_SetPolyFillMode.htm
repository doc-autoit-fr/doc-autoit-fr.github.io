_WinAPI_SetPolyFillMode

Sets the polygon fill mode for functions that fill polygons

#include <WinAPIGdi.au3>
_WinAPI_SetPolyFillMode ( $hDC [, $iMode = 1] )

ParametersParamètres

$hDC Handle to the device context.
$iMode [optional][optionnel] The new fill mode. This parameter can be one of the following values.
$ALTERNATE (Default)
$WINDING

Return ValueValeur de retour

Success:Succès : The previous filling mode.
Failure:Échec : 0.Retourne 0.
