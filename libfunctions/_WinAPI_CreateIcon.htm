_WinAPI_CreateIcon

Creates an icon that has the specified size, colors, and bit patterns

#include <WinAPIRes.au3>
_WinAPI_CreateIcon ( $hInstance, $iWidth, $iHeight, $iPlanes, $iBitsPixel, $pANDBits, $pXORBits )

ParametersParamètre

$hInstance Handle to the instance of the module creating the icon.
$iWidth The width, in pixels, of the icon.
$iHeight The height, in pixels, of the icon.
$iPlanes The number of planes in the XOR bitmask of the icon.
$iBitsPixel The number of bits-per-pixel in the XOR bitmask of the icon.
$pANDBits An array of bytes that contains the bit values for the AND bitmask of the icon. This bitmask describes a monochrome bitmap.
$pXORBits An array of bytes that contains the bit values for the XOR bitmask of the icon. This bitmask describes a monochrome or device-dependent color bitmap.

Return ValueRemarque

Success: the handle to the icon that is created.
Failure: 0, call _WinAPI_GetLastError() to get extended error information.

Remarks<i0>

When you are finished using the icon, destroy it using the _WinAPI_DestroyIcon() function.
