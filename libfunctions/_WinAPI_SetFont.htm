_WinAPI_SetFont

Sets a window font

#include <WinAPI.au3>
_WinAPI_SetFont ( $hWnd, $hFont [, $bRedraw = True] )

ParametersParamètre

$hWnd Window handle
$hFont Font handle
$bRedraw [optional][optional] False - i, v, x ... True to redraw the control

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.
