_WinAPI_ShellGetKnownFolderPath

Retrieves the full path of a known folder identified

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetKnownFolderPath ( $sGUID [, $iFlags = 0 [, $hToken = 0]] )

ParametersParamètres

$sGUID The GUID ($FOLDERID_*) that identifies the standard folders registered with the system.
$iFlags [optional][optionnel][optionnel] The flags that specify special retrieval options. This parameter can be one or more of the following values.
    $KF_FLAG_CREATE
    $KF_FLAG_DONT_VERIFY
    $KF_FLAG_DONT_UNEXPAND
    $KF_FLAG_NO_ALIAS
    $KF_FLAG_INIT
    $KF_FLAG_DEFAULT_PATH
    $KF_FLAG_NOT_PARENT_RELATIVE
    $KF_FLAG_SIMPLE_IDLIST

Windows 7 or later
    $KF_FLAG_ALIAS_ONLY
$hToken [optional][optionnel][optionnel] The access token that represents a particular user. If this parameter is 0, the function requests
the known folder for the current user. Assigning the $hToken parameter a value of (-1) indicates the
Default User. Note that access to the Default User folders requires administrator privileges.

Return ValueValeur de retour

Success:Succès : The path of the known folder.
Failure:Échec : Empty string and sets the @error flag to non-zero, @extended flag may contain the HRESULT error code.

RemarksRemarques

This function requires Windows Vista or later.
