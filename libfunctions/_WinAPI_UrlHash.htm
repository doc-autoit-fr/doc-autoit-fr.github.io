_WinAPI_UrlHash

Hashes a URL string

#include <WinAPIShPath.au3>
_WinAPI_UrlHash ( $sUrl [, $iLength = 32] )

ParametersParamètres

$sUrl The URL.
$iLength [optional][optionnel] The length of the hash data, in bytes. It should be no larger than 256, otherwise, the function fails. Default is 32.

Return ValueValeur de retour

Success:Succès : The hash data in binary form.
Failure:Échec : Sets the @error flag to non-zero, @extended flag may contain the HRESULT error code.

See AlsoVoir aussi

Search UrlHash in MSDN Library.

ExampleExemple

#include <WinAPIShPath.au3>

Local $sUrl = 'http://www.microsoft.com'

ConsoleWrite(_WinAPI_UrlHash($sUrl) & @CRLF)
