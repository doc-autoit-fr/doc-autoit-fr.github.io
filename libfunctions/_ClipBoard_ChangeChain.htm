_ClipBoard_ChangeChain

Removes a specified window from the chain of clipboard viewersSupprime une fenêtre spécifiée de la chaîne des vues du presse-papiers

#include <Clipboard.au3>
_ClipBoard_ChangeChain ( $hRemove, $hNewNext )

ParametersParamètres

$hRemove Handle to the window to be removed from the chain.Handle de la fenêtre qui doit être retirée de la chaîne.
The handle must have been passed to the _ClipBoard_SetViewer() function.La handle doit avoir été passé à la fonction _ClipBoard_SetViewer().
$hNewNext Handle to the window that follows the $hRemove window in the clipboard viewer chain.Handle de la fenêtre qui suit la fenêtre $hRemove dans la chaîne des vues du presse-papiers.
This is the handle returned by _ClipBoard_SetViewer(), unless the sequence was changed in response to a $WM_CHANGECBCHAIN message.C'est le handle renvoyé par _ClipBoard_SetViewer() , à moins que la séquence ait été modifiée en réponse à un message $WM_CHANGECBCHAIN.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

The window identified by $hNewNext replaces the $hRemove window in the chain.La fenêtre identifiée par $hNewNext remplace la fenêtre $hRemove dans la chaîne.
The _ClipBoard_SetViewer() function sends a $WM_CHANGECBCHAIN message to the first window in the clipboard viewer chain.La fonction _ClipBoard_SetViewer() envoie le message $WM_CHANGECBCHAIN à la première fenêtre de la chaîne des vues du presse-papiers.

ExampleExemple

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <WindowsConstants.au3>

Global $g_idMemo, $g_hNext = 0

Example()

Func Example()
    Local $hGUI

    ; Create GUICrée la GUI
    $hGUI = GUICreate("Clipboard", 600, 400)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Initialize clipboard viewerInitialise le visualiseur du presse-papiers
    $g_hNext = _ClipBoard_SetViewer($hGUI)

    GUIRegisterMsg($WM_CHANGECBCHAIN, "WM_CHANGECBCHAIN")
    GUIRegisterMsg($WM_DRAWCLIPBOARD, "WM_DRAWCLIPBOARD")

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Shut down clipboard viewerArrête le visualiseur du presse-papiers
    _ClipBoard_ChangeChain($hGUI, $g_hNext)
EndFunc   ;==>Example

; Write message to memoEcrire un message dans memo
Func MemoWrite($sMessage = "")
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite

; Handle $WM_CHANGECBCHAIN messagesHandle des messages $WM_CHANGECBCHAIN
Func WM_CHANGECBCHAIN($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg
    ; Show that message was receivedMontre que le message a été reçu
    MemoWrite("***** $WM_CHANGECBCHAIN *****")

    ; If the next window is closing, repair the chainSi la fenêtre suivante se ferme, répare la chaîne
    If $wParam = $g_hNext Then
        $g_hNext = $lParam
        ; Otherwise pass the message to the next viewerSinon passe le message au visualiseur suivant
    ElseIf $g_hNext <> 0 Then
        _SendMessage($g_hNext, $WM_CHANGECBCHAIN, $wParam, $lParam, 0, "hwnd", "hwnd")
    EndIf
EndFunc   ;==>WM_CHANGECBCHAIN

; Handle $WM_DRAWCLIPBOARD messagesHandle des messages $WM_DRAWCLIPBOARD
Func WM_DRAWCLIPBOARD($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg
    ; Display any text on clipboardAffiche un texte dans le presse-papiers
    MemoWrite(_ClipBoard_GetData())

    ; Pass the message to the next viewerPasse le message au visualiseur suivant
    If $g_hNext <> 0 Then _SendMessage($g_hNext, $WM_DRAWCLIPBOARD, $wParam, $lParam)
EndFunc   ;==>WM_DRAWCLIPBOARD
