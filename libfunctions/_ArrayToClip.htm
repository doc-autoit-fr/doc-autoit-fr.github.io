_ArrayToClip

Sends the contents of a 1D or 2D array to the clipboard, each element separated by a specified delimiterEnvoie le contenu d'un tableau 1D ou 2D dans le presse-papier, chaque élément étant séparé par un séparateur spécifié

#include <Array.au3>
_ArrayToClip ( Const ByRef $aArray [, $sDelim_Col = "|" [, $iStart_Row = -1 [, $iEnd_Row = -1 [, $sDelim_Row = @CRLF [, $iStart_Col = -1 [, $iEnd_Col = -1]]]]]] )

ParametersParamètres

$aArray Array to copy to clipboardTableau à copier dans le presse papier
$sDelim_Col [optional][optionnel] Delimiter for elements of 1D array or columns of 2D arrayDélimiteur des éléments du tableau 1D ou des colonnes de tableau 2D
$iStart_Row [optional][optionnel] Index of array row to start copyIndex de ligne du tableau pour commencer la copie
$iEnd_Row [optional][optionnel] Index of array row to stop copyIndex de ligne du tableau pour arrêter la copie
$sDelim_Row [optional][optionnel] Delimiter for rows of 2D array (2D only)Délimiteur des lignes du tableau 2D (2D uniquement)
$iStart_Col [optional][optionnel] Index of array column to start copy (2D only)Index de colonne du tableau pour commencer la copie (2D uniquement)
$iEnd_Col [optional][optionnel] Index of array column to stop copy (2D only)Index de colonne du tableau pour arrêter la copie (2D uniquement)

ExampleExemple

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray = StringSplit("A,B,C,D,E,F,G,H,I", ",")
_ArrayDisplay($aArray, "1D Array")
_ArrayToClip($aArray, " - ", Default, 1, 6)
MsgBox($MB_SYSTEMMODAL, "_ArrayToClip() 1D Test", ClipGet())

Local $aArray[4][4]
For $i = 0 To 3
    For $j = 0 To 3
        $aArray[$i][$j] = $i & $j
    Next
Next
_ArrayDisplay($aArray, "2D Array")
_ArrayToClip($aArray, " :: ", Default, 1, 2, 1, 2)
MsgBox($MB_SYSTEMMODAL, "_ArrayToClip() 2D Test", ClipGet())
