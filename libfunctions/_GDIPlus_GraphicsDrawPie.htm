_GDIPlus_GraphicsDrawPie

Draw a pieDessine un secteur d'angle (morceau de tarte)

#include <GDIPlus.au3>
_GDIPlus_GraphicsDrawPie ( $hGraphics, $nX, $nY, $nWidth, $nHeight, $fStartAngle, $fSweepAngle [, $hPen = 0] )

ParametersParamètres

$hGraphics Handle to a Graphics objectHandle de l'objet Graphics
$nX The X coordinate of the upper left corner of the rectangle that bounds the ellipse in which to draw the pieLa coordonnée X du coin supérieur gauche du rectangle qui délimite l'ellipse dans laquelle extraire le secteur angulaire
$nY The Y coordinate of the upper left corner of the rectangle that bounds the ellipse in which to draw the pieLa coordonnée Y du coin supérieur gauche du rectangle qui délimite l'ellipse dans laquelle extraire le secteur angulaire
$nWidth The width of the rectangle that bounds the ellipse in which to draw the pieLa largeur du rectangle qui délimite l'ellipse dans laquelle dessiner le secteur angulaire
$nHeight The height of the rectangle that bounds the ellipse in which to draw the pieLa hauteur du rectangle qui délimite l'ellipse dans laquelle dessiner le secteur angulaire
$fStartAngle The angle, in degrees, between the X axis and the starting point of the arc that defines the pie.L'angle, en degrés, entre l'axe X et le point de début de l'arc qui définit le secteur angulaire.
A positive value specifies clockwise rotation.Une valeur positive indique une rotation dans le sens horaire.
$fSweepAngle The angle, in degrees, between the starting and ending points of the arc that defines the pie.L'angle, en degrés, entre les points de début et de fin de l'arc qui définit le secteur angulaire.
A positive value specifies clockwise rotation.Une valeur positive indique une rotation dans le sens horaire.
$hPen [optional][optionnel][optionnel] Handle to a pen object that is used to draw the arc.Handle de l'objet Pen qui sera utilisé pour dessiner le secteur. If 0, a solid black pen with a width of 1 will be used.Si 0, un crayon noir qui trace des traits continus de largeur de 1 sera utilisé.

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    ; Draw a pieDessine un secteur d'angle (morceau de tarte)
    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    _GDIPlus_GraphicsDrawPie($hGraphic, 150, 80, 100, 100, 45, 90)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $iWidth = 600, $iHeight = 600, $iBgColor = 0x303030 ;$iBGColor format RRGGBB$iBGColor est de la forme RRGGBB

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;create a test GUIcréer une interface de test
    GUISetBkColor($iBgColor, $hGUI) ;set GUI background colorDéfinit la couleur de fond de la GUI
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    _GDIPlus_GraphicsSetSmoothingMode($hGraphics, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;sets the graphics object rendering quality (antialiasing)Définit l'objet graphique en qualité de rendu antialiasing
    Local $hPen = _GDIPlus_PenCreate(0xFFA08080, 2) ;color format AARRGGBB (hex)Le format de couleur est AARRGGBB (hex)
    _GDIPlus_GraphicsDrawPie($hGraphics, 100.5, 0.5, 400.75, 400.125, 45, 90, $hPen)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _GDIPlus_PenDispose($hPen)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
