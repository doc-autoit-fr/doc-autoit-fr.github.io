Keyword Reference

$tagNMHEADER

Contains information about control notification messagesContient des informations sur les messages de notification de contrôle

Global Const $tagNMHEADER = $tagNMHDR & ";int Item;int Button;ptr pItem"

ParametersParamètres

$tagNMHDR Contains information about a notification messageContient des informations sur un message de notification
Item 0-based index of the control to which this structure refers
Button Index of the mouse button used to generate the notification message. This member can be one of the following values:
    0 - Left button
    1 - Right button
    2 - Middle button
pItem Optional pointer to a tagHDITEM structure containing information about the item specified by Item.
The mask member of the tagHDITEM structure indicates which of its members are valid.
