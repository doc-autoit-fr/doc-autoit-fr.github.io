_GDIPlus_BitmapCreateFromGraphics

Creates a Bitmap object based on a Graphics object, a width, and a heightCrée un objet Bitmap à partir d'un objet Graphics, avec une largeur et une hauteur données

#include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromGraphics ( $iWidth, $iHeight, $hGraphics )

ParametersParamètres

$iWidth Specifies the width, in pixels, of the bitmapSpécifie la largeur, en pixels, du bitmap
$iHeight Specifies the height, in pixels, of the bitmapSpécifie la hauteur, en pixels, du Bitmap
$hGraphics Handle to a Graphics objectHandle de l'objet Graphics

Return ValueValeur de retour

Success:Succès : a handle to a Bitmap object Q .Retourne le handle de l'objet Bitmap
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

When you are done with the Bitmap object, call _GDIPlus_BitmapDispose() to release the resources.Lorsque vous en avez terminé avec l'objet bitmap, appelez _GDIPlus_BitmapDispose() pour libérer les ressources

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Global $g_hGUI, $g_hBrush, $g_hGfx, $g_hGfxCtxt, $g_hBitmap

Example()

Func Example()
    AutoItSetOption("GUIOnEventMode", 1)

    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $iWidth = 600, $iHeight = 600, $iBgColor = 0x303030 ;$iBGColor format RRGGBB$iBGColor est de la forme RRGGBB

    $g_hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;create a test GUIcréer une interface de test
    GUISetBkColor($iBgColor, $g_hGUI) ;set GUI background colorDéfinit la couleur de fond de la GUI
    GUISetState(@SW_SHOW)

    ;create buffered graphics frame set for smoother gfx object movementsCrée un buffer graphique pour lisser les objets gfx en mouvement
    $g_hGfx = _GDIPlus_GraphicsCreateFromHWND($g_hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    $g_hBitmap = _GDIPlus_BitmapCreateFromGraphics($iWidth, $iHeight, $g_hGfx) ;create a Bitmap object based on a graphics objectCrée un objet Bitmap basé sur un objet graphique
    $g_hGfxCtxt = _GDIPlus_ImageGetGraphicsContext($g_hBitmap) ;get the graphics context of the image / bitmap to draw on image / bitmapObtient le contexte graphique de l'image/bitmap pour dessiner dans l'image/bitmap
    _GDIPlus_GraphicsSetSmoothingMode($g_hGfxCtxt, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;sets the graphics object rendering quality (antialiasing)Définit l'objet graphique en qualité de rendu antialiasing

    $g_hBrush = _GDIPlus_BrushCreateSolid(0xFF8080FF) ;create a solid Brush objectCrée un objet Brush solide

    GUISetOnEvent($GUI_EVENT_CLOSE, "_Exit")

    Local Const $iDeg = ACos(-1) / 180 ;ACos(-1) is nearly piACos(-1) est presque pi
    Local $iSize = 50, $iX_Center = ($iWidth - $iSize) / 2, $iY_Center = ($iHeight - $iSize) / 2, $iXPos, $iYPos, $iAngle = 0
    Local Const $iDots = 16, $iAngelDist = 360 / $iDots, $iRadius = 200

    Do
        _GDIPlus_GraphicsClear($g_hGfxCtxt, 0xFF000000 + $iBgColor) ;clear bitmap with given color (AARRGGBB format)Efface le bitmap avec une couleur donnée (en format AARRGGBB)
        For $i = 1 To $iDots
            $iXPos = $iX_Center + Cos($iAngle * $iDeg) * $iRadius
            $iYPos = $iY_Center + Sin($iAngle * $iDeg) * $iRadius
            _GDIPlus_GraphicsFillEllipse($g_hGfxCtxt, $iXPos, $iYPos, $iSize, $iSize, $g_hBrush) ;draw dots in a circleDessine des points dans un cercle
            $iAngle += $iAngelDist ;increase angle to next dotAugmente l'angle pour le prochain point
        Next
        $iAngle += 1 ;increase overall angleAugmente l'angle global
        _GDIPlus_GraphicsDrawImageRect($g_hGfx, $g_hBitmap, 0, 0, $iWidth, $iHeight) ;copy drawn bitmap to GUICopie le bitmap dessiné dans la GUI
    Until Not Sleep(20) ;Sleep() always returns 1 and Not 1 is 0Sleep() retourne toujours 1 et non 1 ou 0
EndFunc   ;==>Example

Func _Exit() ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _GDIPlus_BrushDispose($g_hBrush)
    _GDIPlus_GraphicsDispose($g_hGfxCtxt)
    _GDIPlus_GraphicsDispose($g_hGfx)
    _GDIPlus_BitmapDispose($g_hBitmap)
    _GDIPlus_Shutdown()
    GUIDelete($g_hGUI)
    Exit
EndFunc   ;==>_Exit
