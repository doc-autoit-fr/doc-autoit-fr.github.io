_WinAPI_GetBufferedPaintTargetRect

Retrieves the target rectangle

#include <WinAPITheme.au3>
_WinAPI_GetBufferedPaintTargetRect ( $hBP )

ParametersParamètres

$hBP Handle of the buffered paint context.

Return ValueValeur de retour

Success:Succès : $tagRECT structure that contains the rectangle specified by _WinAPI_BeginBufferedPaint() function.
Failure:Échec : Sets the @error flag to non-zero, @extended flag may contain the HRESULT error code.

RemarksRemarques

This function requires Windows Vista or later.
