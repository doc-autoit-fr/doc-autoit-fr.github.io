_GDIPlus_GraphicsMeasureString

Measures the size of a stringMesure la taille d'une chaîne

#include <GDIPlus.au3>
_GDIPlus_GraphicsMeasureString ( $hGraphics, $sString, $hFont, $tLayout, $hFormat )

ParametersParamètres

$hGraphics Handle to a Graphics objectHandle de l'objet Graphics
$sString String to be drawnChaîne à dessiner
$hFont Handle to the font to use to draw the stringHandle de la police à utiliser pour dessiner la chaîne
$tLayout $tagGDIPRECTF structure that bounds the stringStructure $tagGDIPRECTF qui délimite la chaîne
$hFormat Handle to the string format to draw the stringHandle du format de chaîne pour dessiner la chaîne

Return ValueValeur de retour

Success:Succès : an array with the following format:Retourne un tableau au format suivant :
    [0] - $tagGDIPRECTF structure that receives the rectangle that bounds the string
    [1] - The number of characters that actually fit into the layout rectangle
    [2] - The number of lines that fit into the layout rectangle
Failure:Échec : sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic, $hBrush, $hFormat, $hFamily, $hFont, $tLayout, $aInfo
    Local $sString = "Hello world"

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    ; Draw a stringDessine une chaîne
    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    $hBrush = _GDIPlus_BrushCreateSolid(0xFF00007F)
    $hFormat = _GDIPlus_StringFormatCreate()
    $hFamily = _GDIPlus_FontFamilyCreate("Arial")
    $hFont = _GDIPlus_FontCreate($hFamily, 12, 2)
    $tLayout = _GDIPlus_RectFCreate(140, 110, 0, 0)
    $aInfo = _GDIPlus_GraphicsMeasureString($hGraphic, $sString, $hFont, $tLayout, $hFormat)
    _GDIPlus_GraphicsDrawStringEx($hGraphic, $sString, $hFont, $aInfo[0], $hFormat, $hBrush)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_FontDispose($hFont)
    _GDIPlus_FontFamilyDispose($hFamily)
    _GDIPlus_StringFormatDispose($hFormat)
    _GDIPlus_BrushDispose($hBrush)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
