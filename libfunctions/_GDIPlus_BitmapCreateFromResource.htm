_GDIPlus_BitmapCreateFromResource

Creates a Bitmap object based on an iconCrée un objet Bitmap à partir d'une icône

#include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromResource ( $hInst, $vResourceName )

ParametersParamètres

$hInst Handle to an instance of a module whose executable file contains a bitmap resourceHandle d'une instance d'un module dont le fichier exécutable contient une ressource bitmap
$vResourceName The resource name string or identifierLa chaîne du nom ou l'identifiant de la ressource

Return ValueValeur de retour

Success:Succès : a handle to a new Bitmap object.Retourne le handle d'un nouvel objet Bitmap.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

After you are done with the object, call _GDIPlus_ImageDispose() to release the object resources.Lorsque vous en avez terminé avec l'objet, appelez _GDIPlus_ImageDispose() pour libérer les ressources de l'objet.

RelatedLiens

_GDIPlus_ImageDispose, _WinAPI_GetModuleHandle

See AlsoVoir aussi

Search GdipCreateBitmapFromResource in MSDN Library.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()

    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $iWidth = 300, $iHeight = 300, $iBgColor = 0x404040 ;$iBgColor format RRGGBB$iBGColor est au format RRGGBB

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;create a test GUIcréer une interface de test
    GUISetBkColor($iBgColor, $hGUI) ;set GUI background colorDéfinit la couleur de fond de la GUI
    GUISetState(@SW_SHOW)

    Local $sAut2Exe = StringRegExpReplace(@AutoItExe, "\\\w+.exe", "\\Aut2Exe\\Aut2Exe.exe") ;get path to the Aut2Exe.exe fileObtient le chemin vers le fichier Aut2Exe.exe
    If @AutoItX64 Then $sAut2Exe = StringRegExpReplace(@AutoItExe, "\\\w+.exe", "\\Aut2Exe\\Aut2Exe_X64.exe")

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    Local $hInst = _WinAPI_LoadLibrary($sAut2Exe) ;maps a specified executable module into the address space of the calling processMappe un module exécutable spécifié dans l'espace d'adressage du processus appelant
    Local $hBitmap = _GDIPlus_BitmapCreateFromResource($hInst, "BMP_MAINLOGO") ;load bitmap resource "BMP_MAINLOGO" and convert it to GDI+ bitmap formatCharge la ressource bitmap "BMP_MAINLOGO" et la convertit au format bitmap GDI+
    Local $iW = _GDIPlus_ImageGetWidth($hBitmap), $iH = _GDIPlus_ImageGetHeight($hBitmap)
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap, ($iWidth - $iW) / 2, ($iHeight - $iH) / 2) ;display image in GUI centeredAffiche l'image dans la GUI centrée

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _WinAPI_FreeLibrary($hInst)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
    Exit
EndFunc   ;==>Example
