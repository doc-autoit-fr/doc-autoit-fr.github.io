_Word_Create

Connects to an existing Word instance or creates a new Word application object

#include <Word.au3>
_Word_Create ( [$bVisible = True [, $bForceNew = False]] )

ParametersParamètres

$bVisible [optional][optionnel] True specifies that the window will be visible (default)
$bForceNew [optional][optionnel] True forces to create a new Word instance even if there is already a running instance (default = False)

Return ValueValeur de retour

Success:Succès : a variable pointing to a Word.Application object. Sets @extended to:
0 - Word was already running
1 - Word was not running or $bForceNew was set to True. Word has been started by this function
Failure:Échec : 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0.
@error:@error : 1 - Error returned by function ObjCreate(). @extended is set to the COM error code
