_PathFull

Creates a path based on the relative path you provide.Crée un chemin d'accès basé sur le chemin d'accès relatif que vous fournissez. The newly created absolute path is returnedLe chemin d'accès absolu nouvellement créé est retourné.

#include <File.au3>
_PathFull ( $sRelativePath [, $sBasePath = @WorkingDir] )

ParametersParamètres

$sRelativePath The relative path to be createdLe chemin d'accès relatif à être créé
$sBasePath [optional][optionnel][optionnel] The base path to be used.Le chemin de base à utiliser. Default = @WorkingDir

Return ValueValeur de retour

Returns a relative path will be built relative to $sBasePath.Retourne un chemin d'accès relatif qui sera construit par rapport à $sBasePath. To bypass this behavior, use an absolute path.Pour contourner ce comportement, utilisez un chemin d'accès absolu.

RemarksRemarques

UNC paths are supported.Les chemins UNC ne sont pas pris en charge.
Pass "\" to get the root drive of $sBasePath.Utilisez "\" pour obtenir la racine du lecteur de $sBasePath.
Pass "" or "." to return $sBasePath.Utilisez "" ou "." pour retourner $sBasePath.

ExampleExemple

#include <File.au3>
#include <MsgBoxConstants.au3>

Local $sTestPath = _PathFull(@ScriptDir & "\..\..\test")
