_FTP_DirPutContents

Puts an folder on an FTP server. Recursivley if selected

#include <FTPEx.au3>
_FTP_DirPutContents ( $hFTPSession, $sLocalFolder, $sRemoteFolder, $bRecursivePut [, $iContext = 0] )

ParametersParamètre

$hFTPSession as returned by _FTP_Connect().
$sLocalFolder The local folder i.e. "c:\temp".
$sRemoteFolder The remote folder i.e. '/website/home'.
$bRecursivePut Recurse through sub-dirs. 0=Non recursive, 1=Recursive
$iContext [optional][optional] False - i, v, x ... A variable that contains the application-defined value that associates this search with any application data.
This is only used if the application has already called _FTP_SetStatusCallback() to set up a status callback function.

Return ValueRemarque

Success: the directory name.
Failure: 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0 :
