_SetTime

Sets the current time of the system

#include <Date.au3>
_SetTime ( $iHour, $iMinute [, $iSecond = 0 [, $iMSeconds = 0]] )

ParametersParamètres

$iHour the hour. Values: 0-23
$iMinute the minute. Values: 0-59
$iSecond [optional][optionnel] the seconds. Values: 0-59
$iMSeconds [optional][optionnel] the milliseconds. Values: 0-999

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0.
@error:@error : 1 - Failure
@extended: _WinAPI_GetLastError()
Error code(s): MSDN: ms681381.aspx

RemarksRemarques

If the optional parameters ($iSecond and iMSeconds) are not defined, the function will not change the current value.
