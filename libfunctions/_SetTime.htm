_SetTime

Sets the current time of the systemDéfinit l'heure actuelle du système

#include <Date.au3>
_SetTime ( $iHour, $iMinute [, $iSecond = 0 [, $iMSeconds = 0]] )

ParametersParamètres

$iHour the hour.les heures. Values:Valeurs : 0-23
$iMinute the minute.les minutes. Values:Valeurs : 0-59
$iSecond [optional][optionnel][optionnel] the seconds.les secondes. Values:Valeurs : 0-59
$iMSeconds [optional][optionnel][optionnel] the milliseconds.les millisecondes. Values:Valeurs : 0-999

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 and sets the @error flag to non-zero.0 et @error <> 0.
@error:@error : 1 - Failure1 - Échec
@extended:@extended : _WinAPI_GetLastError()
Error code(s):Code(s) d'erreur : MSDN: ms681381.aspxMSDN : ms681381.aspx

RemarksRemarques

If the optional parameters ($iSecond and iMSeconds) are not defined, the function will not change the current value.Si les paramètres optionnels ($iSecond et $iMSeconds) ne sont pas définis, la fonction ne changera pas la valeur actuelle.
