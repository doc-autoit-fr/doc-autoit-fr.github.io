_GDIPlus_BitmapCreateApplyEffect

Creates a new Bitmap by applying a specified effect to an existing BitmapCrée un nouveau Bitmap en appliquant un effet spécifié à un Bitmap existant

#include <GDIPlus.au3>
_GDIPlus_BitmapCreateApplyEffect ( $hBitmap, $hEffect [, $tRECT = Null [, $tOutRECT = Null]] )

ParametersParamètres

$hBitmap Handle to a Bitmap to which the effect is applied.Handle du bitmap auquel le traitement est appliqué.
$hEffect Handle to an Effect to be applied.Handle de l'effet à appliquer.
$tRECT [optional][optionnel][optionnel] a $tagRECT structure that specifies the portion of the input bitmap that is used.Structure $tagRECT qui définit la partie du bitmap d'entrée qui est utilisé.
$tOutRECT [optional][optionnel][optionnel] a $tagRECT structure that receives the portion of the input bitmap that was used.Structure $tagRECT qui reçoit la partie du bitmap d'entrée qui est utilisé.

Return ValueValeur de retour

Success:Succès : a handle to a Bitmap object.Retourne le handle de l'objet Bitmap
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).
@error:@error : -1 - GDIPlus.dll does not support this function.-1 - GDIPlus.dll ne supporte pas cette fonction.
10 - Invalid parameters.10 - Paramètres invalides.

RemarksRemarques

When you are done with the Effect object, call _GDIPlus_EffectDispose() to release the resources.Lorsque vous enavez terminé avec l'objet Effect, appelez _GDIPlus_EffectDispose() pour libérer les ressources.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

_Example()

Func _Example()
    If Not _GDIPlus_Startup() Or @extended < 6 Then
        MsgBox($MB_SYSTEMMODAL, "ERROR", "GDIPlus.dll v1.1 not available")
        Return
    EndIf

    Local $sFile = FileOpenDialog("Select an image", "", "Images (*.bmp;*.png;*.jpg;*.gif;*.tif)")
    If @error Or Not FileExists($sFile) Then Return

    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile)

    Local $hEffect = _GDIPlus_EffectCreateBlur()

    Local $tRECT = DllStructCreate($tagRECT)
    Local $hBitmap = _GDIPlus_BitmapCreateApplyEffect($hImage, $hEffect, Null, $tRECT)

    Local $iBmpW = $tRECT.Right - $tRECT.Left
    Local $iBmpH = $tRECT.Bottom - $tRECT.Top

    Local $iWidth = 600
    Local $iHeight = $iBmpH * 600 / $iBmpW

    Local $hGui = GUICreate("GDI+ v1.1 (" & @ScriptName & ")", $iWidth, $iHeight)
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)
    GUISetState(@SW_SHOW)

    _GDIPlus_GraphicsDrawImageRect($hGraphics, $hBitmap, 0, 0, $iWidth, $iHeight)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    _GDIPlus_EffectDispose($hEffect)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>_Example
