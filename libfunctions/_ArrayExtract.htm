_ArrayExtract

Extracts an array from the specified element(s) of a 1D or 2D arrayExtrait un tableau à partir des éléments spécifiés d'un tableau 1D ou 2D

#include <Array.au3>
_ArrayExtract ( Const ByRef $aArray [, $iStart_Row = -1 [, $iEnd_Row = -1 [, $iStart_Col = -1 [, $iEnd_Col = -1]]]] )

ParametersParamètres

$aArray Array from which extraction should occurTableau à partir de laquelle doit se faire l'extraction
$iStart_Row [optional][optionnel] First row of the extracted arrayPremière ligne du tableau extrait
$iEnd_Row [optional][optionnel] Last row of the extracted arrayDernière ligne du tableau extrait
$iStart_Col [optional][optionnel] First column of the extracted array (2D only)La première colonne du tableau extrait (2D uniquement )
$iEnd_Col [optional][optionnel] Last column of the extracted array (2D only)

Return ValueValeur de retour

Success:Succès : 1Retourne 1

ExampleExemple

#include <Array.au3>

Local $aArray[4][4]
For $i = 0 To 3
    For $j = 0 To 3
        $aArray[$i][$j] = $i & $j
    Next
Next
_ArrayDisplay($aArray, "Original")
Local $aExtract = _ArrayExtract($aArray, 1, 2, 2, 3)
_ArrayDisplay($aExtract, "Row 1-2 cols 2-3")
