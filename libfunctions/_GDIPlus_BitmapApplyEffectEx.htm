_GDIPlus_BitmapApplyEffectEx

Alters a Bitmap by applying a specified effectModifie un bitmap en appliquant un effet spécifié

#include <GDIPlus.au3>
_GDIPlus_BitmapApplyEffectEx ( $hBitmap, $hEffect [, $iX = 0 [, $iY = 0 [, $iW = 0 [, $iH = 0]]]] )

ParametersParamètres

$hBitmap Handle to a Bitmap to which the effect is applied.Handle du bitmap auquel le traitement est appliqué.
$hEffect Handle to an Effect to be applied.Handle de l'effet à appliquer.
$iX [optional][optionnel] The X coordinate of the upper left corner of the portion to which the effect is applied.La coordonnée X du coin supérieur gauche de la partie à laquelle l'effet est appliqué.
$iY [optional][optionnel] The Y coordinate of the upper left corner of the portion to which the effect is applied.La coordonnée Y du coin supérieur gauche de la partie à laquelle l'effet est appliqué.
$iW [optional][optionnel] Specifies the width of the portion to which the effect is applied.Spécifie la largeur de la partie à laquelle l'effet est appliqué.
$iH [optional][optionnel] Specifies the height of the portion to which the effect is applied.Spécifie la hauteur de la partie à laquelle l'effet est appliqué.

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).
@error:@error : -1 - GDIPlus.dll does not support this function.-1 - GDIPlus.dll ne supporte pas cette fonction.
10 - Invalid parameters.10 - Paramètres invalides.

RemarksRemarques

Pass 0 to the coordinate parameters to specify that the effect applies to the entire input bitmap (Default).Passez 0 dans les paramètres des coordonnées pour spécifier que l'effet s'applique au bitmap d'entrée entier (par défaut).

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

_Example()

Func _Example()
    If Not _GDIPlus_Startup() Or @extended < 6 Then
        MsgBox($MB_SYSTEMMODAL, "ERROR", "GDIPlus.dll v1.1 not available")
        Return
    EndIf

    Local $sFile = FileOpenDialog("Select an image", "", "Images (*.bmp;*.png;*.jpg;*.gif;*.tif)")
    If @error Or Not FileExists($sFile) Then Return

    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile)
    Local $iImgW = _GDIPlus_ImageGetWidth($hImage)
    Local $iImgH = _GDIPlus_ImageGetHeight($hImage)

    Local $iWidth = 600
    Local $iHeight = $iImgH * 600 / $iImgW

    Local $hGui = GUICreate("GDI+ v1.1 (" & @ScriptName & ")", $iWidth, $iHeight)
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)
    GUISetState(@SW_SHOW)

    Local $hEffect = _GDIPlus_EffectCreateHueSaturationLightness(0, -100, 0)
    _GDIPlus_BitmapApplyEffectEx($hImage, $hEffect, $iImgW * 0.25, $iImgH * 0.25, $iImgW * 0.5, $iImgH * 0.5)

    _GDIPlus_GraphicsDrawImageRect($hGraphics, $hImage, 0, 0, $iWidth, $iHeight)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    _GDIPlus_EffectDispose($hEffect)
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>_Example
