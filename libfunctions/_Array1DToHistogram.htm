_Array1DToHistogram

Transform a 1D array to HistogramTransforme un tableau 1D en histogramme

#include <Array.au3>
_Array1DToHistogram ( $aArray [, $iSizing = 100] )

ParametersParamètres

$aArray Array to transformTableau à transformer
$iSizing [optional][optionnel][optionnel] Allows to zoom the histogram.Permet de modifier l'échelle de l'histogramme. Default = 100 percentPar défaut, 100%.

Return ValueValeur de retour

Success:Succès : the array transformed.Retourne un tableau contenant les données de l'histogramme.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.
@error:@error : 1 - $aArray is not a 1D array1 - $aArray n'est pas un tableau 1D

RemarksRemarques

Right alignment of negative values is not perfect due to non fixed font usage.L'alignement à droite des valeurs négatives n'est pas parfaite en raison de l'utilisation des polices non fixes.

ExampleExemple

Example 1Exemple 1

#include <Array.au3>

Local $aNEW[] = [0, -80, -49, -44, 80, 100, 8, 7, 6, 5, 4, 3, 2, 1]
Local $aArray = _Array1DToHistogram($aNEW, 125)
_ArrayDisplay($aArray, "_Array1DToHistogram")

Example 2Exemple 2

#include <Array.au3>

Local $iLimit = 500

Local $t, $n
