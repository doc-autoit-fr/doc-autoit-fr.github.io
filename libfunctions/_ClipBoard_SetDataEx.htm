_ClipBoard_SetDataEx

Places data on the clipboard in a specified clipboard formatPlace des données dans le presse-papiers dans un format spécifié de presse-papiers

#include <Clipboard.au3>
_ClipBoard_SetDataEx ( ByRef $hMemory [, $iFormat = 1] )

ParametersParamètres

$hMemory Handle to the data in the specified format.Handle des données dans le format spécifié. This parameter can be NULL, indicating that the window provides data in the specified clipboard format upon request.Ce paramètre peut être NULL, ce qui indique que la fenêtre fournit des données dans le format spécifié du presse-papiers sur demande.
If a window delays rendering, it must process the $WM_RENDERFORMAT and $WM_RENDERALLFORMATS messages.Si une fenêtre retarde à s'afficher, elle doit traiter les messages $WM_RENDERFORMAT et $WM_RENDERALLFORMATS.
If this function succeeds, the system owns the object identified by the $hMemory parameter.Si cette fonction réussit, le système s'approprie l'objet identifié par le paramètre $hMEMORY.
The application may not write to or free the data once ownership has been transferred to the system, but it can lock and read from the data until the _ClipBoard_Close() function is called.L'application ne peut écrire ou libérer les données une fois que la propriété a été transférée au système, mais elle peut verrouiller et lire les données jusqu'à ce que la fonction _ClipBoard_Close() soit appelée.
The memory must be unlocked before the clipboard is closed.La mémoire doit être déverrouillée avant que le presse-papiers soit fermé.
If the $hMemory parameter identifies a memory object, the object must have been allocated using the function with the $GMEM_MOVEABLE flag.Si le paramètre $hMEMORY identifie un objet mémoire, l'objet doit avoir été alloué à l'aide de la fonction avec le flag $GMEM_MOVEABLE.

RemarksRemarques

The $iFormat parameter can identify a registered clipboard format, or it can be one of the standard clipboard formats.Le paramètre $iFormat peut identifier un format de presse-papiers enregistré, ou il peut être l'un des formats standards du presse-papiers.
If an application calls this function in response to $WM_RENDERFORMAT or $WM_RENDERALLFORMATS, the application should not use the handle after this function has been called.Si une application appelle cette fonction en réponse à $WM_RENDERFORMAT ou $WM_RENDERALLFORMATS, l'application ne doit pas utiliser le handle après que cette fonction ait été appelée.
If an application calls _ClipBoard_Open() with a NULL handle, _ClipBoard_Empty() sets the clipboard owner to NULL; this causes this function to fail.Si une application appelle _ClipBoard_Open() avec un handle NULL, _ClipBoard_Empty() définit le propriétaire du presse-papiers à NULL; ce qui fait échouer la fonction.

ExampleExemple

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <WinAPI.au3>
#include <WindowsConstants.au3>

Global $g_idMemo

Example()

Func Example()
    Local $id_SetData, $id_GetData, $hMemory, $hLock, $tData, $sData, $iSize

    ; Create GUICrée la GUI
    GUICreate("Clipboard", 600, 450)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    $id_SetData = GUICtrlCreateButton("Set ClipBoard Data", 150, 410, 120, 30)
    $id_GetData = GUICtrlCreateButton("Get ClipBoard Data", 300, 410, 120, 30)
    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $id_SetData
                ; Open clipboardOuvrir le presse-papiers
                If _ClipBoard_Open(0) Then

                    ; Empty clipboardPresse-papiers vide
                    If _ClipBoard_Empty() Then

                        ; Create global memory buffer (show why using _ClipBoard_SetData is MUCH easier!)Crée tampon de mémoire global (montre pourquoi l'utilisation de _ClipBoard_SetData est beaucoup plus facile!)
                        $sData = "Hello from AutoIt"
                        $iSize = StringLen($sData) + 1
                        $hMemory = _MemGlobalAlloc($iSize, $GHND)
                        If $hMemory <> 0 Then
                            $hLock = _MemGlobalLock($hMemory)
                            If $hLock = 0 Then _WinAPI_ShowError("_Mem_GlobalLock failed")
                            $tData = DllStructCreate("char Text[" & $iSize & "]", $hLock)
                            DllStructSetData($tData, "Text", $sData)
                            _MemGlobalUnlock($hMemory)

                            ; Write clipboard textEcrit du texte dans le presse-papiers
                            If Not _ClipBoard_SetDataEx($hMemory, $CF_TEXT) Then _WinAPI_ShowError("_ClipBoard_SetDataEx failed")
                        Else
                            _WinAPI_ShowError("_Mem_GlobalAlloc failed")
                        EndIf

                        ; Close clipboardFerme le presse-papiers
                        _ClipBoard_Close()
                    Else
                        ; Close clipboardFerme le presse-papiers
                        _ClipBoard_Close()
                        _WinAPI_ShowError("_ClipBoard_Empty failed")
                    EndIf
                Else
                    _WinAPI_ShowError("_ClipBoard_Open failed")
                EndIf

            Case $id_GetData
                MemoWrite(_ClipBoard_GetData())
        EndSwitch
    WEnd
EndFunc   ;==>Example

; Write message to memoEcrire un message dans memo
Func MemoWrite($sMessage = "")
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
