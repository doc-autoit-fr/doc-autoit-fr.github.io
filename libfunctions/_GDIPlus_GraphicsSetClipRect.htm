_GDIPlus_GraphicsSetClipRect

Updates the clipping region of a Graphics object to a region that is the combination of itself and a rectangleRedéfinit la région de découpage d'un objet Graphics à une région qui est la combinaison d'elle-même et d'un rectangle

#include <GDIPlus.au3>
_GDIPlus_GraphicsSetClipRect ( $hGraphics, $nX, $nY, $nWidth, $nHeight [, $iCombineMode = 0] )

ParametersParamètres

$hGraphics Pointer to a Graphics objectLe handle de l'objet Graphics
$nX X coordinate of the upper-left corner of the rectangleCoordonnée X du coin supérieur gauche du rectangle
$nY Y coordinate of the upper-left corner of the rectangleCoordonnée Y du coin supérieur gauche du rectangle
$nWidth Width of the rectangleLargeur du rectangle
$nHeight Height of the rectangleHauteur du rectangle
$iCombineMode [optional][optionnel][optionnel] Regions combination mode:Mode de combinaison des régions:
    0 - The existing region is replaced by the new region
    1 - The existing region is replaced by the intersection of itself and the new region
    2 - The existing region is replaced by the union of itself and the new region
    3 - The existing region is replaced by the result of performing an XOR on the two regions
    4 - The existing region is replaced by the portion of itself that is outside of the new region
    5 - The existing region is replaced by the portion of the new region that is outside of the existing region

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI = GUICreate("GDI+", 320, 240)
    GUICtrlCreateButton("Button", 20, 20, 100, 25)
    GUICtrlCreateSlider(100, 100, 100, 30)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;Create a graphics object from a window handleCrée une GUI

    _GDIPlus_GraphicsSetClipRect($hGraphics, 20, 20, 100, 25, 4) ;subtract button-area from clipregionSoustrait la zone du bouton de la zone de découpage
    _GDIPlus_GraphicsSetClipRect($hGraphics, 100, 100, 100, 30, 4) ;subtract slider-area from clipregionSoustrait la zone du slider de la zone de découpage

    Local $iTimer = TimerInit()
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
        If TimerDiff($iTimer) > 100 Then
            _GDIPlus_GraphicsClear($hGraphics, BitOR(0xFF000000, Random(0, 0xFFFFFF, 1)))
            $iTimer = TimerInit()
        EndIf
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
