_ArrayCombinations

Returns an array of the combinations of a set of elements from a selected 1D arrayRetourne le tableau des combinaisons de taille donnée de l'ensemble des éléments d'un tableau

#include <Array.au3>
_ArrayCombinations ( Const ByRef $aArray, $iSet [, $sDelimiter = ""] )

ParametersParamètres

$aArray The Array to useLe tableau à traiter
$iSet Size of the combinations set
$sDelimiter [optional][optionnel][optionnel] String result separator, default is "" for noneSéparateur à utiliser dans les chaînes résultats , par défaut est "" pour aucun séparateur.

Return ValueValeur de retour

Success:Succès : an array of Combinations
The first element ($aArray[0]) contains the number of strings returned.
The remaining elements ($aArray[1], $aArray[2], etc.) contain the Combinations.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.
@error:@error : 1 - The Input Must be an Array1 - L'entrée doit être un tableau
2 - $aArray is not a 1D array2 - $aArray n'est pas un tableau 1D

RemarksRemarques

The input array must be 0-based, i.e. no counter in $aArray[0].Le tableau d'entrée doit être de base 0, c'est à dire qu'il ne doit pas y avoir le nombre d'éléments dans $aArray[0]. Based on an algorithm by Kenneth H. Rosen.

ExampleExemple

; Declare a 1-dimensional array, and create an array showing the Possible Combinations

#include <Array.au3>

Local $aArray[5] = [1, 2, 3, 4, 5]

For $i = 1 To UBound($aArray)
    Local $aArrayCombo = _ArrayCombinations($aArray, $i, ",")
    _ArrayDisplay($aArrayCombo, "iSet = " & $i)
Next
