_GDIPlus_ImageSaveToStream

Saves an Image object to a streamEnregistre un objet Image dans un flux

#include <GDIPlus.au3>
_GDIPlus_ImageSaveToStream ( $hImage, $pStream, $tEncoder [, $tParams = 0] )

ParametersParamètres

$hImage Pointer to an Image objectHandle de l'objet Image
$pStream Pointer to an IStream interfaceHandle de l'interface IStream
$tEncoder a $tagGUID structure that defines the image encoder GUID
$tParams [optional][optionnel] a $tagGDIPENCODERPARAMS structure or a pointer to it

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

RemarksRemarques

Use this function to save selected frames from a multiple-frame image to another multiple-frame image.Utilisez cette fonction pour enregistrer des trames sélectionnées à partir d'une image à trames multiples vers une autre image à trames multiples.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    _GDIPlus_Startup()
    Local Const $iW = @DesktopWidth / 2, $iH = @DesktopHeight / 2
    Local $hGUI = GUICreate("GDI+ test", $iW, $iH, -1, -1)
    GUISetState(@SW_SHOW)

    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, @DesktopWidth / 2, @DesktopHeight / 2) ;create a GDI bitmap by capturing 1/4 of desktopCrée un bitmap GDI en capturant un quart de bureau
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;convert GDI bitmap to GDI+ bitmapConverti le bitmap GDI en GDI+
    _WinAPI_DeleteObject($hHBmp) ;release GDI bitmap resource because not needed anymoreLibére la ressource du bitmap GDI car n'est plus nécessaire

    Local $sImgCLSID = _GDIPlus_EncodersGetCLSID("jpg") ;create CLSID for a JPG image file typeCrée le CLSID pour un type JPG de fichier d'image
    Local $tGUID = _WinAPI_GUIDFromString($sImgCLSID) ;convert CLSID GUID to binary form and returns $tagGUID structureConvertit le GUID du CLSID sous forme binaire et retourne la structure $tagGUID
    Local $tParams = _GDIPlus_ParamInit(1) ;initialize an encoder parameter list and return $tagGDIPENCODERPARAMS structureInitialise une liste de paramètres de l'encodeur et retourne la structure $tagGDIPENCODERPARAMS
    Local $tData = DllStructCreate("int Quality") ;create struct to set JPG quality settingCrée la struct pour définir le degré de qualité JPG
    DllStructSetData($tData, "Quality", 10) ;quality 0-100 (0: lowest, 100: highest)Qualité 0-100 (0 : plus bas, 100 : le plus élevé)
    Local $pData = DllStructGetPtr($tData) ;get pointer from quality structObtient le pointeur sur la struct de qualité
    _GDIPlus_ParamAdd($tParams, $GDIP_EPGQUALITY, 1, $GDIP_EPTLONG, $pData) ;add a value to an encoder parameter listAjoute une valeur à la liste des paramètres de l'encodeur
    Local $pStream = _WinAPI_CreateStreamOnHGlobal() ;create streamCrée le flux
    _GDIPlus_ImageSaveToStream($hBitmap, $pStream, $tGUID, $tParams) ;save the bitmap in JPG format in memoryEnregistre l'image au format JPG dans la mémoire
    Local $hBitmapFromStream = _GDIPlus_BitmapCreateFromStream($pStream) ;create bitmap from a stream (here from the JPG in memory)Crée le bitmap à partir du flux (ici du JPG en mémoire)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmapFromStream, 0, 0) ;display streamed imageAffiche l'image en streaming

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    ;cleanup resourcesNettoie les ressources
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_BitmapDispose($hBitmapFromStream)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
