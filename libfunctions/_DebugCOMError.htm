_DebugCOMError

Sets, resets or queries the debug level for COM errorsDéfinit, réinitialise ou interroge le niveau de débogage pour les erreurs COM

#include <Debug.au3>
_DebugCOMError ( [$iComDebug = 1 [, $bExit = False]] )

ParametersParamètres

$iComDebug [optional] Debug level.Niveau de débogage. Possible values are:Les valeurs possibles sont :
-1 - Return the current settings-1 - Retourne les paramètres actuels
 0 - Disable debugging 0 - Désactive le débogage
 1 - Enable debugging. 1 - Active le débogage. Output the debug info (default)Affiche les informations de débogage (par défaut)
$bExit [optional] True if the script must be terminatedTrue si le script doit être terminé

Return ValueValeur de retour

Success: 1 if $iComDebug = 0 or 1 and sets @extended to:1 si $iComDebug = 0 ou 1 et définit @extended à :
0 - The COM error handler is not initialized0 - Le gestionnaire d'erreur COM n'est pas initialisé
1 - The COM error handler has been initialized1 - Le gestionnaire d'erreur COM a été initialisé
2 - The COM error handler was already active2 - Le gestionnaire d'erreur COM était déjà actif
3 - The COM error handler is already disabled3 - Le gestionnaire d'erreur COM est déjà désactivé
1 if $iComDebug = -1 and sets the @error flag and @extented to current setting:1 si $iComDebug = -1 et définit les drapeaux @error et @extented d'après le réglage actuel :
@error = current Debug level@error = niveau de Débogage actuel
@extended = current script termination.@extended = résiliation de script en cours.
Failure: 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0.
@error:@error : 1 - $iComDebug is not an integer or < -1 or > 11 - $iComDebug n'est pas un nombre entier ou < -1 or > 1
2 - The COM error handler is already set to another function (= not set by this function)2 - Le gestionnaire d'erreur COM est déjà défini sur une autre fonction (= pas défini par cette fonction)
3 - _DebugSetup() did not run properly.3 - _DebugSetup() ne fonctionne pas correctement. Make sure _DebugSetup() ran properly before calling this functionAssurez-vous que _DebugSetup() s'est exécuté correctement avant d'appeler cette fonction
4 - Installation of the custom error handler failed.4 - L'installation du gestionnaire d'erreurs personnalisé a échoué. @extended is set to the error code returned by ObjEvent()@extended est défini sur le code d'erreur retourné par ObjEvent()

RemarksRemarques

Before calling this function, _DebugSetup() must be called first to create the debug session.Avant d'appeler cette fonction, _DebugSetup() doit être appelée en premier pour créer la session de débogage.

; *****************************************************************************
; ExampleExemple
; Generate a COM error and let _DebugCOMError display the errorGénère une erreur COM et laisse _DebugCOMError afficher l'erreur
; *****************************************************************************
Func Example()
    ; Create application object for the Microsoft Internet ExplorerCrée un objet application pour Microsoft Internet Explorer
    Local $oIE = ObjCreate("InternetExplorer.Application")
    If @error <> 0 Then Exit MsgBox($MB_SYSTEMMODAL, "_DebugCOMError UDFUDF _DebugCOMError", "Error creating a new IE application object.Erreur lors de la création d'un nouvel objet application IE." & _
            @CRLF & "@error = " & @error & ", @extended = " & @extended)
    ; Set up the COM error handler to write error messages using a GUI and exit on errorMet en place le gestionnaire d'erreur COM pour écrire les messages d'erreur via une interface graphique (GUI) et quitte lors d'une erreur
    _DebugCOMError(1)
    ; Provoke a COM error.Provoque une erreur COM. Functions is unknownFonction inconnue
    $oIE.xyz()
    If @error <> 0 Then Return MsgBox($MB_SYSTEMMODAL, "_DebugCOMError UDFUDF _DebugCOMError", "Error occurred and handled.Erreur produite et gérée." & _
            @CRLF & "@error = " & @error)
    ; Close the IEFerme IE
    $oIE.Quit()
    $oIE = 0
EndFunc   ;==>Example
