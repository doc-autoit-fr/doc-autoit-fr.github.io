_GDIPlus_LineBrushSetBlend

Sets the blend factors and the blend positions of a linear gradient brush to create a custom blendDéfinit les taux de mélange des couleurs et des positions associées pour créer un dégradé personnalisé

#include <GDIPlus.au3>
_GDIPlus_LineBrushSetBlend ( $hLineGradientBrush, $aBlends )

ParametersParamètres

$hLineGradientBrush Pointer to a LinearGradientBrush objectHandle de l'objet LinearGradientBrush
$aBlends Array of blend factors and blend positions:
    [0][0] - Number of blend factors and blend positions, must be at least 2
    [1][0] - Factor 1
    [1][1] - Position 1
    [2][0] - Factor 2
    [2][1] - Position 2
    [n][0] - Factor n
    [n][1] - Position n

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

RemarksRemarques

Each factor in the array specifies a percentage of the ending color and should be in the range 0.0 to 1.0.
Each position in the array indicates a percentage of the distance between the starting boundary and the ending boundary and is in the range 0.0 to 1.0, where 0.0 indicates the starting boundary of the gradient and 1.0 indicates the ending boundary.

See AlsoVoir aussi

Search GdipSetLineBlend in MSDN Library.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $iWidth = 600, $iHeight = 600, $iBgColor = 0x303030 ;$iBgColor format RRGGBB$iBGColor est au format RRGGBB

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;create a test GUIcréer une interface de test
    GUISetBkColor($iBgColor, $hGUI) ;set GUI background colorDéfinit la couleur de fond de la GUI
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    _GDIPlus_GraphicsSetSmoothingMode($hGraphics, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;sets the graphics object rendering quality (antialiasing)Définit l'objet graphique en qualité de rendu antialiasing
    Local $hBrush = _GDIPlus_LineBrushCreate(0, 0, 280, 570, 0xFFFFFF00, 0xFF4020FF, 1) ;create linear gradient flipped brushCrée un pinceau à dégradé linéaire renversé

    Local $aBlends[5][2]
    $aBlends[0][0] = 4 ; Using 4 factors and positionUtilise 4 facteurs et 4 emplacements
    $aBlends[1][0] = 0 ; FactorFacteur
    $aBlends[1][1] = 0 ; PositionEmplacement
    $aBlends[2][0] = 1 ; The percentage of the blending is 100% (red to green is gradually completed in just 20% from the left)Le pourcentage du mélange est de 100% (le dégradé du rouge au vert est progressivement complété dans seulement 20% à partir de la gauche)
    $aBlends[2][1] = 0.2 ; The distance percentage from the left boundary of the brush is 20%Le pourcentage de la distance de la frontière gauche du pinceau est de 20%
    $aBlends[3][0] = 0 ; The percentage of the blending is 0% (green to red is gradually completed from 20% to 70% from the left)Le pourcentage de mélange est de 0% (le dégradé du vert au rouge passe progressivement de 20% à 70 % à partir la gauche)
    $aBlends[3][1] = 0.7 ; The distance percentage from the left boundary of the brush is 70%Le pourcentage de la distance de la frontière gauche du pinceau est de 70%
    $aBlends[4][0] = 1 ; The percentage of the blending is 100% (red to green is gradually completed from 70% to 100% from the left)Le pourcentage de mélange est de 100% (le dégradé du rouge au vert est progressivement complété de 70% à 100 % à partir de la gauche
    $aBlends[4][1] = 1 ; The distance percentage from the left boundary of the brush is 100%Le pourcentage de la distance de la frontière gauche du pinceau est à 100%

    _GDIPlus_LineBrushSetBlend($hBrush, $aBlends); set the linear gradient brush color positions and factorsDéfinit les emplacements de couleurs de pinceau de dégradé linéaire et les facteurs

    _GDIPlus_GraphicsFillEllipse($hGraphics, 100, 50, 380, 500, $hBrush) ;draw the eggDessine l'œuf

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _GDIPlus_BrushDispose($hBrush)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
