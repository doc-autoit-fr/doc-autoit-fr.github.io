_GDIPlus_BitmapCreateHBITMAPFromBitmap

Create a handle to a bitmap from a bitmap objectCrée le handle d'un Bitmap à partir d'un objet Bitmap

#include <GDIPlus.au3>
_GDIPlus_BitmapCreateHBITMAPFromBitmap ( $hBitmap [, $iARGB = 0xFF000000] )

ParametersParamètres

$hBitmap Handle to a bitmap objectHandle de l'objet Bitmap
$iARGB [optional][optionnel][optionnel] Color object that specifies the background colorCouleur de l'arrière-plan

Return ValueValeur de retour

Success:Succès : a handle to a HBITMAP.Retourne le handle du HBITMAP
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

When you are done with the Bitmap object, call _WinAPI_DeleteObject() to release the resources.Lorsque vous en avez terminé avec l'objet bitmap, appelez la fonction _WinAPI_DeleteObject pour libérer les ressources

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    Local $hBMP, $hImage, $iX, $iY, $hClone

    ; Initialize GDI+ libraryInitialise GDI+ Librairie
    _GDIPlus_Startup()

    ; Capture 32 bit bitmapCapture un bitmap 32 bits
    $hBMP = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBMP)

    ; Create 24 bit bitmap cloneCrée un clone 24 bits
    $iX = _GDIPlus_ImageGetWidth($hImage)
    $iY = _GDIPlus_ImageGetHeight($hImage)
    $hClone = _GDIPlus_BitmapCloneArea($hImage, 0, 0, $iX, $iY, $GDIP_PXF24RGB)

    ; Save bitmap to fileEnregistre le bitmap dan un fichier
    _GDIPlus_ImageSaveToFile($hClone, @TempDir & "\GDIPlus_Image.bmp")

    ; Clean up resourcesLibère les ressources
    _GDIPlus_BitmapDispose($hClone)
    _GDIPlus_BitmapDispose($hImage)
    _WinAPI_DeleteObject($hBMP)

    ; Load imageCharge l'image
    $hImage = _GDIPlus_ImageLoadFromFile(@TempDir & "\GDIPlus_Image.bmp")
    $hBMP = _GDIPlus_BitmapCreateHBITMAPFromBitmap($hImage)

    ; Save bitmap to fileEnregistre le bitmap dan un fichier
    _ScreenCapture_SaveImage(@TempDir & "\Image.bmp", $hBMP, True) ; True -> $hBMP destroyedTrue - > détruit $hBMP

    ; Clean up resourceNettoie les ressources
    _GDIPlus_ImageDispose($hImage)

    ; Shut down GDI+ libraryFerme la librairie GDI+
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
