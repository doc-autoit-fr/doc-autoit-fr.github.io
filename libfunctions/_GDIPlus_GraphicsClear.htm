_GDIPlus_GraphicsClear

Clears a Graphics object to a specified colorEfface un objet Graphics avec une couleur spécifiée

#include <GDIPlus.au3>
_GDIPlus_GraphicsClear ( $hGraphics [, $iARGB = 0xFF000000] )

ParametersParamètres

$hGraphics Handle to a Graphics objectHandle de l'objet Graphics
$iARGB [optional][optionnel][optionnel] Alpha, Red, Green and Blue components of colorComposantes de la couleur:Alpha, Rouge, Vert et Bleu

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

See AlsoVoir aussi

Search GdipGraphicsClear in MSDN Library.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPI.au3>

Example()

Func Example()
    Local $hBitmap, $hImage, $hGraphic

    ; Initialize GDI+ libraryInitialise GDI+ Librairie
    _GDIPlus_Startup()

    ; Capture screen regionCapture une région de l'écran
    $hBitmap = _ScreenCapture_Capture("", 0, 0, 400, 300)
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; Clear the screen captureEfface la capture d'écran
    $hGraphic = _GDIPlus_ImageGetGraphicsContext($hImage)
    _GDIPlus_GraphicsClear($hGraphic)

    ; Save resultant imageEnregistre l'image qui en résulte
    _GDIPlus_ImageSaveToFile($hImage, @MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Clean up resourcesLibère les ressources
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; Shut down GDI+ libraryFerme la librairie GDI+
    _GDIPlus_Shutdown()

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.jpg")
EndFunc   ;==>Example
