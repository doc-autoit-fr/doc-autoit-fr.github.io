_GDIPlus_BitmapCloneArea

Create a clone of a Bitmap object from the coordinates and format specifiedCrée un clone d'un objet Bitmap à partir des coordonnées et d'un format spécifiés

#include <GDIPlus.au3>
_GDIPlus_BitmapCloneArea ( $hBitmap, $nLeft, $nTop, $nWidth, $nHeight [, $iFormat = 0x00021808] )

ParametersParamètre

$hBitmap Handle to a Bitmap objectHandle de l'objet Bitmap
$nLeft X coordinate of upper left corner of the rectangle to copyCoordonnée X du coin supérieur gauche du rectangle à copier
$nTop Y coordinate of upper left corner of the rectangle to copyCoordonnée Y du coin supérieur gauche du rectangle à copier
$nWidth The width of the rectangle that specifies the portion of this bitmap to copyLargeur du rectangle qui spécifie la partie du Bitmat à copier
$nHeight The height of the rectangle that specifies the portion of this bitmap to copyHauteur du rectangle qui spécifie la partie du Bitmat à copier
$iFormat [optional][optional] False - i, v, x ... Pixel format for the new bitmap:Format de pixel pour le nouveau Bitmap:
    $GDIP_PXF01INDEXED = 1 bit per pixel, indexed
    $GDIP_PXF04INDEXED = 4 bits per pixel, indexed
    $GDIP_PXF08INDEXED = 8 bits per pixel, indexed
    $GDIP_PXF16GRAYSCALE = 16 bits per pixel, grayscale
    $GDIP_PXF16RGB555 = 16 bits per pixel; 5 bits for each RGB component
    $GDIP_PXF16RGB565 = 16 bits per pixel; 5 bits for red, 6 bits for green and 5 bits blue
    $GDIP_PXF16ARGB1555 = 16 bits per pixel; 1 bit for alpha and 5 bits for each RGB component
    $GDIP_PXF24RGB = 24 bits per pixel; 8 bits for each RGB component
    $GDIP_PXF32RGB = 32 bits per pixel; 8 bits for each RGB component. No alpha component.Aucune composante alpha.
    $GDIP_PXF32ARGB = 32 bits per pixel; 8 bits for each RGB and alpha component
    $GDIP_PXF32PARGB = 32 bits per pixel; 8 bits for each RGB and alpha component, pre-multiplied
    $GDIP_PXF48RGB = 48 bits per pixel; 16 bits for each RGB component
    $GDIP_PXF64ARGB = 64 bits per pixel; 16 bits for each RGB and alpha component
    $GDIP_PXF64PARGB = 64 bits per pixel; 16 bits for each RGB and alpha component, pre-multiplied

Return ValueRemarque

Success: a handle to a new Bitmap object.Retourne le handle d'un nouvel objet Bitmap.
Failure: 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <GDIPlus.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    Local $hBitmap, $hClone, $hImage, $iX, $iY

    ; Initialize GDI+ libraryInitialise GDI+ Librairie
    _GDIPlus_Startup()

    ; Capture 32 bit bitmapCapture un bitmap 32 bits
    $hBitmap = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; Create 24 bit bitmap cloneCrée un clone 24 bits
    $iX = _GDIPlus_ImageGetWidth($hImage)
    $iY = _GDIPlus_ImageGetHeight($hImage)
    $hClone = _GDIPlus_BitmapCloneArea($hImage, 0, 0, $iX, $iY, $GDIP_PXF24RGB)

    ; Save bitmap to fileEnregistre le bitmap dan un fichier
    _GDIPlus_ImageSaveToFile($hClone, @MyDocumentsDir & "\GDIPlus_Image.bmp")

    ; Clean up resourcesLibère les ressources
    _GDIPlus_ImageDispose($hClone)
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; Shut down GDI+ libraryFerme la librairie GDI+
    _GDIPlus_Shutdown()

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.jpg")
EndFunc   ;==>Example
