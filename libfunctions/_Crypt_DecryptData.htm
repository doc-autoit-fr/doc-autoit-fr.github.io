_Crypt_DecryptData

Decrypts data using the supplied keyDéchiffre les données à l'aide de la clé fournie

#include <Crypt.au3>
_Crypt_DecryptData ( $vData, $vCryptKey, $iAlgID [, $bFinal = True] )

ParametersParamètres

$vData Data to decryptDonnées à déchiffrer
$vCryptKey Password or handle to a key if the CALG_USERKEY flag is specifiedMot de passe ou descripteur (handle)'d'une clé si l'indicateur CALG_USERKEY est spécifié
$iAlgID The algorithm to useL'algorithme à utiliser
$bFinal [optional] False if this is only a segment of the full dataFalse si c'est seulement un segment de données complet

Return ValueValeur de retour

Success: a binary string containing decrypted data.une chaîne binaire contenant les données déchiffrées.
Failure: sets the @error flag to non-zero.met @error <> 0.
@error:@error : 100+ - Cannot create key100+ - Ne peut créer la clé
20 - Failed to decrypt data20 - Échec lors du déchiffrement des données

RemarksRemarques

The decrypted data is always returned as a binary string even if the encrypted data is in fact a string (cast with BinaryToString()).La donnée déchiffrée est toujours retournée comme une chaîne binaire même si la donnée chiffrée est en fait une chaîne (transtypage avec BinaryToString()).

ExampleExemple

#include <Crypt.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local Const $sUserKey = "CryptPassword" ; Declare a password string to decrypt/encrypt the data.Déclare un de mot de passe sous forme de chaîne pour déchiffrer/chiffrer les données.
    Local $sData = "..upon a time there was a language without any standardized cryptographic functions.... il était une fois un langage sans aucune fonction cryptographique normalisée. That language is no more.Ce langage n'est plus." ; Data that will be encrypted.Données qui seront chiffrées.

    Local $bEncrypted = _Crypt_EncryptData($sData, $sUserKey, $CALG_RC4) ; Encrypt the data using the generic password string.Chiffre les données à l'aide du mot de passe générique.

    $bEncrypted = _Crypt_DecryptData($bEncrypted, $sUserKey, $CALG_RC4) ; Decrypt the data using the generic password string.Déchiffre les données à l'aide du mot de passe générique. The return value is a binary string.La valeur de retour est une chaîne binaire.
    MsgBox($MB_SYSTEMMODAL, "Decrypted dataDonnées déchiffrées", BinaryToString($bEncrypted)) ; Convert the binary string using BinaryToString to display the initial data we encrypted.Convertit la chaîne binaire à l'aide de BinaryToString pour afficher les données initiales que nous avions chiffrées.
EndFunc   ;==>Example
