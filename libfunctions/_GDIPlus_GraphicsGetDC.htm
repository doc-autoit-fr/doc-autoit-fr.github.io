_GDIPlus_GraphicsGetDC

Gets a handle to the device context of the Graphics objectObtient le handle du contexte de périphérique de l'objet Graphics

#include <GDIPlus.au3>
_GDIPlus_GraphicsGetDC ( $hGraphics )

ParametersParamètres

$hGraphics Handle to a Graphics objectHandle de l'objet Graphics

Return ValueValeur de retour

Success:Succès : a handle DC of the Graphics object.Retourne le handle du DC (Device Context)de l'objet Graphics.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

Each call to the _GDIPlus_GraphicsGetDC() should be paired with a call to the _GDIPlus_GraphicsReleaseDC().Chaque appel de la fonction _GDIPlus_GraphicsGetDC() devrait être associé à un appel de la fonction _GDIPlus_GraphicsReleaseDC().
Do not call any methods of the Graphics object between the calls.Ne pas appeler de méthode de l'objet Graphics entre les appels.
