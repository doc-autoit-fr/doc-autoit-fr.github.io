_WinAPI_OemToChar

Converts a string from the OEM-defined character set into either an ANSI string

#include <WinAPIMisc.au3>
_WinAPI_OemToChar ( $sStr )

ParametersParamètres

$sStr The string of characters from the OEM-defined character set.

Return ValueValeur de retour

Success:Succès : The converted string.
Failure:Échec : Sets the @error flag to non-zero, call _WinAPI_GetLastError() to get extended error information.
