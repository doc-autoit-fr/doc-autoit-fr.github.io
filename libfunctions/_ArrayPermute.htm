_ArrayPermute

Returns an array of the Permutations of all Elements in a 1D arrayRetourne un tableau des permutations de tous les éléments d'un tableau 1D

#include <Array.au3>
_ArrayPermute ( ByRef $aArray [, $sDelimiter = ""] )

ParametersParamètres

$aArray The Array to get PermutationsLe tableau dont les éléments sont à permuter.
$sDelimiter [optional][optionnel][optionnel] String result separator, default is "" for noneSéparateur à utiliser dans les chaînes résultats , par défaut est "" pour aucun séparateur.

Return ValueValeur de retour

Success:Succès : an array of Permutations.Retourne un tableau de permutations.
$aArray[0] contains the number of strings returned.$aArray[0] contient le nombre de chaînes renvoyées.
The remaining elements ($aArray[1], $aArray[2] ...Les autres éléments ($aArray[1], $aArray[2], ... $aArray[n]) contain the Permutations.$aArray[n]) contiennent les permutations sous forme de chaînes.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.
@error:@error : 1 - The Input Must be an Array1 - L'entrée doit être un tableau
2 - $aArray is not a 1 dimensional array2 - $aArray n'est pas un tableau 1D

RemarksRemarques

The input array must be 0-based, i.e. no counter in $aArray[0].Le tableau d'entrée doit être de base 0, c'est à dire qu'il ne doit pas y avoir le nombre d'éléments dans $aArray[0].
Based on the algorithm by Alexander Bogomolny (http://www.bearcave.com/random_hacks/permute.html).Sur la base de l'algorithme de Alexander Bogomolny ( http://www.bearcave.com/random_hacks/permute.html ).
