_ClipBoard_GetSequenceNumber

Retrieves the clipboard sequence number for the current window stationRécupère le numéro de séquence du presse-papiers pour la station de fenêtre actuelle

#include <Clipboard.au3>
_ClipBoard_GetSequenceNumber ( )

Return ValueValeur de retour

Success:Succès : The clipboard sequence numberRetourne le numéro de séquence du presse-papiers
Failure:Échec : 0Retourne 0

RemarksRemarques

The system keeps a serial number for the clipboard for each window station.Le système conserve un numéro de série pour le presse-papiers pour chaque station de fenêtre. This number is incremented when the contents of the clipboard change or the clipboard is emptied.Ce nombre est incrémenté lorsque le contenu du presse-papiers change ou le presse-papiers est vidé. You can track this value to determine if the clipboard contents have changed and optimize creating data objects.Vous pouvez suivre cette valeur pour déterminer si le contenu du presse-papiers a changé et optimiser la création d'objets de données. If clipboard rendering is delayed, the sequence number is not incremented until the changes are rendered.Si l'actualisation du presse-papiers est retardée, le numéro de séquence n'est pas incrémenté jusqu'à ce que les changements soient actualisés.

ExampleExemple

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <WinAPI.au3>
#include <WindowsConstants.au3>

Global $g_idMemo

Example()

Func Example()
    Local $hGUI

    ; Create GUICrée la GUI
    $hGUI = GUICreate("Clipboard", 600, 400)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Open the clipboardOuvre le presse-papiers
    If _ClipBoard_Open($hGUI) Then

        ShowData($hGUI)

        ; Close the clipboardFerme le presse-papiers
        _ClipBoard_Close()
    Else
        _WinAPI_ShowError("_ClipBoard_Open failed")
    EndIf

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Show clipboard statisticsAffiche les statistiques du presse-papiers
Func ShowData($hGUI)
    MemoWrite("GUI handle ............: " & $hGUI)
    MemoWrite("Clipboard owner .......: " & _ClipBoard_GetOwner())
    MemoWrite("Clipboard open window .: " & _ClipBoard_GetOpenWindow())
    MemoWrite("Clipboard sequence ....: " & _ClipBoard_GetSequenceNumber())
    MemoWrite()
EndFunc   ;==>ShowData

; Write message to memoEcrire un message dans memo
Func MemoWrite($sMessage = "")
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
