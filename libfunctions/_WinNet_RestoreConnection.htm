_WinNet_RestoreConnection

Restores the connection to a network resource

#include <WinNet.au3>
_WinNet_RestoreConnection ( [$sDevice = "" [, $hWnd = 0 [, $bUseUI = True]]] )

ParametersParamètres

$sDevice [optional][optionnel] The local name of the drive to connect to, such as "Z:".
If blank, the function reconnects all persistent drives stored in the registry for the current user.
$hWnd [optional][optionnel] Handle to the parent window that the function uses to display the user interface that prompts
the user for a name and password when making the network connection.
If 0, there is no owner window.
$bUseUI [optional][optionnel] If True, display a username/password prompt to the caller

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False.Retourne False.
