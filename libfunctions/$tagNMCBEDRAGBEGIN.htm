$tagNMCBEDRAGBEGIN

Contains information used with the $CBEN_DRAGBEGIN notification messageContient des informations utilisées par le message de notification $CBEN_DRAGBEGIN

Global Const $tagNMCBEDRAGBEGIN = $tagNMHDR & ";int ItemID;wchar szText[260]"

ParametersParamètres

$tagNMHDR Contains information about a notification messageContient des informations sur un message de notification
ItemID The ze0-ro-based index of the item being dragged. This value will always be -1, indicating that the item being dragged is the item displayed in the edit portion of the control.
szText The character buffer that contains the text of the item being dragged
