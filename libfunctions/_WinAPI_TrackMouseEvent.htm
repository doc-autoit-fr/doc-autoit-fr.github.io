_WinAPI_TrackMouseEvent

Posts messages when the mouse pointer leaves a window or hovers over a window for a specified amount of time

#include <WinAPISys.au3>
_WinAPI_TrackMouseEvent ( $hWnd, $iFlags [, $iTime = -1] )

ParametersParamètres

$hWnd Handle to the window to track.
$iFlags The services requested. This parameter can be a combination of the following values.
$TME_CANCEL
$TME_HOVER
$TME_LEAVE
$TME_NONCLIENT
$TME_QUERY
$iTime [optional][optionnel] The hover time-out (if $TME_HOVER was specified in $Flags), in milliseconds. Can be (-1) (Default), which
means to use the system default hover time-out.

Return ValueValeur de retour

Success:Succès : True
Failure:Échec : FalseRetourne False
