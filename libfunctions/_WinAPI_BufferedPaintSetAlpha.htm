_WinAPI_BufferedPaintSetAlpha

Sets the alpha to a specified value in a given rectangle

#include <WinAPITheme.au3>
_WinAPI_BufferedPaintSetAlpha ( $hBP [, $iAlpha = 255 [, $tRECT = 0]] )

ParametersParamètre

$hBP Handle of the buffered paint context.
$iAlpha [optional][optional] False - i, v, x ... The alpha value to set. The alpha value can range from zero (fully transparent) to 255 (Default)(fully opaque).
$tRECT [optional][optional] False - i, v, x ... $tagRECT structure that specifies the rectangle in which to set the alpha.
If this parameter is 0 (Default), the entire buffer is used.

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 and sets the @error flag to non-zero, @extended flag may contain the HRESULT error code.

Remarks<i0>

This function requires Windows Vista or later.
