_ArrayAdd

Adds a specified value at the end of an existing 1D or 2D arrayAjoute une valeur spécifiée à la fin d'un tableau existant 1D ou 2D

#include <Array.au3>
_ArrayAdd ( ByRef $aArray, $vValue [, $iStart = 0 [, $sDelim_Item = "|" [, $sDelim_Row = @CRLF [, $iForce = $ARRAYFILL_FORCE_DEFAULT]]]] )

RemarksRemarques

Addition mode depends on $vValue type, but using $iForce can affect this. Default behaviour is as follows:

1D arrays:Tableaux 1D :
    Single item - adds 1 element    Elément unique - insère un élément
    $sDelim_Item delimited string - adds as many elements as items    $sDelim_Item: chaîne morcellée - insère autant d'éléments que de morceaux
    0-based 1D array - adds as many elements as items    Tableau 1D de base 0 - insére autant d'éléments qu'il y a d'éléments dans le tableau

2D arrays:Tableaux 2D :
    $sDelim_Item delimited string - adds 1 row, columns filled if enough items    $sDelim_Item: chaîne morcellée - insère une ligne, les colonnes sont remplies si il y a assez de morceaux
    $sDelim_Row delimited string - adds as many rows as items, only 1 column filled    $sDelim_Row: chaîne morcellée - insère autant de lignes que de morceaux, une seule colonne est remplie
    $sDelim_Item & Row delimited string - adds rows and items, depending on split points    $sDelim_Item & Row: chaîne morcellée - insère des lignes et des éléments, selon les morceaux
    0-based 2D array - adds as many rows/columns as in array    Tableau 2D de base 0 - insère autant de lignes/colonnes que dans le tableau

An empty string ("") is added if there are insufficient items specified in $vValue.Une chaîne vide ("") est insérée s'il n'y a pas suffisamment d'éléments spécifiés dans $vValeur. Excess items are ignored.Les éléments en excès sont ignorés.

Setting the $iForce parameter can change the default insertion behaviour or amend the datatype for added items. It can be set as follows:
    - $ARRAYFILL_FORCE_DEFAULT (default):
- Items are split as described above. Single items and array elements retain their datatypes - delimited strings are added as strings.
    - $ARRAYFILL_FORCE_SINGLEITEM:
- If $aArray is 1D, $vValue is added as a single element.
- if $aArray is 2D the parameter is ignored.
    - $ARRAYFILL_FORCE_INT, $ARRAYFILL_FORCE_NUMBER, $ARRAYFILL_FORCE_PTR, $ARRAYFILL_FORCE_HWND, $ARRAYFILL_FORCE_STRING
- If $vValue is a single item, it is forced into the defined datatype - by default it would retain its existing datatype.
- If $vValue is a delimited string, all items are forced into the specified datatype - by default they would be added to the array as strings. If different datatypes are required for the items they must be passed as an array.
- If $vValue is an array, the parameter is ignored and the array elements are added retaining their existing datatypes.

If $iForce is set to any other value it is ignored.

ExampleExemple

Example 1Exemple 1

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray_Base[2] = ["Org Item 0", "Org item 1"]
_ArrayDisplay($aArray_Base, "1D - Base array")

; Add a single itemInsère un élément unique
Local $aArray = $aArray_Base
Local $sSingleFill = "New Item 2"
_ArrayAdd($aArray, $sSingleFill)
_ArrayDisplay($aArray, "1D - Single")

; Add a delimited string - each item adds new elementInsère une chaîne avec délimiteurs - chaque item insère un nouvel élément
$aArray = $aArray_Base
Local $sFill = ""
For $i = 1 To 5
    $sFill &= "New Item " & $i + 1 & "|"
Next
$sFill = StringTrimRight($sFill, 1)
MsgBox($MB_SYSTEMMODAL, "Delimited string to add", $sFill)
_ArrayAdd($aArray, $sFill)
_ArrayDisplay($aArray, "1D - Delim string")

; Add a 1D array - each element adds new elementInsère un tableau 1D - chaque item insère un nouvel élément
$aArray = $aArray_Base
Local $aFill[5]
For $i = 0 To 4
    $aFill[$i] = "New Item " & $i + 2
Next
_ArrayDisplay($aFill, "Array to add")
_ArrayAdd($aArray, $aFill)
_ArrayDisplay($aArray, "1D - 1D array")

Example 2Exemple 2

#include <Array.au3>

Local $aArray, $sFill

Local $aArray_Base[2][2] = [["Item 0 - 0", "Item 0 - 1"], ["Item 1 - 0", "Item 1 - 1"]]
_ArrayDisplay($aArray_Base, "2D - Base array")

; Add item delimited stringInsère les items de la chaîne avec séparateurs
$aArray = $aArray_Base
$sFill = "New Item 2 - 0|New Item 2 - 1"
_ArrayAdd($aArray, $sFill)
_ArrayDisplay($aArray, "2D - Item delimited")

; Add row delimited string - load in col 1Insère une ligne à partir d'un chaîne avec séparateurs - chargé dans la colonne 1
$aArray = $aArray_Base
$sFill = "New Item 2 - 1" & @CRLF & "New Item 3 - 1"
_ArrayAdd($aArray, $sFill, 1)
_ArrayDisplay($aArray, "2D - Row Delimited")

; Add item & row delimited stringInsère item & ligne à partir d'une chaîne avec délimiteurs
$aArray = $aArray_Base
$sFill = "New Item 2 - 0|New Item 2 - 1" & @CRLF & "New Item 3 - 0|New Item 3 - 1"
_ArrayAdd($aArray, $sFill)
_ArrayDisplay($aArray, "2D - Item & row Delimited")

; Add a 2D arrayInsère un tableau 2D
$aArray = $aArray_Base
Local $aFill[2][2] = [["New Item 2 - 1", "New Item 2 - 2"], ["New Item 3 - 1", "New Item 3 - 2"]]
_ArrayAdd($aArray, $aFill)
_ArrayDisplay($aArray, "2D - 2D Array")

; Add a 2D array - Single item/column - load in col 1Insère un tableau 2D - un seul item/colonne - chargé dans la colonne 1
$aArray = $aArray_Base
Local $aFill[2][1] = [["New Item 2 - 1"], ["New Item 3 - 1"]]
_ArrayAdd($aArray, $aFill, 1)
_ArrayDisplay($aArray, "2D - 2D Array")
