_WinAPI_DrawThemeParentBackground

Draws the part of a parent control that is covered by a partially-transparent or alpha-blended child control

#include <WinAPITheme.au3>
_WinAPI_DrawThemeParentBackground ( $hWnd, $hDC [, $tRECT = 0] )

ParametersParamètres

$hWnd Handle of the child control.
$hDC Handle to the child control's DC.
$tRECT [optional][optionnel] $tagRECT structure that defines the area, in the child window's coordinates, to be drawn.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended flag may contain the HRESULT error code.
