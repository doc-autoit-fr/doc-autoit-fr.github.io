_GDIPlus_TextureCreateIA

Creates a TextureBrush object based on an image, a defining rectangle, and a set of image propertiesCrée un objet TextureBrush à partir d'une image, d'une définition de rectangle, et d'un ensemble de propriétés de l'image

#include <GDIPlus.au3>
_GDIPlus_TextureCreateIA ( $hImage, $nX, $nY, $nWidth, $nHeight [, $pImageAttributes = 0] )

ParametersParamètres

$hImage Pointer to an image objectHandle de l'objet Image
$nX Leftmost coordinate of the image portion to be used by this brushCoordonnée X la plus à gauche de la partie d'image qui doit être utilisée par ce pinceau
$nY Uppermost coordinate of the image portion to be used by this brushCoordonnée Y la plus haute de la partie d'image qui doit être utilisée par ce pinceau
$nWidth Width of the brush and width of the image portion to be used by the brushLargeur du pinceau et largeur de la partie d'image qui doit être utilisée par ce pinceau
$nHeight Height of the brush and height of the image portion to be used by the brushHauteur du pinceau et hauteur de la partie d'image qui doit être utilisée par ce pinceau
$pImageAttributes [optional] Pointer to an ImageAttributes object that contains properties of the imageHandle de l'objet ImageAttributes qui contient des propriétés de l'image

Return ValueValeur de retour

Success: a pointer to a new TextureBrush object.Retourne le handle du nouvel objet TextureBrush.
Failure: False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

RemarksRemarques

After you are done with the object, call _GDIPlus_BrushDispose() to release the object resources.Lorsque vous en avez terminé avec l'objet, appelez _GDIPlus_BrushDispose() pour libérer les ressources de l'objet.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <Screencapture.au3>

Global $g_hHBitmap, $g_hGfxCtxt, $g_hBMP, $g_hGfx
Global $g_iTiles = 24, $g_a_hBmp_Tiles[$g_iTiles][$g_iTiles]

Example()

Func Example()
    AutoItSetOption("GUIOnEventMode", 1)

    Local $iW = 576, $iH = 576, $iTMP, $iSin = 0, $iMulti = 256 / $g_iTiles, $2Multi = 2 * $iMulti, _
            $hCtxt, $hTextureIA, $fValue, $iFrameTimer = TimerInit()

    _GDIPlus_Startup()
    $g_hHBitmap = _ScreenCapture_Capture("", 0, 0, $g_iTiles ^ 2, $g_iTiles ^ 2) ;capture the desktop and save it in memory in GDI formatcapturer le bureau et l'enregistrer dans la mémoire au format GDI
    Local $hImage = _GDIPlus_BitmapCreateFromHBITMAP($g_hHBitmap) ;convert the GDI bitmap to GDI+ bitmapConvertir l'image bitmap GDI en bitmap GDI+
    ; Local $iImgW = _GDIPlus_ImageGetWidth($hImage) ;get the width of the image$Locale = iImgW _GDIPlus_ImageGetWidth ($Himage); Obtient la largeur de l'image
    ; Local $iImgH = _GDIPlus_ImageGetHeight($hImage) ;get the height of the image$Locale = iImgH _GDIPlus_ImageGetHeight ($Himage); Obtient la hauteur de l'image

    Local $hGUI = GUICreate("GDI+ TextureIA", $iW, $iH) ;create a GUICrée une GUI
    $g_hGfx = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphic object for displaying GDI+ graphicsCrée un objet graphique pour l'affichage graphique GDI+
    $g_hBMP = _GDIPlus_BitmapCreateFromGraphics($iW, $iH, $g_hGfx) ;create a bitmap object based on a graphic objectCrée un objet bitmap à partir d'un objet graphique
    $g_hGfxCtxt = _GDIPlus_ImageGetGraphicsContext($g_hBMP) ;get the graphics context of the image / bitmapObtient le contexte graphique de l'image/bitmap

    Local $hBitmap = _GDIPlus_BitmapCreateFromScan0($g_iTiles ^ 2, $g_iTiles ^ 2) ;create an empty bitmap (needed later for texture creating)Crée un bitmap vide (nécessaire plus tard pour créer la texture)
    Local $hContext = _GDIPlus_ImageGetGraphicsContext($hBitmap) ;get the graphic context of the image / bitmapObtient le contexte graphique de l'image/bitmap
    _GDIPlus_GraphicsDrawImageRect($hContext, $hImage, 0, 0, $g_iTiles ^ 2, $g_iTiles ^ 2) ;copy the image into the empty bitmapCopie l'image dans le bitmap vide
    _GDIPlus_GraphicsDispose($hContext) ;release graphic context handleLibére le handle du contexte graphique
    _GDIPlus_ImageDispose($hImage) ;release image handleLibère le handle de l'image

    Local $tCM = _GDIPlus_ColorMatrixCreateScale(0, 0, 0) ;create and initializes a scaling color matrix (5 x 5 homogeneous matrix for a 4-space transformation) -> see http://msdn.microsoft.com/en-us/library/windows/desktop/ms533875(v=vs.85).aspxCrée et initialise une matrice couleur de mise à l'échelle (matrice 5 x 5 homogène pour une transformation dans l'espace de dimension 4) -> voir http://msdn.microsoft.com/en- us/Librairie/windows/desktop/ms533875(v=vs.85).aspx
    Local $pImgAttr = _GDIPlus_ImageAttributesCreate() ;create an image attribute object (needed for color manipulation)Crée un objet d'attributs d'image (nécessaire pour la manipulation des couleurs)
    Local $fSizeInv = 1 / $g_iTiles

    ;create $g_iTiles * $g_iTiles small bitmaps (tiles) to fill it up with parts of the captured imageCrée $g_iTiles * $g_iTiles de petites images bitmaps (pavés) pour remplir avec des parties de l'image capturée
    For $iY = 0 To $g_iTiles - 1
        For $iX = 0 To $g_iTiles - 1
            $g_a_hBmp_Tiles[$iY][$iX] = _GDIPlus_BitmapCreateFromScan0($g_iTiles, $g_iTiles) ;create an empty bitmapCrée un bitmap vide
            $hCtxt = _GDIPlus_ImageGetGraphicsContext($g_a_hBmp_Tiles[$iY][$iX]) ;get the graphics context of the image / bitmapObtient le contexte graphique de l'image/bitmap
            $fValue = BitXOR($iX, $iY) * $fSizeInv
            $tCM.m(1) = 1 - $fValue ; redrouge
            $tCM.m(7) = $fValue ;greenvert
            $tCM.m(19) = 0.4 ;alpha
            _GDIPlus_ImageAttributesSetColorMatrix($pImgAttr, 0, True, $tCM) ;adjust the ImageAttribute color matrix for each bitmapAjuste la matrice couleur ImageAttribute pour chaque bitmap
            $hTextureIA = _GDIPlus_TextureCreateIA($hBitmap, $iX * $g_iTiles, $iY * $g_iTiles, $g_iTiles, $g_iTiles, $pImgAttr) ;creates a texture brush object based on an image with changed image attributesCrée un objet TextureBrush à partir d'une image avec des attributs d'image modifiées
            _GDIPlus_GraphicsFillRect($hCtxt, 0, 0, $g_iTiles, $g_iTiles, $hTextureIA) ;draw rectangle filled with manipulated tile of the image to the bitmapDessine un rectangle rempli avec des pavés manipulée de l'image dans le bitmap
            _GDIPlus_GraphicsDispose($hCtxt) ;release graphic context handleLibére le handle du contexte graphique
            _GDIPlus_BrushDispose($hTextureIA) ;release textureLibére la texture
        Next
    Next
    _GDIPlus_ImageAttributesDispose($pImgAttr) ;releases the image attributes objectLibère l'objet ImageAttributes

    GUISetOnEvent($GUI_EVENT_CLOSE, "_Exit", $hGUI)
    GUISetState(@SW_SHOW, $hGUI)

    While Sleep(10)
        $iSin += 25 / (1000 / TimerDiff($iFrameTimer))
        _GDIPlus_GraphicsClear($g_hGfxCtxt, 0xB0808080) ;Clears a graphic object to a specified colorEfface l'objet graphique avec une couleur spécifiée
        For $iX = 0 To $g_iTiles - 1
            $iTMP = 2 * $iX * $iMulti
            For $iY = 0 To $g_iTiles - 1
                ;move each tile to its new positionDéplace chaque mosaïque à sa nouvelle position
                _GDIPlus_GraphicsDrawImageRect($g_hGfxCtxt, $g_a_hBmp_Tiles[$iY][$iX], _ ;bitmap handleHandle de bitmap
                        25 + 15 * Sin($iY * $iMulti / 64 + $iSin) + $iTMP, _ ;x pos
                        15 * Cos($iX * $iMulti / 64 + $iSin) + 25 + $iY * $2Multi, _ ;y pos
                        $2Multi, _ ;widthlargeur
                        $2Multi) ;heighthauteur
            Next
        Next
        _GDIPlus_GraphicsDrawImage($g_hGfx, $g_hBMP, 0, 0) ;draw complete drawn image into GUIDessine l'image complète dans la GUI
        $iFrameTimer = TimerInit()
    WEnd
EndFunc   ;==>Example

Func _Exit() ;clean up resourcesNettoie les ressources
    _WinAPI_DeleteObject($g_hHBitmap)
    _GDIPlus_GraphicsDispose($g_hGfxCtxt)
    _GDIPlus_BitmapDispose($g_hBMP)
    _GDIPlus_GraphicsDispose($g_hGfx)
    For $iY = 0 To $g_iTiles - 1
        For $iX = 0 To $g_iTiles - 1
            _GDIPlus_BitmapDispose($g_a_hBmp_Tiles[$iY][$iX])
        Next
    Next
    _GDIPlus_Shutdown()
    Exit
EndFunc   ;==>_Exit
