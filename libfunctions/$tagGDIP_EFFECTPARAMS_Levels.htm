$tagGDIP_EFFECTPARAMS_Levels

Levels effect parameter structureStructure du paramètre de l'effet Levels (brillance)

Global Const $tagGDIP_EFFECTPARAMS_Levels = "int Highlight; int Midtone; int Shadow"

ParametersParamètres

Highlight Integer in the range 0 through 100 that specifies which pixels should be lightened.Entier compris entre 0 et 100 qui spécifie les pixels qui doivent être mis en surbrillance.
Midtone Integer in the range -100 through 100 that specifies how much to lighten or darken an image.Entier compris entre -100 et 100 qui spécifie de combien il faut éclaircir ou assombrir une image.
Shadow Integer in the range 0 through 100 that specifies which pixels should be darkened.Entier compris entre 0 et 100 qui spécifie les pixels qui doivent être obscurcis.
