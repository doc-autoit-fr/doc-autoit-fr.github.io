RemarksRemarques

In the $tagSYSTEMTIME structure, the wHour and wMinute members represent the transition time, the wDayOfWeek member to the appropriate weekday, and the wDay member indicating the occurrence of the day of the week within the month (1 to 5, where 5 indicates the final occurrence during the month if that day of the week does not occur 5 times).Dans la structure $tagSYSTEMTIME, les membres wHour et wMinute représentent le changement d'heure, le membre wDayOfWeek du jour de la semaine correspondant, et le membre wDay indiquant l'occurence du jour de la semaine dans le mois (1 à 5, où 5 indique l'occurrence finale au cours du mois si ce jour de la semaine ne répète pas 5 fois).

Using this notation, specify 02:00 on the first Sunday in April as follows: wHour = 2, wMonth = 4, wDayOfWeek = 0, wDay = 1.En utilisant cette notation, indiquez 02:00 pour le premier dimanche d'avril, comme suit : wHour = 2, wMonth = 4, wDayOfWeek = 0, wDay = 1. Specify 02:00 on the last Thursday in October as follows: wHour = 2, wMonth = 10, wDayOfWeek = 4, wDay = 5.Indiquez 02:00 pour le dernier jeudi d'octobre comme suit : wHour = 2, wMonth = 10, wDayOfWeek = 4, wDay = 5.

#include <Date.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsConstants.au3>

; Under Vista the Windows API "SetTimeZoneInformation" may be rejected due to system securitySous Vista, l'API Windows « SetTimeZoneInformation » peut être rejetée en raison de la sécurité du système

Global $g_idMemo

Example()

Func Example()
    Local $aOld, $aNew

    ; Create GUICrée la GUI
    GUICreate("TimeTemps", 400, 300)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 396, 296, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Show current time zone informationAffiche les informations de fuseau horaire actuel
    $aOld = _Date_Time_GetTimeZoneInformation()
    ShowTimeZoneInformation($aOld, "CurrentActuel")

    ; Set new time zone informationDéfinit les nouvelles informations de fuseau horaire
    If Not _Date_Time_SetTimeZoneInformation($aOld[1], "A3L CST", $aOld[3], $aOld[4], "A3L CDT", $aOld[6], $aOld[7]) Then
        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "System timezone cannot be SETImpossible de définir le fuseau horaire système" & @CRLF & @CRLF & _WinAPI_GetLastErrorMessage())
        Exit
    EndIf

    ; Show new time zone informationAffiche les nouvelles informations de fuseau horaire
    $aNew = _Date_Time_GetTimeZoneInformation()
    ShowTimeZoneInformation($aNew, "NewNouveau")

    ; Reset original time zone informationRéinitialise les informations de fuseau horaire d'origine
    _Date_Time_SetTimeZoneInformation($aOld[1], $aOld[2], $aOld[3], $aOld[4], $aOld[5], $aOld[6], $aOld[7])

    ; Show current time zone informationAffiche les informations de fuseau horaire actuel
    $aOld = _Date_Time_GetTimeZoneInformation()
    ShowTimeZoneInformation($aOld, "ResetRéinitialisé")

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

Func MemoWrite($sMessage)

; Show time zone informationAffiche les informations de fuseau horaire
Func ShowTimeZoneInformation(ByRef $aInfo, $sComment)
    MemoWrite("************************************************ " & $sComment & " ************************************************")
    MemoWrite("Result ............:Résultat..................... : " & $aInfo[0])
    MemoWrite("Current bias ......:Décalage actuel.............. : " & $aInfo[1])
    MemoWrite("Standard name .....:Nom standard................. : " & $aInfo[2])
    MemoWrite("Standard date/time :Date/heure standard.......... : " & _Date_Time_SystemTimeToDateTimeStr($aInfo[3]))
    MemoWrite("Standard bias......:Décalage standard............ : " & $aInfo[4])
    MemoWrite("Daylight name .....:Nom pour l'heure d'été....... : " & $aInfo[5])
    MemoWrite("Daylight date/time :Date/heure pour l'heure d'été : " & _Date_Time_SystemTimeToDateTimeStr($aInfo[6]))
    MemoWrite("Daylight bias......:Décalage pour l'heure d'été.. : " & $aInfo[7])
EndFunc   ;==>ShowTimeZoneInformation
