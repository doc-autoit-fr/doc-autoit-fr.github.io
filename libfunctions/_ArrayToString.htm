_ArrayToString

Places the elements of a 1D or 2D array into a single string, separated by the specified delimitersPlace les éléments d'un tableau 1D ou 2D dans une seule chaîne, séparés par des séparateurs spécifiés

#include <Array.au3>
_ArrayToString ( Const ByRef $aArray [, $sDelim_Col = "|" [, $iStart_Row = -1 [, $iEnd_Row = -1 [, $sDelim_Row = @CRLF [, $iStart_Col = -1 [, $iEnd_Col = -1]]]]]] )

ParametersParamètre

$aArray Array to convert to a stringTableau à convertir en chaîne
$sDelim_Col [optional][optional] False - i, v, x ... Delimiter for elements of 1D array or columns of 2D arrayDélimiteur des éléments du tableau 1D ou des colonnes de tableau 2D
$iStart_Row [optional][optional] False - i, v, x ... Index of array row to start copyIndex de ligne du tableau pour commencer la copie
$iEnd_Row [optional][optional] False - i, v, x ... Index of array row to stop copyIndex de ligne du tableau pour arrêter la copie
$sDelim_Row [optional][optional] False - i, v, x ... Delimiter for rows of 2D array (2D only)Délimiteur des lignes du tableau 2D (2D uniquement)
$iStart_Col [optional][optional] False - i, v, x ... Index of array column to start copy (2D only)Index de colonne du tableau pour commencer la copie (2D uniquement)
$iEnd_Col [optional][optional] False - i, v, x ... Index of array column to stop copy (2D only)Index de colonne du tableau pour arrêter la copie (2D uniquement)

Return ValueRemarque

Success: a string which combined selected elements separated by the delimiters.Retourne une chaîne qui combine des éléments sélectionnés séparés par des séparateurs.
Failure: sets the @error flag to non-zero.Retourne @error

4 - $iStart_Row greater then $iEnd_Row4 - $iStart_Row plus grand que $iEnd_Row
5 - $iStart_Col or $iEnd_Col outside array bounds5 - $iStart_Col ou $iEnd_Col en dehors des limites du tableau

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray[20]
For $i = 0 To 19
    $aArray[$i] = $i
Next
_ArrayDisplay($aArray, "1D Array")
MsgBox($MB_SYSTEMMODAL, "Items 1-7", _ArrayToString($aArray, @TAB, 1, 7))

Local $aArray[10][10]
For $i = 0 To 9
    For $j = 0 To 9
        $aArray[$i][$j] = $i & "-" & $j
    Next
Next
_ArrayDisplay($aArray, "2D Array")
MsgBox($MB_SYSTEMMODAL, "Rows 4-7, cols 2-5", _ArrayToString($aArray, " :: ", 4, 7, @CRLF, 2, 5))
