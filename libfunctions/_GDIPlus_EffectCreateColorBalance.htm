_GDIPlus_EffectCreateColorBalance

Creates a ColorBalance class effect objectCrée un objet Effect qui agit sur la distorsion des couleurs

#include <GDIPlus.au3>
_GDIPlus_EffectCreateColorBalance ( [$iCyanRed = 0 [, $iMagentaGreen = 0 [, $iYellowBlue = 0]]] )

ParametersParamètre

$iCyanRed [optional][optional] False - i, v, x ... Integer in the range -100 through 100 that specifies a change in the amount of red in the image.Entier compris entre -100 et 100 qui définit la variation de la quantité de rouge dans l'image.
$iMagentaGreen [optional][optional] False - i, v, x ... Integer in the range -100 through 100 that specifies a change in the amount of green in the image.Entier compris entre -100 et 100 qui définit la variation de la quantité de vert dans l'image.
$iYellowBlue [optional][optional] False - i, v, x ... Integer in the range -100 through 100 that specifies a change in the amount of blue in the image.Entier compris entre -100 et 100 qui définit la variation de la quantité de bleu dans l'image.

Return ValueRemarque

Success: a handle to an Effect object.Retpurne le handle de l'objet Effect.
Failure: 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).
@error: -1 - GDIPlus.dll does not support this function.-1 - GDIPlus.dll ne supporte pas cette fonction.
10 - Invalid parameters.10 - Paramètres invalides.

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

_Example()

Func _Example()
    If Not _GDIPlus_Startup() Or @extended < 6 Then
        MsgBox($MB_SYSTEMMODAL, "ERROR", "GDIPlus.dll v1.1 not available")
        Return
    EndIf

    Local $sFile = FileOpenDialog("Select an image", "", "Images (*.bmp;*.png;*.jpg;*.gif;*.tif)")
    If @error Or Not FileExists($sFile) Then Return

    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile)

    Local $iWidth = 600
    Local $iHeight = _GDIPlus_ImageGetHeight($hImage) * 600 / _GDIPlus_ImageGetWidth($hImage)

    Local $hGui = GUICreate("GDI+ v1.1 (" & @ScriptName & ")", $iWidth, $iHeight)
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)
    GUISetState(@SW_SHOW)

    Local $hEffect = _GDIPlus_EffectCreateColorBalance(-30, 20, 10)
    _GDIPlus_BitmapApplyEffect($hImage, $hEffect)

    _GDIPlus_GraphicsDrawImageRect($hGraphics, $hImage, 0, 0, $iWidth, $iHeight)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    _GDIPlus_EffectDispose($hEffect)
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>_Example
