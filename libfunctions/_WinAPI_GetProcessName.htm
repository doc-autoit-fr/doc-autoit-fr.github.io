_WinAPI_GetProcessName

Retrieves the name for the specified process

#include <WinAPIProc.au3>
_WinAPI_GetProcessName ( [$iPID = 0] )

ParametersParamètres

$iPID [optional][optionnel][optionnel] The PID of the process. Default (0) is the current process.

Return ValueValeur de retour

Success:Succès : The process name.
Failure:Échec : Empty string and sets the @error flag to non-zero.
