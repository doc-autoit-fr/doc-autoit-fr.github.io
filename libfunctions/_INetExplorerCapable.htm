_INetExplorerCapable

Converts a string to IE(Internet Explorer) capable line

#include <Inet.au3>
_INetExplorerCapable ( $sIEString )

ParametersParamètre

$sIEString String to be converted

Return ValueRemarque

Success: the converted string.
Failure: an empty string and sets the @error flag to non-zero.Retourne une chaîne vide et @error <> 0.

Remarks<i0>

This converts IE uncapable characters to '%' and their hex value.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <Inet.au3>
#include <MsgBoxConstants.au3>
