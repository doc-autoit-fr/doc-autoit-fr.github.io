Function ReferenceRéférence de fonction

_Crypt_GenRandom

Fill a buffer with cryptographically random dataRemplit une mémoire tampon (buffer) avec des données cryptographiquement aléatoires

#include <Crypt.au3>
_Crypt_GenRandom ( $pBuffer, $iSize )

ParametersParamètres

$pBuffer Pointer to buffer to fill with random data.Pointeur vers la mémoire tampon à remplir avec des données aléatoires.
$iSize Size of the buffer pointed to by $pBuffer.Taille de la mémoire tampon pointée par $pBuffer.

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero.False et met @error <> 0.

RemarksRemarques

Optionally, the application can fill this buffer with data to use as an auxiliary random seed.Éventuellement, l'application peut remplir cette mémoire tampon avec des données à utiliser comme une graine (seed) aléatoire auxiliaire.

See AlsoVoir aussi

Search CryptGenRandom in MSDN Library.Recherchez CryptGenRandom dans la bibliothèque MSDN.

ExampleExemple

#include <Crypt.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Generate 16 bytes of random dataGénére 16 octets de données aléatoires
    Local $tBuff = DllStructCreate("byte[16]")
    _Crypt_GenRandom($tBuff, DllStructGetSize($tBuff))
    MsgBox($MB_SYSTEMMODAL, "Random data:Données aléatoires :", DllStructGetData($tBuff, 1))
EndFunc   ;==>Example
