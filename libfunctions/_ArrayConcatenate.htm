_ArrayConcatenate

Concatenate two arrays - either 1D or 2D with the same number of columnsConcaténe deux tableaux - tous les deux 1D ou 2D avec le même nombre de colonnes

#include <Array.au3>
_ArrayConcatenate ( Const ByRef $aArrayTarget, ByRef $aArraySource [, $iStart = 0] )

ParametersParamètres

$aArrayTarget The array to which the source array will be concatenatedLe tableau auquel le tableau source sera concaténé
$aArraySource The array to concatenate to the target arrayLe tableau source à concaténer au tableau cible
$iStart [optional][optionnel] index of the first Source array entry (Default = 0)Index de la première entrée du tableau source (par défaut, 0)

Return ValueValeur de retour

Success:Succès : the number of elemets/rows in the new target arrayRetourne le nombre d'élémets/lignes du nouveau tableau cible
Failure:Échec : -1 and sets the @error flag to non-zero.-1 et met @error <> 0.
@error:@error : 1 - $aArrayTarget is not an array1 - $aArrayTarget n'est pas un tableau
2 - $aArraySource is not an array2 - $aArraySource n'est pas un tableau
3 - $aArrayTarget is not a 1D or 2D array3 - $aArrayTarget n'est pas un tableau 1D ou 2D
4 - $aArrayTarget and $aArraySource 1D/2D mismatch4 - $aArrayTarget et $aArraySource 1D/2D ne correspondent pas
5 - $aArrayTarget and $aArraySource column number mismatch (2D only)5 - $aArrayTarget et $aArraySource ont des nombres de colonne différents (2D uniquement)
6 - $iStart outside array bounds6 - $istart est hors des limites du tableau

RemarksRemarques

The $iStart parameter is useful when concatenating arrays which have a count in the [0] element.Le paramètre $iStart est utile lors de la concaténation des tableaux qui ont le nombre d'éléments à la position [0].

ExampleExemple

#include <Array.au3>

Local $aArrayTarget[5] = ["0", "1", "2", "3", "4"]
Local $aArraySource[5] = ["5", "6", "7", "8", "9"]

_ArrayDisplay($aArrayTarget, "1D Target")
_ArrayDisplay($aArraySource, "1D Source")
_ArrayConcatenate($aArrayTarget, $aArraySource)
_ArrayDisplay($aArrayTarget, "1D Target and Source concatenated")

Local $aArrayTarget[4][3]
For $i = 0 To 3
    For $j = 0 To 2
        $aArrayTarget[$i][$j] = $i & $j
    Next
Next

Local $aArraySource[4][3]
For $i = 0 To 3
    For $j = 0 To 2
        $aArraySource[$i][$j] = (4 + $i) & $j
    Next
Next

_ArrayDisplay($aArrayTarget, "2D Target")
_ArrayDisplay($aArraySource, "2D Source")
_ArrayConcatenate($aArrayTarget, $aArraySource)
_ArrayDisplay($aArrayTarget, "2D Target and Source concatenated")
