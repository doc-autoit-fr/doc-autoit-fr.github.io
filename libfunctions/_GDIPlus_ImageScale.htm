_GDIPlus_ImageScale

Scales an image by a given factorChange les échelles horizontale et verticale d'une image

#include <GDIPlus.au3>
_GDIPlus_ImageScale ( $hImage, $iScaleW, $iScaleH [, $iInterpolationMode = $GDIP_INTERPOLATIONMODE_HIGHQUALITYBICUBIC] )

ParametersParamètres

$hImage A handle valueHandle d'une matrice
$iScaleW A floating point valueUne valeur à virgule flottante: échelle verticale
$iScaleH A floating point valueUne valeur à virgule flottante: échelle verticale
$iInterpolationMode [optional][optionnel][optionnel] An integer value.Une valeur entière: mode d'interpolation. Default is $GDIP_INTERPOLATIONMODE_HIGHQUALITYBICUBIC.Par défaut, $GDIP_INTERPOLATIONMODE_HIGHQUALITYBICUBIC.

Return ValueValeur de retour

Success:Succès : Returns a handle to a new Bitmap objectRetourne le handle du nouvel objet Bitmap
Failure:Échec : 0 and sets the @error flag to non-zero.0 et @error <> 0.
@error:@error : 1 - unable to get width of image1 - Impossible d'obtenir la largeur de l'image
2 - unable to get height of image2 - Impossible d'obtenir la hauteur de l'image
3 - unable to create new bitmap3 - Impossible de créer un nouveau bitmap
4 - unable to create graphics context4 - Impossible de créer le contexte graphique
5 - unable to set interpolation mode on graphics context5 - Impossible de définir le mode d'interpolation sur le contexte graphique
6 - unable to copy image to scaled bitmap6 - Impossible de copier l'image dans le bitmap mis à l'échelle

RemarksRemarques

Scale factor greater than 1 means enlarging the image whereas smaller than 1 means decrease the size.Un facteur d'échelle supérieur à 1 signifie un agrandissement de l'image alors que plus petit que 1 signifie une diminution de la taille.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    _GDIPlus_Startup()
    Local Const $iW = @DesktopWidth / 4, $iH = @DesktopHeight / 4

    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, $iW, $iH) ;create a GDI bitmap by capturing 1/16 of desktopCrée un bitmap GDI en capturant un seizième de bureau
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;convert GDI bitmap to GDI+ bitmapConverti le bitmap GDI en GDI+
    _WinAPI_DeleteObject($hHBmp) ;release GDI bitmap resource because not needed anymoreLibére la ressource du bitmap GDI car n'est plus nécessaire

    Local $iScale = 2.75 ;1.0 is without any scaling1.0 est sans redimensionnement
    Local $hBitmap_Scaled = _GDIPlus_ImageScale($hBitmap, $iScale, $iScale, $GDIP_INTERPOLATIONMODE_NEARESTNEIGHBOR) ;scale image by 275% (magnify)Image à l'échelle de 275% (agrandissement)

    Local $hGUI = GUICreate("GDI+ test", $iW * $iScale, $iH * $iScale, -1, -1) ;create a test gui to display the resized imageCrée une GUI de test pour afficher l'image redimensionnée
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap_Scaled, 0, 0) ;display scaled imageAffiche l'image à l'échelle

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    ;cleanup resourcesNettoie les ressources
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_BitmapDispose($hBitmap_Scaled)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
