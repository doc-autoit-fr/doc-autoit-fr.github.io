_ArrayDisplay

Displays a 1D or 2D array in a ListViewAffiche un tableau 1D ou 2D dans un contrôle ListView

#include <Array.au3>
_ArrayDisplay ( Const ByRef $aArray [, $sTitle = "ArrayDisplay" [, $sArrayRange = "" [, $iFlags = 0 [, $vUser_Separator = Default [, $sHeader = Default [, $iMax_ColWidth = Default [, $iAlt_Color = Default [, $hUser_Function = ""]]]]]]]] )

ParametersParamètres

$aArray Array to displayTableau à afficher
$sTitle [optional][optionnel] Title for dialog.Titre de l'affichage. Default = "ArrayDisplay".Par défaut ="ArrayDisplay ".
$sArrayRange [optional][optionnel] Range of rows/columns to display.Plage de lignes/colonnes à afficher. Default ("") = entire array. (See below for details)(Voir ci-dessous pour plus de détails)
$iFlags [optional][optionnel] Determine UDF options.Déterminer les options UDF. Add required values together
    0 = (default) Column text alignment - left
    1 = Transposes the array
    2 = Column text alignment - right
    4 = Column text alignment - center
    8 = Verbose - display MsgBox on error and splash screens during processing of large arrays
    16 = Only 'Copy' buttons displayed
    32 = No buttons displayed
    64 = No 'Row' column displayed
$vUser_Separator [optional][optionnel] Sets column display option when copying data to clipboard.Définit l'option d'affichage des colonnes lors de la copie des données du presse-papiers.
    Character = Delimiter between columns.    Caractère = Séparateur entre les colonnes.
    Number = Fixed column width - longer items will be truncated.    Nombre = fixe la largeur des colonnes - les éléments plus longs seront tronqués.
    Default = Current separator character (usually "|").    Par défaut = séparateur courant (généralement "|").
$sHeader [optional][optionnel] Column names in header (string of names separated by current separator character - usually "|").Les noms de colonnes d'entête (chaîne de noms séparés par le séparateur courant - en général "|" ). Default see Remarks.Par défaut voir Remarques.
$iMax_Colwidth [optional][optionnel] Max width to which a ListView column will expand to show content.Largeur maximale à laquelle une colonne de ListView s'élargit pour afficher le contenu. Default = 350 pixels.Valeur par défaut = 350 pixels.
$iAlt_Color [optional][optionnel] ListView alternate rows set to defined color.Couleur à utiliser une ligne sur deux sur les lignes de ListView. Default = all rows ListView background color.Par défaut = toutes les lignes de ListView ont la couleur de fond.
$hUser_Function [optional][optionnel] A variable assigned to the user defined function to run.Une variable affectée avec la fonction utilisateur à exécuter. Default = none.Par défaut = Aucun. See remarks.Voir remarques.

Return ValueValeur de retour

Success:Succès : 1Retourne 1
Failure:Échec : 0 and @error flag set as follows:Retourne 0 et @error non nul:
@error:@error : 1 - $aArray is not an array1 - $aArray n'est pas un tableau
2 - $aArray has too many dimensions (only 1D and 2D supported)2 - $aArray a trop de dimensions (seulement 1D et 2D est supporté)

RemarksRemarques

If the function is passed a non-array variable or an array of more than 2 dimensions the function returns an error and the script continues.Si le paramètre $aArray n'est pas un tableau ou est un tableau de plus de 2 dimensions, la fonction renvoie une erreur et le script continue. If the "verbose" parameter is set in $iFlags a MsgBox will be displayed which offers the chance to exit the script immediately or to continue the script with the normal error return.Si le paramètre "verbose" est installé dans $iFlags un MsgBox s'affiche qui offre la possibilité de quitter immédiatement le script ou de continuer le script avec un retour normal d'erreur.

Only 65525 rows of an array can be displayed - this is an AutoIt limitation on the total number of controls that can be displayed in a GUI - and an abitrary display limit of 250 columns has also been set.Seuls 65 525 lignes d'un tableau peuvent être affichés - il s'agit d'une limitation de AutoIt sur le nombre total de contrôles qui peuvent être affichés dans une GUI - et une limite d'affichage abitrary de 250 colonnes a également été mis. If the user tries to display an array larger than these limits an "*" is added to the truncated dimension value which will be displayed in red text.Si l'utilisateur tente d'afficher un tableau plus grand que ces limites, une "*" est ajoutée à la valeur de la dimension tronquée qui sera affiché en rouge. Note that using $iArrayRange to determine the elements to be displayed allows arrays larger than the display limits to be displayed in sections.Notez que l'utilisation de $iArrayRange pour déterminer les éléments à afficher permet des tableaux plus grands que les limites d'affichage le permettraient.

The $sArrayRange parameter syntax is as follows:La syntaxe du paramètre $sArrayRange est la suivante:

"7" Show rows 0-7 with all columns
"7:" Show rows 7-end with all columns
"|7" Show all rows with columns 0-7
"|7:" Show all rows with columns 7-end
"7|7" Show rows 0-7 with columns 0-7
"5:7" Show rows 5-7 with all columns
"|5:7" Show all rows with columns 5-7
"7|5:7" Show rows 0-7 with columns 5-7
"5:7|7" Show rows 5-7 with columns 0-7
"5:7|5:7" Show rows 5-7 with columns 5-7

Any column values are ignored for 1D arrays.Toutes les valeurs de colonnes sont ignorées pour les tableaux 1D.

$sHeader names (separated by the current separator character) will be used for as many columns as there are names.Les noms $sHeader (séparés par le séparateur courant) seront utilisés pour autant de colonnes qu'il y a de noms. If no, or not enough, custom names are specified then the default header of "Row|Col0" for 1D arrays or "Row|Col0|Col1|...|Col n" for 2D is substituted.S'il n'y a pas de nom, ou pas assez, les noms suivants pour les entêtes seront utilisés:"Row|Col0" pour les tableaux 1D, ou "Row|Col0|Col1|...|Col n" pour les tableaux 2D.

The 4 buttons at the bottom of the dialog have the following functions:Les 4 boutons en bas de la boîte de dialogue ont les fonctions suivantes :
Copy Data & Hdr/Row Copy the array or the selected row(s) to the clipboard adding full header and row identification.
Copy Data Only Copy the array or the selected row(s) to the clipboard with no header or row identification.
Run User Func Run the user-defined function passed in $sUser_Function. This function is entirely separate from the UDF and must be created and coded by the user to accept 2 (and only 2) parameters which will be provided by the UDF itself: the full array being displayed and a 1D array holding the selected rows indices with a count in the [0] element. These parameters can then be used inside the user function as required. The button is not displayed if no function is specified.Le bouton ne s'affiche pas si aucune fonction n'est spécifiée.
Exit script Exit the script immediately.

The $iFlags parameter enables some or all of these buttons to be hidden.Le paramètre $iFlags permet de cacher tout ou partie de ces boutons.

If all buttons are displayed the array dimensions are displayed at lower left.Si tous les boutons sont affichés les dimensions du tableau sont affichées en bas à gauche. They are in red text if any the following occur: row/column truncation, array transposition, or displaying only a range of elements - a tooltip displays the particular occurence(s).Ils sont en rouge si tout ce qui suit se produit: ligne/colonne tronquée, tableau transposé, ou affichage uniquement d'une plage d'éléments - une info-bulle affiche les occurrences particulières.

If the "verbose" parameter is set in $iFlags a splash dialog is displayed during initial processing when the array to display has more than 10000 elements.Si le paramètre "verbose" est installé dans $iFlags une boîte de dialogue 'splash' est affichée pendant le traitement initial lorsque le tableau d'affichage a plus de 10000 éléments. A similar dialog is displayed when the "Copy Data" buttons are used on a 10000+ element display - if this is likely to occur consideration should be given to using a "User Function" to store the array for later processing.

An array element containing another array is displayed as {Array}.

ExampleExemple

#include <Array.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()

    ; Create 1D array to displayCrée un tableau 1D à afficher
    Local $aArray_1D[5] = ["Item 0", "Item 1", "A longer Item 2 to show column expansion", "Item 3", "Item 4"]

    _ArrayDisplay($aArray_1D, "1D display")
    _ArrayDisplay($aArray_1D, "1D display transposed - 'Copy' buttons only", Default, 1 + 16) ; No buttons displayedPas de boutons affichés

    ; Create 2D array to displayCrée un tableau 2D à afficher
    Local $aArray_2D[75][255]
    For $i = 0 To UBound($aArray_2D) - 1
        For $j = 0 To UBound($aArray_2D, 2) - 1
            $aArray_2D[$i][$j] = "Item " & StringFormat("%02i", $i) & StringFormat("%02i", $j)
        Next
    Next

    _ArrayDisplay($aArray_2D, "2D display") ; Note columns truncatedNotez les colonnes tronquées
    _ArrayDisplay($aArray_2D, "2D display transposed", Default, 1) ; Note all array elements displayedNotez tous les éléments du tableau affichées

    ReDim $aArray_2D[20][10]
    $aArray_2D[5][5] = "A longer item to show column expansion"
    _ArrayDisplay($aArray_2D, "Expanded column - custom titles - alternate line color - no buttons or 'Row' column", Default, 32 + 64, Default, "AA|BB|CC|DD|EE|FF|GG|HH|II|JJ", Default, 0xDDFFDD)

    ; Assign the user function to a variable to pass as a parameterAssigne la fonction utilisateur à une variable pour la passer en paramètre
    Local $hUserFunction = _UserFunc

    $aArray_2D[5][5] = "Column alignment set to right"
    _ArrayDisplay($aArray_2D, "Range set - right align - copy column width - user function", "3:7|4:9", 2, 15, "AA|BB|CC|DD|EE|FF", Default, Default, $hUserFunction)
    _ArrayDisplay($aArray_2D, "Range set - right align - copy column width - transposed", "3:7|4:9", 3, 15, "AA|BB|CC|DD|EE|FF") ; Note no col names as transposedNotez qu'aucun nom de col n'a été transposé

    ; Create non-array variable to force error - MsgBox displayed as $iFlags setCréé une variable non tableau pour forcer une erreur - MsgBox s'affiche car $iFlags est installé
    Local $vVar = 0
    _ArrayDisplay($vVar, "MsgBox on Error", Default, 8)

EndFunc   ;==>Example

    ; But if a parameter is not used do this to prevent an Au3Check warningMais si un paramètre n'est pas utilisé, faire ceci pour éviter un avertissement de Au3Check
    #forceref $aArray_2D

    _ArrayDisplay($aSelected, "Selected cols")

EndFunc   ;==>_UserFunc
