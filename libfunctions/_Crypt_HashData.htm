_Crypt_HashData

Hash data with specified algorithmDonnées de hachage avec l'algorithme spécifié

#include <Crypt.au3>
_Crypt_HashData ( $vData, $iAlgID [, $bFinal = True [, $hCryptHash = 0]] )

ParametersParamètres

$vData Data to hashDonnées à hacher
$iAlgID Hash ID to useID de hashage à utiliser
$bFinal [optional][optionnel] False if this is only a segment of the full data, also makes the function return a hash object instead of hashFalse si c'est seulement un segment de données complet, et fait que la fonction retourne un objet de hachage au lieu d'un hachage
$hCryptHash [optional][optionnel] Hash object returned from a previous call to _Crypt_HashData()Objet de hachage retourné par un précédent appel à _Crypt_HashData()

Return ValueValeur de retour

Success:Succès : a hash or hash object if $bFinal = False.un hachage ou objet de hachage si $bFinal = False.
Failure:Échec : -1 and sets the @error flag to non-zero.-1 et met @error <> 0.
@error:@error : 10 - Failed to create hash object10 - Impossible de créer l'objet de hachage
20 - Failed to hash data20 - Impossible de hacher les données
30 - Failed to get hash size30 - Impossible d'obtenir la taille de hachage
40 - Failed to get hash40 - Impossible d'obtenir le hachage

RemarksRemarques

The hash digest will be returned as a binary string, the size is specified by the algorithm.La synthèse de hachage (hash digest) sera retournée comme une chaîne binaire, la taille est spécifiée par l'algorithme.
To use this with segments of data set the $bFinal flag to False for all non ending segments and use the returned hash object with all subsequent calls.Pour utiliser ceci avec des segments de données, mettez l'indicateur $bFinal à False pour tous les segments non finaux et utilise l'objet de hachage retourné avec tous les appels ultérieurs.

ExampleExemple

#include <ComboConstants.au3>
#include <Crypt.au3>
#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <StringConstants.au3>
#include <WinAPI.au3>
#include <WindowsConstants.au3>

Global $g_iAlgorithm = $CALG_SHA1, $g_idInputEdit = -1, $g_idOutputEdit = -1

Example()

Func Example()
    Local $hGUI = GUICreate("Realtime HashingHachage en temps réel", 400, 320)
    $g_idInputEdit = GUICtrlCreateEdit("", 0, 0, 400, 150, $ES_WANTRETURN)
    $g_idOutputEdit = GUICtrlCreateEdit("", 0, 150, 400, 150, $ES_READONLY)
    Local $idCombo = GUICtrlCreateCombo("", 0, 300, 100, 20, $CBS_DROPDOWNLIST)
    GUICtrlSetData($idCombo, "MD2|MD4|MD5|SHA1", "SHA1")
    GUIRegisterMsg($WM_COMMAND, "WM_COMMAND")
    GUISetState(@SW_SHOW, $hGUI)

    _Crypt_Startup() ; To optimize performance start the crypt library.Exécute la bibliothèque crypt pour optimiser les performances.

    Local $dHash = 0, _
            $sRead = ""
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idCombo ; Check when the combobox is selected and retrieve the correct algorithm.Vérifie si le combobox est sélectionné et récupère le bon algorithme.
                Switch GUICtrlRead($idCombo) ; Read the combobox selection.Lit la sélection du combobox.
                    Case "MD2"
                        $g_iAlgorithm = $CALG_MD2

                    Case "MD4"
                        $g_iAlgorithm = $CALG_MD4

                    Case "MD5"
                        $g_iAlgorithm = $CALG_MD5

                    Case "SHA1"
                        $g_iAlgorithm = $CALG_SHA1

                EndSwitch

                $sRead = GUICtrlRead($g_idInputEdit)
                If StringStripWS($sRead, $STR_STRIPALL) <> "" Then ; Check there is text available to hash.Vérifie qu'il y a du texte à hacher.
                    $dHash = _Crypt_HashData($sRead, $g_iAlgorithm) ; Create a hash of the text entered.Crée un hachage du texte entré.
                    GUICtrlSetData($g_idOutputEdit, $dHash) ; Set the output box with the hash data.Définit la boîte de dialogue de sortie avec les données de hachage.
                EndIf
        EndSwitch
    WEnd

    GUIDelete($hGUI) ; Delete the previous GUI and all controls.Supprime la GUI précédente et tous ses contrôles.
    _Crypt_Shutdown() ; Shutdown the crypt library.Arrête la bibliothèque crypt.
EndFunc   ;==>Example

Func WM_COMMAND($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg, $lParam

    Switch _WinAPI_LoWord($wParam)
        Case $g_idInputEdit
            Switch _WinAPI_HiWord($wParam)
                Case $EN_CHANGE
                    Local $dHash = _Crypt_HashData(GUICtrlRead($g_idInputEdit), $g_iAlgorithm) ; Create a hash of the text entered.Crée un hachage du texte entré.
                    GUICtrlSetData($g_idOutputEdit, $dHash) ; Set the output box with the hash data.Définit la boîte de dialogue de sortie avec les données de hachage.
            EndSwitch
    EndSwitch
EndFunc   ;==>WM_COMMAND
