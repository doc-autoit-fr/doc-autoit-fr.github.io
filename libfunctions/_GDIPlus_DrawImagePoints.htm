_GDIPlus_DrawImagePoints

Draws an image at a specified locationDesssine une image à une position spécifiée

#include <GDIPlus.au3>
_GDIPlus_DrawImagePoints ( $hGraphic, $hImage, $nULX, $nULY, $nURX, $nURY, $nLLX, $nLLY [, $iCount = 3] )

ParametersParamètres

$hGraphic Handle to a Graphics objectHandle de l'objet Graphics
$hImage Handle to an Image objectHandle de l'objet l'image
$nULX The X coordinate of the upper left corner of the source imageLa coordonnée X du coin supérieur gauche de l'image source
$nULY The Y coordinate of the upper left corner of the source imageLa coordonnée Y du coin supérieur gauche de l'image source
$nURX The X coordinate of the upper right corner of the source imageLa coordonnée X du coin supérieur droit de l'image source
$nURY The Y coordinate of the upper right corner of the source imageLa coordonnée Y du coin supérieur droit de l'image source
$nLLX The X coordinate of the lower left corner of the source imageLa coordonnée X du coin inférieur gauche de l'image source
$nLLY The Y coordinate of the lower left corner of the source imageLa coordonnée Y du coin inférieur gauche de l'image source
$iCount [optional][optionnel] Specifies the number of points (x,y)'s in the structure.Le nombre de points (x, y) de la structure.

Return ValueValeur de retour

Success:Succès : True
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code (GDIP_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS ( GDIP_ERR*).

RemarksRemarques

The value of the count parameter must equal 3 to specify the coordinates of the upper-left corner, upper-right corner, and lower-left corner of the parallelogram.La valeur du paramètre $iCount doit être égal à 3 pour spécifier les coordonnées du coin supérieur gauche, coin supérieur droit, et le coin inférieur gauche du parallélogramme.
The coordinate of the lower-right corner, the width, and the height of the image, are calculated using the three given coordinates.La coordonnée du coin en bas à droite, la largeur et la hauteur de l'image, sont calculés à l'aide des trois coordonnées données.
The image is scaled to fit the parallelogram.L'image est redimensionnée à la taille du parallélogramme.

ExampleExemple

#include <GDIPlus.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    Local $hBitmap1, $hBitmap2, $hImage1, $hImage2, $hGraphic

    ; Initialize GDI+ libraryInitialise GDI+ Librairie
    _GDIPlus_Startup()

    ; Capture full screenCapture un plein écran
    $hBitmap1 = _ScreenCapture_Capture("")
    $hImage1 = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap1)

    ; Capture screen regionCapture une région de l'écran
    $hBitmap2 = _ScreenCapture_Capture("", 0, 0, 400, 300)
    $hImage2 = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap2)

    ; Draw one image in anotherDessine une image dans une autre
    $hGraphic = _GDIPlus_ImageGetGraphicsContext($hImage1)

    _GDIPlus_DrawImagePoints($hGraphic, $hImage2, 100, 100, 600, 170, 130, 570)

    ; Draw a frame around the inserted imageDessine un cadre autour de l'image insérée
    _GDIPlus_GraphicsDrawRect($hGraphic, 100, 100, 400, 300)

    ; Save resultant imageEnregistre l'image qui en résulte
    _GDIPlus_ImageSaveToFile($hImage1, @MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Clean up resourcesLibère les ressources
    _GDIPlus_ImageDispose($hImage1)
    _GDIPlus_ImageDispose($hImage2)
    _WinAPI_DeleteObject($hBitmap1)
    _WinAPI_DeleteObject($hBitmap2)

    ; Shut down GDI+ libraryFerme la librairie GDI+
    _GDIPlus_Shutdown()

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.jpg")
EndFunc   ;==>Example
