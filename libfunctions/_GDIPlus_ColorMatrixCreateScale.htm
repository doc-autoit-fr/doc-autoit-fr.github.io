_GDIPlus_ColorMatrixCreateScale

Creates and initializes a scaling color matrixCrée et initialise une matrice couleur de mise à niveaux

#include <GDIPlus.au3>
_GDIPlus_ColorMatrixCreateScale ( $fRed, $fGreen, $fBlue [, $fAlpha = 1] )

ParametersParamètres

$fRed Red component scaling factorFacteur de la composante Rouge de la mise à niveau
$fGreen Green component scaling factorFacteur de la composante Vert de la mise à niveau
$fBlue Blue component scaling factorFacteur de la composante Bleu de la mise à niveau
$fAlpha [optional][optionnel][optionnel] Alpha component scaling factor

Return ValueValeur de retour

Returns a $tagGDIPCOLORMATRIX structure that contains a scaling color matrix.Retourne une structure $tagGDIPCOLORMATRIX qui contient une matrice couleur de mise à niveau.

RemarksRemarques

A scaling color matrix is used to multiply components of a color by multiplier factors.Une matrice couleur de mise à niveaux est utilisée pour multiplier les trois composantes d'une couleur par trois coefficients.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $iWidth = 600, $iHeight = 600

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;create a test GUIcréer une interface de test
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    Local $hIA = _GDIPlus_ImageAttributesCreate() ;create an ImageAttribute objectCrée un objet ImageAttribute

    Local $tColorMatrix = _GDIPlus_ColorMatrixCreateScale(0.5, 0.5, 0.5) ;create the scale factor color matrix for RGB color channelCrée la matrice couleur de mise à niveaux avec des facteurs 0.5

    _GDIPlus_ImageAttributesSetColorMatrix($hIA, 0, True, $tColorMatrix) ;decrease each color (RGB) level by 50% (darken)Diminue le niveau de chaque couleur (RVB) de 50% (assombrit)

    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, $iWidth, $iHeight) ;create a GDI bitmap by capturing an area on desktopCrée un bitmap GDI en capturant une zone du bureau
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;convert GDI to GDI+ bitmapCrée un bitmap GDI en capturant une région du bureau
    _WinAPI_DeleteObject($hHBmp) ;release GDI bitmap resource because not needed anymoreLibére la ressource du bitmap GDI car n'est plus nécessaire

    _GDIPlus_GraphicsDrawImageRectRect($hGraphics, $hBitmap, 0, 0, $iWidth, $iHeight, 0, 0, $iWidth, $iHeight, $hIA) ;draw the bitmap while applying the color adjustmentDessine le bitmap tout en appliquant la modification des couleurs

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _GDIPlus_ImageAttributesDispose($hIA)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
