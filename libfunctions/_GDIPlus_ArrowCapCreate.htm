_GDIPlus_ArrowCapCreate

Creates an adjustable arrow line cap with the specified height and widthCrée un embout de flèche, réglable en hauteur et en largeur, et pouvant être plein ou creux

#include <GDIPlus.au3>
_GDIPlus_ArrowCapCreate ( $fHeight, $fWidth [, $bFilled = True] )

ParametersParamètres

$fHeight Specifies the length, in units, of the arrow from its base to its pointSpécifie la distance, en unités, de la base de l'embout à la pointe
$fWidth Specifies the distance, in units, between the corners of the base of the arrowSpécifie, en unités, la largeur de la base de l'embout de la flèche
$bFilled [optional][optionnel] Fill flag:Paramètre de remplissage:
    True - Arrow will be filled
    False - Arrow will not be filled

Return ValueValeur de retour

Success:Succès : a handle to a new arrowCap object.Retourne le handle du nouvel objet 'embout de flèche'
Failure:Échec : sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

After you are done with the object, call _GDIPlus_ArrowCapDispose() to release the object resources.Après en avoir terminé avec l’objet, appelez la fonction _GDIPlus_ArrowCapDispose() pour libérer les ressources de l’objet.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic, $hPen, $hEndCap

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    ; Create resourcesCrée GUI
    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    $hPen = _GDIPlus_PenCreate(0xFF000000, 2)
    $hEndCap = _GDIPlus_ArrowCapCreate(3, 6)
    _GDIPlus_PenSetCustomEndCap($hPen, $hEndCap)

    ; Show pen end capAffiche l'embout du crayon
    MsgBox($MB_SYSTEMMODAL, "Information", "Pen end cap: 0x" & Hex(_GDIPlus_PenGetCustomEndCap($hPen)))

    ; Draw arrowsDessine des flèches
    _GDIPlus_GraphicsDrawLine($hGraphic, 10, 120, 390, 120, $hPen)
    _GDIPlus_PenSetWidth($hPen, 4)
    _GDIPlus_GraphicsDrawLine($hGraphic, 10, 150, 390, 150, $hPen)
    _GDIPlus_PenSetWidth($hPen, 6)
    _GDIPlus_GraphicsDrawLine($hGraphic, 10, 180, 390, 180, $hPen)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_ArrowCapDispose($hEndCap)
    _GDIPlus_PenDispose($hPen)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
