_GUICtrlTab_DeleteItem

Deletes a tab

#include <GuiTab.au3>
_GUICtrlTab_DeleteItem ( $hWnd, $iIndex )

ParametersParamètre

$hWnd Control ID/Handle to the control
$iIndex 0-based index of the item

Return ValueRemarque

Success: True.Vrai.
Failure: False.Retourne False.

Remarks<i0>

This does not delete the controls on the tabitems.

#include <GUIConstantsEx.au3>
#include <GuiTab.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $idTab

    ; Create GUICrée la GUI
    GUICreate("Tab Control Delete Item", 400, 300)
    $idTab = GUICtrlCreateTab(2, 2, 396, 296)
    GUISetState(@SW_SHOW)

    ; Add tabs
    _GUICtrlTab_InsertItem($idTab, 0, "Tab 1")
    _GUICtrlTab_InsertItem($idTab, 1, "Tab 2")
    _GUICtrlTab_InsertItem($idTab, 2, "Tab 3")

    ; Delete last tabs
    MsgBox($MB_SYSTEMMODAL, "Information", "Deleting last tab")
    _GUICtrlTab_DeleteItem($idTab, 2)

    ; Loop until the user exits.Création d'un objet chemin
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
