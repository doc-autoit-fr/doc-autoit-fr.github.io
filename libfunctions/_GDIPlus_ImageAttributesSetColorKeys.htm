_GDIPlus_ImageAttributesSetColorKeys

Sets or clears the color key (transparency range) for a specified categoryDéfinit ou efface la couleur clé (plage de transparence) pour une catégorie déterminée

#include <GDIPlus.au3>
_GDIPlus_ImageAttributesSetColorKeys ( $hImageAttributes [, $iColorAdjustType = 0 [, $bEnable = False [, $iARGBLow = 0 [, $iARGBHigh = 0]]]] )

ParametersParamètres

$hImageAttributes Pointer to an ImageAttribute objectHandle de l'objet ImageAttribute
$iColorAdjustType [optional][optionnel] The category for which the color key is set or cleared:La catégorie pour laquelle la couleur clé doit être définie ou effacée:
    0 - Color adjustment applies to all categories that do not have adjustment settings of their own
    1 - Color adjustment applies to bitmapped images
    2 - Color adjustment applies to brush operations in metafiles
    3 - Color adjustment applies to pen operations in metafiles
    4 - Color adjustment applies to text drawn in metafiles
$bEnable [optional][optionnel] If True, transparency range for the specified category is applied; otherwise, transparencySi Vrai, la plage de transparence pour la catégorie spécifiée est appliquée; autrement, transparence
$iARGBLow [optional][optionnel] Alpha, Red, Green and Blue components of a color that specifies the low color-key valueComposantes Alpha, Rouge, Vert et Bleu d'une couleur qui indique la valeur de la couleur clé la plus faible.
$iARGBHigh [optional][optionnel] Alpha, Red, Green and Blue components of a color that specifies the high color-key valueComposantes Alpha, Rouge, Vert et Bleu d'une couleur qui indique la valeur de la couleur clé la plus haute

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

RemarksRemarques

Any color that has each of its three components (red, green, blue) between the corresponding components of the high and low color keys is made transparent.Toute couleur qui a chacune de ses trois composantes (rouge, vert, bleu) entre les composantes correspondantes des couleurs clés haute et basse est transparente.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $iWidth = 600, $iHeight = 600

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;create a test GUIcréer une interface de test
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    Local $hIA = _GDIPlus_ImageAttributesCreate() ;create an ImageAttribute objectCrée un objet ImageAttribute

    _GDIPlus_ImageAttributesSetColorKeys($hIA, 0, True, 0x00000000, 0x00F7F7F7) ;make a range of colors transparentDéfinit une page de couleurs transparentes

    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, $iWidth, $iHeight) ;create a GDI bitmap by capturing an area on desktopCrée un bitmap GDI en capturant une zone du bureau
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;convert GDI to GDI+ bitmapCrée un bitmap GDI en capturant une région du bureau
    _WinAPI_DeleteObject($hHBmp) ;release GDI bitmap resource because not needed anymoreLibére la ressource du bitmap GDI car n'est plus nécessaire

    _GDIPlus_GraphicsDrawImageRectRect($hGraphics, $hBitmap, 0, 0, $iWidth, $iHeight, 0, 0, $iWidth, $iHeight, $hIA) ;draw the bitmap while applying the color adjustmentDessine le bitmap tout en appliquant la modification des couleurs

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _GDIPlus_ImageAttributesDispose($hIA)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
