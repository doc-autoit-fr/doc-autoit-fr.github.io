_SetDate

Sets the current date of the system

#include <Date.au3>
_SetDate ( $iDay [, $iMonth = 0 [, $iYear = 0]] )

ParametersParamètre

$iDay Day of the month. Values: 1-31
$iMonth [optional][optional] False - i, v, x ... month. Values: 1-12
$iYear [optional][optional] False - i, v, x ... year. Values: > 0 (windows might restrict this further!!)

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0 :
@error: 1 - Failure
@extended: _WinAPI_GetLastError()
Error code(s): MSDN: ms681381.aspx

Remarks<i0>

If the optional parameters ($iMonth,$iYear) are NOT defined, the function will not change the current value!
