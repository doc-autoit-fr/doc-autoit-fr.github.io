_WinAPI_EmptyWorkingSet

Removes as many pages as possible from the working set of the specified process

#include <WinAPIProc.au3>
_WinAPI_EmptyWorkingSet ( [$iPID = 0] )

ParametersParamètres

$iPID [optional][optionnel][optionnel] The PID of the process. Default (0) is the current process.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 and sets the @error flag to non-zero.0 et @error <> 0.
