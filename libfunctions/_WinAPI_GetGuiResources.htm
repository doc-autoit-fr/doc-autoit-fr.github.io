_WinAPI_GetGuiResources

Retrieves the count of handles to graphical user interface (GUI) objects in use by the specified process

#include <WinAPI.au3>
_WinAPI_GetGuiResources ( [$iFlag = 0 [, $hProcess = -1]] )

ParametersParamètres

$iFlag [optional][optionnel][optionnel] 0 (Default) Return the count of GDI objects.
1 Return the count of USER objects.
$hProcess [optional][optionnel][optionnel] A handle to the process. By default the current process.

Return ValueValeur de retour

Success:Succès : The count of handles to GUI objects in use by the process, according to $iFlag
Failure:Échec : 0, call _WinAPI_GetLastError() to get extended error information
