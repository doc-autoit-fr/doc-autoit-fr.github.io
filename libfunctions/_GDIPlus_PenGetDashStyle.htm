_GDIPlus_PenGetDashStyle

Gets the pen dash styleObtient le style de trait du crayon

#include <GDIPlus.au3>
_GDIPlus_PenGetDashStyle ( $hPen )

ParametersParamètre

$hPen Handle to a pen objectHandle d'un objet Pen

Return ValueRemarque

Success: the dash style:Retourne le style de trait:
    $GDIP_DASHSTYLESOLID - A solid line
    $GDIP_DASHSTYLEDASH - A dashed line
    $GDIP_DASHSTYLEDOT - A dotted line
    $GDIP_DASHSTYLEDASHDOT - An alternating dash-dot line
    $GDIP_DASHSTYLEDASHDOTDOT - An alternating dash-dot-dot line
    $GDIP_DASHSTYLECUSTOM - A a user-defined, custom dashed line
Failure: -1 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne -1 et @error <> 0, @extended GPSTATUS contien le code erreur GPSTATUS ($GPID_ERR*).

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic, $hPen

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    ; Create resourcesCrée GUI
    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    $hPen = _GDIPlus_PenCreate(0xFF000000, 8)
    _GDIPlus_PenSetDashStyle($hPen, $GDIP_DASHSTYLEDASHDOT)

    ; Show pen dash styleAffiche le style de crayon
    MsgBox($MB_SYSTEMMODAL, "Information", "Pen dash style: " & _GDIPlus_PenGetDashStyle($hPen))

    ; Draw lineDessine un segment
    _GDIPlus_GraphicsDrawLine($hGraphic, 10, 150, 390, 150, $hPen)

    ; Loop until the user exits.Création d'un objet chemin
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_PenDispose($hPen)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
