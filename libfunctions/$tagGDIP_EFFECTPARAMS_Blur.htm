$tagGDIP_EFFECTPARAMS_Blur

Blur effect parameter structureStructure du paramètre de l'effet Blur (flou)

Global Const $tagGDIP_EFFECTPARAMS_Blur = "float Radius; bool ExpandEdge"

ParametersParamètres

Radius Real number that specifies the sharpening radius (the radius of the convolution kernel) in pixels.Nombre réel qui indique le rayon de netteté (le rayon du noyau de convolution) en pixels.
The radius must be in the range 0.0 through 255.0Le rayon doit être compris entre 0,0 et 255,0
ExpandEdge Boolean value that specifies whether the bitmap expands by an amount equal to the blur radius.Valeur booléenne pour indiquer si la taille du bitmap augmente d'un montant égal au rayon de flou.
If TRUE, the bitmap expands by an amount equal to the radius so that it can have soft edges.Si TRUE, l'image bitmap se dilate d'un montant égal au rayon de sorte qu'il peut y avoir des bords flous.
If FALSE, the bitmap remains the same size and the soft edges are clipped
