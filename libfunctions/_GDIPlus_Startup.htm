_GDIPlus_Startup

Initialize Microsoft Windows GDI+Initialise Microsoft Windows GDI+

#include <GDIPlus.au3>
_GDIPlus_Startup ( [$sGDIPDLL = Default [, $bRetDllHandle = False]] )

ParametersParamètres

$sGDIPDLL [optional][optionnel][optionnel] the filename of the dll to be used.Le nom du fichier de la DLL à utiliser. Default is the installed GDI Dll.Par défaut, la DLL GDI installée.
$bRetDllHandle [optional][optionnel][optionnel] True if the handle to opened GDI Dll is to be returned.True si le handle pour ouvrir la DLL GDI doit être retourné. Default is False.Par défaut, False.

Return ValueValeur de retour

Success:Succès : True or a handle to the opened GDI Dll if $bRetDllHandle = True.Retourne True ou le handle de la DLL GDI ouvert si $bRetDllHandle = True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

RemarksRemarques

Call _GDIPlus_Startup() before you create any GDI+ objects.Appelez _GDIPlus_Startup() avant de créer tout objet GDI+.

If GDI+ V1.1 functions are available @extended will be set to value greater than 5.
For Vista or Server 2008 the $sGDIPDLL must be defined. The gdiplus.dll can be found in @WindowsDir & "\winsxs\*.gdiplus*\gdiplus.dll"

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPI.au3>

Example()

Func Example()
    Local $hGUI, $hBMP, $hBitmap, $hGraphic

    ; Capture upper left corner of screenCapturez le coin supérieur gauche de l'écran
    $hBMP = _ScreenCapture_Capture("", 0, 0, 400, 300)

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    ; Initialize GDI+ libraryInitialise GDI+ Librairie
    _GDIPlus_Startup()

    ; Draw bitmap to GUIDessine le bitmap dans la GUI
    $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hBMP)
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    _GDIPlus_GraphicsDrawImage($hGraphic, $hBitmap, 0, 0)

    ; Clean up resourcesLibère les ressources
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_ImageDispose($hBitmap)
    _WinAPI_DeleteObject($hBMP)

    ; Shut down GDI+ libraryFerme la librairie GDI+
    _GDIPlus_Shutdown()

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
