FontPrivate user defined functions Reference

Below is a complete list of the user defined functions available in AutoIt.Voici une liste complète des fonctions UDF de la rubrique. Click on a user defined function name for a detailed description.Cliquez sur le nom pour une description détaillée.

When using them you need to add a #include <GDIPlus.au3>.Pour les utiliser, vous aurez besoin d'ajouter #include <GDIPlus.au3>.

 

User Defined Function Description
_GDIPlus_FontPrivateAddFont Adds a font file to the private font collection
_GDIPlus_FontPrivateAddMemoryFont Adds a font that is contained in system memory to a Microsoft® Windows® GDI+ font collection
_GDIPlus_FontPrivateCollectionDispose Deletes the specified PrivateFontCollection object
_GDIPlus_FontPrivateCreateCollection Creates a PrivateFont Collection object
