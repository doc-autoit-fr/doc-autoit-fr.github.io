_FTP_FileTimeLoHiToStr

Converts filetime Loword and Hiword to a string

#include <FTPEx.au3>
_FTP_FileTimeLoHiToStr ( $iLoDWORD, $iHiDWORD [, $bFmt = 0] )

ParametersParamètres

$iLoDWORD FileTime Low
$iHiDWORD File Time Hi
$bFmt [optional][optionnel][optionnel]
    0 returns mm/dd/yyyy hh:mm:ss (Default)    0 retourne mm/dd/aaaa hh:mm:ss (par défaut)
    1 returns yyyy/mm/dd hh:mm:ss    1 retourne aaaa/mm/dd hh:mm:ss

Return ValueValeur de retour

Success:Succès : a datetime according to $bFmt.
Failure:Échec : an empty string.Retourne une chaîne vide.
