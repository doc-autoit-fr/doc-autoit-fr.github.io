_WinAPI_StrLen

Returns the length of the specified string

#include <WinAPIMisc.au3>
_WinAPI_StrLen ( $pString [, $bUnicode = True] )

ParametersParamètre

$pString Pointer to a null-terminated string.
$bUnicode [optional][optional] False - i, v, x ... Specifies whether a string is Unicode or ASCII code of a character, valid values:
True - Unicode (Default).
False - ASCII.

Return ValueRemarque

Success: The length of the string, in TCHARs values (not including the null-terminating character).
Failure: 0.Retourne 0.
