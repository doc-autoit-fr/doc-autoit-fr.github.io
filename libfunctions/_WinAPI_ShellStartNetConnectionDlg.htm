_WinAPI_ShellStartNetConnectionDlg

Displays a general browsing dialog box for a network resource connection

#include <WinAPIDlg.au3>
_WinAPI_ShellStartNetConnectionDlg ( [$sRemote = '' [, $iFlags = 0 [, $hParent = 0]]] )

ParametersParamètre

$sRemote [optional][optional] False - i, v, x ... The remote network name.
$iFlags [optional][optional] False - i, v, x ... The flags that identify the type of resource that the dialog is set to find. This parameter can be
one or more of the following values.
$hParent [optional][optional] False - i, v, x ... Handle of the parent window.

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.
