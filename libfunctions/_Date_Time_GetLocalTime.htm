_Date_Time_GetLocalTime

Retrieves the current local date and timeRécupère les date et heure locales actuelles

#include <Date.au3>
_Date_Time_GetLocalTime ( )

Return ValueValeur de retour

Returns a $tagSYSTEMTIME structure with the current local date and time.Retourne une structure $tagSYSTEMTIME avec la date et l'heure locale actuelles.

RelatedLiens

$tagSYSTEMTIME, _Date_Time_SetLocalTime

ExampleExemple

#include <Date.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsConstants.au3>

; Under Vista the Windows API "SetLocalime" may be rejected due to system securitySous Vista, l'API Windows « SetLocalime » peut être rejetée en raison de la sécurité du système

Global $g_idMemo

Example()

Func Example()
    Local $tCur, $tNew

    ; Create GUICrée la GUI
    GUICreate("TimeTemps", 400, 300)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 396, 296, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Show current local timeAffiche l'heure locale actuelle
    $tCur = _Date_Time_GetLocalTime()
    MemoWrite("Current date/time .:Date/heure actuelles . : " & _Date_Time_SystemTimeToDateTimeStr($tCur))

    ; Set new local timeDéfinit la nouvelle heure locale
    $tNew = _Date_Time_EncodeSystemTime(8, 19, @YEAR, 3, 10, 45)
    If Not _Date_Time_SetLocalTime($tNew) Then
        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "System clock cannot be SETL'horloge système ne peut pas être définie" & @CRLF & @CRLF & _WinAPI_GetLastErrorMessage())
        Exit
    EndIf
    $tNew = _Date_Time_GetLocalTime()
    MemoWrite("New date/time .....:Nouvelle date/heure... : " & _Date_Time_SystemTimeToDateTimeStr($tNew))

    ; Reset local timeRéinitialise l'heure locale
    _Date_Time_SetLocalTime($tCur)

    ; Show current local timeAffiche l'heure locale actuelle
    $tCur = _Date_Time_GetLocalTime()
    MemoWrite("Current date/time .:Date/heure actuelles . : " & _Date_Time_SystemTimeToDateTimeStr($tCur))

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Write a line to the memo controlÉcrit une ligne dans le contrôle mémo
Func MemoWrite($sMessage)
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
