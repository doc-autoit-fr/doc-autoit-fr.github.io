_WinAPI_CoInitialize

Initializes the COM library for use by the calling process

#include <WinAPICom.au3>
_WinAPI_CoInitialize ( [$iFlags = 0] )

ParametersParamètres

$iFlags [optional][optionnel][optionnel] This parameter can be one or more of the following values.
    $COINIT_APARTMENTTHREADED
    $COINIT_DISABLE_OLE1DDE
    $COINIT_MULTITHREADED (Default)
    $COINIT_SPEED_OVER_MEMORY

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended flag may contain the HRESULT error code.
