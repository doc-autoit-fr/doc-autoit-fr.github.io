$tagGDIP_EFFECTPARAMS_ColorCurve

ColorCurve effect parameter structureStructure du paramètre de l'effet ColorCurve (distorsion des couleurs)

Global Const $tagGDIP_EFFECTPARAMS_ColorCurve = "int Adjustment; int Channel; int AdjustValue"

ParametersParamètres

Adjustment CurveAdjustments constant that specifies the adjustment to be applied ($GDIP_Adjust*).Constante CurveAdjustments qui spécifie l'ajustement à appliquer ($GDIP_Adjust*).
Channel CurveChannel constant that specifies the color channel to which the adjustment applies ($GDIP_CurveChannel*).Constante CurveChannel spécifie le canal de couleur auquel l'ajustement s'applique ($GDIP_CurveChannel*).
AdjustValue Integer that specifies the intensity of the adjustment.Entier qui spécifie l'intensité de l'ajustement.
