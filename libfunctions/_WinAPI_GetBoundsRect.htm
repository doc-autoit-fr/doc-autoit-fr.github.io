_WinAPI_GetBoundsRect

Obtains the current accumulated bounding rectangle for a specified device context

#include <WinAPIGdi.au3>
_WinAPI_GetBoundsRect ( $hDC [, $iFlags = 0] )

ParametersParamètres

$hDC Handle to the device context whose bounding rectangle the function will return.
$iFlags [optional][optionnel][optionnel] The flags that specifies how the function will behave. This parameter can be the following value:
    $DCB_RESET

Return ValueValeur de retour

Success:Succès : $tagRECT structure that contains the current bounding rectangle, @extended flag will contain the value specifies the state of the accumulated bounding rectangle ($DCB_*).
Failure:Échec : Sets the @error flag to non-zero.Retourne @error <> 0
