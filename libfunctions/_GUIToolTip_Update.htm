_GUIToolTip_Update

Forces the current tool to be redrawn

#include <GuiToolTip.au3>
_GUIToolTip_Update ( $hWnd )

ParametersParamètre

$hWnd Handle to the ToolTip control (returned by _GUIToolTip_Create.)

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.

Remarks<i0>

This will redraw the currently displaying tooltip, it will have no effect on a tool if it is not showing when you call this function.
