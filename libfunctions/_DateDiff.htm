_DateDiff

Returns the difference between 2 dates, expressed in the type requestedRetourne la différence entre 2 dates, exprimée dans le type demandé

#include <Date.au3>
_DateDiff ( $sType, $sStartDate, $sEndDate )

ParametersParamètres

$sType One of the following:Un des suivants :
D = Difference in days between the given datesD = Différence en jours entre les dates données
M = Difference in months between the given datesM = Différence en mois entre les dates données
Y = Difference in years between the given datesY = Différence en années entre les dates données
w = Difference in Weeks between the given datesw = Différence en semaines entre les dates données
h = Difference in hours between the given datesh = Différence en heures entre les dates données
n = Difference in minutes between the given datesn = Différence en minutes entre les dates données
s = Difference in seconds between the given datess = Différence en secondes entre les dates données
$sStartDate Input Start date in the format "YYYY/MM/DD[ HH:MM:SS]"Entrez la date de début dans le format « AAAA/MM/JJ[ HH:MM:SS] »
$sEndDate Input End date in the format "YYYY/MM/DD[ HH:MM:SS]"Entrez la date de fin dans le format « AAAA/MM/JJ[ HH:MM:SS] »

Return ValueValeur de retour

Success:Succès : Difference between the 2 dates.Différence entre les 2 dates.
Failure:Échec : 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0.
@error:@error : 1 - Invalid $sType1 - $sType non valide
2 - Invalid $sStartDate2 - $sStartDate non valide
3 - Invalid $sEndDate3 - $sEndDate non valide

RemarksRemarques

Valid dates must be between "2000/01/01 00:00:00".Les dates valides doivent être comprises entre « 2000/01/01 00:00:00 » and "3000/12/31 23:59:59"et « 3000/12/31 23:59:59 ».

See _DateTimeSplit() for other possible variations of the start and end date formats.Voir _DateTimeSplit() pour d'autres variations possibles concernant les formats des dates de début et de fin.

ExampleExemple

#include <Date.au3>
#include <MsgBoxConstants.au3>

; Calculated the number of seconds since EPOCH (1970/01/01 00:00:00)Calcule le nombre de secondes depuis l'EPOCH (1970/01/01 00:00:00)
Local $iDateCalc = _DateDiff('s', "1970/01/01 00:00:00", _NowCalc())
MsgBox($MB_SYSTEMMODAL, "", "Number of seconds since EPOCH:Nombre de secondes depuis l'EPOCH : " & $iDateCalc)

; Calculated the number of Hours this yearCalcule le nombre d'heures pour l'année en cours
$iDateCalc = _DateDiff('h', @YEAR & "/01/01 00:00:00", _NowCalc())
MsgBox($MB_SYSTEMMODAL, "", "Number of Hours this year:Nombre d'heures pour l'année en cours : " & $iDateCalc)
