_GDIPlus_TextureCreate2

Creates a TextureBrush object based on an image, a wrap mode and a defining rectangleCrée un objet TextureBrush à partir d'une image, d'un mode d'habillage et d'une définition de rectangle

#include <GDIPlus.au3>
_GDIPlus_TextureCreate2 ( $hImage, $nX, $nY, $nWidth, $nHeight [, $iWrapMode = 0] )

ParametersParamètre

$hImage Pointer to an Image objectHandle de l'objet Image
$nX Leftmost coordinate of the image portion to be used by this brushCoordonnée X la plus à gauche de la partie d'image qui doit être utilisée par ce pinceau
$nY Uppermost coordinate of the image portion to be used by this brushCoordonnée Y la plus haute de la partie d'image qui doit être utilisée par ce pinceau
$nWidth Width of the brush and width of the image portion to be used by the brushLargeur du pinceau et largeur de la partie d'image qui doit être utilisée par ce pinceau
$nHeight Height of the brush and height of the image portion to be used by the brushHauteur du pinceau et hauteur de la partie d'image qui doit être utilisée par ce pinceau
$iWrapMode [optional][optional] False - i, v, x ... Wrap mode that specifies how repeated copies of an image are used to tile an area when it isMode wrap:Indique comment une image est disposée en mosaïque lorsqu'elle est plus petite que la zone à remplir:
    0 - Tiling without flipping
    1 - Tiles are flipped horizontally as you move from one tile to the next in a row
    2 - Tiles are flipped vertically as you move from one tile to the next in a column
    3 - Tiles are flipped horizontally as you move along a row and flipped vertically as you move along a column
    4 - No tiling takes place

Return ValueRemarque

Success: a pointer to a new TextureBrush object.Retourne le handle du nouvel objet TextureBrush.
Failure: 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

Remarks<i0>

After you are done with the object, call _GDIPlus_BrushDispose() to release the object resources.Lorsque vous en avez terminé avec l'objet, appelez _GDIPlus_BrushDispose() pour libérer les ressources de l'objet.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; X64 running supportX64 appui fonctionnement
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    ;get AutoIt install dirObtient le dossier d'installation de AutoIt
    Local $sRegPath = "HKLM\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt"

    Local $sFile = RegRead($sRegPath, "InstallDir") & "\Examples\GUI\logo4.gif"
    If Not FileExists($sFile) Then
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", $sFile & " not found!", 30)
        Return False
    EndIf

    _GDIPlus_Startup()
    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile) ;create an image object based on a fileCrée un objet Image à partir d'un fichier
    If @error Then
        _GDIPlus_Shutdown()
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "An error has occured - unable to load image!", 30)
        Return False
    EndIf

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", 320, 200)
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a Graphics object from a window handleCrée un objet Graphics à partir d'un handle fenêtre
    _GDIPlus_GraphicsClear($hGraphics, 0xFF404040) ;clear graphic handle with dark grey (background)Efface la surface graphique en gris foncé (fond)
    _GDIPlus_GraphicsSetSmoothingMode($hGraphics, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;sets the graphics object rendering quality (antialiasing)Définit l'objet graphique en qualité de rendu antialiasing
    Local $hTexture = _GDIPlus_TextureCreate2($hImage, 5, 4, 59, 59) ;create texture brush only from a defined rectangleCrée un pinceau de texture seulement à partir d'un rectangle défini
    _GDIPlus_GraphicsFillEllipse($hGraphics, 10, 40, 300, 120, $hTexture) ;draw ellipse with texture as a brushDessine une ellipse avec une texture comme pinceau

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup resourcesNettoie les ressources
    _GDIPlus_BrushDispose($hTexture)
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
