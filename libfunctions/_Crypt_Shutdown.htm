Référence Fonction

_Crypt_Shutdown

Uninitialize the Crypt libraryArrête la bibliothèque Crypt

#include <Crypt.au3>
_Crypt_Shutdown ( )

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

Calling this function is only needed if _Crypt_Startup() has been called.Appelez cette fonction n'est nécessaire que si _Crypt_Startup() a été appelée.

RelatedLiens

_Crypt_Startup

See AlsoVoir aussi

Search CryptReleaseContext in MSDN Library.Recherchez CryptReleaseContext dans la bibliothèque MSDN.

ExampleExemple

#include <Crypt.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    _Crypt_Startup() ; To optimize performance start the crypt library, though the same results will be shown if it isn't.Pour optimiser les performances exécuter la bibliothèque crypt, même si ce sont les mêmes résultats affichés dans le cas contraire.

    Local $sData = "..upon a time there was a language without any standardized cryptographic functions.... il était une fois un langage sans aucune fonction cryptographique normalisée. That language is no more.Ce langage n'est plus." ; Data that will be hashed.Données qui seront hachées.

    Local $sOutput = "The following results show the supported algorithms for retrieving the hash of the data.Les résultats suivants montrent les algorithmes pris en charge pour récupérer le hachage des données." & @CRLF & @CRLF & _
            "Text:Texte : " & $sData & @CRLF & _
            "MD2:MD2 : " & _Crypt_HashData($sData, $CALG_MD2) & @CRLF & _
            "MD4:MD4 : " & _Crypt_HashData($sData, $CALG_MD4) & @CRLF & _
            "MD5:MD5 : " & _Crypt_HashData($sData, $CALG_MD5) & @CRLF & _
            "SHA1:SHA1 : " & _Crypt_HashData($sData, $CALG_SHA1)

    MsgBox($MB_SYSTEMMODAL, "Supported algorithmsAlgorithmes pris en charge", $sOutput)

    _Crypt_Shutdown() ; Shutdown the crypt library.Arrête la bibliothèque crypt.
EndFunc   ;==>Example
