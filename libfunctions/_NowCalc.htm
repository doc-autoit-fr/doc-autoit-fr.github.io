Function ReferenceRéférence UDF

_NowCalc

Returns the current Date and Time in format YYYY/MM/DD HH:MM:SS for use in date calculationsRetourne la Date et l'heure actuelle au format AAAA/MM/JJ HH:MM:SS pour une utilisation dans les calculs de la date

#include <Date.au3>
_NowCalc ( )

Return ValueValeur de retour

Returns the current Date and Time in format YYYY/MM/DD HH:MM:SS.Retourne la Date et l'heure actuelle au format AAAA/MM/JJ HH:MM:SS.

RelatedLiens

_NowDate, _NowTime

ExampleExemple

#include <Date.au3>
#include <MsgBoxConstants.au3>

; Calculated the number of seconds since EPOCH (1970/01/01 00:00:00)Calcule le nombre de secondes depuis l'EPOCH (1970/01/01 00:00:00)
Local $iDateCalc = _DateDiff('s', "1970/01/01 00:00:00", _NowCalc())
MsgBox($MB_SYSTEMMODAL, "", "Number of seconds since EPOCH:Nombre de secondes depuis l'EPOCH : " & $iDateCalc)

; Calculated the number of Hours this yearCalcule le nombre d'heures pour l'année en cours
$iDateCalc = _DateDiff('h', @YEAR & "/01/01 00:00:00", _NowCalc())
MsgBox($MB_SYSTEMMODAL, "", "Number of Hours this year:Nombre d'heures pour l'année en cours : " & $iDateCalc)
