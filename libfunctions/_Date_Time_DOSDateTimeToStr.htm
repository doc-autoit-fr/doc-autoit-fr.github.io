_Date_Time_DOSDateTimeToStr

Decode a DOS date to a stringDécode une date DOS en une chaîne

#include <Date.au3>
_Date_Time_DOSDateTimeToStr ( $iDosDate, $iDosTime )

ParametersParamètres

$iDosDate MS-DOS date, packed as follows:Date MS-DOS, rempli comme suit :
Bits 0- 4 Day of the month (1–31)Bits 0-4 Jour du mois (1–31)
Bits 5- 8 Month (1 = January, 2 = February, and so on)Bits 5-8 Mois (1 = janvier, 2 = février, et ainsi de suite)
Bits 9-15 Year offset from 1980 (add 1980 to get actual year)Bits 9-15 Décalage de l'année par rapport à 1980 (ajoutez 1980 pour obtenir l'année actuelle)
$iDosTime MS-DOS date, packed as follows:Date MS-DOS, rempli comme suit :
Bits 0- 4 Second divided by 2Bits 0-4 Seconde divisée par 2
Bits 5-10 Minute (0–59)
Bits 11-15 Hour (0–23 on a 24-hour clock)Bits 11-15 Heure (0–23 sur une horloge de 24 heures)

Return ValueValeur de retour

Returns a date/time string formatted as mm/dd/yyyy hh:mm:ss.Retourne la date/heure sous forme d'une chaîne au format mm/jj/aaaa hh:mm.ss.

ExampleExemple

#include <Date.au3>
#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Global $g_idMemo

Example()

Func Example()
    Local $sDate

    ; Create GUICrée la GUI
    GUICreate("TimeTemps", 400, 300)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 396, 296, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Show FAT date/timeAffiche la date/heure FAT
    $sDate = _Date_Time_DOSDateTimeToStr(0x3621, 0x944a) ; 01/01/2007 18:34:20
    MemoWrite("FAT date/time .:date/heure FAT . : " & $sDate)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Write a line to the memo controlÉcrit une ligne dans le contrôle mémo
Func MemoWrite($sMessage)
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
