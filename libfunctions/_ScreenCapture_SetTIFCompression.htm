_ScreenCapture_SetTIFCompression

Sets the compression used for TIFF screen captures

#include <ScreenCapture.au3>
_ScreenCapture_SetTIFCompression ( $iCompress )

ParametersParamètre

$iCompress Image compression type:
    0 - Default encoder compression
    1 - No compression
    2 - LZW compression

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.

Remarks<i0>

If not explicitly set, TIF screen captures default to LZW compression.
