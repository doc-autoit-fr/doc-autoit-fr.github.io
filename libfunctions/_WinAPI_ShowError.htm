_WinAPI_ShowError

Displays an error message box with an optional exit

#include <WinAPI.au3>
_WinAPI_ShowError ( $sText [, $bExit = True] )

ParametersParamètre

$sText Error text to display
$bExit [optional][optional] False - i, v, x ... Specifies whether to exit after the display:
True - Exit program after display
False - Return normally after display

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.
