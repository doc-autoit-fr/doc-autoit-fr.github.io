_WinAPI_EnumChildProcess

Enumerates a child processes that belong to the specified process

#include <WinAPIProc.au3>
_WinAPI_EnumChildProcess ( [$iPID = 0] )

ParametersParamètres

$iPID [optional][optionnel][optionnel] The PID of the process. Default (0) is the current process.

Return ValueValeur de retour

Success:Succès : The 2D array of the PIDs and process names.
[0][0] - Number of rows in array (n)
[0][1] - Unused
[n][0] - PID
[n][1] - Process name
Failure:Échec : Sets the @error flag to non-zero.Retourne @error <> 0

ExampleExemple

#include <Array.au3>
#include <WinAPIProc.au3>

Local $aData = _WinAPI_EnumChildProcess(_WinAPI_GetParentProcess())

_ArrayDisplay($aData, '_WinAPI_EnumChildProcess')
