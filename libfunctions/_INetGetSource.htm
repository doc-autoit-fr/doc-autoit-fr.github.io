_INetGetSource

Gets the source from an URL without writing a temp file

#include <Inet.au3>
_INetGetSource ( $sURL [, $bString = True] )

ParametersParamètres

$sURL (The URL of the site.) eg 'http://www.autoitscript.com'(The URL of the site.)eg 'http://www.autoitscript.com'
$bString [optional][optionnel] If True the data is returned in string format, otherwise binary format.

Return ValueValeur de retour

Success:Succès : the read string and sets @extended to the number of bytes returned.
Failure:Échec : an empty string and and sets the @error flag to non-zero.
