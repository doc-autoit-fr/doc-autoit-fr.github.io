_GUICtrlMenu_GetItemGrayed

Retrieves the status of the menu item grayed state

#include <GuiMenu.au3>
_GUICtrlMenu_GetItemGrayed ( $hMenu, $iItem [, $bByPos = True] )

ParametersParamètres

$hMenu Handle of the menu
$iItem Identifier or position of the menu item
$bByPos [optional][optionnel] Menu identifier flag:
    True - $iItem is a 0-based item position
    False - $iItem is a menu item identifier

Return ValueValeur de retour

True: $iItem is grayed.
False: $iItem is not grayed.

RelatedLiens

_GUICtrlMenu_SetItemGrayed

ExampleExemple

#include <GuiMenu.au3>

Example()

Func Example()
    Local $hWnd, $hMain, $hFile

    ; Open Notepad
    Run("notepad.exe")
    WinWaitActive("[CLASS:Notepad]")
    $hWnd = WinGetHandle("[CLASS:Notepad]")
    $hMain = _GUICtrlMenu_GetMenu($hWnd)
    $hFile = _GUICtrlMenu_GetItemSubMenu($hMain, 0)

    ; Get/Set Open grayed
    Writeln("Open is grayed: " & _GUICtrlMenu_GetItemGrayed($hFile, 1))
    _GUICtrlMenu_SetItemGrayed($hFile, 1)
    Writeln("Open is grayed: " & _GUICtrlMenu_GetItemGrayed($hFile, 1))
EndFunc   ;==>Example

; Write a line of text to Notepad
Func Writeln($sText)
    ControlSend("[CLASS:Notepad]", "", "Edit1", $sText & @CRLF)
EndFunc   ;==>Writeln
