_ArrayDelete

Deletes the specified element(s) from the specified 1D or 2D arraySupprime les éléments spécifiés dans un tableau 1D ou 2D

#include <Array.au3>
_ArrayDelete ( ByRef $aArray, $vRange )

ParametersParamètres

$aArray Array to modifyTableau à transposer
$vRange Element(s) to delete - either a single index, a range string or a 1D array with a count in the [0] element (see example for details)Élément(s) à supprimer - soit un index unique, une chaîne contenant des plages (exemple "0;11-15;24") soit un tableau 1D avec le nombre d'éléments en position [0] (voir l'exemple pour plus de détails)

Return ValueValeur de retour

Success:Succès : the new size of the array (see remarks)Retourne la nouvelle taille du tableau (voir Remarques)
Failure:Échec : -1 and sets the @error flag to non-zero.-1 et met @error <> 0.
@error:@error : 1 - $aArray is not an array1 - $aArray n'est pas un tableau
2 - $aArray is not a 1D or 2D array2 - $aArray n'est pas un tableau 1D ou 2D
3 - $vRange is not a valid range string3 - $vRange n'est pas une chaîne de plage valide
4 - $vRange is not a 1D array or has only 1 element4 - $vRange n'est pas un tableau 1D ou a seulement 1 élément
5 - $vRange content is outside array bounds5 - $vRange a un contenu en dehors des limites du tableau

RemarksRemarques

$vRange can be a string containing the rows which are to be deleted.$vRange peut être une chaîne contenant les lignes qui doivent être supprimés. It can be a single number or a range denoted by the first and last lines separated by a hyphen (-) - multiple items are separated by a semi-colon (;).Il peut être un nombre unique ou une plage désignée par les première et dernière lignes séparées par un tiret (-) - plusieurs éléments sont séparés par un point-virgule (;).

$vRange can also be a 1D array listing all rows to be deleted with the count in the [0] element].$vRange peut aussi être un tableau 1D représentant toutes les lignes à supprimer avec le nombre d'éléments en position [0].

In either case, the rows need not be in ascending order and can be duplicated.Dans les deux cas, les lignes ne sont pas nécessairement dans l'ordre croissant et peuvent être dupliqués.

ExampleExemple

#include <Array.au3>

Local $aArray[5] = [0, 1, 2, 3, 4]

_ArrayDisplay($aArray, "Original")
_ArrayDelete($aArray, 2)
_ArrayDisplay($aArray, "Element 2 deleted")

Local $aArray_Base[25][4]
For $i = 0 To 24
    For $j = 0 To 3
        $aArray_Base[$i][$j] = $i & "-" & $j
    Next
Next

; Single rowLigne unique
$aArray = $aArray_Base
_ArrayDisplay($aArray, "BEFORE deletion")
_ArrayDelete($aArray, 7)
_ArrayDisplay($aArray, "SINGLE ROW deleted")

; Range stringChaîne contenant des plages
$aArray = $aArray_Base
Local $sRange = "0;11-15;24"
_ArrayDisplay($aArray, "BEFORE deletion")
_ArrayDelete($aArray, $sRange)
ConsoleWrite(" " & @error & @CRLF)
_ArrayDisplay($aArray, "RANGE STRING deleted")

; 1D arrayTableau 1D
$aArray = $aArray_Base
Local $aDel[4] = [3, 5, 11, 13]
_ArrayDisplay($aArray, "BEFORE deletion")
_ArrayDelete($aArray, $aDel)
_ArrayDisplay($aArray, "RANGE ARRAY deleted")
