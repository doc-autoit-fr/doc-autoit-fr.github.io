_GUICtrlHeader_AddItem

Adds a new header item

#include <GuiHeader.au3>
_GUICtrlHeader_AddItem ( $hWnd, $sText [, $iWidth = 50 [, $iAlign = 0 [, $iImage = -1 [, $bOnRight = False]]]] )

ParametersParamètre

$hWnd Handle to the control
$sText Item text
$iWidth [optional][optional] False - i, v, x ... Item width
$iAlign [optional][optional] False - i, v, x ... Text alignment:
    0 - Text is left-aligned
    1 - Text is right-aligned
    2 - Text is centered
$iImage [optional][optional] False - i, v, x ... 0-based index of an image within the image list
$bOnRight [optional][optional] False - i, v, x ... If True, the column image appears to the right of text

Return ValueRemarque

Success: the index of the new item.
Failure: -1.Retourne -1.

#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hHeader

    ; Create GUICrée la GUI
    $hGUI = GUICreate("Header", 400, 300)
    $hHeader = _GUICtrlHeader_Create($hGUI)
    _GUICtrlHeader_SetUnicodeFormat($hHeader, True)
    GUISetState(@SW_SHOW)

    ; Add columns
    _GUICtrlHeader_AddItem($hHeader, "Column 1", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 2", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 3", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 4", 100)

    ; Loop until the user exits.Création d'un objet chemin
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
