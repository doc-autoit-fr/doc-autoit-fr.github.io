File user defined functions ReferenceFile - Référence

Below is a complete list of the user defined functions available in AutoIt.Voici une liste complète des fonctions UDF de la rubrique. Click on a user defined function name for a detailed description.Cliquez sur le nom pour une description détaillée.

When using them you need to add a #include <File.au3>.Pour les utiliser, vous aurez besoin d'ajouter #include <File.au3>.

 

User Defined Function Description
_FileCountLines Returns the number of lines in the specified file
_FileCreate Creates or zero's out the length of the file specified
_FileListToArray Lists files and\or folders in a specified folder (Similar to using Dir with the /B Switch)
_FileListToArrayRec Lists files and\or folders in specified path with optional recursion to defined level and result sorting
_FilePrint Prints a plain text file
_FileReadToArray Reads the specified file into a 1D or 2D array
_FileWriteFromArray Writes an array to a specified file
_FileWriteLog Writes current date, time and the specified text to a log file
_FileWriteToLine Writes text to a specific line in a file
_PathFull Creates a path based on the relative path you provide.Crée un chemin d'accès basé sur le chemin d'accès relatif que vous fournissez. The newly created absolute path is returnedLe chemin d'accès absolu nouvellement créé est retourné.
_PathGetRelative Returns the relative path to a directoryRetourne le chemin d'accès relatif à un répertoire
_PathMake Creates a path from drive, directory, file name and file extension parts
_PathSplit Splits a path into the drive, directory, file name and file extension parts. An empty string is set if a part is missing
_ReplaceStringInFile Replaces substrings in a file
_TempFile Generate a name for a temporary file. The file is guaranteed not to exist yet
