_WinAPI_RegDeleteValue

Removes a named value from the specified registry key

#include <WinAPIReg.au3>
_WinAPI_RegDeleteValue ( $hKey, $sValueName )

ParametersParamètres

$hKey Handle to an open registry key. The key must have been opened with the $KEY_SET_VALUE access right.
This handle is returned by the _WinAPI_RegCreateKey() or _WinAPI_RegOpenKey() function, or it can be
one of the following predefined keys.
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sValueName The registry value to be removed. If this parameter is empty string, the key's unnamed or default
value is removed.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended flag may contain the system error code.
