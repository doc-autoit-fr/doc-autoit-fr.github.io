_GDIPlus_GraphicsSetTextRenderingHint

Sets the contrast value of a Graphics objectDéfinit la valeur du contraste d'un objet Graphics

#include <GDIPlus.au3>
_GDIPlus_GraphicsSetTextRenderingHint ( $hGraphics, $iTextRenderingHint )

ParametersParamètres

$hGraphics Pointer to a Graphics objectLe handle de l'objet Graphics
$iTextRenderingHint Text rendering mode:Mode de rendu de texte:
    0 - Character is drawn using the currently selected system font smoothing mode (also called a rendering hint)
    1 - Character is drawn using its glyph bitmap and hinting to improve character appearance on stems and
    2 - Character is drawn using its glyph bitmap and no hinting. This results in better performance at theIl en résulte de meilleures performances au détriment de la qualité
    3 - Character is drawn using its antialiased glyph bitmap and hinting. This results in much better qualityIl en résulte une bien meilleure qualité
    4 - Character is drawn using its antialiased glyph bitmap and no hinting. Stem width differences may beDes différences de largeur des jambes peuvent être perceptible
    5 - Character is drawn using its glyph Microsoft ClearType bitmap and hinting. This type of text rendering

Success:Succès : True.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hGUI = GUICreate("GDI+ test", 800, 400)
    GUISetState(@SW_SHOW)

    If @OSBuild < 6000 Then MsgBox($MB_SYSTEMMODAL, "", "Antialiasing is automatically turned on for your operating system - no visible differences!")

    _GDIPlus_Startup()
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    _GDIPlus_GraphicsDrawString($hGraphics, "AutoIt rulez!", 0, 0, "Impact", 110)
    _GDIPlus_GraphicsSetTextRenderingHint($hGraphics, $GDIP_TEXTRENDERINGHINT_ANTIALIASGRIDFIT)
    _GDIPlus_GraphicsDrawString($hGraphics, "AutoIt rulez!", 0, 200, "Impact", 110)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    ;cleanup resourcesNettoie les ressources
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
