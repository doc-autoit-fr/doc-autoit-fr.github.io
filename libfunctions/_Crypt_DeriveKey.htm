_Crypt_DeriveKey

Creates a key from algorithm and passwordCrée une clé d'après l'algorithme et le mot de passe

#include <Crypt.au3>
_Crypt_DeriveKey ( $vPassword, $iAlgID [, $iHashAlgID = $CALG_MD5] )

ParametersParamètres

$vPassword Password to useMot de passe à utiliser
$iAlgID Encryption ID of algorithm to be used with the keyID de chiffrement de l'algorithme à utiliser avec la clé
$iHashAlgID [optional][optionnel] Id of the algo to hash the password withID de l'algorithme de hachage du mot de passe

Return ValueValeur de retour

Success:Succès : a handle to a cryptographic key.un handle vers une clé de chiffrement.
Failure:Échec : -1 and sets the @error flag to non-zero.-1 et met @error <> 0.
@error:@error : 10 - Failed to create hash object10 - Impossible de créer l'objet de hachage
20 - Failed to hash password20 - Impossible de hacher le mot de passe
30 - Failed to generate key30 - Impossible de générer la clé

RemarksRemarques

The key needs to be destroyed with _Crypt_DestroyKey().La clé doit être détruite avec _Crypt_DestroyKey().

ExampleExemple

#include <Crypt.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aStringsToEncrypt[6] = ["AutoIt", "SciTE", "Crypt", ".au3", 42, "42"]
    Local $sOutput = ""

    Local $hKey = _Crypt_DeriveKey("CryptPassword", $CALG_RC4) ; Declare a password string and algorithm to create a cryptographic key.Déclare un mot de passe sous forme de châine et l'algorithme pour créer une clé de chiffrement.

    For $iWord In $aStringsToEncrypt
        $sOutput &= $iWord & @TAB & " = " & _Crypt_EncryptData($iWord, $hKey, $CALG_USERKEY) & @CRLF ; Encrypt the text with the cryptographic key.Chiffre le texte avec la clé de chiffrement.
    Next

    MsgBox($MB_SYSTEMMODAL, "Encrypted dataDonnées chiffrées", $sOutput)

    _Crypt_DestroyKey($hKey) ; Destroy the cryptographic key.Détruit la clé de chiffrement.
EndFunc   ;==>Example
