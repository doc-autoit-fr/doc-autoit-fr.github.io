_GDIPlus_BrushSetSolidColor

Set the color of a Solid Brush objectDéfinit la couleur d'un pinceau de couleur unie

#include <GDIPlus.au3>
_GDIPlus_BrushSetSolidColor ( $hBrush [, $iARGB = 0xFF000000] )

ParametersParamètre

$hBrush Handle to a Brush objectHandle de l'objet Brush
$iARGB [optional][optional] False - i, v, x ... Alpha, Red, Green and Blue components of brushComposantes couleur du pinceau:Alpha, Rouge, Vert et Bleu

Return ValueRemarque

Success: TrueRetourne True
Failure: False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Example()

Func Example()
    Local $hGUI, $idLabel1, $idLabel2, $hGraphic, $hBrush1, $iClr1, $iClr2

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 345, 150)
    $idLabel1 = GUICtrlCreateLabel("", 2, 2, 150, 30)
    $idLabel2 = GUICtrlCreateLabel("", 202, 2, 150, 30)
    GUISetState(@SW_SHOW)
    Sleep(100)

    ; Start GDIPlusDémarre GDIPlus
    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)

    ; Create solid brushCrée un pinceau uni
    $hBrush1 = _GDIPlus_BrushCreateSolid()

    ; Get solid brush original colorObtient la couleur du pinceau d'origine d'origine
    $iClr1 = _GDIPlus_BrushGetSolidColor($hBrush1)

    ; Draw some graphics with the original brush colorDessine des graphiques avec la couleur du pinceau d'origine
    _GDIPlus_GraphicsFillEllipse($hGraphic, 25, 35, 100, 100, $hBrush1)

    ; Set new brush color (0xFFFF0000 = Red)Définit une nouvelle couleur du pinceau (0xFFFF0000 = rouge)
    _GDIPlus_BrushSetSolidColor($hBrush1, 0xFFFF0000)

    ; Get solid brush new colorObtient la nouvelle couleur du pinceau
    $iClr2 = _GDIPlus_BrushGetSolidColor($hBrush1)

    ; Draw some graphics with the new brush colorDessine des graphiques avec la nouvelle couleur du pinceau
    _GDIPlus_GraphicsFillRect($hGraphic, 220, 35, 100, 100, $hBrush1)

    ; Write original brush color to Label1Ecrit avec le pinceau d'origine dans Label1
    GUICtrlSetData($idLabel1, "Brush orignal color: " & Hex($iClr1))

    ; Write the new brush color to Label2Ecrit avec la nouvelle couleur du pinceau dans label2
    GUICtrlSetData($idLabel2, "Brush new color: " & Hex($iClr2))

    ; Loop until the user exits.Création d'un objet chemin
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_BrushDispose($hBrush1)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
