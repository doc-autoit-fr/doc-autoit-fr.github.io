_GUICtrlListView_GetItemText

Retrieves the text of an item or subitem

#include <GuiListView.au3>
_GUICtrlListView_GetItemText ( $hWnd, $iIndex [, $iSubItem = 0] )

ParametersParamètres

$hWnd Control ID/Handle to the control
$iIndex 0-based index of the item
$iSubItem [optional][optionnel][optionnel] 1-based sub item index

Return ValueValeur de retour

Returns the item or subitem text.

RemarksRemarques

To retrieve the item text, set $iSubItem to 0.
To retrieve the text of a subitem, set $iSubItem to the 1-based subitem's index.

RelatedLiens

_GUICtrlListView_GetItemTextArray, _GUICtrlListView_GetItemTextString, _GUICtrlListView_SetItemText

ExampleExemple

#include <GUIConstantsEx.au3>
#include <GuiListView.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $idListview

    GUICreate("ListView Get Item Text", 400, 300)
    $idListview = GUICtrlCreateListView("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Add columns
    _GUICtrlListView_AddColumn($idListview, "Items", 100)

    ; Add items
    _GUICtrlListView_AddItem($idListview, "Item 1")
    _GUICtrlListView_AddItem($idListview, "Item 2")
    _GUICtrlListView_AddItem($idListview, "Item 3")

    ; Set item 2 text
    _GUICtrlListView_SetItemText($idListview, 1, "New Item 2")
    MsgBox($MB_SYSTEMMODAL, "Information", "Item 2 Text: " & _GUICtrlListView_GetItemText($idListview, 1))

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
