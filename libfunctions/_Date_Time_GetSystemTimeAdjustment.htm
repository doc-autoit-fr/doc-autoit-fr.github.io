_Date_Time_GetSystemTimeAdjustment

Determines whether the system is applying periodic time adjustmentsDétermine si le système applique des ajustements périodiques de l'heure

#include <Date.au3>
_Date_Time_GetSystemTimeAdjustment ( )

Return ValueValeur de retour

Returns an array with the following format:Retourne un tableau (array) au format suivant :
[1] - The number of 100 nanosecond units added to the clock at each periodic time adjustment[1] - Le nombre d'unités de 100 nanosecondes ajoutées à l'horloge à chaque réglage périodique de l'heure
[2] - The number of 100 nanosecond units between periodic time adjustments.[2] - Le nombre d'unités de 100 nanosecondes entre les ajustements périodiques de l'heure
This interval is the time period between a system's clock interrupts.Cet intervalle correspond à la période de temps entre les interruptions d'horloge du système.
[3] - True indicates that periodic time adjustment is disabled.[3] - True indique que l'ajustement périodique de l'heure est désactivée.
At each clock interrupt, the system merely adds the interval between clock interrupts to the clock.À chaque interruption d'horloge, le système ajoute simplement, l'intervalle entre les interruptions d'horloge, à l'horloge.
If False, periodic time adjustment is being used to adjust the clock.Si False, l'ajustement périodique de l'heure est utilisé pour ajuster l'heure.

#include <Date.au3>

Example()

Func Example()
    Local $aInfo

    ; Open the clock so we can watch the funOuvre l'horloge pour voir un truc amusant
    Run("RunDll32.exe shell32.dll,Control_RunDLL timedate.cpl")
    WinWaitActive("[CLASS:#32770]")

    ; Get current time adjustmentsRécupère les réglages de l'heure actuelle
    $aInfo = _Date_Time_GetSystemTimeAdjustment()

    ; Slow down clockRalentit l'horloge
    If Not _Date_Time_SetSystemTimeAdjustment($aInfo[1] / 10, False) Then
        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "System clock cannot be DOWNL'horloge système ne peut pas être RALENTIE." & @CRLF & @CRLF & _WinAPI_GetLastErrorMessage())
        Exit
    EndIf
    MsgBox($MB_SYSTEMMODAL, "Information", "Slowing down system clockRalentit l'horloge système", 2)

    Sleep(5000)

    ; Speed up clockAccélére l'horloge
    If Not _Date_Time_SetSystemTimeAdjustment($aInfo[1] * 10, False) Then
        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "System clock cannot be UPL'horloge système ne peut pas être ACCÉLÉRÉE." & @CRLF & @CRLF & _WinAPI_GetLastErrorMessage())
    EndIf
    MsgBox($MB_SYSTEMMODAL, "Information", "Speeding up system clockAccélération de l'horloge système", 2)

    Sleep(5000)

    ; Reset time adjustmentRéinitialise le réglage de l'heure
    If Not _Date_Time_SetSystemTimeAdjustment($aInfo[1], True) Then
        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "System clock cannot be RESETL'horloge système ne peut pas être RÉINITIALISÉE" & @CRLF & @CRLF & _WinAPI_GetLastErrorMessage())
    Else
        MsgBox($MB_SYSTEMMODAL, "Information", "System clock restoredHorloge système restaurée")
    EndIf
EndFunc   ;==>Example
