_WinAPI_GetDesktopWindow

Returns the handle of the Windows desktop window

#include <WinAPI.au3>
_WinAPI_GetDesktopWindow ( )

Return ValueValeur de retour

Returns the Handle of the desktop window

#include <MsgBoxConstants.au3>
