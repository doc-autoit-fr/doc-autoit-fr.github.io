_SQLite_ErrMsg

Returns a string describing in english the error condition for the most recent sqlite3_* API call

#include <SQLite.au3>
_SQLite_ErrMsg ( [$hDB = -1] )

ParametersParamètres

$hDB [optional][optionnel][optionnel] An open database, default is the last opened database

Return ValueValeur de retour

Success:Succès : an error message string.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.
@error:@error : 1 - error calling SQLite API 'sqlite3_errmsg16'
2 - Call Prevented by SafeMode
