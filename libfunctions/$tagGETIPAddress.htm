$tagGETIPAddress

Contains information for all 4 fields of the IP Address controlContient des informations pour tous les 4 champs du contrôle de l'adresse IP

Global Const $tagGETIPAddress = "byte Field4;byte Field3;byte Field2;byte Field1"

ParametersParamètres

Field4 contains bits 0 through 7
Field3 contains bits 8 through 15
Field2 contains bits 16 through 23
Field1 contains bits 24 through 31
