_ArraySearch

Finds an entry within a 1D or 2D array.Cherche une valeur dans un tableau 1D ou 2D. Similar to _ArrayBinarySearch(), except that the array does not need to be sortedSimilaire à _ArrayBinarySearch(), sauf que le tableau n'a pas besoin d'être trié

#include <Array.au3>
_ArraySearch ( Const ByRef $aArray, $vValue [, $iStart = 0 [, $iEnd = 0 [, $iCase = 0 [, $iCompare = 0 [, $iForward = 1 [, $iSubItem = -1 [, $bRow = False]]]]]]] )

ParametersParamètre

$aArray The array to searchLe tableau où chercher
$vValue What to search $aArray forCe qu'il faut rechercher dans $aArray
$iStart [optional][optional] False - i, v, x ... Index of array to start searching atIndex du tableau où commencer la recherche
$iEnd [optional][optional] False - i, v, x ... Index of array to stop searching atIndex du tableau où arrêter la recherche
$iCase [optional][optional] False - i, v, x ... If set to 1, search is case sensitiveSi la valeur est 1, la recherche est sensible à la casse
$iCompare [optional][optional] False - i, v, x ...
    0 Casting of variables to the same type (default), "string" = 0, "" = 0 or "0" = 0 match (If $iCase = 0)
    1 executes a partial search
    2 comparison match if variables have same type and same value
    3 compares using a regular expression pattern provided as $vValue
$iForward [optional][optional] False - i, v, x ... If set to 0, searches the array from end to beginning (instead of beginning to end)Si défini à 0, la recherche dans le tableau se fait de la fin au début (au lieu du début à la fin)
$iSubItem [optional][optional] False - i, v, x ... Sub-index to search on in 2D arraysSous-index de recherche dans un tableau 2D
$bRow [optional][optional] False - i, v, x ... If True then $iSubItem sets the row to search - False (default) searches columnsSi True, $iSubItem définit la ligne où chercher.Par défaut, False, définit la colonne où chercher

Return ValueRemarque

Success: the index that $vValue was found at.Retourne l'index où $vValeur a été trouvé.
Failure: -1 and sets the @error flag to non-zero.Retourne -1 et @error <> 0.
@error: 1 - $aArray is not an array1 - $aArray n'est pas un tableau
2 - $aArray is not a 1D or 2D array2 - $aArray n'est pas un tableau 1D ou 2D
3 - $aArray is empty3 - $aArray est vide
4 - $iStart is greater than $iEnd4 - $iStart est supérieur à $iEnd
5 - Array not 2D and $bRow set True5 - Le tableau n'est pas 2D et $bRow est à true
6 - $vValue was not found in array6 - $vValeur n'a pas été trouvée dans le tableau

Remarks<i0>

This function is likely to be slower than _ArrayBinarySearch() but is useful when the array cannot be sorted before the search.Cette fonction est susceptible d'être plus lente que _ArrayBinarySearch() mais est utile lorsque le tableau ne peut pas être trié avant la recherche.

The values of $iCompare can not be combined together.

Example 1Exemple 1

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray[6][4]
For $i = 0 To 5
    For $j = 0 To 3
        $aArray[$i][$j] = "#" & $i & $j
    Next
Next
_ArrayDisplay($aArray, "Looking for '#32'", Default, 8)

; Search by column (looking in col 2)Recherche par colonne (en regardant dans la colonne 2)
Local $iIndex = _ArraySearch($aArray, "#32", 0, 0, 0, 0, 1, 2)
MsgBox($MB_SYSTEMMODAL, "Found '#32'", "Column 2 on Row " & $iIndex)

; Search by row (looking in row 3)Recherche par rangée (en regardant la ligne 3)
$iIndex = _ArraySearch($aArray, "#32", 0, 0, 0, 0, 1, 3, True)
MsgBox($MB_SYSTEMMODAL, "Found '#32'", "Row 3 in Col " & $iIndex)

Example 2Exemple 2

; using a 2D arrayTri d'un tableau 2D

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $avArray[6][2] = [ _
        ["String0", "SubString0"], _
        ["String1", "SubString1"], _
        ["String2", "SubString2"], _
        ["String3", "SubString3"], _
        ["String4", "SubString4"], _
        ["String5", "SubString5"]]

_ArrayDisplay($avArray, "$avArray")

Local $sSearch = InputBox("_ArraySearch() demo", "String to find?")
If @error Then Exit

Local $sColumn = InputBox("_ArraySearch() demo", "Column to search?")
If @error Then Exit
$sColumn = Int($sColumn)

Local $iIndex = _ArraySearch($avArray, $sSearch, 0, 0, 0, 1, 1, $sColumn)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Not Found", '"' & $sSearch & '" was not found on column ' & $sColumn & '.')
Else
    MsgBox($MB_SYSTEMMODAL, "Found", '"' & $sSearch & '" was found in the array at position ' & $iIndex & ' on column ' & $sColumn & '.')
EndIf
