_WinAPI_ProgIDFromCLSID

Retrieves the ProgID for a given CLSID

#include <WinAPICom.au3>
_WinAPI_ProgIDFromCLSID ( $sCLSID )

ParametersParamètres

$sCLSID The string that represents the CLSID for which ProgID is to be retrieved.

Return ValueValeur de retour

Success:Succès : The string that contains a ProgID.
Failure:Échec : Empty string and sets the @error flag to non-zero, @extended flag may contain the HRESULT error code.
