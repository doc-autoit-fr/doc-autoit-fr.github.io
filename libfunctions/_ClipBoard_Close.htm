_ClipBoard_Close

Closes the clipboardFerme le presse-papiers

#include <Clipboard.au3>
_ClipBoard_Close ( )

Return ValueValeur de retour

Success:Succès : True
Failure:Échec : FalseRetourne False

RemarksRemarques

When the window has finished examining or changing the clipboard close the clipboard by calling this function.Lorsque la fenêtre a fini d'examiner ou de modifier le presse-papiers, fermez le presse-papiers en appelant cette fonction.
This enables other windows to access the clipboard.Cela permet à d'autres fenêtres d'accéder au presse-papiers. Do not place an object on the clipboard after calling this function.Ne placez pas d'objet dans le presse-papiers après l'appel de cette fonction.

ExampleExemple

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <WinAPI.au3>
#include <WindowsConstants.au3>

Global $g_idMemo

Example()

Func Example()
    Local $hGUI

    ; Create GUICrée la GUI
    $hGUI = GUICreate("Clipboard", 600, 400)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Open the clipboardOuvre le presse-papiers
    If Not _ClipBoard_Open($hGUI) Then _WinAPI_ShowError("_ClipBoard_Open failed")

    ; Show clipboard formats availableAffiche les formats du presse-papiers disponibles
    MemoWrite("Clipboard formats ..: " & _ClipBoard_CountFormats())

    ; Empty the clipboardVide le presse-papiers
    If Not _ClipBoard_Empty() Then _WinAPI_ShowError("_ClipBoard_Empty failed")

    ; Show clipboard formats availableAffiche les formats du presse-papiers disponibles
    MemoWrite("Clipboard formats ..: " & _ClipBoard_CountFormats())

    ; Close the clipboardFerme le presse-papiers
    _ClipBoard_Close()

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Write message to memoEcrire un message dans memo
Func MemoWrite($sMessage = "")
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
