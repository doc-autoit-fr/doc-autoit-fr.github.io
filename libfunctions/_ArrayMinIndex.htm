_ArrayMinIndex

Returns the index where the lowest value occurs in a 1D or 2D arrayRetourne l'index de la valeur la plus petite d'un tableau 1D ou 2D

#include <Array.au3>
_ArrayMinIndex ( Const ByRef $aArray [, $iCompNumeric = 0 [, $iStart = -1 [, $iEnd = -1 [, $iSubItem = 0]]]] )

ParametersParamètres

$aArray Array to searchTableau à traiter
$iCompNumeric [optional][optionnel][optionnel] Comparison method:Méthode de comparaison :
    0 - compare alphanumerically
    1 - compare numerically
$iStart [optional][optionnel][optionnel] Index of array to start searchIndex du tableau où commence la recherche
$iEnd [optional][optionnel][optionnel] Index of array to end searchIndex de tableau où finir la recherche
$iSubItem [optional][optionnel][optionnel] Column of array to searchColonne du tableau où effectuer la recherche

Return ValueValeur de retour

Success:Succès : the index of the minimum value in the array.Retourne l'index de la valeur minimum du tableau.
Failure:Échec : -1 and sets the @error flag to non-zero.-1 et met @error <> 0.
@error:@error : 1 - $aArray is not an array or is empty
2 - $aArray is not a 1D or 2D array2 - $aArray n'est pas un tableau 1D ou 2D
3 - $iStart or $iEnd outside array bounds3 - $iStart ou $iEnd est en dehors des limites du tableau
4 - $iStart is greater than $iEnd4 - $iStart est supérieur à $iEnd
5 - $aArray is empty5 - $aArray est vide
6 - $iSubItem outside array bounds6 - iSubItem en dehors des limites du tableau

ExampleExemple

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray = StringSplit("4,2,06,8,12,5", ",")

MsgBox($MB_SYSTEMMODAL, 'Min Index String value', _ArrayMinIndex($aArray, 0, 1))
MsgBox($MB_SYSTEMMODAL, 'Min Index Numeric value', _ArrayMinIndex($aArray, 1, 1))

Local $aArray[4][4]
For $i = 0 To 3
    For $j = 0 To 3
        $aArray[$i][$j] = Random(0, 99, 1)
    Next
Next
_ArrayDisplay($aArray, "2D Array")
MsgBox($MB_SYSTEMMODAL, 'Min Index Numeric value in column 2', _ArrayMinIndex($aArray, 1, 0, 0, 2))
