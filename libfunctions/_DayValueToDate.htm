_DayValueToDate

Add the given days since noon 4713 BC January 1 and returns the Gregorian dateAjoute les jours donnés depuis midi le 1ᵉʳ janvier 4713 av. J.-C. et retourne la date grégorienne

#include <Date.au3>
_DayValueToDate ( $iJulianDate, ByRef $iYear, ByRef $iMonth, ByRef $iDay )

ParametersParamètres

$iJulianDate A valid number of days.Un nombre valide de jours.
$iYear will return the year in format YYYYretournera l'année au format AAAA
$iMonth will return the month in format MMretournera le mois au format MM
$iDay will return the day in format DDretournera la date au format JJ

Return ValueValeur de retour

Success:Succès : the date calculated.la date calculée.
Failure:Échec : 0 and sets the @error flag to non-zero.0 et @error <> 0.
@error:@error : 1 - Invalid Input number of days.1 - Nombre de jours saisi non valide.

RelatedLiens

ExampleExemple

#include <Date.au3>
#include <MsgBoxConstants.au3>

; Julian date of today.Date julienne d'aujourd'hui.
Local $sJulDate = _DateToDayValue(@YEAR, @MON, @MDAY)
MsgBox($MB_SYSTEMMODAL, "", "Todays Julian date is:La date julienne d'aujourd'hui est : " & $sJulDate)

; 14 days ago calculation.Calcule des 14 jours avant.
Local $Y, $M, $D
$sJulDate = _DayValueToDate($sJulDate - 14, $Y, $M, $D)
MsgBox($MB_SYSTEMMODAL, "", "14 days ago:14 jours avant :" & $M & "/" & $D & "/" & $Y & "  (" & $sJulDate & ")")
