_ClipBoard_SetViewer

Adds the specified window to the chain of clipboard viewersAjoute la fenêtre spécifiée à la chaîne des vues du presse-papiers

#include <Clipboard.au3>
_ClipBoard_SetViewer ( $hViewer )

ParametersParamètres

$hViewer Handle to the window to be added to the clipboard chainHandle de la fenêtre à ajouter à la chaîne du presse-papiers

Return ValueValeur de retour

Success:Succès : The handle to the next window in the clipboard viewer chainRetourne le handle de la fenêtre suivante dans la chaîne des vues du presse-papiers
Failure:Échec : 0 if there is no clipboard viewerRetourne 0 si il n'y a pas de vue dans le presse-papier

RemarksRemarques

The windows that are part of the clipboard viewer chain must process the clipboard messages $WM_CHANGECBCHAIN and $WM_DRAWCLIPBOARD.Les fenêtres qui font partie de la chaîne des vues du presse-papiers doivent traiter les messages $WM_CHANGECBCHAIN ​et $WM_DRAWCLIPBOARD.
Each clipboard viewer window calls the _SendMessage() function to pass these messages to the next window in the clipboard viewer chain.Chaque fenêtre des vues du presse-papiers appelle la fonction _SendMessage() pour passer ces messages à la fenêtre suivante dans la chaîne des vues.
A clipboard viewer window must eventually remove itself from the clipboard viewer chain by calling the _ClipBoard_ChangeChain() function.Une fenêtre de vue du presse-papiers doit finalement se retirer de la chaîne en appelant la fonction _ClipBoard_ChangeChain().

ExampleExemple

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <WindowsConstants.au3>

Global $g_idMemo, $g_hNext = 0

Example()

Func Example()
    Local $hGUI

    ; Create GUICrée la GUI
    $hGUI = GUICreate("Clipboard", 600, 400)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Initialize clipboard viewerInitialise le visualiseur du presse-papiers
    $g_hNext = _ClipBoard_SetViewer($hGUI)

    GUIRegisterMsg($WM_CHANGECBCHAIN, "WM_CHANGECBCHAIN")
    GUIRegisterMsg($WM_DRAWCLIPBOARD, "WM_DRAWCLIPBOARD")

    MemoWrite("GUI handle ....: " & $hGUI)
    MemoWrite("Viewer handle .: " & _ClipBoard_GetViewer())

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Shut down clipboard viewerArrête le visualiseur du presse-papiers
    _ClipBoard_ChangeChain($hGUI, $g_hNext)
EndFunc   ;==>Example

; Write message to memoEcrire un message dans memo
Func MemoWrite($sMessage = "")
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite

; Handle $WM_CHANGECBCHAIN messagesHandle des messages $WM_CHANGECBCHAIN
Func WM_CHANGECBCHAIN($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg
    ; Show that message was receivedMontre que le message a été reçu
    MemoWrite("***** $WM_CHANGECBCHAIN *****")

    ; If the next window is closing, repair the chainSi la fenêtre suivante se ferme, répare la chaîne
    If $wParam = $g_hNext Then
        $g_hNext = $lParam
        ; Otherwise pass the message to the next viewerSinon passe le message au visualiseur suivant
    ElseIf $g_hNext <> 0 Then
        _SendMessage($g_hNext, $WM_CHANGECBCHAIN, $wParam, $lParam, 0, "hwnd", "hwnd")
    EndIf
EndFunc   ;==>WM_CHANGECBCHAIN

; Handle $WM_DRAWCLIPBOARD messagesHandle des messages $WM_DRAWCLIPBOARD
Func WM_DRAWCLIPBOARD($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg
    ; Display any text on clipboardAffiche un texte dans le presse-papiers
    MemoWrite(_ClipBoard_GetData())

    ; Pass the message to the next viewerPasse le message au visualiseur suivant
    If $g_hNext <> 0 Then _SendMessage($g_hNext, $WM_DRAWCLIPBOARD, $wParam, $lParam)
EndFunc   ;==>WM_DRAWCLIPBOARD
