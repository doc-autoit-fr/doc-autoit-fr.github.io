_GDIPlus_EffectCreateLevels

Creates a Levels class effect objectCrée un objet Effect qui agit sur la brillance

#include <GDIPlus.au3>
_GDIPlus_EffectCreateLevels ( [$iHighlight = 100 [, $iMidtone = 0 [, $iShadow = 0]]] )

ParametersParamètres

$iHighlight [optional][optionnel][optionnel] Integer in the range 0 through 100 that specifies which pixels should be lightened.Entier compris entre 0 et 100 qui spécifie les pixels qui doivent être mis en surbrillance.
$iMidtone [optional][optionnel][optionnel] Integer in the range -100 through 100 that specifies how much to lighten or darken an image.Entier compris entre -100 et 100 qui spécifie de combien il faut éclaircir ou assombrir une image.
$iShadow [optional][optionnel][optionnel] Integer in the range 0 through 100 that specifies which pixels should be darkened.Entier compris entre 0 et 100 qui spécifie les pixels qui doivent être obscurcis.

Return ValueValeur de retour

Success:Succès : a handle to an Effect object.Retpurne le handle de l'objet Effect.
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).
@error:@error : -1 - GDIPlus.dll does not support this function.-1 - GDIPlus.dll ne supporte pas cette fonction.
10 - Invalid parameters.10 - Paramètres invalides.

RemarksRemarques

When you are done with the Effect object, call _GDIPlus_EffectDispose() to release the resources.Lorsque vous enavez terminé avec l'objet Effect, appelez _GDIPlus_EffectDispose() pour libérer les ressources.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

_Example()

Func _Example()
    If Not _GDIPlus_Startup() Or @extended < 6 Then
        MsgBox($MB_SYSTEMMODAL, "ERROR", "GDIPlus.dll v1.1 not available")
        Return
    EndIf

    Local $sFile = FileOpenDialog("Select an image", "", "Images (*.bmp;*.png;*.jpg;*.gif;*.tif)")
    If @error Or Not FileExists($sFile) Then Return

    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile)

    Local $iWidth = 600
    Local $iHeight = _GDIPlus_ImageGetHeight($hImage) * 600 / _GDIPlus_ImageGetWidth($hImage)

    Local $hGui = GUICreate("GDI+ v1.1 (" & @ScriptName & ")", $iWidth, $iHeight)
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)
    GUISetState(@SW_SHOW)

    Local $hEffect = _GDIPlus_EffectCreateLevels(60, -50, 20)
    _GDIPlus_BitmapApplyEffect($hImage, $hEffect)

    _GDIPlus_GraphicsDrawImageRect($hGraphics, $hImage, 0, 0, $iWidth, $iHeight)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    _GDIPlus_EffectDispose($hEffect)
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>_Example
