_WinAPI_BeginDeferWindowPos

Allocates memory for a multiple-window-position structure

#include <WinAPISys.au3>
_WinAPI_BeginDeferWindowPos ( [$iAmount = 1] )

ParametersParamètres

$iAmount [optional][optionnel] The initial number of windows for which to store position information. Default is 1.Par défaut c'est 1.

Return ValueValeur de retour

Success:Succès : The value identifies the multiple-window-position structure.
Failure:Échec : 0, call _WinAPI_GetLastError() to get extended error information.
