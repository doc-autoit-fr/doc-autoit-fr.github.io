_GDIPlus_PathSetMarker

Designates the last point in a path as a marker pointDésigne le dernier point d'un tracé (GraphicsPath) comme point marqueur

#include <GDIPlus.au3>
_GDIPlus_PathSetMarker ( $hPath )

ParametersParamètre

$hPath Pointer to a GraphicsPath objectHandle de l'objet GraphicsPath

Return ValueRemarque

Success: True.Vrai.
Failure: False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

Remarks<i0>

A path can have markers that divide the path into sections.Un tracé peut avoir des marqueurs qui divisent le tracé en sections. You can use a GraphicsPathIterator object to isolate one or more of those sections.Vous pouvez utiliser un objet GraphicsPathIterator afin d'isoler une ou plusieurs de ces sections.
