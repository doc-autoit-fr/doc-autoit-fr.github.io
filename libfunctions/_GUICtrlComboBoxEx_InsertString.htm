_GUICtrlComboBoxEx_InsertString

Inserts a new item in the control

#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_InsertString ( $hWnd, $sText [, $iIndex = -1 [, $iImage = -1 [, $iSelectedImage = -1 [, $iOverlayImage = -1 [, $iIndent = -1 [, $iParam = -1]]]]]] )

ParametersParamètre

$hWnd Handle to the control
$sText Item text. If set to -1, the item set is set via the $CBEN_GETDISPINFO notification message.
$iIndex [optional][optional] False - i, v, x ... 0-based index at which the new string should be inserted.
To insert an item at the end of the list, set the $iIndex member to -1
$iImage [optional][optional] False - i, v, x ... 0-based index of the item's icon in the control's image list
$iSelectedImage [optional][optional] False - i, v, x ... 0-based index of the item's icon in the control's image list
$iOverlayImage [optional][optional] False - i, v, x ... 0-based index of the item's icon in the control's image list
$iIndent [optional][optional] False - i, v, x ... Number of image widths to indent the item. A single indentation equals the width of an image.
$iParam [optional][optional] False - i, v, x ... Value specific to the item

Return ValueRemarque

Success: the index of the new item.
Failure: -1.Retourne -1.

#include <GuiComboBoxEx.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hGUI, $hCombo

    ; Create GUICrée la GUI
    $hGUI = GUICreate("ComboBoxEx Insert String", 400, 300)
    $hCombo = _GUICtrlComboBoxEx_Create($hGUI, "", 2, 2, 394, 100)
    GUISetState(@SW_SHOW)

    ; Add files
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    _GUICtrlComboBoxEx_AddDir($hCombo, "", $DDL_DRIVES, False)
    _GUICtrlComboBoxEx_AddDir($hCombo, "", $DDL_DRIVES)
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    _GUICtrlComboBoxEx_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBoxEx_EndUpdate($hCombo)
    _GUICtrlComboBoxEx_EndUpdate($hCombo)

    ; Insert string
    _GUICtrlComboBoxEx_InsertString($hCombo, "Text inserted", 1)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
