_WinAPI_InvalidateRgn

Adds a region to the specified window's update region

#include <WinAPIGdi.au3>
_WinAPI_InvalidateRgn ( $hWnd [, $hRgn = 0 [, $bErase = True]] )

ParametersParamètre

$hWnd Handle to the window with an update region that is to be modified.
$hRgn [optional][optional] False - i, v, x ... Handle to the region to be added to the update region. The region is assumed to have client coordinates.
If this parameter is 0, the entire client area is added to the update region.
$bErase [optional][optional] False - i, v, x ... Specifies whether the background within the update region is to be erased when the update region is processed, valid values:
    True - The background is erased (Default).
    False - The background remains unchanged.

Return ValueRemarque

Returns always True
