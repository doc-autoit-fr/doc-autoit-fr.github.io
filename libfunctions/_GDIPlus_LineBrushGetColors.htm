_GDIPlus_LineBrushGetColors

Gets the starting color and ending color of a linear gradient brushObtient la couleur de début et la couleur de fin d'un pinceau à dégradé linéaire

#include <GDIPlus.au3>
_GDIPlus_LineBrushGetColors ( $hLineGradientBrush )

ParametersParamètre

$hLineGradientBrush Pointer to a LinearGradientBrush objectHandle de l'objet LinearGradientBrush

Return ValueRemarque

Success: an array containing the starting and ending colors of the LinearGradientBrush:Retourne un tableau contenant les couleurs de début et de fin de l'objet LinearGradientBrush:
        [0] - Starting color
        [1] - Ending color
Failure: sets the @error flag to non-zero, @extended may contain GPSTATUS error code error code ($GPID_ERR*).Retourne @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;initialize GDI+Initialize GDI+
    Local Const $fWidth = 600, $fHeight = 600, $iBgColor = 0x303030 ;$fBGColor format RRGGBB$fBGColor est au format RRGGBB

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $fWidth, $fHeight) ;create a test GUIcréer une interface de test
    GUISetBkColor($iBgColor, $hGUI) ;set GUI background colorDéfinit la couleur de fond de la GUI
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    _GDIPlus_GraphicsSetSmoothingMode($hGraphics, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;sets the graphics object rendering quality (antialiasing)Définit l'objet graphique en qualité de rendu antialiasing
    Local $hBrush = _GDIPlus_LineBrushCreate(0, 0, 150, 0, 0xFF000000 + Random(0x111111, 0xFFFFFF, 1), 0xFF000000 + Random(0x111111, 0xFFFFFF, 1), 1) ;create linear gradient flipped brushCrée un pinceau à dégradé linéaire renversé

    _GDIPlus_GraphicsClear($hGraphics, 0xFF000000 + $iBgColor) ;clear bitmapEfface le bitmap
    _GDIPlus_GraphicsFillRect($hGraphics, 100, 50, 380, 500, $hBrush) ;draw the egg againDessine de nouveau l'œuf

    Local $aColors = _GDIPlus_LineBrushGetColors($hBrush)
    MsgBox($MB_SYSTEMMODAL, "", "Starting hex color: " & Hex($aColors[0], 8) & @CRLF & "End hex color: " & Hex($aColors[1], 8))

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;cleanup GDI+ resourcesNettoie les ressources de GDI+
    _GDIPlus_BrushDispose($hBrush)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
