Référence Fonction

_Assert

Display a message if assertion failsAffiche un message si une assertion échoue

#include <Debug.au3>
_Assert ( $sCondition [, $bExit = True [, $iCode = 0x7FFFFFFF [, $sLine = @ScriptLineNumber]]] )

ParametersParamètres

$sCondition The condition (expression quoted) that must evaluate to true.La condition (expression entre quote) qui doit être évaluée à vraie.
$bExit [optional] If true, the script is aborted.Si vrai, le script est interrompu lorsque la condition est vraie.
$iCode [optional] The exit code to use if the script is aborted.Le code de sortie à utiliser si le script est interrompu.
$sLine [optional] Displays the line number where the assertion failed.Affiche le numéro de ligne où l'assertion a échoué. If this value is not changed, then the default value will show the correct line.Si cette valeur n'est pas modifiée, la valeur par défaut affichera la ligne correcte.

Return ValueValeur de retour

Returns the result of the condition (Only valid when not exiting).Retourne le résultat de la condition (Valide uniquement lorsque le script n'est pas sorti).

RemarksRemarques

@error and @extended are not destroyed on return.@error et @extended ne sont pas détruits lors du retour.

ExampleExemple

#include <Debug.au3>

Local $sABC = ""
_Assert('$sABC > ""')
