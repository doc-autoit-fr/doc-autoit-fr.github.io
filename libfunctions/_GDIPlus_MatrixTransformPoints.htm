_GDIPlus_MatrixTransformPoints

Multiplies each point in an array by a matrixMultiplie chaque point d'un tableau par une matrice

#include <GDIPlus.au3>
_GDIPlus_MatrixTransformPoints ( $hMatrix, ByRef $aPoints )

ParametersParamètres

$hMatrix Pointer to a Matrix objectHandle de l'objet Matrix
$aPoints Array of points to be transformed:Tableau de points à transformer:
    [0][0] - Number of points
    [1][0] - Point 1 X coordinate
    [1][1] - Point 1 Y coordinate
    [2][0] - Point 2 X coordinate
    [2][1] - Point 2 Y coordinate
    [n][0] - Point n X coordinate
    [n][1] - Point n Y coordinate

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne False et @error <> 0, @extended contient le code erreur GPSTATUS($GPID_ERR*).

RemarksRemarques

Each point in the array is treated as a row matrix.Chaque point du tableau est considéré comme une matrice ligne.
The multiplication is performed with the row matrix on the left and the matrix on the right.La multiplication est effectuée avec la matrice ligne à gauche et de la matrice carrée à droite.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Create GUICrée la GUI

    Local $aGui[6]
    Local $aPoints[6][2] = [[5]]
    For $i = 1 To 5
        $aPoints[$i][0] = Cos(6.28 * $i / 5) * @DesktopHeight / 3 + @DesktopWidth / 2
        $aPoints[$i][1] = Sin(6.28 * $i / 5) * @DesktopHeight / 3 + @DesktopHeight / 2
        $aGui[$i] = GUICreate($i, 120, 100, $aPoints[$i][0] - 60, $aPoints[$i][1] - 50)
        GUISetBkColor(0x55)
        GUISetState(@SW_SHOW)
    Next

    _GDIPlus_Startup()
    Local $hMatrix = _GDIPlus_MatrixCreate()
    _GDIPlus_MatrixTranslate($hMatrix, -@DesktopWidth / 2, -@DesktopHeight / 2)
    _GDIPlus_MatrixRotate($hMatrix, 1, True)
    _GDIPlus_MatrixTranslate($hMatrix, @DesktopWidth / 2, @DesktopHeight / 2, True)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Local $iTimer = TimerInit()
    Do
        If TimerDiff($iTimer) > 40 Then
            _GDIPlus_MatrixTransformPoints($hMatrix, $aPoints)
            For $i = 1 To 5
                WinMove($aGui[$i], "", $aPoints[$i][0] - 60, $aPoints[$i][1] - 50)
            Next
            $iTimer = TimerInit()
        EndIf
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _GDIPlus_MatrixDispose($hMatrix)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
