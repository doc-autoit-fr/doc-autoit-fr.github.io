_ArrayTrim

Trims a certain number of characters from all elements in a 1D or 2D arrayCoupe un nombre donné de caractères dans tous les éléments d'un tableau 1D ou 2D

#include <Array.au3>
_ArrayTrim ( ByRef $aArray, $iTrimNum [, $iDirection = 0 [, $iStart = 0 [, $iEnd = 0 [, $iSubItem = 0]]]] )

ParametersParamètres

$aArray Array to modifyTableau à transposer
$iTrimNum Number of characters to removeNombre de caractères à supprimer
$iDirection [optional][optionnel][optionnel] Direction to trim:Emplacement de la coupe:
    0 - trim left
    1 - trim right
$iStart [optional][optionnel][optionnel] Index of array to start trimIndex du tableau où commencer la coupe
$iEnd [optional][optionnel][optionnel] Index of array to stop trimIndex du tableau où arrêter la coupe
$iSubItem [optional][optionnel][optionnel] Column to trimColonne dans laquelle effectuer la coupe

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 and sets the @error flag to non-zero.0 et @error <> 0.
@error:@error : 1 - $aArray is not an array1 - $aArray n'est pas un tableau
2 - $aArray is not an 1D or 2D array2 - $aArray n'est pas un tableau 1D ou 2D
3 - $iStart or $iEnd outside array bounds3 - $iStart ou $iEnd est en dehors des limites du tableau
4 - $iStart is greater than $iEnd4 - $iStart est supérieur à $iEnd
5 - $iSubItem outside array bounds5 - $iSubItem est en dehors des limites du tableau

ExampleExemple

#include <Array.au3>

Local $aArray[5] = ["ab", "bc", "cd", "de", "ef"]

_ArrayDisplay($aArray, "1D arrayTableau 1D")
_ArrayTrim($aArray, 1, 1, 1, 3)
_ArrayDisplay($aArray, "Trimmed right 1 char from items 1-3")

Local $aArray[4][4], $iChr
For $i = 0 To 3
    For $j = 0 To 3
        $iChr = Random(97, 121, 1)
        $aArray[$i][$j] = Chr($iChr) & Chr($iChr + 1)
    Next
Next
_ArrayDisplay($aArray, "2D array")
_ArrayTrim($aArray, 1, 0, 1, 2, 3)
_ArrayDisplay($aArray, "Trimmed left 1 char from col 2-3 in row 1-2")
