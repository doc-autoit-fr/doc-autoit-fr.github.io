_WinAPI_LoadKeyboardLayout

Loads a new input locale identifier into the system

#include <WinAPISys.au3>
_WinAPI_LoadKeyboardLayout ( $iLanguage [, $iFlag = 0] )

ParametersParamètre

$iLanguage The input locale identifier to load.
$iFlag [optional][optional] False - i, v, x ... The flag that specifies how the input locale identifier is to be loaded. This parameter can be
one of the following values.
$KLF_ACTIVATE
$KLF_NOTELLSHELL
$KLF_REORDER
$KLF_REPLACELANG
$KLF_SUBSTITUTE_OK
$KLF_SETFORPROCESS

Return ValueRemarque

Success: The input locale identifier to the locale matched with the requested name.
Failure: 0, call _WinAPI_GetLastError() to get extended error information.
