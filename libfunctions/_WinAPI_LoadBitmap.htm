_WinAPI_LoadBitmap

Loads the specified bitmap resource from a module's executable file

#include <WinAPI.au3>
_WinAPI_LoadBitmap ( $hInstance, $sBitmap )

ParametersParamètres

$hInstance Handle to the instance of the module whose executable file contains the bitmap to be loaded
$sBitmap The name of the bitmap resource to be loaded. Alternatively this can consist of the resource
identifier in the low order word and 0 in the high order word.

Return ValueValeur de retour

Success:Succès : The handle to the specified bitmap
Failure:Échec : 0Retourne 0
