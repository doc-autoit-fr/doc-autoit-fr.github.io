_GDIPlus_ImageResize

Resize an image to a new given sizeRedimensionne une image à une taille donnée

#include <GDIPlus.au3>
_GDIPlus_ImageResize ( $hImage, $iNewWidth, $iNewHeight [, $iInterpolationMode = $GDIP_INTERPOLATIONMODE_HIGHQUALITYBICUBIC] )

ParametersParamètres

$hImage A handle valueHandle d'une matrice
$iNewWidth An integer valueLa nouvelle hauteur (nombre entier)
$iNewHeight An integer valueLa nouvelle hauteur (nombre entier)
$iInterpolationMode [optional][optionnel][optionnel] An integer value.Une valeur entière: mode d'interpolation. Default is $GDIP_INTERPOLATIONMODE_HIGHQUALITYBICUBIC.Par défaut, $GDIP_INTERPOLATIONMODE_HIGHQUALITYBICUBIC.

Return ValueValeur de retour

Success:Succès : a handle to a new Bitmap objectRetourne le handle du nouvel objet Bitmap
Failure:Échec : 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0.
@error:@error : 1 - unable to create new bitmap1 - Impossible de créer un nouveau bitmap
2 - unable to create graphics context2 - Impossible de créer le contexte graphique
3 - unable to set interpolation mode on graphics context3 - Impossible de régler le mode d'interpolation sur le contexte graphique
4 - unable to copy image to scaled bitmap4 - Impossible de copier l'image dans le bitmap redimensionné

ExampleExemple

#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    _GDIPlus_Startup()
    Local Const $iW = @DesktopWidth, $iH = @DesktopHeight

    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, $iW, $iH) ;create a GDI bitmap by capturing full screen of the desktopCrée un bitmap GDI en capturant le bureau en entier
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;convert GDI bitmap to GDI+ bitmapConverti le bitmap GDI en GDI+
    _WinAPI_DeleteObject($hHBmp) ;release GDI bitmap resource because not needed anymoreLibére la ressource du bitmap GDI car n'est plus nécessaire

    Local $hBitmap_Scaled = _GDIPlus_ImageResize($hBitmap, $iW / 4, $iH / 4) ;resize imageRedimensionne l'image

    Local $hGUI = GUICreate("GDI+ test", $iW / 4, $iH / 4, -1, -1) ;create a test gui to display the resized imageCrée une GUI de test pour afficher l'image redimensionnée
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;create a graphics object from a window handleCrée un objet graphique à partir du handle de fenêtre
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap_Scaled, 0, 0) ;display scaled imageAffiche l'image à l'échelle

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    ;cleanup resourcesNettoie les ressources
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_BitmapDispose($hBitmap_Scaled)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
