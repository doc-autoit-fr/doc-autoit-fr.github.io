$tagNMLVSCROLL

Provides information about a scrolling operationFournit des informations sur une opération de défilement

Global Const $tagNMLVSCROLL = $tagNMHDR & ";int DX;int DY"

ParametersParamètres

$tagNMHDR Contains information about a notification messageContient des informations sur un message de notification
DX Specifies in pixels the horizontal position where a scrolling operation should begin or end
DY Specifies in pixels the vertical position where a scrolling operation should begin or end
