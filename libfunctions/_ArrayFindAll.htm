_ArrayFindAll

Find the indices of all ocurrences of a search query between two points in a 1D or 2D array using _ArraySearch()Trouve les index de toutes les occurrences d'une requête de recherche entre deux points d'un tableau 1D ou 2D en utilisant _ArraySearch()

#include <Array.au3>
_ArrayFindAll ( Const ByRef $aArray, $vValue [, $iStart = 0 [, $iEnd = 0 [, $iCase = 0 [, $iCompare = 0 [, $iSubItem = 0 [, $bRow = False]]]]]] )

ParametersParamètres

$aArray The array to searchLe tableau où chercher
$vValue What to search $aArray forCe qu'il faut rechercher dans $aArray
$iStart [optional][optionnel] Index of array to start searchIndex du tableau où commence la recherche
$iEnd [optional][optionnel] Index of array to end searchIndex de tableau où finir la recherche
$iCase [optional][optionnel] If set to 1, search is case sensitiveSi la valeur est 1, la recherche est sensible à la casse
$iCompare [optional][optionnel]
    0 Casting of variables to the same type (default), "string" = 0, "" = 0 or "0" = 0 match (If $iCase = 0)
    1 executes a partial search
    2 comparison match if variables have same type and same value
    3 compares using a regular expression pattern provided as $vValue
$iSubItem [optional][optionnel] Sub-index to search on in 2D arraysSous-index de recherche dans un tableau 2D
$bRow [optional][optionnel] If True then $iSubItem sets the row to search - False (default) searches columnsSi True, $iSubItem définit la ligne où chercher.Par défaut, False, définit la colonne où chercher

Return ValueValeur de retour

Success:Succès : an array of all index numbers in array containing $vValue.Retourne le tableau de tous les index du tableau initial contenant $vValeur.
Failure:Échec : sets the @error flag to non-zero (see _ArraySearch() description for @error).Retourne @error <> 0 (consultez _ArraySearch() pour une description de @error ).

RemarksRemarques

The values of $iCompare can not be combined together.

RelatedLiens

_ArrayBinarySearch, _ArraySearch

ExampleExemple

#include <Array.au3>

Local $aArray[5] = [0, 1, 2, 1, 0]
_ArrayDisplay($aArray, "1D arrayTableau 1D")
Local $aiResult = _ArrayFindAll($aArray, 0)
_ArrayDisplay($aiResult, "Found")

Local $aArray[5][5] = [[0, 1, 2, 1, 0], _
        [4, 5, 5, 4, 2], _
        [4, 1, 3, 1, 3], _
        [0, 3, 2, 1, 0], _
        [1, 5, 5, 4, 1]]
_ArrayDisplay($aArray, "2D array")

Local $aResult = _ArrayFindAll($aArray, 0, Default, Default, Default, Default, 4)
_ArrayDisplay($aResult, "Found in Column 4")

$aResult = _ArrayFindAll($aArray, 1, Default, Default, Default, Default, 2, True)
_ArrayDisplay($aResult, "Found in Row 2")
