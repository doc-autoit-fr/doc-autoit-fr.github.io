_SendMessageA

Send a Message to a Window/Control (Force Ansi Call)

#include <SendMessage.au3>
_SendMessageA ( $hWnd, $iMsg [, $wParam = 0 [, $lParam = 0 [, $iReturn = 0 [, $wParamType = "wparam" [, $lParamType = "lparam" [, $sReturnType = "lresult"]]]]]] )

ParametersParamètres

$hWnd Window/control handle
$iMsg Message to send to control (number)
$wParam [optional][optionnel] Specifies additional message-specific information
$lParam [optional][optionnel] Specifies additional message-specific information
$iReturn [optional][optionnel] What to return:
    0 - Return value from DLL call
    1 - $ihWnd
    2 - $iMsg
    3 - $wParam
    4 - $lParam
    <0 or > 4 - array same as DllCall
$wParamType [optional][optionnel] See DllCall in Related
$lParamType [optional][optionnel] See DllCall in Related
$sReturnType [optional][optionnel] See DllCall in Related

Return ValueValeur de retour

Success:Succès : the user selected value from the DllCall() result.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

When a message is blocked by UIPI the last error, retrieved with _WinAPI_GetLastError(), is set to 5 (access denied).
