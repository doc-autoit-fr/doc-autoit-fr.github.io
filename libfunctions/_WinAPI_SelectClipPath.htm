_WinAPI_SelectClipPath

Selects the current path as a clipping region, combining the new region with any existing clipping region

#include <WinAPIGdi.au3>
_WinAPI_SelectClipPath ( $hDC [, $iMode = 5] )

ParametersParamètres

$hDC Handle to the device context of the path.
$iMode [optional][optionnel] The way to use the path. This parameter can be one of the following values.
$RGN_AND
$RGN_COPY (Default)
$RGN_DIFF
$RGN_OR
$RGN_XOR

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False, call _WinAPI_GetLastError() to get extended error information.
