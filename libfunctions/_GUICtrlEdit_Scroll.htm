_GUICtrlEdit_Scroll

Scrolls the text vertically

#include <GuiEdit.au3>
_GUICtrlEdit_Scroll ( $hWnd, $iDirection )

ParametersParamètres

$hWnd Control ID/Handle to the control
$iDirection This parameter can be one of the following values:
    $SB_LINEDOWN - Scrolls down one line
    $SB_LINEUP - Scrolls up one line
    $SB_PAGEDOWN - Scrolls down one page
    $SB_PAGEUP - Scrolls up one page
    $SB_SCROLLCARET - Scrolls the caret into view

Return ValueValeur de retour

Success:Succès : The high-order word of the return value is 1.
The low-order word is the number of lines that the command scrolls.
Failure:Échec : 0.Retourne 0.

#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $idEdit, $sText, $iLen

    ; Create GUICrée la GUI
    GUICreate("Edit Scroll", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268, BitOR($ES_WANTRETURN, $WS_VSCROLL))
    GUISetState(@SW_SHOW)

    ; Set Text
    For $i = 1 To 50
        $sText &= $i & @CRLF
    Next
    _GUICtrlEdit_SetText($idEdit, $sText)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Line Down")
    _GUICtrlEdit_Scroll($idEdit, $SB_LINEDOWN)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Line Up")
    _GUICtrlEdit_Scroll($idEdit, $SB_LINEUP)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Page Down")
    _GUICtrlEdit_Scroll($idEdit, $SB_PAGEDOWN)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Page Up")
    _GUICtrlEdit_Scroll($idEdit, $SB_PAGEUP)

    $iLen = _GUICtrlEdit_GetTextLen($idEdit)
    _GUICtrlEdit_SetSel($idEdit, $iLen, $iLen)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Caret")
    _GUICtrlEdit_Scroll($idEdit, $SB_SCROLLCARET)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
