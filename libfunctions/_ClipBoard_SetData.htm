_ClipBoard_SetData

Places data on the clipboard in a specified clipboard formatPlace des données dans le presse-papiers dans un format spécifié de presse-papiers

#include <Clipboard.au3>
_ClipBoard_SetData ( $vData [, $iFormat = 1] )

ParametersParamètres

$vData Data in Binary or String format, or optionally NULL (0) (owner must render, see below).Données au format Binary ou String, ou éventuellement NULL(0) (propriétaire doit rendre, voir ci-dessous ).
IMPORTANT:IMPORTANT : If a String is passed, and it is not of type $CF_TEXT, $CF_OEMTEXT, or $CF_UNICODETEXT, it will be treated as an ANSI string.Si le format est String, et que la chaîne n'est pas de type $CF_TEXT, $CF_OEMTEXT, ou $CF_UNICODETEXT, elle sera traitée comme une chaîne ANSI.
To force Unicode strings for other types, you must pass the data in Binary format.Pour forcer une chaîne Unicode en un autre type, vous devez transmettre les données au format binaire.
Also, do NOT pass $CF_UNICODETEXT in Binary format (causes garbled data).Aussi, ne PAS passer $CF_UNICODETEXT au format binaire (brouillage des données ).
When $vData is NULL, it indicates that the window provides data in the specified clipboard format upon request.Lorsque $vData est NULL, cela indique que la fenêtre fournit des données dans le format spécifié du presse-papiers, sur demande.
If a window delays rendering, it must process the $WM_RENDERFORMAT and $WM_RENDERALLFORMATS messages.Si une fenêtre retarde à s'afficher, elle doit traiter les messages $WM_RENDERFORMAT et $WM_RENDERALLFORMATS.
If this function succeeds, the system owns the object identified by the $hMemory parameter.Si cette fonction réussit, le système s'approprie l'objet identifié par le paramètre $hMEMORY.
The application may not write to or free the data once ownership has been transferred to the system, but it can lock and read from the data until the _ClipBoard_Close() function is called.L'application ne peut écrire ou libérer les données une fois que la propriété a été transférée au système, mais elle peut verrouiller et lire les données jusqu'à ce que la fonction _ClipBoard_Close() soit appelée.
The memory must be unlocked before the clipboard is closed.La mémoire doit être déverrouillée avant que le presse-papiers soit fermé.
If the $hMemory parameter identifies a memory object, the object must have been allocated using the function with the $GMEM_MOVEABLE flag.Si le paramètre $hMEMORY identifie un objet mémoire, l'objet doit avoir été alloué à l'aide de la fonction avec le flag $GMEM_MOVEABLE.

Return ValueValeur de retour

Success:Succès : The handle to the dataRetourne le handle des données
Failure:Échec : 0Retourne 0

RemarksRemarques

This function performs all of the steps neccesary to put data on the clipboard.Cette fonction effectue toutes les étapes nécessaires pour placer les données dans le presse-papiers. It will allocate the global memory object, open the clipboard, place the data on the clipboard and close the clipboard.Elle va allouer un objet de mémoire globale, ouvrir le presse-papiers, placer les données dans le presse-papiers et fermer le presse-papiers. If you need more control over putting data on the clipboard, you may want to use the _ClipBoard_SetDataEx() function.Si vous avez besoin de plus de contrôle sur la mise en place des données dans le presse-papiers, vous pouvez utiliser la fonction _ClipBoard_SetDataEx().

ExampleExemple

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Global $g_idMemo

Example()

Func Example()
    Local $id_SetData, $id_GetData

    ; Create GUICrée la GUI
    GUICreate("Clipboard", 600, 450)
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    $id_SetData = GUICtrlCreateButton("Set ClipBoard Data", 150, 410, 120, 30)
    $id_GetData = GUICtrlCreateButton("Get ClipBoard Data", 300, 410, 120, 30)
    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $id_SetData
                _ClipBoard_SetData("ClipBoard Library")
            Case $id_GetData
                MemoWrite(_ClipBoard_GetData())
        EndSwitch
    WEnd
EndFunc   ;==>Example

; Write message to memoEcrire un message dans memo
Func MemoWrite($sMessage = "")
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
