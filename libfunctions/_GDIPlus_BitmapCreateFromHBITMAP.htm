_GDIPlus_BitmapCreateFromHBITMAP

Create a Bitmap object from a bitmap handleCrée un objet Bitmap à partir du handle d'un Bitmap

#include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromHBITMAP ( $hBitmap [, $hPal = 0] )

ParametersParamètres

$hBitmap Handle to a HBITMAPHandle du HBITMAP
$hPal [optional][optionnel][optionnel] Handle to a HPALETTEHandle du HPALETTE

Return ValueValeur de retour

Success:Succès : a handle to a Bitmap object.Retourne le handle de l'objet Bitmap
Failure:Échec : 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

When you are done with the Bitmap object, call _GDIPlus_BitmapDispose() to release the resources.Lorsque vous en avez terminé avec l'objet bitmap, appelez _GDIPlus_BitmapDispose() pour libérer les ressources

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    Local $hGUI, $hBMP, $hBitmap, $hGraphic

    ; Capture upper left corner of screenCapturez le coin supérieur gauche de l'écran
    $hBMP = _ScreenCapture_Capture("", 0, 0, 400, 300)

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    ; Initialize GDI+ libraryInitialise GDI+ Librairie
    _GDIPlus_Startup()

    ; Draw bitmap to GUIDessine le bitmap dans la GUI
    $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hBMP)
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    _GDIPlus_GraphicsDrawImage($hGraphic, $hBitmap, 0, 0)

    ; Clean up resourcesLibère les ressources
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_BitmapDispose($hBitmap)
    _WinAPI_DeleteObject($hBMP)

    ; Shut down GDI+ libraryFerme la librairie GDI+
    _GDIPlus_Shutdown()

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
