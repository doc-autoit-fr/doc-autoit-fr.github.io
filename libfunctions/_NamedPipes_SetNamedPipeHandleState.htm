_NamedPipes_SetNamedPipeHandleState

Sets the read mode and the blocking mode of the specified named pipe

#include <NamedPipes.au3>
_NamedPipes_SetNamedPipeHandleState ( $hNamedPipe, $iRead, $iWait [, $iBytes = 0 [, $iTimeOut = 0]] )

ParametersParamètres

$hNamedPipe Handle to the named pipe instance
$iRead Pipe read mode. Must be one of the following:Doit être l'un des suivants:
    0 - Data is read from the pipe as a stream of bytes
    1 - Data is read from the pipe as a stream of messages
$iWait Pipe wait mode. Must be one of the following:Doit être l'un des suivants:
    0 - Blocking mode is enabled
    1 - Nonblocking mode is enabled
$iBytes [optional][optionnel] Maximum number of bytes collected on the client computer before transmission to the server
$iTimeout [optional][optionnel] Maximum time, in milliseconds, that can pass before a remote named pipe transfers information

Return ValueValeur de retour

Success:Succès : True.
Failure:Échec : False.Retourne False.
