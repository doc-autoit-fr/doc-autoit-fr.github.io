_GDIPlus_BitmapCreateFromHICON32

Creates a Bitmap object based on a 32bit iconCrée un objet Bitmap à partir d'une icône 32 bits

#include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromHICON32 ( $hIcon )

ParametersParamètres

$hIcon Handle to an iconHandle de l'icône

Return ValueValeur de retour

Success: a handle to a new Bitmap object.Retourne le handle d'un nouvel objet Bitmap.
Failure: 0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).Retourne 0 et définit @error <> 0, @extended contient le code erreur GPSTATUS ($GPID_ERR*).

RemarksRemarques

When you are done with the Bitmap object, call _GDIPlus_ImageDispose() to release the resources.Lorsque vous en avez terminé avec l'objet Bitmap, appelez _GDIPlus_ImageDispose() pour libérer les ressources.

RelatedLiens

_GDIPlus_ImageDispose, _WinAPI_LoadIcon, _WinAPI_LoadImage

See AlsoVoir aussi

Search GdipCreateBitmapFromHICON in MSDN Library.

ExampleExemple

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIShellEx.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic, $hIcon, $hBitmap

    ; Create GUICrée la GUI
    $hGUI = GUICreate("GDI+", 168, 168)
    GUISetState()

    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;Create a graphics object from a window handleCrée une GUI
    _GDIPlus_GraphicsClear($hGraphic, 0xFFFFFFFF)

    $hIcon = _WinAPI_ShellExtractIcon(@SystemDir & '\shell32.dll', 32, 48, 48)

    $hBitmap = _GDIPlus_BitmapCreateFromHICON32($hIcon)

    _GDIPlus_GraphicsDrawImage($hGraphic, $hBitmap, 60, 60)

    ; Loop until user exitsBoucle jusqu'à ce que l'utilisateur quitte
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Clean up resourcesLibère les ressources
    _WinAPI_DestroyIcon($hIcon)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
