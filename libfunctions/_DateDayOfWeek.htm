_DateDayOfWeek

Returns the name of the weekday, based on the specified dayRetourne le nom du jour de la semaine, basé sur le jour spécifié

#include <Date.au3>
_DateDayOfWeek ( $iDayNum [, $iFormat = 0] )

ParametersParamètres

$iDayNum Day numberNuméro du jour
    1 = Sunday    1 = Sunday / dimanche
    2 = Monday    2 = Monday / lundi
    3 = Tuesday    3 = Tuesday / mardi
    4 = Wednesday    4 = Wednesday / mercredi
    5 = Thursday    5 = Thursday / jeudi
    6 = Friday    6 = Friday / vendredi
    7 = Saturday    7 = Saturday / samedi
$iFormat [optional][optionnel][optionnel] Format:Format :
$DMW_LOCALE_USER (0) - Long name of the weekday (Default)$DMW_LOCALE_USER (0) - Nom long du jour de la semaine (par défaut)
$DMW_SHORTNAME (1) - Short name of the weekday e.g.$DMW_SHORTNAME (1) - Nom court du jour de la semaine, p. ex. Mon
$DMW_LOCALE_LONGNAME (2) - Long name of the weekday in the user's default language$DMW_LOCALE_LONGNAME (2) - Nom long du jour de la semaine dans la langue par défaut de l'utilisateur
$DMW_LOCALE_SHORTNAME (3) - Short name of the weekday in the user's default language$DMW_LOCALE_SHORTNAME (3) - Nom court du jour de la semaine dans la langue par défaut de l'utilisateur

Return ValueValeur de retour

Success:Succès : Weekday nameNom du jour de la semaine
Failure:Échec : An empty string and sets @error to non-zeroUne chaîne vide et met @error <> 0.
@error:@error : 1 - Invalid weekday number.1 - Numéro du jour non valide.

RemarksRemarques

The function returns weekday names in English or the user's default language.La fonction retourne les noms des jours de la semaine en anglais ou dans la langue par défaut de l'utilisateur.
