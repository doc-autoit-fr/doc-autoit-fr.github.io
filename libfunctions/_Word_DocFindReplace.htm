_Word_DocFindReplace

Runs the specified find and replace operation

#include <Word.au3>
_Word_DocFindReplace ( $oDoc [, $sFindText = "" [, $sReplaceWith = "" [, $iReplace = $WdReplaceAll [, $vSearchRange = 0 [, $bMatchCase = False [, $bMatchWholeWord = False [, $bMatchWildcards = False [, $bMatchSoundsLike = False [, $bMatchAllWordForms = False [, $bForward = True [, $iWrap = $WdFindContinue [, $bFormat = False]]]]]]]]]]]] )

ParametersParamètre

$oDoc Word document object
$sFindText [optional][optional] False - i, v, x ... The text to be searched for. Use an empty string ("") to search for formatting only.
You can search for special characters by specifying appropriate character codes.
For example, "^p" corresponds to a paragraph mark and "^t" corresponds to a tab character (default = "")
$sReplaceWith [optional][optional] False - i, v, x ... The replacement text. To delete found text use an empty string ("").
You specify special characters and advanced search criteria just as you do for the Find argument (default = "")
$iReplace [optional][optional] False - i, v, x ... How many replacements are to be made: one, all, or none.
Can be anyWdReplace constant (default = $WdReplaceAll)
$vSearchRange [optional][optional] False - i, v, x ... Specifies the selection or range to search. Can be:
    -1 - Specifies the current selection
    0 - Specifies the entire document (default)
Any Word range object.
$bMatchCase [optional][optional] False - i, v, x ... If True the find is case sensitive (default = False)
$bMatchWholeWord [optional][optional] False - i, v, x ... If True only find entire words (default = False)
$bMatchWildcards [optional][optional] False - i, v, x ... If True the find text has special search operators (default = False)
$bMatchSoundsLike [optional][optional] False - i, v, x ... If True finds words that sound similar to the find text (default = False)
$bMatchAllWordForms [optional][optional] False - i, v, x ... If True finds all forms of the find text
(e.g. "sit" locates "sitting" and "sat") (default = False)
$bForward [optional][optional] False - i, v, x ... True to search forward (toward the end of the document) (default = True)
$iWrap [optional][optional] False - i, v, x ... True wraps when the bottom or top of the document, selection or range is reached.
Can be one of the WdFindWrap constants (default = $WdFindContinue)
$bFormat [optional][optional] False - i, v, x ... True to have the find operation locate formatting in addition to or instead of the find text (default = False)

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0 :
@error: 1 - $oDoc is not an object
2 - $vSearchRange is not an object
3 - An error occurred when the replace operation was executed. @extended is set to the COM error code
