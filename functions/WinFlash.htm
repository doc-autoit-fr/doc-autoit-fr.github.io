WinFlash

Flashes a window in the taskbar.Fait clignoter une fenêtre dans la barre des tâches.

WinFlash ( "title" [, "text" [, flashes = 4 [, delay = 500]]] )

ParametersParamètres

title The title/hWnd/class of the window to change.Le titre/hWnd/class de la fenêtre à changer. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel] The text of the window to change.Le texte de la fenêtre à changer. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .
flashes [optional][optionnel] The amount of times to flash the window.Le nombre de clignotements de la fenêtre. Default 4.Par défaut 4.
delay [optional][optionnel] The time in milliseconds to sleep between each flash.Le temps d'attente en milliseconde entre deux clignotements. Default 500 ms.Par défaut 500 ms.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

This function is very useful for catching the user's eye!Cette fonction est très utile pour attirer l'attention de l'utilisateur.

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Flash the window 4 times with a break in between each one of 1/2 second.Fait clignoter la fenêtre quatre fois avec une pause entre chaque de 2.1 secondes.
    WinFlash($hWnd, "", 4, 500)
EndFunc   ;==>Example
