MsgBox

Displays a simple message box with optional timeout.Affiche une simple boîte de message, avec temporisation optionnelle.

MsgBox ( flag, "title", "text" [, timeout = 0 [, hwnd]] )

ParametersParamètres

flag The flag indicates the type of message box and the possible button combinations.Ce paramètre indique le type de boîte de message et les combinaisons de boutons possibles. See remarks.Voir remarques.
title The title of the message box.Le titre de la boîte de message.
text The text of the message box.Le texte de la boîte de message.
timeout [optional][optionnel][optionnel] Timeout in seconds.Temporisation en secondes. After the timeout has elapsed the message box will be automatically closed.Après la temporisation, la boite de message sera automatiquement fermée. The default is 0, which is no timeout.La valeur par défaut est 0, ce qui équivaut à aucune temporisation.
hwnd [optional][optionnel][optionnel] The window handle to use as the parent for this dialog.Le handle de la fenêtre à utiliser comme fenêtre parent pour ce dialogue.

Return ValueValeur de retour

Success:Succès : the ID of the button pressed.Retourne ID du bouton pressé.
Failure:Échec : $IDTIMEOUT (-1) if the message box timed out.Retourne $IDTIMEOUT (-1) si la temporisation de la boîte de dialogue est dépassée.

For example, to specify a SYSTEMMODAL box with the YES/NO buttons the flag value would be 4096+4 (or 4100) If using hexadecimal flags, that would be 0x1000+0x4 (or 0x1004).Par exemple, pour spécifier une boîte SYSTEMMODAL avec les boutons YES/NO, la valeur du 'flag' serait 4096+4 (ou 4100).En utilisant les valeurs hexadécimales des flags, ce serait 0x1000+0x4 (ou 0x1004).

A message box appears centered on screen and resizes according to the text it contains.Une boîte de dialogue apparaît centrée sur l'écran et est redimensionnée selon le texte qu'elle contient. However, the title could get truncated if the SYSTEMMODAL flag (4096) is used..Cependant, le titre pourrait être tronqué si l'option SYSTEMMODAL (4096) est utilisée.

If "title" is equal to Default keyword @Scriptname is usedSi "title" est égal au mot-clé Default, @Scriptname est utilisé.

Special note; MsgBox does not support the use of the timeout parameter when the $MB_SERVICE_NOTIFICATION flag is used when calling MsgBox from a service. The message box window is not owned by the process calling MsgBox when $MB_SERVICE_NOTIFICATION is specified.

Example 1Exemple 1

Example 2Exemple 2

#include <MsgBoxConstants.au3>

Local $iTimeout = 10

; Display a message box with a nested variable in its text.Affiche une boîte de message avec une variable incluse dans le texte.
MsgBox($MB_SYSTEMMODAL, "Title", "This message box will timeout after " & $iTimeout & " seconds or select the OK button.", $iTimeout)
