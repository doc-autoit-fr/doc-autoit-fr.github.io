Send

Sends simulated keystrokes to the active window.Simule l'appui sur des touches du clavier et envoie la valeur dans la fenêtre active.

Send ( "keys" [, flag = 0] )

ParametersParamètres

keys The sequence of keys to send.La séquence de touches à envoyer.
flag [optional][optionnel][optionnel] Changes how "keys" is processed:Change la manière dont les touches sont envoyées:
    $SEND_DEFAULT (0) = Text contains special characters like + and ! to indicate SHIFT and ALT key-presses (default).qui sont utilisés pour indiquer les touches SHIFT et ALT (défaut).
    $SEND_RAW (1) = keys are sent raw.

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

See the Appendix for some tips on using Send().Consultez l'Appendice pour quelques astuces d'utilisation de Send(). AutoIt can send all ASCII and Extended ASCII characters (0-255). To send UNICODE characters you must use the "ASC" option and the code of the character you wish to send (see {ASC} at the bottom of the table below).

The "Send()" command syntax is similar to that of ScriptIt and the Visual Basic "SendKeys" command.La syntaxe de la commande Send() est similaire à celle de ScriptIt et à celle de la commande "SendKeys" en Visual Basic. Characters are sent as written with the exception of the following characters:Les caractères sont envoyés tels qu'ils sont écrits à l'exception des caractères suivants:

'!'
This tells AutoIt to send an ALT keystroke, therefore Send("This is text!a") would send the keys "This is text" and then press "ALT+a".Ce caractère dit à AutoIt de simuler l'appui de la touche ALT.Par exemple, Send()("Ceci est un texte!a") va envoyer les touches "Ceci est un texte" puis presser "ALT+a".

N.B.Remarque: Some programs are very choosy about capital letters and ALT keys, i.e.,Certains programmes sont très difficiles quand à l'utilisation des lettres majuscules combinées avec l'appui sur ALT. "!A" is different from "!a". The first says ALT+SHIFT+A, the second is ALT+a.Le premier code dit ALT+SHIFT+A et le deuxième ALT+a. If in doubt, use lowercase!Si vous avez un doute, utilisez les minuscules.

'+'
This tells AutoIt to send a SHIFT keystroke; therefore, Send("Hell+o") would send the text "HellO".Ce caractère dit à AutoIt de simuler l'appui de la touche SHIFT. Send("!+a") would send "ALT+SHIFT+a".Par exemple, Send()("Hell+o") va envoyer le texte "HellO", et <a1>Send()</a1>("!+a") va envoyer "ALT+SHIFT+a".

'^'
This tells AutoIt to send a CONTROL keystroke; therefore, Send("^!a") would send "CTRL+ALT+a".Ce caractère dit à AutoIt de simuler l'appui de la touche CTRL.Par exemple, Send()("^!a") va envoyer "CTRL+ALT+a".

N.B.Remarque: Some programs are very choosy about capital letters and CTRL keys, i.e.,Certains programmes sont très difficiles quand à l'utilisation des lettres majuscules combinées avec l'appui sur CTRL. "^A" is different from "^a".Par exemple, "^A" est différent de "^a". The first says CTRL+SHIFT+A, the second is CTRL+a.Le premier code dit CTRL+SHIFT+A et le deuxième CTRL+a. If in doubt, use lowercase!Si vous avez un doute, utilisez les minuscules.

'#'
The hash now sends a Windows keystroke; therefore, Send("#r") would send Win+r which launches the Run() dialog box.Ce caractère dit à AutoIt de simuler l'appui de la touche Windows.Par exemple, Send("#r") va envoyer Win+r qui va lancer la boîte de dialogue 'Exécuter' de Windows.


You can set SendCapslockMode to make CAPS LOCK disabled at the start of a Send() operation and restored upon completion.Vous pouvez utiliser SendCapslockMode pour désactiver le verrouillage majuscule au début de chaque utilisation de Send() et le réactiver à la fin.
However, if a user is holding down the Shift key when a Send() function begins, text may be sent in uppercase.Cependant, si un utilisateur maintient l'appui sur SHIFT au démarrage de la fonction Send(), le texte peut être envoyé en majuscule.
One workaround is to Send("{SHIFTDOWN}{SHIFTUP}") before the other Send() operations.Une manière de contourner est d'utiliser Send()("{SHIFTDOWN}{SHIFTUP}") avant les autres utilisations de Send().

Certain keyboards as the Czech keyboard sends different characters when using the Shift Key or with CAPS LOCK enabled while sending a character.Certains claviers comme le Tchèque envoient des caractères différents lors de l'appui sur la touche Shift ou pendant que le VERROUILLAGE MAJUSCULE est activé. Due to the send AutoIt implementation the CAPS LOCKed character will be sent as a Shifted character so it will not work.En raison du fonctionnement de la fonction Send, (les caractères majuscules sont envoyés comme des minuscules affublées d'un SHIFT) l'envoi des majuscules ne marchera pas.
Certain special keys can be sent and should be enclosed in braces:Certains caractères spéciaux peuvent être envoyés mais doivent être encadrés de crochets.

N.B.Remarque: Windows does not allow the simulation of the "CTRL-ALT-DEL" combination!Windows n'autorise pas la simulation de la combinaison "CTRL-ALT-DEL"!

Single keys can also be sent repeated:Des touches seules peuvent aussi être répétées, comme dans les exemples ci-dessous.

No key is sent if the count is zero allowing the count to be set programatically:Aucune touche n'est envoyée si le nombre est 0, ce qui permet de définir ce nombre par programme:

When running a script on a remote computer through a program as psexec (www.sysinternals.com) or beyondexec (www.beyondlogic.org) it is necessary, specially when sending strokes to a program launch by the script with a Run() function, to use ControlSend() or other ControlXXX functions to directly communicate with the control.Quand un script est lancé sur un ordinateur distant à travers un programme tiers comme psexec (www.sysinternals.com) ou beyondexec (www.beyondlogic.org), il est nécessaire, spécialement en envoyant des touches à un programme lancé par le script via la fonction Run(), d'utiliser ControlSend() ou d'autres fonctions ControlXXX pour communiquer directement avec le contrôle. Send() even with Opt("SendAttachMode", 1) will not work.
Using the -s mode when submitting can help to have better rights on the remote computer.Utilisez le mode -s quand vous envoyez des touches peut vous aider à avoir des meilleurs droits sur l'ordinateur distant.

Opt("SendKeyDelay",...) alters the the length of the brief pause in between sent keystrokes.
Opt("SendKeyDownDelay",...) alters the length of time a key is held down before being released during a keystroke.
Set both "SendKeyDelay" and "SendKeyDownDelay" to 0 to remove all delays when sending keystrokes.Définissez "SendKeyDelay" et "SendKeyDownDelay" à 0 pour enlever tous les délais lors de l'envoi de touches. This may be required under certain circumstances, for example, when locking the system ("#l") it may be necessary to remove the delays in order to prevent the WIN key from being stuck down.Cela peut être nécessaire dans certaines circonstances, par exemple, lors du verrouillage du ...système ("#L"), il peut être nécessaire d'enlever les délais afin de prévenir que la touche WIN reste enfoncée.

Example()

Func Example()
    ; Simulate the key combination Win + R to open the Run dialogue window.Simuler la combinaison de touches Win + R pour ouvrir la fenêtre de dialogue Exécuter.
    Send("#r")

    ; Wait 10 seconds for the Run dialogue window to appear.Attend 10 secondes que la fenêtre de dialogue Exécuter apparaît.
    WinWait("Run", "", 10)

    ; Simulate entering notepad.exe and pressing the 'ENTER' key.Simule une entrée dans notepad.exe et presse la touche 'ENTER'.
    Send("notepad.exe{Enter}")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Simulate entering the following string and pressing the 'F5' key to input the date and time into edit control of Notepad.Simuler l'entrée de la chaîne suivante et appuie sur la touche "F5" pour entrer la date et l'heure dans le contrôle d'édition du bloc-notes.
    Send("Today's time/date is {F5}")

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)

    ; Now a screen will pop up and ask to save the changes, the classname of the window is calledMaintenant, une fenêtre s'ouvre et demande d'enregistrer les modifications, le nom de classe de la fenêtre est appelé
    ; "#32770" and simulating the "TAB" key to move to the second button in which the "ENTER" is simulated to not "save the file""# 32770" et la simulation de la touche "TAB" pour passer au deuxième bouton dans lequel le "ENTER" est simulé pour ne pas enregistrer le fichier
    WinWaitActive("[CLASS:#32770]")
    Sleep(500)
    Send("{TAB}{ENTER}")
EndFunc   ;==>Example
