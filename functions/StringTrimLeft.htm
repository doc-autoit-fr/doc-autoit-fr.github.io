StringTrimLeft

Trims a number of characters from the left hand side of a string.Coupe un nombre de caractères depuis la gauche de la chaîne.

StringTrimLeft ( "string", count )

ParametersParamètres

string The string to evaluate.La chaîne à traiter.
count The number of characters to trim.Le nombre de caractères à couper.

Return ValueValeur de retour

Returns the string trimmed by count characters from the left.Retourne la chaîne coupée du nombre count de caractères depuis la gauche.

RemarksRemarques

If count is out-of-bounds, an empty string is returned.Si count est hors-limites, une chaîne vide est retournée.
StringTrimLeft($str, $n) is functionally equivalent to StringRight($str, StringLen($str) - $n)StringTrimLeft($str, $n) est fonctionnellement équivalent à StringRight($str, StringLen($str) - $n)

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sString = StringTrimLeft("This is a sentence with whitespace.", 5) ; Remove the 5 leftmost characters from the string.Coupe les 5 caractères les plus à gauche de la chaîne.
MsgBox($MB_SYSTEMMODAL, "", $sString)
