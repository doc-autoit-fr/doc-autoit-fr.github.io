StringAddCR

Takes a string and prefixes all linefeed characters ( Chr(10) ) with a carriage return character ( Chr(13) ).Ajoute @CR devant tous les caractèrs @LF d'une chaîne.

StringAddCR ( "string" )

ParametersParamètres

string The string to convert.La chaîne à traiter.

Return ValueValeur de retour

Returns the string with all instances of linefeed characters ( @LF ) prefixed with a carriage return character ( @CR ).Retourne la chaîne avec toutes les instances des caractères saut de ligne (@LF) préfixées par un caractère de retour chariot (@CR).

RelatedLiens

StringReplace, StringStripCR

ExampleExemple

#include <MsgBoxConstants.au3>

; Add the carriage return charcter to all the line feed characters in the string.Ajoute le caractère 'retour chariot' à tous les caractères 'ligne suivante' dans la chaîne.
Local $sString = StringAddCR("This is a sentence " & @LF & "with " & Chr(10) & "whitespace.")
MsgBox($MB_SYSTEMMODAL, "", $sString)
