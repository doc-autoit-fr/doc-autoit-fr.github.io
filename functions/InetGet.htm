InetGet

Downloads a file from the internet using the HTTP, HTTPS or FTP protocol.Télécharge un fichier depuis internet en utilisant le protocole HTTP, HTTPS ou FTP.

InetGet ( "URL", "filename" [, options = 0 [, background = 0]] )

ParametersParamètres

URL URL of the file to download.URL du fichier à télécharger. See remarks below.Voir remarques ci-dessous.
filename Local filename to download to.Chemin local du fichier à télécharger.
options [optional][optionnel]$INET_LOCALCACHE (0) = Get the file from local cache if available (default).[optionnel]$INET_LOCALCACHE (0) = Récupère le fichier depuis le cache local si disponible (défaut).
    $INET_FORCERELOAD (1) = Forces a reload from the remote site.
    $INET_IGNORESSL (2) = Ignore all SSL errors (with HTTPS connections).
    $INET_ASCIITRANSFER (4) = Use ASCII when transferring files with the FTP protocol (Can not be combined with flag $INET_BINARYTRANSFER (8)).
    $INET_BINARYTRANSFER (8) = Use BINARY when transferring files with the FTP protocol (Can not be combined with flag $INET_ASCIITRANSFER (4)). This is the default transfer mode if none are provided.Ceci est le mode de transfert par défaut si aucun n'est fourni.
    $INET_FORCEBYPASS (16) = By-pass forcing the connection online (See remarks).

Constants are defined in InetConstants.au3.
background [optional][optionnel]
    $INET_DOWNLOADWAIT (0) = Wait until the download is complete before continuing (default).
    $INET_DOWNLOADBACKGROUND (1) = return immediately and download in the background (see remarks).
Constants are defined in InetConstants.au3Les constantes sont définies dans InetConstants.au3

Return ValueValeur de retour

Success:Succès : the return value changes depending on if the download is in the background:La valeur de retour change si le téléchargement est en arrière-plan ou pas:
Background: a handle is returned.Arrière-plan:Retourne un handle.
Wait: the number of bytes downloaded.Attente:Retourne le nombre d'octets téléchargés.
Failure:Échec : Background: a handle is returned.Arrière-plan:Retourne un handle.
Wait:Attente: 0 and sets the flag to non-zero.Retourne 0 et @error <&lg; 0.

RemarksRemarques

Internet Explorer 3 or greater must be installed for this function to work.Internet Explorer 3 ou plus récent doit être installé pour faire fonctionner la fonction.

Use the returned handle with InetGetInfo() to determine if there was an error with the download.Utilisez le handle retourné par InetGetInfo() pour déterminer si il y a une erreur dans le téléchargement.
The returned handle must be closed with InetClose().The handle retourné doit être fermé avec InetClose().

The URL parameter should be in the form "http://www.somesite.com/path/file.html" - just like an address you would type into your web browser.Le paramètre URL devrait être de la forme "http://www.lesite.com/chemin/fichier.html" - tout comme une adresse que vous tapez dans votre navigateur web.

To use a username and password when connecting simply prefix the servername with "username:password@", e.g.Pour utiliser un nom d'utilisateur et un mot de passe lors de la connexion, préfixez simplement le nom de serveur avec "nomutilisateur:motdepasse@", par exemple:
"http://myuser:mypassword@www.somesite.com""http://nomutilisateur:motdepasse@www.lesite.fr"


Notes about the "background" ParameterNotes à propos du paramètre "background"

By default the function waits until the download has finished before returning.Par défaut, la fonction attend que le téléchargement soit terminé avant d'envoyer la valeur de retour. If the background parameter is set to $INET_DOWNLOADBACKGROUND (1) the function returns immediately and the download continues in the background.Si le paramètre background est mis à $INET_DOWNLOADBACKGROUND (1), le retour de la fonction est immédiat et le téléchargement continue en arrière-plan. The function InetGetInfo() can be used to check the status of the download.La fonction InetGetInfo() peut-être utilisée pour vérifier l'état du téléchargement. It takes the handle returned from InetGet().Elle utilise le handle retourné par InetGet().

Multiple downloads are supported if they are started in background mode.Les téléchargements multiples sont supportés s'ils sont démarrés en mode arrière-plan.

To abort a download call InetClose() and pass it the handle returned by InetGet().Pour interrompre un téléchargement, appelez InetClose() en lui passant le handle retourné par InetGet().

By default AutoIt forces a connection before starting a download.Par défaut, AutoIt force une connexion avant de démarrer un téléchargement. For dial-up users this will prompt to go online or dial the modem (depending on how the system is configured).Les utilisateurs de modems analogique seront invités à passer en ligne ou le numéro sera composé (selon la configuration du système). The options value $INET_FORCEBYPASS (16) disables this behavior.L'option $INET_FORCEBYPASS (16) désactive ce comportement. Disabling the behavior can be useful for persistent connects (Broadband, LAN).Désactiver ce comportement peut-être utile pour les connexions continues (Large bande , LAN). However, it is also required to work around certain issues in Windows Vista and Windows 7.Cependant, il est aussi nécessaire de travailler autour de certains problèmes dans Windows Vista et Windows 7.

ExampleExemple

Example 1Exemple 1

#include <InetConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

; Download a file in the background.Télécharge un fichier en arrière-plan.
; Wait for the download to complete.Attend la fin du téléchargement.

Example()

Func Example()
    ; Save the downloaded file to the temporary folder.Enregistre le fichier téléchargé dans le dossier temporaire.
    Local $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Download the file in the background with the selected option of 'force a reload from the remote site.'Télécharge le fichier en arrière-plan avec l'option choisie de 'forcer le chargement du site distant'.
    Local $hDownload = InetGet("http://www.autoitscript.com/autoit3/files/beta/update.dat", $sFilePath, $INET_FORCERELOAD, $INET_DOWNLOADBACKGROUND)

    ; Wait for the download to complete by monitoring when the 2nd index value of InetGetInfo returns True.Attend la fin du téléchargement pour vérifier si la valeur de l'indice de 2e InetGetInfo renvoie True.
    Do
        Sleep(250)
    Until InetGetInfo($hDownload, $INET_DOWNLOADCOMPLETE)

    ; Retrieve the number of total bytes received and the filesize.Récupére le nombre total d'octets reçus et la taille du fichier.
    Local $iBytesSize = InetGetInfo($hDownload, $INET_DOWNLOADREAD)
    Local $iFileSize = FileGetSize($sFilePath)

    ; Close the handle returned by InetGet.Ferme le handle renvoyé par InetGet.
    InetClose($hDownload)

    ; Display details about the total number of bytes read and the filesize.Affiche les détails sur le nombre total d'octets lus et la taille du fichier.
    MsgBox($MB_SYSTEMMODAL, "", "The total download size: " & $iBytesSize & @CRLF & _
            "The total filesize: " & $iFileSize)

    ; Delete the file.Supprime le fichier.
    FileDelete($sFilePath)
EndFunc   ;==>Example
