DllOpen

Opens a DLL file for use in DllCall.Ouvre un fichier DLL pour une utilisation dans DllCall.

DllOpen ( "filename" )

ParametersParamètres

filename Filename of the DLL file to open.Nom du fichier DLL à ouvrir.

Return ValueValeur de retour

Success:Succès : a dll "handle" to be used with subsequent Dll functions.Retourne un handle de DLL pour utilisation ultérieure avec les fonctions DLL.
Failure:Échec : -1 if error occurs.Retourne -1 si une erreur est rencontrée.

RemarksRemarques

When finished working with a dll, call the DllClose() function to close it.Lorsque vous avez terminé d'utiliser une DLL, utilisez la fonction DllClose() pour la fermer. AutoIt normally closes all files upon termination, but explicitly calling DllClose() is still a good idea.Normalement AutoIt ferme tous les fichiers lors de sa fermeture, mais appeler explicitement DllClose() est toujours une bonne idée.

RelatedLiens

DllCall, DllClose

ExampleExemple

Local $hDLL = DllOpen("user32.dll")
DllCall($hDLL, "int", "MessageBox", "hwnd", 0, "str", "Some text", "str", "Some title", "int", 0)
DllClose($hDLL)
