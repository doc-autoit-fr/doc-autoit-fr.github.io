TrayItemSetState

Sets the state of a tray menu/item control.Définit l'état d'un contrôle menu ou item de la zone de notification.

TrayItemSetState ( controlID, state )

ParametersParamètres

controlID The control identifier (controlID) as returned by a TrayCreateItem or TrayCreateMenu function.L'identifiant du contrôle (controlID) tel que retourné par la fonction TrayCreateItem ou TrayCreateMenu fonction.
state See the State table below.Consultez le Tableau des états ci-dessous.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

State table

StateEtat ValueValeur CommentsCommentaires
No Change 0Retourne 0
$TRAY_CHECKED 1Retourne 1 Menuitem will be checkedL'item du menu sera coché
$TRAY_UNCHECKED 4 Menuitem will be uncheckedL'item du menu sera décoché
$TRAY_ENABLE 64 Menuitem will be enabledL'item du menu sera activé
$TRAY_DISABLE 128 Menuitem will be greyed outL'item du menu sera grisé
$TRAY_FOCUS 256 Menuitem will be selectedL'item du menu sera sélectionné
$TRAY_DEFAULT 512 Menuitem will be set as default menuitemL'item du menu sera défini comme l'item par défaut

State values can be summed up as for example $TRAY_CHECKED + $TRAY_DEFAULT sets the menuitem in an checked and default state.Les valeurs de l'État peuvent s'ajouter comme par exemple $TRAY_CHECKED + $TRAY_DEFAULT qui définit l'item du menu dans l'état coché et par défaut.

To reset/delete the $TRAY_DEFAULT-state for a menuitem just use this function on the item with another state, for instance with $TRAY_ENABLE.Pour réinitialiser ou effacer l'état $TRAY_DEFAULT pour un item de menu il suffit d'utiliser cette fonction sur l'item avec un autre état, par exemple avec $TRAY_ENABLE.

When you set $TRAY_DEFAULT state for a menuitem, the default behavior of primary double click on tray icon is to execute this default menuitem.Lorsque vous définissez l'état $TRAY_DEFAULT pour un item de menu, le comportement par défaut d'un double-clic gauche sur l'icône de la zone de notification est d'exécuter cet item par défaut. To disable this behavior use Opt("TrayMenuMode", 4).

The above constants are defined in #include <TrayConstants.au3>.

Example()

Func Example()
    Local $idSetState = TrayCreateItem("Set 'About' State")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

            Case $idSetState
                ; Set the 'About' item state to checked.Définit l'état de l'item 'About' à coché.
                TrayItemSetState($idAbout, $TRAY_CHECKED)

            Case $idExit ; Exit the loop.Sort de la boucle.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
