GUICtrlCreateDummy

Creates a Dummy control for the GUI.Crée un contrôle fictif dans la GUI.

GUICtrlCreateDummy ( )

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

This control can receive messages through a GUICtrlSendToDummy() call. The control will "notify" as normal and the value sent with GUISendToDummy can be read with GUICtrlRead().

RelatedLiens

GUICtrlRead, GUICtrlSendToDummy, GUICtrlSetData, GUICtrlSetOnEvent

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()

    GUICreate("GUICtrlCreateDummy", 250, 200, 100, 200)
    GUISetBkColor(0x00E0FFFF) ; will change background colorchange la couleur de fond

    Local $idUser = GUICtrlCreateDummy()
    Local $idButton = GUICtrlCreateButton("event", 75, 170, 70, 20)
    Local $idCancel = GUICtrlCreateButton("Cancel", 150, 170, 70, 20)
    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
        $idMsg = GUIGetMsg()

        Select
            Case $idMsg = $idButton
                GUICtrlSendToDummy($idUser)
            Case $idMsg = $idCancel
                GUICtrlSendToDummy($idUser)
            Case $idMsg = $idUser
                ; special action before closing
                ; ...
                Exit
        EndSelect
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
