StringToASCIIArray

Converts a string to an array containing the ASCII code of each character.Convertit une chaîne en un tableau contenant le code ASCII de chaque caractère.

StringToASCIIArray ( "string",[start = 0 [, end [, encoding = 0]]] )

ParametersParamètre

"string" The string to convert to an array of ASCII codes.La chaîne à convertir en tableau de codes ASCII.
start [optional][optional] False - i, v, x ... The 0-based position to start processing from (Default:La position de base 0 à partir de laquelle le traitement doit commencer ( par défaut : 0).0 ).
end [optional][optional] False - i, v, x ... The 0-based position to end processing at (Default:La position de base 0 à partir de laquelle le traitement doit s'arrêter ( par défaut : StringLen("string")).StringLen("chaîne")).
encoding [optional][optional] False - i, v, x ... The returned array will contain values in the specified character set:Le tableau retourné contiendra les valeurs dans le jeu de caractères spécifié :
    $SE_UTF16 (0) = UTF-16 (Default)
    $SE_ANSI (1) = ANSI
    $SE_UTF8 (2) = UTF-8

Constants are defined in StringConstants.au3.

Return ValueRemarque

Success: an array where each element is the UNICODE code of the character at the corresponding position.Retourne un tableau où chaque élément est le code Unicode du caractère à la position correspondante.
Failure: an empty string.Retourne une chaîne vide.

Remarks<i0>

The string may contain embedded Chr(0).La chaîne peut embarquer des caractères Chr(0). These will appear in the returned array as well as any data past them.Ceux-ci apparaissent dans le tableau retourné ainsi que les données passées avec eux. Processing only stops when the end of the string is reached or the user specified end.Traitement ne s'arrête que lorsque la fin de la chaîne est atteinte ou que l'utilisateur ait spécifié la fin.

In order to convert binary data to an array using this function it must first be converted to a string with the BinaryToString() function.Afin de convertir les données binaires en un tableau en utilisant cette fonction, il doit d'abord être converti en une chaîne avec la fonction BinaryToString().

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Convert the string to an ASCII array.Convertit la chaîne en un tableau ASCII.
    Local $aArray = StringToASCIIArray("This is a sentence with whitespace.")

    ; Display the array to see that it contains the ASCII values for each character in the string.Affiche le tableau pour voir qu'il contient les valeurs ASCII de chaque caractère de la chaîne.
    _ArrayDisplay($aArray)

    ; Convert the array into a string.Convertit le tableau en une chaîne.
    Local $sString = StringFromASCIIArray($aArray)

    ; Display the string to see that it matches the original string initially converted to an array.Affiche la chaîne pour voir qu'elle correspond à la chaîne d'origine d'abord converti en un tableau.
    MsgBox($MB_SYSTEMMODAL, "", $sString)
EndFunc   ;==>Example
