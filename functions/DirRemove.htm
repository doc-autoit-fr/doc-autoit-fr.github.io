Function ReferenceRéférence de fonction

DirRemove

Deletes a directory/folder.Supprime un répertoire/dossier.

DirRemove ( "path" [, recurse = 0] )

ParametersParamètres

path Path of the directory to remove.Chemin du répertoire à supprimer.
recurse [optional][optionnel] Use this flag to specify if you want to delete sub-directories too.Utilisez ce drapeau pour spécifier si vous voulez également supprimer les sous-répertoires.
    $DIR_DEFAULT (0) = (default) deletes the folder, only if it is empty
    $DIR_REMOVE (1) = remove files and subdirectories (like the DOS DelTree command)

Constants are define in "AutoItConstants.au3".

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if there is an error removing the directory (or if directory does not exist).Retourne 0 s'il y a eu une erreur de suppression du répertoire (ou si le répertoire n'existe pas).

RemarksRemarques

Some directory attributes can make the deletion impossible, therefore if this is the case look at FileSetAttrib() to change the attributes of a directory.Certains attributs du dossier peut rendre la suppression impossible, cependant si c'est le cas, consultez FileSetAttrib() pour changer les attributs d'un répertoire.

RelatedLiens

DirCopy, DirCreate, DirMove, FileDelete, FileRecycle

ExampleExemple

#include <MsgBoxConstants.au3>

; Delete C:\Test1 and all subdirs and filesSupprimer C :\\ Test1 et tous les sous-répertoires et fichiers
Example()

Func Example()
    Local $sFldr1 = "C:\Test1\"
    Local $sFldr2 = "C:\Test1\Folder1\"
    Local $sFldr3 = "C:\Test1\Folder1\Folder2\"

    If DirGetSize($sFldr1) <> -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "Directory already exists!")
        Return False
    EndIf

    DirCreate($sFldr3)

    RunWait("explorer /root, C:\Test1\Folder1")
    Local $hWnd = WinGetHandle("[TITLE:Folder1;CLASS:CabinetWClass]")

    MsgBox($MB_SYSTEMMODAL, "", "Explorer is opened with Folder2 displayed.")

    DirRemove($sFldr3, 1)
    MsgBox($MB_SYSTEMMODAL, "", "The sub folder: Folder2 has been deleted.")

    WinClose($hWnd)

    DirRemove($sFldr2) ;clean up test foldersnettoyer les dossiers de test
    DirRemove($sFldr1) ;clean up test foldersnettoyer les dossiers de test
EndFunc   ;==>Example
