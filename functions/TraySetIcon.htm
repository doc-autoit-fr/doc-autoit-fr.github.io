TraySetIcon

Loads/Sets a specified tray icon.Charge/Définit l'icône spécifiée pour la zone de notification.

TraySetIcon ( [filename [, iconID]] )

ParametersParamètres

filename [optional][optionnel] The filename of the icon to be display in the tray.Le nom de l'icône à afficher dans la zone de notification.
iconID [optional][optionnel] Icon identifier if the file contains multiple icons.Identifiant de l'icône si le fichier contient plusieurs icônes.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

To reset the icon to the default, use the function with no parameters TraySetIcon().

Passing a positive number will reference the string equivalent icon name.Passer un nombre positif, fera référence à l'équivalent chaîne du nom de l'icône.
Passing a negative number causes 1-based "index" behaviour.Passer un nombre négatif, provoque un comportement d'"index" basé sur 1. Some Dll can have icon extracted just with negative numbers.Les icônes de certaines Dll ne peuvent être extraites seulement qu'avec des nombres négatifs.

The filename parameter can be set to "blank", "info", "question", "stop" or "warning" to select some corresponding standard icon.Le paramètre filename peut être "blank", "info", "question", "stop" ou "warning" afin de sélectionner l'icône standard correspondante.

Example()

Func Example()
    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    Local $hTimer = TimerInit() ; Begin the timer and store the handle in a variable.Démarre le chrono et stocke Le handle dans une variable.
    Local $fDiff = 0, $iIndex = 0

    While 1
        $fDiff = TimerDiff($hTimer) ; Find the difference in time from the previous call of TimerInitTrouve la différence de temps avec l'appel précédent de TimerInit
        If $fDiff > 1000 Then ; If the difference is greater than 1 second then change the tray menu icon.Si la différence est supérieure à 1 seconde, alors change l'icône du menu de la zone de notification.

            $iIndex = -Random(0, 100, 1) ; Use a negative number for ordinal numbering.Utilise un nombre négatif pour la numérotation ordinale.
            TraySetToolTip("Currently using the icon shell32.dll, " & $iIndex & ".") ; Set the tray menu tooltip with information about the icon index.Définit l'info-bulle du menu avec des informations sur l'indice de l'icône.
            TraySetIcon("shell32.dll", $iIndex) ; Set the tray menu icon using the shell32.dll and the random index number.Définit l'icône du menu de la barre à l'aide de shell32.dll et le numéro d'index aléatoire.
            $hTimer = TimerInit() ; Reset the timer.Réinitialise le compteur.

        EndIf

        Switch TrayGetMsg()
            Case $idExit ; Exit the loop.Sort de la boucle.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
