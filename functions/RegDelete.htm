RegDelete

Deletes a key or value from the registry.Supprime une clé ou une valeur dans le registre.

RegDelete ( "keyname" [, "valuename"] )

ParametersParamètres

keyname The registry key to delete.La clé de registre à supprimer.
valuename [optional][optionnel] The valuename to delete.La valeur à supprimer.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Special: 0 if the key/value does not exist.Retourne 0 si la clé/valeur n'existe pas.
Failure:Échec : 2 if error deleting key/value and sets the @error flag to non-zero.Retourne 2 si erreur lors de la suppression et @error <> 0.
@error:@error : 1 = unable to open requested key1 = impossible d'ouvrir la clé demandée
 2 = unable to open requested main key
 3 = unable to remote connect to the registry
-1 = unable to delete requested value-1 = impossibilité de supprimer la valeur demandée
-2 = unable to delete requested key/value-2 = impossibilité de supprimer la clé/valeur demandée

RemarksRemarques

A registry key must start with "HKEY_LOCAL_MACHINE" ("HKLM") or "HKEY_USERS" ("HKU") or "HKEY_CURRENT_USER" ("HKCU") or "HKEY_CLASSES_ROOT" ("HKCR") or "HKEY_CURRENT_CONFIG" ("HKCC").Une clé de registre doit commencer par "HKEY_LOCAL_MACHINE" ("HKLM") ou "HKEY_USERS" ("HKU") ou "HKEY_CURRENT_USER" ("HKCU") ou "HKEY_CLASSES_ROOT" ("HKCR") ou "HKEY_CURRENT_CONFIG" ("HKCC") .

When running on 64-bit Windows if you want to delete a key or value specific to the 64-bit environment you have to suffix the HK... with 64 i.e.Dans un système Windows 64-bits, si vous souhaitez supprimer une clé ou une valeur spécifique à l'environnement 64-bits, il est nécessaire de faire suivre HK...par 64. HKLM64.avec 64 c'est à dire HKLM64.

To access the (Default) value use "" (an empty string) for the valuename.Pour accéder à la valeur par (par défaut) utilisez "" (une chaîne vide) pour le nom de la valeur.

Deleting from the registry is potentially dangerous--please exercise caution!Supprimer une clé dans le registre est potentiellement dangereux -- A utiliser avec précaution!

It is possible to access remote registries by using a keyname in the form "\\computername\keyname".Il est possible d'accéder à des registres à distance en utilisant des noms de valeur sous la forme "\\nom_ordinateur\nom_de_la_clé". To use this feature you must have the correct access rights.Pour utiliser cette fonctionnalité vous devez avoir les droits d'accès corrects.
