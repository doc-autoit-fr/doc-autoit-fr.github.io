GUICtrlSetDefBkColor

Sets the default background color of all the controls of the GUI window.Définit la couleur de fond par défaut de tous les contrôles de la fenêtre GUI.

GUICtrlSetDefBkColor ( defbkcolor [, winhandle] )

ParametersParamètre

defbkcolor Default background color for all controls.Couleur de fond par défaut pour tous les contrôles.
winhandle [optional][optional] False - i, v, x ... Windows handle as returned by GUICreate() (default is the previously used window).Handle Windows retourné par GUICreate() (par défaut, la fenêtre précédemment utilisée ).

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0.Retourne 0.
