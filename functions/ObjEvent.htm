ObjEvent

Handles incoming events from the given Object.Gère les événements entrants de l'objet donné.

ObjEvent ( $ObjectVar, "functionprefix" [, "interface name"] )
ObjEvent ( "AutoIt.Error" [, "function"] )

ParametersParamètres

$ObjectVar A variable containing an Object from which you want to receive eventsUne variable contenant un objet à partir duquel vous souhaitez recevoir des événements
"functionprefix" The prefix of the functions you define to handle receiving events.Le préfixe des fonctions que vous définissez pour gérer les événements reçus.
The prefix is appended by the Objects method name.Le préfixe est ajouté au nom des méthodes des objets.
"interface name" [optional][optionnel] name of an Event interface to use.nom d'une interface d'événement à utiliser.
Note:Note : It must be a supported as outgoing for the Object AND it must be of type DISPATCH.

Return ValueValeur de retour

Success:Succès : an object or a function name.Retourne un objet ou un nom de fonction.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

The first format is used to receive Events from the given Object.Le premier format est utilisé pour recevoir des événements de l'objet donné.
To receive a specific event, create an AutoIt function name using the given prefix appended with the event name.Pour recevoir un événement particulier, créez un nom de fonction AutoIt en utilisant le préfixe donné en annexe avec le nom de l'événement.

The second format is used for COM Error Handling.Le second format est utilisé pour gérer les erreurs COM.
If any COM error occurs, the given function is being called.Si une erreur COM se produit, la fonction donnée est appelée. First parameter for the function will be error object.Le premier paramètre de la fonction sera l'objet d'erreur.
You can use it to access different properties of this object.Vous pouvez l'utiliser pour accéder aux propriétés différentes de cet objet.
If the second parameter is omitted, it will return the name of the current Error handler function or function, depending what was used when registering.Si le second paramètre est omis, elle retournera le nom de la fonction du gestionnaire d'erreur en cours ou de la fonction, selon ce qui a été utilisé lors de l'enregistrement.

Properties of the AutoIt Error Object:Propriétés de l'objet Erreur AutoIt :

.number The Windows HRESULT value from a COM callLa valeur HRESULT de Windows d'un appel COM
.windescription The FormatWinError() text derived from .numberLe texte FormatWinError() dérivé de .number
.source Name of the Object generating the error (contents from ExcepInfo.source)Nom de l'objet généré par l'erreur (contenu de ExcepInfo.source )
.description Source Object's description of the error (contents from ExcepInfo.description)Description de l'objet source de l'erreur (contenu de ExcepInfo.description)
.helpfile Source Object's helpfile for the error (contents from ExcepInfo.helpfile)Le fichier d'aide de l'objet source de l'erreur (contenu de ExcepInfo.helpfile)
.helpcontext Source Object's helpfile context id number (contents from ExcepInfo.helpcontext)L'ID du contexte du fichier d'aide de l'objet de la source (contenu de ExcepInfo.helpcontext)
.lastdllerror The number returned from GetLastError()Le nombre renvoyé par GetLastError()
.scriptline The script line on which the error was generatedLa ligne du script sur laquelle l'erreur est apparue

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Error monitoring.Suivi d'erreur. This will trap all COM errors while alive.Ce code piège toutes les erreurs COM pendant l'exécution.
    ; This particular object is declared as local, meaning after the function returns it will not exist.Cet objet particulier est déclarée en local, ce qui signifie qu'après le retour de la fonction, il n'existera plus.
    Local $oErrorHandler = ObjEvent("AutoIt.ErrorErreur", "_ErrFunc")

    ; Create Internet Explorer objectCrée un objet Internet Explorer
    Local $oIE = ObjCreate("InternetExplorer.Application")
    ; Check for errorsVérifie les erreurs
    If @error Then Return

    $oIE.Visible = True ; set visibilityDéfinit la visibilité

    ; Custom sink objectObjet sink personnel
    Local $oIEEvents = ObjEvent($oIE, "_IEEvent_", "DWebBrowserEvents2")

    ; Navigate somewhereNavigue quelque part
    $oIE.navigate("http://www.google.com/")
    ; Check for errors while loadingVérifie les erreurs lors du chargement
    If @error Then
        $oIE.Quit()
        Return
    EndIf

    ; Wait for page to loadAttend que la page se charge
    While 1
        If $oIE.readyState = "complete" Or $oIE.readyState = 4 Then ExitLoop
        Sleep(10)
    WEnd

    ; Deliberately cause error by calling non-existing methodProvoque délibérément une erreur en appelant une méthode qui n'existe pas
    $oIE.PlayMeARockAndRollSong()

    ; Check for errorsVérifie les erreurs
    If @error Then MsgBox($MB_SYSTEMMODAL, "COM Error", "@error is set to COM error number." & @CRLF & "@error = 0x" & Hex(@error))

    ; Wait few seconds to see if more events will be firedAttend quelques secondes pour voir si d'autres événements seront déclenchés
    Sleep(3000)

    ; Nothing more to do.Rien de plus à faire. Close IE and return from the functionFerme IE et retourne
    $oIE.Quit()

    #forceref $oErrorHandler, $oIEEvents
EndFunc   ;==>Example

Global $__g_oTemplateCOMErrorHandler = 0

Local $oShell = ObjCreate("Shell.Application")

; Wrap calls that are likely to fatal error

; This won't fatal error, it will set @error as in 3.3.11.0
Template_COMErrorRegister()
$oShell.InvalidFunction()
If @error Then MsgBox(4096, "COM Error Detected", @error)
Template_COMErrorUnregister()

; This will fatal error, the 2nd line will not be reached.
$oShell.InvalidFunction()
If @error Then MsgBox(4096, "COM Error Detected", @error)

Func Template_COMErrorRegister()
    $__g_oTemplateCOMErrorHandler = ObjEvent("AutoIt.ErrorErreur", "Template_COMErrFunc")
EndFunc   ;==>Template_COMErrorRegister

Func Template_COMErrorUnregister()
    $__g_oTemplateCOMErrorHandler = 0
EndFunc   ;==>Template_COMErrorUnregister

Func Template_COMErrFunc()
    ; Do nothing special, just check @error after suspect functions.
EndFunc   ;==>Template_COMErrFunc
