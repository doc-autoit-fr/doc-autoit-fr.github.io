InetGetInfo

Returns detailed data for a handle returned from InetGet().Retourne les données détaillées d'un handle retourné par InetGet().

InetGetInfo ( [handle [, index = -1]] )

ParametersParamètres

handle [optional][optionnel][optionnel] A handle return from InetGet().Un handle retourné par InetGet().
index [optional][optionnel][optionnel] The index for the data to retrieve.L'index pour les données à récupérer. If this value is -1 an array containing all of the below data will be returned.Si cette valeur est -1, un tableau contenant toutes les données ci-dessous sera retourné.
    $INET_DOWNLOADREAD (0) - Bytes read so far (this is updated while the download progresses).
    $INET_DOWNLOADSIZE (1) - The size of the download in bytes (this may not always be present).
    $INET_DOWNLOADCOMPLETE (2) - Set to True if the download is complete, False if the download is still ongoing.
    $INET_DOWNLOADSUCCESS (3) - True if the download was successful. If this is False then the next data member will be non-zero.Si celui-ci est à False alors le membre de données suivante sera non-nulle.
    $INET_DOWNLOADERROR (4) - The error value for the download. The value itself is arbitrary.La valeur elle-même est arbitraire. Testing that the value is non-zero is sufficient for determining if an error occurred.Tester que la valeur est non-nulle est suffisant pour déterminer si une erreur s'est produite.
    $INET_DOWNLOADEXTENDED (5) - The extended value for the download. The value is arbitrary and is primarily only useful to the AutoIt developers.La valeur est arbitraire et est principalement utile aux développeurs d'AutoIt.

Constants are defined in InetConstants.au3.

Return ValueValeur de retour

Success:Succès : the request data.Retourne les données demandées.
Failure:Échec : an empty string and sets the @error flag to non-zero.Retourne une chaîne vide et @error <> 0.

RemarksRemarques

If called with no arguments then the total number of active downloads will be returned.Si la fonction est appelée sans aucun argument alors le nombre total de téléchargements actifs sera retourné.

This function can be called in a loop to query the number of bytes download or to pause until a download is complete.Cette fonction peut-être appelée dans une boucle pour récupérer le nombre d'octets téléchargés ou pour faire une pause jusqu'à ce que le téléchargement soit complet.

ExampleExemple

#include <InetConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Save the downloaded file to the temporary folder.Enregistre le fichier téléchargé dans le dossier temporaire.
    Local $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Download the file in the background with the selected option of 'force a reload from the remote site.'Télécharge le fichier en arrière-plan avec l'option choisie de 'forcer le chargement du site distant'.
    Local $hDownload = InetGet("http://www.autoitscript.com/autoit3/files/beta/update.dat", $sFilePath, $INET_FORCERELOAD, $INET_DOWNLOADBACKGROUND)

    ; Wait for the download to complete by monitoring when the 2nd index value of InetGetInfo returns True.Attend la fin du téléchargement pour vérifier si la valeur de l'indice de 2e InetGetInfo renvoie True.
    Do
        Sleep(250)
    Until InetGetInfo($hDownload, $INET_DOWNLOADCOMPLETE)

    ; Retrieve details about the download file.Consulte les détails sur le fichier de téléchargement.
    Local $aData = InetGetInfo($hDownload)
    If @error Then
        FileDelete($sFilePath)
        Return False ; If an error occurred then return from the function and delete the file.
    EndIf

    ; Close the handle returned by InetGet.Ferme le handle renvoyé par InetGet.
    InetClose($hDownload)

    ; Display details about the downloaded file.Affiche les détails sur le fichier téléchargé.
    MsgBox($MB_SYSTEMMODAL, "", "Bytes read: " & $aData[$INET_DOWNLOADREAD] & @CRLF & _
            "Size: " & $aData[$INET_DOWNLOADSIZE] & @CRLF & _
            "Complete: " & $aData[$INET_DOWNLOADCOMPLETE] & @CRLF & _
            "successful: " & $aData[$INET_DOWNLOADSUCCESS] & @CRLF & _
            "@error:@error : " & $aData[$INET_DOWNLOADERROR] & @CRLF & _
            "@extended: " & $aData[$INET_DOWNLOADEXTENDED] & @CRLF)

    ; Delete the file.Supprime le fichier.
    FileDelete($sFilePath)
EndFunc   ;==>Example
