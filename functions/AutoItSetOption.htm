AutoItSetOption

Changes the operation of various AutoIt functions/parameters.Change le mode de fonctionnement de divers fonctions/paramètres d'AutoIt.

AutoItSetOption ( "option" [, param] )

ParametersParamètres

option The option to change. See Remarks.Voir Remarque.
param [optional][optionnel] The value to assign to the option.La valeur à assigner à l'option. The type and meaning vary by option.Le type et la signification varient selon l'option. See remarks below.Voir remarques ci-dessous. If the param is not provided, then the function just returns the value already assigned to the option.Si le paramètre n'est pas indiqué, alors la fonction retourne la valeur qui lui est assignée. The keyword Default can be used for the parameter to reset the option to its default value.Le mot clé Default peut être utilisé en paramètre pour réinitialiser la valeur par défaut de l'option.

Return ValueValeur de retour

Success:Succès : the value of the previous setting for the option.Retourne la valeur du paramètre précédent pour l'option.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0. Failure will occur if the parameters are invalid (such as an option that does not exist).L'échec peut se produire si les paramètres sont invalides (tel qu'un paramètre qui n'existe pas).

1 = (default) Complete / Slow mode1 = (défaut) Complète / Mode lent
