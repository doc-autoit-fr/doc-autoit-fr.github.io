GUICtrlSendToDummy

Sends a message to a Dummy control.Envoie un message à un contrôle factice.

GUICtrlSendToDummy ( controlID [, state] )

ParametersParamètres

controlID The control identifier (controlID) as returned by GUICtrlCreateDummy()L'identifiant du contrôle (controlID) tel que renvoyé par GUICtrlCreateDummy()
state [optional][optionnel] value that can be retrieved later on by GUICtrlRead()Valeur qui peut être récupéré plus tard par GUICtrlRead()

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

When this function is called a notification that can be handled through the message loop or with a GUICtrlSetOnEvent() function is generated (as if the control had been "clicked" on).Lorsque cette fonction est appelée, une notification, qui peut être manipulé à travers la boucle de messages ou avec la fonction GUICtrlSetOnEvent(), est générée (comme si le contrôle avait été "cliqué").

Note that the function will not action the dummy control if the GUI in which it was created is hidden, as by design none of the controls on such a GUI can be actioned.Notez que la fonction ne sera pas l'action du contrôle factice si la GUI dans laquelle il a été créé est cachée, comme pendant la conception où aucun contrôle ne peut être exécuté.

ExampleExemple

#include <GUIConstantsEx.au3>

Global $g_idUserDummy, $g_iState = 0

Example()

Func Example()
    Opt("GUIOnEventMode", 1) ; Set the option to use GUIOnEventMode.Définit l'option GUIOnEventMode.

    GUICreate("GUISendToDummy", 220, 200, 100, 200)
    GUISetBkColor(0x00E0FFFF) ; Change the background color of the GUI.Change la couleur du fond de la GUI.
    GUISetOnEvent($GUI_EVENT_CLOSE, "OnExit") ; Set an event to call the 'OnExit' function.Définit un événement pour appeler la fonction 'OnExit'.

    $g_idUserDummy = GUICtrlCreateDummy()
    GUICtrlSetOnEvent(-1, "OnDummy") ; Set an event to call the 'OnExit' function when this control is selected.Définit un événement pour appeler la fonction 'OnExit' lorsque ce contrôle est sélectionné.

    GUICtrlCreateButton("Click", 70, 170, 85, 25)
    GUICtrlSetOnEvent(-1, "OnClick") ; Set an event to call the 'OnClick' function when this control is selected.Définir un événement pour appeler la fonction 'OnClick' lorsque ce contrôle est sélectionné.

    GUICtrlSendToDummy($g_idUserDummy, 1) ; Set state to be checked OnclickDéfinition de l'état d'être coché sur Onclick

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW)
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Sleep(100)
    WEnd
EndFunc   ;==>Example

Func OnClick()
    Return GUICtrlSendToDummy($g_idUserDummy) ; Send a message to the dummy control that the close button was selected, which will then proceed to call the function 'OnExit'.Envoie un message au contrôle factice que le bouton de fermeture a été choisi, et que la fonction 'OnExit' sera appelée.
EndFunc   ;==>OnClick

Func OnDummy()
    If GUICtrlRead($g_idUserDummy) Then
        GUISetBkColor(0x000000FF) ; Change the background color of the GUI on dummy stateChange la couleur de fond de la GUI sur l'état factice
    Else
        Exit
    EndIf
EndFunc   ;==>OnDummy

Func OnExit()
    Exit ; Exit the script.Sortie du script.
EndFunc   ;==>OnExit
