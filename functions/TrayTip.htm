TrayTip

Displays a balloon tip from the AutoIt Icon.Affiche une info-bulle 'balloon' depuis l'icône du script AutoIt.

TrayTip ( "title", "text", timeout [, option = 0] )

ParametersParamètres

title Text appears in bold at the top of the balloon tip.Le texte qui apparait en gras en haut de l'info-bulle 'balloon'. (63 characters maximum)(63 caractères maximum)
text Message the balloon tip will display.Message qui sera affiché par l'info-bulle 'balloon'. (255 characters maximum)(255 caractères maximum)
timeout A rough estimate of the time (in seconds) the balloon tip should be displayed.Une estimation approximative du temps (en secondes) d'affichage de l'info-bulle balloon. (Windows has a min and max of about 10-30 seconds but does not always honor a time in that range.)(Windows a un mini/maxi d'environ 10-30 secondes, mais ne le respecte pas toujours.)
option [optional][optionnel]
    $TIP_ICONNONE (0) = No icon (default)
    $TIP_ICONASTERISK (1) = Info icon
    $TIP_ICONEXCLAMATION (2) = Warning icon
    $TIP_ICONHAND (3) = Error icon
    $TIP_NOSOUND (16) = Disable sound

Constants are defined in TrayConstants.au3.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

A Tray tip will close if the AutoIt tray icon disappears.L'info-bulle se fermera si l'icône du script AutoIt disparaît. Consequently, the TrayTip() may not appear if Opt("TrayIconHide", 1) is set or if the user has disabled balloon tips via registry settings!

Windows XP usually plays a sound when displaying a balloon tip.Windows XP joue habituellement un son lors de l'affichage d'une info-bulle. This sound can be disabled by adding $TIP_NOSOUND to the option parameter or by setting an empty title.Ce son peut être désactivé en ajoutant $TIP_NOSOUND au paramètre option ou en utilisant un titre vide.

To clear a balloon tip that is being displayed, call another tip with an empty text string (and any title).Pour effacer une info-bulle affiché, appelez en une autre avec une chaîne de texte vide (et n'importe quel titre).

RelatedLiens

MsgBox, ToolTip, TrayIconHide (Option), TraySetToolTip

ExampleExemple

#include <MsgBoxConstants.au3>
#include <TrayConstants.au3>

TrayTip("I'm a title", "I'm the message", 0, $TIP_ICONASTERISK)
MsgBox($MB_SYSTEMMODAL, "", "Press OK to reset the tip.")
TrayTip("clears any tray tip", "", 0)
MsgBox($MB_SYSTEMMODAL, "", "Press OK to see another tip.")
TrayTip("", "A different tray tip.", 5)
Sleep(5000)
