FileGetTime

Returns the time and date information for a file.Retourne les informations de date et heure d'un fichier.

FileGetTime ( "filename" [, option = 0 [, format = 0]] )

ParametersParamètres

filename The path to the file or directory to check.Nom du fichier à vérifier.
option [optional][optionnel][optionnel] Flag to indicate which timestampParamètre du timestamp (horodatage)
    $FT_MODIFIED (0) = Last modified (default)
    $FT_CREATED (1) = Created
    $FT_ACCESSED (2) = Last accessed

Constants are defined in FileConstants.au3Les constantes sont définies dans FileConstants.au3
format [optional][optionnel][optionnel] to specify type of returnpour spécifier le type de retour
    $FT_ARRAY (0) = return an array (default)
    $FT_STRING (1) = return a string YYYYMMDDHHMMSS

Constants are defined in FileConstants.au3Les constantes sont définies dans FileConstants.au3

Return ValueValeur de retour

Success:Succès : an array or string that contains the file time information.Retourne un tableau ou une chaîne qui contient les informations de temps du fichier. See Remarks.Voir Remarque.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

The array is a single dimension array containing six elements:Le tableau est un tableau à une dimension contenant six éléments:
    $aArray[0] = year (four digits)
    $aArray[1] = month (range 01 - 12)
    $aArray[2] = day (range 01 - 31)
    $aArray[3] = hour (range 00 - 23)
    $aArray[4] = min (range 00 - 59)
    $aArray[5] = sec (range 00 - 59)
Notice that return values are zero-padded.Notez que les valeurs de retour sont complétées avec des zéros.

ExampleExemple

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Set the modified timestamp of the file to 1st Nov 2003 and use the current time.Régle l'horodatage de modification du fichier de 1 novembre 2003 et utilise l'heure actuelle.
    Local $iFileSetTime = FileSetTime($sFilePath, "20031101", $FT_MODIFIED)

    ; Display the modified timestamp of the file and return as a string in the format YYYYMMDDHHMMSS.Affiche l'horodatage de modification du fichier et retourne une chaîne au format AAAAMMJJHHMMSS.
    If $iFileSetTime Then
        MsgBox($MB_SYSTEMMODAL, "", "Timestamp:" & @CRLF & FileGetTime($sFilePath, $FT_MODIFIED, 1))
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst setting the timestamp of the file.")
    EndIf

    ; Delete the temporary file.Supprimez le fichier temporaire.
    FileDelete($sFilePath)
EndFunc   ;==>Example
