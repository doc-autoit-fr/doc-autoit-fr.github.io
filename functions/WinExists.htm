WinExists

Checks to see if a specified window exists.Vérifie si la fenêtre indiquée existe.

WinExists ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to check.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the window to check.Le texte de la fenêtre à contrôler. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 if the window is not found.Retourne 0 si la fenêtre n'existe pas.

RemarksRemarques

WinExists() will return 1 even if the window is hidden.WinExists() retournera 1 même si la fenêtre est cachée.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    WinWait("[CLASS:Notepad]", "", 10)

    ; Test if the window exists and display the results.Teste si la fenêtre existe et affiche les résultats.
    If WinExists("[CLASS:Notepad]") Then
        MsgBox($MB_SYSTEMMODAL, "", "Window exists")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Window does not exist")
    EndIf
EndFunc   ;==>Example
