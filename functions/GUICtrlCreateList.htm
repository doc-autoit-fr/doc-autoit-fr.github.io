GUICtrlCreateList

Creates a List control for the GUI.Crée un contrôle liste dans une GUI.

GUICtrlCreateList ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

text The text of the control.Le texte de la commande.
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default ( -1) : $LBS_SORT, $WS_BORDER, $WS_VSCROLL
    forced styles : $WS_TABSTOP, $LBS_NOTIFY
exStyle [optional][optionnel] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .
    default ( -1) : $WS_EX_CLIENTEDGE

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

To obtain the value of the control see GUICtrlRead().Pour obtenir la valeur du contrôle, consultez GUICtrlRead() .
To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

The different list entries that can be selected can be set with GUICtrlSetData()Les différentes entrées de liste qui peuvent être sélectionnées peuvent être réglées avec GUICtrlSetData

To limit horizontal scrolling use GUICtrlSetLimit()Pour limiter l'utilisation de défilement horizontal GUICtrlSetLimit

To combine styles with the default style use BitOR($GUI_SS_DEFAULT_LIST, newstyle, ... ).)..
To use the values specified above you must #include <ListBoxConstants.au3> in your script.

Default resizing is $GUI_DOCKAUTO size and position will occur.Pour redimensionner par défaut, utiliez $GUI_DOCKAUTO.

RelatedLiens

ExampleExemple

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $sMESSAGE = "The following buttons have been clicked"

    GUICreate("My GUI list") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    Local $idAdd = GUICtrlCreateButton("Add", 64, 32, 75, 25)
    Local $idClear = GUICtrlCreateButton("Clear", 64, 72, 75, 25)
    Local $idMylist = GUICtrlCreateList("buttons that have been clicked", 176, 32, 121, 97)
    GUICtrlSetLimit(-1, 200) ; to limit horizontal scrollingLimite le défilement horizontal
    GUICtrlSetData(-1, $sMESSAGE)
    Local $idClose = GUICtrlCreateButton("my closing button", 64, 160, 175, 25)

    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idAdd
                GUICtrlSetData($idMylist, "You clicked button No1|")
            Case $idClear
                GUICtrlSetData($idMylist, "")
            Case $idClose
                MsgBox($MB_SYSTEMMODAL, "", "the closing button has been clicked", 2)
                Exit
        EndSwitch
    WEnd
EndFunc   ;==>Example
