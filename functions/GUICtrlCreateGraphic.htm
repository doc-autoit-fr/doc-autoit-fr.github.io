GUICtrlCreateGraphic

Creates a Graphic control for the GUI.Crée un contrôle graphique dans une GUI.

GUICtrlCreateGraphic ( left, top [, width [, height [, style]]] )

ParametersParamètres

left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default ( -1) : $SS_NOTIFY.

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

To draw in the control see GUICtrlSetGraphic().Pour dessiner dans le contrôle, Consultez GUICtrlSetGraphic.

The border and background color can be set with GUICtrlSetBkColor() and GUICtrlSetColor().La couleur du contour et la couleur de fond peuvent être définies avec GUICtrlSetBkColor() et GUICtrlSetColor().

Graphic control can be clicked so they should not overlap with other controls.Le contrôle graphique peut être cliqué donc il ne devrait pas recouvrir d'autres contrôles. If overlap is needed it is important to disable the graphic control :Si le recouvrement est nécessaire, il est important de désactiver le contrôle graphique: GUICtrlSetState(-1, $GUI_DISABLE).Ce n'est pas suffisant pour l'onglet ou le contrôle ListView qui se comportent différemment.Dans ce cas, vous avez besoin de créer l'image avec le style $WS_CLIPSIBLINGS, GuiCtrlSetState(-1, $GUI_ONTOP) est nécessaire pour l'onglet, TreeView ou contrôle ListView.

This control cannot be resized due to fix graphic relative addressing creation but can be dock with GUICtrlSetResizing().Ce contrôle ne peut pas être redimensionné en raison d'une correction relative à l'adressage lors de sa création, mais il peut être attaché avec GUICtrlSetResizing().

#include <GUIConstantsEx.au3>

Global Const $g_MAXGr = 6
Global $g_a_idArray[$g_MAXGr + 1] ; 0 and $g_MAXGr entries not used to allow GUICtrlDelete result0 et les entrées $g_MAXGr ne sont pas utilisés pour permettre GUICtrlDelete
Global $g_idDel

Example()

Func Example()
    Local $idMsg, $iInc, $i

    CreateChild()

    $i = 1
    $iInc = 1
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
        $idMsg = GUIGetMsg()

        If $idMsg = $g_idDel Then
            GUICtrlDelete($g_a_idArray[$i])
            $i = $i + $iInc
            If $i < 0 Or $i > $g_MAXGr Then Exit
        EndIf
        If $idMsg > 0 Then MsgBox($MB_SYSTEMMODAL, "clicked", $idMsg & @CRLF & $g_a_idArray[5], 2)
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
