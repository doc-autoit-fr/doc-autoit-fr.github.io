GUICtrlCreateCheckbox

Creates a Checkbox control for the GUI.Crée un contrôle Chekbox (cases à cocher) dans une GUI.

GUICtrlCreateCheckbox ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètre

text The text of the control checkbox.Le texte du contrôle de la case à cocher
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optional] False - i, v, x ... The width of the control (default text autofit in width).La largeur du contrôle (par défaut ajustement automatique du texte en largeur ).
height [optional][optional] False - i, v, x ... The height of the control (default text autofit in height).La hauteur du contrôle (par défaut ajustement automatique du texte en hauteur ).
style [optional][optional] False - i, v, x ... Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default ( -1) : $BS_AUTOCHECKBOX.
    forced styles : $WS_TABSTOP, and $BS_AUTOCHECKBOX if no checkbox style defined.$WS_TABSTOP, et $BS_AUTOCHECKBOX si aucun styles chekkbox n'est défini.
exStyle [optional][optional] False - i, v, x ... Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .

Return ValueRemarque

Success: the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure: 0.Retourne 0.

Remarks<i0>

To obtain the value of the control see GUICtrlRead().Pour obtenir la valeur du contrôle, consultez GUICtrlRead() .
To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

A Checkbox control can display an icon or image by using the $BS_ICON or $BS_BITMAP style.Un contrôle de case à cocher permet d'afficher une icône ou une image en utilisant le style $BS_ICON ou $BS_BITMAP. Use GUICtrlSetImage() to specify the picture to use.Utilisez GUICtrlSetImage pour spécifier l'image à utiliser.

To combine styles with the default style use BitOR ( $GUI_SS_DEFAULT_CHECKBOX, newstyle, ... ).)..

To use the values specified above you must #include <ButtonConstants.au3> in your script.

Default resizing is $GUI_DOCKHEIGHT.Par défaut redimensionnement est $ GUI_DOCKHEIGHT.

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Create a GUI with various controls.Crée une GUI avec différents contrôles.
    Local $hGUI = GUICreate("Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>", 300, 200)

    ; Create a checkbox control.Crée un contrôle Checkbox (cases à cocher).
    Local $idCheckbox = GUICtrlCreateCheckbox("Standard Checkbox", 10, 10, 185, 25)
    Local $idClose = GUICtrlCreateButton("Close", 210, 170, 85, 25)

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Loop until the user exits.Création d'un objet chemin
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idClose
                ExitLoop

            Case $idCheckbox
                If _IsChecked($idCheckbox) Then
                    MsgBox($MB_SYSTEMMODAL, "", "The checkbox is checked.", 0, $hGUI)
                Else
                    MsgBox($MB_SYSTEMMODAL, "", "The checkbox is not checked.", 0, $hGUI)
                EndIf

        EndSwitch
    WEnd

    ; Delete the previous GUI and all controls.Supprime la GUI précédente et tous ses contrôles.
    GUIDelete($hGUI)
EndFunc   ;==>Example

Func _IsChecked($idControlID)
    Return BitAND(GUICtrlRead($idControlID), $GUI_CHECKED) = $GUI_CHECKED
EndFunc   ;==>_IsChecked
