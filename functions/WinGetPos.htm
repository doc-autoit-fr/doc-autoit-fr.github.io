WinGetPos

Retrieves the position and size of a given window.Récupère la position et la taille de la fenêtre indiquée.

WinGetPos ( "title" [, "text"] )

ParametersParamètre

title The title/hWnd/class of the window to get the position/size.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optional] False - i, v, x ... The text of the window to get the position/size.Le texte de la fenêtre à traiter. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueRemarque

Success: a 4-element array containing the following information:Retourne un tableau de 4 éléments contenant les informations suivantes :
    $aArray[0] = X position
    $aArray[1] = Y position
    $aArray[2] = Width
    $aArray[3] = Height
Failure: sets the @error flag to non-zero if the window is not found.Retourne @error <> 0 si la fenêtre n'est pas trouvée.

Remarks<i0>

WinGetPos() returns negative numbers such as -32000 for minimized windows, but works fine with (non-minimized) hidden windows.WinGetPos() retourne des nombres négatifs tels que -32000 pour les fenêtres minimisées, mais fonctionne très bien avec des fenêtres masquées (non réduites).
If the window title "Program Manager" is used, the function will return the size of the desktop.Si le titre de la fenêtre "Program Manager» est utilisé, la fonction retourne la taille du bureau. If multiple windows match the criteria, the most recently active window is used.Si plusieurs fenêtres correspondent aux critères, la fenêtre active la plus récente sera utilisé.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Retrieve the position as well as height and width of the active window.Récupére la position ainsi que la hauteur et la largeur de la fenêtre active.
    Local $aPos = WinGetPos("[ACTIVE]")

    ; Display the array values returned by WinGetPos.Affiche les valeurs de tableau retourné par WinGetPos.
    MsgBox($MB_SYSTEMMODAL, "", "X-Pos: " & $aPos[0] & @CRLF & _
            "Y-Pos: " & $aPos[1] & @CRLF & _
            "Width: " & $aPos[2] & @CRLF & _
            "Height: " & $aPos[3])
EndFunc   ;==>Example
