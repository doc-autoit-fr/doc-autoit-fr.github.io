IsHWnd

Checks if a variable's base type is a pointer and window handle.Vérifie si le type de base d'une variable est un pointeur et un handle de fenêtre.

IsHWnd ( variable )

ParametersParamètres

variable The variable/expression to check.La variable/expression à vérifier.

Return ValueValeur de retour

Success:Succès : 1 if the expression is a pointer type AND a valid window handle.Retourne 1 si l'expression est un type de pointeur et un handle fenêtre valide.
Failure:Échec : 0 if expression is not a pointer OR not a valid window handle.Retourne 0 si l'expression n'est pas un pointeur OU n'est pas un handle fenêtre valide.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Test if the window was found and display the results.Teste si la fenêtre a été trouvé et affiche les résultats.
    If IsHWnd($hWnd) Then
        MsgBox($MB_SYSTEMMODAL, "", "It's a valid HWND")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "It's not a valid HWND")
    EndIf
EndFunc   ;==>Example
