DriveGetFileSystem

Returns File System Type of a drive.Retourne le type de système de fichiers d'un disque.

DriveGetFileSystem ( "path" )

ParametersParamètres

path Path of drive to receive information from.Le chemin sur lequel agir.

Return ValueValeur de retour

Success:Succès : the File System Type of the drive as a string; see table below.Retourne le type de système de fichiers du lecteur sous la forme d'une chaîne; voir le tableau ci-dessous.
Failure:Échec : sets the @error flag to 1.Retourne 0 et @error = 1.



Return ValueValeur de retour InterpretationInterprétation
$DT_UNDEFINED (1) (numeric) Drive does NOT contain media (CD, Floppy, Zip) or media is unformatted (RAW).Le lecteur ne contient pas de média (CD, Disquette, Zip) ou le média n'est pas formaté (RAW).
$DT_FAT ("FAT") Typical file system for drives under ~500 MB such as Floppy, RAM disks, USB "pen" drives, etc.Système de fichier typique des disques inférieurs à ~ 500 Mo tels que les disquettes, disques RAM, lecteurs USB "pen", etc
$DT_FAT32 ("FAT32") Typical file system for Windows 9x/Me hard drives.Système de fichier typique des diques durs Windows 9x/Me
$DT_EXFAT ("exFAT") Typical file system for drives with files over 4 GB such as RAM disks, USB "pen" drives, etc.
$DT_NTFS ("NTFS") Typical file system for Windows hard drives.Système de fichier typique des diques durs Windows NT/2000/XP
$DT_UNWFS ("NWFS") Typical file system for Novell Netware file servers.Système de fichier typique des serveurs de fichiers Novell.
$DT_CDFS ("CDFS") Typically indicates a CD (or an ISO image mounted as a virtual CD drive).Indique généralement un CD (ou une image ISO montée comme un lecteur de CD virtuel).
$DT_UDF ("UDF") Typically indicates a DVD.Indique généralement un DVD.

Constants are defined in "AutoItConstants.au3".

RemarksRemarques

The list of possible return values might be incomplete.La liste des valeurs de retour possibles est peut-être incomplète.

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sFileSystem = DriveGetFileSystem(@HomeDrive & "\") ; Find the file system type of the home drive, generally this is the C:\ drive.Trouver le type de lecteur de base du système de fichiers, en général c'est le lecteur C:\\.
MsgBox($MB_SYSTEMMODAL, "", "File System Type: " & $sFileSystem)
