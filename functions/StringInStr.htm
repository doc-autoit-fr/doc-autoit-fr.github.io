StringInStr

Checks if a string contains a given substring.Vérifie si une chaîne contient une sous-chaîne donnée.

StringInStr ( "string", "substring" [, casesense = 0 [, occurrence = 1 [, start = 1 [, count]]]] )

ParametersParamètres

string The string to evaluate.La chaîne à traiter.
substring The substring to search for.La sous-chaîne à rechercher.
casesense [optional][optionnel] Flag to indicate if the operations should be case sensitive.Paramètre pour indiquer si les opérations doivent être sensibles à la casse.
    $STR_NOCASESENSE (0) = not case sensitive, using the user's locale (default)
    $STR_CASESENSE (1) = case sensitive
    $STR_NOCASESENSEBASIC (2) = not case sensitive, using a basic/faster comparison

Constants are defined in StringConstants.au3.
occurrence [optional][optionnel] Which occurrence of the substring to find in the string.Quelle occurrence de la sous-chaîne à trouver dans la chaîne. Use a negative occurrence to search from the right side.Utiliser une occurrence négative pour rechercher à partir de la droite de la chaîne. The default value is 1 (finds first occurrence).La valeur par défaut est 1 (trouve la première occurrence).
start [optional][optionnel] The starting position of the search.La position de départ de la recherche.
count [optional][optionnel] The number of characters to search.Le nombre de caractères à traiter. This effectively limits the search to a portion of the full string.Ceci limite la recherche à une partie de la chaîne complète . See remarks.Voir remarques.

Return ValueValeur de retour

Success:Succès : the position of the substring.Retourne la position de la sous-chaîne.
Failure:Échec : 0 if substring not found.Retourne 0 si la sous-chaîne n'est pas trouvée.
@error:@error : 1 = Invalid "start" or "occurrence" parameter given.1 = Paramètres 'start' ou 'occurrence' invalide.

RemarksRemarques

The first character position is 1.La position du premier caractère est 1.

The count parameter must not be shorter than the substring being searched for.Le paramètre 'count' doit être plus long que la sous-chaîne recherchée . The count parameter (combined with the start parameter) effectively limits the search to a substring of the full string.Le paramètre 'count' (combiné avec le paramètre 'start') limite efficacement la recherche à une sous-chaîne de la chaîne complète. The following two statements are equivalent:Les instructions suivantes sont équivalentes:
StringInStr("the string to search", "string", 0, 1, 1, 11)StringInStr("La chaîne à chercher", "chaîne", 0, 1, 1, 11)
StringInStr(StringMid("the string to search", 1, 11), "string")StringInStr(StringMid("La chaîne à chercher", 1, 11), "chaîne")

ExampleExemple

#include <MsgBoxConstants.au3>

; Retrieve the character position of where the string 'white' first occurs in the sentence.Récupére la position du caractère à l'endroit où la chaîne 'white' apparaît la première fois dans la phrase.
Local $iPosition = StringInStr("This is a sentence with whitespace.", "white")
MsgBox($MB_SYSTEMMODAL, "", "The search string 'white' first appears at position: " & $iPosition)
