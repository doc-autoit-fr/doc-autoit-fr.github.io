TrayItemSetOnEvent

Defines a user-defined function to be called when a tray item is clicked.Définit une fonction utilisateur qui est appelée quand un item de la zone de notification est cliqué.

TrayItemSetOnEvent ( itemID, "function" )

ParametersParamètres

itemID The item identifier (itemID) as returned by a TrayCreateItem function.L'identifiant de l'item (itemID) tel que retourné par la fonction TrayCreateItem.
function The name of the user function to call.Le nom de la fonction utilisateur à appeler.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0.
@error:@error : 1 = the "function" is not defined.1 = la "fonction" n'est pas définie.

RemarksRemarques

OnEvent functions are only called when the option Opt("TrayOnEventMode", 1) - when in this mode TrayGetMsg() is NOT used at all.

Within the called user function the item identifier can be retrieved with @TRAY_ID.Dans la fonction utilisateur appelée l'identifiant de l'item peut être récupéré avec TRAY_ID.

If the function is an empty string "" the previous user-defined is disabled.Si la fonction est une chaîne vide "" la précédente fonction utilisateur est désactivée.

Example()

Func Example()
    TrayCreateItem("About")
    TrayItemSetOnEvent(-1, "About")

    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    TrayCreateItem("Exit")
    TrayItemSetOnEvent(-1, "ExitScript")

    TraySetOnEvent($TRAY_EVENT_PRIMARYDOUBLE, "About") ; Display the About MsgBox when the tray icon is double clicked on with the primary mouse button.Affiche la MsgBox Aboutlorsque l'icône de la zone de notification est double-cliquée par le bouton principal de la souris.

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Sleep(100) ; An idle loop.Une boucle d'inactivité.
    WEnd
EndFunc   ;==>Example

Func About()
    ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
    MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
            "Version: " & @AutoItVersion & @CRLF & _
            "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.
EndFunc   ;==>About
