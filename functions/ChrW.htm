ChrW

Returns a character corresponding to a unicode code.Retourne un caractère correspondant à un code unicode.

ChrW ( UNICODEcode )

ParametersParamètres

UNICODEcode A unicode code in the range 0-65535 (e.g., 65 returns the capital letter A).

Return ValueValeur de retour

Success:Succès : a string containing the representation of the given code.Retourne une chaîne correspondant à la représentation du code donné.
Failure:Échec : an empty string and sets the @error flag to non-zero if the UNICODE value is greater than 65535.Retourne une chaîne vide et @error = 1 si le code est plus grand que 65535.

RemarksRemarques

See the ASCII Character Code table for a complete list of available values.Consultez la table des Codes de caractères ASCII pour une liste complète des valeurs possibles.

RelatedLiens

Asc, AscW, Chr, String

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sText = ""
For $i = 256 To 512
    $sText = $sText & ChrW($i) ; Or $sText &= ChrW($i) can be used as well.Ou $sText &= ChrW($i) peut être utilisé.
Next
MsgBox($MB_SYSTEMMODAL, "Unicode chars 256 to 512", $sText) ; Display the unicode characters between 256 to 512.Affiche les caractères Unicode entre 256-512.
