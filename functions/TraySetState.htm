TraySetState

Sets the state of the tray icon.Définit l'état de l'icône de la zone de notification.

TraySetState ( [flag = 1] )

ParametersParamètres

flag [optional][optionnel] A combination of the following:Une combinaison des éléments suivants:
    $TRAY_ICONSTATE_SHOW (1) = Shows the tray icon (default)
    $TRAY_ICONSTATE_HIDE (2) = Destroys/Hides the tray icon
    $TRAY_ICONSTATE_FLASH (4) = Flashes the tray icon
    $TRAY_ICONSTATE_STOPFLASH (8) = Stops tray icon flashing
    $TRAY_ICONSTATE_RESET (16) = Resets the icon to the defaults (no flashing, default tip text)

Constants are defined in TrayConstants.au3.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

This function overrides Opt("TrayIconHide") and #NoTrayIcon.
The normal and pause tray icon are NOT reset by this function!Les icônes normal et pause de la zone de notification ne sont PAS réinitialisées par cette fonction!

Example()

Func Example()
    Local $idFlash = TrayCreateItem("Flash Icon")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

            Case $idFlash
                ; Start flashing the tray icon.Clignoter l'icône de la barre.
                TraySetState($TRAY_ICONSTATE_FLASH)
                Sleep(5000) ; Wait for 5 seconds.Attendez 5 secondes.
                ; Stop flashing the tray icon.Arrête le clignotement de l'icône.
                TraySetState($TRAY_ICONSTATE_STOPFLASH)

            Case $idExit ; Exit the loop.Sort de la boucle.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
