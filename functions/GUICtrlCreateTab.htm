GUICtrlCreateTab

Creates a Tab control for the GUI.Crée un contrôle Tab (avec onglet) dans une GUI.

GUICtrlCreateTab ( left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default ( -1) : none.
    forced styles : $TCS_TOOLTIPS, $WS_TABSTOP, $WS_CLIPSIBLINGS
exStyle [optional][optionnel] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

A tab control is a control which can only contain tabitem controls.Un contrôle onglet est un contrôle qui peut seulement contenir des contrôles TabItem. Any other controls should be created within these tabitems using GUICtrlCreate... functions as shown in the example.Tous les autres contrôles devraient être créés dans ces TabItems à l'aide des fonctions GUICtrlCreate...comme dans l'exemple. It is important to close the tab structure by creating a final tabitem control with a null text - GUICtrlCreateTabItem("").Il est important de fermer la structure Tab en créant un contrôle TabItem finale avec un texte null - GUICtrlCreateTabItem("").

To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

To combine styles with the default style use BitOR($GUI_SS_DEFAULT_TAB, newstyle, ...Pour combiner des styles avec le style par défaut, utilisez BitOR($ GUI_SS_DEFAULT_TAB, newstyle,... ).)..
To use the default value add #include <TabConstants.au3> in your script.

Default resizing is $GUI_DOCKSIZE.Par défaut redimensionnement est $ GUI_DOCKSIZE.

A GUI can only hold a single tab control, but it is possible to create child GUIs each holding a tab control.Une GUI peut contenir un seul contrôle Tab, mais il est possible de créer des GUI enfants contenant chacun un contrôle Tab.

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI Tab") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUISetBkColor(0x00E0FFFF)
    GUISetFont(9, 300)

    GUICtrlCreateTab(10, 10, 200, 100)

    GUICtrlCreateTabItem("tab0")
    GUICtrlCreateLabel("label0", 30, 80, 50, 20)
    GUICtrlCreateButton("OK0", 20, 50, 50, 20)
    GUICtrlCreateInput("default", 80, 50, 70, 20)

    GUICtrlCreateTabItem("tab----1")
    GUICtrlCreateLabel("label1", 30, 80, 50, 20)
    GUICtrlCreateCombo("", 20, 50, 60, 120)
    GUICtrlSetData(-1, "Trids|CyberSlug|Larry|Jon|Tylo|guinness", "Jon") ; default JonJon défaut
    GUICtrlCreateButton("OK1", 80, 50, 50, 20)

    GUICtrlCreateTabItem("tab2")
    GUICtrlSetState(-1, $GUI_SHOW) ; will be display firsts'affichera en premier
    GUICtrlCreateLabel("label2", 30, 80, 50, 20)
    GUICtrlCreateButton("OK2", 140, 50, 50)

    GUICtrlCreateTabItem("") ; end tabitem definitionfin de la définition de TabItem

    GUICtrlCreateLabel("label3", 20, 130, 50, 20)

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        $idMsg = GUIGetMsg()

        If $idMsg = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd
EndFunc   ;==>Example
