InetClose

Closes a handle returned from InetGet().Ferme un handle retourné par InetGet().

InetClose ( handle )

ParametersParamètres

handle A handle return from InetGet().Un handle retourné par InetGet().

Return ValueValeur de retour

True: if the handle was found and closed.le handle a été trouvé et fermé.
False: if not.sinon.

RemarksRemarques

InetGet() handles must be closed or resources will leak.Les handles d'InetGet() doivent être fermés sinon, vous aurez une fuite de ressources.

Closing the handle to a download still in progress will cancel the download.Fermer le handle d'un téléchargement en cours annulera le téléchargement.

ExampleExemple

#include <InetConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Save the downloaded file to the temporary folder.Enregistre le fichier téléchargé dans le dossier temporaire.
    Local $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Download the file in the background with the selected option of 'force a reload from the remote site.'Télécharge le fichier en arrière-plan avec l'option choisie de 'forcer le chargement du site distant'.
    Local $hDownload = InetGet("http://www.autoitscript.com/autoit3/files/beta/update.dat", $sFilePath, $INET_FORCERELOAD, $INET_DOWNLOADBACKGROUND)

    ; Wait for the download to complete by monitoring when the 2nd index value of InetGetInfo returns True.Attend la fin du téléchargement pour vérifier si la valeur de l'indice de 2e InetGetInfo renvoie True.
    Do
        Sleep(250)
    Until InetGetInfo($hDownload, $INET_DOWNLOADCOMPLETE)

    ; Retrieve the number of total bytes received and the filesize.Récupére le nombre total d'octets reçus et la taille du fichier.
    Local $iBytesSize = InetGetInfo($hDownload, $INET_DOWNLOADREAD)
    Local $iFileSize = FileGetSize($sFilePath)

    ; Close the handle returned by InetGet.Ferme le handle renvoyé par InetGet.
    InetClose($hDownload)

    ; Display details about the total number of bytes read and the filesize.Affiche les détails sur le nombre total d'octets lus et la taille du fichier.
    MsgBox($MB_SYSTEMMODAL, "", "The total download size: " & $iBytesSize & @CRLF & _
            "The total filesize: " & $iFileSize)

    ; Delete the file.Supprime le fichier.
    FileDelete($sFilePath)
EndFunc   ;==>Example
