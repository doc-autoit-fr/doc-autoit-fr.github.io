Mod

Performs the modulus operation.Effectue l'opération modulo (reste d'une division).

Mod ( value1, value2 )

ParametersParamètres

value1 The dividend.Le dividende.
value2 The divisor.Le diviseur.

Return ValueValeur de retour

Success:Succès : the remainder when value1 is divided by value2.Retourne le reste lorsque valeur1 est divisé par valeur2.
Failure:Échec : -1.#IND if the divisor is zero.Retourne -1.#IND (indéterminé) si le diviseur est zéro.

RemarksRemarques

This function guarantees that Mod(dividend, divisor) = dividend - Int(dividend / divisor) * divisor.Cette fonction garantit que Mod(dividend, divisor) = dividend - Int ( dividende/diviseur) * diviseur.

This function does not guarantee that dividend or divisor can be represented, specifically with floating point numbers.Cette fonction ne garantit pas que le dividende ou le diviseur peut être représentée, en particulier avec des nombres à virgule flottante.

If integers are passed this function does an integral modulo operation.Si des 'integer' sont passés, cette fonction fait une opération modulo. Otherwise it falls back to a floating point operation which per the previous remark means it may not produce the expected output.Sinon, l'opération revient à une opération en virgule flottante qui par la remarque précédente signifie qu'elle ne peut pas produire le résultat attendu.

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local variable the number 18.Assigne une variable locale le nombre 18.
    Local $iNb = 18

    ; If the mudulus operation of $iNb by 2 equals to 0 then $iNb is even.Si l'opération modulo de $INB par 2 est égal à 0, alors $INB est pair.
    If Mod($iNb, 2) = 0 Then
        MsgBox($MB_SYSTEMMODAL, "", $iNb & " is an even number.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", $iNb & " is an odd number.")
    EndIf

    ; Assign a Local variable the modulus operation of 4 by 7.Assigne une variable locale avec l'opération modulo de 4 par 7.
    Local $iMod1 = Mod(4, 7) ; 4%7 = 4 because the divisor > dividend4 % 7 = 4 car le diviseur > dividende

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iMod1)

    ; Assign a Local variable the modulus operation of 1 by 3/4.Assigne une variable locale avec l'opération modulo de 1 par 3/4 par.
    Local $iMod2 = Mod(1, 3 / 4) ; 1%(3/4) = 0.25 because the divisor is a float1 % (3/4) = 0,25, car le diviseur est un flotteur

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iMod2)
EndFunc   ;==>Example
