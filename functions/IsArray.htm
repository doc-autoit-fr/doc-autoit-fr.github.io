IsArray

Checks if a variable is an array type.Vérifie si une variable est de type tableau.

IsArray ( variable )

ParametersParamètres

variable The variable/expression to check.La variable/expression à vérifier.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 if parameter is not an array variable.Retourne 0 si le paramètre n'est pas une variable tableau.

RemarksRemarques

Can be useful to validate array/non-array parameters to user-defined functions.Peut être utile afin de vérifier dans une librairie de fonctions (UDF) si un paramètre est de type tableau.

See language datatypes for a detailed description.Consultez Types de données du langage pour une description détaillée.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Retrieve the position and size of the Notepad window by passing the handle to WinGetPos.Récupére la position et la taille de la fenêtre du Bloc-notes en passant Le handle de WinGetPos.
    Local $aPos = WinGetPos($hWnd)

    ; Check if the variable is an array.Vérifie si la variable est un tableau.
    If IsArray($aPos) Then
        MsgBox($MB_SYSTEMMODAL, "", "Window height: " & $aPos[3])
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred.")
    EndIf

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
