Run

Runs an external program.Exécute un programme externe.

Run ( "program" [, "workingdir" [, show_flag [, opt_flag]]] )

ParametersParamètres

program The full path of the program (EXE, BAT, COM, or PIF) to run (see remarks).Le chemin d'accès complet du programme (EXE, BAT, COM, ou PIF) pour exécuter (Voir Remarques).
workingdir [optional][optionnel][optionnel] The working directory.Le répertoire de travail. Blank ("") uses the current working directory.Si vide ("") utilise le répertoire de travail courant. This is not the path to the program.Ce n'est pas le chemin du programme.
show_flag [optional][optionnel][optionnel] The "show" flag of the executed program:L'option de visualisation du programme exécuté:
    @SW_HIDE = Hidden window (or Default keyword)
    @SW_MINIMIZE = Minimized window
    @SW_MAXIMIZE = Maximized window
opt_flag [optional][optionnel][optionnel] Controls various options related to how the parent and child process interact.Options diverses liées à la façon dont les processus parent-enfant inter-agissent.
    $STDIN_CHILD (0x1) = Provide a handle to the child's STDIN stream
    $STDOUT_CHILD (0x2) = Provide a handle to the child's STDOUT stream
    $STDERR_CHILD (0x4) = Provide a handle to the child's STDERR stream
    $STDERR_MERGED (0x8) = Provides the same handle for STDOUT and STDERR. Implies both $STDOUT_CHILD and $STDERR_CHILD.Implique à la fois $STDOUT_CHILD et $STDERR_CHILD.
    $STDIO_INHERIT_PARENT (0x10) = Provide the child with the parent's STDIO streams. This flag can not be combined with any other STDIO flag.Ce drapeau ne peut être associé à aucun autre drapeau STDIO. This flag is only useful when the parent is compiled as a Console application.Ce drapeau n'est utile que lorsque le parent est compilé comme une application console.
    $RUN_CREATE_NEW_CONSOLE (0x10000) = The child console process should be created with it's own window instead of using the parent's window. This flag is only useful when the parent is compiled as a Console application.Ce drapeau n'est utile que lorsque le parent est compilé comme une application console.

Constants are defined in AutoItConstants.au3.

Return ValueValeur de retour

Success:Succès : the PID of the process that was launched.Retourne le PID du processus qui a été lancé.
Failure:Échec : 0 and sets the @error flag to non-zero.0 et @error <> 0.

RemarksRemarques

Paths with spaces need to be enclosed in quotation marks.Les chemins avec des espaces doivent être placés entre guillemets.

To run DOS (console) commands, try Run(@ComSpec & " /c " & 'commandName', "", @SW_HIDE)    ; don't forget " " before "/c"

After running the requested program the script continues.Après avoir exécuté le programme demandé, le script continue. To pause execution of the script until the spawned program has finished use the RunWait() function instead.Pour mettre en pause l'exécution du script jusqu'à ce que le programme soit terminé, utilisez la fonction RunWait() à la place.

Providing the Standard I/O parameter with the proper values permits interaction with the child process through the StderrRead(), StdinWrite() and StdoutRead() functions.Fournir le paramètre Standard I/O avec les valeurs propres permet une interaction avec le processus de l'enfant à travers les fonctions StderrRead(), StdinWrite() et StdoutRead(). Combine the flag values (or use $STDERR_CHILD, $STDIN_CHILD & $STDOUT_CHILD) to manage more than one stream.

In order for the streams to close, the following conditions must be met:Pour les flux de fermeture, les conditions suivantes doivent être remplies:
1) The child process has closed its end of the stream (this happens when the child closes).1) Le processus enfant a fermé la fin de son flux (ce qui arrive lorsque l'enfant ferme).
2) AutoIt must read any captured streams until there is no more data.2) AutoIt doit lire tous les flux capturés jusqu'à il n'y a pas plus de données.
3) If STDIN is provided for the child, StdinWrite() must be called to close the stream.3) Si STDIN est fourni pour l'enfant, StdinWrite() doit être appelée pour fermer le flux. Once all streams are detected as no longer needed, all internal resources will automatically be freed.Une fois que tous les flux sont détectés comme n'étant plus nécessaires, toutes les ressources internes seront automatiquement libérés.

StdioClose() can be used to force the STDIO streams closed.StdioClose() peut être utilisé pour forcer le flux STDIO à se fermer.

ExampleExemple

Example()

Func Example()
    ; Run Notepad with the window maximized.Exécute le Bloc-notes avec la fenêtre agrandie.
    Local $iPID = Run("notepad.exe", "", @SW_SHOWMAXIMIZED)

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    WinWait("[CLASS:Notepad]", "", 10)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Close the Notepad process using the PID returned by Run.Ferme le processus Notepad en utilisant le PID renvoyé par Run.
    ProcessClose($iPID)
EndFunc   ;==>Example
