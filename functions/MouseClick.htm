Référence Fonction

MouseClick

Perform a mouse click operation.Effectue une opération de clic de souris.

MouseClick ( "button" [, x, y [, clicks = 1 [, speed = 10]]] )

ParametersParamètres

button The button to click:Le bouton à cliquer :
    $MOUSE_CLICK_RIGHT ("right")
    $MOUSE_CLICK_MIDDLE ("middle")
    $MOUSE_CLICK_MAIN ("main")
    $MOUSE_CLICK_MENU ("menu")
    $MOUSE_CLICK_PRIMARY ("primary")
    $MOUSE_CLICK_SECONDARY ("secondary")

Constants are defined in "AutoItConstants.au3".
x, y [optional] The x/y coordinates to move the mouse to.Les coordonnées x,y où déplacer la souris. If no x and y coords are given, the current position is used (default).Si aucune coordonnée n'est donné, la position actuelle est utilisée (par défaut).
clicks [optional] The number of times to click the mouse.Le nombre de clics de souris. Default is 1.Par défaut c'est 1.
speed [optional] the speed to move the mouse in the range 1 (fastest) to 100 (slowest).la vitesse de déplacement de la souris dans la plage 1 (le plus rapide) à 100 (le plus lent). A speed of 0 will move the mouse instantly.Une vitesse de 0 déplace la souris instantanément. Default speed is 10.La vitesse par défaut est 10.

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0, the button is not in the list or invalid parameter as x without y.Retourne 0, le bouton n'est pas dans la liste ou un paramètre est invalide comme x sans y.

RemarksRemarques

If the button is an empty string, the left button will be clicked.
If x or y equal to the Default keyword no move occurs in the corresponding coordinate.Si x ou y est égal au mot-clé Default aucun mouvement ne se produit dans la coordonnée correspondante.

If the user has swapped the left and right mouse buttons in the control panel, then the behaviour of the buttons is different.Si l'utilisateur a permuté les boutons gauche et droite de la souris dans le panneau de configuration, alors le comportement de la fonction est différent. "Left" and "right" always click those buttons, whether the buttons are swapped or not."Left" et "right" cliquent respectivement sur les boutons gauche et droite, qu'ils soient inversés ou non. The "primary" or "main" button will be the main click, whether or not the buttons are swapped.Le bouton "primary" ou "main" sera le clic principal, que les boutons soient inversés ou non. The "secondary" or "menu" buttons will usually bring up the context menu, whether the buttons are swapped or not.Le bouton "secondary" ou "menu" fera généralement apparaître le menu contextuel, que les boutons soiennt inversés ou non.

ExampleExemple

#include <AutoItConstants.au3>

; Double click at the current mouse position.Double-clique sur la position actuelle de la souris.
MouseClick($MOUSE_CLICK_LEFT)
MouseClick($MOUSE_CLICK_LEFT)

; Double click at the x, y position of 0, 500.Double-clique sur la position (x, y) = (0, 500).
MouseClick($MOUSE_CLICK_LEFT, 0, 500, 2)

; Double click at the x, y position of 0, 500.Double-clique sur la position (x, y) = (0, 500). This is a better approach as it takes into account left/right handed users.C'est une meilleure approche car elle prend en compte gaucher/droitier.
MouseClick($MOUSE_CLICK_PRIMARY, 0, 500, 2)
