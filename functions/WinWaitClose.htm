WinWaitClose

Pauses execution of the script until the requested window does not exist.Met le script en pause jusqu'à ce que la fenêtre indiquée n'existe plus.

WinWaitClose ( "title" [, "text" [, timeout = 0]] )

ParametersParamètres

title The title/hWnd/class of the window to check.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the window to check.Le texte de la fenêtre à contrôler. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .
timeout [optional][optionnel][optionnel] Timeout in seconds if the window is not closed.Temporisation en secondes si la fenêtre n'est pas fermée. Default is 0 (no timeout).La valeur par défaut est 0 (pas de temporisation).

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 if timeout occurred.Retourne 0 si la temporisation est dépassée.

RemarksRemarques

If the window already doesn't exist when this function is called it will return 1 immediately.Si la fenêtre n'existe déjà plus lorsque cette fonction est appelée, elle retournera 1 immédiatement. The window is polled every 250 milliseconds or so.La fenêtre est interrogée toutes les 250 millisecondes ou plus.

RelatedLiens

ProcessWaitClose, WinActive, WinExists, WinWait, WinWaitActive, WinWaitDelay (Option), WinWaitNotActive

ExampleExemple

; Wait for the window "[CLASS:Notepad]" to not exist.Attend que la fenêtre "[CLASS:Notepad]" n'existe plus.
WinWaitClose("[CLASS:Notepad]")

; Wait a maximum of 5 seconds for "[CLASS:Notepad]" to not exist.Attend un maximum de 5 secondes pour que "[CLASS:Notepad]" n'existe plus.
WinWaitClose("[CLASS:Notepad]", "", 5)
