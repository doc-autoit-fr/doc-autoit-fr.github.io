TCPStartup, UDPStartup

Starts TCP or UDP services.Démarre les services TCP ou UDP.

TCPStartup ( )
UDPStartup ( )

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0 :
@error: Windows API WSAStartup return value (see MSDN).

Remarks<i0>

There must be a TCPShutdown() call to avoid memory consumption.Il doit y avoir un appel à TCPShutdown() pour éviter une consommation de mémoire.
A script must call one TCPShutdown() call for every successful TCPStartup() call.Un script doit appeler un TCPShutdown() pour chaque appel réussi à un TCPStartup().
UDPStartup() is just an alias of TCPStartup().UDPStartup() n'est qu'un alias de TCPStartup().
