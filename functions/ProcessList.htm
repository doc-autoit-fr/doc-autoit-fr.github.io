ProcessList

Returns an array listing the currently running processes (names and PIDs).Retourne un tableau listant les processus en cours d'exécution (noms et PIDs).

ProcessList ( ["name"] )

ParametersParamètres

name [optional][optionnel] If a name is given only processes of the same name will be returned.Si un nom est donné, seul les processus de même nom seront retournés.

Return ValueValeur de retour

Success:Succès : an array of process names and PIDs (See Remarks).Retourne le tableau des noms de processus et leur PID (Voir Remarques).
Failure:Échec : sets the @error flag to 1 when the array cannot be built.Retourne @error = 1 lorsque la liste ne peut pas être construite.

RemarksRemarques

The array returned is two-dimensional and is made up as follows:Le tableau retourné est un tableau à deux dimensions composé comme ceci :
    $aArray[0][0] = Number of processes
    $aArray[1][0] = 1st Process name
    $aArray[1][1] = 1st Process ID (PID)
    $aArray[2][0] = 2nd Process name
    $aArray[2][1] = 2nd Process ID (PID)
    ...
    $aArray[n][0] = nth Process name
    $aArray[n][1] = nth Process ID (PID)

The list can be empty if $aArray[0][0] = 0.La liste peut être vide si $array[0][0] = 0. No @error set in this case.Dans ce cas aucune @error n'est déterminée.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Display a list of Notepad processes returned by ProcessList.Affiche la liste des processus Bloc-notes renvoyées par PROCESSLIST.
    Local $aProcessList = ProcessList("notepad.exe")
    For $i = 1 To $aProcessList[0][0]
        MsgBox($MB_SYSTEMMODAL, "", $aProcessList[$i][0] & @CRLF & "PID: " & $aProcessList[$i][1])
    Next

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
