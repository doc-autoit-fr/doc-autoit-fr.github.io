GUICtrlCreateUpdown

Creates an UpDown control for the GUI.Crée un contrôle UpDown dans une GUI.

GUICtrlCreateUpdown ( inputcontrolID [, style = -1] )

ParametersParamètres

inputcontrolID The controlID of the input control in which the updown control will be created, or -1 for the last created control.Le controlID du contrôle d'entrée dans lequel le contrôle updown sera créé, ou -1 pour le dernier contrôle créé.
style [optional] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default (-1) : $UDS_ALIGNRIGHT.
    forced style : $UDS_SETBUDDYINT and $UDS_ALIGNRIGHT if no align defined.$UDS_SETBUDDYINT et $UDS_ALIGNRIGHT si aucun alignement n'est défini.

Return ValueValeur de retour

Success: the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure: 0.Retourne 0.

RemarksRemarques

To use the values specified above you must #include <UpDownConstants.au3> in your script.

Max and min value can be set with GUICtrlSetLimit().Les valeurs minimun et maximum peuvent être définies avec GUICtrlSetLimit() .
By default Windows increases the value when clicking the upper arrow button.Par défaut, Windows incrémente la valeur lors d'un clic sur le bouton flèche vers le haut.

Default height resizing is done according to the one of the related input control.Par défaut le redimensionnement de la hauteur se fait avec le contrôle d'entrée qui contient le contrôle UpDown.

RelatedLiens

GUICtrlCreateInput, GUICtrlSetData, GUICtrlSetLimit

ExampleExemple

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsConstants.au3>

Example()

Func Example()
    GUICreate("My GUI UpDown", -1, -1, -1, -1, $WS_SIZEBOX)

    Local $idInput = GUICtrlCreateInput("2", 10, 10, 50, 20)
    GUICtrlCreateUpdown($idInput)

    ; Attempt to resize input controlTente de redimensionner le contrôle d'entrée
    GUICtrlSetPos($idInput, 10, 10, 100, 40)

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        $idMsg = GUIGetMsg()

        Switch $idMsg
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    MsgBox($MB_SYSTEMMODAL, "Updown", GUICtrlRead($idInput))
EndFunc   ;==>Example
