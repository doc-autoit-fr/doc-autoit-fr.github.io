WinGetClientSize

Retrieves the size of a given window's client area.Récupère la taille de la zone client d'une fenêtre.

WinGetClientSize ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to get the size.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel] The text of the window to get the size.Le texte de la fenêtre à traiter. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : a 2-element array containing the following information:Retourne un tableau de 2 éléments contenant les informations suivantes :
    $aArray[0] = Width of window's client area
    $aArray[1] = Height of window's client area
Failure:Échec : sets the @error flag to non-zero if the window is not found.Retourne @error <> 0 si la fenêtre n'est pas trouvée.

RemarksRemarques

If the window is minimized, the returned width and height values are both zero.Si la fenêtre est réduite, la largeur et la hauteur retournées sont toutes deux nulles. However, WinGetClientSize() works correctly on (non-minimized) hidden windows.Toutefois, WinGetClientSize() fonctionne correctement sur les fenêtres cachées (non réduites). If the window title "Program Manager" is used, the function will return the size of the desktop.Si le titre de la fenêtre "Program Manager» est utilisé, la fonction retourne la taille du bureau. If multiple windows match the criteria, the most recently active window is used.Si plusieurs fenêtres correspondent aux critères, la fenêtre active la plus récente sera utilisé.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Retrieve the client area of the Notepad window using the handle returned by WinWait.Récupére la zone client de la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    Local $aClientSize = WinGetClientSize($hWnd)

    ; Display the height and width of the client area.Affiche la hauteur et la largeur de la zone client.
    MsgBox($MB_SYSTEMMODAL, "", "Width: " & $aClientSize[0] & @CRLF & "Height: " & $aClientSize[1])

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
