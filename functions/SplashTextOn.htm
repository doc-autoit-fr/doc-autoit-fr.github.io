SplashTextOn

Creates a customizable text popup window.Crée une fenêtre de texte popup personnalisable.

SplashTextOn ( "title", "text" [, w = 500 [, h = 400 [, x pos [, y pos [, opt = 0 [, "fontname" [, fontsz = 12 [, fontwt]]]]]]]] )

ParametersParamètres

title Title for splash window.Titre de la fenêtre.
text Text for splash window.Texte de la fenêtre splash.
w [optional][optionnel] Width of window in pixels.Largeur de la fenêtre en pixels. (default 500)(500 par défaut)
h [optional][optionnel] Height of window in pixels.Hauteur de la fenêtre en pixels. (default 400)(400 par défaut)
x pos [optional][optionnel] Position from left (in pixels) of splash window.Position de la fenêtre splash à partir de la gauche (en pixels). (default is centered)(centrée par défaut)
y pos [optional][optionnel] Position from top (in pixels) of splash window.Position de la fenêtre splash à partir du haut (en pixels). (default is centered)(centrée par défaut)
opt [optional][optionnel] Add them up - default is 'center justified/always on top/with title'Ajoutez les options - par défaut c'est 'centrée justifiée/toujours au premier plan/avec titre".
    $DLG_CENTERONTOP (0) = Center justified/always on top/with title (default)
    $DLG_NOTITLE (1) = Thin bordered titleless window
    $DLG_NOTONTOP (2) = Without "always on top" attribute
    $DLG_TEXTLEFT (4) = Left justified text
    $DLG_TEXTRIGHT (8) = Right justified text
    $DLG_MOVEABLE (16) = Windows can be moved
    $DLG_TEXTVCENTER (32) = Center text vertically

Constants are defined in AutoItConstants.au3.
fontname [optional][optionnel] Name of the font to use.Nom de la police à utiliser. (OS default GUI font is used if the font is "" or is not found)(La police GUI par défaut dans l'OS est utilisée si la police est "" ou si elle n'est pas trouvée)
fontsz [optional][optionnel] Font size.Taille de la police. (default is 12; standard sizes are 6 8 9 10 11 12 14 16 18 20 22 24 26 28 36 48 72)(12 par défaut; les tailles standard sont 6 8 9 10 11 12 14 16 18 20 22 24 26 28 36 48 72)
fontwt [optional][optionnel] The weight of the font in the range 0 through 1000.La graisse de la police de 0 à 1000. For example, 400 is normal and 700 is bold.Par exemple, 400 est normal et 700 est gras. If this value is zero, a default weight is used.Si cette valeur est 0, une graisse par défaut est utilisée.
The following values are defined for convenience.Les valeurs suivantes sont définies pour en faciliter l'utilisation.
    $FW_DONTCARE (0) (Use the default font weight)
    $FW_THIN (100)
    $FW_EXTRALIGHT (200)
    $FW_LIGHT (300)
    $FW_NORMAL (400)
    $FW_MEDIUM (500)
    $FW_SEMIBOLD (600)
    $FW_BOLD (700)
    $FW_EXTRABOLD (800)
    $FW_HEAVY (900)
Constants are defined in FontConstants.au3.

Return ValueValeur de retour

Return the Handle of the splash window that can be used in ControlSetText().Retourne le handle de la fenêtre splash qui peut être utilisé dans ControlSetText().

RemarksRemarques

To skip an optional parameter, leaving its default value intact, use:Pour sauter un paramètre optionnel, en laissant sa valeur par défaut intacte, utilisez:
    "" for string parameters
    -1 for numeric parameters

Only one SplashImage/Text window is allowed at one time; so if you wish to cycle through multiple images/text, simply call SplashImageOn()/SplashTextOn() again with the new information.Une seule fenêtre splashImage/splashtext est permise en même temps, donc si vous souhaitez faire défiler plusieurs images/textes, utilisez simplement SplashImageOn()/SplashTextOn() à nouveau avec les nouvelles informations.

Even better is to use ControlSetText() to update text without flicker...Il est préférable d'utiliser ControlSetText() pour mettre à jour le texte sans faire flasher la fenêtre.
If the text is center and multiline, the ControlSetText() will not override the number of lines created by the SplashTextOn().Si le texte est centré et multi-lignes, le ControlSetText() ne modifiera pas le nombre de lignes créées par le SplashTextOn().

Splash with opt=1 cannot be moved and cannot be activated by click.Un splash avec opt=1 ne peut être déplacé et ne peut être activé par un clic.

Standard font names include:Noms de polices standard incluses:
    Arial, Comic Sans MS, Courier New, Lucida Console, Microsoft Sans Serif, System, Tahoma, Times New Roman, and WingDings
See the Appendix for a complete list of Windows fonts and the Windows versions under which they are supported.Consultez l'Appendice pour une liste complète des polices Windows et les versions de Windows sous lesquelles elles sont supportées.

Use @LF to display several lines.Utilisez @LF pour afficher plusieurs lignes.

RelatedLiens

ExampleExemple

#include <AutoItConstants.au3>

SplashTextOn("Title", "Message goes here.", -1, -1, -1, -1, $DLG_TEXTLEFT, "", 24)
Sleep(3000)
SplashOff()

; ; FLICKER
Local $sMessage = ""
SplashTextOn("TitleFoo", $sMessage, -1, -1, -1, -1, $DLG_TEXTLEFT, "")
For $i = 1 To 20
    $sMessage = $sMessage & $i & @CRLF
    SplashTextOn("TitleFoo", $sMessage, -1, -1, -1, -1, $DLG_TEXTLEFT, "")
    Sleep(100)
Next

; ; SMOOTH; LISSE
$sMessage = ""
SplashTextOn("TitleFoo", $sMessage, -1, -1, -1, -1, $DLG_TEXTLEFT, "")
For $i = 1 To 20
    $sMessage = $sMessage & $i & @CRLF
    ControlSetText("TitleFoo", "", "Static1", $sMessage)
    Sleep(100)
Next
