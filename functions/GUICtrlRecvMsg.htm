GUICtrlRecvMsg

Send a message to a control and retrieve information in lParam.Envoie un message à un contrôle et retourne des informations dans lParam.

GUICtrlRecvMsg ( controlID , msg [, wParam [, lParamType]] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
msg type of message to be send to the control as defined in the Windows controls documentation.Type de message à envoyer au contrôle tel que défini dans la documentation des contrôles Windows.
wParam [optional][optionnel] An integer first param to be send to the control.Un premier paramètre 'integer' à envoyer au contrôle.
lParamType [optional][optionnel] Define the type of lParam that will be returned:Définit le type de lParam qui sera retourné : 0 (default) for wParam and lParam, 1 for lParam String, 2 for lParam RECT struct.0 (par défaut) pour wParam et lParam, 1 pour lParam-String, 2 pour lParam RECT-structure.

Return ValueValeur de retour

Success:Succès : the value returned by the SendMessage Windows API.Retourne la valeur renvoyée par l'API SendMessage Windows.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

This function allows the sending of special Windows messages directly to the control using the SendMessage API.Cette fonction permet d'envoyer des messages spéciaux de Windows directement au contrôle en utilisant l'API SendMessage. It is used to enable special control features not available with the simple GUICtrlRead() and GUICtrlUpdate... range of functions.Elle est utilisée pour activer des fonctions spéciales de contrôle non disponibles avec le simple éventail de fonctions GUICtrlRead() and GUICtrlUpdate....

If the wParam and lParamType parameters are not specified then an array of two elements is returned (LPwParam, LPlParam).Si les paramètres wParam et lParamType ne sont pas spécifiés, alors un tableau de deux éléments est retourné (LPwParam, LPlParam).

The RECT is returned in an array of 4 elements (Left, Top, Right, Bottom).Le RECT est retourné dans un tableau de 4 éléments ((Left, Top, Right, Bottom).

ExampleExemple

#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Create a GUI with an edit control.Crée une interface graphique avec un contrôle d'édition.
    Local $hGUI = GUICreate("ExampleExemple")
    Local $idEdit = GUICtrlCreateEdit("Line 0" & @CRLF, 0, 0, 400, 350)
    Local $idOK = GUICtrlCreateButton("OK", 310, 370, 85, 25)

    ; Set data of the edit control.Réglages des données du contrôle d'édition.
    For $i = 1 To 25
        GUICtrlSetData($idEdit, "Line " & $i & @CRLF, 1)
    Next

    ; Set focus to the edit control.Donne le focus au au contrôle d'édition.
    GUICtrlSetState($idEdit, $GUI_FOCUS)

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Initialize the variable $aCtrlRecvMsg for storing the value returned by GUICtrlRecvMsg.Initialise la variable $aCtrlRecvMsg pour stocker la valeur retournée par GUICtrlRecvMsg.
    Local $aCtrlRecvMsg = 0

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idOK
                ; Send the message EM_GETSEL, to retrieve the current selection of the edit control.Envoie le message EM_GETSEL, pour récupérer la sélection en cours du contrôle d'édition.
                $aCtrlRecvMsg = GUICtrlRecvMsg($idEdit, $EM_GETSEL)

                ; Set focus to the edit control.Donne le focus au au contrôle d'édition.
                GUICtrlSetState($idEdit, $GUI_FOCUS)

                ; If GUICtrlRecvMsg returned the value of 0, then an error occurred otherwise display the contents of the array.Si GUICtrlRecvMsg a retourné la valeur 0, une erreur s'est produite sinon afficher le contenu du tableau.
                If $aCtrlRecvMsg = 0 Then
                    MsgBox($MB_SYSTEMMODAL, "", "An error occurred. The value returned was - " & $aCtrlRecvMsg)
                Else
                    MsgBox($MB_SYSTEMMODAL, "", "Start: " & $aCtrlRecvMsg[0] & " End: " & $aCtrlRecvMsg[1])
                EndIf

        EndSwitch
    WEnd

    ; Delete the previous GUI and all controls.Supprime la GUI précédente et tous ses contrôles.
    GUIDelete($hGUI)
EndFunc   ;==>Example
