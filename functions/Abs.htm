Abs

Calculates the absolute value of a number.Calcule la valeur absolue d'un nombre.

Abs ( expression )

ParametersParamètres

expression Any valid numeric expression.Toute expression numérique valide.

Return ValueValeur de retour

Returns absolute value of expression.Retourne la valeur absolue de l'expression.

RemarksRemarques

A string has a value of zero.Une chaîne de caractères donne zéro pour valeur.

ExampleExemple

#include <MsgBoxConstants.au3>

; Assign a Local variable an absolute number.Assigne une variable locale avec la valeur absolue d'un nombre.
