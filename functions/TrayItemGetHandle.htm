TrayItemGetHandle

Returns the handle for a tray menu(item).Retourne le handle d'un menu ou d'un item de la zone de notification.

TrayItemGetHandle ( controlID )

ParametersParamètres

controlID The control identifier (controlID) as returned by a TrayCreateItem or TrayCreateMenu function.L'identifiant du contrôle (controlID) tel que retourné par la fonction TrayCreateItem ou TrayCreateMenu fonction.

Return ValueValeur de retour

Success: the handle of the given control ID.Retourne le handle de l'ID du contrôle donné.
Failure: 0.Retourne 0.

RemarksRemarques

To get the handle to the tray context menu itself use '0' as controlID.Pour obtenir Le handle dans le menu contextuel de la zone de notification lui-même, utilisez '0' comme controlID.

Func Example()
    Local $idSettings = TrayCreateMenu("Settings") ; Create a tray menu sub menu with two sub items.Crée un sous-menu du menu de la zone de notification avec deux sous-items.
    Local $idDisplay = TrayCreateItem("Display", $idSettings)
    Local $idPrinter = TrayCreateItem("Printer", $idSettings)
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    SetMenuColor(0, 0xEEBB99) ; BGR color value, '0' refers to the tray menu itself.Valeur de couleur RGB, '0' se réfère au menu de la zone de notification elle-même.
    SetMenuColor($idSettings, 0x66BB99); BGR color value for the 'Settings' menu.Valeur de couleur RGB pour le menu "Settings".

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

            Case $idDisplay, $idPrinter
                MsgBox($MB_SYSTEMMODAL, "", "A sub menu item was selected from the tray menu.")

            Case $idExit ; Exit the loop.Sort de la boucle.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example

Func SetMenuColor($iMenuID, $iColor)
    Local $hMenu = TrayItemGetHandle($iMenuID) ; Get the internal menu handleObtient le handle de menu interne

    Local $hBrush = DllCall("gdi32.dll", "hwnd", "CreateSolidBrush", "int", $iColor)
    $hBrush = $hBrush[0]

    Local $tMenuInfo = DllStructCreate("dword;dword;dword;uint;ptr;dword;ptr")
    DllStructSetData($tMenuInfo, 1, DllStructGetSize($tMenuInfo))
    DllStructSetData($tMenuInfo, 2, BitOR($MIM_APPLYTOSUBMENUS, $MIM_BACKGROUND))
    DllStructSetData($tMenuInfo, 5, $hBrush)

    DllCall("user32.dll", "int", "SetMenuInfo", "hwnd", $hMenu, "struct*", $tMenuInfo)
EndFunc   ;==>SetMenuColor
