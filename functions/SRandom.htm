Référence Fonction

SRandom

Set Seed for random number generation.Définit la valeur de départ pour le générateur de nombre aléatoire.

SRandom ( Seed )

ParametersParamètre

Seed Seed value for random number generation.Valeur de départ pour le générateur de nombre aléatoire. Number between -2^31 and 2^31-1Nombre compris entre -2^31 et 2^31-1.

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.

Remarks<i0>

When using SRandom() sequence of random numbers will be repeated each time you run the script.Lors de l'utilisation de SRandom() une séquence de nombres aléatoires sera répétée chaque fois que vous exécutez le script. For example, you can test the speed of the script and the random numbers will not affect the test results.Par exemple, vous pouvez tester la vitesse du script et les nombres aléatoires n'affecteront pas les résultats du test.
After each call to SRandom() random number generator starts a new sequence.Après chaque appel de SRandom() le générateur de nombre aléatoire démarre une nouvelle séquence. Use the SRandom(@SEC), for random sequences.Utilisez SRandom(@SEC), pour des séquences aléatoires.
