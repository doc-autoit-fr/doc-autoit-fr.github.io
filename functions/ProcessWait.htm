ProcessWait

Pauses script execution until a given process exists.Met en pause l'exécution du script jusqu'à ce qu'un processus donné existe.

ProcessWait ( "process" [, timeout = 0] )

ParametersParamètres

process The name of the process to check.Le nom du processus à vérifier.
timeout [optional][optionnel][optionnel] Specifies how long to wait (in seconds).Spécifie la durée d'attente (en secondes ). Default is to wait indefinitely.Par défaut attendre indéfiniment.

Return ValueValeur de retour

Success:Succès : the Pid of the process.Retourne le PID du processus.
Failure:Échec : 0 if the wait timed out.Retourne 0 si l'attente a expiré.

RemarksRemarques

Process names are executables without the full path, e.g., "notepad.exe" or "winword.exe"

The process is polled approximately every 250 milliseconds.Le processus est interrogé environ toutes les 250 millisecondes.

This function is the only process function not to accept a PID.Cette fonction est la seule fonction de processus qui n'accepte pas un PID. Because PIDs are allocated randomly, waiting for a particular PID to exist doesn't make sense.Parce que les PID sont alloués de façon aléatoire, attendre qu'un PID particulier existe n'a pas de sens.
