Process functions Reference

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
DllCall Dynamically calls a function in a DLL.
DllCallAddress Dynamically calls a function at a specific memory address.Appel dynamique d'une fonction à une adresse mémoire spécifique.
DllCallbackFree Frees a previously created handle created with DllCallbackRegister.Libère un handle créé précédemment avec DllCallbackRegister.
DllCallbackGetPtr Returns the pointer to a callback function that can be passed to the Win32 API.Retourne le pointeur sur une fonction de rappel qui peut être transmis à l'API Win32.
DllCallbackRegister Creates a user-defined DLL Callback function.Crée une fonction utilisateur de rappel DLL.
DllClose Closes a previously opened DLL.Ferme une DLL précédemment ouverte.
DllOpen Opens a DLL file for use in DllCall.Ouvre un fichier DLL pour une utilisation dans DllCall.
DllStructCreate Creates a C/C++ style structure to be used in DllCall.Crée une structure style C/C++ à utiliser dans DllCall.
DllStructGetData Returns the data of an element of the struct.Retourne les données d'un élément de la structure.
DllStructGetPtr Returns the pointer to the struct or an element in the struct.Retourne le pointeur vers la structure ou un élément dans la structure.
DllStructGetSize Returns the size of the struct in bytes.Retourne la taille de la structure en octets.
DllStructSetData Sets the data of an element in the struct.
ProcessClose Terminates a named process.Termine un processus identifié.
ProcessExists Checks to see if a specified process exists.Vérifie si un processus spécifié existe.
ProcessGetStats Returns an array about Memory or IO infos of a running process.Retourne un tableau sur la mémoire ou des infos IO sur le processus en cours d'exécution.
ProcessList Returns an array listing the currently running processes (names and PIDs).Retourne un tableau listant les processus en cours d'exécution (noms et PIDs).
ProcessSetPriority Changes the priority of a process.Modifie la priorité d'un processus.
ProcessWait Pauses script execution until a given process exists.Met en pause l'exécution du script jusqu'à ce qu'un processus donné existe.
ProcessWaitClose Pauses script execution until a given process does not exist.Met en pause l'exécution du script jusqu'à ce qu'un processus donné n'existe plus.
Run Runs an external program.Exécute un programme externe.
RunAs Runs an external program under the context of a different user.Exécute un programme externe dans le contexte d'un autre utilisateur.
RunAsWait Runs an external program under the context of a different user and pauses script execution until the program finishes.Exécute un programme externe dans le contexte d'un autre utilisateur et interrompt l'exécution du script jusqu'à ce que le programme se termine.
RunWait Runs an external program and pauses script execution until the program finishes.Exécute un programme externe et interrompt l'exécution du script jusqu'à ce que le programme se termine.
ShellExecute Runs an external program using the ShellExecute API.Exécute un programme externe en utilisant l'API ShellExecute.
ShellExecuteWait Runs an external program using the ShellExecute API and pauses script execution until it finishes.Exécute un programme externe en utilisant l'API ShellExecute et interrompt l'exécution du script jusqu'à ce qu'il se termine.
Shutdown Shuts down the system.Arrête le système.
StderrRead Reads from the STDERR stream of a previously run child process.Lit dans le flux STDERR d'un processus enfant lancé au préalable.
StdinWrite Writes a number of characters to the STDIN stream of a previously run child process.Ecrit un nombre de caractères dans le flux STDIN d'un processus enfant lancé au préalable.
StdioClose Closes all resources associated with a process previously run with STDIO redirection.Ferme toutes les ressources associées à un processus lancé précédemment avec la redirection STDIO.
StdoutRead Reads from the STDOUT stream of a previously run child process.Lit dans le flux STDOUT d'un processus enfant lancé au préalable.
