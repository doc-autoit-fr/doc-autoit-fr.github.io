MemGetStats

Retrieves memory related information.Récupère des informations relatives à la mémoire.

MemGetStats ( )

Return ValueValeur de retour

Returns a seven-element array containing the memory information:Retourne un tableau à sept éléments contenant les informations de la mémoire :
$aArray[0] = Memory Load (Percentage of memory in use)
$aArray[1] = Total physical RAM
$aArray[2] = Available physical RAM
$aArray[3] = Total Pagefile
$aArray[4] = Available Pagefile
$aArray[5] = Total virtual
$aArray[6] = Available virtual

All memory sizes are in kilobytes.Toutes les tailles de mémoire sont en kilo-octets .

ExampleExemple

#include <MsgBoxConstants.au3>

Local $aMem = MemGetStats()
MsgBox($MB_SYSTEMMODAL, "", "Total physical RAM (KB): " & $aMem[1])
