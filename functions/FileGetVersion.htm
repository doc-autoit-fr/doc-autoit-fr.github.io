FileGetVersion

Returns version information stored in a file.

FileGetVersion ( "filename" [, "stringname"] )

ParametersParamètres

filename Filename.
stringname [optional][optionnel] Name of the string field to be retrieved from the header version file info:
    $FV_COMMENTS ("Comments")
    $FV_COMPANYNAME ("CompanyName")
    $FV_FILEDESCRIPTION ("FileDescription")
    $FV_FILEVERSION ("FileVersion")
    $FV_INTERNALNAME ("InternalName")
    $FV_LEGALCOPYRIGHT ("LegalCopyright")
    $FV_LEGALTRADEMARKS ("LegalTrademarks")
    $FV_ORIGINALFILENAME ("OriginalFilename")
    $FV_PRODUCTNAME ("ProductName")
    $FV_PRODUCTVERSION ("ProductVersion")
    $FV_PRIVATEBUILD ("PrivateBuild")
    $FV_SPECIALBUILD ("SpecialBuild")

Constants are defined in FileConstants.au3.

Return ValueValeur de retour

Success:Succès : The version number ("#.#.#.#" format) - or the content of the specified string field.
Failure:Échec : "0.0.0.0" if no version information (or other error) or "" if string field not filled - @error flag set to 1.

RemarksRemarques

Another special "stringname" is "DefaultLangCodepage" can be used to retrieve the default language and codepage.

The language and codepage can be used if needed to differentiate the "stringname" i.e.La langue et les pages de code peuvent être utilisés si nécessaire pour différencier les "stringname" (noms de chaîne) exemple : "080904b0\Comments" (see MSDN StringFileInfo in VerQueryValue function)."080904b0\Comments" (Consultez StringFileInfo dans la fonction VerQueryValue sur MSDN).

To get the version number of an AutoIt compiled script it is better to use the @AutoItVersion macro.Pour obtenir le numéro de version d'un script Autoit compilé il est mieux d'utiliser la macro @AutoItVersion.
