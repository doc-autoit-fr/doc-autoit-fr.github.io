StringReverse

Reverses the contents of the specified string.Inverse le contenu de la chaîne spécifiée.

StringReverse ( "string" [, flag = 0] )

ParametersParamètre

string The string to reverse.La chaîne à inverser.
flag [optional][optional] False - i, v, x ... Changes the way the string is reversedDéfinit la façon dont la chaîne est inversée
    $STR_UTF16 (0) = reversed in full UTF-16 mode. (default)(par défaut)
    $STR_UCS2 (1) = a much faster method - only use if using UCS-2 text.

Constants are defined in "StringConstants.au3".

Return ValueRemarque

Returns a reversed string.Retourne la chaîne inversée.
