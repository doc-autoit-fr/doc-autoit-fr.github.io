FileCopy

Copies one or more files.Copie un ou plusieurs fichiers.

FileCopy ( "source", "dest" [, flag = 0] )

ParametersParamètres

source The source path of the file(s) to copy. (* and ?( caractères spéciaux * et ? wildcards accepted - See Remarks)acceptés - Voir Remarques)
dest The destination path of the copied file(s).Chemin du répertoire de destination du/des fichier(s) copié(s).
flag [optional][optionnel] this flag determines whether to overwrite files if they already exist.ce paramètre détermine s'il faut écraser les fichiers existants.
Can be a combination of the following:Peut-être une combinaison des éléments suivants:
    $FC_NOOVERWRITE (0) = (default) do not overwrite existing files    $FC_NOOVERWRITE (0) = (par défaut) n'écrase pas les fichiers existants
    $FC_OVERWRITE (1) = overwrite existing files    $FC_OVERWRITE (1) = écrase les fichiers existants
    $FC_CREATEPATH (8) = Create destination directory structure if it doesn't exist (See Remarks).

Constants are defined in FileConstants.au3.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

See FileFindFirstFile() for a discussion about wildcards.Consultez FileFindFirstFile() pour des précisions sur les caractères spéciaux.

The destination directory must already exist, except using with flag value $FC_CREATEPATH (8).Le répertoire de destination doit exister, sauf si le paramètre "8" est utilisé.
For instance the combined flag $FC_OVERWRITE (1) + $FC_CREATEPATH (8) overwrites the target file and pre-checks for the destination directory structure and if it doesn't exist creates it automatically.Par exemple le paramètre combiné $FC_OVERWRITE (1) + $FC_CREATEPATH (8), écrase le fichier cible et vérifie la structure du répertoire de destination et si elle n'existe pas, il la crée automatiquement.

Some file attributes can make the overwriting impossible, if this is the case look at FileSetAttrib() to change the attributes of a file.Certains attributs de fichier peuvent rendre l'écrasement impossible, si c'est le cas consultez FileSetAttrib() pour changer les attributs d'un fichier.

ExampleExemple

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create a temporary file to copy.Crée un fichier temporaire pour copier.
    If Not FileWrite($sFilePath, "This is an example of using FileCopy.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Copy Au3 files in the temporary directory to a new folder/directory called Au3Files.Copie les fichiers Au3 dans le répertoire temporaire vers un nouveau dossier/répertoire appelé Au3Files.
    FileCopy(@TempDir & "\*.au3", @TempDir & "\Au3Files\", $FC_OVERWRITE + $FC_CREATEPATH)

    ; Display the temporary directory.Affiche le répertoire temporaire.
    ShellExecute(@TempDir)
EndFunc   ;==>Example
