Function ReferenceRéférence de fonction

ToolTip

Creates a tooltip anywhere on the screen.Crée une info-bulle n'importe où sur l'écran.

ToolTip ( "text" [, x [, y [, "title" [, icon = 0 [, options]]]]] )

ParametersParamètres

text The text of the tooltip.Le texte de l'info-bulle. (An empty string clears a displaying tooltip)(Une chaîne vide efface l'affichage d'une info-bulle)
x [optional][optionnel] The x position of the tooltip.La position x de l'info-bulle.
y [optional][optionnel] The y position of the tooltip.La position y de l'info-bulle.
title [optional][optionnel] The title for the tooltip.
icon [optional][optionnel] Pre-defined icon to show next to the title:Icône pré-définie pour l'afficher à côté du titre: Requires a title.Nécessite un titre.
    $TIP_NOICON (0) = No icon
    $TIP_INFOICON (1) = Info icon
    $TIP_WARNINGICON (2) = Warning icon
    $TIP_ERRORICON (3) = Error Icon

Constants are defined in "AutoItConstants.au3".
options [optional][optionnel] Sets different options for how the tooltip will be displayed (Can be added together):Définit différentes options déterminant la façon avec laquelle l'info-bulle sera affichée (Peuvent être additionnées ensemble):
    $TIP_BALLOON (1) = Display as Balloon Tip
    $TIP_CENTER (2) = Center the tip at the x,y coordinates instead of using them for the upper left corner.
    $TIP_FORCEVISIBLE (4) = Force the tooltip to always be visible confining it to monitor borders if necessary. If multiple monitors are used, then the tooltip will "snap-to" the nearest monitor.Si plusieurs moniteurs sont utilisés, alors l'info-bulle sera "snap-to" le moniteur le plus proche.

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 when title length is greater than 99.Retourne 0 lorsque la longueur du titre est supérieur à 99.

RemarksRemarques

To skip an optional parameter, leaving its default value intact, use the Default keyword.Pour passer un paramètre optionnel, en laissant sa valeur par défaut intacte, utilisez le mot-clé Default.

If the x and y coordinates are omitted, the tip is placed near the mouse cursor.Si les coordonnées x et y sont omises, l'info-bulle est placée près du curseur de la souris.
A Tooltip will appear until the script terminates or ToolTip("") is called.Une info-bulle s'affiche jusqu'à ce que le script se termine ou ToolTip("") soit appelée.
You may use @CR or @LF to create multi-line tooltips.Vous pouvez utiliser @CR ou @LF pour créer des info-bulles multi-lignes.
To display an icon, you must specify a non-empty title.Pour afficher une icône, vous devez indiquer un titre non vide. The icon appears on the same row as the title and thus requires a title to be present.L'icône apparaît sur la même ligne que le titre et nécessite donc qu'un titre soit présent.
If using the center flag, the center of the tooltip will be at the coordinates specified.Si vous utilisez le flag 'center', le centre de l'info-bulle sera aux coordonnées spécifiées. If using the center flag with a Balloon Tip, the stem will be centered on the balloon and will point to the coordinates specified.Si vous utilisez le flag 'center' avec la forme ballon, la tige sera centrée sur le ballon et pointera vers les coordonnées spécifiées.

ExampleExemple

; This will create a tooltip in the upper left of the screenCréation d'une info-bulle dans le coin supérieur gauche de l'écran

ToolTip("This is a tooltip", 0, 0)
Sleep(2000) ; Sleep to give tooltip time to displayPause pout donner à l'info-bulle le temps de s'afficher
