Référence Fonction

Eval

Return the value of the variable defined by a string.Retourne la valeur de la variable définie par une chaîne de caractère.

Eval ( string )

ParametersParamètres

string string representing name of the variable.Chaîne de caractères représentant la variable.

Return ValueValeur de retour

Success: the value of the variable.Retourne la valeur de la variable.
Failure: "" (empty string) and sets the @error flag to non-zero.Retourne "" (Chaîne vide ) et @error <> 0.

RemarksRemarques

If there is a need to use Eval() to read a variable, then in most situations Assign() should be used to create/write to the variable and IsDeclared() should be used to check that the variable exists.Si l'utilisation de la fonction Eval() s'avère nécessaire pour lire une variable, dans la plupart des situations Assign() devrait être utilisé pour la créer ou l'écrire, et la fonction IsDeclared() devra être utilisée afin de vérifier qu'elle existe.

RelatedLiens

Assign, Execute, IsDeclared

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign the variable string sString with data.Affecte la variable chaîne sChaine de données.
    Assign("sString", "This is a string which is declared using the function Assign")

    ; Find the value of the variable string sString and assign to the variable $sEvalString.Trouve la valeur de la chaîne sChaine et l'affecte à la variable $sEvalString.
    Local $sEvalString = Eval("sString")

    ; Display the value of $sEvalString.Affiche la valeur de $sEvalString. This should be the same value as $sString.Cela devrait être la même valeur que $sChaine.
    MsgBox($MB_SYSTEMMODAL, "", $sEvalString)
EndFunc   ;==>Example
