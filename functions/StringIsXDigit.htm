StringIsXDigit

Checks if a string contains only hexadecimal digit (0-9, A-F) characters.Vérifie si une chaîne contient uniquement des caractères hexadécimaux (0-9, A-F).

StringIsXDigit ( "string" )

ParametersParamètres

string The string to checkLa chaîne à traiter

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if string contains non-hexadecimal characters.Retourne 0 si la chaîne contient des caractères non-hexadécimaux.

RemarksRemarques

Valid characters are 0123456789abcdefABCDEF.
Note that a string containing whitespace or the "0x" prefix will cause StringIsXDigit() to return 0.

RelatedLiens

StringIsAlNum, StringIsAlpha, StringIsASCII, StringIsDigit, StringIsLower, StringIsSpace, StringIsUpper, StringLower, StringUpper

ExampleExemple

#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "", "Is the string 42 a hexadecimal digit: " & StringIsXDigit("42") & @CRLF & _ ; Returns 1, as the string contains only hexadecimal (0-9, A-F) characters.Retourne 1, car la chaîne ne contient que des chiffres hexadécimaux (0-9, AF) caractères.
        "Is the string 00 a hexadecimal digit: " & StringIsXDigit("00") & @CRLF & _ ; Returns 1, as the string contains only hexadecimal (0-9, A-F) characters.Retourne 1, car la chaîne ne contient que des chiffres hexadécimaux (0-9, AF) caractères.
        "Is the string 1.0 a hexadecimal digit: " & StringIsXDigit("1.0") & @CRLF & _ ; Returns 0, due to the decimal point.Retourne 0, en raison de la virgule.
        "Is the number 1.0 a hexadecimal digit: " & StringIsXDigit(1.0) & @CRLF & _ ; Returns 1, due to the number to string conversion.Retourne 1, en raison du nombre que contient la chaîne à convertir.
        "Is the string 'A string' a hexadecimal digit: " & StringIsXDigit("A string.") & @CRLF) ; Returns 0, due to the string containing whitespace.Retourne 0, car la chaîne de caractères contenant des espaces.
