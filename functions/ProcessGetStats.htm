ProcessGetStats

Returns an array about Memory or IO infos of a running process.Retourne un tableau sur la mémoire ou des infos IO sur le processus en cours d'exécution.

ProcessGetStats ( ["process" [, type = 0]] )

ParametersParamètres

process [optional][optionnel] The name or PID of the process to get infos from.Le nom ou le PID du processus dont on veut obtenir des infos de. Default (-1) is the current process.Par défaut (-1) est le processus courant.
type [optional][optionnel]
    $PROCESS_STATS_MEMORY (0) = (default) memory infos.
    $PROCESS_STATS_IO (1) = IO infos.

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Success:Succès : an array of infos data (See Remarks).Retourne un tableau de données d'infos (Voir Remarques).
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

The array returned is mono-dimensional and is made up as follows:Le tableau renvoyé est unidimensionnel et se compose comme suit:

Type = $PROCESS_STATS_MEMORY
    $aArray[0] = WorkingSetSize
    $aArray[1] = PeakWorkingSetSize


Type = $PROCESS_STATS_IO
    $aArray[0] = number of read operations performed.
    $aArray[1] = number of write operations performed.
    $aArray[2] = number of I/O operations performed, other than read and write operations.
    $aArray[3] = number of bytes read.
    $aArray[4] = number of bytes write.
    $aArray[5] = number of bytes transferred during operations other than read and write operations.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Retrieve memory details about the current process.Récupèrer les détails de la mémoire sur le processus en cours.
    Local $aMemory = ProcessGetStats()

    ; If $aMemory is an array then display the following details about the process.Si $aMemory est un tableau, alors affiche les informations suivantes sur le processus.
    If IsArray($aMemory) Then
        MsgBox($MB_SYSTEMMODAL, "", "WorkingSetSize: " & $aMemory[0] & @CRLF & _
                "PeakWorkingSetSize: " & $aMemory[1])
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred.")
    EndIf
EndFunc   ;==>Example
