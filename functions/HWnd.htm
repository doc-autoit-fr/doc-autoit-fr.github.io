Référence Fonction

HWnd

Converts an expression into an HWND handle.Convertit une expression en un handle HWND.

HWnd ( expression )

ParametersParamètre

expression An expression to convert into an HWND handle.L'expression à convertir en un handle HWND.

Return ValueRemarque

Success: If the value can be converted to an HWND, the HWND representation will be returned.Si la valeur peut être convertie en un HWND, la représentation HWND sera retournée.
Failure: If the HWND does not denote a valid window, a 0 (NULL) HWND will be returned and sets the @error flag to 1.Si le HWND n'indique pas une fenêtre valide, un HWND égal à 0 (NULL) sera retourné ainsi que @error = 1.

Remarks<i0>

Double numbers can not be converted to an HWND.Les nombres 'double' ne peuvent pas être convertit en HWND.
No literal string should be converted to an HWND because there is no guarantee that a given window will have the same HWND ever again.Aucune chaîne littérale ne devrait être convertie en HWND parce qu'il n'y a aucune garantie qu'une fenêtre donnée aura le même HWND à chaque fois. This is not strictly forbidden, but it is a programming mistake and should be avoided.Ceci n'est pas interdit, mais c'est une erreur de programmation qui devrait être évitée.

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Convert the handle to a string.Convertit le handle en une chaîne.
    Local $sHWnd = String($hWnd)

    ; Minimize the Notepad window and wait for 2 seconds.Réduit la fenêtre du Bloc-notes et attend 2 secondes.
    WinSetState(HWnd($sHWnd), "", @SW_MINIMIZE)
    Sleep(2000)

    ; Restore the Notepad window and wait for 2 seconds.Restaure la fenêtre du Bloc-notes et attend 2 secondes.
    WinSetState(HWnd($sHWnd), "", @SW_RESTORE)
    Sleep(2000)

    ; Close the Notepad window.Ferme la fenêtre du Bloc-notes.
    WinClose(HWnd($sHWnd))
EndFunc   ;==>Example
