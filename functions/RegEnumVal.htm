RegEnumVal

Reads the name of a value according to its instance.Lit le nom d'une valeur dont l'instance est donnée.

RegEnumVal ( "keyname", instance )

ParametersParamètres

keyname The registry key to read.La clé de registre à lire.
instance The 1-based value instance to retrieveL'instance de la valeur, à partir de 1, à récupérer

Return ValueValeur de retour

Success: the requested registry value.Retourne la valeur de Registre demandé. @extended is set to the type of the value.@extended contient le type de la valeur.
Failure: sets the @error flag to non-zero.met @error <> 0.
@error:@error : 1 = unable to open requested key1 = impossible d'ouvrir la clé demandée
 2 = unable to open requested main key
 3 = unable to remote connect to the registry
-1 = unable to retrieve requested value name (value instance out of range)-1 = impossible de récupérer le nom de la valeur demandée ( instance de valeur en dehors de la plage)

RemarksRemarques

A registry key must start with "HKEY_LOCAL_MACHINE" ("HKLM") or "HKEY_USERS" ("HKU") or "HKEY_CURRENT_USER" ("HKCU") or "HKEY_CLASSES_ROOT" ("HKCR") or "HKEY_CURRENT_CONFIG" ("HKCC").Une clé de registre doit commencer par "HKEY_LOCAL_MACHINE" ("HKLM") ou "HKEY_USERS" ("HKU") ou "HKEY_CURRENT_USER" ("HKCU") ou "HKEY_CLASSES_ROOT" ("HKCR") ou "HKEY_CURRENT_CONFIG" ("HKCC") .

When running on 64-bit Windows if you want to enum a value specific to the 64-bit environment you have to suffix the HK... with 64 i.e.Lors de l'exécution sur Windows 64 bits si vous voulez lire une valeur spécifique à l'environnement 64 bits, vous devez suffixer HK... HKLM64.avec 64 c'est à dire HKLM64.

ExampleExemple

#include <MsgBoxConstants.au3>

; X64 running supportX64 appui fonctionnement
Local $sWow64 = ""
If @AutoItX64 Then $sWow64 = "\Wow6432Node"

Local $sVar = ""

For $i = 1 To 100
    $sVar = RegEnumVal("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", $i)
    If @error <> 0 Then ExitLoop
    MsgBox($MB_SYSTEMMODAL, "Value Name #" & $i & " under in AutoIt3 key", $sVar)
Next
