SetError

Manually set the value of the @error macro (and optionally @extended, and "Return Value").

SetError ( code [, extended = 0 [, return value]] )

ParametersParamètres

code The required value (integer) to set into the @error macro.La valeur requise (entière) pour définir la macro @error.
extended [optional][optionnel] The required value (integer) to set into the @extended macro.La valeur requise (entière) à affecter à @extended. This sets the same macro as the SetExtended() function.Ceci définit la même macro que la fonction SetExtended().
return value [optional][optionnel] The value to be returned by the function - if no parameter used the return value is undefined.La valeur à retourner par la fonction - si aucun paramètre n'est utilisé la valeur de retour est indéfinie.

Return ValueValeur de retour

Returns the "return value" parameter (undefined if the parameter is not used).Retourne le paramètre "return value"( indéfini si le paramètre n'est pas donné).
@error and @extended will be set to the values set in the function call (both default = 0)@error et @extended seront définies avec les valeurs installées dans l'appel de la fonction (les deux, par défaut = 0).

RemarksRemarques

When entering a function @error is set to 0.Lorsque vous entrez dans une fonction, @error est remis à 0. Unless SetError() is called, then @error will remain 0 when the function ends.A moins que SetError() ne soit appelée, @error restera à 0 à la sortie de la fonction. This means that in order for @error to be set after a function, it must be explicitly set.Cela signifie que vous devez assigner explicitement @error à la sortie d'une fonction, il doit être explicitement défini. This also means you may need to backup the status of @error in a variable if you are testing it in a While-WEnd loop.Cela signifie également que vous pouvez avoir besoin de sauvegarder le statut de @error dans une variable si vous la testez dans une boucle While-WEnd.
The extended parameter is optional.Le paramètre extended est optionnel. It is provided as a way to set both @error and @extended at the same time.Il est seulement prévu comme un moyen de définir à la fois @error et @extended au même moment. If only @extended needs to be set, then it is recommended to use the SetExtended() function instead.
The return value parameter is optional. It is provided as a way to use the Return SetError(...) syntax to define the value to be returned at the same time as setting @error (and possibly @extended). If a specific value is not set then the return value will be undefined and should not be used subsequently by the code.

@error is limited between the values of -2147483648 to 2147483647.@error est limitée entre les valeurs de -2147483648 à 2147483647.

#include <MsgBoxConstants.au3>

Local $fResult = myDiv(5, 0)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "Division by Zero")
Else
    MsgBox($MB_SYSTEMMODAL, "Result", $fResult)
EndIf
Exit

Func myDiv($iDividend, $iDivisor)
    If $iDividend = 0 And $iDivisor = 0 Then
        SetError(2) ; Indeterminate form 0/0.Forme indéterminée 0/0.
    ElseIf $iDivisor = 0 Then
        SetError(1) ; Plain division by zero.Division par zéro plaine.
    EndIf
    Return $iDividend / $iDivisor
EndFunc   ;==>myDiv
