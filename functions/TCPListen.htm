TCPListen

Creates a socket listening for an incoming connection.Crée un socket d'écoute pour une connexion entrante.

TCPListen ( IPAddr, port [, MaxPendingConnection] )

ParametersParamètres

IPAddr Internet Protocol dotted address(IpV4) as "192.162.1.1".L'adresse IPv4 séparée par des points tel que :"192.162.1.1".
port port on which the created socket will be connected.Port sur lequel le socket créé sera connecté.
MaxPendingConnection [optional][optionnel] Maximum length of the queue of pending connections.Longueur maximale de la file d'attente des connexions en attente. By default the maximum reasonable value will be set.Par défaut, une valeur maximale raisonnable est définie.

Return ValueValeur de retour

Success:Succès : the main socket identifier.Retourne l'identifiant principale du socket.
Failure:Échec : -1 or 0 and sets the @error flag to non-zero.Retourne -1 ou 0, et @error <> 0.
@error:@error : 1 IPAddr is incorrect.1 = IPAddr incorrect.
2 port is incorrect.2 = port incorrect.
Windows API WSAGetError return value (see MSDN).Valeur retournée par l'API Windows WSAGetError (Consultez MSDN).

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    TCPStartup() ; Start the TCP service.Démarre le service TCP.

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

    ; Assign Local variables the loopback IP Address and the Port.Assigne des variables locales avec l'adresse IP et le port.
    Local $sIPAddress = "127.0.0.1" ; This IP Address only works for testing on your own computer.Cette adresse IP ne fonctionne que pour les tests sur votre propre ordinateur.
    Local $iPort = 65432 ; Port used for the connection.Port utilisé pour la connexion.

    ; Assign a Local variable the Listening socket and bind to the IP Address and Port specified with a maximum of 100 pending connexions.Assigne une variable locale avec le socket d'écoute et se connecte à l'adresse IP et au port spécifiés avec un maximum de 100 connexions en attente.
    Local $iListenSocket = TCPListen($sIPAddress, $iPort, 100)

    ; Notes: You can only listen on private IPs, such as the one used here;Vous pouvez seulement écouter sur des IPs privées, comme celui utilisé ici;
    ; or on the range of 192 to 223 (generally 192.168.X.X, use @IPAddress1 to test on your local IP [you will need another computer]).
    ; The Listen socket identifier is only used for the TCP Accept function.L'identifiant de la socket Listen est utilisée uniquement pour la fonction TCPAccept.

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        ; Someone is probably already listening on this IP Address and Port (script already running?).Quelqu'un est probablement déjà à l'écoute sur cette adresse IP et ce port (script déjà en cours d'exécution?).
        Local $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Could not listen, Error code: " & $iError)
        Return False
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Listen successful.")
    EndIf

    ; Close the Listening socket to allow afterward binds.Ferme le socket d'écoute pour permettre pour permettre une connexion ultérieure.
    ; While not closed, any other program can NOT bind to the same IP Address and Port.Tant que n'est pas fermé, aucun autre programme ne peut pas se connecter à la même adresse IP et à ce port.
    TCPCloseSocket($iListenSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Close the TCP service.Ferme le service TCP.
EndFunc   ;==>OnAutoItExit
