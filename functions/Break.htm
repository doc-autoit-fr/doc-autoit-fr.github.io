Break

Enables or disables the users' ability to exit a script from the tray icon menu.Active ou désactive la possibilité d'arrêter un script depuis le menu de l'icône de la zone de notification.

Break ( mode )

ParametersParamètres

mode Sets the script break mode:Règle le mode break du script :
    $BREAK_ENABLE (1) = Break is enabled (user can quit) (default)
    $BREAK_DISABLE (0) = Break is disabled (user cannot quit)

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

Please only disable break with good reason.Désactivez le break seulement si vous avez une bonne raison de le faire.
AutoIt normally creates a tray icon when running, and right-clicking this icon allows the user to pause or exit the script.Autoit crée normalement une icône dans la barre des tâches lorsqu'il s'exécute et un clic droit sur cette icône permet à l'utilisateur de suspendre ou de sortir du script. If Break() is disabled (0), then the user cannot terminate the script this way.Si Break() est désactivé (0), alors l'utilisateur ne peut pas terminer le script de cette façon.

ExampleExemple

Break(0) ;Disable breakDésactive le break
