Référence Fonction

TrayGetMsg

Polls the tray to see if any events have occurred.Interroge la zone de notification pour savoir si des événements ont été déclenchés.

TrayGetMsg ( )

Return ValueValeur de retour

Returns an event.Retourne un événement.

The "event" returned is the control ID of the control sending the message, or it is a special event (like the mouse clicking on the tray icon).L'événement retourné est l'ID du contrôle qui a envoyé le message, ou il s'agit d'un événement spécial (comme le clic de la souris sur l'icône de la zone de notification). Or if there is no message, the event is 0.Ou s'il n'y a pas de message, l'événement est égal à 0.


Event IDsIDs d'événements
$TRAY_EVENT_NONE (0): No eventAucun évènement
Control ID: the ID of the control sending the messagel'identifiant du contrôle qui a envoyé le message
$TRAY_EVENT_PRIMARYDOWN: the primary mouse button was pressedle bouton principal de la souris a été pressé
$TRAY_EVENT_PRIMARYUP: the primary mouse button was releasedle bouton principal de la souris a été relâché
$TRAY_EVENT_SECONDARYDOWN: the secondary mouse button was pressedle bouton secondaire (droit) de la souris a été pressé
$TRAY_EVENT_SECONDARYUP: the secondary mouse button was releasedle bouton secondaire (droit) de la souris a été relâché
$TRAY_EVENT_PRIMARYDOUBLE: the primary mouse button was double pressedle bouton principal de la souris a été pressé deux fois
$TRAY_EVENT_SECONDARYDOUBLE: the secondary mouse button was double pressedle bouton secondaire (droit) de la souris a été pressé deux fois

Constants are defined in TrayConstants.au3Les constantes sont définies dans TrayConstants.au3

RemarksRemarques

This function automatically idles the CPU when required so that it can be safely used in tight loops without hogging all the CPU.Cette fonction ralentit automatiquement lorsque le CPU en a besoin, de façon qu'elle puisse être utilisée en toute sécurité dans des boucles courtes sans monopoliser tout le CPU.

These are options 1 and 2 for TrayMenuMode.Ce sont les options 1 et 2 de TrayMenuMode.

Example()

Func Example()
    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

            Case $idExit ; Exit the loop.Sort de la boucle.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
