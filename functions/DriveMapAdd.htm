DriveMapAdd

Maps a network drive.Mappe un lecteur réseau.

DriveMapAdd ( "device", "remote share" [, flags = 0 [, "user" [, "password"]]] )

ParametersParamètres

device The device to map, for example "O:" or "LPT1:".Périphérique à mapper, par exemple "O:" ou "LPT1:". If you pass an empty string for this parameter a connection is made but not mapped to a specific drive.Si vous passez une chaîne vide pour ce paramètre une connexion est établie mais pas mappée à un lecteur spécifique. If you specify "*" an unused drive letter will be automatically selected.Si vous spécifiez "*" une lettre de lecteur non utilisée sera automatiquement sélectionnée.
remote share The remote share to connect to in the form "\\server\share".Le partage distant à connecter sous la forme "\\server\share".
flags [optional][optionnel] A combination of the following:Une combinaison des éléments suivants:
    $DMA_DEFAULT (0) = default
    $DMA_PERSISTENT (1) = Persistent mapping
    $DMA_AUTHENTICATION (8) = Show authentication dialog if required

Constants are defined in "AutoItConstants.au3"
user [optional][optionnel] The username to use to connect.Le nom de l'utilisateur à utiliser pour la connexion. In the form "username" or "domain\username".Sous la forme "username" ou "domain\username".
password [optional][optionnel] The password to use to connect.Le mot de passe à utiliser pour la connexion.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe. (See Remarks)(Consultez les remarques)
Failure:Échec : 0 if a new mapping could not be created and sets the @error flag to non-zero.Retourne 0 si une nouvelle connexion ne peut pas être créée et assigne @error.(Consultez les remarques).
@error:@error : 1 = Undefined / Other error.1 = Erreur Indéfinie/Autre. @extended set with Windows API return@extended est défini avec le code de retour de l'API Windows
2 = Access to the remote share was denied2 = L'accès au partage distant a été refusé
3 = The device is already assigned3 = Le périphérique est déjà assigné
4 = Invalid device name4 = Nom de périphérique invalide
5 = Invalid remote share5 = Partage distant invalide
6 = Invalid password6 = Mot de passe invalide

RemarksRemarques

Note:Note : When using "*" for the device parameter the drive letter selected will be returned instead of 1 or 0, e.g. "U:". If there was an error using "*" then an empty string "" will be returned.S'il y avait une erreur en utilisant "*" alors une chaîne vide "" serait retournée.

If defined the user/password will be presented to the remote computer that will validate the credential.Si user/password sont définis, ils seront présentés à l'ordinateur distant qui validera le certificat.

ExampleExemple

#include <AutoItConstants.au3>

; Map X drive to \\myserver\stuff using current userMappe le lecteur X sur \\myserver\stuff en utilisant l'utilisateur courant
DriveMapAdd("X:", "\\myserver\stuff")

; Map X drive to \\myserver2\stuff2 using the user "jon" from "domainx" with password "tickle"Mappe le lecteur X sur \\myserver2\stuff2 en utilisant l'utilisateur \"jon\" du \"domainx\" avec le mot de passe \"tickle\"
DriveMapAdd("X:", "\\myserver2\stuff2", $DMA_DEFAULT, "domainx\jon", "tickle")
