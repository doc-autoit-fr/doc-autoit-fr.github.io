EnvUpdate

Refreshes the OS environment.Actualise l'environnement de l'OS.

EnvUpdate ( )

Return ValueValeur de retour

Success:Succès : none.Aucune valeur de retour.
Failure:Échec : sets the @error flag to 1.Retourne 0 et @error = 1.

RemarksRemarques

Similar effect as logging off and then on again.Effet similaire à la déconnexion, puis reconnexion de la session. For example, changes to the %path% environment might not take effect until you call EnvUpdate() (or you logoff/reboot).Par exemple, les modifications de la variable d'environnement %PATH% pourraient ne pas prendre effet tant que vous n'appelez EnvUpdate() (ou déconnecter la session/rebooter).

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Retrieve the system environment variable called %PATH%.Récupére la variable d'environnement système appelé %PATH%.
    Local $sEnvVar = EnvGet("PATH")

    ; Assign the system environment variable called %PATH% with its current value as well as the script directory.Assigne la variable d'environnement système appelé %PATH% avec sa valeur actuelle ainsi que le répertoire de script.
    ; When you assign an environment variable you do so minus the percentage signs (%).Lorsque vous affectez une variable d'environnement, vous le faites moins les signes de pourcentage (%).
    EnvSet("PATH", $sEnvVar & ";" & @ScriptDir)

    ; Refresh the OS environment for changes to take affect.Actualise l'environnement OS pour que les modifications prennent effet.
    EnvUpdate()

    ; Retrieve the system environment variable that was just assigned a value previously.Récupére la variable d'environnement système dont on vient d'attribuer une valeur auparavant.
    $sEnvVar = EnvGet("PATH")

    ; Display the value of the environment variable $PATH%.Affiche la valeur de la variable d'environnement %PATH%.
    MsgBox($MB_SYSTEMMODAL, "", "The environment variable %PATH% has the value of: " & $sEnvVar)
EndFunc   ;==>Example
