Int

Returns the integer (whole number) representation of an expression.Retourne la partie entière d'une expression (l'entier immédiatement inférieur).

Int ( expression [, flag = 0] )

ParametersParamètres

expression An expression to convert into an integer.Une expression à convertir en un entier.
flag [optional][optionnel] Defines behavior.Définit le comportement.
Can be one of the following:Peut être un des suivants:
    $NUMBER_AUTO (0) = (default) the result is auto-sized integer. See remarks.Voir remarques.
    $NUMBER_32BIT (1) = string is interpreted as a 32bit integer
    $NUMBER_64BIT (2) = string is interpreted as a 64bit integer

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Returns an integer.Retourne un entier.

RemarksRemarques

Default behavior is that if the result is within range of 32bit integer then 32bit integer is returned.Le comportement par défaut est que si le résultat est dans la plage des entiers 32 bits, alors un entier 32 bits est retourné. If not, 64bit integer is returned.Si non, un entier 64 bits est retourné. Both are signed.
Fractional portions are truncated, so Int(1.999999) returns 1La partie fractionnaire est tronqué, de sorte que Int(1.999999) renvoie 1
Int(0/0) returns -9223372036854775807, if you were wondering.Int(0/0 ) retourne -9223372036854775807, si vous vous posiez la question.
This function makes minor corrections to floating point numbers to account for the imprecise nature of floating point numbers. For example, the floating point expression 0.7 + 0.2 + 0.1 produces a floating point number that is not quite 1.0.Par exemple, l'expression en virgule flottante 0,7 + 0,2 + 0,1 produit un nombre à virgule flottante qui n'est pas tout à fait 1.0. Int() corrects for this anomaly, however, certain extremely rare circumstances may lead to Int() returning an unexpected value (the odds of getting an unexpected value are less than if Int() did not attempt any correction at all).
