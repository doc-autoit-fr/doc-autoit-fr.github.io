ObjGet

Retrieves a reference to a COM object from an existing process or filename.Récupère une référence sur un objet COM à partir d'un processus ou d'un fichier existant.

ObjGet ( "filename" [, "classname" [, instance]] )

ParametersParamètres

filename The full path and name to the file containing the object (See remarks).Le chemin d'accès complet et le nom du fichier contenant l'objet (Voir Remarques ).
classname [optional][optionnel] Class identifier.Identifiant de classe. Can be in either ProgID or the string representation of the CLSID.Peut être ProgID ou la représentation chaîne du CLSID.
instance [optional][optionnel] Instance of the object for ROT objects of the same (co)class.Instance de l'objet pour les objets ROT da la même classe.

Return ValueValeur de retour

Success:Succès : an object.Retourne un objet.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

The filename is optional if you want to use only the class identifier, but the parameter can't be omitted.Le nom du fichier est facultatif si vous souhaitez utiliser uniquement l'identifiant de classe, mais le paramètre ne peut pas être omis.
Use an empty string if you want to access objects from running objects table (ROT).Utilisez une chaîne vide si vous voulez accéder à des objets de la Table des Objets en cours d'Exécution (Running Object Table). You can use third parameter to specify wanted instance in that case.Vous pouvez utiliser la troisième paramètre pour spécifier l'instance désirée dans ce cas.
In all other cases third parameter is ignored.Dans tous les autres cas, la troisième paramètre est ignoré.

If you use a filename, the classname is optional.Si vous utilisez un nom de fichier, le nom de classe est facultatif. It's only required when you want to load an object of a specific class.Il est uniquement nécessaire si vous voulez charger un objet d'une classe spécifique.
Classname parameter does not have default value.Le paramètre Classname ne peut pas avoir une valeur par défaut. If it's specified then it's processed.Si il est spécifié, alors il est traité.

See the Obj/COM Reference for more information about Objects.Consultez Référence Obj/COM pour plus d'informations sur les objets.

Func Example()
    Local $oExcel = ObjGet("", "Excel.Application") ; Get an existing Excel ObjectObtient un objet Excel existant

    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Excel File Test" & @CRLF & "Error Getting an active Excel Object. Error code: " & Hex(@error, 8))

    EndIf

    $oExcel.Visible = 1 ; Let the guy show himselfAffiche la gui

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $sFileName = @ScriptDir & "\Worksheet.xls"

    If Not FileExists($sFileName) Then
        MsgBox($MB_SYSTEMMODAL, "", "Excel File Test" & @CRLF & "Can't run this test, because you didn't create the Excel file " & $sFileName)
        Return False
    EndIf

    Local $oExcelDoc = ObjGet($sFileName) ; Get an Excel Object from an existing filenameObtient un objet Excel à partir d'un nom de fichier existant

    If IsObj($oExcelDoc) Then

        ; Tip:Astuce: Uncomment these lines to make Excel visible (credit:Supprimez ces lignes pour rendre Excel visible:(crédit : DaleHohm)
        ; $oExcelDoc.Windows(1).Visible = 1; Set the first worksheet in the workbook visible
        ; $oExcelDoc.Application.Visible = 1; Set the application visible (without this Excel will exit)

        Local $sString = "" ; String for displaying purposesChaîne à des fins d'affichage

        For $oProperty In $oExcelDoc.BuiltinDocumentProperties
            $sString &= $oProperty.Name & ":" & $oProperty.Value & @CRLF
        Next

        MsgBox($MB_SYSTEMMODAL, "", "Excel File Test:" & @CRLF & "The document properties of " & $sFileName & " are:" & @CRLF & @CRLF & $sString)

        $oExcelDoc.Close ; Close the Excel documentFerme le document Excel

    Else
        MsgBox($MB_SYSTEMMODAL, "", "Excel File Test" & @CRLF & "Error: Could not open " & $sFileName & " as an Excel Object.")
    EndIf
EndFunc   ;==>Example
