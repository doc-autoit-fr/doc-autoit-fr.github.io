GUICtrlCreateListView

Creates a ListView control for the GUI.Crée un contrôle ListView dans une GUI.

GUICtrlCreateListView ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

text definition of columns heading.Définition des entêtes des colonnes. Each of them are separated with Opt("GUIDataSeparatorChar").Elles sont séparées avec Opt("GUIDataSeparatorChar").
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default (-1) : $LVS_SHOWSELALWAYS, $LVS_SINGLESEL
    forced style : $LVS_REPORT
exStyle [optional][optionnel] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table or ListView Extended Style Table.Consultez Extended Style Table ou ListView Extended Style Table.
    default (-1) : $LVS_EX_FULLROWSELECT, $WS_EX_CLIENTEDGE

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

To add items to a ListView control use GUICtrlCreateListViewItem()Pour ajouter des items à un contrôle ListView utilisez GUICtrlCreateListViewItem()

The ListView will appear by default as in the Explorer view "Details" (LVS_REPORT style is forced).Le ListView apparaîtra par défaut comme dans la vue des détails de l' Explorer (le style LVS_REPORT est forcé).
You can control initial column size by padding blanks to the column heading definition.Vous pouvez contrôler la taille initiale des colonnes par des blancs de remplissage dans la définition des entêtes de colonnes. The column can be extend during the GUICtrlCreateListViewItem() according to item size.La colonne peut être étendue avec GUICtrlCreateListViewItem() selon la taille de l'item. Size of a column will be up to around 25 characters.La taille d'une colonne ira jusqu'à environ 25 caractères. No resizing will be done during an update by GUICtrlSetData().Aucun redimensionnement se fera lors d'une mise à jour par GUICtrlSetData().

To create a ListView with Icon-, SmallIcon- or List-style just use after creation:Pour créer un ListView avec icône, petit icône ou style de liste, utilisez après création:
GUICtrlSetStyle() with the styles $LVS_ICON, $LVS_LIST or $LVS_SMALLICON.GUICtrlSetStyle() avec les styles $LVS_ICON, $LVS_LIST or $LVS_SMALLICON.

Sorting the list by clicking the column name (as in Explorer) is not currently implemented.Le tri d'une liste en cliquant sur le nom de la colonne (comme dans l' Explorer) n'est pas mis en oeuvre actuellement.

To have the entire line visually selected use the extended style LVS_EX_FULLROWSELECT.Pour avoir la ligne entière visuellement sélectionnée utilisez le extended style LVS_EX_FULLROWSELECT.

To combine styles with the default style use BitOR($GUI_SS_DEFAULT_LISTVIEW, newstyle, ... ).)..
To use the values specified above you must #include <ListViewConstants.au3> in your script.

If you add the style $LVS_EX_CHECKBOXES to the ListView, then the borders are removed.Si vous ajoutez le style $LVS_EX_CHECKBOXES à la ListView, alors les bordures sont enlevées. Add the style $WS_EX_CLIENTEDGE to display the borders.Ajoutez le style $WS_EX_CLIENTEDGE pour afficher les bordures.


The special flag $GUI_BKCOLOR_LV_ALTERNATE can be used with Listview control to give alternate background of the ListviewItems lines.Le drapeau spécial $GUI_BKCOLOR_LV_ALTERNATE peut être utilisé avec le contrôle ListView pour lui donner un fond différent suivant les lignes des ListviewItems.
The odd lines will get the color set by GUICtrlSetBkColor() of the Listview control.Les lignes impaires obtiendront la couleur définie par GUICtrlSetBkColor() du contrôle ListView.
The even lines will get the color set by GUICtrlSetBkColor() of the ListviewItem control.Les lignes paires auront la couleur définie par GUICtrlSetBkColor() du contrôle ListViewItem.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsConstants.au3>

Example()

Func Example()
    GUICreate("listview items", 220, 250, 100, 200, -1, $WS_EX_ACCEPTFILES)
    GUISetBkColor(0x00E0FFFF) ; will change background colorchange la couleur de fond

    Local $idListview = GUICtrlCreateListView("col1 |col2|col3  ", 10, 10, 200, 150) ;,$LVS_SORTDESCENDING), $LVS_SORTDESCENDING)
    Local $idButton = GUICtrlCreateButton("Value?", 75, 170, 70, 20)
    Local $idItem1 = GUICtrlCreateListViewItem("item2|col22|col23", $idListview)
    Local $idItem2 = GUICtrlCreateListViewItem("item1|col12|col13", $idListview)
    Local $idItem3 = GUICtrlCreateListViewItem("item3|col32|col33", $idListview)
    GUICtrlCreateInput("", 20, 200, 150)
    GUICtrlSetState(-1, $GUI_DROPACCEPTED) ; to allow drag and droppingpermet le glisser-déposer
    GUISetState(@SW_SHOW)
    GUICtrlSetData($idItem2, "ITEM1")
    GUICtrlSetData($idItem3, "||COL33")
    GUICtrlDelete($idItem1)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idButton
                MsgBox($MB_SYSTEMMODAL, "listview item", GUICtrlRead(GUICtrlRead($idListview)), 2)

            Case $idListview
                MsgBox($MB_SYSTEMMODAL, "listview", "clicked=" & GUICtrlGetState($idListview), 2)

        EndSwitch
    WEnd
EndFunc   ;==>Example
