GUICtrlSetFont

Sets the font for a control.Définit la police d'un contrôle.

GUICtrlSetFont ( controlID, size [, weight [, attribute [, fontname [, quality]]]] )

ParametersParamètre

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
size Fontsize (default is 8.5).Taille de la police (par défaut 8.5).
weight [optional][optional] False - i, v, x ... The weight of the font in the range 0 through 1000.La graisse de la police de 0 à 1000. For example, 400 is normal and 700 is bold.Par exemple, 400 est normal et 700 est gras. If this value is zero, a default weight is used.Si cette valeur est 0, une graisse par défaut est utilisée.
The following values are defined for convenience.Les valeurs suivantes sont définies pour en faciliter l'utilisation.
    $FW_DONTCARE = 0 (Use the default font weight)
    $FW_THIN = 100
    $FW_EXTRALIGHT = 200
    $FW_LIGHT = 300
    $FW_NORMAL = 400
    $FW_MEDIUM = 500
    $FW_SEMIBOLD = 600
    $FW_BOLD = 700
    $FW_EXTRABOLD = 800
    $FW_HEAVY = 900

Constants are defined in FontConstants.au3.
attribute [optional][optional] False - i, v, x ... Font attributes, which can be a combination of the following added together:Attributs de police, qui peut être une combinaison des éléments suivants additionnés :
    $GUI_FONTITALIC (2) = Italic
    $GUI_FONTUNDER (4) = Underlined
    $GUI_FONTSTRIKE (8) = Strike

Constants are defined in GUIConstantsEx.au3.
fontname [optional][optional] False - i, v, x ... Name of the font to use.Nom de la police à utiliser. (OS default GUI font is used if the font is "" or is not found).(police par défaut de l'OS utilisée si la police est "" ou introuvable).
quality [optional][optional] False - i, v, x ... Font quality to select.Qualité de la police à sélectionner. The following qualities are accepted:Les qualités suivantes sont acceptées:
    $DEFAULT_QUALITY (0) = Appearance of the font does not matter.
    $DRAFT_QUALITY (1) = Appearance of the font is less important than when $PROOF_QUALITY is used. For GDI raster fonts, scaling is enabled, which means that more font sizes are available, but the quality may be lower. Bold, italic, underline, and strikeout fonts are synthesized if necessary.Les polices gras, italique, souligné, et barré sont synthétisés si nécessaire.
    $PROOF_QUALITY (2) = (default) Character quality of the font is more important than exact matching of the logical-font attributes. For GDI raster fonts, scaling is disabled and the font closest in size is chosen.Pour les polices raster GDI, la mise à l'échelle est désactivée et la police la plus proche en taille est choisie. Although the chosen font size may not be mapped exactly when $PROOF_QUALITY is used, the quality of the font is high and there is no distortion of appearance.Bien que la taille de la police choisie ne peut pas être mappé exactement quand $PROOF_QUALITY est utilisé, la qualité de la police est élevé et il n'y a pas de distorsion de l'apparence. Bold, italic, underline, and strikeout fonts are synthesized if necessary.Les polices gras, italique, souligné, et barré sont synthétisés si nécessaire.
    $NONANTIALIASED_QUALITY (3) = Font is never antialiased.
    $ANTIALIASED_QUALITY (4) = Font is always antialiased if the font supports it and the size of the font is not too small or too large.
    $CLEARTYPE_QUALITY (5) = If set, text is rendered (when possible) using ClearType antialiasing method. See the remarks on the msdn page for LOGFONT for details about when cleartype is not available.Voir Remarques sur la page MSDN pour LOGFONT pour savoir quand ClearType n'est pas disponible.
If neither $ANTIALIASED_QUALITY nor $NONANTIALIASED_QUALITY is selected, the font is antialiased only if the user chooses smooth screen fonts in Control Panel.Si ni $ANTIALIASED_QUALITY ni $NONANTIALIASED_QUALITY n'est sélectionné, la police est anti-crénelée seulement si l'utilisateur choisit les polices d'écran lisses dans le Panneau de configuration.
Constants are defined in FontConstants.au3Les constantes sont définies dans FontConstants.au3

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0.Retourne 0.

Remarks<i0>

By default, controls use the font set by GUISetFont().Par défaut, les contrôles utilisent la police définit par GUISetFont().
See the Appendix for a complete list of Windows fonts and the Windows versions under which they are supported.Consultez l'Appendice pour une liste complète des polices Windows et les versions de Windows sous lesquelles elles sont supportées.

Size can contain a decimal as in 8.5.La taille peut contenir un nombre décimal comme 8.5.

For some controls such as labels, the default size can be 8.5 instead of 9 according to Windows Theme Values.Pour certains contrôles tels que les Label, la taille par défaut peut être de 8,5 au lieu de 9 en accord avec les valeurs du thème de Windows.

For Quality parameter check MSDN, some Windows XP installation need CLEARTYPE_QUALITY=5Pour le paramètre Quality consultez MSDN, certaines installations de Windows XP nécessite CLEARTYPE_QUALITY = 5

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Font type to be used for setting the font of the controls.Type de police à utiliser pour les contrôles.
    Local Const $sFont = "Comic Sans Ms"

    ; Create a GUI with various controls.Crée une GUI avec différents contrôles.
    Local $hGUI = GUICreate("Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>", 300, 200)

    ; Create label controls.Crée des contrôles Label.
    GUICtrlCreateLabel("A string of text underlined", 10, 10, 185, 17)
    GUICtrlSetFont(-1, 9, $FW_NORMAL, $GUI_FONTUNDER, $sFont) ; Set the font of the previous control.Définit la police du contrôle précédent.

    Local $idLabel2 = GUICtrlCreateLabel("A string of italic text", 10, 30, 185, 17)
    GUICtrlSetFont($idLabel2, 9, $FW_NORMAL,  $GUI_FONTITALIC, $sFont) ; Set the font of the controlID stored in $iLabel2.Définit la police de la controlID stocké dans $iLabel2.

    Local $idLabel3 = GUICtrlCreateLabel("A string of text with a strike through", 10, 50, 290, 17)
    GUICtrlSetFont($idLabel3, 9, $FW_NORMAL, $GUI_FONTSTRIKE, $sFont) ; Set the font of the controlID stored in $iLabel3.Définit la police de la controlID stocké dans $iLabel3.

    Local $idClose = GUICtrlCreateButton("Close", 210, 170, 85, 25)

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Loop until the user exits.Création d'un objet chemin
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idClose
                ExitLoop

        EndSwitch
    WEnd

    ; Delete the previous GUI and all controls.Supprime la GUI précédente et tous ses contrôles.
    GUIDelete($hGUI)
EndFunc   ;==>Example
