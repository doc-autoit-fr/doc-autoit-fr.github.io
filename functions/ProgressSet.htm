ProgressSet

Sets the position and/or text of a previously created Progress bar window.Définit la position et/ou le texte d'une fenêtre de barre de progression précédemment créé.

ProgressSet ( percent [, "subtext" [, "maintext"]] )

ParametersParamètres

percent Percentage (value between 0.Pourcentage (valeur comprise entre 0. and 100.) to set the progress bar at.et 100.)pour régler la barre de progression.
subtext [optional][optionnel] Set the text for the Sub, Normal, Lower label.Sélectionnez le texte pour le Sub, Normale, Lower label.
maintext [optional][optionnel] Set the text for the Main, Bold, Upper label.Sélectionnez le texte pour le Main, Bold, Upper label.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

Notice that the subtext argument comes before maintext.Notez que l'argument de sous-texte vient avant mainText.

RelatedLiens

ProgressOff, ProgressOn

ExampleExemple

Example()

Func Example()
    ; Display a progress bar window.Affiche une fenêtre de barre de progression.
    ProgressOn("Progress Meter", "Increments every second", "0%")

    ; Update the progress value of the progress bar window every second.Met à jour la valeur de la progression de la fenêtre de la barre de progression à chaque seconde.
    For $i = 10 To 100 Step 10
        Sleep(1000)
        ProgressSet($i, $i & "%")
    Next

    ; Set the "subtext" and "maintext" of the progress bar window.Régle le "sous-texte" et "MainText" de la fenêtre de la barre de progression.
    ProgressSet(100, "Done", "Complete")
    Sleep(5000)

    ; Close the progress window.Ferme la fenêtre de progression.
    ProgressOff()
EndFunc   ;==>Example
