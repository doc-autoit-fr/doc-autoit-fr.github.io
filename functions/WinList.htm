WinList

Retrieves a list of windows.Récupère une liste de fenêtres.

WinList ( ["title" [, "text"]] )

ParametersParamètres

title [optional][optionnel][optionnel] The title/hWnd/class of the windows to get the list.Le titre/hWnd/class des fenêtres à lister. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the windows to get the list.Le texte des fenêtres à lister. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Returns an array of matching window titles and handles.Retourne un tableau correspondant aux titres et handles des fenêtres trouvées.

RemarksRemarques

If no title and text is given then all top-level windows are returned.Si aucun titre ou texte n'est donné, alors toutes les fenêtres de niveau supérieur sont retournées.

The array returned is two-dimensional and is made up as follows:Le tableau retourné est un tableau à deux dimensions composé comme ceci :
    $aArray[0][0] = Number of windows returned
    $aArray[1][0] = 1st window title
    $aArray[1][1] = 1st window handle (HWND)
    $aArray[2][0] = 2nd window title
    $aArray[2][1] = 2nd window handle (HWND)
    ...
    $aArray[n][0] = nth window title
    $aArray[n][1] = nth window handle (HWND)

Example 1Exemple 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Retrieve a list of window handles.Récupère une liste de handles de fenêtre.
    Local $aList = WinList()

    ; Loop through the array displaying only visable windows with a title.Boucle dans le tableau pour afficher uniquement les fenêtres visibles avec un titre.
    For $i = 1 To $aList[0][0]
        If $aList[$i][0] <> "" And BitAND(WinGetState($aList[$i][1]), 2) Then
            MsgBox($MB_SYSTEMMODAL, "", "Title: " & $aList[$i][0] & @CRLF & "Handle: " & $aList[$i][1])
        EndIf
    Next
EndFunc   ;==>Example

Func Example()
    ; Retrieve a list of window handles using a regular expression.Récupère une liste de handles de fenêtre à l'aide d'une expression régulière.     Local $aWinList = WinList("[REGEXPTITLE:(?i)(.*SciTE.*|.*Internet Explorer.*)]")
