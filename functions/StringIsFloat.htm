StringIsFloat

Checks if a string is a floating point number.Vérifie si une chaîne contient est un nombre à virgule flottante.

StringIsFloat ( "string" )

ParametersParamètres

string The string or expression to check.La chaîne ou l'expression à vérifier.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if not floating point format.Retourne 0 si ce n'est pas le format d'un nombre à virgule flottante.

RemarksRemarques

A string is a float if it contains at least one decimal digit and exactly one period; the only other character allowed is an optional plus or minus at the beginning.Une chaîne est un 'float' si elle contient au moins un chiffre décimal et un seul point, les seuls autres caractères permis sont un plus (+) ou un moins (-) optionnel au début. (StringIsFloat() does not accept a comma character as the decimal point even if the computer Locale settings use that symbol.)(StringIsFloat() n'accepte pas le caractère virgule comme séparateur décimal, même si les paramètres régionaux de l'ordinateur utilisent ce symbole). If parameter is not a string, its value is converted to a string.Si le paramètre n'est pas une chaîne, sa valeur est convertie en une chaîne. See examples.Voir les exemples.

ExampleExemple

#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "", "Is the string 1.5 a floating point number: " & StringIsFloat("1.5") & @CRLF & _ ; Returns 1, due to the decimal point.Retourne 1, en raison de la virgule.
        "Is the string 7. a floating point number: " & StringIsFloat("7.") & @CRLF & _ ; Returns 1, due to the decimal point.Retourne 1, en raison de la virgule.
        "Is the string 3/4 a floating point number: " & StringIsFloat("3/4") & @CRLF & _ ; Returns 0, as the string 3 / (forward slash) 4 is not a floating point number.Retourne 0, car la chaîne 3/4 n'est pas un nombre à virgule flottante.
        "Is the number 1.0 a floating point number: " & StringIsFloat(1.0) & @CRLF & _ ; Returns 1, due to the number to string conversion as well as the decimal point.Retourne 1.
        "Is the string 2 a floating point number: " & StringIsFloat("2") & @CRLF) ; Returns 0, due to 2 being an integer.Retourne 0, car 2 est un nombre 'integer'.
