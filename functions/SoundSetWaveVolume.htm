SoundSetWaveVolume

Sets the system wave volume by percent.Règle le volume du son du système en pourcentage.

SoundSetWaveVolume ( percent )

ParametersParamètre

percent percentage number between 0 and 100nombre en pourcentage entre 0 et 100

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 and sets the @error flag to 1 if percent is invalid.Retourne 0 et @error = 1 si pourcent est invalide.

Remarks<i0>

This controls the Wave volume, not the master volume control.Cette fonction contrôle le volume du son, pas le contrrôle du volume maître. Also, a value of Zero does not set mute status.En outre, une valeur de zéro ne définit pas le statut muet.

On Windows Vista, there is no system-wide wave volume.Sous Windows Vista, il n'y a pas de volume du son du système. This function only changes the wave volume for the script.Cette fonction change seulement le volume du son pour le script. It can not be used to change the wave volume of other programs.Elle ne peut pas être utilisée pour modifier le volume du son d'autres programmes.
