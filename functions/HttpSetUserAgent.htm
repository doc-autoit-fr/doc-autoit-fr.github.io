Référence Fonction

HttpSetUserAgent

Sets the HTTP user-agent string which is sent with all Inet requests.Configure la chaîne User-Agent envoyée par toutes les requêtes Inet.

HttpSetUserAgent ( "user agent" )

ParametersParamètres

user agent The string to set as the user-agent.La chaîne à utiliser pour le User-Agent. The default user-agent is "AutoIt".Le User-Agent par défaut est "AutoIt". Passing an empty string will reset the user-agent back to the default.Passer une chaine vide comme user-agent, réinitialisera sa valeur par défaut.

Return ValueValeur de retour

The previous user agent.La chaîne précédente user-agent.

RelatedLiens

InetGet, InetGetSize, InetRead

ExampleExemple

HttpSetUserAgent("MyUserAgent")
