GUICtrlSetColor

Sets the text color of a control.Définit la couleur du texte d'un contrôle.

GUICtrlSetColor ( controlID, textcolor )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
textcolor The RGB color to use.La couleur RVB à utiliser.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

Only Button, Label, Checkbox, Group, Radio, Edit, Input, List, Listview, ListviewItem, Treeview, TreeviewItem, Graphic, Progress and Combo controls can currently be colored.Seuls les contrôles Button, Label, Checkbox, Group, Radio, Edit, Input, List, Listview, ListviewItem, Treeview, TreeviewItem, Graphic, Progress et Combo peuvent actuellement être coloré.

Checkbox, Radio, Group or Progress controls cannot be painted if the "Windows XP/Vista style" is used.Les contrôles Checkbox, Radio, Group or Progress ne peuvent pas être peints si le "style de Windows XP/Vista" est utilisé.

Button controls are always painted in "Windows Classic style".Les contrôles Button sont toujours peints en "style Windows Classique".

ExampleExemple

#include <ColorConstants.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Create a GUI with various controls.Crée une GUI avec différents contrôles.
    Local $hGUI = GUICreate("ExampleExemple", 300, 200)

    ; Create a label control.Crée un contrôle Label.
    Local $idLabel = GUICtrlCreateLabel("A string of text", 10, 10, 185, 17)
    Local $idClose = GUICtrlCreateButton("Close", 210, 170, 85, 25)

    ; Set the color of the label control.Définit la couleur du contrôle Label.
    GUICtrlSetColor($idLabel, $COLOR_RED)

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idClose
                ExitLoop

        EndSwitch
    WEnd

    ; Delete the previous GUI and all controls.Supprime la GUI précédente et tous ses contrôles.
    GUIDelete($hGUI)
EndFunc   ;==>Example
