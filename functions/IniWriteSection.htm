IniWriteSection

Writes a section to a standard format .ini file.Écrit une section dans un fichier ini de format standard.

IniWriteSection ( "filename", "section", "data" [, index = 1] )

ParametersParamètres

filename The filename of the .ini file.Le nom du fichier ini.
section The section name in the .ini file.Le nom de la section dans le fichier ini.
data The data to write.La donnée à écrire. The data can either be a string or an array.La donnée peuvent être une chaîne ou un tableau. If the data is a string, then each key=value pair must be delimited by @LF.Si la donnée est une chaîne, chaque paire clé/valeur doit être délimitée par @LF. If the data is an array, the array must be 2-dimensional and the second dimension must be 2 elements.Si la donnée est un tableau, le tableau doit être de dimension 2 et la deuxième dimension doit être de 2 éléments.
index [optional][optionnel] If an array is passed as data, this specifies the index to start writing from.Si un tableau est transmis en données, ce paramètre indique l'indice à partir duquel l'écriture doit commencer. By default, this is 1 so that the return value of IniReadSection() can be used immediately.Par défaut, c'est 1 de telle façon que la valeur de retour de IniReadSection() peut être utilisée immédiatement. For manually created arrays, this value may need to be different depending on how the array was created.Pour les tableaux créés à la main, cette valeur peut être différente en fonction de la façon dont le réseau a été créé. This parameter is ignored if a string is passed as data.Ce paramètre est ignoré si une chaîne est passée comme donnée.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0. The function will sets the @error flag to 1 if the data format is invalid.

RemarksRemarques

A standard ini file looks like:Un fichier ini standard ressemble à :
[SectionName][NomdelaSection]
Key=Valueclé=Valeur

If file does not exist, it is created.Si le fichier n'existe pas, il est créé. Any directories that do not exist, will not be created.Tous les répertoires qui n'existent pas, ne seront pas créés. Keys and/or sections are added to the end and are not sorted in any way.Les clés et/ou les sections sont ajoutés à la fin et ne sont pas triés en aucune façon.
If the section being written already exists, its contents will be overwritten.Si la section à écrire existe déjà, son contenu sera écrasé.

If you want to use an ini file with unicode encoding, first create an .ini file by using the FileOpen() function with the mode parameter set to "Unicode UTF16 Little Endian".Si vous souhaitez utiliser un fichier ini avec le codage Unicode, créez d'abord un fichier ini en utilisant la fonction FileOpen() avec le paramètre de mode réglé sur "Unicode UTF16 Little Endian".

ExampleExemple

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create an INI section structure as an array.Crée une structure de section INI dans un tableau. The zeroth element is how many items are in the array, in this case 3.L'élément zéro contient le nombre d'éléments du tableau, dans ce cas 3.
    Local $aSection[4][2] = [[3, ""], ["Title", "AutoIt"], ["Version", @AutoItVersion], ["OS", @OSVersion]]

    ; Write the array to the section labelled 'General'.Ecrit le tableau à la section 'General'.
    IniWriteSection($sFilePath, "General", $aSection)

    ; Read the INI section labelled 'General'.Lit la section INI 'General'. This will return a 2 dimensional array.Ce qui renvoie un tableau à 2 dimensions.
    Local $aArray = IniReadSection($sFilePath, "General")

    ; Check if an error occurred.Vérifie si une erreur s'est produite.
    If Not @error Then
        ; Enumerate through the array displaying the keys and their respective values.Énumére le tableau pour afficher les clés et leurs valeurs respectives.
        For $i = 1 To $aArray[0][0]
            MsgBox($MB_SYSTEMMODAL, "", "Key: " & $aArray[$i][0] & @CRLF & "Value: " & $aArray[$i][1])
        Next
    EndIf

    ; Delete the INI file.Supprime le fichier INI.
    FileDelete($sFilePath)
EndFunc   ;==>Example
