FileOpenDialog

Initiates a Open File Dialog.Ouvre un dialogue permettant à l'utilisateur de choisir un fichier.

FileOpenDialog ( "title", "init dir", "filter" [, options = 0 [, "default name" [, hwnd]]] )

ParametersParamètre

title Title text of the Dialog GUI.Titre de la boîte de dialogue.
init dir Initial directory selected in the GUI file tree.Répertoire initial proposé à l'utilisateur.
filter File type single filter such as "All (*.*)" or "Text files (*.txt)" or multiple filter groups such as "All (*.*)|Text files (*.txt)" (See Remarks).Filtre de type de fichier unique tel que "Tous les fichiers (*.*)" ou "Fichiers texte (*.txt)" ou plusieurs groupes de filtres tels que "Tous les fichiers (*.*)|Fichiers texte (*.txt)" (Voir Remarques).
options [optional][optional] False - i, v, x ... Dialog Options:Options de dialogue: To use more than one option, BitOR the required values together.Pour utiliser plusieurs options, rassemblez-les avec BitOr.
    $FD_FILEMUSTEXIST (1) = File Must Exist (if user types a filename)
    $FD_PATHMUSTEXIST (2) = Path Must Exist (if user types a path, ending with a backslash)
    $FD_MULTISELECT (4) = Allow MultiSelect
    $FD_PROMPTCREATENEW (8) = Prompt to Create New File (if does not exist)

Constants are defined in FileConstants.au3.
default name [optional][optional] False - i, v, x ... Suggested file name for the user to open.Nom de fichier suggéré à l'ouverture. Default is blank ("").Par défaut est vide ("").
hwnd [optional][optional] False - i, v, x ... The window handle to use as the parent for this dialog.Le handle de la fenêtre à utiliser comme fenêtre parent pour ce dialogue.

Return ValueRemarque

Success: the full path of the file(s) chosen.Retourne le chemin complet du(des) fichier(s) choisi. Results for multiple selections are "Directory|file1|file2|...".Les résultats pour les sélections multiples sont "Répertoire|fichier1|fichier2|..."
Failure: sets the @error flag to non-zero.Retourne @error
@error: 1 = File selection failed.1 - La sélection de fichier a échoué.
2 = Bad file filter2 - Filtre de fichier incorrect.

Remarks<i0>

Separate the file filters with a semicolon as shown in the example.Séparez les filtres de fichiers avec un point-virgule comme le montre l'exemple.
Multiple groups of filters are separated by a pipe "|".Plusieurs groupes de filtres sont séparés par une barre verticale "|".

If default name is given, options must also be given.Si le nom par défaut est donné, les options doivent également être donnée. If none of the options are wanted, use 0 for options.Si une option n'est pas désirée, utilisez 0 pour cette option.

Special Windows folders (such as "My Documents") can be set as root by using the right CLSID detailed in the Appendix.Les dossiers Windows spéciaux (tels que "Mes Documents") peuvent être définis en tant que root en utilisant le bon CLSID détaillée dans l'Appendix.

@WorkingDir is changed on successful return.

Example 1Exemple 1

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the message to display in FileOpenDialog.Créez une variable constante portée locale du message à afficher dans FileOpenDialog.
    Local Const $sMessage = "Hold down Ctrl or Shift to choose multiple files."

    ; Display an open dialog to select a list of file(s).Ouvre une boîte de dialogue pour sélectionner une liste de fichier(s).
    Local $sFileOpenDialog = FileOpenDialog($sMessage, @WindowsDir & "\", "Images (*.jpg;*.bmp)|Videos (*.avi;*.mpg)", $FD_FILEMUSTEXIST + $FD_MULTISELECT)
    If @error Then
        ; Display the error message.Affiche le message d'erreur.
        MsgBox($MB_SYSTEMMODAL, "", "No file(s) were selected.")

        ; Change the working directory (@WorkingDir) back to the location of the script directory as FileOpenDialog sets it to the last accessed folder.Change le répertoire de travail (@WorkingDir) vers l'emplacement du répertoire de script comme FileOpenDialog l'a défini au dernier dossier consulté.
        FileChangeDir(@ScriptDir)
    Else
        ; Change the working directory (@WorkingDir) back to the location of the script directory as FileOpenDialog sets it to the last accessed folder.Change le répertoire de travail (@WorkingDir) vers l'emplacement du répertoire de script comme FileOpenDialog l'a défini au dernier dossier consulté.
        FileChangeDir(@ScriptDir)

        ; Replace instances of "|" with @CRLF in the string returned by FileOpenDialog.Remplace les instances de "|" par @CRLF dans la chaîne renvoyée par FileOpenDialog.
        $sFileOpenDialog = StringReplace($sFileOpenDialog, "|", @CRLF)

        ; Display the list of selected files.Affiche la liste des fichiers sélectionnés.
        MsgBox($MB_SYSTEMMODAL, "", "You chose the following files:" & @CRLF & $sFileOpenDialog)
    EndIf
EndFunc   ;==>Example

        ; Change the working directory (@WorkingDir) back to the location of the script directory as FileOpenDialog sets it to the last accessed folder.Change le répertoire de travail (@WorkingDir) vers l'emplacement du répertoire de script comme FileOpenDialog l'a défini au dernier dossier consulté.
        FileChangeDir(@ScriptDir)
    Else
        ; Change the working directory (@WorkingDir) back to the location of the script directory as FileOpenDialog sets it to the last accessed folder.Change le répertoire de travail (@WorkingDir) vers l'emplacement du répertoire de script comme FileOpenDialog l'a défini au dernier dossier consulté.
        FileChangeDir(@ScriptDir)

        ; Display the selected file.Affiche le fichier sélectionné.
        MsgBox($MB_SYSTEMMODAL, "", "You chose the following file:" & @CRLF & $sFileOpenDialog)
    EndIf
EndFunc   ;==>Example
