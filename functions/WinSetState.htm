WinSetState

Shows, hides, minimizes, maximizes, or restores a window.Montre, cache, minimise, maximise ou restaure une fenêtre.

WinSetState ( "title", "text", flag )

ParametersParamètres

title The title/hWnd/class of the window to change the state.Le titre/hWnd/class de la fenêtre dont l'état doit changer. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to change the state.Le texte de la fenêtre dont l'état doit changer. See Text special definition.Consultez Définition spéciale de Texte .
flag The "show" flag of the executed program:L'option de visualisation du programme exécuté:
    @SW_HIDE = Hide window
    @SW_SHOW = Shows a previously hidden window
    @SW_MINIMIZE = Minimize window
    @SW_MAXIMIZE = Maximize window
    @SW_RESTORE = Undoes a window minimization or maximization
    @SW_DISABLE = Disables the window
    @SW_ENABLE = Enables the window

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if the window is not found.Retourne 0 si la fenêtre n'existe pas.

RemarksRemarques

If multiple windows match the criteria, the most recently active window is used.Si plusieurs fenêtres correspondent aux critères, la fenêtre active la plus récente sera utilisé.
@SW_MINIMIZE and @SW_MAXIMIZE even work on modal dialog windows.@SW_MINIMIZE et @SW_MAXIMIZE fonctionnent aussi sur les boîtes de dialogue modales Windows.

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Set the state of the Notepad window to "hide".Définit l'état de la fenêtre du Bloc-notes à "cachée".
    WinSetState($hWnd, "", @SW_HIDE)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Set the state of the Notepad window to "show".Définit l'état de la fenêtre du Bloc-notes de "affichée".
    WinSetState($hWnd, "", @SW_SHOW)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
