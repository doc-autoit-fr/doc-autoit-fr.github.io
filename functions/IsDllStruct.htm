IsDllStruct

Checks if a variable is a DllStruct type.Vérifie si une variable est de type DllStruct.

IsDllStruct ( variable )

ParametersParamètres

variable The variable/expression to check.La variable/expression à vérifier.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 if expression is not DllStruct type as return by DllStructCreate.Retourne 0 si l'expression n'est pas de type DllStruct comme la valeur de retour de DllStructCreate.

RemarksRemarques

See language datatypes for a detailed description.Consultez Types de données du langage pour une description détaillée.

RelatedLiens

DllStructCreate, VarGetType

ExampleExemple

#include <MsgBoxConstants.au3>

Local $tStruct = DllStructCreate("wchar[256]")
If IsDllStruct($tStruct) Then
    MsgBox($MB_SYSTEMMODAL, "", "The variable is a dll structure")
Else
    MsgBox($MB_SYSTEMMODAL, "", "The variable is not a dll structure")
EndIf
