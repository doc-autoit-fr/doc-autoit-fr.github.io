GUICtrlCreateTabItem

Creates a TabItem control within an existing tab control in the GUI.Crée un contrôle TabItem dans un contrôle Tab existant dans une GUI.

GUICtrlCreateTabItem ( "text" )

ParametersParamètres

text The text of the control.Le texte de la commande.

Return ValueValeur de retour

Success: the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure: 0.Retourne 0.

RemarksRemarques

To select a specific tabitem to be shown when the GUI opens use GUICtrlSetState(-1, $GUI_SHOW) as shown in the example.Pour sélectionner un TabItem spécifique à afficher lorsque la GUI s'ouvre, utilisez GUICtrlSetState( -1, $ GUI_SHOW ), comme illustré dans l'exemple.

It is important to close the tab structure by creating a final tabitem control with a null text - GUICtrlCreateTabItem("").Il est important de fermer la structure Tab en créant un contrôle TabItem finale avec un texte null - GUICtrlCreateTabItem("").

Controls for a specific tabitem should be created after the tabitem and before the creating a further tabitem or closing the tab structure.Les contrôles pour un TabItem spécifique devraient être créés après la TabItem et avant la création d'une nouveau TabItem ou la fermeture de la structure du Tab. To create a new control on an existing tabitem, use GUISwitch($hWin, $tabitem) to select the correct tabitem and then create the new control.Pour créer un nouveau contrôle dans un TabItem existant, utilisez GUISwitch($hWin, $tabitem) pour choisir le TabItem correct et ensuite créer le nouveau contrôle. Do not forget to close the tabitem structure once again with GUICtrlCreateTabItem("").N'oubliez pas de fermer la structure TabItem avec GUICtrlCreateTabItem("").

The tabitem control cannot be colored (too much code ...).Le contrôle TabItem ne peut être coloré ( trop de code...).

GUICtrlRead() will return the index of the clicked tab item, or the controlID when advanced mode is used.GUICtrlRead() retournera l'index du TabItem cliqué, ou le controlID lorsque le mode avancé est utilisé.

To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI Tab", 250, 150); will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUISetBkColor(0x00E0FFFF)
    GUISetFont(9, 300)

    Local $idTab = GUICtrlCreateTab(10, 10, 200, 100)

    GUICtrlCreateTabItem("tab0")
    GUICtrlCreateLabel("label0", 30, 80, 50, 20)
    GUICtrlCreateButton("OK0", 20, 50, 50, 20)
    GUICtrlCreateInput("default", 80, 50, 70, 20)

    GUICtrlCreateTabItem("tab----1")
    GUICtrlCreateLabel("label1", 30, 80, 50, 20)
    GUICtrlCreateCombo("", 20, 50, 60, 120)
    GUICtrlSetData(-1, "Trids|CyberSlug|Larry|Jon|Tylo|guinness", "Jon"); default JonJon défaut
    GUICtrlCreateButton("OK1", 80, 50, 50, 20)

    GUICtrlCreateTabItem("tab2")
    GUICtrlSetState(-1, $GUI_SHOW); will be display firsts'affichera en premier
    GUICtrlCreateLabel("label2", 30, 80, 50, 20)
    GUICtrlCreateButton("OK2", 140, 50, 50)

    GUICtrlCreateTabItem(""); end tabitem definitionfin de la définition de TabItem

    GUICtrlCreateLabel("Click on tab and see the title", 20, 130, 250, 20)

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        $idMsg = GUIGetMsg()

        If $idMsg = $GUI_EVENT_CLOSE Then ExitLoop
        If $idMsg = $idTab Then
            ; display the clicked tabAffiche l'onglet cliqué
            WinSetTitle("My GUI Tab", "", "My GUI Tab" & GUICtrlRead($idTab))
        EndIf
    WEnd
EndFunc   ;==>Example
