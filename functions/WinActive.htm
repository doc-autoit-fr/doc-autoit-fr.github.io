Référence Fonction

WinActive

Checks to see if a specified window exists and is currently active.Vérifie si une fenêtre spécifiée existe et si elle est active.

WinActive ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to check.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional] The text of the window to check.Le texte de la fenêtre à contrôler. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success: the handle to the window if it is active.Retourne le handle de la fenêtre si elle est active.
Failure: 0 otherwise.Retourne 0.

RelatedLiens

WinExists, WinTitleMatchMode (Option), WinWait, WinWaitActive, WinWaitClose, WinWaitNotActive

ExampleExemple

#include <MsgBoxConstants.au3>

If WinActive("[CLASS:Notepad]") Then ; Check if Notepad is currently active.Vérifie si le bloc-notes est actuellement actif.
    MsgBox($MB_SYSTEMMODAL, "", "WinActive" & @CRLF & "Notepad is active.")
Else
    MsgBox($MB_SYSTEMMODAL, "", "WinActive" & @CRLF & "Notepad is not active.")
EndIf
