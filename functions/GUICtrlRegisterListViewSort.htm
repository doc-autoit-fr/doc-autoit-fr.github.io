GUICtrlRegisterListViewSort

Register a user defined function for an internal listview sorting callback function.Enregistre une fonction 'callback' définie par l'utilisateur pour un tri interne d'une ListView.

GUICtrlRegisterListViewSort ( controlID, "function" )

ParametersParamètres

controlID The listview controlID for which the user function should proceed.Le controlID de la Listview pour lequel la fonction de l'utilisateur doit procéder.
function The name of the user function to call when the sorting callback runs.Le nom de la fonction utilisateur à appeler lorsque le tri 'callback' doit être exécuté.

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0.Retourne 0.

RemarksRemarques

!!! To make the user function workable you have to define it with maximum 4 function parameters otherwise the function won't be called !!!Pour réaliser une fonction utilisateur valide, vous devez la définir avec au plus 4 paramètres sinon la fonction ne sera pas appelée !!!
i.e:Par exemple:
Func MySortFunction($nListViewID, $LParam1, $LParam2, $nColumn)Func MySortFunction($nListViewID, $LParam1, $LParam2, $nColumn )
...
EndFunc

Orou

Func MySortFunction($nListViewID, $LParam1, $LParam2)
...
EndFunc


When the user function is called then these 4 parameters have the following values:Lorsque la fonction utilisateur sera appelée, ces quatre paramètres auront les valeurs suivantes:

Position ParameterParamètre MeaningSignification
1Retourne 1 controlID The controlID of the listview control for which the callback function is used.Le controlID du contrôle Listview pour lequel la fonction de rappel est utilisée.
2 lParam1 The lParam value of the first item (by default the item controlID).La valeur lParam du premier item (par défaut l'item controlID).
3 lParam2 The lParam value of the second item (by default the item controlID).La valeur lParam du deuxième item (par défaut l'item controlID ).
4 columncolonne The column that was clicked for sorting (the first column number is 0).La colonne qui a été cliquée pour être triée (le numéro de la première colonne est 0).

The following values have to be Returned to change the behaviour of the sorting callback:Les valeurs suivantes ont été Retournées pour changer le comportement de la fonction de tri 'callback':

Return valueValeur retournée MeaningSignification
-1Retourne -1 1st item should precede the 2nd.Le 1er item doit précéder la 2nd.
0Retourne 0 No Change.Pas de changement.
1Retourne 1 1st item should follow the 2nd.Le 1er item devrait suivre le 2ème.

See also examples for sorting with self created GUI listview items.Consultez aussi les exemples pour trier avec auto-création les items de la Listview.

#include <GUIConstantsEx.au3>
#include <ListViewConstants.au3>

Global $g_iCurCol = -1
Global $g_iSortDir = 1
Global $g_bSet = False
Global $g_iCol = -1

Example()

Func Example()
    GUICreate("Test", 300, 200)

    Local $idLv = GUICtrlCreateListView("Column1|Col2|Col3", 10, 10, 280, 180)
    GUICtrlRegisterListViewSort(-1, "LVSort") ; Register the function "SortLV" for the sorting callbackCrée la fonction 'SortLV' pour le tri 'callback'

    GUICtrlCreateListViewItem("ABC|666|10.05.2004", $idLv)
    GUICtrlSetImage(-1, "shell32.dll", 7)
    GUICtrlCreateListViewItem("DEF|444|11.05.2005", $idLv)
    GUICtrlSetImage(-1, "shell32.dll", 12)
    GUICtrlCreateListViewItem("CDE|444|12.05.2004", $idLv)
    GUICtrlSetImage(-1, "shell32.dll", 3)

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        $idMsg = GUIGetMsg()
        Switch $idMsg
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idLv
                $g_bSet = False
                $g_iCurCol = $g_iCol
                GUICtrlSendMsg($idLv, $LVM_SETSELECTEDCOLUMN, GUICtrlGetState($idLv), 0)
        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example

; Our sorting callback funtionNotre fonction de tri callback
Func LVSort($hWnd, $nItem1, $nItem2, $iColumn)
    Local $sVal1, $sVal2, $iResult

    ; Switch the sorting directionChange la direction de tri
    If $iColumn = $g_iCurCol Then
        If Not $g_bSet Then
            $g_iSortDir = $g_iSortDir * -1
            $g_bSet = True
        EndIf
    Else
        $g_iSortDir = 1
    EndIf
    $g_iCol = $iColumn

    $sVal1 = GetSubItemText($hWnd, $nItem1, $iColumn)
    $sVal2 = GetSubItemText($hWnd, $nItem2, $iColumn)

    ; If it is the 3rd colum (column starts with 0) then compare the datesSi c'est le 3ème colonne (celle qui commence par 0), alors compare les dates
    If $iColumn = 2 Then
        $sVal1 = StringRight($sVal1, 4) & StringMid($sVal1, 4, 2) & StringLeft($sVal1, 2)
        $sVal2 = StringRight($sVal2, 4) & StringMid($sVal2, 4, 2) & StringLeft($sVal2, 2)
    EndIf

    $iResult = 0 ; No change of item1 and item2 positionsPas de changement des positions de item1 et de item2

    If $sVal1 < $sVal2 Then
        $iResult = -1 ; Put item2 before item1Met item2 avant item1
    ElseIf $sVal1 > $sVal2 Then
        $iResult = 1 ; Put item2 behind item1Mettez item2 derrière élément1
    EndIf

    $iResult = $iResult * $g_iSortDir

    Return $iResult
EndFunc   ;==>LVSort

; Retrieve the text of a listview item in a specified columnRécupére le texte d'un item listview dans une colonne spécifiée
Func GetSubItemText($idCtrl, $idItem, $iColumn)
    Local $tLvfi = DllStructCreate("uint;ptr;int;int[2];int")

    DllStructSetData($tLvfi, 1, $LVFI_PARAM)
    DllStructSetData($tLvfi, 3, $idItem)

    Local $tBuffer = DllStructCreate("char[260]")

    Local $nIndex = GUICtrlSendMsg($idCtrl, $LVM_FINDITEM, -1, DllStructGetPtr($tLvfi));

    Local $tLvi = DllStructCreate("uint;int;int;uint;uint;ptr;int;int;int;int")

    DllStructSetData($tLvi, 1, $LVIF_TEXT)
    DllStructSetData($tLvi, 2, $nIndex)
    DllStructSetData($tLvi, 3, $iColumn)
    DllStructSetData($tLvi, 6, DllStructGetPtr($tBuffer))
    DllStructSetData($tLvi, 7, 260)

    GUICtrlSendMsg($idCtrl, $LVM_GETITEMA, 0, DllStructGetPtr($tLvi));

    Local $sItemText = DllStructGetData($tBuffer, 1)

    Return $sItemText
EndFunc   ;==>GetSubItemText

; Create and insert items directly into the listviewCrée et insére des items directement dans la Listview
Func MyGUICtrlCreateListViewItem($sText, $idCtrl, $iIndex)
    Local $tLvitem = DllStructCreate("uint;int;int;uint;uint;ptr;int;int;int;int;")
    Local $tText = DllStructCreate("char[260]")
    Local $aText = StringSplit($sText, "|")

    If $iIndex = -1 Then $iIndex = GUICtrlSendMsg($idCtrl, $LVM_GETITEMCOUNT, 0, 0)

    DllStructSetData($tText, 1, $aText[1]) ; Save the item text in the structEnregistrez le texte de l'élément dans la structure

    DllStructSetData($tLvitem, 1, BitOR($LVIF_TEXT, $LVIF_PARAM))
    DllStructSetData($tLvitem, 2, $iIndex)
    DllStructSetData($tLvitem, 6, DllStructGetPtr($tText))
    ; Set the lParam of the struct to the line index - unique within the listviewDéfinit le lParam de la structure sur l'index unique de ligne dans le listview
    DllStructSetData($tLvitem, 9, $iIndex)

    $iIndex = GUICtrlSendMsg($idCtrl, $LVM_INSERTITEMA, 0, DllStructGetPtr($tLvitem))

    If $iIndex > -1 Then
        ; Insert now the rest of the column textInsére maintenant le reste du texte de la colonne
        For $i = 2 To $aText[0]
            DllStructSetData($tText, 1, $aText[$i])
            DllStructSetData($tLvitem, 3, $i - 1) ; Store the subitem indexStocker l'index de sous-élément

            GUICtrlSendMsg($idCtrl, $LVM_SETITEMTEXTA, $iIndex, DllStructGetPtr($tLvitem))
        Next
    EndIf

    ; Change the column width to fit the item textChange la largeur de colonne pour ajuster le texte de l'item
    For $i = 0 To 2
        GUICtrlSendMsg($idCtrl, $LVM_SETCOLUMNWIDTH, $i, -1)
        GUICtrlSendMsg($idCtrl, $LVM_SETCOLUMNWIDTH, $i, -2)
    Next
EndFunc   ;==>MyGUICtrlCreateListViewItem
