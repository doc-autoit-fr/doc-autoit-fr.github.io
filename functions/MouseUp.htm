Référence Fonction

MouseUp

Perform a mouse up event at the current mouse position.Déclenche l'événement de souris 'bouton relaché' à la position actuelle de la souris.

MouseUp ( "button" )

ParametersParamètres

button The button to click:Le bouton à cliquer :
    $MOUSE_CLICK_RIGHT ("right")
    $MOUSE_CLICK_MIDDLE ("middle")
    $MOUSE_CLICK_MAIN ("main")
    $MOUSE_CLICK_MENU ("menu")
    $MOUSE_CLICK_PRIMARY ("primary")
    $MOUSE_CLICK_SECONDARY ("secondary")

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0, the button is not in the list.Retourne 0, le bouton n'est pas dans la liste.

RemarksRemarques

See MouseClick remarks.Consultez Remarques dans MouseClick.
User responsibility:Responsabilité de l'utilisateur : For every MouseDown() there should eventually be a corresponding MouseUp() event.Pour chaque MouseDown() il devrait correspondre un MouseUp().

RelatedLiens

MouseClick, MouseClickDrag, MouseCoordMode (Option), MouseDown, MouseGetPos, MouseMove

ExampleExemple

#include <AutoItConstants.au3>

MouseDown($MOUSE_CLICK_LEFT) ; Set the left mouse button state as down.Définit l'état du bouton gauche de la souris comme 'pressé'.
Sleep(100)
MouseUp($MOUSE_CLICK_LEFT) ; Set the left mouse button state as up.Définit l'état du bouton gauche de la souris comme 'relaché'.
