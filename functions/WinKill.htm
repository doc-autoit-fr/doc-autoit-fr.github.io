WinKill

Forces a window to close.Force la fermeture d'une fenêtre.

WinKill ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to close.Le titre/hWnd/class de la fenêtre à fermer. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the window to close.Le texte de la fenêtre à fermer. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : The function only returns the success value.La fonction retourne une valeur seulement en cas de succès.

RemarksRemarques

The difference between this function and WinClose() is that WinKill() will forcibly terminate the window if it doesn't close quickly enough.La différence entre cette fonction et WinClose() est que WinKill() fermera la fenêtre de force si elle ne se referme pas assez vite. Consequently, a user might not have time to respond to dialogs prompting the user to save data.Par conséquent, un utilisateur pourrait ne pas avoir le temps de répondre aux boîtes de dialogue lui demandant d'enregistrer les données. Although WinKill() can work on both minimized and hidden windows, some windows (notably explorer windows) can only be terminated using WinClose().Bien que WinKill() peut travailler sur les fenêtres réduites et cachées, certaines fenêtres (Windows Explorer notamment) ne peuvent être fermées qu'avec WinClose().

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Wait for 2 seconds before calling the WinKill function.Attend deux secondes avant d'appeler la fonction WinKill.
    Sleep(2000)

    ; Kill the window using the handle returned by WinWait.Tue la fenêtre en utilisant Le handle retournée par WinWait.
    WinKill($hWnd, "")
EndFunc   ;==>Example
