MouseClickDrag

Perform a mouse click and drag operation.Effectue un clic de souris et une opération de glissement.

MouseClickDrag ( "button", x1, y1, x2, y2 [, speed = 10] )

ParametersParamètres

button The button to click:Le bouton à cliquer :
    $MOUSE_CLICK_RIGHT ("right")
    $MOUSE_CLICK_MIDDLE ("middle")
    $MOUSE_CLICK_MAIN ("main")
    $MOUSE_CLICK_MENU ("menu")
    $MOUSE_CLICK_PRIMARY ("primary")
    $MOUSE_CLICK_SECONDARY ("secondary")

Constants are defined in "AutoItConstants.au3".
x1, y1 The x/y coords to start the drag operation from.Les coordonnées x,y pour démarrer l'opération de glissement.
x2, y2 The x/y coords to end the drag operation at.Les coordonnées x,y pour mettre fin à l'opération de glissement.
speed [optional][optionnel][optionnel] the speed to move the mouse in the range 1 (fastest) to 100 (slowest).la vitesse de déplacement de la souris dans la plage 1 (le plus rapide) à 100 (le plus lent). A speed of 0 will move the mouse instantly.Une vitesse de 0 déplace la souris instantanément. Default speed is 10.La vitesse par défaut est 10.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0, the button is not in the list.Retourne 0, le bouton n'est pas dans la liste.

RemarksRemarques

If the button is an empty string, the left button will be clicked.Si le bouton est une chaîne vide, le bouton gauche sera cliqué.

If the user has swapped the left and right mouse buttons in the control panel, then the behaviour of the buttons is different.Si l'utilisateur a permuté les boutons gauche et droite de la souris dans le panneau de configuration, alors le comportement de la fonction est différent. "Left" and "right" always click those buttons, whether the buttons are swapped or not."Left" et "right" cliquent respectivement sur les boutons gauche et droite, qu'ils soient inversés ou non. The "primary" or "main" button will be the main click, whether or not the buttons are swapped.Le bouton "primary" ou "main" sera le clic principal, que les boutons soient inversés ou non. The "secondary" or "menu" buttons will usually bring up the context menu, whether the buttons are swapped or not.Le bouton "secondary" ou "menu" fera généralement apparaître le menu contextuel, que les boutons soiennt inversés ou non. See the table in MouseClick for more explanationConsultezle tableau de MouseClick pour plus d'explications.
