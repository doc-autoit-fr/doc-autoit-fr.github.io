StringTrimRight

Trims a number of characters from the right hand side of a string.Coupe un nombre de caractères depuis la droite de la chaîne.

StringTrimRight ( "string", count )

ParametersParamètre

string The string to evaluate.La chaîne à traiter.
count The number of characters to trim.Le nombre de caractères à couper.

Return ValueRemarque

Returns the string trimmed by count characters from the right.Retourne la chaîne coupée du nombre <e0>count</e0> de caractères depuis la droite.

Remarks<i0>

If count is out-of-bounds, an empty string is returned.Si <e0>count</e0> est hors-limites, une chaîne vide est retournée.
StringTrimRight($str, $n) is functionally equivalent to StringLeft($str, StringLen($str) - $n)StringTrimRight($str, $n) est fonctionnellement équivalent à StringLeft($str, StringLen($str) - $n).
