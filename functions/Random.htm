Random

Generates a pseudo-random float-type number.Génère un nombre pseudo-aléatoire de type 'float'.

Random ( [Min = 0 [, Max = 1 [, Flag = 0]]] )

ParametersParamètre

Min [optional][optional] False - i, v, x ... The smallest number to be generated.Le plus petit nombre qui peut être généré. The default is 0.Par défaut c'est 0.
Max [optional][optional] False - i, v, x ... The largest number to be generated.Le plus grand nombre qui peut être généré. The default is 1.La valeur par défaut est 1.
Flag [optional][optional] False - i, v, x ... If this is set to 1 then an integer result will be returned.S'il est mis à 1, un résultat de type 'integer' sera retourné. Default is a floating point number.Par défaut c'est un nombre de type 'float'.

Return ValueRemarque

Success: a pseudo-random number between Min and Max.Retourne un nombre pseudo-aléatoire entre Min et Max.
Failure: sets the @error flag to non-zero.Retourne @error

Remarks<i0>

If only one argument is provided, then this is interpreted to be the Max parameter.Si un seul argument est fourni, il est considéré comme le paramètre Max.

The result will be in the range of Min to Max INCLUSIVE when using integers (just short of Max when using floats).Le résultat sera dans l'intervalle de Min à Max, bornes INCLUSES lors de l'utilisation des résultats de type 'integer' (un peu moins du Max lorsqu'il s'agit de 'float').

If Min and Max are the same value then Random() will return that value - the @error flag will not be set in this case

When using integers Max-Min must be less than 2^31.Quand on utilise des 'integers' Min et MAx doivent être inférieurs à 2^31.

This function uses the Mersenne Twister random number generator, MT19937, written by Takuji Nishimura, Makoto Matsumoto, Shawn Cokus, Matthe Bellew and Isaku Wada.Cette fonction utilise le générateur de nombre aléatoire Mersenne Twister, MT19937, écrit par Takuji Nishimura, Makoto Matsumoto, Shawn Cokus, Matthe Bellew et Isaku Wada.

The Mersenne Twister is an algorithm for generating random numbers.Le Mersenne Twister est un algorithme de génération de nombres aléatoires. It was designed with consideration of the flaws in various other generators.Il a été conçu en tenant compte des failles de divers autres générateurs. The period, 219937-1, and the order of equidistribution, 623 dimensions, are far greater.La période, 219937-1 et l'ordre de l'équidistribution, 623 dimensions, sont beaucoup plus grands. The generator is also fast; it avoids multiplication and division, and it benefits from caches and pipelines.Le générateur est également rapide, il évite la multiplication et la division, et bénéficie de mémoire cache et de plusieurs canaux de traitement.
