ControlGetText

Retrieves text from a control.Retourne le texte d'un contrôle.

ControlGetText ( "title", "text", controlID )

ParametersParamètre

title The title/hWnd/class of the window to access.Le titre/handle/class de la fenêtre à atteindre. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to access.Le texte de la fenêtre à accéder. See Text special definition.Consultez Définition spéciale de Texte .
controlID The control to interact with.Le contrôle avec lequel interagir. See Controls.Consultez Controls.

Return ValueRemarque

Success: the text from a control.Retourne le texte d'un contrôle (contenu, label).
Failure: "" (empty string) and sets the @error flag to 1.Retourne "" (chaîne vide) et @error = 1.

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Set the edit control in Notepad with some text.Définit le texte du contrôle d'édition du Bloc-notes. The handle returned by WinWait is used for the "title" parameter of ControlSetText.Le handle renvoyé par WinWait est utilisé pour le paramètre \"titre\" de ControlSetText.
    ControlSetText($hWnd, "", "Edit1", "This is some text")

    ; Retrieve the text of the edit control in Notepad.Récupére le texte du contrôle d'édition du le Bloc-notes. The handle returned by WinWait is used for the "title" parameter of ControlGetText.Le handle renvoyé par WinWait est utilisé pour le paramètre \"titre\" de ControlGetText.
    Local $sText = ControlGetText($hWnd, "", "Edit1")

    ; Display the text of the edit control.Affiche le texte du contrôle d'édition.
    MsgBox($MB_SYSTEMMODAL, "", "The text in Edit1 is: " & $sText)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
