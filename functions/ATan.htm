ATan

Calculates the arctangent of a number.Calcule l’arc tangente d'un nombre.

ATan ( expression )

ParametersParamètres

expression Any valid numeric expression.Toute expression numérique valide.

Return ValueValeur de retour

Returns the trigonometric arctangent of the number.Retourne l’arc tangente trigonométrique du nombre. Result is in radians.Le résultat est exprimé en radians.

RemarksRemarques

The result of 4 * ATan(1) is pi.Le résultat de 4 * ATan(1) est pi.

RelatedLiens

ExampleExemple

Example 1Exemple 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local variable the arcTangent of 0.5.Assigne une variable locale avec l'arctangente de 0,5.
    Local $fArcTan1 = ATan(0.5)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fArcTan1 & " rad.")

    ; Assign a Local constant variable the approximate PI number.Assigne une constante locale avec une approximation du nombre de PI.
    Local Const $PI = 3.141592653589793

    ; Assign a Local variable the formula to switch from radian to degree (equals to one radian in degree).Assigne une variable locale avec le facteur de conversion de radian en degré (égal à un radian en degré).
    Local $fRadToDeg = 180 / $PI

    ; Assign a Local variable a number in degree.Assigne une variable locale avec un certain nombre de degrés.
    Local $fArcTan2 = $fRadToDeg * ATan(1)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fArcTan2 & " deg.")
EndFunc   ;==>Example
