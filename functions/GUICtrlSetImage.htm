GUICtrlSetImage

Sets the bitmap or icon image to use for a control.Définit l'image bitmap ou icône à utiliser pour un contrôle.

GUICtrlSetImage ( controlID, filename [, iconname [, icontype]] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
filename The filename containing the picture to be display on the control.Le nom du fichier contenant l'image à afficher sur le contrôle.
iconname [optional][optionnel] Icon name if the file contains multiple icons.Nom de l'icône si le fichier contient plusieurs icônes. Can be an ordinal name if negative number.Peut être un nom ordinal si nombre négatif. Otherwise -1.Sinon -1.
icontype [optional][optionnel] To select a specific icon size :Pour sélectionner la taille d'une icône spécifique : 0 = small, 1 = normal (default).0 = petite, 1 = normale (par défaut).
For a TreeViewItem the icon size :Pour un TreeViewItem la taille de l'icône : 2 = selected, 4 for non-selected item.2 = sélectionné, 4 pour un item non sélectionné.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

Use a resource hacker to know the value of the valid icon name in a file.Utilisez un hacker de ressources pour connaître la valeur du nom de l'icône valide dans un fichier.

If used on a Button control the image will be displayed on the button.Si elle est utilisée sur un contrôle Button, l'image sera affichée sur le bouton. Images can also be set for Checkbox controls as long as the $BS_PUSHLIKE style is used.Les images peuvent également être définies pour des contrôles Checkbox aussi longtemps que le style $BS_PUSHLIKE est utilisé. In both case the $BS_ICON or $BS_BITMAP styles are needed to select the type of the picture used.Dans les deux cas, les styles $BS_ICON ou $BS_BITMAP sont nécessaires pour sélectionner le type de l'image utilisée. The first icon resolution will be used in a multi icon resolution file.La première résolution de l'icône sera utilisée dans un fichier multi icône de résolution. I.E. if a 128x128 is the first resolution and the control is 64x64 the image will be truncated.I.E.si 128x128 est la première résolution et le contrôle est 64x64 l'image sera tronquée.

!!! If you use this command on a TreeViewItem the first time, then all other items will use this icon/image automatically by default !!!Si vous utilisez ce contrôle sur un TreeViewItem la première fois, alors tous les autres items vont utiliser cette icône/image automatiquement par défaut!!!
If you use GUICtrlSetImage() on a TreeView or ListView then all items of it will change to this icon/image.Si vous utilisez GUICtrlSetImage() sur un TreeView ou une ListView, tous les items de celui-ci vont adopter cette icône/image.

Passing a positive number will reference the string equivalent icon name.Passer un nombre positif, fera référence à l'équivalent chaîne du nom de l'icône.
Passing a negative number causes 1-based "index" behaviour.Passer un nombre négatif, provoque un comportement d'"index" basé sur 1. Some Dll can have icon extracted just with negative numbers.Les icônes de certaines Dll ne peuvent être extraites seulement qu'avec des nombres négatifs.

Example()

Func Example()
    GUICreate("My GUI") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUICtrlCreateButton("my picture button", 10, 20, 40, 40, $BS_ICON)
    GUICtrlSetImage(-1, "shell32.dll", 22)

    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
