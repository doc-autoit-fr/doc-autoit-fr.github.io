AutoItWinGetTitle

Retrieves the title of the AutoIt window.Récupère le titre d'une fenêtre Autoit.

AutoItWinGetTitle ( )

Return ValueValeur de retour

Returns a string containing the title of the AutoIt window.Retourne une chaine de caractères contenant le titre de la fenêtre Autoit.

RelatedLiens

AutoItWinSetTitle, WinGetTitle

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Set the title of the AutoIt Hidden Window.
    AutoItWinSetTitle("My AutoIt Window")

    ; Display AutoIt's Hidden Window.Affiche la fenêtre cachée de AutoIt.
    AutoItWinShow()

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example

; Display AutoIt's Hidden Window.Affiche la fenêtre cachée de AutoIt. Returns the handle of the window.Retourne le handle de la fenêtre.
Func AutoItWinShow()
    Local $hWnd = WinGetHandle(AutoItWinGetTitle()) ; Get the handle of the AutoIt Hidden Window by finding out the title of the AutoIt Hidden Window.Obtient le handle de la fenêtre cachée AutoIt à partir du titre de la fenêtre cachée AutoIt.
    WinMove($hWnd, "", (@DesktopWidth / 2) - 250, (@DesktopHeight / 2) - 250, 500, 500) ; Move the AutoIt Hidden Window and re-size for a better view.Déplace la fenêtre cachée AutoIt et la redimensionne pour une meilleure vue.
    WinSetState($hWnd, "", @SW_SHOW) ; Show the AutoIt Hidden Window, normally this is hidden, but in the interest of this example I'm displaying it.
    Return $hWnd
EndFunc   ;==>AutoItWinShow
