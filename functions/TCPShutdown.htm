Function ReferenceRéférence de fonction

TCPShutdown, UDPShutdown

Stops TCP/UDP services.Arrête les services TCP/UDP.

TCPShutdown ( )
UDPShutdown ( )

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0.
@error:@error : Windows API WSACleanup return value (see MSDN).

RemarksRemarques

A script must call one TCPShutdown() call for every successful TCPStartup() call.Un script doit appeler un TCPShutdown() pour chaque appel réussi à un TCPStartup().
UDPShutdown() is just an alias of TCPShutdown().
