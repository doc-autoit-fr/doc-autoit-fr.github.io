ProcessExists

Checks to see if a specified process exists.Vérifie si un processus spécifié existe.

ProcessExists ( "process" )

ParametersParamètres

process The name or PID of the process to check.Le nom ou le PID du processus à vérifier.

Return ValueValeur de retour

Success:Succès : the PID of the process.Retourne le PID du processus.
Failure:Échec : 0 if process does not exist.Retourne 0 si le processus n'existe pas.

RemarksRemarques

Process names are executables without the full path, e.g., "notepad.exe" or "winword.exe"

PID is the unique number which identifies a Process.PID est le numéro unique qui identifie un processus.

The process is polled approximately every 250 milliseconds.Le processus est interrogé environ toutes les 250 millisecondes.

RelatedLiens

ProcessClose, ProcessList, ProcessWait, ProcessWaitClose, WinExists

ExampleExemple

#include <MsgBoxConstants.au3>

If ProcessExists("notepad.exe") Then ; Check if the Notepad process is running.Vérifie si le processus Notepad est en cours d'exécution.
    MsgBox($MB_SYSTEMMODAL, "", "Notepad is running")
Else
    MsgBox($MB_SYSTEMMODAL, "", "Notepad is not running")
EndIf
