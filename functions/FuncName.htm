FuncName

Returns the name of a function stored in a variable.Retourne le nom d'une fonction stockée dans une variable.

FuncName ( $Functionvariable )

ParametersParamètres

$Functionvariable A variable containing a Function whose name you want to retrieve.Une variable contenant une Function dont vous voulez obtenir le nom.

Return ValueValeur de retour

Success:Succès : a string containing the name of the function.Retourne une chaîne contenant le nom de la function.
Failure:Échec : "" (empty string) and sets the @error flag to 1.Retourne "" (chaîne vide) et @error = 1.

Func Example()
    ; Create a first class object of MsgBox and assign to a Local variable.Crée un premier objet de la classe de MsgBox et assigner à une variable locale.
    Local $hMsgBox = MsgBox

    ; Display a MsgBox using the previously assigned variable.Affiche une MsgBox en utilisant la variable précédemment attribuée.
    $hMsgBox($MB_SYSTEMMODAL, "", "This is a sentence with whitespace.")

    ; Display a MsgBox that shows the function name $hMsgBox.Affiche une MsgBox qui indique le nom de la fonction $hMsgBox.
    $hMsgBox($MB_SYSTEMMODAL, "", "The function name of $hMsgBox is: " & FuncName($hMsgBox))
EndFunc   ;==>Example
