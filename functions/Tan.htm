Tan

Calculates the tangent of a number.Calcule la tangente d'un nombre.

Tan ( expression )

ParametersParamètre

expression Value in radians.Valeur en radians.

Return ValueRemarque

Returns the trigonometric tangent of the number.Retourne la tangente trigonométrique du nombre.

Remarks<i0>

1° = pi / 180 radians.
Note that Tan() is "infinite" for ...Notez que Tan() est "infini" pour ... -3PI/2, -PI/2, PI/2, 3PI/2, 5PI/2, ... and tends to return 1.63317787283838e+016 for such multiples of Pi-halves.-3PI/2, -PI/2, PI/2, 3PI/2, 5PI/2, ...et tend à retourner 1.63317787283838e+016 pour les multiples de Pi/2.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local constant variable the approximate PI number.Assigne une constante locale avec une approximation du nombre de PI.
    Local Const $PI = 3.141592653589793

    ; Assign a Local variable the tangent of PI / 4.Assigne une variable locale avec la tangente de PI / 4.
    Local $fTan1 = Tan($PI / 4)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fTan1)

    ; Assign a Local variable the formula to switch from radian to degree (equals to one degree in radian).Assigne une variable locale avec le coefficient de conversion de radian à degré (égal à un degré en radian).
    Local $iDegToRad = $PI / 180

    ; Assign a Local variable the tangent of 90 degrees (firstly converted to radians).Assigne une variable locale avec la tangente de 90 degrés (d'abord converti en radians).
    Local $fTan2 = Tan(90 * $iDegToRad)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fTan2)
EndFunc   ;==>Example
