FileSetTime

Sets the timestamp of one of more files.Attribue le timestamp (l'horodatage) d'un ou plusieurs fichiers/dossiers.

FileSetTime ( "file pattern", "time" [, type = 0 [, recurse = 0]] )

ParametersParamètres

file pattern The path of the file(s) to set, e.g.Le chemin des Fichier(s) à modifier, ex : C:\*.au3, C:\Dir. (* and ?( caractères spéciaux * et ? wildcards accepted - See Remarks)acceptés - Voir Remarques)
time The new time to set in the format "YYYYMMDDHHMMSS" (Year, month, day, hours (24hr clock), seconds).La nouvelle date et heure à attribuer au format "YYYYMMDDHHMMSS" (Année, mois, jour, heures (au format 24hr), secondes). If the time is blank "" then the current time is used.Si time est vide "" alors la date et l'heure en cours est utilisée.
type [optional] The timestamp to change:Horodatage à modifier:
    $FT_MODIFIED (0) = Last modified (default)
    $FT_CREATED (1) = Created
    $FT_ACCESSED (2) = Last accessed

Constants are defined in FileConstants.au3Les constantes sont définies dans FileConstants.au3
recurse [optional]
    $FT_NONRECURSIVE (0) - no recursion (Default).
    $FT_RECURSIVE (1) - directories are recursed into.

Constants are defined in FileConstants.au3Les constantes sont définies dans FileConstants.au3

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if error changing timestamp(s).Retourne 0 en cas d'erreur lors du changement du timestamp.

RemarksRemarques

See FileFindFirstFile() for a discussion about wildcards.Consultez FileFindFirstFile() pour des précisions sur les caractères spéciaux.

Using a date earlier than 1980-01-01 will have no effect.Utiliser une date antérieure au 01/01/1980 n'aura aucun effet.
Trying to change a timestamp on read-only files will result in an error.Le changement de la date d'un fichier en lecture seule provoquera une erreur.

ExampleExemple

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create a temporary file to read data from.Crée un fichier temporaire pour y lire les données.
    If Not FileWrite($sFilePath, "This is an example of using FileSetTime.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Set the modified timestamp of the file to 1st Nov 2003 and use the current time.Régle l'horodatage de modification du fichier de 1 novembre 2003 et utilise l'heure actuelle.
    Local $iFileSetTime = FileSetTime($sFilePath, "20031101", $FT_MODIFIED)

    ; Display the modified timestamp of the file and return as a string in the format YYYYMMDDHHMMSS.Affiche l'horodatage de modification du fichier et retourne une chaîne au format AAAAMMJJHHMMSS.
    If $iFileSetTime Then
        MsgBox($MB_SYSTEMMODAL, "", "Timestamp:" & @CRLF & FileGetTime($sFilePath, $FT_MODIFIED, 1))
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst setting the timestamp of the file.")
    EndIf

    ; Delete the temporary file.Supprimez le fichier temporaire.
    FileDelete($sFilePath)
EndFunc   ;==>Example
