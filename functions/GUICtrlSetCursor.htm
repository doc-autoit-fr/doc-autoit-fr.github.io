GUICtrlSetCursor

Sets the mouse cursor icon for a particular control.Définit l'icône du curseur de la souris pour un contrôle particulier.

GUICtrlSetCursor ( controlID, cursorID )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
cursorID cursor ID as used by Windows SetCursor API (use -1 for the default mouse cursor for the control)L'ID du curseur utilisé par l'API Windows SetCursor (utilisez -1 pour le curseur de la souris par défaut pour le contrôle)

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

Unlike GUISetCursor() which changes the mouse cursor for an entire window, this function sets the mouse cursor that is used when the mouse is hovering over the specified control.Contrairement à GUISetCursor() qui modifie le curseur de la souris pour une fenêtre entière, cette fonction définit le curseur de la souris qui est utilisé lorsque la souris survole le contrôle spécifié.

If the cursorID is invalid the standard arrow will be displayed.Si l'ID du curseur est invalide la flèche standard sera utilisée.

For a list of valid cursor IDs see MouseGetCursor().Pour une liste d'IDs de curseur valides voir MouseGetCursor().
CursorId = 16 will hide the mouse cursor.CursorID = 16 permet de cacher le curseur souris.

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("put cursor over label", 300, 100)
    GUICtrlCreateLabel("label", 125, 40)
    GUICtrlSetCursor(-1, 4)
    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
