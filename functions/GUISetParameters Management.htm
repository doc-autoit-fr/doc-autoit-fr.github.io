GUI set parameters functions ReferenceGUI set parameters - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

See Gui Constants include files if you need to use the related controls Constants .Voir Fichiers 'include' des constantes GUI si vous avez besoin d'utiliser des constantes relatives aux contrôles.

 

FunctionFonction Description
GUISetAccelerators Sets the accelerator table to be used in a GUI window.Définit les raccourcis clavier (accélérateurs) pour une fenêtre GUI.
GUISetBkColor Sets the background color of the GUI window.Définit la couleur de fond de la fenêtre GUI.
GUISetCoord Sets absolute coordinates for the next control.Définit les coordonnées absolues pour le contrôle suivant.
GUISetCursor Sets the mouse cursor icon for a GUI window.Définit l'icône du curseur de la souris pour une fenêtre GUI.
GUISetFont Sets the default font for a GUI window.Définit la police par défaut pour une fenêtre GUI.
GUISetHelp Sets an executable file that will be run when F1 is pressed.Définit un fichier exécutable qui sera exécuté lorsque F1 sera pressé.
GUISetIcon Sets the icon used in a GUI window.Définit l'icône utilisée dans une fenêtre GUI.
GUISetOnEvent Defines a user function to be called when a system button is clicked.Définit une fonction utilisateur à appeler quand un bouton système est cliqué.
GUISetState Changes the state of a GUI window.Modifie l'état d'une fenêtre GUI.
GUISetStyle Changes the styles of a GUI window.Modifie les styles d'une fenêtre GUI.
