IsKeyword

Checks if a variable is a keyword (for example, Default).Vérifie si une variable est un mot-clé (par exemple, 'Default' ).

IsKeyword ( variable )

ParametersParamètre

variable The variable to check.La variable à vérifier.

Return ValueRemarque

Success: $KEYWORD_DEFAULT (1) the Default keyword.Retourne $KEYWORD_DEFAULT (1), le mot-clé 'Default'.
    $KEYWORD_NULL (2) the Null keyword.    $KEYWORD_NULL (2), le mot-clé Null.
Failure: 0 if not a keyword.Retourne 0 si ce n'est pas un mot-clé.

Constants are defined in AutoItConstants.au3

#include <Constants.au3>

Example()

Func Example()
    ; Check if a variable is the Default keyword.Vérifie si une variable est le mot-clé 'Default'.
    Local $vDefault = Default
    If IsDefault($vDefault) Then
        MsgBox($MB_SYSTEMMODAL, "", "The variable is the Default keyword")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The variable is not the Default keyword")
    EndIf

    ; Check if a variable is the Null keyword.Vérifie si une variable est le mot clé Null.
    Local $vNull = Null
    If IsNull($vNull) Then
        MsgBox($MB_SYSTEMMODAL, "", "The variable is the Null keyword")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The variable is not the Null keyword")
    EndIf

    ; Check if a variable is the Null keyword.Vérifie si une variable est le mot clé Null. This will be false as $sString is a string datatype.C'est faux que $sChaine est un type de donnée chaîne.
    Local $sString = 'Default'
    If IsDefault($sString) Then
        MsgBox($MB_SYSTEMMODAL, "", "The variable is the Default keyword")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The variable is not the Default keyword")
    EndIf
EndFunc   ;==>Example

Func IsDefault($vKeyword)
    Return IsKeyword($vKeyword) = $KEYWORD_DEFAULT
EndFunc   ;==>IsDefault

Func IsNull($vKeyword)
    Return IsKeyword($vKeyword) = $KEYWORD_NULL
EndFunc   ;==>IsNull
