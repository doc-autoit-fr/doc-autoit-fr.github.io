SoundPlay

Play a sound file.Joue un fichier audio.

SoundPlay ( "filename" [, wait = 0] )

ParametersParamètre

filename Name of the file to be played (typically a WAV or MP3)Nom du fichier à jouer (généralement un fichier WAV ou MP3)
wait [optional][optional] False - i, v, x ... This flag determines if the script should wait for the sound to finish before continuing:Cet indicateur détermine si le script doit attendre que le son soit fini avant de poursuivre :
    $SOUND_WAIT(1) = wait until sound has finished
    $SOUND_NOWAIT(0) = continue script while sound is playing (default)

Constants are defined in "AutoItConstants.au3".

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges. (Always returns 1 regardless of success.)(Retourne toujours 1 quel que soit le résultat.)

Remarks<i0>

Terminating the script will stop the sound (if it is still playing).La fermeture du script stoppera le son (s'il est encore joué).
Calling SoundPlay("") can be used to stop a currently playing sound.L'appel à SoundPlay("") peut être utilisé pour arrêter un son en cours de lecture. This has the side effect of also closing the open handle.Ceci à pour effet secondaire de fermer le handle ouvert.
If you need to delete a sound file which you have played in the script, you should call SoundPlay("") first to ensure the handle is closed.Si vous avez besoin de supprimer un fichier audio que vous avez joué dans le script, vous devez utiliser SoundPlay("") en premier afin d'être sûr que le handle soit fermé.
