BitRotate

Performs a bit shifting operation, with rotation.Exécute un décalage bit à bit avec rotation.

BitRotate ( value [, shift = 1 [, size = "W"]] )

ParametersParamètres

value The number to be operate on.La valeur à traiter.
shift [optional][optionnel] Number of bits to rotate to the left (negative numbers rotate right). If not given, the default is 1.Si non donné, la valeur par défaut est 1.
size [optional][optionnel] A string that determines the rotation size, the default is (16 bits).Une chaîne de caractères déterminant la taille de la rotation (16 bits par défaut). See below.Voir ci-dessous.
Size parameter :Paramètre Size :
"B" rotate bits within the low-order byte (8 bits).Rotation à l'intérieur de l'octet de poids faible (8 bits).
"W" rotate bits within the low-order word (16 bits).Rotation à l'intérieur du mot de poids faible (16 bits).
"D" rotate bits within the entire double-word (32 bits).Rotation à l'intérieur d'un mot double (32 bits).

Return ValueValeur de retour

Success:Succès : the value rotated by the required number of bits.Retourne la valeur de la rotation par le nombre de bits demandés.
Failure:Échec : Sets the flag @error to non-zero if size is invalid.@error <> 0 si la taille est incorrecte.

RemarksRemarques

Remember hex notation can be used for numbers.La notation hexadécimale peut être utilisée comme arguments.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Note:Note : "b" is the symbol for byte."b" est le symbole de l'octet.

    ; Assign a Local variable the bitwise left-rotate operation of 2.Assigne une variable locale avec une opération bit à bit de rotation à gauche de 2.
    Local $iBitRotate1 = BitRotate(2, 1) ; 2 = 0010b left-rotated once -> 4 = 0100b2 = 0010b rotation gauche une fois -> 4 = 0100b

    ; Note:Note : It is equivalent to do this:Il est équivalent à faire : BitShift(2, -1)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate1)

    ; Assign a Local variable the bitwise right-rotate operation of 1.Assigne une variable locale avec l'opération bit à bit d'une rotation vers la droite de 1.
    Local $iBitRotate2 = BitRotate(1, -1) ; 1 = 0001b right-rotated once -> 32768 (32 bits) = 1000 0000 0000 0000b1 = 0001b rotation droite une fois -> 32768 (32 bits) = 1000 0000 0000 0000b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate2)

    ; Assign a Local variable the bitwise right-rotate operation of 14.Assigne une variable locale avec une opération bit à bit de Rotation droite de 14.
    Local $iBitRotate3 = BitRotate(14, -2) ; 14 = 1110b right-rotated twice -> 32771 (16 bits) = 1000 0000 0000 0011b14 = 1110b rotation droite deux fois -> 32771 (16 bits) = 1000 0000 0000 0011b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate3)

    ; Assign a Local variable the bitwise right-rotate operation of 14 on 32 bits.Assigne une variable locale avec l'opération bit à bit de rotation droite de 14 sur 32 bits.
    Local $iBitRotate4 = BitRotate(14, -2, "D")
    ; 14 = 1110b right-rotated twice -> -2147483645 (32 bits) = 1000 0000 0000 0000 0000 0000 0000 0011b (the first bit is signed)14 = 1110b rotation droite deux fois -> -2147483645 (32 bits) = 1000 0000 0000 0000 0000 0000 0000 0011b (le premier bit est signé)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate4)
EndFunc   ;==>Example
