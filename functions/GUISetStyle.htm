GUISetStyle

Changes the styles of a GUI window.Modifie les styles d'une fenêtre GUI.

GUISetStyle ( Style [, ExStyle [, winhandle]] )

ParametersParamètres

style defines the style of the window.Définit le style de la fenêtre. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.

Use -1 to leave it unchanged.Utilisez -1 pour laisser inchangé.
exStyle [optional][optionnel] defines the extended style of the window.Définit le style étendu de la fenêtre. See the Extended Style Table.Voir le Tableau des Styles Etendus. -1 is the default.-1 est la valeur par défaut.
Use -1 to leave it unchanged.Utilisez -1 pour laisser inchangé.
winhandle [optional][optionnel] Windows handle as returned by GUICreate() (default is the previously used window).Handle Windows retourné par GUICreate() (par défaut, la fenêtre précédemment utilisée ).

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

No checking is done on style value, neither non interaction with already defined control.Aucune vérification n'est effectuée sur la valeur du style, pas plus que sur l'interaction avec contrôle déjà défini. It is the designer responsibility to take care of this compatibility.Il est de la responsabilité du concepteur d'assurer cette compatibilité.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Example()

Func Example()

    GUICreate("Gui Style", 260, 100)
    Local $idButton = GUICtrlCreateButton("Set Style", 45, 50, 150, 20)
    GUISetState(@SW_SHOW)

    Local $bNewStyle = False, $idMsg
    While 1
        $idMsg = GUIGetMsg()
        Switch $idMsg
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idButton
                If Not $bNewStyle Then
                    GUISetStyle(BitOR($WS_POPUPWINDOW, $WS_THICKFRAME), BitOR($WS_EX_CLIENTEDGE, $WS_EX_TOOLWINDOW))
                    GUICtrlSetData($idButton, 'Undo Style')
                    $bNewStyle = True
                Else
                    GUISetStyle(BitOR($WS_MINIMIZEBOX, $WS_CAPTION, $WS_POPUP, $WS_SYSMENU), 0)
                    GUICtrlSetData($idButton, 'Set Style')
                    $bNewStyle = False
                EndIf
        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example
