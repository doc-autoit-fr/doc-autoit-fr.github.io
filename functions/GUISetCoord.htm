GUISetCoord

Sets absolute coordinates for the next control.Définit les coordonnées absolues pour le contrôle suivant.

GUISetCoord ( left, top [, width [, height [, winhandle]]] )

ParametersParamètres

left The left side of the control.La distance à gauche à la GUI.
top The top of the control.La distance en haut à la GUI.
width [optional][optionnel][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
winhandle [optional][optionnel][optionnel] Windows handle as returned by GUICreate() (default is the previously used window).Handle Windows retourné par GUICreate() (par défaut, la fenêtre précédemment utilisée ).

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0.

RemarksRemarques

To be used specifically in Opt ("GUICoordMode", 2).Pour être utilisé spécifiquement dans Opt ("GUICoordMode", 2) . It allows you to set the current position to a precise point and from that position to create controls by row (x_offset,-1) or by columns (-1, y_offset).Cette fonction vous permet de définir la position courante en un point précis et à partir de cette position créer des contrôles par ligne (x_offset, -1) ou par colonne (-1, y_offse ).

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    Opt("GUICoordMode", 2) ; relative to cell modepar rapport au mode cellule

    GUICreate("My GUI Set Coord", 200, 100)
    GUICtrlCreateCheckbox("Check #1", 20, 10, 75)
    GUICtrlCreateCheckbox("Notify #2", 10, -1) ; next cell in the lineprochaine cellule dans la ligne

    GUISetCoord(20, 60)

    GUICtrlCreateButton("OK #3", -1, -1)
    GUICtrlCreateButton("Cancel #4", 10, -1)
    GUICtrlSetState(-1, $GUI_FOCUS)

    GUISetState(@SW_SHOW) ; will display an empty dialog boxAffiche une boîte de dialogue vide

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example
