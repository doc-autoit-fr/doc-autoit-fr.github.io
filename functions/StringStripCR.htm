StringStripCR

Removes all carriage return values ( Chr(13) ) from a string.Supprime tous les retours à la ligne ( Chr(13) ) dans une chaîne.

StringStripCR ( "string" )

ParametersParamètres

string The string to convert.La chaîne à traiter.

Return ValueValeur de retour

Returns the string with all instances of the @CR character ( Chr(13) ) removed.Retourne la chaîne avec toutes les instances du caractère @CR ( Chr(13) ) supprimées.

RelatedLiens

StringAddCR, StringIsSpace, StringStripWS

ExampleExemple

#include <MsgBoxConstants.au3>

; A string with carriage return characters.Une chaîne avec des caractères 'retour chariot'.
Local $sString = "This is a sentence " & @CR & "with " & Chr(13) & "whitespace."

; Strip all carriage return characters in the string.Supprime tous les caractères 'retour chariot' dans la chaîne.
Local $sStrippedString = StringStripCR($sString)

; Display the before and after string.Affiche la chaîne avant et après.
MsgBox($MB_SYSTEMMODAL, "", "Before: " & $sString & @CRLF & @CRLF & "After: " & $sStrippedString)
