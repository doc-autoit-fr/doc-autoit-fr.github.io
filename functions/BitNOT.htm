Function ReferenceRéférence UDF

BitNOT

Performs a bitwise NOT operation.Exécute un "NON" logique bit à bit.

BitNOT ( value )

ParametersParamètres

value The number to operate on.La valeur à traiter.

Return ValueValeur de retour

Returns the bitwise NOT of the value.Retourne un "NON" logique de la valeur.
Bit operations are performed as 32-bit integers.Les opérations sont effectuées comme entiers 32 bits.

RemarksRemarques

Remember hex notation can be used for numbers.La notation hexadécimale peut être utilisée comme arguments.
Remember that in 2's-complement notation, BitNOT() is functionally equivalent to adding 1 and negating the result.Rappelez-vous que dans la notation en complément à 2, la fonction BitNOT() est fonctionnellement équivalent à ajouter 1 et à prendre la négation du résultat.
Also remember that NOT changes a 0 bit to 1 and vice-versa.De même, rappelez-vous que le "NON" logique change 0 en 1 et vice-versa.

ExampleExemple

#include <MsgBoxConstants.au3>

; Assign a Local variable the bitwise NOT operation of 5.Assigne une variable locale le opération NON 5.
Local $iBitNOT1 = BitNOT(5)

#cs Comments:
    Note: "b" is the symbol for byte.

    Result is -6 because for 32-bit numbers
    +5 == 0000 0000 0000 0000 0000 0000 0000 0101b
    -6 == 1111 1111 1111 1111 1111 1111 1111 1010b
    and the first bit is signed
#ce

; Display the result.Affiche le résultat.
MsgBox($MB_SYSTEMMODAL, "", $iBitNOT1)
