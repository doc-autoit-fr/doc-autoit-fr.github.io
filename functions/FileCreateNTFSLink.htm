FileCreateNTFSLink

Creates an NTFS hardlink to a file or a directory.Crée un lien dur NTFS vers un fichier ou un répertoire.

FileCreateNTFSLink ( "source", "hardlink" [, flag = 0] )

ParametersParamètres

source Path of the source to which the hardlink will be created.Chemin de la source à laquelle le hardlink sera créé.
hardlink Path of the hardlink.Chemin du lien dur.
flag [optional][optionnel] this flag determines whether to overwrite link if they already exist.cet indicateur détermine s'il faut remplacer un lien s'il existe déjà.
Can be a combination of the following:Peut-être une combinaison des éléments suivants:
    $FC_NOOVERWRITE (0) = (default) do not overwrite existing link
    $FC_OVERWRITE (1) = overwrite existing link

Constants are defined in FileConstants.au3.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

The destination directory must already exist.Le répertoire de destination doit déjà exister.

This function works only on volumes with the NTFS File system.Cette fonction ne fonctionne que sur un volume avec un système de fichiers NTFS.

If the source is a file, the hardlink must be on the same volume.Si la source est un fichier, le lien dur doit être sur le même volume.
If the source is a directory cross volume is allowed.Si la source est un répertoire, le cross volume est autorisé.

FileDelete() or FileMove() can be used on hardlink.FileDelete() ou FileMove() peut être utilisée sur un 'hardlink'.

To manage the link with the explorer you can use the shell extension NTFSLinkPour gérer le lien avec l'explorateur, vous pouvez utiliser l'extension du shell NTFSLink

ExampleExemple

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create a temporary file to link to.Crée un fichier temporaire pour le relier.
    If Not FileWrite($sFilePath, "This is an example of using FileCreateNTFSLink.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Create a NTFS link of the .txt file to the .log file on the desktop.Crée un lien NTFS du fichier .txt au fichier .log du bureau.
    Local $iNTFSLink = FileCreateNTFSLink($sFilePath, @DesktopDir & "\ExampleNTFSLink.log")

    ; Display a message of whether the NTFS link was created.Affiche un message si le lien NTFS a été créé.
    If $iNTFSLink Then
        ; Open the desktop directory.Ouvre le dossier du bureau.
        ShellExecute(@DesktopDir)

        MsgBox($MB_SYSTEMMODAL, "", "The NTFS link was created." & @CRLF & "FileCreateNTFSLink returned: " & $iNTFSLink)
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The NTFS link wasn't created." & @CRLF & "FileCreateNTFSLink returned: " & $iNTFSLink)
    EndIf

    ; Delete the temporary files.Supprime les fichiers temporaires.
    FileDelete($sFilePath)
    FileDelete(@DesktopDir & "\ExampleNTFSLink.log")
EndFunc   ;==>Example
