HotKeySet

Sets a hotkey that calls a user function.Définit un raccourci clavier qui appelle une fonction utilisateur.

HotKeySet ( "key" [, "function"] )

ParametersParamètres

key The key combination to use as the hotkey.La combinaison de touches à utiliser comme raccourci clavier. Same format as Send(). See remarks.Voir remarques.
function [optional][optionnel][optionnel] The name of the function to call when the key is pressed.Le nom de la fonction à appeler lorsque la touche est appuyée. Not specifying this parameter will unset a previous hotkey.Si cette valeur n'est pas spécifiée, le raccourci précédemment créé est effacé.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0.

RemarksRemarques

It is recommended to use lower-case keys/characters (e.g. "b" and not "B") when setting hotkeys to avoid errors as with some keyboard layouts upper and lower case keys may be mapped differently.

Keyboard with 102 keys as the Hungarian keyboard need to use "{OEM_102}" to capture the "í" key.

If two AutoIt scripts set the same hotkeys, you should avoid running those scripts simultaneously as the second script cannot capture the hotkey unless the first script terminates or unregisters the key prior to the second script setting the hotkey.Si deux scripts AutoIt fixent les mêmes raccourcis clavier, vous devriez éviter de faire fonctionner ces scripts ...simultanément, le second script ne peut pas saisir le raccourci clavier à moins que le premier script ... If the scripts use GUIs, then consider using GUISetAccelerators as these keys are only active when the parent GUI is active.termine ou annule l'enregistrement du raccourci avant la définition du raccourci du deuxième script.

A hotkey-press *typically* interrupts the active AutoIt function/statement and runs its user function until it completes or is interrupted.L'appui sur un raccourci clavier interrompt généralement la fonction ou l'instruction en cours dans le script, et exécute la fonction définie jusqu'à ce que celle-ci se termine ou qu'elle soit interrompue. Exceptions are as follows:Les exceptions sont les suivantes :
1) If the current function is a "blocking" function, then the key-presses are buffered and execute as soon as the blocking function completes.1) Si la fonction en cours est une fonction "bloquante", l'appui sur le raccourci clavier est mis en attente puis traité dès que la fonction bloquante se termine. MsgBox() and FileSelectFolder() are examples of blocking functions. Try the behavior of Shift-Alt-d in the Example.Essayez le comportement de Shift-Alt-d dans l'exemple.
2) If you have paused the script by clicking on the AutoIt Tray icon, any hotkeys pressed during this paused state are ignored.2) Si vous avez mis le script en pause par un clic sur l'icône AutoIt de la zone de notification, tous les raccourcis clavier utilisés pendant cette pause sont ignorés.

The following hotkeys cannot be set:Les combinaisons suivantes ne peuvent pas être attribuées:

ASC nnnn Cannot be used as in Send(). It will use "A" first char as Send() does when the "{...}" is invalid.
Ctrl+Alt+Delete It is reserved by WindowsRaccourci réservé par Windows.
F12 It is also reserved by Windows, according to its API.Raccourci également réservé à Windows, selon son API.
NumPad's Enter Key Instead, use {Enter} which captures both Enter keys on the keyboard.A la place, utilisez {Enter} qui capture les deux touches "Entrée" du clavier.
Win+B,D,E,F,L,M,R,U; and Win+Shift+M These are built-in Windows shortcuts.Ce sont des raccourcis intégrés à Windows. Note:Note : Win+B and Win+L might only be reserved on Windows XP and above.Win+B et Win+L sont probablement réservés sur Windows XP et supérieur.
Alt, Ctrl, Shift, Win These are the modifier keys themselves!Ce sont les touches de modification elles mêmes !
Other Any global hotkeys a user has defined using third-party software, any combos of two or more "base keys" such as '{F1}{F2}', and any keys of the form '{LALT}' or '{ALTDOWN}'.Tout raccourci clavier global qu'un utilisateur a défini par une application tierce, et toute combinaison de deux "touches de base" ou plus telle que '{F1}{F2}' et n'importe qu'elle touche de la forme '{LALT}' ou '{ALTDOWN}'.

When you set a hotkey, AutoIt captures the key-press and does not pass it on to the active application, with one exception: the Lock keys (NumLock, CapsLock, and ScrollLock) still toggle their respective state!Lorsque vous définissez un raccourci clavier, AutoIt capture les appuis de touche sans les faire passer à l'application active en cours, avec une exception : les touches de verrouillage (NumLock, CapsLock, et ScrollLock) basculeront toujours leur état respectif!
To Send() a key combination which will trigger a HotKeySet() event, either use ControlSend() or unregister the HotKeySet() event, otherwise, the Send() event may trigger an infinite loop.Pour Send() (envoyer) une combinaison de touches qui déclenchera un événement HotKeySet(), utilisez ControlSend(), ou annulez l'enregistrement de l'événement HotKeySet(), sinon, l'événement Send() provoquera une boucle infinie.

; capture and pass along a keypress; capture et envoi d'un appui de touche
HotKeySet("{Esc}", "captureEsc")
Func captureEsc()
    ; ... can do stuff here
    HotKeySet("{Esc}")
    Send("{Esc}")
    HotKeySet("{Esc}", "captureEsc")
EndFunc


The called function can not be given parameters.La fonction appelée ne peut pas recevoir de paramètres. They will be ignored.Ils seront ignorés.

@HotKeyPressed macro can be used inside the function to handle several keys in the same function.La macro @HotKeyPressed peut être utilisée à l'intérieur de la fonction pour traiter différentes touches dans la même fonction.

ExampleExemple

Example 1Exemple 1

#include <MsgBoxConstants.au3>

; Press Esc to terminate script, Pause/Break to "pause"Appuyez sur Echap pour terminer le script, Pause/Break pour "pause"

Global $g_bPaused = False

HotKeySet("{PAUSE}", "TogglePause")
HotKeySet("{ESC}", "Terminate")
HotKeySet("+!d", "ShowMessage") ; Shift-Alt-dMaj -Alt- d

While 1
    Sleep(100)
WEnd

Func TogglePause()
    $g_bPaused = Not $g_bPaused
    While $g_bPaused
        Sleep(100)
        ToolTip('Script is "Paused"', 0, 0)
    WEnd
    ToolTip("")
EndFunc   ;==>TogglePause

Func Terminate()
    Exit
EndFunc   ;==>Terminate

Func ShowMessage()
    MsgBox($MB_SYSTEMMODAL, "", "This is a message.")
EndFunc   ;==>ShowMessage

Example 2Exemple 2

#include <MsgBoxConstants.au3>

; Press Esc to terminate script, Pause/Break to "pause"Appuyez sur Echap pour terminer le script, Pause/Break pour "pause"

Global $g_bPaused = False

HotKeySet("{PAUSE}", "HotKeyPressed")
HotKeySet("{ESC}", "HotKeyPressed")
HotKeySet("+!d", "HotKeyPressed") ; Shift-Alt-dMaj -Alt- d

While 1
    Sleep(100)
WEnd

Func HotKeyPressed()
    Switch @HotKeyPressed ; The last hotkey pressed.Le dernier raccourci utilisé.
        Case "{PAUSE}" ; String is the {PAUSE} hotkey.la chaîne est le raccourci {PAUSE}.
            $g_bPaused = Not $g_bPaused
            While $g_bPaused
                Sleep(100)
                ToolTip('Script is "Paused"', 0, 0)
            WEnd
            ToolTip("")

        Case "{ESC}" ; String is the {ESC} hotkey.La chaîne est le raccourci {ESC}.
            Exit

        Case "+!d" ; String is the Shift-Alt-d hotkey.La chaîne est le raccourci Maj -Alt-d.
            MsgBox($MB_SYSTEMMODAL, "", "This is a message.")

    EndSwitch
EndFunc   ;==>HotKeyPressed
