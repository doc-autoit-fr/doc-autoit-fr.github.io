FileDelete

Delete one or more files.Supprime un ou plusieurs fichiers.

FileDelete ( "filename" )

ParametersParamètres

Filename The path of the file(s) to delete. (* and ?( caractères spéciaux * et ? wildcards accepted - See Remarks)acceptés - Voir Remarques)

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 if files are not deleted or do not exist.Retourne 0 si le ou les fichiers n'ont pas étés supprimés ou s'ils n'existent pas.

RemarksRemarques

See FileFindFirstFile() for a discussion about wildcards.Consultez FileFindFirstFile() pour des précisions sur les caractères spéciaux.

Note:Note : If the "path" passed to FileDelete() is a folder, the files therein will be deleted just as if you had used the *.* mask.Si le "chemin" transmis à FileDelete() est un dossier, les fichiers qui y sont, seront supprimés comme si vous aviez utilisé le masque *.*.

Some file attributes can make the deletion impossible, if this is the case look at FileSetAttrib() to change the attributes of a file.Certains attributs de fichier peuvent rendre la suppression impossible, si c'est le cas consultez FileSetAttrib() pour changer les attributs d'un fichier.

ExampleExemple

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create a temporary file to read data from.Crée un fichier temporaire pour y lire les données.
    If Not FileWrite($sFilePath, "This is an example of using FileDelete.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Display the contents of the file passing the filepath to FileRead instead of a handle returned by FileOpen.Affiche le contenu du fichier en passant le chemin du fichier à FILEREAD au lieu du handle retournée par FileOpen.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & FileRead($sFilePath))

    ; Delete the temporary file.Supprimez le fichier temporaire.
    Local $iDelete = FileDelete($sFilePath)

    ; Display a message of whether the file was deleted.Affiche un message si le fichier a été supprimé.
    If $iDelete Then
        MsgBox($MB_SYSTEMMODAL, "", "The file was successfuly deleted.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst deleting the file.")
    EndIf
EndFunc   ;==>Example
