GUICtrlCreatePic

Creates a Picture control for the GUI.Crée un contrôle Picture (image) dans une GUI.

GUICtrlCreatePic ( filename, left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

filename filename of the picture to be loaded : supported types BMP, JPG, GIF(but not animated).Adresse de l'image à charger: types pris en charge BMP, JPG, GIF (mais sans animation).
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional][optionnel][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default (-1) : $SS_NOTIFY
    forced style : $SS_BITMAP
exStyle [optional][optionnel][optionnel] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0 if picture cannot be created.Retourne 0 si le contrôle ne peut pas être créé.

RemarksRemarques

To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

To update the picture after the dialog box is displayed just use GUICtrlSetImage().Pour mettre à jour l'image lorsque le contrôle est affichée il suffit d'utiliser GUICtrlSetImage

To set the picture control to the same size as the file content set width and height to 0.Si vous voulez avoir une image ayant la même taille que le contenu du fichier il suffit d'utiliser width=height=0.

To have a transparent picture create the GUI with the WS_EX_LAYERED extended style.Pour avoir une image transparente, il est nécessaire pour créer la fenêtre GUI avec le style étendu WS_EX_LAYERED. The left-top pixel will be used as the transparency color.Le pixel supérieur gauche est utilisé en tant que couleur de transparence. If several pictures are created the last picture defines the transparent color.Si plusieurs photos sont créés la dernière photo définit la couleur de transparence. See example 2.Consultez l'exemple 2.

To combine styles with the default style use BitOR($GUI_SS_DEFAULT_PIC, newstyle, ... ).)..
To use the values specified above you must #include <StaticConstants.au3> in your script.

Default resizing is $GUI_DOCKSIZE.Par défaut redimensionnement est $ GUI_DOCKSIZE.

If a picture is set as a background picture the other controls will overlap, so it is important to disable the pic control:Si une image est définie comme une image de fond, comme les autres contrôles se chevauchent, il est important de désactiver le contrôle d'image et de le créer après les autres contrôles:<a0>GUICtrlSetState</a0>(-1, $GUI_DISABLE). GUICtrlSetState(-1, $GUI_DISABLE).Ce n'est pas suffisant pour l'onglet ou le contrôle ListView qui se comportent différemment.Dans ce cas, vous avez besoin de créer l'image avec le style $WS_CLIPSIBLINGS, GuiCtrlSetState(-1, $GUI_ONTOP) est nécessaire pour l'onglet, TreeView ou contrôle ListView.

The extended style $GUI_WS_EX_PARENTDRAG can be used to allow the dragging of the parent window for windows that do not have a titlebar (no $WS_CAPTION style in GUICreate()).Le style étendu $GUI_WS_EX_PARENTDRAG peut être utilisé pour permettre le glissement de la fenêtre parent pour les fenêtres qui n'ont pas de barre de titre (pas de style $WS_CAPTION dans GUICreate()).

The background is always set to transparent.Le fond est toujours mis à transparent. GUICtrlSetBkColor() has no effect on pic control.

ExampleExemple

Example 1Exemple 1

#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Example()

Func Example()
    GUICreate("My GUI picture", 350, 300, -1, -1, $WS_SIZEBOX + $WS_SYSMENU) ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUISetBkColor(0xE0FFFF)
    Local $idPic = GUICtrlCreatePic("..\GUI\mslogo.jpg", 50, 50, 200, 50)

    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    ; resize the controlRedimensionne le contrôle
    GUICtrlSetPos($idPic, 50, 50, 200, 100)

    Local $idMsg
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        $idMsg = GUIGetMsg()

        If $idMsg = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd

    GUIDelete()
EndFunc   ;==>Example

Example 2Exemple 2

#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Global $g_hGui, $g_aGuiPos, $g_hPic, $g_aPicPos

Example()

Func Example()
    $g_hGui = GUICreate("test transparentpic", 200, 100)
    $g_hPic = GUICreate("", 68, 71, 10, 20, $WS_POPUp, BitOR($WS_EX_LAYERED, $WS_EX_MDICHILD), $g_hGui)
    GUICtrlCreatePic("..\GUI\merlin.gif", 0, 0, 0, 0)

    GUISetState(@SW_SHOW, $g_hPic)
    GUISetState(@SW_SHOW, $g_hGui)

    HotKeySet("{ESC}", "Main")
    HotKeySet("{Left}", "Leftgauche")
    HotKeySet("{Right}", "Rightdroit")
    HotKeySet("{Down}", "Down")
    HotKeySet("{Up}", "Up")
    $g_aPicPos = WinGetPos($g_hPic)
    $g_aGuiPos = WinGetPos($g_hGui)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    HotKeySet("{ESC}")
    HotKeySet("{Left}")
    HotKeySet("{Right}")
    HotKeySet("{Down}")
    HotKeySet("{Up}")
EndFunc   ;==>Example

Func Main()
    $g_aGuiPos = WinGetPos($g_hGui)
    WinMove($g_hGui, "", $g_aGuiPos[0] + 10, $g_aGuiPos[1] + 10)
EndFunc   ;==>Main

Func Left()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0] - 10, $g_aPicPos[1])
EndFunc   ;==>Left

Func Right()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0] + 10, $g_aPicPos[1])
EndFunc   ;==>Right

Func Down()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0], $g_aPicPos[1] + 10)
EndFunc   ;==>Down

Func Up()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0], $g_aPicPos[1] - 10)
EndFunc   ;==>Up
