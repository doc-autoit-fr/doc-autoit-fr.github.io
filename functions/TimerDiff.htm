TimerDiff

Returns the difference in time from a previous call to TimerInit().Retourne la différence en temps à partir d'un précédent appel à TimerInit().

TimerDiff ( handle )

ParametersParamètres

handle Handle returned from a previous call to TimerInit().Handle retourné à partir d'un précédent appel à TimerInit().

Return ValueValeur de retour

Returns the time difference (in milliseconds) from a previous call to TimerInit().Retourne la différence de temps (en millisecondes) à partir d'un précédent appel à TimerInit().

RemarksRemarques

Some processors are known to cause the function to return incorrect result.Quelques processeurs sont connus pour fausser les résultats de la fonction.
