FileGetEncoding

Determines the text encoding used in a file.Détermine l'encodage du texte utilisé dans un fichier.

FileGetEncoding ( "filehandle/filename" [, mode = 1] )

ParametersParamètres

filehandle/filename The handle of a file, as returned by a previous call to FileOpen().Le handle d'un fichier, renvoyé par l'utilisation de la commande FileOpen(). Alternatively you may use a string filename as the first parameter.Alternativement le nom du fichier.
mode [optional][optionnel] The UTF8 detection mode to use.Le mode de détection UTF8 à utiliser.
    $FE_ENTIRE_UTF8 (1) = Check entire file for UTF8 sequences (default)
    $FE_PARTIALFIRST_UTF8 (2) = Check first part of file for UTF8 sequences (the same as FileOpen() uses by default)

Constants are defined in FileConstants.au3.

Return ValueValeur de retour

Success:Succès : the file encoding using similar values to the FileOpen() function:Retourne l'encodage du fichier en utilisant des valeurs similaires à la fonction FileOpen():
    $FO_UTF16_LE (32) = UTF16 Little Endian.
    $FO_UTF16_BE (64) = UTF16 Big Endian.
    $FO_UTF8 (128) = UTF8 (with BOM).
    $FO_UTF8_NOBOM (256) = UTF8 (without BOM).
    $FO_ANSI (512) = ANSI (containing char > 127 and < 255)
Failure:Échec : -1.Retourne -1.

Constants are defined in FileConstants.au3.

RemarksRemarques

If a filename is given rather than a file handle - the file will be opened and closed during the function call.Si un fichier est donné plutôt qu'un handle, le fichier sera ouvert puis fermé durant l'appel de fonction.
Note:Note : Do not mix filehandles and filenames, i.e., don't FileOpen() a file and then use a filename in this function.Ne pas mélanger handles et des chemins d'accès, c'est à dire, ne pas utiliser FileOpen() pour un fichier et ensuite utiliser un chemin d'accès dans cette fonction. Use either filehandles or filenames in your routines, not both!Utilisez des handles de fichiers ou des noms de fichiers dans vos routines, mais pas les deux!

If a file handle is used then the "mode" parameter has no effect - the mode used for FileOpen() takes precedence.Si un handle de fichier est utilisé alors le paramètre "mode" n'a aucun effet - Le mode utilisé pour FileOpen() a la priorité.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iEncoding = FileGetEncoding(@ScriptFullPath) ; Retrieve the file encoding of the current script.Récupére l'encodage du fichier du script courant.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error: Could not obtain the file encoding.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "FileGetEncoding: The value returned was: " & $iEncoding) ; The value returned for this example should be 0 or $FO_ANSI.
    EndIf
EndFunc   ;==>Example
