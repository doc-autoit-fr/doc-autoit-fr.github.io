TraySetToolTip

(Re)Sets the tooltip text for the tray icon.(Re)Définit le texte de l'info-bulle pour l'icône de la zone de notification.

TraySetToolTip ( [text] )

ParametersParamètres

text [optional][optionnel][optionnel] The new text to be displayed as tooltip.Le nouveau texte qui doit être affiché comme info-bulle. The length is limited - see Remarks.La longueur est limitée - Voir Remarques.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0.

RemarksRemarques

The tooltip length is limited to 128 characters.La longueur de l'info-bulle est limitée à 128 caractères.

To reset the tooltip to the default text, use the function with no parameters TraySetToolTip().

The tooltip text is only changed when the tray icon is visible!Le texte de l'info-bulle n'est modifié que lorsque l'icône est visible!
