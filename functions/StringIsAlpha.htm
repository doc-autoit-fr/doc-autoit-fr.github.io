Référence Fonction

StringIsAlpha

Checks if a string contains only alphabetic characters.Vérifie si la chaîne contient uniquement des caractères alphabétiques.

StringIsAlpha ( "string" )

ParametersParamètres

string The string to checkLa chaîne à traiter

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if string contains non-alphabetic characters.Retourne 0 si la chaîne contient des caractères non-alphabétiques.

RemarksRemarques

Keep in mind that any whitespace will cause StringIsAlpha() to return 0.Notez que les espaces blancs feront que StringIsAlpha retournera 0.

RelatedLiens

StringIsAlNum, StringIsASCII, StringIsDigit, StringIsLower, StringIsSpace, StringIsUpper, StringIsXDigit, StringLower, StringUpper

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sString = "This is a sentence with whitespace." ; Check whether this string contains only alphabetic characters.Vérifie si cette chaîne ne contient que des caractères alphabétiques.
If StringIsAlpha($sString) Then
    MsgBox($MB_SYSTEMMODAL, "", "The variable contains only alphabetic characters.")
Else
    MsgBox($MB_SYSTEMMODAL, "", "The variable does not contain alphabetic characters.")
EndIf
