GUICtrlSetData

Modifies the data for a control.Modifie les données d'un contrôle.

GUICtrlSetData ( controlID, data [, default] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
data Combo, List, ListView, ListViewItem:Combo, Liste, ListView, ListViewItem : An Opt("GUIDataSeparatorChar",...) separated list of items.Le caractère Opt("GUIDataSeparatorChar",...)est utilisé comme séparateur.
Progress:Progress : The percentage.Le pourcentage.
Slider:Slider : The value.La valeur.
Button, Checkbox, Combo, Edit, Group, Input, Label, List, Menu, MenuItem, Radio, TabItem, TreeViewItem: Replaces the text.Remplace le texte.
Date :Date: The date or time depending the style of the control and the regional settings.La date ou l'heure selon le style du contrôle et des paramètres régionaux.
Dummy: The value.La valeur.
default [optional][optionnel]
Combo, List:Combo, List : The default value.La valeur par défaut.
Edit, Input: If non-empty (""), the string is inserted at the current insertion point (caret).Si non vide (""), la chaîne est insérée au point d'insertion courant (caret).

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.
-1 in case of invalid data-1 En cas de données non valides

RemarksRemarques

For Combo or List control :Pour les contrôles Combo ou List :
If the "data" corresponds to an already existing entry it is set as the default.Si la "donnée" correspond à une entrée déjà existante, elle est définie comme la valeur par défaut.
If the "data" starts with GUIDataSeparatorChar or is an empty string "" the previous list is destroyed.Si la "donnée" commence par GUIDataSeparatorChar ou est une chaîne vide "" la liste précédente est détruite. A trailing GUIDataSeparatorChar is ignored.

For ListView, ListViewItem controls :Pour les contrôles ListView, ListViewItem :
To update a specific column just forget about the others ie "||update" to update 3rd column.Pour mettre à jour une colonne spécifique il suffit d'oublier les autres c'est-à-dire "||update" pour mettre à jour la 3ème colonne.
If "update" is empty the column/subitem will be erased.Si "update" est vide, la colonne/sous-item sera effacé. For example "|" will erase the second column/subitem, "" will erase the first.Par exemple "|" effacera la deuxième colonne/sous-item, "" effacera la première.

For Monthcal controls :Pour les contrôles Monthcal :
The "data" date format is "yyyy/mm/dd".Le format date de la "donnée" est "yyyy/mm/dd".

For Date controls:Pour les contrôles Date :
The date and time is in the format defined by the regional settings.La date et l'heure est au format défini par les paramètres régionaux. GUICtrlRead() use the same default format.

    GUISetState(@SW_SHOW) ; will display an empty dialog box with a combo control with focus onAffiche une boîte de dialogue vide avec un contrôle Combo et le focus dessus

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
