DriveMapDel

Disconnects a network drive.Déconnecte un lecteur réseau.

DriveMapDel ( "drive" )

ParametersParamètres

drive The device to disconnect, e.g. "O:" or "LPT1:".

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if the disconnection was unsuccessful.Retourne 0 si la déconnexion à échouée.

RemarksRemarques

If a connection has no drive letter mapped you may use the connection name to disconnect, e.g. \\server\share

RelatedLiens

DriveMapAdd, DriveMapGet

ExampleExemple

; Map X drive to \\myserver\stuff using current userMappe le lecteur X sur \\myserver\stuff en utilisant l'utilisateur courant
DriveMapAdd("X:", "\\myserver\stuff")

; DisconnectDéconnecte
DriveMapDel("X:")
