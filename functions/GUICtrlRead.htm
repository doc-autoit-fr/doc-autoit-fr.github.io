GUICtrlRead

Read state or data of a control.Lit l'état ou les données d'un contrôle.

GUICtrlRead ( controlID [, advanced = 0] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
advanced [optional][optionnel][optionnel] returns extended information of a control.Retourne des informations étendus d'un contrôle.
    $GUI_READ_DEFAULT (0) = (Default) Returns a value with state or data of a control.
    $GUI_READ_EXTENDED (1) = Returns extended information of a control (see Remarks).

Constants are defined in GUIConstantsEx.au3.

Return ValueValeur de retour

Success:Succès : depends on the control (see below).Retourne une valeur qui dépend du contrôle (voir ci-dessous).
Failure:Échec : 0.Retourne 0.


Type ValueValeur
Checkbox, Radio The checked state ($GUI_CHECKED or $GUI_UNCHECKED)
Combo, List The value selectedLa valeur sélectionnée
Input, Edit The text enteredLe texte entré
Button The display textLe texte affiché
Date The selected date in the format defined by the regional settingsLa date sélectionnée dans le format défini par les paramètres régionaux
Progress Current percentagePourcentage en cours
Slider Current valueValeur en cours

TreeViewItem State of the TreeViewItemÉtat du TreeViewItem
ListView Control identifier (controlID) of the selected ListViewItem.Identifiant du contrôle (controlID) du ListViewItem sélectionné. 0 means no item is selected0 signifie aucun item sélectionné
ListViewItem The text of the selected item/lines in the ListViewLe texte de l'item/ligne sélectionné dans le ListView
Dummy The value set by GUICtrlSendToDummy() or GUICtrlSetData()

RemarksRemarques

In 'advanced' mode the return value contains additional data of the control (see below).En mode 'avancé' la valeur de retour contient des données supplémentaires sur le contrôle (Voir ci-dessous).

Note: not all controls return additional data!Remarque: pas tous les contrôles renvoient des données supplémentaires!

Type Additional ValueValeurs supplémentaires
Checkbox, Radio The text of the control.Le texte de la commande.
Menu, MenuItem The text of the control.Le texte de la commande.
TreeView The text of the current selected TreeViewItem.Le texte du TreeViewItem actuellement sélectionné.
TreeViewItem The text of the TreeViewItem.Le texte du TreeViewItem.
ListViewItem The state of the ListViewItem if $LVS_EX_CHECKBOXES exStyle used in advanced mode.L'état du ListViewItem si le style étendu $LVS_EX_CHECKBOXES est utilisé en mode avancé. See State tableVoir Tableau des états
Tab The controlID of the tabitem selectedLe controlID du TabItem sélectionné

For Checkbox and Radio controls only the $GUI_CHECKED (1), $GUI_UNCHECKED (4) or $GUI_INDETERMINATE (2) states are returned so the value can be used directly.Pour les contrôles Checkbox et Radio seuls les états $GUI_CHECKED (1), $GUI_UNCHECKED (4) ou $GUI_INDETERMINATE (2) sont retournés, aussi la valeur peut être utilisée directement.

For Listview items several states can be returned as $GUI_CHECKED and $GUI_UNCHECKED (only for listview controls with LVS_EX_CHECKBOXES-exstyle and on 'advanced' return) . So use i.e. BitAND(GUICtrlRead($Item),$GUI_CHECKED) to test if the item is checked.Donc utilisez BitAND(GUICtrlRead($item),$GUI_CHECKED) pour tester si l'item est cochée.

For Treeview items several states can be returned as $GUI_FOCUS, $GUI_EXPAND and $GUI_CHECKED, $GUI_UNCHECKED (only for treeview controls with TVS_CHECKBOXES-style . So use i.e. BitAND(GUICtrlRead($Item),$GUI_CHECKED) to test if the item is checked.Donc utilisez BitAND(GUICtrlRead($item),$GUI_CHECKED) pour tester si l'item est cochée.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("My GUICtrlRead") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    Local $idMenu1 = GUICtrlCreateMenu("File")

    Local $idList = GUICtrlCreateList("", 10, 10, -1, 100)
    GUICtrlSetData(-1, "item1|item2|item3", "item2")

    Local $idButton = GUICtrlCreateButton("Read", 10, 110, 50)
    GUICtrlSetState(-1, $GUI_FOCUS) ; the focus is on this buttonle focus est mis sur ce bouton

    GUISetState(@SW_SHOW) ; will display an empty dialog boxAffiche une boîte de dialogue vide

    Local $idMsg, $iMenustate, $sMenutext
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    Do
        $idMsg = GUIGetMsg()
        If $idMsg = $idButton Then
            MsgBox($MB_SYSTEMMODAL, "Selected listbox entry", GUICtrlRead($idList)) ; display the selected listbox entryAffiche l'entrée de zone de liste sélectionnée
            $iMenustate = GUICtrlRead($idMenu1) ; return the state of the menu itemrenvoie l'état de l'item du menu
            $sMenutext = GUICtrlRead($idMenu1, $GUI_READ_EXTENDED) ; return the text of the menu itemrenvoie le texte de l'item du menu
            MsgBox($MB_SYSTEMMODAL, "State and text of the menuitem", "state:" & $iMenustate & @CRLF & "text:" & $sMenutext)
        EndIf
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
