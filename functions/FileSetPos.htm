Référence Fonction

FileSetPos

Sets the current file position.Définit la position actuelle dans un fichier.

FileSetPos ( "filehandle", offset, origin )

ParametersParamètres

filehandle The handle of a file, as returned by a previous call to FileOpen().Le handle d'un fichier, renvoyé par l'utilisation de la commande FileOpen().
offset The offset to move from the origin.Le décalage pour déplacer à partir de l'origine. This value may be positive or negative.Cette valeur peut être positive ou négative. Negative values move backwards from the origin.Les valeurs négatives déplacent vers l'arrière de l'origine.
origin Must be one of the following:Doit être l'un des suivants:
    $FILE_BEGIN (0) = Beginning of the file.
    $FILE_CURRENT (1) = Current position.
    $FILE_END (2) = End of the file.

Constants are defined in FileConstants.au3.

Return ValueValeur de retour

Success: True if the operation succeeded.Retourne True si l'opération a réussi.
Failure: False.Retourne False.

RemarksRemarques

Include Constants.au3 in your script to use the symbolic name in parentheses to specify the origin.Inclure Constants.au3 dans votre script pour utiliser le nom symbolique entre parenthèses pour préciser l'origine.
Using FileSetPos() it is possible to both read and write to the same file.En utilisant FileSetPos() il est possible de lire et écrire sur le même fichier. When attempting to read and write to the same file, always call FileFlush() between each write and read operation.Lorsque vous tentez de lire et d'écrire sur le même fichier, toujours utiliser FileFlush() entre chaque opération de lecture et d'écriture.
Moving the pointer to the middle of the data can be used to overwrite data.Déplacer le pointeur vers le milieu des données peut être utilisé pour écraser les données.

RelatedLiens

FileFlush, FileGetPos, FileOpen, FileRead, FileReadLine, FileWrite, FileWriteLine

ExampleExemple

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Open the file for writing (overwrite the file) and store the handle to a variable.Ouvre le fichier en écriture (écrase le fichier) et stocke le handle dans une variable.
    Local $hFileOpen = FileOpen($sFilePath, $FO_OVERWRITE)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Write data to the file using the handle returned by FileOpen.Écrit des données dans le fichier à l'aide du handle retournée par FileOpen.
    FileWriteLine($hFileOpen, "Line 1")
    FileWriteLine($hFileOpen, "Line 2")
    FileWriteLine($hFileOpen, "Line 3")

    ; Flush the file to disk.Vide le fichier sur le disque.
    FileFlush($hFileOpen)

    ; Check file position and try to read contents for current position.Vérifie la position dans le fichier et essaie de lire le contenu à partir de la position actuelle.
    MsgBox($MB_SYSTEMMODAL, "", "Position: " & FileGetPos($hFileOpen) & @CRLF & "Data: " & @CRLF & FileRead($hFileOpen))

    ; Now, adjust the position to the beginning.Maintenant, ajuste la position sur le début.
    FileSetPos($hFileOpen, 0, $FILE_BEGIN)

    ; Check file position and try to read contents for current position.Vérifie la position dans le fichier et essaie de lire le contenu à partir de la position actuelle.
    MsgBox($MB_SYSTEMMODAL, "", "Position: " & FileGetPos($hFileOpen) & @CRLF & "Data: " & @CRLF & FileRead($hFileOpen))

    ; Close the handle returned by FileOpen.Ferme le handle renvoyé par FileOpen.
    FileClose($hFileOpen)

    ; Delete the temporary file.Supprimez le fichier temporaire.
    FileDelete($sFilePath)
EndFunc   ;==>Example
