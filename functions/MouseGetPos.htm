Référence Fonction

MouseGetPos

Retrieves the current position of the mouse cursor.Récupère la position actuelle du curseur de la souris.

MouseGetPos ( [dimension] )

ParametersParamètres

dimension [optional] argument that determines what the return value will be.Argument qui détermine ce que sera la valeur de retour. See Return Value.Voir Valeur de retour.

Return ValueValeur de retour

Dimension Valuesvaleurs
None Returns a two-element array containing the mouse coordinates:Retourne un array à 2 éléments contenant les coordonnées de la souris : $aArray[0] = X coord (horizontal), $aArray[1] = Y coord (vertical)$aArray[0] = X (abscisse), $aArray[1] = Y (ordonnée)
0Retourne 0 Returns the X co-ordinate as an integer.Retourne l'abscisse X sous la forme d'un 'integer'.
1Retourne 1 Returns the Y co-ordinate as an integer.Retourne l'ordonnée Y sous la forme d'un 'integer'.

If dimension is not a number @error will be set to 1.Si 'dimension' n'est pas un nombre @error = 1.

RemarksRemarques

See MouseCoordMode for relative/absolute position settings.Voir MouseCoordMode pour paramètrer la position relative/absolue. If relative positioning, numbers may be negative.Si le positionnement est relatif, les nombres retournés peuvent être négatifs.

ExampleExemple

#include <MsgBoxConstants.au3>

Local $aPos = MouseGetPos()
MsgBox($MB_SYSTEMMODAL, "Mouse x, y:", $aPos[0] & ", " & $aPos[1])
