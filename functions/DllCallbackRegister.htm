DllCallbackRegister

Creates a user-defined DLL Callback function.Crée une fonction utilisateur de rappel DLL.

DllCallbackRegister ( "function", "return type", "params" )

ParametersParamètres

function The name of the User Defined Function to call.Le nom défini par l'utilisateur de la fonction à appeler.
return type The return type and calling convention of the function (see DllCall).Le type de retour et de convention d'appel de la fonction (ConsultezDllCall).
params A semi-colon separated list of parameters that will be passed to this function.Un point-virgule sépare la liste des paramètres qui seront passés à cette fonction. See Remarks.Voir Remarque.

Return ValueValeur de retour

Success:Succès : a dll "handle" to be used with DllCallbackGetPtr() and DllCallbackFree() functions.Retourne le handle de la DLL pour être utilisé avec les fonctions DllCallbackGetPtr() et DllCallbackFree().
Failure:Échec : 0 if error occurs.Retourne 0 en cas d'erreur.

RemarksRemarques

Uses all DllCall() types except "struct".Utilisez tous les types de DllCall() excepté "struct".
When finished working with a callback, call the DllCallbackFree() function to close it.Lorsque le travail est terminé avec un rappel, appelez la fonction DllCallbackFree() pour le fermer. AutoIt normally closes all files upon termination, but explicitly calling DllCallbackFree() is still a good idea.AutoIt ferme normalement tous les fichiers à la résiliation, mais un appel explicite à DllCallbackFree() est toujours une bonne idée.

RelatedLiens

DllCall, DllCallbackFree, DllCallbackGetPtr

ExampleExemple

#include <MsgBoxConstants.au3>

; Create callback function.Crée une fonction de rappel.
Local $hHandle = DllCallbackRegister("_EnumWindowsProc", "int", "hwnd;lparam")

; Call EnumWindows.Appelle EnumWindows.
DllCall("user32.dll", "int", "EnumWindows", "ptr", DllCallbackGetPtr($hHandle), "lparam", 10)

; Delete callback function.Supprime une fonction de rappel.
DllCallbackFree($hHandle)

; Callback Procedureprocédure de rappel
Func _EnumWindowsProc($hWnd, $lParam)
    ; If the Title is empty or if the window is not visible then continue enumeration.Si le titre est vide ou si la fenêtre n'est pas visible, alors continuer l'énumération.
    If WinGetTitle($hWnd) = "" Or BitAND(WinGetState($hWnd), 2) = 0 Then Return 1

    Local $iRes = MsgBox(BitOR($MB_SYSTEMMODAL, $MB_OKCANCEL), _
            WinGetTitle($hWnd), "$hWnd=" & $hWnd & @CRLF & _
            "$lParam=" & $lParam & @CRLF & _
            "$hWnd(type)=" & VarGetType($hWnd))

    If $iRes <> $IDOK Then Return 0 ; Cancel/Close button clicked, return 0 to stop enumeration.Annuler / Fermer bouton cliqué, retourner 0 pour arrêter l'énumération.

    Return 1 ; Return 1 to continue enumeration.Retour 1 pour continuer l'énumération.
EndFunc   ;==>_EnumWindowsProc
