GUICtrlCreateEdit

Creates an Edit control for the GUI.Crée un contrôle Edit pour l'interface graphique.

GUICtrlCreateEdit ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

text The text of the control.Le texte de la commande.
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default ( -1) : $ES_WANTRETURN, $WS_VSCROLL, $WS_HSCROLL, $ES_AUTOVSCROLL, $ES_AUTOHSCROLL
    forced styles : $ES_MULTILINE, $WS_TABSTOP only if not $ES_READONLY$ES_MULTILINE, $WS_TABSTOP seulement sans $ES_READONLY
exStyle [optional][optionnel] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .
    default ( -1) : $WS_EX_CLIENTEDGE

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

To obtain the value of the control see GUICtrlRead().Pour obtenir la valeur du contrôle, consultez GUICtrlRead() .
To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

To combine styles with the default style use BitOR($GUI_SS_DEFAULT_EDIT, newstyle, ... ).)..
To use the values specified above you must #include <EditConstants.au3> in your script.

If you want to drag & drop a filename onto this control just add the WS_EX_ACCEPTFILES extended style on the GUICreate() and set the state to $GUI_DROPACCEPTED.Si vous voulez glisser et déplacer un nom de fichier dans le contrôle, il suffit d'ajouter WS_EX_ACCEPTFILES en style étendu sur le GUICreate() et définir l'état avec $GUI_DROPACCEPTED.
Multiple selected files will be dropped as separate lines.Si plusieurs fichiers sont sélectionnés , ils seront déposés sur des lignes distinctes.

Default resizing is $GUI_DOCKAUTO size and position will occur.Pour redimensionner par défaut, utiliez $GUI_DOCKAUTO.

Creating a RichEdit control is too complex so it will not be included as a basic control.La création d'un contrôle RichEdit est trop complexe donc ne sera pas inclus dans un contrôle de base.
You have to use the _GUICtrlRichEdit_Create().Vous devez utiliser _GUICtrlRichEdit_Create().

ExampleExemple

#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Example()

Func Example()
    GUICreate("My GUI edit") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    Local $idMyedit = GUICtrlCreateEdit("First line" & @CRLF, 176, 32, 121, 97, $ES_AUTOVSCROLL + $WS_VSCROLL)

    GUISetState(@SW_SHOW)

    Send("{END}")

    ; will be append dont' forget 3rd parameterdonnées ajoutées: n'oubliez pas le 3ème paramètre
    GUICtrlSetData($idMyedit, "Second line", 1)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
    GUIDelete()
EndFunc   ;==>Example
