WinWaitNotActive

Pauses execution of the script until the requested window is not active.Met le script en pause jusqu'à ce que la fenêtre indiquée ne soit plus active.

WinWaitNotActive ( "title" [, "text" [, timeout = 0]] )

ParametersParamètres

title The title/hWnd/class of the window to check.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel] The text of the window to check.Le texte de la fenêtre à contrôler. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .
timeout [optional][optionnel] Timeout in seconds if the window is still active.Temporisation en secondes si la fenêtre est encore active. Default is 0 (no timeout).La valeur par défaut est 0 (pas de temporisation).

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if timeout occurred.Retourne 0 si la temporisation est dépassée.

RemarksRemarques

The window is polled every 250 milliseconds or so.La fenêtre est interrogée toutes les 250 millisecondes ou plus.

RelatedLiens

WinActive, WinExists, WinWait, WinWaitActive, WinWaitClose, WinWaitDelay (Option)

ExampleExemple

; Wait for the window "[CLASS:Notepad]" to not be active.Attend que la fenêtre "[CLASS:Notepad]" ne soit plus active.
WinWaitNotActive("[CLASS:Notepad]")

; Wait a maximum of 5 seconds for "[CLASS:Notepad]" to not be active.Attend un maximum de 5 secondes que "[CLASS:Notepad]" ne soit plus active.
WinWaitNotActive("[CLASS:Notepad]", "", 5)
