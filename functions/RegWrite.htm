RegWrite

Creates a key or value in the registry.Crée une clé ou une valeur dans le registre.

RegWrite ( "keyname" [, "valuename", "type", value] )

ParametersParamètres

keyname The registry key to write to.La clé de registre à écrire. If no other parameters are specified this key will simply be created.Si pas d'autre paramètre n'est spécifié cette clé sera simplement créée.
valuename [optional] The valuename to write to.Le nom de la valeur à écrire.
type [optional] Type of key to write:Type de clé à écrire: "REG_SZ", "REG_MULTI_SZ", "REG_EXPAND_SZ", "REG_DWORD", "REG_QWORD", or "REG_BINARY"."REG_SZ", "REG_MULTI_SZ", "REG_EXPAND_SZ", "REG_DWORD", "REG_QWORD", ou "REG_BINARY".
value [optional] The value to write.La valeur à écrire.

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 and sets the @error flag to non-zero if error writing registry key or value.Retourne 0 et @error <> 0 s'il y a une erreur d'écriture dans la clé de registres ou dans la valeur.
@error:@error : 1 = unable to open requested key1 = impossible d'ouvrir la clé demandée
 2 = unable to open requested main key
 3 = unable to remote connect to the registry
-1 = unable to open requested value-1 = impossible d'ouvrir la valeur demandée
-2 = value type not supported-2 = type de valeur non pris en charge

RemarksRemarques

A registry key must start with "HKEY_LOCAL_MACHINE" ("HKLM") or "HKEY_USERS" ("HKU") or "HKEY_CURRENT_USER" ("HKCU") or "HKEY_CLASSES_ROOT" ("HKCR") or "HKEY_CURRENT_CONFIG" ("HKCC").Une clé de registre doit commencer par "HKEY_LOCAL_MACHINE" ("HKLM") ou "HKEY_USERS" ("HKU") ou "HKEY_CURRENT_USER" ("HKCU") ou "HKEY_CLASSES_ROOT" ("HKCR") ou "HKEY_CURRENT_CONFIG" ("HKCC") .

When running on 64-bit Windows if you want to write a key or value specific to the 64-bit environment you have to suffix the HK... with 64 i.e.Lors de l'exécution sur Windows 64 bits si vous voulez écrire une clé ou une valeur spécifique à l'environnement 64 bits, vous devez suffixer HK ...avec 64 i.e. HKLM64.avec 64 c'est à dire HKLM64.

AutoIt supports registry keys of type REG_BINARY, REG_SZ, REG_MULTI_SZ, REG_EXPAND_SZ, REG_QWORD, and REG_DWORD.AutoIt supporte les clés de registre de type REG_BINARY, REG_SZ, REG_MULTI_SZ, REG_EXPAND_SZ, REG_QWORD, et REG_DWORD.

To access the (Default) value use "" (an empty string) for the valuename.Pour accéder à la valeur par (par défaut) utilisez "" (une chaîne vide) pour le nom de la valeur.

When writing a REG_MULTI_SZ key you must separate each value with @LF.Lors de l'écriture d'une clé REG_MULTI_SZ vous devez séparer chaque valeur par @LF. The value must NOT end with @LF unless you want an empty substring (see example).La valeur ne doit pas se terminer par @LF sauf si vous voulez une chaîne vide (Voir exemple).

It is possible to access remote registries by using a keyname in the form "\\computername\keyname".Il est possible d'accéder à des registres à distance en utilisant des noms de valeur sous la forme "\\nom_ordinateur\nom_de_la_clé". To use this feature you must have the correct access rights.Pour utiliser cette fonctionnalité vous devez avoir les droits d'accès corrects.

ExampleExemple

; Write a single REG_SZ valueDonne une valeur REG_SZ unique
RegWrite("HKEY_CURRENT_USER\Software\Test", "TestKey", "REG_SZ", "Hello this is a test")

; Write the REG_MULTI_SZ value of "line1" and "line2"Ecrit la valeur REG_MULTI_SZ de "ligne1" et "ligne2"
RegWrite("HKEY_CURRENT_USER\Software\Test", "TestKey1", "REG_MULTI_SZ", "line1" & @CRLF & "line2")

; Write the REG_MULTI_SZ value of "line1"Ecrit la valeur REG_MULTI_SZ de "ligne1"
RegWrite("HKEY_CURRENT_USER\Software\Test", "TestKey2", "REG_MULTI_SZ", "line1")

; always add and extra null stringtoujours ajouter et chaîne à extra null
RegWrite("HKEY_CURRENT_USER\Software\Test", "TestKey3", "REG_MULTI_SZ", "line1" & @CRLF & "line2" & @CRLF)
RegWrite("HKEY_CURRENT_USER\Software\Test", "TestKey4", "REG_MULTI_SZ", "line1" & @CRLF & @CRLF & "line2" & @CRLF)

; empty REG_MULTI_SZvide REG_MULTI_SZ
RegWrite("HKEY_CURRENT_USER\Software\Test", "TestKey5", "REG_MULTI_SZ", "")

; create just the keycrée la clé
RegWrite("HKEY_CURRENT_USER\Software\Test1")
