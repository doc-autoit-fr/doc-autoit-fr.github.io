FileMove

Moves one or more files.Déplace un ou plusieurs fichiers.

FileMove ( "source", "dest" [, flag = 0] )

ParametersParamètre

source The source path and filename of the file to move.Chemin du/des fichier(s) à déplacer. (* wildcards accepted - See Remarks)(Les Wildcards * sont pris en charge)
dest The destination path and filename of the moved file.Chemin et nom du répertoire de destination du/des fichier(s) déplacer. (* wildcards accepted - See Remarks)(Les Wildcards * sont pris en charge)
flag [optional][optional] False - i, v, x ... this flag determines whether to overwrite files if they already exist:ce paramètre détermine s'il faut écraser les fichiers existants.
Can be a combination of the following:Peut-être une combinaison des éléments suivants:
    $FC_NOOVERWRITE (0) = (default) do not overwrite existing files.
    $FC_OVERWRITE (1) = overwrite existing files.
    $FC_CREATEPATH (8) = Create destination directory structure if it doesn't exist (See Remarks).

Constants are defined in FileConstants.au3.

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if source cannot be moved or if dest already exists and flag=0.Retourne 0 si source ne peut pas être déplacé ou si dest existe déjà et flag=0.

Remarks<i0>

See FileFindFirstFile() for a discussion about wildcards.Consultez FileFindFirstFile() pour des précisions sur les caractères spéciaux.

If the source and destination paths are on different volumes a copy and delete operation is performed rather than a move.Si la source et les chemins de destination sont sur des volumes différents une copie puis une suppression est effectuée au lieu d'un déplacement.

Because AutoIt lacks a "FileRename" function, use FileMove() to rename a file!Parce que AutoIt ne possède pas de fonction "FileRename", utiliser FileMove() pour renommer un fichier!

The destination directory must already exist, except using with flag value $FC_CREATEPATH (8).Le répertoire de destination doit exister, sauf si le paramètre "8" est utilisé.
For instance the combined flag $FC_OVERWRITE (1) + $FC_CREATEPATH (8) overwrites the target file and prechecks for the destination directory structure and if it doesn't exist creates it automatically.Par exemple le paramètre combiné $FC_OVERWRITE (1) + $FC_CREATEPATH (8), écrase le fichier cible et vérifie la structure du répertoire de destination et si elle n'existe pas, il la crée automatiquement.

Some file attributes can make the overwriting impossible, if this is the case look at FileSetAttrib() to change the attributes of a file.Certains attributs de fichier peuvent rendre l'écrasement impossible, si c'est le cas consultez FileSetAttrib() pour changer les attributs d'un fichier.

Example 1Exemple 1

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create a temporary file to move.Crée un fichier temporaire qui sera déplacé.
    If Not FileWrite($sFilePath, "This is an example of using FileMove.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Move Au3 files in the temporary directory to a new folder/directory called Au3Files.Déplace les fichiers Au3 du répertoire temporaire vers un nouveau dossier/répertoire appelé Au3Files.
    FileMove(@TempDir & "\*.au3", @TempDir & "\Au3Files\", $FC_OVERWRITE + $FC_CREATEPATH)

    ; Display the temporary directory.Affiche le répertoire temporaire.
    ShellExecute(@TempDir)
EndFunc   ;==>Example

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepaths that will be renamed.Crée une constante de portée locale avec des chemins de fichiers qui seront renommés.
    Local Const $sSource = _WinAPI_GetTempFileName(@TempDir), _
            $sDestination = _WinAPI_GetTempFileName(@TempDir)

    ; Create a temporary file to rename.Crée un fichier temporaire qui sera renommé.
    If Not FileWrite($sSource, "This is an example of using FileMove.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Rename a file using FileMove and overwrite the new file if it exists.Renomme un fichier en utilisant FileMove et remplace le nouveau fichier s'il existe.
    FileMove($sSource, $sDestination, $FC_OVERWRITE)

    ; Display results that the destination file was renamed.Affiche les résultats pour montrer que le fichier de destination a été renommé.
    MsgBox($MB_SYSTEMMODAL, "", "Does the source file exist?: " & FileExists($sSource) & @CRLF & _ ; FileExists should return 0.FileExists doivent toujours retourner 0.
            "Does destination file exist?: " & FileExists($sDestination) & @CRLF) ; FileExists should return 1.FileExists devraient revenir 1.

    ; Delete the temporary files.Supprime les fichiers temporaires. FileDelete checks if the file exists.FileDelete vérifie si le fichier existe.
    FileDelete($sSource)
    FileDelete($sDestination)
EndFunc   ;==>Example
