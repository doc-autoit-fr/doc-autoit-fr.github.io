TCPRecv

Receives data from a connected socket.Reçoit des données d'un socket connecté.

TCPRecv ( mainsocket, maxlen [, flag = 0] )

ParametersParamètres

mainsocket The connected socket identifier (SocketID) as returned by a TCPAccept() or a TCPConnect() function.
maxlen max # of characters to receive.Nombre maximum de caractères à recevoir.
flag [optional][optionnel] Forces the function to return binary data if set to 1 (default is 0, and will auto detect between binary/string).Force la fonction à retourner des données binaire si le paramètre est à 1 (par défaut 0 et détectera automatiquement entre binaire et chaîne).
    $TCP_DATA_DEFAULT (0) - (Default) will auto detect between binary/string
    $TCP_DATA_BINARY (1) - return binary data

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Success:Succès : the binary/string sent by the connected socket.Retourne les données binaire/chaîne envoyées par le socket connecté.
Failure:Échec : "" and sets the @error flag to non-zero.Retourne "" et @error <> 0.
@error:@error : -1 invalid socket.
-2 not connected.
Windows API WSAGetError return value (see MSDN).Valeur retournée par l'API Windows WSAGetError (Consultez MSDN).
@extended: 1 no byte received.

RemarksRemarques

For backwards compatibility reasons this function will try to return strings by default.Pour des raisons de rétro-compatibilité, cette fonction essayera de retourner des chaînes de caractères par défaut. If null (0x00) characters are received then the return value will be a binary type.Si des caractères nul (0x00) sont reçus, alors la valeur retournée sera de type binaire.
To force the function to always return binary data (the most sensible option) then use the "flag" parameter set to 1.Pour forcer la fonction à toujours retourner des données binaires (l'option la plus sensible) utilisez le paramètre "flag" à 1.
If Unicode strings need to be transmitted they must be encoded/decoded with StringToBinary()/BinaryToString().Si des chaînes Unicode doivent être envoyées, elles doivent être encodées/décodées avec StringToBinary()/BinaryToString().

Func Example()
    TCPStartup() ; Start the TCP service.Démarre le service TCP.

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

Func MyTCP_Client($sIPAddress, $iPort)
    ; Assign a Local variable the socket and connect to a listening socket with the IP Address and Port specified.Assigne une variable locale avec le socket et se connecte à un socket d'écoute avec l'adresse IP et le port spécifié.
    Local $iSocket = TCPConnect($sIPAddress, $iPort)
    Local $iError = 0

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        ; The server is probably offline/port is not opened on the server.Le serveur est probablement hors ligne ou le port n'est pas ouvert sur le serveur.
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Client:" & @CRLF & "Could not connect, Error code: " & $iError)
        Return False
    EndIf

    ; Send the string "tata" to the server.Envoie la chaîne "tata" au serveur.
    TCPSend($iSocket, "tata")

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Client:" & @CRLF & "Could not send the data, Error code: " & $iError)
        Return False
    EndIf

    ; Close the socket.Ferme le socket.
    TCPCloseSocket($iSocket)
EndFunc   ;==>MyTCP_Client

Func MyTCP_Server($sIPAddress, $iPort)
    ; Assign a Local variable the socket and bind to the IP Address and Port specified with a maximum of 100 pending connexions.Assigne une variable locale avec le socket et se connecte à l'adresse IP et au port spécifié avec un maximum de 100 connexions en attente.
    Local $iListenSocket = TCPListen($sIPAddress, $iPort, 100)
    Local $iError = 0

    If @error Then
        ; Someone is probably already listening on this IP Address and Port (script already running?).Quelqu'un est probablement déjà à l'écoute sur cette adresse IP et ce port (script déjà en cours d'exécution?).
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Server:" & @CRLF & "Could not listen, Error code: " & $iError)
        Return False
    EndIf

    ; Assign a Local variable to be used by the Client socket.Assigne une variable locale pour être utilisé par le socket client.
    Local $iSocket = 0

    Do ; Wait for someone to connect (Unlimited).Attend que quelqu'un se connecte (illimité).
        ; Accept incomming connexions if present (Socket to close when finished; one socket per client).Accepte des connexions entrantes si présent (Socket fermé une fois terminé, un socket par client).
        $iSocket = TCPAccept($iListenSocket)

        ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
        If @error Then
            $iError = @error
            MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Server:" & @CRLF & "Could not accept the incoming connection, Error code: " & $iError)
            Return False
        EndIf

        If GUIGetMsg() = $GUI_EVENT_CLOSE Then Return False
    Until $iSocket <> -1 ;if different from -1 a client is connected.si il est différent de -1 le client est connecté.

    ; Close the Listening socket to allow afterward binds.Ferme le socket d'écoute pour permettre pour permettre une connexion ultérieure.
    TCPCloseSocket($iListenSocket)

    ; Assign a Local variable the data received.Assigne une variable locale avec les données reçues.
    Local $sReceived = TCPRecv($iSocket, 4) ;we're waiting for the string "tata" OR "toto" (example script TCPRecv):nous attendons la chaîne "tata" OU "toto" (exemple de script TCPRecv): 4 bytes length.4 octets de longueur.

    ; Notes: If you don't know how much length will be the data,Si vous ne savez pas de quelle longueur seront les données,
    ; use e.g:utilisez, par exemple: 2048 for maxlen parameter and call the function until the it returns nothing/error.2048 pour paramètre maxlen et appelez la fonction jusqu'à ce qu'elle ne retourne rien ou erreur.

    ; Display the string received.Affiche la chaîne reçue.
    MsgBox($MB_SYSTEMMODAL, "", "Server:" & @CRLF & "Received: " & $sReceived)

    ; Close the socket.Ferme le socket.
    TCPCloseSocket($iSocket)
EndFunc   ;==>MyTCP_Server

Func OnAutoItExit()
    TCPShutdown() ; Close the TCP service.Ferme le service TCP.
EndFunc   ;==>OnAutoItExit

#include <AutoItConstants.au3>
#include <FileConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    TCPStartup() ; Start the TCP service.Démarre le service TCP.

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

    ; Assign Local variables the loopback IP Address and the Port.Assigne des variables locales avec l'adresse IP et le port.
    Local $sIPAddress = "127.0.0.1" ; This IP Address only works for testing on your own computer.Cette adresse IP ne fonctionne que pour les tests sur votre propre ordinateur.
    Local $iPort = 65432 ; Port used for the connection.Port utilisé pour la connexion.

    ; Assign a Local variable the socket and connect to a listening socket with the IP Address and Port specified.Assigne une variable locale avec le socket et se connecte à un socket d'écoute avec l'adresse IP et le port spécifié.
    Local $iSocket = TCPConnect($sIPAddress, $iPort)
    Local $iError = 0

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        ; The server is probably offline/port is not opened on the server.Le serveur est probablement hors ligne ou le port n'est pas ouvert sur le serveur.
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Client:" & @CRLF & "Could not connect, Error code: " & $iError)
        Return False
    EndIf

    ; Assign a Local variable the path of the file which will be received.Assigne une variable locale avec le chemin du fichier qui sera reçu.
    Local $sFilePath = FileSaveDialog("Save as", @MyDocumentsDir, "All types (*.*)", BitOR($FD_PATHMUSTEXIST, $FD_PROMPTOVERWRITE))

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONEXCLAMATION), "", "Client:" & @CRLF & "Invalid file chosen, Error code: " & $iError)
        Return False
    EndIf

    ; Assign a Local variable the handle of the file opened in binary overwrite mode.Assigne une variable locale avec le handle du fichier ouvert en mode écrasement binaire.
    Local $hFile = FileOpen($sFilePath, BitOR($FO_BINARY, $FO_OVERWRITE))

    ; Assign Locales Constant variables the number representing 4 KiB; the binary code for the end of the file and the length of the binary code.Assigne des constantes locales avec le nombre représentant 4 Ko; le code binaire pour la fin du fichier, et la longueur du code binaire.
    Local Const $i4KiB = 4096, $bEOF = Binary(@CRLF & "{EOF}"), $iEOFLen = BinaryLen($bEOF)

    ; Assign a Local variable the empty binary data which will contain the binary data of the file.Assigne une variable locale avec les données binaires vide qui contiendra les données binaires du fichier.
    Local $bData = Binary("")

    ; Assign a Local variable to store the length of the data received.Assigne une variable locale pour stocker la longueur des données reçues.
    Local $iDataLen = 0

    ; Assign a Local variable a boolean.Assigne une variable locale avec un booléen.
    Local $bEOFReached = False

    Do
        $bData = TCPRecv($iSocket, $i4KiB, $TCP_DATA_BINARY)

        ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
        If @error Then
            $iError = @error
            MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Client:" & @CRLF & "Connection lost, Error code: " & $iError)
            Return False
        EndIf

        $iDataLen = BinaryLen($bData)

        ; If nothing is received, retry for the incoming data.Si rien n'est reçu, recommencez pour les données entrantes.
        If $iDataLen = 0 Then ContinueLoop

        ; If the end of the file is reached.Si la fin du fichier est atteinte.
        If BinaryMid($bData, 1 + $iDataLen - $iEOFLen, $iEOFLen) = $bEOF Then
            ; Strip the EOF code from the file data.Coupe le code de EOF dans le fichier des données.
            $bData = BinaryMid($bData, 1, $iDataLen - $iEOFLen)

            ; Set the EOFReached to True.Définit EOFReached à True.
            $bEOFReached = True
        EndIf

        FileWrite($hFile, $bData)
    Until $bEOFReached

    ; Close the file handle.Ferme le handle du fichier.
    FileClose($hFile)

    ; Display the successful message.Affiche un message de succès.
    MsgBox($MB_SYSTEMMODAL, "", "Client:" & @CRLF & "File received.")

    ; Close the socket.Ferme le socket.
    TCPCloseSocket($iSocket)
EndFunc   ;==>Example
