StringIsLower

Checks if a string contains only lowercase characters.Vérifie si une chaîne contient uniquement des lettres minuscules.

StringIsLower ( "string" )

ParametersParamètre

string The string to checkLa chaîne à traiter

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if string contains non-lowercase characters.Retourne 0 si la chaîne contient des caractères autres que des lettres minuscules.

Remarks<i0>

Note that digits/punctuation/whitespace will cause StringIsLower() to return 0.Notez que les chiffres, la ponctuation et les espaces blancs feront que StringIsLower() retournera 0 comme valeur de retour.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <MsgBoxConstants.au3>

Local $sString = "lowercase" ; Check whether this string contains only lowercase characters.Vérifie si cette chaîne ne contient que des caractères minuscules.
If StringIsLower($sString) Then
    MsgBox($MB_SYSTEMMODAL, "", "The variable is a lowercase string")
Else
    MsgBox($MB_SYSTEMMODAL, "", "The variable is not a lowercase string")
EndIf
