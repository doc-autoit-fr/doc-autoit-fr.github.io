OnAutoItExitRegister

Registers a function to be called when AutoIt exits.Enregistre une fonction qui est appelée quand AutoIt s'arrête.

OnAutoItExitRegister ( "function" )

ParametersParamètres

function The name of the user function to call.Le nom de la fonction utilisateur à appeler.

Return ValueValeur de retour

Success:Succès : 1. @extended can be set if already registered.@extended peut être défini si déjà enregistré.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

The exit code can be retrieved with @exitCode.Le code de sortie peut être récupéré avec @exitCode.
The mode of exit can be retrieved with @exitMethod.Le mode de sortie peut être récupéré avec @exitMethod.

$EXITCLOSE_NORMAL (0) Natural closing.Termine normalement.
$EXITCLOSE_BYEXIT (1) close by Exit function.Termine par la fonction exit.
$EXITCLOSE_BYCLICK (2) close by clicking on exit of the systray.Termine par un clic sur Fermer dans la zone de notification.
$EXITCLOSE_BYLOGOFF (3) close by user logoff.Termine par déconnexion de l'utilisateur.
$EXITCLOSE_BYSUTDOWN (4) close by Windows shutdown.Termine par arrêt de Windows.

Constants are defined in "AutoItConstants.au3".

ExampleExemple

#include <MsgBoxConstants.au3>

OnAutoItExitRegister("MyTestFunc")
OnAutoItExitRegister("MyTestFunc2")

Sleep(1000)

Func MyTestFunc()
    MsgBox($MB_SYSTEMMODAL, "Exit Results 1", 'Exit Message from MyTestFunc()')
EndFunc   ;==>MyTestFunc

Func MyTestFunc2()
    MsgBox($MB_SYSTEMMODAL, "Exit Results 2", 'Exit Message from MyTestFunc()')
EndFunc   ;==>MyTestFunc2
