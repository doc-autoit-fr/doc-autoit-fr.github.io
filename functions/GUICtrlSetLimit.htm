GUICtrlSetLimit

Limits the number of characters/pixels for a control.Limite le nombre de caractères/pixels pour un contrôle.

GUICtrlSetLimit ( controlID, max [, min = 0] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
max For List controls it is the extent you can scroll horizontally in pixels.Pour les contrôles List, max est la limite en pixels du défilement horizontalement.
For Input/Edit controls it is the max number of characters that can be entered.Pour les contrôles Input/Edit, max est le nombre maximum de caractères qui peuvent être entrés.
min [optional][optionnel][optionnel] For Slider and UpDown controls you can specify a min value.Pour les contrôles Slider et UpDown, vous pouvez spécifier une valeur minimale. Default = 0Par défaut = 0

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0.Retourne 0.

RelatedLiens

GUICtrlCreateEdit, GUICtrlCreateInput, GUICtrlCreateList, GUICtrlCreateSlider, GUICtrlCreateUpdown

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI limit input 3 chars") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUICtrlCreateInput("", 10, 20)
    GUICtrlSetLimit(-1, 3) ; to limit the entry to 3 charslimite l'entrée à 3 caractères

    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
