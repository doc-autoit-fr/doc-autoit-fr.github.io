GUICtrlSetStyle

Changes the style of a control.Change le style d'un contrôle.

GUICtrlSetStyle ( controlID, style [, exStyle] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
style Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
exStyle [optional][optionnel] Defines the extended Style of the control.Définit le style étendu d'un contrôle. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

Some styles cannot be changed dynamically, check MSDN documentation.Certains styles ne peuvent pas être modifiés dynamiquement, consultez la documentation MSDN. $CBS_UPPERCASE combo style is one example.Le style $CBS_UPPERCASE d'un contrôle Combo est un exemple.

Example()

Func Example()
    GUICreate("My GUI style") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUICtrlCreateLabel("my label which will split on several lines", 10, 20, 100, 100)
    GUICtrlSetStyle(-1, $SS_RIGHT)

    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
