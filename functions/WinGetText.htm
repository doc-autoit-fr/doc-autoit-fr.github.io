WinGetText

Retrieves the text from a window.Récupère le texte d'une fenêtre.

WinGetText ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to get the text.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the window to get the text.Le texte de la fenêtre à traiter. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : a string containing the window text read.Retourne une chaine de caractères contenant le texte lu dans la fenêtre.
Failure:Échec : "" if the window is not found.Retourne "" si la fenêtre n'est pas trouvée.

RemarksRemarques

Up to 64KB of window text can be retrieved.Jusqu'à 64 Ko de texte peut être récupéré. WinGetText() works on minimized windows, but only works on hidden windows if you've set Opt("WinDetectHiddenText", 1)
If multiple windows match the criteria for WinGetText(), the information for the most recently active match is returned.Si plusieurs fenêtres correspondent aux critères pour WinGetText(), les informations de la fenêtre la plus récente seront retournés.
Use WinGetText("[active]") to get the active window's text.Utilisez WinGetText("[active]") pour obtenir un texte de la fenêtre active.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Retrieve the window text of the active window.Récupére le texte de la fenêtre de la fenêtre active.
    Local $sText = WinGetText("[ACTIVE]")

    ; Display the window text.Affiche le texte de la fenêtre.
    MsgBox($MB_SYSTEMMODAL, "", $sText)
EndFunc   ;==>Example
