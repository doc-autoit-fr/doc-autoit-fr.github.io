DriveSpaceFree

Returns the free disk space of a path in Megabytes.Retourne l'espace disque libre d'un lecteur en Méga-octet.

DriveSpaceFree ( "path" )

ParametersParamètres

path Path of drive to receive information from.Le chemin sur lequel agir.

Return ValueValeur de retour

Success:Succès : the free disk space in Megabytes as a float number.Retourne l'espace libre du lecteur en Méga-octets comme nombre flottant.
Failure:Échec : 0 and sets the @error flag to 1.Retourne 0 et @error = 1.

RemarksRemarques

DriveSpaceFree() may even work when a complete directory path (that exists) is given.DriveSpaceFree() peut fonctionner même si le chemin donné est celui d'un répertoire existant sur le lecteur. However, a file path won't work.Un chemin vers un fichier ne fonctionnera pas.
Use the Round() function if the return value is too precise.Utilisez la fonction Round() pour obtenir une valeur arrondie.

ExampleExemple

#include <MsgBoxConstants.au3>

Local $iFreeSpace = DriveSpaceFree(@HomeDrive & "\") ; Find the free disk space of the home drive, generally this is the C:\ drive.Trouve l'espace libre sur le disque du lecteur principal, en général c'est le lecteur C:\
MsgBox($MB_SYSTEMMODAL, "", "Free Space: " & $iFreeSpace & " MB")
