TrayItemSetText

Sets the itemtext of a tray menu/item control.Définit le texte d'un contrôle menu ou item de menu de la zone de notification.

TrayItemSetText ( controlID, text )

ParametersParamètre

controlID The control identifier (controlID) as returned by a TrayCreateItem or TrayCreateMenu function.L'identifiant du contrôle (controlID) tel que retourné par la fonction TrayCreateItem ou TrayCreateMenu fonction.
text The new text of the tray menu/item control.Le nouveau texte du contrôle menu ou item de menu de la zone de notification.

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0.Retourne 0.

Remarks<i0>

To change the itemtext of the default menuitems (Script Paused/Exit) use the constants $TRAY_ITEM_EXIT and $TRAY_ITEM_PAUSE as controlID.Pour changer le texte des items de menu par défaut (script en Pause ou Exit), utilisez les constantes $TRAY_ITEM_EXIT et $TRAY_ITEM_PAUSE comme controlID.

The above constants are defined in #include <TrayConstants.au3>.

#NoTrayIcon
#include <MsgBoxConstants.au3>
#include <TrayConstants.au3> ; Required for the $TRAY_ICONSTATE_SHOW, $TRAY_ITEM_EXIT and $TRAY_ITEM_PAUSE constants.Nécessaire pour les constantes $TRAY_ICONSTATE_SHOW, $TRAY_ITEM_EXIT et $TRAY_ITEM_PAUSE.

Opt("TrayAutoPause", 0) ; The script will not pause when selecting the tray icon.Le script ne sera pas une pause lors de la sélection de l'icône de la zone de notification.
Opt("TrayMenuMode", 2) ; Items are not checked when selected.Les items ne sont pas cochés lorsqu'ils sont sélectionnés.

Example()

Func Example()
    Local $idRandom = TrayCreateItem("Random:") ; Select this item to change the text with a random number.Sélectionne cette option pour modifier le texte avec un nombre aléatoire.
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idAbout = TrayCreateItem("About")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    TrayItemSetText($TRAY_ITEM_EXIT, "Exit Program") ; Set the text of the default 'Exit' item.Définit le texte de l'item par défaut 'Exit'.
    TrayItemSetText($TRAY_ITEM_PAUSE, "Pause Program") ; Set the text of the default 'Pause' item.Définit le texte de l'item par défaut 'Pause'.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

            Case $idRandom
                ; Set the text of the 'Random' item with a random integer.Définit le texte de l'item 'Random' avec un nombre aléatoire.
                TrayItemSetText($idRandom, "Random: " & Int(Random(1, 10, 1)))

        EndSwitch
    WEnd
EndFunc   ;==>Example
