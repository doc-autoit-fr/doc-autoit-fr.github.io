WinSetTrans

Sets the transparency of a window.Définit la transparence d'une fenêtre.

WinSetTrans ( "title", "text", transparency )

ParametersParamètres

title The title/hWnd/class of the window to change.Le titre/hWnd/class de la fenêtre à changer. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to change.Le texte de la fenêtre à changer. See Text special definition.Consultez Définition spéciale de Texte .
transparency A number in the range 0 - 255.Un nombre entre 0 et 255. The lower the number, the more transparent the window will become.Plus le nombre est petit, plus la fenêtre sera transparente. 255 = Solid, 0 = Invisible.255 = Normal, 0 = Invisible.

Return ValueValeur de retour

Success:Succès : Non-zero.Retourne une valeur différente de zéro.
Failure:Échec : 0 and sets the @error flag to non-zero if no supported on the OS.Retourne 0 et @error <> 0 si la fonction n'est pas supportée par l'OS.

RemarksRemarques

Screen color must be greater or equal to 16-bit.La couleur de l'écran doit être égale ou supérieure à 16-bit.

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Set the transparency of the Notepad window to 170.Définit la transparence de la fenêtre du Bloc-notes à 170.
    WinSetTrans($hWnd, "", 170)

    ; Wait for 2 seconds to display the Notepad window.Attend 2 secondes pour afficher la fenêtre du Bloc-notes.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
