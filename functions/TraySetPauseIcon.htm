TraySetPauseIcon

Loads/Sets a specified tray pause icon.Charge/Définit l'icône 'pause' spécifiée pour la zone de notification.

TraySetPauseIcon ( [filename [, iconID]] )

ParametersParamètres

filename [optional][optionnel] The filename of the icon to be display as pause icon in the tray.Le nom de l'icône à afficher dans la zone de notification.
iconID [optional][optionnel] Icon identifier if the file contains multiple icons.Identifiant de l'icône si le fichier contient plusieurs icônes.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

To reset the pause icon to the default (red cross), use the function with no parameters TraySetPauseIcon().

Passing a positive number will reference the string equivalent icon name.Passer un nombre positif, fera référence à l'équivalent chaîne du nom de l'icône.
Passing a negative number causes 1-based "index" behaviour.Passer un nombre négatif, provoque un comportement d'"index" basé sur 1. Some Dll can have icon extracted just with negative numbers.Les icônes de certaines Dll ne peuvent être extraites seulement qu'avec des nombres négatifs.

RelatedLiens

TraySetIcon, TraySetState

ExampleExemple

#NoTrayIcon
#include <TrayConstants.au3> ; Required for the $TRAY_ICONSTATE_SHOW constant.Nécessaire pour la constante $TRAY_ICONSTATE_SHOW.

Example()

Func Example()
    TraySetPauseIcon("shell32.dll", 12) ; Set the pause icon.Définit l'icône pause. This will flash on and off when the tray menu is selected and the script is paused.Elle clignotera lorsque le menu de la barre est sélectionné et le script sera en pause.
    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Sleep(100) ; An idle loop.Une boucle d'inactivité.
    WEnd
EndFunc   ;==>Example
