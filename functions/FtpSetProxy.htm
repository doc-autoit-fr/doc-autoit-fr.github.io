FtpSetProxy

Sets the internet proxy to use for ftp access.Configure le proxy internet pour l'accès FTP.

FtpSetProxy ( mode = 0 [, "proxy:port" [, "username" [, "password"]]] )

ParametersParamètres

mode The proxy mode to use:Le mode de proxy à utiliser :
    $PROXY_IE (0) = (default) Use current Internet Explorer settings for proxy.
    $PROXY_NONE (1) = Use no proxy (direct access)
    $PROXY_SPECIFIED (2) = Use the proxy specified

Constants are deined in "AutoItConstants.au3".
proxy:port [optional][optionnel][optionnel] The address and port of the proxy to use.L'adresse et le port du proxy à utiliser. See remarks below.Voir remarques ci-dessous.
username [optional][optionnel][optionnel] If required, the username for the proxySi nécessaire, le nom d'utilisateur pour le proxy.
password [optional][optionnel][optionnel] If required, the password for the proxySi nécessaire, le mot de passe pour le proxy.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

Internet Explorer 3 or greater must be installed for this function to work.Internet Explorer 3 ou plus récent doit être installé pour faire fonctionner la fonction.

Only CERN proxy servers are supported.Seuls les serveurs proxy CERN (W3C) sont pris en charge.

Proxy servers are usually made up of a web address and port number.Les serveurs proxy sont habituellement définis par une adresse web et un numéro de port. Most ISPs use the port number of 8080.La plupart des prestataires utilisent le port 8080. If your proxy was "www.proxy.com" and port was "8080" then you would use this to set the proxy:Si votre proxy est "www.proxy.com" et le port est 8080 alors, vous utiliserez les paramètres suivants:
FtpSetProxy(2, "www.proxy.com:8080")

Most proxy servers do not require a username and password.La plupart des serveurs proxy ne nécessites ni nom d'utilisateur ni mot de passe.

When using mode 0 you may specify a username and password and these will be used with the Internet Explorer configured proxy.En utilisant le mode 0, vous pouvez spécifier un nom d'utilisateur et un mot de passe qui seront utilisés avec le proxy configuré dans Internet Explorer.

ExampleExemple

; Use no proxyNe pas utiliser de proxy
FtpSetProxy(1)

; Use IE defaults for proxyUtilisez les valeurs par défaut d'IE pour proxy
FtpSetProxy(0)

; Use the proxy "www-cache.myisp.net" on port 8080Utilise le proxy "www-cache.myisp.net" sur le port 8080
FtpSetProxy(2, "www-cache.myisp.net:8080")
