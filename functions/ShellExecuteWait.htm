ShellExecuteWait

Runs an external program using the ShellExecute API and pauses script execution until it finishes.Exécute un programme externe en utilisant l'API ShellExecute et interrompt l'exécution du script jusqu'à ce qu'il se termine.

ShellExecuteWait ( "filename" [, "parameters" [, "workingdir" [, "verb" [, showflag]]]] )

ParametersParamètres

filename The name of the file to run (EXE, .txt, .lnk, etc).Le nom du fichier à exécuter (EXE, .txt, .lnk, etc..).
parameters [optional][optionnel] Any parameters for the program.Les paramètres du programme. Blank ("") uses none.Vide ("") si aucun.
workingdir [optional][optionnel] The working directory.Le répertoire de travail. Blank ("") uses the current working directory.Si vide ("") utilise le répertoire de travail courant.
verb [optional][optionnel] The "verb" to use, common verbs include:Le "verbe" à utiliser, les verbes courants comprennent :
    $SHEX_OPEN ("open") = Opens the file specified. The file can be an executable file, a document file, or a folderLe fichier peut être un fichier exécutable, un fichier de document ou un dossier
    $SHEX_EDIT ("edit") = Launches an editor and opens the document for editing. If "filename" is not a document file, the function will failSi "filename" n'est pas un fichier document, la fonction échouera
    $SHEX_PRINT ("print") = Prints the document file specified. If "filename" is not a document file, the function will failSi "filename" n'est pas un fichier document, la fonction échouera
    $SHEX_PROPERTIES ("properties") = Displays the file or folder's properties
See remarks for more information on the default behavior when a verb is not specified.Voir remarques pour plus d'information sur le comportement par défaut quand le verbe n'est pas spécifié.

Constants are defined in "AutoItConstants.au3".
showflag [optional][optionnel] The "show" flag of the executed program:L'option de visualisation du programme exécuté:
    @SW_HIDE = Hidden window
    @SW_MINIMIZE = Minimized window
    @SW_MAXIMIZE = Maximized window

Return ValueValeur de retour

Success:Succès : the exit code of the program that was run.Retourne le code de sortie du programme qui a été exécuté.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

After running the requested program the script pauses until the requested program terminatesAprès avoir exécuté le programme demandé le script s'interrompt jusqu'à ce que le programme demandé termine

When no verb is specified the default verb is used.Lorsque aucun verbe n'est spécifié, un verbe est utilisé par défaut. The default verb is the verb configured in the registry.Le verbe par défaut est le verbe configuré dans le Registre. If no verb is set as default in the registry then the "open" verb is used.Si aucun verbe est défini par défaut dans le Registre, alors le verbe "open" est utilisé. If the "open" verb is not present then the first verb listed in the registry is used.Si le verbe "open" n'est pas présent, alors le premier verbe inscrit au registre est utilisé.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Execute Notepad and wait for the Notepad process to close.Exécute le Bloc-notes et attend que le processus du Bloc-notes se termine.
    Local $iReturn = ShellExecuteWait("notepad.exe")

    ; Display the return code of the Notepad process.Affiche le code de retour du processus du Bloc-notes.
    MsgBox($MB_SYSTEMMODAL, "", "The return code from Notepad was: " & $iReturn)
EndFunc   ;==>Example
