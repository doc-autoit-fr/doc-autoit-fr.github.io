GUICreate

Create a GUI window.Crée une fenêtre GUI.

GUICreate ( "title" [, width [, height [, left = -1 [, top = -1 [, style = -1 [, exStyle = -1 [, parent = 0]]]]]]] )

ParametersParamètres

title The title of the dialog box.Le titre de la boîte de dialogue.
width [optional][optionnel] The width of the window.La largeur de la fenêtre.
height [optional][optionnel] The height of the window.La hauteur de la fenêtre.
left [optional][optionnel] The left side of the dialog box.La position du côté gauche de la boîte de dialogue. By default (-1), the window is centered.Par défaut (-1), la fenêtre est centrée. If defined, top must also be defined.Si défini, la position haute doit également être définie.
top [optional][optionnel] The top of the dialog box.La position haute de la boîte de dialogue. Default (-1) is centeredPar défaut (-1), la fenêtre est centrée.
style [optional][optionnel] defines the style of the window.Définit le style de la fenêtre. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
Use -1 for the default style which includes a combination of $WS_MINIMIZEBOX, $WS_CAPTION, $WS_POPUP, $WS_SYSMENU styles.Utilisez -1 pour le style par défaut, qui inclut une combinaison des styles $WS_MINIMIZEBOX, $WS_CAPTION, $WS_POPUP et $WS_SYSMENU.
Some styles are always included:Certains styles sont toujours inclus: $WS_CLIPSIBLINGS, and $WS_SYSMENU if $WS_MAXIMIZEBOX or $WS_SIZEBOX is specified.$WS_CLIPSIBLINGS et $WS_SYSMENU si $WS_MAXIMIZEBOX ou $WS_SIZEBOX sont spécifiés.
exStyle [optional][optionnel] defines the extended style of the window.Définit le style étendu de la fenêtre. See the Extended Style Table below.Consultez la Table des Styles Étendus ci-dessous.
Use -1 for the default, which is no extended styles.Utilisez -1 pour valeur par défaut, ce qui correspond à aucun style étendu.
Forced styles:Styles forcés: $WS_EX_WINDOWEDGE
parent [optional][optionnel] The handle of another previously created window - this new window then becomes a child of that window.Le handle d'une fenêtre précédemment créé - La nouvelle fenêtre devient alors un enfant de cette fenêtre.

Return ValueValeur de retour

Success:Succès : a windows handle.Retourne le handle de la fenêtre.
Failure:Échec : 0 if the window cannot be created and sets the @error flag to 1.Retourne 0 si la fenêtre ne peut pas être créé et @error = 1.

RemarksRemarques

By default the dialog box is non sizable and non maximizable.Par défaut la taille de la boîte de dialogue ne peut pas être modifiée ni maximisée. So WS_SIZEBOX or WS_MAXIMIZEBOX can be used in the style parameter.WS_SIZEBOX ou WS_MAXIMIZEBOX peuvent être utilisés dans le paramètre de style.
As defining only one style just set this style don't forget to combine it with default ones, i.e. just defining WS_SIZEBOX will not set $WS_MINIMIZEBOX, $WS_CAPTION, $WS_POPUP, $WS_SYSMENU.La définition d'un seul style n'applique que le style définit, donc n'oubliez pas de le combiner avec les styles par défaut.Si par exemple vous ne définissez que WS_SIZEBOX, alors WS_MINIMIZEBOX, WS_CAPTION, WS_POPUP et WS_SYSMENU ne seront pas utilisés. So the best method to define a resizeable window is to use $WS_OVERLAPPEDWINDOW.Donc, la meilleure méthode pour définir une fenêtre redimensionnable est d'utiliser WS_OVERLAPPEDWINDOW.
When using $WS_EX_MDICHILD the position is relative to client area of the parent window.Lorsque vous utilisez $WS_EX_MDICHILD, la position est relative à la zone client de la fenêtre parent. With $WS_EX_LAYERED it is possible to have a transparent picture on a background picture defined in the parent window.Avec $WS_EX_LAYERED il est possible d'avoir une image transparente sur une image de fond défini dans la fenêtre parent.
Adding $WS_CLIPCHILDREN style can avoid some flickering when resizing GUI containing Edit control for example.L'ajout du style $WS_CLIPCHILDREN peut éviter un certain scintillement lors du redimensionnement d'une GUI avec par exemple un contrôle Edit.
You can enable window dragging for GUI without $WS_CAPTION by using $WS_EX_CONTROLPARENT in the exStyle parameter.Vous pouvez activer le déplacement d'une fenêtre sans $WS_CAPTION en utilisant $WS_EX_CONTROLPARENT dans les paramètres étendus (exStyle).

To combine styles with the default style use BitOR($GUI_SS_DEFAULT_GUI, newstyle, ...).Pour combiner les styles avec les styles par défaut, utilisez BitOR($GUI_SS_DEFAULT_GUI, newstyle, ...).
The size specified is the size of the client area of the window.La taille spécifiée est la taille de la zone cliente de la fenêtre. The border and title bar will make the window slightly larger than specified.La bordure et la barre de titre feront que la fenêtre sera légèrement plus grande que la taille spécifiée. Using menu controls will also change the windows height.L'utilisation de menu de commandes changera aussi la hauteur des fenêtres.

$WS_EX_TOPMOST Specifies that a window created with this style should be placed above all non-topmost windows and should stay above them, even when the window is deactivated.Indique que la fenêtre créée avec ce style doit être placée par-dessus toutes les fenêtres qui ne sont pas d'avant-plan et devrait rester par-dessus, même lorsque la fenêtre est désactivée.
$WS_EX_TRANSPARENT The window appears transparent because the bits of underlying sibling windows have already been painted.La fenêtre apparaît transparente car les bits des fenêtres enfant sous-jacentes de même parent ont déjà été peints.
$WS_EX_TOOLWINDOW Creates a tool window; that is, a window intended to be used as a floating toolbar.Crée une fenêtre outil, qui est une fenêtre destinée à être utilisée comme une barre d'outils flottante. A tool window has a title bar that is shorter than a normal title bar, and the window title is drawn using a smaller font.Une fenêtre outil a une barre de titre qui est plus petite que la barre de titre normale, et dont la police du titre est plus petite. A tool window does not appear in the taskbar or in the dialog box that appears when the user presses ALT+TAB.Une fenêtre outil n'apparaît pas dans la barre des tâches ou dans la boîte de dialogue qui s'affiche lorsque l'utilisateur appuie sur ALT+TAB. If a tool window has a system menu, its icon is not displayed on the title bar.Si une fenêtre outil dispose d'un menu système, son icône n'est pas affichée dans la barre de titre. However, you can display the system menu by typing ALT+SPACE.Cependant, vous pouvez afficher le menu système en tapant ALT+ESPACE.
$WS_EX_WINDOWEDGE Specifies that a window has a border with a raised edge.Indique que la fenêtre a une bordure en relief.
$WS_EX_LAYERED Creates a layered window.Crée une fenêtre de superposition (alphablend). Note that this cannot be used for child windows.Notez que cela ne peut pas être utilisé pour les fenêtres enfants.

To use the values specified above you must #include <WindowsConstants.au3> in your script.

Note:Note : The handle returned from this function is a real windows handle, which means it can be used in the same way as the result of WinGetHandle().Le handle renvoyé par cette fonction est un véritable handle de fenêtres, ce qui signifie qu'il peut être utilisé de la même manière que le résultat de WinGetHandle().
WinMove() use before GuiSetState() to change GUI window size will not change the position/resizing of defined Controls.

ExampleExemple

Example 1Exemple 1

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Create a GUI with various controls.Crée une GUI avec différents contrôles.
    Local $hGUI = GUICreate("ExampleExemple")
    Local $idOK = GUICtrlCreateButton("OK", 310, 370, 85, 25)

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idOK
                ExitLoop

        EndSwitch
    WEnd

    ; Delete the previous GUI and all controls.Supprime la GUI précédente et tous ses contrôles.
    GUIDelete($hGUI)
EndFunc   ;==>Example

#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Example()

Func Example()
    Local $sFilePath = "..\GUI\logo4.gif"

    ; Create a GUI with various controls.Crée une GUI avec différents contrôles.
    Local $hGUI = GUICreate("ExampleExemple", 400, 100)
    GUICtrlCreatePic("..\GUI\msoobe.jpg", 0, 0, 400, 100)

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW, $hGUI)

    Local $hChild = GUICreate("", 169, 68, 20, 20, $WS_POPUP, BitOR($WS_EX_LAYERED, $WS_EX_MDICHILD), $hGUI)

    ; Create a picture control with a transparent image.Crée un contrôle d'image avec une image transparente.
    GUICtrlCreatePic($sFilePath, 0, 0, 169, 68)

    ; Display the child GUI.Affiche l'interface graphique enfant.
    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    ; Delete the previous GUIs and all controls.Supprime les GUI précédentes et tous les contrôles.
    GUIDelete($hGUI)
    GUIDelete($hChild)
EndFunc   ;==>Example
