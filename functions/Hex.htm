Hex

Returns a string representation of an integer or of a binary type converted to hexadecimal.Retourne une chaîne de caractères représentant un nombre entier ou binaire converti en hexadécimal.

Hex ( expression [, length] )

ParametersParamètres

expression The expression to convert.L'expression à convertir.
length [optional][optionnel] Number of characters to be returned for integer.Le nombre de caractères à retourner pour des nombres entiers.
Characters are truncated from the left-hand side if length is too small.Les caractères sont tronqués à partir de la gauche si la longueur est trop petite.
This parameter is ignored if the data is binary data.Ce paramètre est ignoré pour les données binaires.

Return ValueValeur de retour

Success:Succès : a string of length (never more than 16) characters, zero-padded if necessary for integer.Retourne une chaîne de longueur (au plus 16) caractères, complété de zéro si nécessaire pour les nombres entiers. Returns the binary type converted.Retourne le type binaire convertit.
Special: "" (empty string) if length is less than 1.Retourne "" (chaîne vide) si la longueur est inférieure à 1.

RemarksRemarques

Omitting the second parameter (length) or using the keyword Default results in automatic sizing of the output; 8-characters display for numbers that can fit in that and 16-characters display for others.En omettant le second paramètre (length)ou en utilisant le mot-clé Default, le résultat est automatiquement calibré; 8 caractères s'affichent pour les nombres qui peuvent y tenir et ...16 caractères s'affichent pour les autres.
A pointer type parameter is displayed in size of pointer by default (8 characters for x86 and 16 characters for x64).Un paramètre de type pointeur est affiché en taille de pointeur par défaut (8 caractères pour x86 et 16 caractères pour x64).
64bit integers have 16 characters displayed by default.Les entiers 64 bits s'affichent sur 16 caractères par défaut.
Numbers passed as non-integers (those with decimal separator or exponent) are processed as doubles.Les nombres non entiers passés (ceux avec un séparateur décimal ou un exposant) sont traitées comme doubles.

ExampleExemple

#include <MsgBoxConstants.au3>

; Assign a Local variable the hex number representation of 1033.Assigne une variable locale avec la représentation hexadécimal du nombre 1033.

; Display the result.Affiche le résultat.
