BinaryMid

Extracts a number of bytes from a binary variant.Extrait un nombre d'octets à partir d'une variable de type 'variant' binaire.

BinaryMid ( binary, start [, count] )

ParametersParamètre

binary The binary data to evaluate.Les données binaires à évaluer.
start The byte position to start.La position de l'octet où commencer l'extraction. (1 = first byte)(1 = premier octet)
count [optional][optional] False - i, v, x ... The number of bytes to extract.Le nombre d'octets à extraire. By default the entire remainder of the binary data.Par défaut tout le reste des données binaires.

Return ValueRemarque

Returns the extracted binary data.Retourne les données binaires extraites.

Remarks<i0>

If start is out-of-bounds, an empty binary variant is returned.Si start est en dehors des limites, une variable binaire vide est retournée. If start is valid but count is out-of-bounds, the entire remainder of the binary data is returned.Si start est valide mais count est en dehors des limites, tout le reste des données binaires est retourné.
