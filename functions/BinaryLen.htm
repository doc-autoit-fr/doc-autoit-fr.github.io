BinaryLen

Returns the number of bytes in a binary variant.Retourne le nombre d'octets d'un variable de type 'variant' binaire.

BinaryLen ( binary )

ParametersParamètres

binary The binary data to evaluate.Les données binaires à évaluer.

Return ValueValeur de retour
