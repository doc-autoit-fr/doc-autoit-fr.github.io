VarGetType

Returns the internal type representation of a variant.Retourne le type de représentation interne d'un variant.

VarGetType ( expression )

ParametersParamètres

expression An expression to check the internal type of.Une expression dont on veut analyser le type interne.

Return ValueValeur de retour

Returns a string representing the type of the expression.Retourne une chaîne représentant le type d'expression.

RemarksRemarques

See language datatypes for a detailed description.Consultez Types de données du langage pour une description détaillée.

IsInt() can return different results as it tries to convert from a float, for example calling VarGetType() with a value of 2.0 will return "Double", but IsInt() will return 1 for the same value.IsInt() peut retourner des résultats différents puisqu'il essaie de convertir depuis un float, par exemple appeler VarGetType() avec une valeur de 2.0 retournera "Double", mais IsInt() retournera 1 pour la même valeur.
