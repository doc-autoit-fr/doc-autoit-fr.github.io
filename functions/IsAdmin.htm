IsAdmin

Checks if the current user has full administrator privileges.Vérifie si l'utilisateur a des droits d'administrateur complets.

IsAdmin ( )

Return ValueValeur de retour

Success:Succès : 1 if the current user has administrator privileges.Retourne 1 si l'utilisateur courant a des droits d'administrateur.
Failure:Échec : 0 if user lacks admin privileges.Retourne 0 si l'utilisateur n'a pas les droits d'administrateur.

RemarksRemarques

It returns 1 under Windows Vista only if running with a full administrator token (i.e. #RequireAdmin has been used, or has already been elevated by UAC).

ExampleExemple

#include <MsgBoxConstants.au3>

If IsAdmin() Then
    MsgBox($MB_SYSTEMMODAL, "", "IsAdmin" & @CRLF & "Admin rights are detected.")
EndIf
