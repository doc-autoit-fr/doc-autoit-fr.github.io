ASin

Calculates the arcsine of a number.Calcule l’arc-sinus d'un nombre.

ASin ( expression )

ParametersParamètres

expression Any value between -1 and 1 (inclusive).Toute valeur comprise entre -1 et 1 (inclus).

Return ValueValeur de retour

Returns the trigonometric arcsine of the number.Retourne l’arc-sinus trigonométrique du nombre. Result is in radians.Le résultat est exprimé en radians.

RemarksRemarques

ASin(x) is mathematically defined only for -1 < x < 1, so ASin() tends to return -1.#IND for other values of x.ASin(x) est défini mathématiquement pour -1 < x < 1, aussi ASin retourne -1.#IND (indéterminé) pour toutes les autres valeurs de x.

Example 1Exemple 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local variable the arcSine of 0.5.Assigne une variable locale avec l'arc sinus de 0,5.
    Local $fArcSin1 = ASin(0.5)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fArcSin1 & " rad.")

    ; Assign a Local constant variable the approximate PI number.Assigne une constante locale avec une approximation du nombre de PI.
    Local Const $PI = 3.141592653589793

    ; Assign a Local variable the formula to switch from radian to degree (equals to one radian in degree).Assigne une variable locale avec le facteur de conversion de radian en degré (égal à un radian en degré).
    Local $fRadToDeg = 180 / $PI

    ; Assign a Local variable a number in degree.Assigne un nombre en degrés à une variable locale.
    Local $fArcSin2 = $fRadToDeg * ASin(1)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fArcSin2 & " deg.")
EndFunc   ;==>Example
