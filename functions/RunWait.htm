RunWait

Runs an external program and pauses script execution until the program finishes.Exécute un programme externe et interrompt l'exécution du script jusqu'à ce que le programme se termine.

RunWait ( "program" [, "workingdir" [, show_flag [, opt_flag]]] )

ParametersParamètres

program The full path of the program (EXE, BAT, COM, or PIF) to run (see remarks).Le chemin d'accès complet du programme (EXE, BAT, COM, ou PIF) pour exécuter (Voir Remarques).
workingdir [optional][optionnel][optionnel] The working directory.Le répertoire de travail. Blank ("") uses the current working directory.Si vide ("") utilise le répertoire de travail courant. This is not the path to the program.Ce n'est pas le chemin du programme.
show_flag [optional][optionnel][optionnel] The "show" flag of the executed program:L'option de visualisation du programme exécuté:
    @SW_HIDE = Hidden window (or Default keyword)
    @SW_MINIMIZE = Minimized window
    @SW_MAXIMIZE = Maximized window
opt_flag [optional][optionnel][optionnel] Controls various options related to how the parent and child process interact.Options diverses liées à la façon dont les processus parent-enfant inter-agissent.
    0x10000 ($RUN_CREATE_NEW_CONSOLE) = The child console process should be created with its own window instead of using the parents window. This flag is only useful when the parent is compiled as a Console application.Ce drapeau n'est utile que lorsque le parent est compilé comme une application console.

Return ValueValeur de retour

Success:Succès : the exit code of the program that was run.Retourne le code de sortie du programme qui a été exécuté.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

Paths with spaces need to be enclosed in quotation marks.Les chemins avec des espaces doivent être placés entre guillemets.

To run DOS commands, try RunWait(@ComSpec & " /c " & "commandName")    ; don't forget " " before "/c"

After running the requested program the script pauses until the program terminates.Après avoir exécuté le programme demandé le script se met en pause jusqu'à la fin du programme. To run a program and then immediately continue script execution use the Run() function instead.Pour exécuter un programme puis continuer immédiatement l'exécution de script utilisez la fonction Run() à la place.

Some programs will appear to return immediately even though they are still running; these programs spawn another process - you may be able to use the ProcessWaitClose() function to handle these cases.Certains programmes semble retourner immédiatement, même si elles sont encore en cours d'exécution; ces programmes engendrent un autre processus - vous pourriez être en mesure d'utiliser la fonction ProcessWaitClose() pour traiter ces cas.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run Notepad and wait for the Notepad process to close.Lance le Bloc-notes et attend que le processus du Bloc-notes se ferme.
    Local $iReturn = RunWait("notepad.exe")

    ; Display the return code of the Notepad process.Affiche le code de retour du processus du Bloc-notes.
    MsgBox($MB_SYSTEMMODAL, "", "The return code from Notepad was: " & $iReturn)
EndFunc   ;==>Example
