Référence Fonction

FileClose

Closes a previously opened file.Ferme un fichier texte précédemment ouvert.

FileClose ( "filehandle" )

ParametersParamètres

filehandle The handle of a file, as returned by a previous call to FileOpen().Le handle d'un fichier, renvoyé par l'utilisation de la commande FileOpen().

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if the filehandle is invalid.Retourne 0 si le handle du fichier n'est pas valide.

RemarksRemarques

Upon termination, AutoIt automatically closes any files it opened, but calling FileClose() is still a good idea.A la fermeture du script, AutoIt ferme automatiquement tous les fichiers qu'il a ouvert, mais appeler FileClose() est néanmoins une bonne idée.

This function is also used to close search handles as returned by FileFindFirstFile().Cette fonction est aussi utilisée pour fermer les handles de recherche renvoyés par FileFindFirstFile().

RelatedLiens

FileFindFirstFile, FileFindNextFile, FileFlush, FileOpen

ExampleExemple

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create a temporary file to read data from.Crée un fichier temporaire pour y lire les données.
    If Not FileWrite($sFilePath, "This is an example of using FileClose.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Open the file for reading and store the handle to a variable.Ouvre le fichier pour la lecture et stocke le handle dans une variable.
    Local $hFileOpen = FileOpen($sFilePath, $FO_READ)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred when reading the file.")
        Return False
    EndIf

    ; Read the contents of the file using the handle returned by FileOpen.Lire le contenu du fichier à l'aide du handle retournée par FileOpen.
    Local $sFileRead = FileRead($hFileOpen)

    ; Close the handle returned by FileOpen.Ferme le handle renvoyé par FileOpen.
    FileClose($hFileOpen)

    ; Display the contents of the file.Affiche le contenu du fichier.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & $sFileRead)

    ; Delete the temporary file.Supprimez le fichier temporaire.
    FileDelete($sFilePath)
EndFunc   ;==>Example
