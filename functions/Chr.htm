Chr

Returns a character corresponding to an ASCII code.Retourne un caractère correspondant à un code ASCII.

Chr ( ASCIIcode )

ParametersParamètres

ASCIIcode An ASCII code in the range 0-255 (e.g., 65 returns the capital letter A).

Return ValueValeur de retour

Success:Succès : a string containing the ASCII representation of the given code.Retourne une chaîne contenant la représentation ASCII du code donné.
Failure:Échec : an empty string and sets the @error flag to 1 if the ASCIIcode is greater than 255.Retourne une chaîne vide et @error = 1 si le code ASCII est supérieur à 255.

RemarksRemarques

See the ASCII Character Code table for a complete list of available values.Consultez la table des Codes de caractères ASCII pour une liste complète des valeurs possibles.

Chr(48) == "0", Chr(57) == "9", Chr(65) == "A", Chr(90) == "Z", Chr(97) == "a", Chr(122) == "z", etc.

RelatedLiens

Asc, AscW, ChrW, String

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sText = ""
For $i = 65 To 90
    $sText = $sText & Chr($i) ; Or $sText &= Chr($i) can be used as well.Ou $sText &= Chr($i) peut être utilisé.
Next
MsgBox($MB_SYSTEMMODAL, "Uppercase alphabet", $sText) ; Display the characters between 65 to 90.Affiche les caractères entre 65 à 90.
