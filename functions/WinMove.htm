WinMove

Moves and/or resizes a window.Déplace et/ou redimensionne une fenêtre.

WinMove ( "title", "text", x, y [, width [, height [, speed]]] )

ParametersParamètres

title The title/hWnd/class of the window to move/resize.Le titre/hWnd/class de la fenêtre à déplacer/redimensionner. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to move/resize.Le texte de la fenêtre à déplacer/redimensionner. See Text special definition.Consultez Définition spéciale de Texte .
x X coordinate to move to.Coordonnée X pour le déplacement.
y Y coordinate to move to.Coordonnée Y pour le déplacement.
width [optional][optionnel] New width of the window.Nouvelle largeur de la fenêtre.
height [optional][optionnel] New height of the window.Nouvelle hauteur de la fenêtre.
speed [optional][optionnel] the speed to move the windows in the range 1 (fastest) to 100 (slowest).La vitesse de déplacement de la fenêtre comprise entre 1 (rapide) et 100 (lent). If not defined the move is instantaneous.Si non défini, le déplacement est instantané.

Return ValueValeur de retour

Success:Succès : a handle to the window.Retourne un handle pour la fenêtre.
Failure:Échec : 0 if the window is not found.Retourne 0 si la fenêtre n'existe pas.

RemarksRemarques

WinMove() has no effect on minimized windows, but WinMove() works on hidden windows.WinMove() n'a pas d'effet sur les fenêtres minimisées, mais fonctionne sur les fenêtres cachées.
If the width and height are small (or negative), the window will go no smaller than 112 x 27 pixels.Si la largeur et la hauteur sont trop petites (ou négative), la fenêtre n'ira pas en dessous de 112 x 27 pixels. If width and height are large, the window will go no larger than approximately [12+@DesktopWidth] x [12+@DesktopHeight] pixels.Si la largeur et la hauteur sont trop grandes, La fenêtre ne sera pas plus grande que [12 + @DesktopWidth] x [12 + @DesktopHeight] pixels.
Negative values are allowed for the x and y coordinates.Des valeurs négatives sont autorisées pour les coordonnées de x et de y. In fact, you can move a window off screen; and if the window's program is one that remembers its last window position, the window will appear in the corner (but fully on-screen) the next time you launch the program.En fait, vous pouvez déplacer la fenêtre hors de la zone de l'écran; et si la fenêtre du programme est une fenêtre qui se rappelle de sa dernière position sur l'écran, la fenêtre apparaîtra dans le coin (mais entièrement à l'écran) la prochaine fois que vous lancerez le programme.
If multiple windows match the criteria, the most recently active window is used.Si plusieurs fenêtres correspondent aux critères, la fenêtre active la plus récente sera utilisé.

If x and y equal to the Default keyword no move occurs, just resizing.Si x et y sont égaux au mot clé Default il n'y aura pas de déplacement, juste un redimensionnement.

When speed is used the move is done with the new size.Lorsque le paramètre vitesse est utilisé, le déplacement est effectué avec la nouvelle dimension.

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Retrieve the position as well as the height and width of the Notepad window.Récupère la position ainsi que la hauteur et la largeur de la fenêtre du Bloc-notes. We will use this when we have to move the window back to the original position.Nous les utiliserons lorsque nous déplacerons la fenêtre à sa position initiale.
    Local $aPos = WinGetPos($hWnd)

    ; Move the Notepad to the x, y position of 0, 0 and set the height and width at 200, 200.Déplace le bloc-notes à la position x, y de 0, 0 et définit la hauteur et la largeur à 200, 200.
    WinMove($hWnd, "", 0, 0, 200, 200)

    ; Wait for 2 seconds to display the new position of the Notepad window.Attend 2 secondes pour afficher la nouvelle position de la fenêtre du Bloc-notes.
    Sleep(2000)

    ; Move the Notepad window back to the original position by using the array returned by WinGetPos.Déplace la fenêtre du Bloc-notes de retour à sa position initiale en utilisant le tableau renvoyé par WinGetPos.
    WinMove($hWnd, "", $aPos[0], $aPos[1], $aPos[2], $aPos[3])

    ; Wait for 2 seconds to display the original position of the Notepad window.Attend 2 secondes pour afficher la position initiale de la fenêtre du Bloc-notes.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
