DriveGetType

Returns drive type.Retourne le type de lecteur.

DriveGetType ( "path" [, operation = 1] )

ParametersParamètres

path Path of drive to receive information from.Le chemin sur lequel agir.
operation [optional][optionnel] The drive type operation to perform.
    $DT_DRIVETYPE (1) = the type of drive (default)
    $DT_SSDSTATUS (2) = SSD status of the drive
    $DT_BUSTYPE (3) = the bus type of drive

Constants are defined in AutoItConstants.au3.

Return ValueValeur de retour

Success:Succès : See remarks.Voir remarques.
Failure:Échec : Sets the @error flag to 1 if a bad path was given, or if the operation failed. Return value is "".

RemarksRemarques

If the mode parameter is $DT_DRIVETYPE (1), returns the type of drive.Si le paramètre du mode est $DT_DRIVETYPE (1), la fonction retourne le type de lecteur. The path must be a string volume name, such as "C:\".Le chemin doit être une string qui contient le nom du volume, comme "C:\". Return values are:Les valeurs retournées sont:
"Unknown", "Removable", "Fixed", "Network", "CDROM", "RAMDisk"

If the mode parameter is $DT_SSDSTATUS (2), returns the SSD status of the drive.Si le paramètre du mode est $DT_SSDSTATUS (2), la fonction retourne le statut SSD du lecteur. The path can be a string volume name, or an integer drive index.Le chemin peut être une chaîne qui contient le nom du volume, ou un entier, index du lecteur. Return values are:Les valeurs retournées sont:
"SSD", "" (blank)

If the mode parameter is $DT_BUSTYPE (3), returns the bus type of the drive.Si le paramètre du mode est $DT_BUSTYPE (3), la fonction retourne le type de bus du lecteur. The path can be a string volume name, or an integer drive index.Le chemin peut être une chaîne qui contient le nom du volume, ou un entier, index du lecteur. Return values are:Les valeurs retournées sont:
"Unknown", "SCSI", "ATAPI", "ATA", "1394", "SSA", "Fibre", "USB", "RAID", "iSCSI", "SAS", "SATA", "SD", "MMC", "Virtual", "FileBackedVirtual"

ExampleExemple

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

; Check the drive type for C:\Vérifiez le type de lecteur C :\\
Local $sInfo = DriveGetType("C:\")
MsgBox($MB_SYSTEMMODAL, "", "Drive Type: " & $sInfo)

; Check the SSD status for C:\Vérifiez l'état SSD C :\\
$sInfo = DriveGetType("C:\", $DT_SSDSTATUS)
MsgBox($MB_SYSTEMMODAL, "", "Drive SSD: " & $sInfo)

; Check the SSD status for disk 0Vérifie l'état de SSD pour le disque 0
$sInfo = DriveGetType(0, $DT_SSDSTATUS)
MsgBox($MB_SYSTEMMODAL, "", "Drive SSD: " & $sInfo)

; Check the bus type status for disk 0Vérifie l'état du type de bus pour le disque 0
$sInfo = DriveGetType(0, $DT_BUSTYPE)
MsgBox($MB_SYSTEMMODAL, "", "Drive Bus: " & $sInfo)
