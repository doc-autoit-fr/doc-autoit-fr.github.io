Math functions Reference

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
Abs Calculates the absolute value of a number.Calcule la valeur absolue d'un nombre.
ACos Calculates the arcCosine of a number.Calcule l’arc-cosinus d'un nombre.
ASin Calculates the arcsine of a number.Calcule l’arc-sinus d'un nombre.
ATan Calculates the arctangent of a number.Calcule l’arc tangente d'un nombre.
BitAND Performs a bitwise AND operation.Exécute un 'ET' logique bit à bit.
BitNOT Performs a bitwise NOT operation.Exécute un "NON" logique bit à bit.
BitOR Performs a bitwise OR operation.Exécute un "OU" logique bit à bit.
BitRotate Performs a bit shifting operation, with rotation.Exécute un décalage bit à bit avec rotation.
BitShift Performs a bit shifting operation.Exécute un décalage de bits.
BitXOR Performs a bitwise exclusive OR (XOR) operation.Exécute un "OU EXCLUSIF" logique bit à bit.
Ceiling Returns a number rounded up to the next integer.Retourne un nombre arrondi à l'entier suivant.
Cos Calculates the cosine of a number.Calcule le cosinus d'un nombre.
Exp Calculates e to the power of a number.Calcule e à la puissance d'un nombre.
Floor Returns a number rounded down to the closest integer.Retourne un nombre arrondi à l'entier inférieur le plus proche.
Log Calculates the natural logarithm of a number.Calcule le logarithme naturel d'un nombre.
Mod Performs the modulus operation.Effectue l'opération modulo (reste d'une division).
Random Generates a pseudo-random float-type number.Génère un nombre pseudo-aléatoire de type 'float'.
Round Returns a number rounded to a specified number of decimal places.Arrondi un décimal avec un nombre spécifié de décimales.
Sin Calculates the sine of a number.Calcule le sinus d'un nombre.
Sqrt Calculates the square-root of a number.Calcule la racine carrée d'un nombre.
SRandom Set Seed for random number generation.Définit la valeur de départ pour le générateur de nombre aléatoire.
Tan Calculates the tangent of a number.Calcule la tangente d'un nombre.
