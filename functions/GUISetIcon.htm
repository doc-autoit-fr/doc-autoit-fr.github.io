GUISetIcon

Sets the icon used in a GUI window.Définit l'icône utilisée dans une fenêtre GUI.

GUISetIcon ( iconfile [, iconID [, winhandle]] )

ParametersParamètres

iconfile used to display the icon in the title area.
iconID [optional][optionnel] The ID of the icon in the iconfile. (Default is -1).
winhandle [optional][optionnel] Windows handle as returned by GUICreate() (default is the previously used window).Handle Windows retourné par GUICreate() (par défaut, la fenêtre précédemment utilisée ).

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

Passing a positive number will reference the string equivalent icon name.Passer un nombre positif, fera référence à l'équivalent chaîne du nom de l'icône.
Passing a negative number causes 1-based "index" behaviour.Passer un nombre négatif, provoque un comportement d'"index" basé sur 1. Some Dll can have icon extracted

just with negative numbers.

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; X64 running supportX64 appui fonctionnement
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    ; Retrieve the following ico file. This can be found in the include folder which is in the installation path of AutoIt.Il peut être trouvé dans le dossier 'include' dans le chemin d'installation de AutoIt.
    Local $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\icons\au3.ico"

    GUICreate("My GUI new icon") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUISetIcon($sFile) ; will change icon

    GUISetState(@SW_SHOW); will display an empty dialog boxAffiche une boîte de dialogue vide

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example
