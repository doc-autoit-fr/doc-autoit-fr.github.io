WinGetCaretPos

Returns the coordinates of the caret in the foreground window.Retourne les coordonnées du curseur dans la fenêtre du premier plan.

WinGetCaretPos ( )

Return ValueValeur de retour

Success:Succès : a 2-element array containing the following information:Retourne un tableau de 2 éléments contenant les informations suivantes :
    $aArray[0] = X coordinate
    $aArray[1] = Y coordinate
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

WinGetCaretPos() might not return accurate values for Multiple Document Interface (MDI) applications if absolute CaretCoordMode is used.WinGetCaretPos() peut ne pas retourner des valeurs précises pour des applications 'Multiple Document Interface' (MDI) si CaretCoordMode est utilisée avec l'option absolute. See example for a workaround.Voir l'exemple pour une solution de contournement. Note:Note : Some applications report static coordinates regardless of caret position!Certaines applications signalent des coordonnées statiques quelle que soit la position du curseur !

ExampleExemple

Local $aCaretPos = WinGetCaretPos()
If Not @error Then
    ToolTip("First Method", $aCaretPos[0], $aCaretPos[1])
EndIf
Sleep(2000)

$aCaretPos = _WinGetCaretPos()
If Not @error Then
    ToolTip("Second Method", $aCaretPos[0], $aCaretPos[1])
EndIf
Sleep(2000)

; A more reliable method to retrieve the caret coordinates in MDI text editors.Une méthode plus fiable pour récupérer les coordonnées du curseur dans des éditeurs de texte MDI.
Func _WinGetCaretPos()
    Local $iXAdjust = 5
    Local $iYAdjust = 40

    Local $iOpt = Opt("CaretCoordMode", 0) ; Set "CaretCoordMode" to relative mode and store the previous option.Définit "CaretCoordMode" en mode relatif et stocke l'option précédente.
    Local $aGetCaretPos = WinGetCaretPos() ; Retrieve the relative caret coordinates.Récupére les coordonnées relatives du curseur.
    Local $aGetPos = WinGetPos("[ACTIVE]") ; Retrieve the position as well as height and width of the active window.Récupére la position ainsi que la hauteur et la largeur de la fenêtre active.
    Local $sControl = ControlGetFocus("[ACTIVE]") ; Retrieve the control name that has keyboard focus.Récupére le nom du contrôle qui a le focus clavier.
    Local $aControlPos = ControlGetPos("[ACTIVE]", "", $sControl) ; Retrieve the position as well as the size of the control.Récupére la position ainsi que la taille du contrôle.
    $iOpt = Opt("CaretCoordMode", $iOpt) ; Reset "CaretCoordMode" to the previous option.Re-initialise "CaretCoordMode" à l'option précédente.

    Local $aReturn[2] = [0, 0] ; Create an array to store the x, y position.Crée un tableau pour stocker la position x, y.
    If IsArray($aGetCaretPos) And IsArray($aGetPos) And IsArray($aControlPos) Then
        $aReturn[0] = $aGetCaretPos[0] + $aGetPos[0] + $aControlPos[0] + $iXAdjust
        $aReturn[1] = $aGetCaretPos[1] + $aGetPos[1] + $aControlPos[1] + $iYAdjust
        Return $aReturn ; Return the array.Retour du tableau.
    Else
        Return SetError(1, 0, $aReturn) ; Return the array and set @error to 1.Retourne le tableau et définit @error à 1.
    EndIf
EndFunc   ;==>_WinGetCaretPos
