TCPNameToIP

Converts an Internet name to IP address.Convertit un nom de domaine en adresse IP.

TCPNameToIP ( name )

ParametersParamètres

name String containing an Internet name.Chaîne de caractères contenant le nom de domaine.

Return ValueValeur de retour

Success:Succès : a string containing IP address corresponding to the name.Retourne une chaîne de caractères contenant l'adresse IP correspondant au nom de domaine.
Failure:Échec : "" and sets the @error flag to non-zero.Retourne "" et @error <> 0.
@error:@error : Windows API WSAGetError return value (see MSDN).Valeur retournée par l'API Windows WSAGetError (Consultez MSDN).

RemarksRemarques

TCPStartup() is needed before this call.TCPStartup() est nécessaire avant un appel à cette fonction.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Start the TCP service.Démarre le service TCP.
    TCPStartup()

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

    ; Assign a Local variable the IP address of www.autoitscript.com.Assigne une variable locale avec l'adresse IP: www.autoitscript.com.
    Local $sIPAddress = TCPNameToIP("www.autoitscript.com")

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error code: " & @error)
        Return False
    Else
        ; Display the IP address.Affiche l'adresse IP.
        MsgBox($MB_SYSTEMMODAL, "", "Domain name to IP :" & $sIPAddress)
    EndIf
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Close the TCP service.Ferme le service TCP.
EndFunc   ;==>OnAutoItExit
