Sin

Calculates the sine of a number.Calcule le sinus d'un nombre.

Sin ( expression )

ParametersParamètres

expression Value in radians.Valeur en radians.

Return ValueValeur de retour

Returns the trigonometric sine of the number.Retourne le sinus trigonométrique du nombre.

RemarksRemarques

1° = pi / 180 radians.

RelatedLiens

ACos, ASin, ATan, Cos, Tan

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local constant variable the approximate PI number.Assigne une constante locale avec une approximation du nombre de PI.
    Local Const $PI = 3.141592653589793

    ; Assign a Local variable the sine of PI / 4 (equals to Sqrt(2) / 2).Assigne une variable locale avec le sinus de PI / 4 (égal à sqrt (2) / 2).
    Local $fSin1 = Sin($PI / 4)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fSin1)

    ; Assign a Local variable the formula to switch from radian to degree (equals to one degree in radian).Assigne une variable locale avec le coefficient de conversion de radian à degré (égal à un degré en radian).
    Local $fDegToRad = $PI / 180

    ; Assign a Local variable the sine of 90 degrees (firstly converted to radians).Assigne une variable locale avec le sinus de 90 degrés (d'abord converti en radians).
    Local $fSin2 = Sin(90 * $fDegToRad)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fSin2)
EndFunc   ;==>Example
