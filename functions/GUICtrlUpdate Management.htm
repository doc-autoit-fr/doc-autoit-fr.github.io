FunctionFonction Description

GUICtrlSetBkColor Sets the background color of a control.Définit la couleur de fond d'un contrôle.
GUICtrlSetColor Sets the text color of a control.Définit la couleur du texte d'un contrôle.
GUICtrlSetCursor Sets the mouse cursor icon for a particular control.Définit l'icône du curseur de la souris pour un contrôle particulier.
GUICtrlSetData Modifies the data for a control.Modifie les données d'un contrôle.
GUICtrlSetDefBkColor Sets the default background color of all the controls of the GUI window.Définit la couleur de fond par défaut de tous les contrôles de la fenêtre GUI.
GUICtrlSetDefColor Sets the default text color of all the controls of the GUI window.Définit la couleur de texte par défaut de tous les contrôles d'une fenêtre GUI.
GUICtrlSetFont Sets the font for a control.Définit la police d'un contrôle.
GUICtrlSetGraphic Modifies the data for a control.Modifie les données d'un contrôle.
GUICtrlSetImage Sets the bitmap or icon image to use for a control.Définit l'image bitmap ou icône à utiliser pour un contrôle.
GUICtrlSetLimit Limits the number of characters/pixels for a control.Limite le nombre de caractères/pixels pour un contrôle.
GUICtrlSetOnEvent Defines a user-defined function to be called when a control is clicked.Définit une fonction utilisateur appelée quand un contrôle est cliqué.
GUICtrlSetPos Changes the position of a control within the GUI window.Modifie la position d'un contrôle dans la fenêtre GUI.
GUICtrlSetResizing Defines the resizing method used by a control.Définit la méthode de redimensionnement utilisée par un contrôle.
GUICtrlSetState Changes the state of a control.Modifie l'état d'un contrôle.
GUICtrlSetStyle Changes the style of a control.Change le style d'un contrôle.
GUICtrlSetTip Sets the tip text associated with a control.Définit le texte de l'info-bulle associée à un contrôle.
