GUICtrlCreateTreeViewItem

Creates a TreeViewItem control for the GUI.Crée un contrôle TreeViewItem dans le contrôle TreeView d'une GUI.

GUICtrlCreateTreeViewItem ( "text", treeviewID )

ParametersParamètres

text The text of the control.Le texte de la commande.
treeviewID treeview identifier as return by treeview or treeviewitem creation if subtree is created.L'identificateur de la TreeView retourné lors de la création ou l'identificateur d'un TreeViewItem si un sous-arbre est créé.

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

For setting more information see GUICtrlUpdate...() functions.Pour plus d'informations consultez les fonctions GUICtrlUpdate...().

To paint a treeview item in bold (to show as default) use GUICtrlSetState($treeviewItem, $GUI_DEFBUTTON), to turn off this behaviour use GUICtrlSetState() with another value but $GUI_DEFBUTTON, for instance GUICtrlSetState($treeviewItem, 0).Pour mettre l'item d'un TreeView en gras (pour le définir comme l'item par défaut ) utilisez GUICtrlSetState($treeviewItem, $GUI_DEFBUTTON), pour annuler cette propriété utilisez GUICtrlSetState() avec une autre valeur que $GUI_DEFBUTTON, par exemple GUICtrlSetState($TreeViewItem, 0 ).

To expand a treeview item use GUICtrlSetState($treeviewItem, $GUI_EXPAND).Pour développer l'item d'un TreeView, utilisez GUICtrlSetState($TreeViewItem, $GUI_EXPAND).

To select a specific treeview item use GUICtrlSetState($treeviewItem, $GUI_FOCUS).Pour sélectionner un item spécifique d'un TreeView, utilisez GUICtrlSetState($TreeViewItem, $GUI_FOCUS ).

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <StaticConstants.au3>
#include <TreeViewConstants.au3>

Example()

Func Example()
    GUICreate("My GUI with treeview", 350, 215)

    Local $idTreeview = GUICtrlCreateTreeView(6, 6, 100, 150, BitOR($TVS_HASBUTTONS, $TVS_HASLINES, $TVS_LINESATROOT, $TVS_DISABLEDRAGDROP, $TVS_SHOWSELALWAYS), $WS_EX_CLIENTEDGE)
    Local $idGeneralitem = GUICtrlCreateTreeViewItem("General", $idTreeview)
    GUICtrlSetColor(-1, 0x0000C0)
    Local $idDisplayitem = GUICtrlCreateTreeViewItem("Display", $idTreeview)
    GUICtrlSetColor(-1, 0x0000C0)
    Local $idAboutitem = GUICtrlCreateTreeViewItem("About", $idGeneralitem)
    Local $idCompitem = GUICtrlCreateTreeViewItem("Computer", $idGeneralitem)
    GUICtrlCreateTreeViewItem("User", $idGeneralitem)
    GUICtrlCreateTreeViewItem("Resolution", $idDisplayitem)
    GUICtrlCreateTreeViewItem("Other", $idDisplayitem)

    Local $idStartlabel = GUICtrlCreateLabel("TreeView Demo", 190, 90, 100, 20)
    Local $idAboutlabel = GUICtrlCreateLabel("This little scripts demonstates the using of a treeview-control.", 190, 70, 100, 60)
    GUICtrlSetState(-1, $GUI_HIDE) ; Hides the "aboutlabel"-text during initializationCache le texte de 'aboutlabel' lors de l'initialisation
    Local $idCompinfo = GUICtrlCreateLabel("Name:" & @TAB & @ComputerName & @CRLF & "OS:" & @TAB & @OSVersion & @CRLF & "SP:" & @TAB & @OSServicePack, 120, 30, 200, 80)
    GUICtrlSetState(-1, $GUI_HIDE) ; Hides the "compinfo"-text during initializationCache le texte de 'CompInfo' lors de l'initialisation

    GUICtrlCreateLabel("", 0, 170, 350, 2, $SS_SUNKEN)
    Local $idTogglebutton = GUICtrlCreateButton("&Toggle", 35, 185, 70, 20)
    Local $idInfobutton = GUICtrlCreateButton("&Info", 105, 185, 70, 20)
    Local $idStatebutton = GUICtrlCreateButton("Col./Exp.", 175, 185, 70, 20)
    Local $idCancelbutton = GUICtrlCreateButton("&Cancel", 245, 185, 70, 20)

    GUICtrlSetState($idGeneralitem, BitOR($GUI_EXPAND, $GUI_DEFBUTTON)) ; Expand the "General"-item and paint in boldDéveloppe l'item 'Général' et le met en gras
    GUICtrlSetState($idDisplayitem, BitOR($GUI_EXPAND, $GUI_DEFBUTTON)) ; Expand the "Display"-item and paint in boldDéveloppe l'item 'Display' et le met en gras

    GUISetState(@SW_SHOW)

    Local $idMsg, $idItem, $hItem, $sText
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        $idMsg = GUIGetMsg()
        Select
            Case $idMsg = $idCancelbutton Or $idMsg = $GUI_EVENT_CLOSE
                ExitLoop

            Case $idMsg = $idTogglebutton ; Toggle the bold paintingBascule la mise en gras
                If BitAND(GUICtrlRead($idGeneralitem), $GUI_DEFBUTTON) Then
                    GUICtrlSetState($idGeneralitem, 0)
                    GUICtrlSetState($idDisplayitem, 0)
                Else
                    GUICtrlSetState($idGeneralitem, $GUI_DEFBUTTON)
                    GUICtrlSetState($idDisplayitem, $GUI_DEFBUTTON)
                EndIf

            Case $idMsg = $idInfobutton
                $idItem = GUICtrlRead($idTreeview) ; Get the controlID of the current selected treeview itemObtient le controlID de l'item sélectionné en cours
                If $idItem = 0 Then
                    MsgBox($MB_SYSTEMMODAL, "TreeView Demo", "No item currently selected")
                Else
                    $sText = GUICtrlRead($idItem, 1) ; Get the text of the treeview itemObtient le texte de l'item de la TreeView
                    If $sText == "" Then
                        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "Error while retrieving infos about item")
                    Else
                        MsgBox($MB_SYSTEMMODAL, "TreeView Demo", "Current item selected is: " & $sText)
                    EndIf
                EndIf

            Case $idMsg = $idStatebutton
                $idItem = GUICtrlRead($idTreeview)
                If $idItem > 0 Then
                    $hItem = GUICtrlGetHandle($idItem)
                    GUICtrlSendMsg($idTreeview, $TVM_EXPAND, $TVE_TOGGLE, $hItem)
                EndIf

                ; The following items will hide the other labels (1st and 2nd parameter) and then show the 'own' labels (3rd and 4th parameter)Les items suivants cacheront les autres labels (1er et 2nd paramètre), puis afficheront les labels 'propres' (3ème et 4ème paramètre)
            Case $idMsg = $idGeneralitem
                GUIChangeItems($idAboutlabel, $idCompinfo, $idStartlabel, $idStartlabel)

            Case $idMsg = $idAboutitem
                GUICtrlSetState($idCompinfo, $GUI_HIDE)
                GUIChangeItems($idStartlabel, $idStartlabel, $idAboutlabel, $idAboutlabel)

            Case $idMsg = $idCompitem
                GUIChangeItems($idStartlabel, $idAboutlabel, $idCompinfo, $idCompinfo)
        EndSelect
    WEnd

    GUIDelete()
EndFunc   ;==>Example

Func GUIChangeItems($idHidestart, $idHideend, $idShowstart, $idShowend)
    Local $idX

    For $idX = $idHidestart To $idHideend
        GUICtrlSetState($idX, $GUI_HIDE)
    Next
    For $idX = $idShowstart To $idShowend
        GUICtrlSetState($idX, $GUI_SHOW)
    Next
EndFunc   ;==>GUIChangeItems
