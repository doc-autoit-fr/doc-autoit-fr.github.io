ControlMove

Moves a control within a window.Déplace un contrôle dans une fenêtre.

ControlMove ( "title", "text", controlID, x, y [, width [, height]] )

ParametersParamètres

title The title/hWnd/class of the window to access.Le titre/handle/class de la fenêtre à atteindre. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to access.Le texte de la fenêtre à accéder. See Text special definition.Consultez Définition spéciale de Texte .
controlID The control to interact with.Le contrôle avec lequel interagir. See Controls.Consultez Controls.
x X coordinate to move to relative to the window client area.La coordonnée x de la nouvelle position du contrôle, relatif à la zone cliente de la fenêtre.
y Y coordinate to move to relative to the window client area.La coordonnée y de la nouvelle position du contrôle, relatif à la zone cliente de la fenêtre.
width [optional][optionnel] New width of the window.Nouvelle largeur de la fenêtre.
height [optional][optionnel] New height of the window.Nouvelle hauteur de la fenêtre.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if window/control is not found.Retourne 0 si la fenêtre/le contrôle n'est pas trouvé.

RemarksRemarques

If x and y equal to the Default keyword no move occurs, just resizing.Si x et y sont égaux au mot clé Default il n'y aura pas de déplacement, juste un redimensionnement.

RelatedLiens

ControlCommand, ControlGetPos

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Move the edit control of Notepad using the handle returned by WinWait.Déplace le contrôle d'édition du Bloc-notes à l'aide du handle renvoyé par WinWait.
    ControlMove($hWnd, "", "Edit1", 0, 0, 150, 150)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
