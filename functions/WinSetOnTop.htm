WinSetOnTop

Change a window's "Always On Top" attribute.Change l'attribut "Always On Top" d'une fenêtre.

WinSetOnTop ( "title", "text", flag )

ParametersParamètres

title The title/hWnd/class of the window to change.Le titre/hWnd/class de la fenêtre à changer. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to change.Le texte de la fenêtre à changer. See Text special definition.Consultez Définition spéciale de Texte .
flag Determines whether the window should have the "TOPMOST" flag set.Détermine si la fenêtre doit avoir le paramètre "Mise en avant plan (TOPMOST)" activé.
    $WINDOWS_NOONTOP (0) = remove on top flag
    $WINDOWS_ONTOP (1) = set on top flag

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if the window is not found.Retourne 0 si la fenêtre n'existe pas.

RemarksRemarques

Third-party programs which add an "Always On Top" context menu entry might not update their menu entry to reflect the AutoIt-induced change in TOPMOST status.Les programmes tiers incluant un menu contextuel "Toujours à l'avant plan (Always on Top)" ne pourront mettre à jour l'entrée de leur menu pour refléter le changement de statut de AutoIt en TOPMOST.

ExampleExemple

#include <AutoItConstants.au3>

Example()

Func Example()
    ; Retrieve the handle of the active window.Récupère le handle de la fenêtre active.
    Local $hWnd = WinGetHandle("[ACTIVE]")

    ; Set the active window as being ontop using the handle returned by WinGetHandle.Définit la fenêtre active comme étant 'on top' à l'aide du handle renvoyé par WinGetHandle.
    WinSetOnTop($hWnd, "", $WINDOWS_ONTOP)

    ; Wait for 2 seconds to display the change.Attend 2 secondes pour afficher le changement.
    Sleep(2000)

    ; Remove the "topmost" state from the active window.Retire l'état "topmost" à la fenêtre active.
    WinSetOnTop($hWnd, "", $WINDOWS_NOONTOP)
EndFunc   ;==>Example
