WinWaitActive

Pauses execution of the script until the requested window is active.Met le script en pause jusqu'à ce que la fenêtre indiquée soit active.

WinWaitActive ( "title" [, "text" [, timeout = 0]] )

ParametersParamètre

title The title/hWnd/class of the window to check.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optional] False - i, v, x ... The text of the window to check.Le texte de la fenêtre à contrôler. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .
timeout [optional][optional] False - i, v, x ... Timeout in seconds if the window is not active.Temporisation en secondes si la fenêtre n'est pas active. Default is 0 (no timeout).La valeur par défaut est 0 (pas de temporisation).

Return ValueRemarque

Success: a handle to the requested window.Retourne un handle sur la fenêtre demandée.
Failure: 0 if timeout occurred.Retourne 0 si la temporisation est dépassée.

Remarks<i0>

The window is polled every 250 milliseconds or so.La fenêtre est interrogée toutes les 250 millisecondes ou plus.

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    WinWaitActive("[CLASS:Notepad]", "", 10)

    ; Wait for 2 seconds to display the Notepad window.Attend 2 secondes pour afficher la fenêtre du Bloc-notes.
    Sleep(2000)

    ; Close the Notepad window using the classname of Notepad.Ferme la fenêtre du Bloc-notes à l'aide du nom de classe du Bloc-notes.
    WinClose("[CLASS:Notepad]")
EndFunc   ;==>Example
