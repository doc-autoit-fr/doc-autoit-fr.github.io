EnvGet

Retrieves an environment variable.Récupère une variable d'environnement.

EnvGet ( "envvariable" )

ParametersParamètres

envvariable Name of the environment variable to get such as "TEMP" or "PATH".Nom de la variable d'environnement à récupérer comme "TEMP" ou "PATH"

Return ValueValeur de retour

Returns the requested variable (or an empty string if the variable does not exist).Retourne la variable demandée (ou une chaîne vide si la variable n'existe pas).

RelatedLiens

EnvSet, EnvUpdate

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Retrieve the value of the environment variable %APPDATA%.Récupére la valeur de la variable d'environnement %APPDATA%.
    ; When you assign or retrieve an envorinment variable you do so minus the percentage signs (%).Lorsque vous affectez ou récupérez une variable d'environnement vous le faites moins les signes de pourcentage (%).
    Local $sEnvVar = EnvGet("APPDATA")

    ; Display the value of the environment variable %APPDATA%.Affiche la valeur de la variable d'environnement %APPDATA%.
    MsgBox($MB_SYSTEMMODAL, "", "The environment variable %APPDATA% has the value of: " & @CRLF & @CRLF & $sEnvVar) ; This returns the same value as the macro @AppDataDir does.Ceci renvoie la même valeur que la macro AppDataDir.
EndFunc   ;==>Example
