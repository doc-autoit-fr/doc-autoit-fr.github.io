GUICtrlCreateCombo

Creates a ComboBox control for the GUI.Crée un Contrôle ComboBox dans une GUI.

GUICtrlCreateCombo ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

text The text which will appear in the combo control.Le texte qui apparaîtra dans le contrôle ComboBox.
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel][optionnel] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optionnel][optionnel] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional][optionnel][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default (-1) : $CBS_DROPDOWN, $CBS_AUTOHSCROLL, $WS_VSCROLL
    forced style : $WS_TABSTOP$ WS_TABSTOP
exStyle [optional][optionnel][optionnel] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .
    default ( -1) : $WS_EX_CLIENTEDGE

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.

RemarksRemarques

To obtain the value of the control see GUICtrlRead().Pour obtenir la valeur du contrôle, consultez GUICtrlRead() .
To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

Under Windows XP/2003 Windows will adjust the size of the opened combo.Sous Windows XP/2003 Windows ajustera la taille du combo ouvert. On other Windows versions you can define this size with the "height" parameter if the default value is not BIG enough to contain at least one line.Sous les autres versions de Windows, vous pouvez déterminer cette taille avec le paramètre "height" si la valeur par défaut n'est pas ASSEZ GRANDE pour contenir au moins une ligne.

To combine styles with the default style use BitOR ( $GUI_SS_DEFAULT_COMBO, newstyle, ... ).)..
To use the values specified above you must #include <ComboConstants.au3> in your script.

Default resizing is $GUI_DOCKHEIGHT.Par défaut redimensionnement est $ GUI_DOCKHEIGHT.

ExampleExemple

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Create a GUI with various controls.Crée une GUI avec différents contrôles.
    Local $hGUI = GUICreate("ExampleExemple", 300, 200)

    ; Create a combobox control.Crée un contrôle ComboBox.
    Local $idComboBox = GUICtrlCreateCombo("Item 1", 10, 10, 185, 20)
    Local $idClose = GUICtrlCreateButton("Close", 210, 170, 85, 25)

    ; Add additional items to the combobox.Ajoute des items supplémentaires à la ComboBox.
    GUICtrlSetData($idComboBox, "Item 2|Item 3", "Item 2")

    ; Display the GUI.Affiche la GUI.
    GUISetState(@SW_SHOW, $hGUI)

    Local $sComboRead = ""

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idClose
                ExitLoop

            Case $idComboBox
                $sComboRead = GUICtrlRead($idComboBox)
                MsgBox($MB_SYSTEMMODAL, "", "The combobox is currently displaying: " & $sComboRead, 0, $hGUI)

        EndSwitch
    WEnd

    ; Delete the previous GUI and all controls.Supprime la GUI précédente et tous ses contrôles.
    GUIDelete($hGUI)
EndFunc   ;==>Example
