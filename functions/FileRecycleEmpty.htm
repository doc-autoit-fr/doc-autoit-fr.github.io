FileRecycleEmpty

Empties the recycle bin.Vide la corbeille.

FileRecycleEmpty ( ["source"] )

ParametersParamètres

source [optional][optionnel] The rootpath to empty - if omitted the recycle bin for all drives is emptied.Le chemin de la racine à vider - si il est omis, les corbeilles de tous les lecteurs sont vidées.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 (the recycle bin cannot be emptied - see below).Retourne 0 (la corbeille ne peut être vidé - Consultez ci-dessous).

RemarksRemarques

For this function to work IE4+ must be available.Pour que cette fonction puisse fonctionner IE4+ doit être disponible.

RelatedLiens

FileDelete, FileRecycle

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Open the recycle bin by using the following CLSID.Ouvre le bac de recyclage en utilisant le CLSID suivant.
    ShellExecute("::{645FF040-5081-101B-9F08-00AA002F954E}")

    ; Display a message box asking whether or not you would like to delete files in the recycle bin.Affiche une boîte de message vous demandant si vous souhaitez supprimer des fichiers dans la corbeille.
    If MsgBox(BitOR($MB_YESNO, $MB_SYSTEMMODAL), "", "Would you like to empty the recycle bin on the " & @HomeDrive & "\ drive?") = $IDNO Then Return False

    ; Delete files in the recycle bin.Supprime les fichiers dans la corbeille. This is for the @HomeDrive only, which is generally the C:\ drive.C'est pour le lecteur principal seulement, qui est généralement le lecteur C:\.
    Local $iRecycle = FileRecycleEmpty(@HomeDrive)

    ; Display a message of whether the recycle bin was emptied successfuly.Affiche un message si la corbeille a été vidée avec succès.
    If $iRecycle Then
        MsgBox($MB_SYSTEMMODAL, "", "The recycle bin was emptied successfuly.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The recycle bin wasn't emptied successfuly.")
    EndIf
EndFunc   ;==>Example
