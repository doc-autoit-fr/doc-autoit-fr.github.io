GUICtrlSetBkColor

Sets the background color of a control.Définit la couleur de fond d'un contrôle.

GUICtrlSetBkColor ( controlID, backgroundcolor )

ParametersParamètre

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
backgroundcolor The RGB color to use.La couleur RVB à utiliser.

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0.Retourne 0.

Remarks<i0>

Only Button, Label, Checkbox, Group, Radio, Edit, Input, List, Listview, ListviewItem, Treeview, TreeviewItem, Graphic, Progress, Slider and Combo controls can currently be colored.Seuls les contrôles Button, Label, Checkbox, Group, Radio, Edit, Input, List, Listview, ListviewItem, Treeview, TreeviewItem, Graphic, Progress, Slider et Combo peuvent actuellement être colorés.

Progress controls cannot be painted if the "Windows XP style" is used.Les contrôles Progress ne peuvent pas être peints si le "style Windows XP" est utilisé.
Checkbox, Radio, Group Progress, Combo controls cannot be painted if the "Windows XP/Vista style" is used.Les contrôles Checkbox, Radio, Groupe Progrès, Combo ne peuvent pas être peints si le "style de Windows XP/Vista" est utilisé.

Button controls are always painted in "Windows Classic style".Les contrôles Button sont toujours peints en "style Windows Classique". They cannot have the $BS_ICON style.Ils ne peuvent pas avoir le style $BS_ICON.

The special flag $GUI_BKCOLOR_TRANSPARENT can be used with Label, Group, Radio, Checkbox controls to give them a transparent background.Le drapeau spécial $GUI_BKCOLOR_TRANSPARENT peut être utilisé avec es contrôles Label, Group, Radio, Checkbox pour leur donner un fond transparent.

The special flag $GUI_BKCOLOR_LV_ALTERNATE can be used with Listview control to give alternate background of the ListviewItems lines.Le drapeau spécial $GUI_BKCOLOR_LV_ALTERNATE peut être utilisé avec le contrôle ListView pour lui donner un fond différent suivant les lignes des ListviewItems.
The odd lines will get the color set by GUICtrlSetBkColor() of the Listview control.Les lignes impaires obtiendront la couleur définie par GUICtrlSetBkColor() du contrôle ListView.
The even lines will get the color set by GUICtrlSetBkColor() of the ListviewItem control.Les lignes paires auront la couleur définie par GUICtrlSetBkColor() du contrôle ListViewItem.
