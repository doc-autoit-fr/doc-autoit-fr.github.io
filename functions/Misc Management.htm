Misc. functions Reference

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
AdlibRegister Registers an Adlib function.Enregistre une fonction AdLib.
AdlibUnRegister Unregisters an adlib function.Annule l'enregistrement d'une fonction AdLib.
AutoItSetOption Changes the operation of various AutoIt functions/parameters.Change le mode de fonctionnement de divers fonctions/paramètres d'AutoIt.
AutoItWinGetTitle Retrieves the title of the AutoIt window.Récupère le titre d'une fenêtre Autoit.
AutoItWinSetTitle Changes the title of the AutoIt window.Modifie le titre d'une fenêtre Autoit.
BlockInput Disable/enable the mouse and keyboard.Désactive/Active la souris et le clavier.
Break Enables or disables the users' ability to exit a script from the tray icon menu.Active ou désactive la possibilité d'arrêter un script depuis le menu de l'icône de la zone de notification.
Call Calls a user-defined or built-in function contained in first parameter.Appelle une fonction utilisateur ou interne passée dans le premier paramètre.
CDTray Opens or closes the CD tray.Ouvre ou ferme le tiroir d'un lecteur CD.
Execute Execute an expression.Exécute une expression.
OnAutoItExitRegister Registers a function to be called when AutoIt exits.Enregistre une fonction qui est appelée quand AutoIt s'arrête.
OnAutoItExitUnRegister UnRegisters a function that was called when AutoIt exits.Annule l'inscription d'une fonction appelée lorsque AutoIt se termine.
SetError Manually set the value of the @error macro (and optionally @extended, and "Return Value").
SetExtended Manually set the value of the @extended macro.Définit manuellement la valeur de la macro @extended.
VarGetType Returns the internal type representation of a variant.Retourne le type de représentation interne d'un variant.
