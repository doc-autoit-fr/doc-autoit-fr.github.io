UDPBind

Create a socket bound to an incoming connection.Crée un socket lié à une connexion entrante.

UDPBind ( IPAddr, port )

ParametersParamètres

IPAddr Internet Protocol dotted address(IpV4) as "192.162.1.1".L'adresse IPv4 séparée par des points tel que :"192.162.1.1".
port port on which the created socket will be bound.Port sur lequel le socket créé sera connecté.

Return ValueValeur de retour

Success:Succès : an array :Retourne un tableau : $aArray[1] contains the real socket, $aArray[2] contains the specified IP address and $aArray[3] contains the port.$array[1] contient le socket réel, $array[2] contient l'adresse IP spécifiée et $array[3] contient le port. We need this information in subsequent calls to UDPRecv(), where we pass this socket structure/array.Nous avons besoin de ces informations pour des appels ultérieurs à UDPRecv(), où nous passons cette structure/tableau de socket.
Failure:Échec : Sets the @error flag to non-zero.Retourne @error <> 0
@error:@error : 1 - IPAddr is incorrect.1 = Le paramètre IPAddr est incorrect.
2 - port is incorrect.2 = Le port est incorrect.
Windows API WSAGetError return value (see MSDN).Valeur retournée par l'API Windows WSAGetError (Consultez MSDN).

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    UDPStartup() ; Start the UDP service.Démarre le service UDP.

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

    ; Assign Local variables the loopback IP Address and the Port.Assigne des variables locales avec l'adresse IP et le port.
    Local $sIPAddress = "127.0.0.1" ; This IP Address only works for testing on your own computer.Cette adresse IP ne fonctionne que pour les tests sur votre propre ordinateur.
    Local $iPort = 65532 ; Port used for the connection.Port utilisé pour la connexion.

    ; Assign a Local variable the socket and bind to the IP Address and Port specified.Assigne une variable locale avec le socket et se connecte à l'adresse IP et au port spécifié.
    Local $aSocket = UDPBind($sIPAddress, $iPort)

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        ; Someone is probably already binded on this IP Address and Port (script already running?).Quelqu'un est probablement déjà connecté sur cette adresse IP et ce port (script déjà en cours d"exécution?).
        Local $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Could not bind, Error code: " & $iError)
        Return False
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Bind successful.")
    EndIf

    ; Close the Listening socket to allow afterward binds.Ferme le socket d'écoute pour permettre pour permettre une connexion ultérieure.
    ; While not closed, any other program can NOT bind to the same IP Address and Port.Tant que n'est pas fermé, aucun autre programme ne peut pas se connecter à la même adresse IP et à ce port.
    UDPCloseSocket($aSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    UDPShutdown() ; Close the UDP service.Ferme le service UDP.
EndFunc   ;==>OnAutoItExit
