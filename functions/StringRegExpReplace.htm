StringRegExpReplace

Replace text in a string based on regular expressions.Remplace du texte, dans une chaîne, selon une expression régulière donnée.

StringRegExpReplace ( "test", "pattern", "replace" [, count = 0] )

ParametersParamètres

test The string to checkLa chaîne à traiter
pattern The regular expression to compare.L'expression régulière qui définit la recherche. See StringRegExp for pattern definition characters.Consultez StringRegExp pour les caractères de définition du modèle.
replace The text to replace the regular expression matching text with.Le texte destiné à remplacer le texte trouvé par l'expression régulière. To insert matched group text, \0 - \9 (or $0 - $9) can be used as back-references. (See remarks)
count [optional][optionnel] The number of times to execute the replacement in the string.Le nombre de fois où exécuter le remplacement de la chaîne. The default is 0.Par défaut c'est 0. Use 0 for global replacement.Utilisez 0 pour un remplacement global.

Return ValueValeur de retour

Success:Succès : an updated string based on regular expressions.Retourne la chaîne modifiée selon l'expression régulière. Check @extended for the number of replacements performed.Testez @extended pour obtenir le nombre de remplacements effectués.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.
@error:@error : 2 = Pattern invalid.2 = Modèle invalide. @extended = offset of error in pattern.@extended = position de l'erreur dans le modèle.

RemarksRemarques

To separate back-reference replacements from actual (replaced) numbers, wrap them with curly braces, i.e:Pour séparer les remplacements avec référence arrière des nombres (remplacés) actuels, les encadrer avec des accolades, exemple: "${1}5".
If a "\" needs to be in the replaced string it must be doubled.Si un "\" doit être remplacé dans la chaîne il doit être doublé. This is a consequence of the back-references mechanism.Ceci est une conséquence du mécanisme de référence arrière.
The "\" and "$" replacement formats are the only valid back-references formats supported.Les formats de remplacement de "\" et "$" sont les seuls formats valides pour les références arrières supportées.

See also the Regular Expression tutorial, in which you can run a script to test your regular expression(s).Consultez aussi le tutoriel Expressions régulières, dans lequel vous pouvez exécuter un script pour tester votre expression régulière.

; This example demonstrates a basic replacement.Cet exemple montre un remplacement de base. It replaces the vowels aeiouIl remplace les voyelles aeiou
; with the @ character.avec le caractère @.
Func Test1()
    Local $sInput = "Where have all the flowers gone, long time passing?"
    Local $sOutput = StringRegExpReplace($sInput, "[aeiou]", "@")
    Display($sInput, $sOutput)
EndFunc   ;==>Test1

; The following example demonstrates using back-references to change the dateL'exemple suivant illustre l'utilisation des références arrières pour changer la date
; from MM/DD/YYYY to DD.MM.YYYY
Func Test2()
    Local $sInput = 'some text1 12/31/2009 01:02:03 some text2' & @CRLF & _
            'some text3 02/28/2009 11:22:33 some text4'
    Local $sOutput = StringRegExpReplace($sInput, '(\d{2})/(\d{2})/(\d{4})', ' $2.$1.$3 ')
    Display($sInput, $sOutput)
EndFunc   ;==>Test2

; The following example demonstrates the need to double backslashL'exemple suivant illustre la nécessité d'une double barre oblique inverse
Func Test3()
    Local $sInput = '%CommonProgramFiles%\Microsoft Shared\'
    Local $sOutput = StringRegExpReplace($sInput, '%([^%]*?)%', 'C:\\WINDOWS\\Some Other Folder$')
    Display($sInput, $sOutput)
EndFunc   ;==>Test3

Func Display($sInput, $sOutput)
    ; Format the output.Formate la sortie.
    Local $sMsg = StringFormat("Input:\t%s\n\nOutput:\t%s", $sInput, $sOutput)
    MsgBox($MB_SYSTEMMODAL, "Results", $sMsg)
EndFunc   ;==>Display
