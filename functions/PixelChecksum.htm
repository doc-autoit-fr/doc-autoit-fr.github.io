PixelChecksum

Generates a checksum for a region of pixels.Génère une somme de contrôle dans une région de pixels.

PixelChecksum ( left, top, right, bottom [, step = 1 [, hwnd [, mode = 0]]] )

ParametersParamètres

left left coordinate of rectangle.Abscisse du coin NW du rectangle.
top top coordinate of rectangle.Ordonnée du coin NW du rectangle.
right right coordinate of rectangle.Abscisse du coin SE du rectangle.
bottom bottom coordinate of rectangle.Ordonnée du coin SE du rectangle.
step [optional][optionnel] Instead of checksumming each pixel use a value larger than 1 to skip pixels (for speed). E.g.Par exemple : A value of 2 will only check every other pixel.Une valeur de 2 va seulement vérifier un pixel sur deux. Default is 1.Par défaut c'est 1. It is not recommended to use a step value greater than 1.
hwnd [optional][optionnel] Window handle to be used.Handle de fenêtre à utiliser. Default is the desktop window.Par défaut c'est la fenêtre du bureau.
mode [optional][optionnel] default 0 ADLER checksum, 1 CRC32 Checksum.0 = par défaut, somme de contrôle de ADLER, 1 = somme de contrôle CRC32.

Return ValueValeur de retour

Success:Succès : the checksum value of the region.Retourne la valeur de la somme de contrôle de la région.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

A checksum only allows you to see if "something" has changed in a region - it does not tell you exactly what has changed.Une somme de contrôle vous permet seulement de voir si "quelque chose" a changé dans une région - elle ne vous dit pas exactement ce qui a changé.

CRC32 checksum is a little slower than ADLDER but detects better pixel variation.Le contrôle CRC32 est un peu plus lent que ADLDER mais détecte mieux une variation de pixel.

ExampleExemple

#include <MsgBoxConstants.au3>

; Wait until something changes in the region 0,0 to 50,50Attend jusqu'à ce que quelque chose change dans la région 0,0 à 50,50

; Get initial checksumObtient une somme de contrôle initiale
Local $iCheckSum = PixelChecksum(0, 0, 50, 50)

; Wait for the region to change, the region is checked every 100ms to reduce CPU loadAttend que la région change, la région est contrôlée toutes les 100ms pour réduire la charge du CPU
While $iCheckSum = PixelChecksum(0, 0, 50, 50)
    Sleep(100)
WEnd

MsgBox($MB_SYSTEMMODAL, "", "Something in the region has changed!")
