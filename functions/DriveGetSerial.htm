DriveGetSerial

Returns Serial Number of a drive.Retourne le numéro de série d'un lecteur.

DriveGetSerial ( "path" )

ParametersParamètres

path Path of drive to receive information from.Le chemin sur lequel agir.

Return ValueValeur de retour

Success:Succès : the Serial Number of the drive as a string.Retourne le numéro de série du lecteur sous forme d'une chaîne.
Failure:Échec : sets the @error flag to 1.Retourne 0 et @error = 1.

RemarksRemarques

The value returned is not the hardware serial number as found on the label of the drive, it is the Windows Volume ID for the drive.Aucune.

RelatedLiens

DriveGetDrive, DriveGetFileSystem, DriveGetLabel, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sSerial = DriveGetSerial(@HomeDrive & "\") ; Find the serial number of the home drive, generally this is the C:\ drive.Trouver le numéro de série de l'entraînement à la maison, en général c'est le lecteur C:\\.
MsgBox($MB_SYSTEMMODAL, "", "Serial Number: " & $sSerial)
