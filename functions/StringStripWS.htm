StringStripWS

Strips the white space in a string.Enlève les 'espaces blancs' dans une chaîne.

StringStripWS ( "string", flag )

ParametersParamètres

string The string to strip.La chaîne à traiterer.
flag Flag to indicate the type of stripping that should be performed (add the flags together for multiple operations):Option pour indiquer le type de nettoyage à effectuer.Peut être une combinaison des éléments suivants :
    $STR_STRIPLEADING (1) = strip leading white space
    $STR_STRIPTRAILING (2) = strip trailing white space
    $STR_STRIPSPACES (4) = strip double (or more) spaces between words
    $STR_STRIPALL (8) = strip all spaces (over-rides all other flags)

Constants are defined in StringConstants.au3.

Return ValueValeur de retour

Returns the new string stripped of the requested white space.Retourne la nouvelle chaîne nettoyée des espaces blancs demandés.

RemarksRemarques

Whitespace includes Chr(9) thru Chr(13) which are HorizontalTab, LineFeed, VerticalTab, FormFeed, and CarriageReturn.Les 'espaces blancs' incluent Chr(9) à Chr(13), qui sont HorizontalTab (tabulation horizontale), LineFeed (saut de ligne), VerticalTab (tabulation verticale), FormFeed (saut de page), et CarriageReturn (retour à la ligne). Whitespace also includes the null string ( Chr(0) ) and the standard space ( Chr(32) ).Les 'espaces blancs' comprennent aussi le caractère nul ( Chr(0) ) et l'espace standard ( Chr(32) ).

To strip single spaces between words, use the function StringReplace().Pour enlever les espaces simples entre les mots, utiliser la fonction StringReplace().

RelatedLiens

StringIsSpace, StringReplace, StringStripCR

ExampleExemple

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

; Strip leading and trailing whitespace as well as the double spaces (or more) in between the words.Coupe avant et après les espaces blancs, ainsi que les doubles espaces (ou plus) entre les mots.
Local $sString = StringStripWS("   This is a sentence with whitespace.    ", $STR_STRIPLEADING + $STR_STRIPTRAILING + $STR_STRIPSPACES)
MsgBox($MB_SYSTEMMODAL, "", $sString)
