Référence Fonction

FileSelectFolder

Initiates a Browse For Folder dialog.Lance une boîte de dialogue pour choisir un dossier.

FileSelectFolder ( "dialog text", "root dir" [, flag = 0 [, "initial dir" [, hwnd]]] )

ParametersParamètres

dialog text Text greeting in dialog.Texte du message d'accueil de la boîte de dialogue.
root dir Root directory of GUI file tree.Répertoire racine de l'arborescence de fichiers. It limits a choice in the root folder. Use "" for Desktop to be root.
flag [optional]
    $FSF_CREATEBUTTON (1) = Show Create Folder Button (XP only)
    $FSF_NEWDIALOG (2) = Use New Dialog Style (XP only)
    $FSF_EDITCONTROL (4) = Show Edit Control (XP only)

Constants are defined in FileConstants.au3.
initial dir [optional] The full path of the folder you selected/highlighted when displaying dialogue (if it exists in the root folder). Default is blank ("").Par défaut est vide ("").
hwnd [optional] The window handle to use as the parent for this dialog.Le handle de la fenêtre à utiliser comme fenêtre parent pour ce dialogue.

Return ValueValeur de retour

Success: the full path of the folder chosen.Retourne le chemin complet du dossier choisi.
Failure: "" (empty string) and sets the @error flag to 1 if user cancels/closes the window.Renvoie "" (chaîne vide) et @error = 1 si l'utilisateur annule/ferme la fenêtre.

RemarksRemarques

The root dir will be chosen if the initial dir (if given) does not exist.Le répertoire racine sera choisi si le dossier initial (si donnée) n'existe pas.
A nonexistent root dir will also cause the Desktop folder to be root.Un répertoire racine inexistant entraînera également que le dossier Bureau soit le répertoire racine.
Special Windows folders (such as "My Documents") can be set as root by using the right CLSID detailed in the Appendix.Les dossiers Windows spéciaux (tels que "Mes Documents") peuvent être définis en tant que root en utilisant le bon CLSID détaillée dans l'Appendix.
UNC paths are not supported.Les chemins UNC ne sont pas pris en charge. If you think that user's may choose files on a UNC path then the path needs to be mapped as a drive first.Si vous pensez que l'utilisateur peut avoir besoin de choisir les fichiers sur un chemin UNC, le chemin doit être mappé comme un premier disque.

The new vista+ dialog style is used if possible, regardless of the Use New Dialog Style flag.Le nouveau style de dialogue de Vista+ est utilisé si possible, suivant le flag Use New Dialog Style. With the vista dialog style the edit control and create folder buttons are always shown, ignoring the flags parameter, and the dialog text parameter sets the title of the window.Avec le style de dialogue de Vista le contrôle edit et les boutons de création de dossier sont toujours affichés, ignorant les paramètres flag, et le paramètre texte du dialogue définit le titre de la fenêtre.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the message to display in FileSelectFolder.Crée une constante de portée locale avec le message à afficher dans FileSelectFolder.
    Local Const $sMessage = "Select a folder"

    ; Display an open dialog to select a file.Ouvre une boîte de dialogue pour sélectionner un fichier.
    Local $sFileSelectFolder = FileSelectFolder($sMessage, "")
    If @error Then
        ; Display the error message.Affiche le message d'erreur.
        MsgBox($MB_SYSTEMMODAL, "", "No folder was selected.")
    Else
        ; Display the selected folder.Affiche le dossier sélectionné.
        MsgBox($MB_SYSTEMMODAL, "", "You chose the following folder:" & @CRLF & $sFileSelectFolder)
    EndIf
EndFunc   ;==>Example
