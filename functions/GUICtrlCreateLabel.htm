GUICtrlCreateLabel

Creates a static Label control for the GUI.Crée un contrôle Label dans une GUI.

GUICtrlCreateLabel ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

text The text of the control.Le texte de la commande.
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optionnel] The width of the control (default text autofit in width).La largeur du contrôle (par défaut ajustement automatique du texte en largeur ).
height [optional][optionnel] The height of the control (default text autofit in height).La hauteur du contrôle (par défaut ajustement automatique du texte en hauteur ).
style [optional][optionnel] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default ( -1) : none.
    forced styles : $SS_NOTIFY, $SS_LEFT
exStyle [optional][optionnel] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

To set or change information in the control see GUICtrlUpdate...() functions.Pour définir ou modifier des informations dans le contrôle, consultez les fonctions GUICtrlUpdate...().

To combine styles with the default style use BitOR($GUI_SS_DEFAULT_LABEL, newstyle, ...Pour combiner les styles avec le style par défaut, utilisez BitOR($GUI_SS_DEFAULT_LABEL, newstyle, ... ).)..
To use the values specified above you must #include <StaticConstants.au3> in your script.

Default resizing is $GUI_DOCKAUTO size and position will occur.Pour redimensionner par défaut, utiliez $GUI_DOCKAUTO.

The extended style $GUI_WS_EX_PARENTDRAG can be used to allow the dragging of the parent window for windows that don't have a titlebar (no $WS_CAPTION style in GUICreate()).Le style étendu $GUI_WS_EX_PARENTDRAG peut être utilisé pour permettre le glissement de la fenêtre parente, pour les fenêtres qui n'ont pas de barre de titre (pas de style $WS_CAPTION avec GUICreate()).

To set the background to transparent, use GUICtrlSetBkColor(-1, $GUI_BKCOLOR_TRANSPARENT).Pour définir la transparence, utiliser GUICtrlSetBkColor(-1, $GUI_BKCOLOR_TRANSPARENT)..

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUISetHelp("notepad.exe") ; will run notepad if F1 is typedexécute le bloc-notes si F1 est pressé
    Local $iOldOpt = Opt("GUICoordMode", 2)

    Local $iWidthCell = 70
    GUICtrlCreateLabel("Line 1 Cell 1", 10, 30, $iWidthCell) ; first cell 70 widthla première cellule de 70 de large
    GUICtrlCreateLabel("Line 2 Cell 1", -1, 0) ; next lineligne suivante
    GUICtrlCreateLabel("Line 3 Cell 2", 0, 0) ; next line and next cellligne suivante et cellule suivante
    GUICtrlCreateLabel("Line 3 Cell 3", 0, -1) ; next cell same linecellule suivante même ligne
    GUICtrlCreateLabel("Line 4 Cell 1", -3 * $iWidthCell, 0) ; next line Cell1ligne suivante Cell1

    GUISetState(@SW_SHOW) ; will display an empty dialog boxAffiche une boîte de dialogue vide

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    $iOldOpt = Opt("GUICoordMode", $iOldOpt)
EndFunc   ;==>Example
