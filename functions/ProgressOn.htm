ProgressOn

Creates a customizable progress bar window.Crée une fenêtre de barre de progression personnalisable.

ProgressOn ( "title", "maintext" [, "subtext" [, x pos [, y pos [, opt]]]] )

ParametersParamètres

title Title for progress window.Titre de la fenêtre de progression.
maintext Text for Main, Bold, Upper label.Texte pour Main, Bold, Upper label.
subtext [optional] text for Sub, Normal, Lower label.texte pour Sub, Normal, Lower label. (default is blank)(par défaut est vide)
x pos [optional] position from left (in pixels) of progress window. (default is centered)(centrée par défaut)
y pos [optional] position from top (in pixels) of progress window. (default is centered)(centrée par défaut)
opt [optional] Default is 'always on top/with title'Par défaut: toujours Au Premier Plan et avec Titre
Add up the following options you want:Ajoute les options que vous souhaitez.
    $DLG_NOTITLE (1) = borderless, titleless window
    $DLG_NOTONTOP (2) = Without "always on top" attribute
    $DLG_MOVEABLE (16) = Window can be moved

Constants are defined in AutoItConstants.au3

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

To skip an optional parameter, leaving its default value intact, use:Pour sauter un paramètre optionnel, en laissant sa valeur par défaut intacte, utilisez:
    "" for string parameters
    -1 for numeric parameters
If ProgressOn() is called while a progress window already exists, the window is redrawn/moved.Si ProgressOn() est appelée alors qu'une fenêtre de progression existe déjà, la fenêtre est redessinée/déplacé. (If you want multiple simultaneous progress windows, you need to use multiple scripts.)(Si vous voulez plusieurs fenêtres de progression simultanées, vous devez utiliser plusieurs scripts.)

RelatedLiens

ProgressOff, ProgressSet

ExampleExemple

Example()

Func Example()
    ; Display a progress bar window.Affiche une fenêtre de barre de progression.
    ProgressOn("Progress Meter", "Increments every second", "0%")

    ; Update the progress value of the progress bar window every second.Met à jour la valeur de la progression de la fenêtre de la barre de progression à chaque seconde.
    For $i = 10 To 100 Step 10
        Sleep(1000)
        ProgressSet($i, $i & "%")
    Next

    ; Set the "subtext" and "maintext" of the progress bar window.Régle le "sous-texte" et "MainText" de la fenêtre de la barre de progression.
    ProgressSet(100, "Done", "Complete")
    Sleep(5000)

    ; Close the progress window.Ferme la fenêtre de progression.
    ProgressOff()
EndFunc   ;==>Example
