GUICtrlSetGraphic

Modifies the data for a control.Modifie les données d'un contrôle.

GUICtrlSetGraphic ( controlID, type [, par1 [, ... par6]] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreateGraphic() function.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreateGraphic().
type type of drawing : dot, line, bezier, rect, ellipse, pie.Type de dessin : point, ligne, Bézier, rectangle, ellipse, arc de cercle.
par1...par6 See the Graphic Type table below.Voir la Table des types graphiques ci-dessous.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.
-1 in case of invalid data-1 En cas de données non valides

RemarksRemarques

The point position (x,y) is relative to the GUICtrlCreateGraphic() coordinates.La position du point (x, y) est relative aux coordonnées GUICtrlCreateGraphic(). It can be outside the graphic control but inside of the GUI window.Il peut être en dehors du contrôle graphique mais à l'intérieur de la fenêtre GUI.

Func Example()
    Local $idMsg, $iInc, $i

    GUICreate("My Main", -1, -1, 100, 100)
    Local $idDel1 = GUICtrlCreateButton("ReCreate", 50, 200, 50)
    GUISetState(@SW_SHOW)
    CreateChild()

    $i = 1
    $iInc = 1
    ;$i=5 ; uncomment to delete starting from last define Graphic control$i = 5 ; décommenter pour supprimer le démarrage à partir du dernier contrôle graphique défini
    ;$iInc=-1$iInc = -1

    Do
        $idMsg = GUIGetMsg()
        If $idMsg = $idDel1 Then $i = Create($iInc)

        If $idMsg = $g_idDel Then
            GUICtrlDelete($g_aidGraphics[$i])
            $i = $i + $iInc
            If $i < 0 Or $i > $g_MAXGr Then Exit
        EndIf
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

Func Create($iInc)
    GUIDelete($g_hChild)
    CreateChild()
    If $iInc = -1 Then Return 5
    Return 1
EndFunc   ;==>Create
