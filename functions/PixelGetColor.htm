PixelGetColor

Returns a pixel color according to x,y pixel coordinates.Retourne la couleur d'un pixel de coordonnées données.

PixelGetColor ( x , y [, hwnd] )

ParametersParamètres

x x coordinate of pixel.Abscisse du pixel.
y y coordinate of pixel.Ordonnée du pixel.
hwnd [optional][optionnel] Window handle to be used.Handle de fenêtre à utiliser. Default is the desktop window.Par défaut c'est la fenêtre du bureau.

Return ValueValeur de retour

Success:Succès : a decimal value of pixel's color.Retourne la valeur décimale de la couleur du pixel.
Failure:Échec : -1 if invalid coordinates.Retourne -1 si les coordonnées sont invalides.

RelatedLiens

MouseGetPos, PixelChecksum, PixelCoordMode (Option), PixelSearch

ExampleExemple

#include <MsgBoxConstants.au3>

Local $iColor = PixelGetColor(10, 100)

MsgBox($MB_SYSTEMMODAL, "", "The decimal color is: " & $iColor)
MsgBox($MB_SYSTEMMODAL, "", "The hex color is: " & Hex($iColor, 6))
