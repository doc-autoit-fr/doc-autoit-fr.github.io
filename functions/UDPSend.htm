Référence Fonction

UDPSend

Sends data on an opened socket.Envoie des données sur un socket ouvert.

UDPSend ( socketarray, data )

ParametersParamètres

socketarray The main socket/array as returned by a UDPOpen() function.
data binary/string to be sent to the connected socketBinaire ou chaine qui doit être envoyé au socket connecté.

Return ValueValeur de retour

Success: the number of bytes sent to the opened socket.Retourne le nombre d'octets envoyés au socket ouvert.
Failure: sets the @error flag to non-zero.met @error <> 0.
@error:@error : -3 or -4 invalid socketarray.
1 - IPAddr is incorrect.1 = Le paramètre IPAddr est incorrect.
2 - port is incorrect.2 = Le port est incorrect.
Windows API WSAGetError return value (see MSDN).Valeur retournée par l'API Windows WSAGetError (Consultez MSDN).

RelatedLiens

UDPBind, UDPOpen, UDPRecv

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

; Start First clicking on "1.Commencez d'abord par cliquer sur "1. Server"Serveur"
; Then start a second instance of the script selecting "2.Puis démarrez une seconde instance du script en sélectionnant "2. Client"Client\"

Example()

Func Example()
    UDPStartup() ; Start the UDP service.Démarre le service UDP.

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

    ; Assign Local variables the loopback IP Address and the Port.Assigne des variables locales avec l'adresse IP et le port.
    Local $sIPAddress = "127.0.0.1" ; This IP Address only works for testing on your own computer.Cette adresse IP ne fonctionne que pour les tests sur votre propre ordinateur.
    Local $iPort = 65532 ; Port used for the connection.Port utilisé pour la connexion.

    #Region GUI
    Local $sTitle = "UDP Start"
    Local $hGUI = GUICreate($sTitle, 250, 70)

    Local $idBtnServer = GUICtrlCreateButton("1.Retourne 1 si la fenêtre existe. Server", 65, 10, 130, 22)

    Local $idBtnClient = GUICtrlCreateButton("2. Client", 65, 40, 130, 22)

    GUISetState(@SW_SHOW, $hGUI)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idBtnServer
                WinSetTitle($sTitle, "", "UDP Server started")
                GUICtrlSetState($idBtnClient, $GUI_HIDE)
                GUICtrlSetState($idBtnServer, $GUI_DISABLE)
                If Not MyUDP_Server($sIPAddress, $iPort) Then ExitLoop
            Case $idBtnClient
                WinSetTitle($sTitle, "", "UDP Client started")
                GUICtrlSetState($idBtnServer, $GUI_HIDE)
                GUICtrlSetState($idBtnClient, $GUI_DISABLE)
                If Not MyUDP_Client($sIPAddress, $iPort) Then ExitLoop
        EndSwitch

        Sleep(10)
    WEnd

    #EndRegion GUI
EndFunc   ;==>Example

Func MyUDP_Server($sIPAddress, $iPort)
    ; Assign a Local variable the socket and bind to the IP Address and Port specified.Assigne une variable locale avec le socket et se connecte à l'adresse IP et au port spécifié.
    Local $iSocket = UDPBind($sIPAddress, $iPort)

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        ; Someone is probably already binded on this IP Address and Port (script already running?).Quelqu'un est probablement déjà connecté sur cette adresse IP et ce port (script déjà en cours d"exécution?).
        Local $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Server:" & @CRLF & "Could not bind, Error code: " & $iError)
        Return False
    EndIf

    ; Assign a Local variable to store the data received.Assigne une variable locale pour stocker les données reçues.
    Local $sReceived = ""

    Do
        ; We are waiting for the string "toto" OR "tata" (example script UDPSend):Nous attendons la chaîne "toto" OU "tata" (exemple de script UDPSend) : 4 bytes length.4 octets de longueur.
        $sReceived = UDPRecv($iSocket, 4)
    Until $sReceived <> ""

    ; Notes: If you don't know how much length will be the data,Si vous ne savez pas de quelle longueur seront les données,
    ; use e.g:utilisez, par exemple: 2048 for maxlen parameter and call the function until the it returns nothing/error.2048 pour paramètre maxlen et appelez la fonction jusqu'à ce qu'elle ne retourne rien ou erreur.

    ; Display the string received.Affiche la chaîne reçue.
    MsgBox($MB_SYSTEMMODAL, "", "Server:" & @CRLF & "Received: " & $sReceived)

    ; Close the socket.Ferme le socket.
    UDPCloseSocket($iSocket)
EndFunc   ;==>MyUDP_Server

Func MyUDP_Client($sIPAddress, $iPort)
    ; Assign a Local variable the socket and connect to a listening socket with the IP Address and Port specified.Assigne une variable locale avec le socket et se connecte à un socket d'écoute avec l'adresse IP et le port spécifié.
    Local $iSocket = UDPOpen($sIPAddress, $iPort)
    Local $iError = 0

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        ; The server is probably offline/port is not opened on the server.Le serveur est probablement hors ligne ou le port n'est pas ouvert sur le serveur.
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Client:" & @CRLF & "Could not connect, Error code: " & $iError)
        Return False
    EndIf

    ; Send the string "toto" converted to binary to the server.Envoie la chaîne "toto" convertie en binaire sur le serveur.
    UDPSend($iSocket, StringToBinary("toto"))

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Client:" & @CRLF & "Could not send the data, Error code: " & $iError)
        Return False
    EndIf

    ; Close the socket.Ferme le socket.
    UDPCloseSocket($iSocket)
EndFunc   ;==>MyUDP_Client

Func OnAutoItExit()
    UDPShutdown() ; Close the UDP service.Ferme le service UDP.
EndFunc   ;==>OnAutoItExit
