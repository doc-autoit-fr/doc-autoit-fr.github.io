Return ValueValeur de retour

RemarksRemarques

The return value from TimerInit() should be treated as an opaque handle and should only be used to pass to TimerDiff().La valeur retournnée par TimerInit() devrait être traiter comme un handle opaque et devrait seulement être utilisé pour le passer à TimerDiff(). Any other usage of the return value is a potential error.D'autres utilisations de la valeur de retour conduisent à des erreurs.
