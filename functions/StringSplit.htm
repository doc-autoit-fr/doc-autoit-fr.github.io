StringSplit

Splits up a string into substrings depending on the given delimiters.Découpe une chaîne en tableau de sous-chaînes selon des délimiteurs donnés.

StringSplit ( "string", "delimiters" [, flag = 0] )

ParametersParamètre

string The string to evaluate.La chaîne à traiter.
delimiters One or more characters to use as delimiters (case sensitive).Un ou plusieurs caractères à utiliser comme délimiteurs (sensible à la casse).
flag [optional][optional] False - i, v, x ... Changes how the string split works, add multiple flag values together if required:Modifie le fonctionnement de la fonction, ajoutez plusieurs valeurs de flag ensemble si besoin:
    $STR_CHRSPLIT (0) = each character in the delimiter string will mark where to split the string (default)
    $STR_ENTIRESPLIT (1) = entire delimiter string is needed to mark the split
    $STR_NOCOUNT (2) = disable the return count in the first element - effectively makes the array 0-based (must use UBound() to get the size of the array in this case).

Constants are defined in StringConstants.au3.

Return ValueRemarque

Returns an array, by default the first element ($aArray[0]) contains the number of strings returned, the remaining elements ($aArray[1], $aArray[2], etc.) contain the delimited strings.Retourne un tableau, par défaut le premier élément $array[0] contient le nombre de chaînes retournées, les éléments qui restent $array[1], $array[2], etc.contiennent les chaînes découpées. If the flag parameter is set to $STR_NOCOUNT (2) then the count is not returned in the first element.Si le paramètre 'flag' est défini à $STR_NOCOUNT (2), alors le comptage n'est pas retourné dans le premier élément.

If no delimiters were found then the @error flag is set to 1:
If flag is not equal to $STR_NOCOUNT, $aArray[0] is set to 1 and the full string is returned in $aArray[1].
If flag is equal to $STR_NOCOUNT, the full string is returned in $aArray[0].

Remarks<i0>

If you use an empty string "" for the delimiters, each character will be returned as an element.Si vous utilisez une chaîne vide "" comme délimiteur, chaque caractère sera retourné comme un élément.

If the delimiter string contains several characters function behaviour depends on the flag setting.Si la chaîne des délimiteurs contient plusieurs caractères, le comportement de la fonction dépend du paramètre flag. If set to $STR_CHRSPLIT the string will be split at every instance of each of the individual characters in the delimiter - if set to $STR_ENTIRESPLIT then the string will only be split when the entire delimiter string is encountered.S'il est défini à $STR_CHRSPLIT la chaîne sera coupée à chaque instance de chaque caractère delimiteur - s'il est défini à $STR_ENTIRESPLIT alors la chaîne sera coupée seulement quand la chaîne entière des délimiteurs est rencontrée. See second example below.Voir le second exemple ci-dessous.

Note that the macro @CRLF is actually a 2 character string, so unless the flag parameter to $STR_ENTIRESPLIT extra blank lines will be generated as the string is split on both @CR and @LF.Remarquez que la macro @CRLF est actuellement une chaîne de 2 caractères, aussi si le paramètre flag est défini à $STR_ENTIRESPLIT des lignes vides seront générées puisque la chaîne sera coupée à la fois par @CR et @LF.

StringSplit() is very useful as an alternative to StringInStr() as well as a means to populate an array.StringSplit() est une très bonne alternative à StringInStr() et comme moyen de remplissage d'un tableau.

Example 1Exemple 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aDays = StringSplit("Mon,Tues,Wed,Thur,Fri,Sat,Sun", ",") ; Split the string of days using the delimiter "," and the default flag value.Diviser la chaîne de jour en utilisant le délimiteur \",\" et la valeur de l'indicateur de défaut.
    #cs
        The array returned will contain the following values:
        $aDays[1] = "Mon"
        $aDays[2] = "Tues"
        $aDays[3] = "Wed"
        ...
        $aDays[7] = "Sun"
    #ce

    For $i = 1 To $aDays[0] ; Loop through the array returned by StringSplit to display the individual values.Boucle dans le tableau renvoyé par StringSplit pour afficher les valeurs individuelles.
        MsgBox($MB_SYSTEMMODAL, "", "$aDays[" & $i & "] - " & $aDays[$i])
    Next
EndFunc   ;==>Example

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

Example()

Func Example()
    Local $sText = "This\nline\ncontains\nC-style breaks." ; Define a variable with a string of text.Définir une variable avec une chaîne de texte.
    Local $aArray = StringSplit($sText, '\n', $STR_ENTIRESPLIT) ; Pass the variable to StringSplit and using the delimiter "\n".Passer la variable à StringSplit et en utilisant le séparateur \"\\ n\".
    ; Note that flag parameter is set to $STR_ENTIRESPLIT (1) as it would split at \ or n otherwise.Notez que le paramètre de drapeau est défini sur $STR_ENTIRESPLIT (1) comme il serait divisé à \\ n ou autrement.
    #cs
        The array returned will contain the following values:
        $aArray[1] = "This"
        $aArray[2] = "line"
        ...
        $aArray[4] = "C-style breaks."
    #ce

    For $i = 1 To $aArray[0] ; Loop through the array returned by StringSplit to display the individual values.Boucle dans le tableau renvoyé par StringSplit pour afficher les valeurs individuelles.
        MsgBox($MB_SYSTEMMODAL, "", "$aArray[" & $i & "] - " & $aArray[$i])
    Next
EndFunc   ;==>Example
