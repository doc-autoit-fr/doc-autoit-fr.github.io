DirGetSize

Returns the size in bytes of a given directory.Retourne la taille en octets d'un répertoire donné.

DirGetSize ( "path" [, flag = 0] )

ParametersParamètres

path The directory path to get the size from, e.g.Le chemin du répertoire dont on veut obtenir la taille. "C:\Windows".Ex :"C:\Windows".
flag [optional] this flag determines the behaviour and result of the function, and can be a combination of the following:Ce paramètre détermine le comportement et le résultat de la fonction et peut être une combinaison de ce qui suit :
    $DIR_DEFAULT (0) = (default)
    $DIR_EXTENDED (1) = Extended mode is On -> returns an array that contains extended information (see Remarks).
    $DIR_NORECURSE (2) = Don't get the size of files in subdirectories (recursive mode is Off)

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Success: the sizes that are greater than or equal to zeroRetourne la taille >=0.
Failure: -1 and sets the @error flag to non-zero if the path doesn't exist.

RemarksRemarques

If the script is paused then this function is paused too and will only continue when the script continues!Si le script est mis en pause alors cette fonction est mise en pause également et elle continuera quand le script sera reprit.

If you use the extended mode then the array returned from this function is a single dimension array containing the following elements:Si vous utilisez le mode étendu, alors le tableau retourné par cette fonction est un tableau à 1 dimension contenant les éléments suivants :
$aArray[0] = Size$array[0] = Taille
$aArray[1] = Files count$array[1] = Nombre de fichiers
$aArray[2] = Dirs Count$array[2] = Nombre de répertoires

ExampleExemple

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iSize = DirGetSize(@HomeDrive)
    MsgBox($MB_SYSTEMMODAL, "", "Size(MegaBytes): " & Round($iSize / 1024 / 1024))

    $iSize = DirGetSize(@WindowsDir, $DIR_NORECURSE)
    MsgBox($MB_SYSTEMMODAL, "", "Size(MegaBytes): " & Round($iSize / 1024 / 1024))

    Local $hTimer = TimerInit()

    Local $aSize = DirGetSize("\\10.0.0.1\h$", $DIR_EXTENDED) ; extended modemode étendu
    If Not @error Then
        Local $iDiff = Round(TimerDiff($hTimer) / 1000) ; time in secondstemps en secondes
        MsgBox($MB_SYSTEMMODAL, "", "Size(Bytes): " & $aSize[0] & @CRLF _
                 & "Files: " & $aSize[1] & @CRLF & "Dirs: " & $aSize[2] & @CRLF _
                 & "TimeDiff(Sec): " & $iDiff)
    EndIf
EndFunc   ;==>Example
