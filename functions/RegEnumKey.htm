RegEnumKey

Reads the name of a subkey according to its instance.Lit le nom d'une sous-clé dont l'instance est donnée.

RegEnumKey ( "keyname", instance )

ParametersParamètres

keyname The registry key to read.La clé de registre à lire.
instance The 1-based key instance to retrieveL'instance, à partir de 1, à récupérer

Return ValueValeur de retour

Success:Succès : the requested subkey name.Retourne le nom de la sous-clé demandée.
Failure:Échec : an empty string and sets the @error flag to non-zero.Retourne une chaîne vide et @error <> 0.
@error:@error : 1 = unable to open requested key1 = impossible d'ouvrir la clé demandée
 2 = unable to open requested main key
 3 = unable to remote connect to the registry
-1 = unable to retrieve requested subkey (key instance out of range)-1 = impossible de récupérer la sous-clé demandée (instance de la clé en dehors de la plage)

RemarksRemarques

A registry key must start with "HKEY_LOCAL_MACHINE" ("HKLM") or "HKEY_USERS" ("HKU") or "HKEY_CURRENT_USER" ("HKCU") or "HKEY_CLASSES_ROOT" ("HKCR") or "HKEY_CURRENT_CONFIG" ("HKCC").Une clé de registre doit commencer par "HKEY_LOCAL_MACHINE" ("HKLM") ou "HKEY_USERS" ("HKU") ou "HKEY_CURRENT_USER" ("HKCU") ou "HKEY_CLASSES_ROOT" ("HKCR") ou "HKEY_CURRENT_CONFIG" ("HKCC") .

When running on 64-bit Windows if you want to enum a key specific to the 64-bit environment you have to suffix the HK... with 64 i.e.Lors de l'exécution sur Windows 64 bits si vous voulez lire une clé spécifique à l'environnement 64 bits, vous devez suffixer HK... HKLM64.avec 64 c'est à dire HKLM64.

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sSubKey = ""
For $i = 1 To 10
    $sSubKey = RegEnumKey("HKEY_LOCAL_MACHINE\SOFTWARE", $i)
    If @error Then ExitLoop
    MsgBox($MB_SYSTEMMODAL, "SubKey #" & $i & " under HKLM\SOFTWARE: ", $sSubKey)
Next
