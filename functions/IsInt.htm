IsInt

Checks if the value of a variable or expression has no fractional component.Vérifie si la valeur d'une variable ou d'une expression n'a pas de partie fractionnaire.

IsInt ( variable )

ParametersParamètres

variable The variable/expression to check.La variable/expression à vérifier.

Return ValueValeur de retour

Success:Succès : 1 - No fractional componentRetourne 1 - Aucune partie fractionnaire
Failure:Échec : 0 - Fractional componentRetourne 0 - une partie fractionnaire existe

RemarksRemarques

The function will return 1 if the value is a float with no fractional component (e.g. 1.000).
See language datatypes for a detailed description.Consultez Types de données du langage pour une description détaillée.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $bIsInt1 = IsInt(-12345) ; Returns 1Retourne 1
    Local $bIsInt2 = IsInt(3.0000) ; Returns 1Retourne 1
    Local $bIsInt3 = IsInt("5432") ; Returns 0 as the value is a string.Retourne 0 car la valeur est une chaîne.
    Local $bIsInt4 = IsInt(7.5 - 4.5) ; Returns 1 since it evaluates to integer 3Retourne 1 car il teste l'entier 3

    MsgBox($MB_SYSTEMMODAL, "", "IsInt: " & @CRLF & _
            $bIsInt1 & @CRLF & $bIsInt2 & @CRLF & $bIsInt3 & @CRLF & $bIsInt4)
EndFunc   ;==>Example
