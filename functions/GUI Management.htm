GUI functions ReferenceGUI - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

See Gui Constants include files if you need to use the related controls Constants .Voir Fichiers 'include' des constantes GUI si vous avez besoin d'utiliser des constantes relatives aux contrôles.

 

FunctionFonction Description
GUICreate Create a GUI window.Crée une fenêtre GUI.
GUICtrlGetHandle Returns the handle for a control and some special (item) handles (Menu, ContextMenu, TreeViewItem).Retourne le handle d'un contrôle et certains handles spéciaux (Menu, ContextMenu, TreeViewItem).
GUICtrlGetState Gets the current state of a control.Obtient l'état actuel d'un contrôle.
GUICtrlRead Read state or data of a control.Lit l'état ou les données d'un contrôle.
GUICtrlRecvMsg Send a message to a control and retrieve information in lParam.Envoie un message à un contrôle et retourne des informations dans lParam.
GUICtrlSendMsg Send a message to a control.Envoie un message à un contrôle.
GUICtrlSendToDummy Sends a message to a Dummy control.Envoie un message à un contrôle factice.
GUIDelete Deletes a GUI window and all controls that it contains.Supprime une fenêtre GUI et tous les contrôles qu'elle contient.
GUIGetCursorInfo Gets the mouse cursor position relative to GUI window.Obtient la position du curseur de la souris par rapport à la fenêtre GUI.
GUIGetMsg Polls the GUI to see if any events have occurred.Sonde la GUI pour voir si des événements ont été déclenchés.
GUIGetStyle Retrieves the styles of a GUI window.Récupère les styles d'une fenêtre GUI.
GUIRegisterMsg Register a user defined function for a known Windows Message ID (WM_MSG).Enregistre une fonction utilisateur pour un ID de message Windows connu (WM_MSG).
GUIStartGroup Defines that any subsequent controls that are created will be "grouped" together.Détermine que les contrôles ultérieurs qui seront créés seront "groupés" ensemble.
GUISwitch Switches the current window used for GUI functions.Commute la fenêtre courante utilisée pour les fonctions de la GUI.
