GUICtrlSetResizing

Defines the resizing method used by a control.Définit la méthode de redimensionnement utilisée par un contrôle.

GUICtrlSetResizing ( controlID, resizing )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
resizing See the Docking Values table below for values that can be used (add together multiple values if required).Voir la Tableau des valeurs d'Ancrage ci-dessous pour les valeurs qui peuvent être utilisées (additionner plusieurs valeurs si nécessaire).

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

When a GUI window is resized the controls within react - how they react is determined by this function.Quand une fenêtre GUI est redimensionnée, ses contrôles réagissent - la façon dont ils réagissent est déterminée par cette fonction. To be able to resize a GUI window it needs to have been created with the $WS_SIZEBOX and $WS_SYSMENU styles.Pour être en mesure de redimensionner une fenêtre GUI, elle doit avoir été créée avec les styles $WS_SIZEBOX et $WS_SYSMENU. See GUICreate().Consultez GUICreate().

The default resizing for a given control is control dependent see the control doc.La valeur par défaut de redimensionnement pour un contrôle donné est $GUI_DOCKAUTO, voir la documentation sur les contrôles.
A default value for any control can be set with GUIResizeMode (Option).Une valeur par défaut pour tous les contrôles peut être définie avec GUIResizeMode (Option).

The automatic resizing event can be disabled if GUIEventOptions (Option) is set to 1.L'événement de redimensionnement automatique peut être désactivée si GUIEventOptions (Option) est mis à 1.

ExampleExemple

#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Example()

Func Example()
    Opt("GUICoordMode", 2)
    GUICreate("My InputBox", 190, 114, -1, -1, $WS_SIZEBOX + $WS_SYSMENU) ; start the definitioncommence la définition

    GUISetFont(8, -1, "Arial")

    GUICtrlCreateLabel("Prompt", 8, 7) ; add prompt infoajoute des informations rapides
    GUICtrlSetResizing(-1, $GUI_DOCKLEFT + $GUI_DOCKTOP)

    Local $idEdit = GUICtrlCreateInput("Default", -1, 3, 175, 20, $ES_PASSWORD) ; add the input areaajoute une zone de saisie
    GUICtrlSetState($idEdit, $GUI_FOCUS)
    GUICtrlSetResizing($idEdit, $GUI_DOCKBOTTOM + $GUI_DOCKHEIGHT)

    Local $idOK = GUICtrlCreateButton("OK", -1, 3, 75, 24) ; add the button that will close the GUIajoute le bouton qui fermera la GUI
    GUICtrlSetResizing($idOK, $GUI_DOCKBOTTOM + $GUI_DOCKSIZE + $GUI_DOCKHCENTER)

    Local $idCancel = GUICtrlCreateButton("Annuler", 25, -1) ; add the button that will close the GUIajoute le bouton qui fermera la GUI
    GUICtrlSetResizing($idCancel, $GUI_DOCKBOTTOM + $GUI_DOCKSIZE + $GUI_DOCKHCENTER)

    GUISetState(@SW_SHOW) ; to display the GUIpour afficher l'interface graphique

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
