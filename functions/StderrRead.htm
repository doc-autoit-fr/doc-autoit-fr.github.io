StderrRead

Reads from the STDERR stream of a previously run child process.Lit dans le flux STDERR d'un processus enfant lancé au préalable.

StderrRead ( process_id [, peek = False [, binary = False]] )

ParametersParamètres

process_id The process ID of a child process, as returned by a previous call to Run.L'ID du processus enfant, tel que retourné par un appel précédent à Run.
peek [optional][optionnel] If True the function does not remove the read characters from the stream.Si vrai, la fonction ne supprime pas les caractères lus à partir du flux.
binary [optional][optionnel] If True the function reads the data as binary instead of text (default is text).Si vrai, la fonction lit les données de manière binaire au lieu de texte (par défaut c'est texte).

Return ValueValeur de retour

Success:Succès : the data read.Retourne les données lues. @extended contains the number of bytes read.@extended contient le nombre d'octets lus.
Failure:Échec : sets the @error flag to non-zero if EOF is reached, STDERR was not redirected for the process or other error.Retourne @error <> 0 si EOF est atteint, STDERR n'a pas été redirigé pour le processus ou une autre erreur.

RemarksRemarques

StderrRead() reads from the console standard output stream of a child process, which is normally used by console applications to write to the screen.StderrRead() lit à partir du flux de sortie standard de la console du processus enfant, qui est normalement utilisé par les applications console pour écrire sur l'écran. During the call to Run() for the child process you wish to read from the STD I/O parameter must have included the value of $STDERR_CHILD (4) for this function to work properly (see the Run() function).Durant l'appel de Run() pour le processus enfant, si vous souhaitez lire à partir du STD I/O, le paramètre doit avoir pris en compte la valeur $STDERR_CHILD (4) sinon cette fonction ne tavaillera pas correctement (voir la fonction Run()).
StderrRead() does not block, it will return immediately.StderrRead() ne bloque pas, il retournera immédiatement. In order to get all data, it must be called in a loop.Afin d'obtenir toutes les données, elle doit être appelée dans une boucle.
Peeking on the stream does not remove the data from the buffer, however, it does return the available data as normal.Une lecture sur le flux ne supprime pas les données de la mémoire tampon, cependant, elle renvoie les données disponibles comme d'habitude.
By default, data is returned in text format.Par défaut, les données sont retournées au format texte. By using the binary option, the data will be returned in binary format.En utilisant l'option binaire, les données seront renvoyées au format binaire.

ExampleExemple

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iPID = Run(@ComSpec & " /c DIR Example.au3", @SystemDir, @SW_HIDE, $STDERR_CHILD + $STDOUT_CHILD)
    Local $sOutput = ""
    While 1
        $sOutput = StdoutRead($iPID)
        If @error Then ; Exit the loop if the process closes or StdoutRead returns an error.Sort de la boucle si le processus ferme ou StdoutRead renvoie une erreur.
            ExitLoop
        EndIf
        MsgBox($MB_SYSTEMMODAL, "Stdout Read:", $sOutput)
    WEnd

    While 1
        $sOutput = StderrRead($iPID)
        If @error Then ; Exit the loop if the process closes or StderrRead returns an error.Sort de la boucle si le processus ferme ou si StderrRead renvoie une erreur.
            ExitLoop
        EndIf
        MsgBox($MB_SYSTEMMODAL, "Stderr Read:", $sOutput)
    WEnd
EndFunc   ;==>Example
