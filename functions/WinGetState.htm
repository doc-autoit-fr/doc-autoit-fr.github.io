WinGetState

Retrieves the state of a given window.Récupère l'état d'une fenêtre donnée.

WinGetState ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to get the state.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the window to get the state.Le texte de la fenêtre à traiter. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : a value indicating the state of the window.Retourne une valeur indiquant l'état de la fenêtre. Multiple values are added together so use BitAND() to examine the part you are interested in:Plusieurs valeurs peuvent être additionnées, donc utilisez BitAND() pour examiner la partie qui vous intéresse:
$WIN_STATE_EXISTS (1) = Window exists
$WIN_STATE_VISIBLE (2) = Window is visible
$WIN_STATE_ENABLED (4) = Window is enabled
$WIN_STATE_ACTIVE (8) = Window is active
$WIN_STATE_MINIMIZED (16) = Window is minimized
$WIN_STATE_MAXIMIZED (32) = Window is maximized
Failure:Échec : 0 and sets the @error flag to non-zero if the window is not found.Retourne 0 et @error <> 0 si la fenêtre n'est pas trouvée.

Constants are defined in "AutoItConstants.au3".

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Retrieve the state of the Notepad window using the handle returned by WinWait.Récupére l'état de la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    Local $iState = WinGetState($hWnd)

    ; Check if the Notepad window is minimized and display the appropriate message box.Teste si la fenêtre du Bloc-notes est minimisée et affiche la boîte de message appropriée.
    If BitAND($iState, 16) Then
        MsgBox($MB_SYSTEMMODAL, "", "Notepad is minimized and the state returned by WinGetState was - " & $iState)
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Notepad isn't minimized and the state returned by WinGetState was - " & $iState)
    EndIf

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
