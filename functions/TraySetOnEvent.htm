TraySetOnEvent

Defines a user function to be called when a special tray action happens.Définit une fonction utilisateur qui sera appelée lors d'une action spéciale dans la zone de notification.

TraySetOnEvent ( specialID, "function" )

ParametersParamètres

specialID See the Special ID table below.Voir le tableau des IDs Spéciaux ci-dessous.
function The name of the user function to call.Le nom de la fonction utilisateur à appeler.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.
@error:@error : 1 = the "function" is not defined.1 = la "fonction" n'est pas définie.

RemarksRemarques

OnEvent functions are only called when Opt("TrayOnEventMode", 1) - when in this mode TrayGetMsg() is NOT used at all.

Example()

Func Example()
    TrayCreateItem("About")

    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    TrayCreateItem("Exit")
    TrayItemSetOnEvent(-1, "ExitScript")

    TraySetOnEvent($TRAY_EVENT_PRIMARYDOUBLE, "TrayEvent")
    TraySetOnEvent($TRAY_EVENT_SECONDARYUP, "TrayEvent")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Sleep(100) ; An idle loop.Une boucle d'inactivité.
    WEnd
EndFunc   ;==>Example

Func TrayEvent()
    Switch @TRAY_ID ; Check the last tray item identifier.Teste le dernier identifiant de l'item.
        Case $TRAY_EVENT_PRIMARYDOUBLE
            ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
            MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                    "Version: " & @AutoItVersion & @CRLF & _
                    "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

        Case $TRAY_EVENT_SECONDARYUP
            MsgBox($MB_SYSTEMMODAL, "", "The secondary mouse button was released on the tray icon.Le bouton secondaire de la souris a été relaché sur l'icône de la zone de notification.")

    EndSwitch
EndFunc   ;==>TrayEvent

Func ExitScript()
    Exit
EndFunc   ;==>ExitScript
