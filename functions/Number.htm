Référence Fonction

Number

Returns the numeric representation of an expression.Retourne la représentation numérique d'une expression.

Number ( expression [, flag = 0] )

ParametersParamètre

expression An expression to convert into a number.Une expression à convertir en nombre.
flag [optional][optional] False - i, v, x ... Defines behavior.Définit le comportement.
Can be one of the following:Peut être un des suivants:
    $NUMBER_AUTO (0) = (default) the result is auto-sized integer. See remarks.Voir remarques.
    $NUMBER_32BIT (1) = the result is 32bit integer.
    $NUMBER_64BIT (2) = the result is 64bit integer.
    $NUMBER_DOUBLE (3) = the result is double.

Constants are defined in "AutoItConstants.au3".

Return ValueRemarque

Returns a number.Retourne un nombre.

Remarks<i0>

If you pass number, function outputs the input.Si vous passez un nombre à la fonction, elle retourne ce nombre.
A string beginning with letters has a numeric value of zero.Une chaîne commençant par des lettres a une valeur numérique de zéro. A string beginning with digits has non-numeric characters stripped at the from the first one. Valid characters for Double numbers does not start the stripping.
Default behavior results in auto-sized integer.Le comportement par défaut donne un résultat convertit automatiquement dans le type 'integer'. If input value after conversion is within the range of 32bit integer the result is 32bit integer.Si la valeur d'entrée après conversion est dans l'intervalle des entiers 32bit, le result est un 'integer' 32bit. If not the result is 64bit integer.Sinon, le résultat est un integer 64bit. Both are signed.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <MsgBoxConstants.au3>

Local $bNumber1 = Number(1 + 2 + 10) ; Returns 13.Retourne 13.
Local $bNumber2 = Number("3.14") ; Returns 3.14.Retourne 3.14.
Local $bNumber3 = Number("24/7") ; Returns 24.Retourne 24.
Local $bNumber4 = Number("tmp3") ; Returns 0 as this is a string.Retourne 0 car c'est une chaîne.
Local $bNumber5 = Number("1,000,000") ; Returns 1 as it strips everything after and including the first comma.Retourne 1 car il supprime tout ce qui suit après la première virgule (le séparateur décimal est le point!).

MsgBox($MB_SYSTEMMODAL, "", "The following values were converted to a numeric value:" & @CRLF & _
        $bNumber1 & @CRLF & $bNumber2 & @CRLF & $bNumber3 & @CRLF & $bNumber4 & @CRLF & $bNumber5)
