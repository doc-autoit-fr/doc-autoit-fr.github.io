GUIGetMsg

Polls the GUI to see if any events have occurred.Sonde la GUI pour voir si des événements ont été déclenchés.

GUIGetMsg ( [advanced = 0] )

ParametersParamètres

advanced [optional] return extended information in an array.retour d'informations complémentaires dans un tableau.
    $GUI_EVENT_SINGLE (0) = (default) Returns a single event.
    $GUI_EVENT_ARRAY (1) = returns an array containing the event and extended information.

Constants are defined in GUIConstantsEx.au3.

Return ValueValeur de retour

Returns an event, or an array depending on the "advanced" parameter.Retourne un événement, ou un tableau selon le paramètre "advanced".

The "event" returned is the control ID of the control sending the message, or it is a special event (like the window is closing, minimizing).L'événement retourné est l'ID du contrôle qui envoie le message, ou un événement spécial (comme la fermeture de la fenêtre, fenêtre minimisé). Or if there is no message, the event is 0.Ou s'il n'y a pas de message, l'événement est égal à 0.


Event ID: the ID of the control sending the messageL'ID du contrôle qui a envoyé le message
$GUI_EVENT_NONE (0): No eventAucun évènement
$GUI_EVENT_CLOSE: dialog box being closed (either by defined button or system menu).boîte de dialogue fermée (par bouton défini ou par menu système ).
$GUI_EVENT_MINIMIZE: dialog box minimized with Windows title bar button.boîte de dialogue minimisée avec le menu système.
$GUI_EVENT_RESTORE: dialog box restored by click on task bar icon.boîte de dialogue restaurée par clic sur l'icône de la barre de tâche.
$GUI_EVENT_MAXIMIZE: dialog box maximized with Windows title bar button.boîte de dialogue maximisée avec le menu système.
$GUI_EVENT_MOUSEMOVE: the mouse cursor has moved.le curseur de la souris s'est déplacé.
$GUI_EVENT_PRIMARYDOWN: the primary mouse button was pressed.le bouton principal de la souris a été enfoncé.
$GUI_EVENT_PRIMARYUP: the primary mouse button was released.le bouton principal de la souris a été relâché.
$GUI_EVENT_SECONDARYDOWN: the secondary mouse button was pressed.le bouton secondaire de la souris a été enfoncé.
$GUI_EVENT_SECONDARYUP: the secondary mouse button was released.le bouton secondaire de la souris a été relâché.
$GUI_EVENT_RESIZED: dialog box has been resized.boîte de dialogue redimensionnée.
$GUI_EVENT_DROPPED: End of a Drag&Drop action @GUI_DragId, @GUI_DragFile and @GUI_DropId will be used to retrieve the ID's/file corresponding to the involve control.Fin d'une action glisser-déposer.@GUI_DragId, @GUI_DragFile et @GUI_DropId seront utilisés pour récupérer de l'ID/fichier du contrôle impliqué.

Constants are defined in GUIConstantsEx.au3.


When using the "advanced" parameter the information is returned in an array with extended information:Lorsque vous utilisez le paramètre "avancé" les informations sont retournées dans un tableau avec des informations complémentaires:

$aArray[0] = 0 or Event ID or Control ID$aArray[0] = 0 ou l'ID de l'événement ou l'ID du contrôle
$aArray[1] = The window handle the event is from$aArray[1] = Le handle de la fenêtre où l'événement s'est déclenché
$aArray[2] = The control handle the event is from (if applicable)$aArray[2] = Le handle du contrôle où l'événement s'est déclenché (le cas échéant)
$aArray[3] = The current X position of the mouse cursor (relative to the GUI window)$aArray[3] = L'abscisse X du curseur de la souris (relative à la fenêtre GUI)
$aArray[4] = The current Y position of the mouse cursor (relative to the GUI window)$aArray[4] = L'ordonnée Y du curseur de la souris (relative à la fenêtre GUI)


If the GUIOnEventMode option is set to 1 then the return from GUIGetMsg is always 0 and the @error is set to 1.Si l'option de GUIOnEventMode est réglé à 1, alors le retour de GUIGetMsg est toujours 0 et @error est mis à 1.

If the option GUIEventOptions is set to 1 the minimize, restore and maximize button will not do any action on the window just a simple notification.Si l'option GUIEventOptions est à 1, les boutons 'minimize', 'restaure' et 'maximize' n'auront aucun effer sur la fenêtre, juste une simple notification.

RemarksRemarques

This function automatically idles the CPU when required so that it can be safely used in tight loops without hogging all the CPU.Cette fonction ralentit automatiquement lorsque le CPU en a besoin, de façon qu'elle puisse être utilisée en toute sécurité dans des boucles courtes sans monopoliser tout le CPU.

Information about the mouse position and the hovering control can be retrieved with GUIGetCursorInfo().Les informations sur la position de la souris et sur le contrôle survolé peuvent être récupérés avec GUIGetCursorInfo(). No event is fired when the mouse is over a control so GUIGetCursorInfo() must be called to retrieve the controlID.Aucun événement n'est déclenché lorsque la souris est sur un contrôle donc GUIGetCursorInfo() doit être appelée pour récupérer le controlID.

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

; --------------------------------------------------------------------------------------------------------------------------------------- -----------------------------------
; Example - Press the button to see the value of the radio boxesExemple - Appuyez sur le bouton pour voir la valeur des boutons radio
; The script also detects state changes (closed/minimized/timeouts, etc).Le script détecte également les changements d'état (fermé/minimisé/délai d'attente, etc).
Func Example()
    Opt("GUICoordMode", 1)

    GUICreate("Radio Box Demo", 400, 280)

    ; Create the controlsCrée les contrôles
    Local $idButton_1 = GUICtrlCreateButton("B&utton 1", 30, 20, 120, 40)
    GUICtrlCreateGroup("Group 1", 30, 90, 165, 160)
    GUIStartGroup()
    Local $idRadio_1 = GUICtrlCreateRadio("Radio &0", 50, 120, 70, 20)
    GUICtrlCreateRadio("Radio &1", 50, 150, 60, 20)
    Local $idRadio_3 = GUICtrlCreateRadio("Radio &2", 50, 180, 60, 20)

    ; Init our vars that we will use to keep track of GUI eventsInitialise les variables que nous allons utiliser pour garder une trace des événements de la GUI
    Local $iRadioVal1 = 0 ; We will assume 0 = first radio button selected, 2 = last buttonNous supposerons 0 = premier RadioButton sélectionné, 2 = dernier RadioButton

    ; Show the GUIAffiche la GUI
    GUISetState(@SW_SHOW)

    Local $idMsg = 0
    ; In this message loop we use variables to keep track of changes to the radios, anotherDans cette boucle de message nous utilisons des variables pour garder une trace des changements apportés aux radios, une autre
    ; way would be to use GUICtrlRead() at the end to read in the state of each controlfaçon serait d'utiliser GUICtrlRead() à la fin de la lecture de l'état de chaque contrôle
    While 1
        $idMsg = GUIGetMsg()
        Select
            Case $idMsg = $GUI_EVENT_CLOSE
                MsgBox($MB_SYSTEMMODAL, "", "Dialog was closed")
                ExitLoop
            Case $idMsg = $GUI_EVENT_MINIMIZE
                MsgBox($MB_SYSTEMMODAL, "", "Dialog minimized", 2)
            Case $idMsg = $GUI_EVENT_MAXIMIZE
                MsgBox($MB_SYSTEMMODAL, "", "Dialog restored", 2)

            Case $idMsg = $idButton_1
                MsgBox($MB_SYSTEMMODAL, "", "Default button clicked:" & @CRLF & "Radio " & $iRadioVal1)

            Case $idMsg >= $idRadio_1 And $idMsg <= $idRadio_3
                $iRadioVal1 = $idMsg - $idRadio_1

        EndSelect
    WEnd

    GUIDelete()
EndFunc   ;==>Example

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Create a GUI.Crée une interface graphique.
    Local $hGUI1 = GUICreate("Example GUI1")
    ; Create a button.Crée un bouton.
    Local $idButton1 = GUICtrlCreateButton("Button1", 10, 10, 80, 22)
    ; Display the GUIAffiche la GUI
    GUISetState(@SW_SHOW, $hGUI1)

    ; Create a GUI.Crée une interface graphique.
    Local $hGUI2 = GUICreate("Example GUI2", 300, 300)
    ; Create a button.Crée un bouton.
    Local $idButton2 = GUICtrlCreateButton("Button2", 10, 10, 80, 22)
    ; Display the GUIAffiche la GUI
    GUISetState(@SW_SHOW, $hGUI2)

    ; Initialize a Local variable.Initialise une variable locale.
    Local $aMsg = 0

    While 1
        ; Assign to $aMsg the advanced GUI messages.Assigne à $aMsg les messages GUI avancés.
        $aMsg = GUIGetMsg($GUI_EVENT_ARRAY)

        ; Switch from GUIsSelon la GUI
        Switch $aMsg[1]
            Case $hGUI1
                ; The event comes from the GUI1L'événement vient de la GUI1

                ; Switch from event IDSelon l'ID de l'événement
                Switch $aMsg[0]
                    Case $GUI_EVENT_CLOSE
                        ExitLoop
                    Case $idButton1
                        MsgBox($MB_SYSTEMMODAL, "", "Button1 clicked.")
                EndSwitch
            Case $hGUI2
                ; The event comes from the GUI2L'événement vient de la GUI2

                ; Switch from event IDSelon l'ID de l'événement
                Switch $aMsg[0]
                    Case $GUI_EVENT_CLOSE
                        GUIDelete($hGUI2)
                    Case $idButton2
                        MsgBox($MB_SYSTEMMODAL, "", "Button2 clicked.")
                EndSwitch
        EndSwitch
    WEnd

    ; Delete the previous GUIs and all controls.Supprime les GUI précédentes et tous les contrôles.
    GUIDelete($hGUI1)
EndFunc   ;==>Example
