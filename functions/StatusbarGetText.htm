StatusbarGetText

Retrieves the text from a standard status bar control.Récupère le texte d'un contrôle de barre d'état standard.

StatusbarGetText ( "title" [, "text" [, part = 1]] )

ParametersParamètres

title The title/hWnd/class of the window to check.Le titre/hWnd/class de la fenêtre à traiter.
text [optional][optionnel] The text of the window to check.Le texte de la fenêtre à contrôler. Default is an empty string.La valeur par défaut est une chaîne vide.
part [optional][optionnel] The "part" number of the status bar to read - the default is 1.Le numéro de la "partie" de la barre d'état à lire - la valeur par défaut est 1. 1 is the first possible part and usually the one that contains the useful messages like "Ready" "Loading...", etc.C'est la première partie possible et le plus souvent celle qui contient les messages utiles comme "Ready ", "Loading", etc...

Return ValueValeur de retour

Success:Succès : the text read.Retourne le texte lu.
Failure:Échec : "" (empty string) and sets the @error flag to 1 if no text could be read.Retourne "" (Chaîne vide ) et @error = 1 si aucun texte n'a pu être lu.

RemarksRemarques

This functions attempts to read the first standard status bar on a window (Microsoft common control: msctls_statusbar32).Cette fonction tente de lire la première partie d'une barre d'état standard dans une fenêtre (contrôle commun Microsoft: msctls_statusbar32 ). Some programs use their own status bars or special versions of the MS common control which StatusbarGetText() cannot read.Certains programmes utilisent leurs propres barres d'état ou des versions spéciales du contrôle commun MS que StatusbarGetText() ne peut pas lire. For example, StatusbarGetText does not work on the program TextPad; however, the first region of TextPad's status bar can be read using ControlGetText("TextPad", "", "HSStatusBar1")Par exemple, StatusbarGetText ne fonctionne pas avec le programme TextPad; cependant la première région de la barre d'état de TextPad peut être lue en utilisant ControlGetText("TextPad ", "", "HSStatusBar1")
StatusbarGetText() can work on windows that are minimized or even hidden.StatusbarGetText() peut travailler sur les fenêtres qui sont minimisées ou même cachées.

#include <MsgBoxConstants.au3>
