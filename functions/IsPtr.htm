IsPtr

Checks if a variable's base type is a pointer.Vérifie si le type de base d'une variable est un pointeur.

IsPtr ( variable )

ParametersParamètres

variable The variable/expression to check.La variable/expression à vérifier.

Return ValueValeur de retour

Success:Succès : 1 if the expression is a pointer type.Retourne 1 si l'expression est du type de pointeur.
Failure:Échec : 0 if expression is not pointer type.Retourne 0 si l'expression n'est pas du type pointeur.

RemarksRemarques

See language datatypes for a detailed description.Consultez Types de données du langage pour une description détaillée.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Test if $hWnd is a valid pointer and display the results.Teste si $hWnd est un pointeur valide et affiche les résultats.
    If IsPtr($hWnd) Then
        MsgBox($MB_SYSTEMMODAL, "", "It's a valid Ptr")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "It's not a valid Ptr")
    EndIf

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
