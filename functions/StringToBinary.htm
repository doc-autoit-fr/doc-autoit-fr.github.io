StringToBinary

Converts a string into binary data.Convertit une chaîne en données binaires.

StringToBinary ( expression [, flag = 1] )

ParametersParamètres

expression An expression to convert into binary data.Une expression à convertir en données binaires.
flag [optional][optionnel] Changes how the string is stored as binary:Change la méthode de conversion binaire :
    $SB_ANSI (1) = string data is ANSI (default)
    $SB_UTF16LE (2) = string data is UTF16 Little Endian
    $SB_UTF16BE (3) = string data is UTF16 Big Endian
    $SB_UTF8 (4) = string data is UTF8

Constants are defined in StringConstants.au3.

Return ValueValeur de retour

Returns a binary variant.Retourne une variable de type Variant binaire.

RemarksRemarques

See "Unicode Support" for a detailed description.Voir "Support Unicode" pour une description détaillée.

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Define the string that will be converted later.Définit la chaîne qui sera convertie plus tard.
    ; NOTE:NOTE : This string may show up as ??Cette chaîne peut apparaître comme ?? in the help file and even in some editors.dans un fichier d'aide et même dans certains éditeurs.
    ; This example is saved as UTF-8 with BOM.Cet exemple est enregistré au format UTF -8 avec BOM. It should display correctly in editorsIl doit s'afficher correctement dans les éditeurs
    ; which support changing code pages based on BOMs.qui prennent en charge la modification des pages de code basées sur les BOMs.
    Local Const $sString = "Hello - 你好"

    ; Temporary variables used to store conversion results.Les variables temporaires utilisées pour stocker les résultats de conversion. $dBinary will hold$dBinary contiendra
    ; the original string in binary form and $sConverted will hold the resultla chaîne d'origine sous forme binaire et $sConverted contiendra le résultat
    ; afte it's been transformed back to the original format.ensuite elle est transformée dans le format original.
    Local $dBinary = Binary(""), $sConverted = ""

    ; Convert the original UTF-8 string to an ANSI compatible binary string.Convertit la chaîne d'origine UTF-8 pour une chaîne binaire compatible ANSI.
    $dBinary = StringToBinary($sString)

    ; Convert the ANSI compatible binary string back into a string.Convertit la chaîne binaire compatible ANSI en une chaîne.
    $sConverted = BinaryToString($dBinary)

    ; Display the resulsts.Affiche les résultats. Note that the last two characters will appearNotez que les deux derniers caractères s'affichent
    ; as ??comme ?? since they cannot be represented in ANSI.car ils ne peuvent pas être représentés dans la norme ANSI.
    DisplayResults($sString, $dBinary, $sConverted, "ANSI")

    ; Convert the original UTF-8 string to an UTF16-LE binary string.Convertit la chaîne d'origine UTF-8 en une chaîne binaire UTF16-LE.
    $dBinary = StringToBinary($sString, $SB_UTF16LE)

    ; Convert the UTF16-LE binary string back into a string.Convertit la chaîne binaire UTF16-LE en une chaîne.
    $sConverted = BinaryToString($dBinary, $SB_UTF16LE)

    ; Display the resulsts.Affiche les résultats.
    DisplayResults($sString, $dBinary, $sConverted, "UTF16-LE")

    ; Convert the original UTF-8 string to an UTF16-BE binary string.Convertit la chaîne d'origine UTF-8 en une chaîne binaire UTF16-BE.
    $dBinary = StringToBinary($sString, $SB_UTF16BE)

    ; Convert the UTF16-BE binary string back into a string.Convertit la chaîne binaire UTF16-BEen une chaîne.
    $sConverted = BinaryToString($dBinary, $SB_UTF16BE)

    ; Display the resulsts.Affiche les résultats.
    DisplayResults($sString, $dBinary, $sConverted, "UTF16-BE")

    ; Convert the original UTF-8 string to an UTF-8 binary string.Convertit la chaîne d'origine UTF-8 en une chaîne UTF-8 binaire.
    $dBinary = StringToBinary($sString, $SB_UTF8)

    ; Convert the UTF8 binary string back into a string.Convertit la chaîne binaire UTF8 en une chaîne.
    $sConverted = BinaryToString($dBinary, $SB_UTF8)

    ; Display the resulsts.Affiche les résultats.
    DisplayResults($sString, $dBinary, $sConverted, "UTF8")
EndFunc   ;==>Example

; Helper function which formats the message for display.Fonction d'aide qui formate le message pour l'affichage. It takes the following parameters:Elle prend les paramètres suivants :
; $sOriginal - The original string before conversions.$sOriginal - La chaîne d'origine avant la conversion.
; $dBinary - The original string after it has been converted to binary.$dBinary - La chaîne d'origine après qu'elle ait été convertie en binaire.
; $sConverted- The string after it has been converted to binary and then back to a string.$sConverted- La chaîne après qu'elle ait été convertie en binaire et puis de nouveau en une chaîne.
; $sConversionType - A human friendly name for the encoding type used for the conversion.$sConversionType - Un nom convivial humain pour le type de codage utilisé pour la conversion.
Func DisplayResults($sOriginal, $dBinary, $sConverted, $sConversionType)
    MsgBox($MB_SYSTEMMODAL, "", "Original:" & @CRLF & $sOriginal & @CRLF & @CRLF & "Binary:" & @CRLF & $dBinary & @CRLF & @CRLF & $sConversionType & ":" & @CRLF & $sConverted)
EndFunc   ;==>DisplayResults
