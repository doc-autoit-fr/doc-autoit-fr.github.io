IniReadSection

Reads all key/value pairs from a section in a standard format .ini file.Lit toutes les paires clé/valeur d'une section d'un fichier .ini au format standard.

IniReadSection ( "filename", "section" )

ParametersParamètres

filename The filename of the .ini file.Le nom du fichier ini.
section The section name in the .ini file.Le nom de la section dans le fichier ini.

Return ValueValeur de retour

Success: a 2 dimensional array where element[n][0] is the key and element[n][1] is the value.Retourne un tableau à 2 dimensions où l'élément [n][0] est la clé et l'élément [n][1] est la valeur.
Failure: sets the @error flag to non-zero if unable to read the section (The INI file may not exist or the section may not exist or is empty)Retourne @error <> 0 si incapable de lire la section (Le fichier INI peut ne pas exister ou la section peut ne pas exister ou est vide )

RemarksRemarques

A standard ini file looks like:Un fichier ini standard ressemble à :
[SectionName][NomdelaSection]
Key=Valueclé=Valeur

The number of elements returned will be in $aArray[0][0].Le nombre d'éléments retournés sera dans $aArray[0][0]. If an @error occurs, no array is created.Si un @error se produit, aucun tableau n'est créé.
    $aArray[0][0] = Number
    $aArray[1][0] = 1st Key
    $aArray[1][1] = 1st Value
    $aArray[2][0] = 2nd Key
    $aArray[2][1] = 2nd Value
    ...
    $aArray[n][0] = nth Key
    $aArray[n][1] = nth Value

Only the first 32767 chars are read for legacy reasons.Seuls les 32767 premiers caractères sont lus pour des raisons historiques.

ExampleExemple

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create an INI section structure as a string.Crée une structure de section INI sous forme de chaîne.
    Local $sSection = "Title=AutoIt" & @LF & "Version=" & @AutoItVersion & @LF & "OS=" & @OSVersion

    ; Write the string to the section labelled 'General'.Ecrit la chaîne à la section "General".
    IniWriteSection($sFilePath, "General", $sSection)

    ; Read the INI section labelled 'General'.Lit la section INI 'General'. This will return a 2 dimensional array.Ce qui renvoie un tableau à 2 dimensions.
    Local $aArray = IniReadSection($sFilePath, "General")

    ; Check if an error occurred.Vérifie si une erreur s'est produite.
    If Not @error Then
        ; Enumerate through the array displaying the keys and their respective values.Énumére le tableau pour afficher les clés et leurs valeurs respectives.
        For $i = 1 To $aArray[0][0]
            MsgBox($MB_SYSTEMMODAL, "", "Key: " & $aArray[$i][0] & @CRLF & "Value: " & $aArray[$i][1])
        Next
    EndIf

    ; Delete the INI file.Supprime le fichier INI.
    FileDelete($sFilePath)
EndFunc   ;==>Example
