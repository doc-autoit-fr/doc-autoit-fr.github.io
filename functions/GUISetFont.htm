GUISetFont

Sets the default font for a GUI window.Définit la police par défaut pour une fenêtre GUI.

GUISetFont ( size [, weight [, attribute [, fontname [, winhandle [, quality]]]]] )

ParametersParamètres

size Fontsize (default is 8.5).Taille de la police (par défaut 8.5).
weight [optional][optionnel] The weight of the font in the range 0 through 1000.La graisse de la police de 0 à 1000. For example, 400 is normal and 700 is bold.Par exemple, 400 est normal et 700 est gras. If this value is zero, a default weight is used.Si cette valeur est 0, une graisse par défaut est utilisée.
The following values are defined for convenience.Les valeurs suivantes sont définies pour en faciliter l'utilisation.
    $FW_DONTCARE = 0 (Use the default font weight)
    $FW_THIN = 100
    $FW_EXTRALIGHT = 200
    $FW_LIGHT = 300
    $FW_NORMAL = 400
    $FW_MEDIUM = 500
    $FW_SEMIBOLD = 600
    $FW_BOLD = 700
    $FW_EXTRABOLD = 800
    $FW_HEAVY = 900

Constants are defined in FontConstants.au3.
attribute [optional][optionnel] Font attributes, which can be a combination of the following added together:Attributs de police, qui peut être une combinaison des éléments suivants additionnés :
    $GUI_FONTNORMAL (0) = Normal (Default)
    $GUI_FONTITALIC (2) = Italic
    $GUI_FONTUNDER (4) = Underlined
    $GUI_FONTSTRIKE (8) = Strike

Constants are defined in GUIConstantsEx.au3.
fontname [optional][optionnel] Font to use.Nom de la police à utiliser. (OS default GUI font is used if the font is "" or is not found).(police par défaut de l'OS utilisée si la police est "" ou introuvable).
winhandle [optional][optionnel] Windows handle as returned by GUICreate() (default is the previously used window).Handle Windows retourné par GUICreate() (par défaut, la fenêtre précédemment utilisée ).
quality [optional][optionnel] Font quality to select.Qualité de la police à sélectionner. The following qualities are accepted:Les qualités suivantes sont acceptées:
    $DEFAULT_QUALITY (0) = Appearance of the font does not matter (Default).
    $DRAFT_QUALITY (1) = Appearance of the font is less important than when $PROOF_QUALITY is used. For GDI raster fonts, scaling is enabled, which means that more font sizes are available, but the quality may be lower. Bold, italic, underline, and strikeout fonts are synthesized if necessary.Les polices gras, italique, souligné, et barré sont synthétisés si nécessaire.
    $PROOF_QUALITY (2) = (default) Character quality of the font is more important than exact matching of the logical-font attributes. For GDI raster fonts, scaling is disabled and the font closest in size is chosen.Pour les polices raster GDI, la mise à l'échelle est désactivée et la police la plus proche en taille est choisie. Although the chosen font size may not be mapped exactly when $PROOF_QUALITY is used, the quality of the font is high and there is no distortion of appearance.Bien que la taille de la police choisie ne peut pas être mappé exactement quand $PROOF_QUALITY est utilisé, la qualité de la police est élevé et il n'y a pas de distorsion de l'apparence. Bold, italic, underline, and strikeout fonts are synthesized if necessary.Les polices gras, italique, souligné, et barré sont synthétisés si nécessaire.
    $NONANTIALIASED_QUALITY (3) = Font is never anti aliased.
    $ANTIALIASED_QUALITY (4) = Font is always anti aliased if the font supports it and the size of the font is not too small or too large.
    $CLEARTYPE_QUALITY (5) = If set, text is rendered (when possible) using ClearType anti aliasing method. See the remarks on the MSDN page for LOGFONT for details about when ClearType is not available.Voir Remarques sur la page MSDN pour LOGFONT pour savoir quand ClearType n'est pas disponible.
If neither $ANTIALIASED_QUALITY nor $NONANTIALIASED_QUALITY is selected, the font is anti aliased only if the user chooses smooth screen fonts in Control Panel.Si ni $ANTIALIASED_QUALITY ni $NONANTIALIASED_QUALITY n'est sélectionné, la police est anti-crénelée seulement si l'utilisateur choisit les polices d'écran lisses dans le Panneau de configuration.
Constants are defined in FontConstants.au3Les constantes sont définies dans FontConstants.au3

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

See the Appendix for a complete list of Windows fonts and the Windows versions under which they are supported.Consultez l'Appendice pour une liste complète des polices Windows et les versions de Windows sous lesquelles elles sont supportées.

Size can contain a decimal as in 8.5.La taille peut contenir un nombre décimal comme 8.5.

For some control as label ones the default can be 8.5 instead of 9 according to Windows Theme Values.Pour certains contrôles tels que les Label, la taille par défaut peut être de 8,5 au lieu de 9 en accord avec les valeurs du thème de Windows.

For Quality parameter check MSDN, some Windows XP installation need CLEARTYPE_QUALITY=5Pour le paramètre Quality consultez MSDN, certaines installations de Windows XP nécessite CLEARTYPE_QUALITY = 5

ExampleExemple

#include <GUIConstantsEx.au3>
#include <FontConstants.au3>

Example()

Func Example()
    GUICreate("My GUI default font") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    Local $sFont = "Comic Sans MS"
    GUISetFont(9,  $FW_NORMAL, $GUI_FONTUNDER, $sFont) ; will display underlined charactersAffiche caractères barrés
    GUICtrlCreateLabel("underlined label", 10, 20)

    GUISetFont(9,  $FW_NORMAL, $GUI_FONTITALIC, $sFont) ; will display underlined charactersAffiche caractères barrés
    GUICtrlCreateLabel("italic label", 10, 40)

    GUISetFont(9,  $FW_NORMAL, $GUI_FONTSTRIKE, $sFont) ; will display underlined charactersAffiche caractères barrés
    GUICtrlCreateLabel("strike label", 10, 60)

    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example
