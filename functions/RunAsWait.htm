RunAsWait

Runs an external program under the context of a different user and pauses script execution until the program finishes.Exécute un programme externe dans le contexte d'un autre utilisateur et interrompt l'exécution du script jusqu'à ce que le programme se termine.

RunAsWait ( "username", "domain", "password", logon_flag, "program" [, "workingdir" [, show_flag [, opt_flag]]] )

ParametersParamètres

username The username to log on with.Le nom d'utilisateur pour se connecter avec.
domain The domain to authenticate against.Le domaine pour s'authentifier.
password The password for the user.Le mot de passe de l'utilisateur.
logon_flag     $RUN_LOGON_NOPROFILE (0) - Interactive logon with no profile.
    $RUN_LOGON_PROFILE (1) - Interactive logon with profile.
    $RUN_LOGON_NETWORK (2) - Network credentials only.
    $RUN_LOGON_INHERIT (4) - Inherit the calling process's environment instead of the user's environment.

Constants are defined in "AutoItConstants.au3".
program The full path of the program (EXE, BAT, COM, or PIF) to run (see remarks).Le chemin d'accès complet du programme (EXE, BAT, COM, ou PIF) pour exécuter (Voir Remarques).
workingdir [optional][optionnel] The working directory.Le répertoire de travail. If not specified, then the value of @SystemDir will be used.S'il n'est pas spécifié, la valeur de SystemDir sera utilisée. This is not the path to the program.Ce n'est pas le chemin du programme.
show_flag [optional][optionnel] The "show" flag of the executed program:L'option de visualisation du programme exécuté:
    @SW_HIDE = Hidden window (or Default keyword)
    @SW_MINIMIZE = Minimized window
    @SW_MAXIMIZE = Maximized window
opt_flag [optional][optionnel] Controls various options related to how the parent and child process interact.Options diverses liées à la façon dont les processus parent-enfant inter-agissent.
    $RUN_CREATE_NEW_CONSOLE (0x10000) = The child console process should be created with its own window instead of using the parent's window. This flag is only useful when the parent is compiled as a Console application.Ce drapeau n'est utile que lorsque le parent est compilé comme une application console.

Constant is defined in AutoItConstants.au3

Return ValueValeur de retour

Success:Succès : the exit code of the program that was run.Retourne le code de sortie du programme qui a été exécuté.
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

Paths with spaces need to be enclosed in quotation marks.Les chemins avec des espaces doivent être placés entre guillemets.

It is important to specify a working directory the user you are running as has access to, otherwise the function will fail.Il est important de spécifier un répertoire de travail accessible par l'utilisateur sur lequel vous exécutez le programme, sinon la fonction échouera.

It is recommended that you only load the user's profile is you are sure you need it.Il est recommandé que vous ne chargiez le profil de l'utilisateur que si vous êtes sûr que vous en avez besoin. There is a small chance a profile can be stuck in memory under the right conditions.Il ya une petite chance qu'un profil soit bloqué en mémoire dans de bonnes conditions. If a script using RunAs() happens to be running as the SYSTEM account (for example, if the script is running as a service) and the user's profile is loaded, then you must take care that the script remains running until the child process closes.Si un script qui exécute RunAs() arrive à être exécuté sous le compte SYSTEM (par exemple, si le script est exécuté en tant que service) et que le profil de l'utilisateur est chargé, vous devez veiller à ce que le script continue de s'exécuter jusqu'à ce que le processus enfant se termine.

When running as an administrator, the Secondary Logon (RunAs()) service must be enabled or this function will fail.Lors de l'exécution en tant qu'administrateur, l'ouverture de session secondaire ( RunAs()) Services doit être activée ou cette fonction échouera. This does not apply when running as the SYSTEM account.Ceci ne s'applique pas lors de l'exécution sous le compte SYSTEM.

After running the requested program the script pauses until the program terminates.Après avoir exécuté le programme demandé le script se met en pause jusqu'à la fin du programme. To run a program and then immediately continue script execution use the RunAs() function instead.Pour exécuter un programme puis continuer immédiatement l'exécution de script utilise la fonction RunAs() à la place.

Some programs will appear to return immediately even though they are still running; these programs spawn another process - you may be able to use the ProcessWaitClose() function to handle these cases.Certains programmes semble retourner immédiatement, même si elles sont encore en cours d'exécution; ces programmes engendrent un autre processus - vous pourriez être en mesure d'utiliser la fonction ProcessWaitClose() pour traiter ces cas.

The "load profile" and "network credentials only" options are incompatible.Les options "charge un profil" et "informations réseau seules" sont incompatibles. Using both will produce undefined results.Les utiliser ensemble provoquera des résultats indéfinis.

There is an issue in the Windows XP generation of Windows which prevents STDIO redirection and the show flag from working.Il y a un problème avec la génération des Windows XP qui empêche la redirection STDIO et le drapeau "mode d'affichage" de fonctionner. See Microsoft Knowledge Base article KB818858 for more information about which versions are affected as well as a hotfix for the issue.Consultez l'article de la Base de connaissances Microsoft KB818858 pour plus d'informations sur les versions concernées ainsi que d'un correctif pour ce problème. Users running Windows XP SP2 or later, or Windows Vista or later are not affected.Les utilisateurs de Windows XP SP2 ou plus, ou Windows Vista ou plus ne sont pas affectés.

ExampleExemple

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Change the username and password to the appropriate values for your system.Changez le nom d'utilisateur et le mot de passe pour des valeurs appropriées à votre système.
    Local $sUserName = "Username"
    Local $sPassword = "Password"

    ; Run Notepad and wait for the Notepad process to close.Lance le Bloc-notes et attend que le processus du Bloc-notes se ferme. Notepad is run under the user previously specified.Le Bloc-notes est exécuté sous l'utilisateur spécifié.
    Local $iReturn = RunAsWait($sUserName, @ComputerName, $sPassword, $RUN_LOGON_NOPROFILE, "notepad.exe")

    ; Display the return code of the Notepad process.Affiche le code de retour du processus du Bloc-notes.
    MsgBox($MB_SYSTEMMODAL, "", "The return code from Notepad was: " & $iReturn)
EndFunc   ;==>Example
