ConsoleWriteError

Writes data to the STDERR stream.Écrit des données dans le flux STDERR. Some text editors can read this stream as can other programs which may be expecting data on this stream.Certains éditeurs de texte peuvent lire ce flux comme d'autres programmes peuvent attendre les données de ce flux.

ConsoleWriteError ( "data" )

ParametersParamètres

data The data you wish to output.Les données que vous voulez sortir. This may either be text or binary.Cela peut être du texte ou du binaire.

Return ValueValeur de retour

The amount of data written.Retourne la quantité de données écrites. If writing binary, the number of bytes written, if writing text, the number of characters written.Si l'écriture est binaire, se sera le nombre d'octets écrits, si l'écriture est du texte, alors ce sera le nombre de caractères écrits.

RemarksRemarques

The purpose for this function is to write to the STDERR stream.Le but de cette fonction est d'écrire dans le flux STDERR. Many popular text editors can read this stream.De nombreux éditeurs texte peuvent lire ce flux. Scripts compiled as Console applications also have a STDERR stream.Les scripts compilés comme applications console ont aussi un flux STDERR.

This does not write to a DOS console unless the script is compiled as a console application.Cette fonction ne peut pas écrire dans une console DOS sauf si le script est compilé en tant qu'application console.

Characters are converted to ANSI before being written.Les caractères sont convertis à la norme ANSI avant d'être écrits.

Binary data is written as-is.Les données binaires sont écrites en l'état. It will not be converted to a string.Elles ne seront pas converties en une chaîne. To print the hex representation of binary data, use the String() function to explicitly cast the data to a string.Pour visualiser la représentation hexadécimale de données binaires, utilisez la fonction String() pour convertir explicitement les données en chaîne.

RelatedLiens

ConsoleRead, ConsoleWrite

ExampleExemple

Local $sString = "An error occurred."
ConsoleWriteError($sString & @CRLF) ; Running this in a text editor which can trap console output e.g. SciTE, will display value of $sString.
