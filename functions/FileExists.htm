FileExists

Checks if a file or directory exists.Vérifie si un fichier ou un dossier existe.

FileExists ( "path" )

ParametersParamètres

Path The directory or file to check.Chemin du répertoire ou du fichier à vérifier.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if path/file does not exist.Retourne 0 si le fichier ou le répertoire n'existe pas.

RemarksRemarques

FileExists() returns 0 if you specify a floppy drive which does not contain a disk.FileExists() retourne 0 si vous spécifiez un lecteur qui ne contient pas de média.

RelatedLiens

DriveStatus, FileGetAttrib

ExampleExemple

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    Local $iFileExists = FileExists($sFilePath)

    ; Display a message of whether the file exists or not.Affiche un message si le fichier existe ou pas.
    If $iFileExists Then
        MsgBox($MB_SYSTEMMODAL, "", "The file exists." & @CRLF & "FileExist returned: " & $iFileExists)
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The file doesn't exist." & @CRLF & "FileExist returned: " & $iFileExists)
    EndIf

    ; Delete the temporary file.Supprimez le fichier temporaire.
    FileDelete($sFilePath)
EndFunc   ;==>Example
