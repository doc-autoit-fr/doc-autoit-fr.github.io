Controls functions ReferenceControls - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
ControlClick Sends a mouse click command to a given control.Envoie un clic de souris sur un contrôle spécifié.
ControlCommand Sends a command to a control.Envoie une commande à un contrôle.
ControlDisable Disables or "grays-out" a control.Désactive ou met en grisé un contrôle.
ControlEnable Enables a "grayed-out" control.Active ou dégrise un contrôle.
ControlFocus Sets input focus to a given control on a window.Donne le focus à un contrôle.
ControlGetFocus Returns the ControlRef# of the control that has keyboard focus within a specified window.Retourne la référence du contrôle qui a le focus clavier dans une fenêtre donnée.
ControlGetHandle Retrieves the internal handle of a control.Retourne le handle d'un contrôle.
ControlGetPos Retrieves the position and size of a control relative to its window.Retourne la position et la taille d'un contrôle.
ControlGetText Retrieves text from a control.Retourne le texte d'un contrôle.
ControlHide Hides a control.Cache un contrôle.
ControlListView Sends a command to a ListView32 control.Envoie une commande à un contrôle ListView32.
ControlMove Moves a control within a window.Déplace un contrôle dans une fenêtre.
ControlSend Sends a string of characters to a control.Envoie une chaîne de caractère à un contrôle.
ControlSetText Sets text of a control.Modifie le texte d'un contrôle.
ControlShow Shows a control that was hidden.Affiche un contrôle précédemment caché.
ControlTreeView Sends a command to a TreeView32 control.Envoie une commande à un contrôle TreeView32.
StatusbarGetText Retrieves the text from a standard status bar control.Récupère le texte d'un contrôle de barre d'état standard.
