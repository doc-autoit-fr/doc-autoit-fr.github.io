WinActivate

Activates (gives focus to) a window.Active (donne le focus) à une fenêtre.

WinActivate ( "title" [, "text"] )

ParametersParamètre

title The title/hWnd/class of the window to activate.Le titre/hWnd/class de la fenêtre à activer. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optional] False - i, v, x ... The text of the window to activate.Le texte de la fenêtre à activer. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueRemarque

Success: the handle of the window.Retourne le handle de la fenêtre.
Failure: 0 if window is not found or cannot be activated.Retourne 0 si la fenêtre n'est pas trouvée ou ne peut pas être activée.

Remarks<i0>

You can use the WinActive() function to check if WinActivate() succeeded.Vous pouvez utiliser la fonction WinActive() pour vérifier si WinActivate() a réussi. If multiple windows match the criteria, the window that was most recently active is the one activated.Si plusieurs fenêtres correspondent aux critères, la fenêtre la plus récente sera activée. WinActivate() works on minimized windows. However, a window that is "Always On Top" could still cover up a window you Activated.Toutefois, une fenêtre qui est "Always On Top" peut encore couvrir une fenêtre que vous avez activée.

After a successful activation @extended is set to 1 if the the window was already active, 2 if not.

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Activate the Notepad window using the handle returned by WinWait.Active la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinActivate($hWnd)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
