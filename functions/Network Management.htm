FunctionFonction Description
FtpSetProxy Sets the internet proxy to use for ftp access.Configure le proxy internet pour l'accès FTP.
HttpSetProxy Sets the internet proxy to use for http access.Configure le proxy internet pour l'accès HTTP.
HttpSetUserAgent Sets the HTTP user-agent string which is sent with all Inet requests.Configure la chaîne User-Agent envoyée par toutes les requêtes Inet.
InetClose Closes a handle returned from InetGet().Ferme un handle retourné par InetGet().
InetGet Downloads a file from the internet using the HTTP, HTTPS or FTP protocol.Télécharge un fichier depuis internet en utilisant le protocole HTTP, HTTPS ou FTP.
InetGetInfo Returns detailed data for a handle returned from InetGet().Retourne les données détaillées d'un handle retourné par InetGet().
InetGetSize Returns the size (in bytes) of a file located on the internet.Retourne la taille (en octets) d'un fichier situé sur internet.
InetRead Downloads a file from the internet using the HTTP, HTTPS or FTP protocol.Télécharge un fichier depuis internet en utilisant le protocole HTTP, HTTPS ou FTP.
Ping Pings a host and returns the roundtrip-time.Ping un hôte et retourne le temps de réponse.
TCPAccept Permits an incoming connection attempt on a socket.Permet une tentative de connexion entrante sur un socket.
TCPCloseSocket Closes a TCP socket.Ferme un socket TCP.
TCPConnect Create a socket connected to an existing server.Crée un socket connecté à un serveur existant.
TCPListen Creates a socket listening for an incoming connection.Crée un socket d'écoute pour une connexion entrante.
TCPNameToIP Converts an Internet name to IP address.Convertit un nom de domaine en adresse IP.
TCPRecv Receives data from a connected socket.Reçoit des données d'un socket connecté.
TCPSend Sends data on a connected socket.Envoie des données à un socket connecté.
TCPShutdown, UDPShutdown Stops TCP/UDP services.Arrête les services TCP/UDP.
TCPStartup, UDPStartup Starts TCP or UDP services.Démarre les services TCP ou UDP.
UDPBind Create a socket bound to an incoming connection.Crée un socket lié à une connexion entrante.
UDPCloseSocket Close a UDP socket.Ferme un socket UDP.
UDPOpen Open a socket connected to an existing server .Ouvre un socket connecté à un serveur existant.
UDPRecv Receives data from an opened socket.Reçoit des données d'un socket ouvert.
UDPSend Sends data on an opened socket.Envoie des données sur un socket ouvert.
