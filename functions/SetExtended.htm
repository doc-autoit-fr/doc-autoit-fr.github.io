SetExtended

Manually set the value of the @extended macro.Définit manuellement la valeur de la macro @extended.

SetExtended ( code [, return value] )

ParametersParamètre

code The required value (integer) to set into the @extended macro.La valeur requise (entière) à affecter à @extended.
return value [optional][optional] False - i, v, x ... The value to be returned by the function - if no parameter used the return value is undefined.La valeur à retourner par la fonction - si aucun paramètre n'est utilisé la valeur de retour est indéfinie.

Return ValueRemarque

Returns the "return value" parameter (undefined if the parameter is not used).Retourne le paramètre "return value"( indéfini si le paramètre n'est pas donné).
@extended will be set to the values set in the function call (default = 0)@extended sera définie avec les valeurs définies dans l'appel de la fonction (par défaut = 0 )

Remarks<i0>

When entering a function @extended is set to 0.Lors de l'entrée dans une fonction, @extended est définie à 0. Unless SetExtended() is called, then @extended will remain 0 when the function ends.A moins que SetExtended() soit appelée, alors @extended restera 0 lorsque la fonction se terminera. This means that in order for @extended to be set after a function returns, it must be explicitly set in the function.Cela signifie que, @extended doit être définie après qu'une fonction retourne, elle doit être définie explicitement dans la fonction. This also means you may need to backup the status of @extended in a variable if you are testing it in a While-WEnd loop.Cela signifie également que vous pouvez avoir besoin de sauvegarder le statut de @extended dans une variable si vous la testez dans une boucle While- WEnd.
The return value parameter is optional. It is provided as a way to use the Return SetExtended(...) syntax to define the value to be returned at the same time as setting @extended. If a specific value is not set then the return value will be undefined and should not be used subsequently by the code.

@extended is limited between the values of -2147483648 to 2147483647.@extended est limitée entre les valeurs de -2147483648 à 2147483647.
