Référence Fonction

Call

Calls a user-defined or built-in function contained in first parameter.Appelle une fonction utilisateur ou interne passée dans le premier paramètre.

Call ( "function" [, param1 [, param2 [, paramN]]] )

ParametersParamètre

function The name of function or function to call.Le nom de la fonction ou la fonction à appeler.
param Arguments that will be passed to the function being invoked.Les arguments qui seront passés à la fonction invoquée.

Return ValueRemarque

Success: the return value of the called function.Retourne la valeur de retour de la fonction appelée. Both @error and @extended may contain values if the called function set them.De même @error et @extended peuvent contenir des valeurs si la fonction appelée les renseigne.
Failure: sets the @error flag to 0xDEAD and @extended to 0xBEEF if the function does not exist or invalid number of parameters.Retourne @error = 0xDEAD et @extended = 0xBEEF si la fonction n'existe pas ou si le nombre d'arguments n'est pas correct.

Remarks<i0>

The function can pass arguments to functions, however, ByRef parameters are not supported; there is no way to retrieve the ByRef parameter.

A special array can be passed in lieu of individual parameters.Un tableau spécial peut être passé à la place des paramètres individuels. This array must have its first element set to "CallArgArray" and elements 1 - n will be passed as separate arguments to the function.Ce tableau doit avoir son premier élément égal à "CallArgArray" et les éléments de 1 à n contiendront les paramètres de la fonction passés séparemment. If using this special array, no other arguments can be passed to Call().Si vous utilisez ce tableau spécial, aucun autre argument ne pourra être passé à Call(). See example for a demonstration.Consultez l'exemple pour une démonstration.
Both Call() itself can set the @error flag or the called function can set the @error flag.Call() peut définir lui-même le flag @error, tout comme la fonction appelée. If Call() sets the @error flag, the value will be 0xDEAD and @extended will also be set to 0xBEEF.Si Call() définit le flag @error, la valeur sera 0xDEAD et @extended sera défini à 0xBEEF. See the example for a demonstration of testing for a function that was not found.Consultez l'exemple pour un exemple de fonction qui n'est pas trouvée.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; This calls a function accepting no arguments.Ceci appelle une fonction acceptant aucun argument.
    Call("Test1")

    ; This calls a function accepting one argument and passes it an argument.Ceci appelle une fonction acceptant un argument et le passe en argument.
    Call("Test2", "Message from Call()!")

    ; This demonstrates how to use the special array argument.Ceci démontre comment utiliser le tableau spécial d'arguments.
    Local $aArgs[4]
    $aArgs[0] = "CallArgArray" ; This is required, otherwise, Call() will not recognize the array as containing argumentsCela est nécessaire, sinon, Call() ne reconnaîtra pas le tableau comme contenant des arguments
    $aArgs[1] = "This is a string" ; Parameter one is a stringLe premier paramètre est une chaîne
    $aArgs[2] = 47 ; Parameter two is a numberLe paramètre deux est un nombre
    Local $aArray[2]
    $aArray[0] = "Array Element 0"
    $aArray[1] = "Array Element 1"
    $aArgs[3] = $aArray ; Parameter three is an arrayLe paramètre trois est un tableau

    ; We've built the special array, now call the functionNous avons construit le tableau spécial, maintenant, appelons la fonction
    Call("Test3", $aArgs)

    ; Test calling a function that does not exist.Test de l'appel d'une fonction qui n'existe pas. This shows the proper way to test byCela montre la bonne façon de tester ensemble
    ; checking that both @error and @extended contain the documented failure values.les deux flags @error et @extended qui contiennent les erreurs documentées.
    Local Const $sFunction = "DoesNotExist"
    Call($sFunction)
    If @error = 0xDEAD And @extended = 0xBEEF Then MsgBox($MB_SYSTEMMODAL, "", "Function does not exist.")
EndFunc   ;==>Example

Func Test1()
    MsgBox($MB_SYSTEMMODAL, "", "Hello")
EndFunc   ;==>Test1

Func Test2($sMsg)
    MsgBox($MB_SYSTEMMODAL, "", $sMsg)
EndFunc   ;==>Test2

Func Test3($sString, $nNumber, $aArray)
    MsgBox($MB_SYSTEMMODAL, "", "The string is: " & @CRLF & $sString)
    MsgBox($MB_SYSTEMMODAL, "", "The number is: " & @CRLF & $nNumber)
    For $i = 0 To UBound($aArray) - 1
        MsgBox($MB_SYSTEMMODAL, "", "Array[" & $i & "] contains:" & @CRLF & $aArray[$i])
    Next
EndFunc   ;==>Test3
