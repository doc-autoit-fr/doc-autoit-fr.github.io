Tray functions ReferenceTray - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
TrayCreateItem Creates a menuitem control for the tray.Crée un contrôle menuitem dans la zone de notification.
TrayCreateMenu Creates a menu control for the tray menu.Crée un contrôle menu pour le menu de la zone de notification.
TrayGetMsg Polls the tray to see if any events have occurred.Interroge la zone de notification pour savoir si des événements ont été déclenchés.
TrayItemDelete Deletes a menu/item control from the tray menu.Supprime un contrôle menu ou item dans le menu de la zone de notification.
TrayItemGetHandle Returns the handle for a tray menu(item).Retourne le handle d'un menu ou d'un item de la zone de notification.
TrayItemGetState Gets the current state of a control.Obtient l'état actuel d'un contrôle.
TrayItemGetText Gets the itemtext of a tray menu/item control.Obtient le texte d'un contrôle menu ou item de la zone de notification.
TrayItemSetOnEvent Defines a user-defined function to be called when a tray item is clicked.Définit une fonction utilisateur qui est appelée quand un item de la zone de notification est cliqué.
TrayItemSetState Sets the state of a tray menu/item control.Définit l'état d'un contrôle menu ou item de la zone de notification.
TrayItemSetText Sets the itemtext of a tray menu/item control.Définit le texte d'un contrôle menu ou item de menu de la zone de notification.
TraySetClick Sets the clickmode of the tray icon - what mouseclicks will display the tray menu.Définit le 'ClickMode' de l'icône de la zone de notification - quel clic de souris affiche le menu.
TraySetIcon Loads/Sets a specified tray icon.Charge/Définit l'icône spécifiée pour la zone de notification.
TraySetOnEvent Defines a user function to be called when a special tray action happens.Définit une fonction utilisateur qui sera appelée lors d'une action spéciale dans la zone de notification.
TraySetPauseIcon Loads/Sets a specified tray pause icon.Charge/Définit l'icône 'pause' spécifiée pour la zone de notification.
TraySetState Sets the state of the tray icon.Définit l'état de l'icône de la zone de notification.
TraySetToolTip (Re)Sets the tooltip text for the tray icon.(Re)Définit le texte de l'info-bulle pour l'icône de la zone de notification.
TrayTip Displays a balloon tip from the AutoIt Icon.Affiche une info-bulle 'balloon' depuis l'icône du script AutoIt.
