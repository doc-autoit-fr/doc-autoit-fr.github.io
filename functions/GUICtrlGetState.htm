GUICtrlGetState

Gets the current state of a control.Obtient l'état actuel d'un contrôle.

GUICtrlGetState ( controlID )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.

Return ValueValeur de retour

Success:Succès : the state.Retourne l'état. See GUICtrlSetState() for values.Consultez GUICtrlSetState() pour les valeurs.
Failure:Échec : -1 if control is not defined.Retourne -1 si le contrôle n'est pas définie.

RemarksRemarques

As opposed to GUICtrlRead() this function returns ONLY the state of a control enabled/disabled/hidden/show/dropacceptedPar opposition à GUICtrlRead() cette fonction retourne ONLY l'état d'un contrôle activé/désactivé/caché/visible/qui accepte le drop

Exceptions:
For ListView controls it returns the number of the clicked column.Pour le contrôle ListView, elle renvoie le nombre de colonnes cliquées.

RelatedLiens

GUICtrlRead, GUICtrlSetState

ExampleExemple

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("My GUI (GetControlState)")
    Local $idCheckbox = GUICtrlCreateCheckbox("checkbox", 10, 10)
    GUICtrlSetState(-1, 1) ; checkedvérifié

    GUISetState(@SW_SHOW) ; will display an empty dialog boxAffiche une boîte de dialogue vide

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    MsgBox($MB_SYSTEMMODAL, "state", StringFormat("GUICtrlRead=%d\nGUICtrlGetState=%d", GUICtrlRead($idCheckbox), GUICtrlGetState($idCheckbox)))
EndFunc   ;==>Example
