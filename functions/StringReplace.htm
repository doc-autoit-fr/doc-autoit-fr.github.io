StringReplace

Replaces substrings in a string.Remplace des sous-chaînes dans une chaîne.

StringReplace ( "string", "searchstring/start", "replacestring" [, occurrence = 0 [, casesense = 0]] )

ParametersParamètres

string The string to evaluate.La chaîne à traiter.
searchstring/start The substring to search for or the character position to start the replacement.La sous-chaîne à rechercher ou la position du caractère de départ du remplacement.
replacestring The replacement string.La chaîne de remplacement.
occurrence [optional][optionnel] The number of times to replace the searchstring.Le nombre de remplacement de la chaîne recherchée. Use a negative occurrence to replace from the right side.Utiliser une occurrence négative pour remplacer à partir de la droite.
    0 = all searchstrings will be replaced (default)
casesense [optional][optionnel] Flag to indicate if the operations should be case sensitive.Paramètre pour indiquer si les opérations doivent être sensibles à la casse.
    $STR_NOCASESENSE (0) = not case sensitive, using the user's locale (default)
    $STR_CASESENSE (1) = case sensitive
    $STR_NOCASESENSEBASIC (2) = not case sensitive, using a basic/faster comparison

Constants are defined in StringConstants.au3.

Return ValueValeur de retour

Returns the new string with the number of replacements performed stored in the @extended macro.Retourne la nouvelle chaîne, le nombre de remplacements effectués est stocké dans la macro @extended.

RemarksRemarques

By default or if the occurrence is positive the search/replace is performed left-to-right.Par défaut ou si le paramètre occurrence est positif, la recherche/remplacement est effectuée de gauche à droite. Thus, StringReplace("aaa", "aa", "bb") returns "bba"Ainsi, StringReplace("aaa", "aa", "bb") retourne "bba"

If the start method is used the occurrence and casesense parameters are ignored.Si la méthode de la position de départ est utilisée, les paramètres occurrence et casesense sont ignorés. The function will replace the characters in "string", starting at the requested position, with the characters in "replacestring" - as many characters will be replaced as are in "replacestring". However, if there are not enough characters in "string" for the entire "replacestring" to be inserted an empty string is returned and @error is set to 1.

ExampleExemple

#include <MsgBoxConstants.au3>

; Replace a blank space (' ') with a - (minus) character.Remplace un espace (' ') avec le caractère - (moins).
Local $sString = StringReplace("This is a sentence with whitespace.", " ", "-")
Local $iReplacements = @extended
MsgBox($MB_SYSTEMMODAL, "", $iReplacements & " replacements were made and the new string is:" & @CRLF & @CRLF & $sString)
