FileGetLongName

Returns the long path+name of the path+name passed.Retourne le chemin+nom long du chemin+nom donné.

FileGetLongName ( "filename" [, flag = 0] )

ParametersParamètre

filename Full path and file name to convertLe chemin complet et nom du fichier à convertir
flag [optional][optional] False - i, v, x ...
    $FN_FULLPATH (0) - Default
    $FN_RELATIVEPATH (1) - file can have relative dir, e.g. "..\file.txt"

Constants are defined in FileConstants.au3.

Return ValueRemarque

Success: the long path+name of the path+name passed.Retourne le chemin + nom long du chemin + nom donné.
Failure: the parameter and sets the @error flag to 1.Retourne le paramètre et @error = 1.
