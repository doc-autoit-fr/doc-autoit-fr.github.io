AutoItWinSetTitle

Changes the title of the AutoIt window.Modifie le titre d'une fenêtre Autoit.

AutoItWinSetTitle ( "newtitle" )

ParametersParamètre

newtitle The new title to give to the window.Le nouveau titre à donner à la fenêtre.

Return ValueRemarque

None.La fonction permet de définir un tableau de plages de caractère qui représentent les plages mesurées par un appel à la fonction MeasureCharacterRanges.

Remarks<i0>

The AutoIt window is usually hidden.La fenêtre Autoit est habituellement cachée. The purpose of changing the title is to allow other programs (or other AutoIt scripts) to interact with AutoIt.L’intérêt de changer son titre est d'autoriser d'autres programmes (ou d'autres scripts Autoit) à interagir avec Autoit.

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Set the title of the AutoIt Hidden Window.
    AutoItWinSetTitle("My AutoIt Window")

    ; Display AutoIt's Hidden Window.Affiche la fenêtre cachée de AutoIt.
    AutoItWinShow()

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example

; Display AutoIt's Hidden Window.Affiche la fenêtre cachée de AutoIt. Returns the handle of the window.Retourne le handle de la fenêtre.
Func AutoItWinShow()
    Local $hWnd = WinGetHandle(AutoItWinGetTitle()) ; Get the handle of the AutoIt Hidden Window by finding out the title of the AutoIt Hidden Window.Obtient le handle de la fenêtre cachée AutoIt à partir du titre de la fenêtre cachée AutoIt.
    WinMove($hWnd, "", (@DesktopWidth / 2) - 250, (@DesktopHeight / 2) - 250, 500, 500) ; Move the AutoIt Hidden Window and re-size for a better view.Déplace la fenêtre cachée AutoIt et la redimensionne pour une meilleure vue.
    WinSetState($hWnd, "", @SW_SHOW) ; Show the AutoIt Hidden Window, normally this is hidden, but in the interest of this example I"m displaying it.Montre la fenêtre cachée de AutoIt, normalement elle est cachée, mais dans l'intérêt de cet exemple, je la montre.
    Return $hWnd
EndFunc   ;==>AutoItWinShow
