CDTray

Opens or closes the CD tray.Ouvre ou ferme le tiroir d'un lecteur CD.

CDTray ( "drive", "status" )

ParametersParamètres

drive The drive letter of the CD tray to control, in the format D:,La lettre du lecteur CD à contrôler, dans le format D:, E:, etc.E: etc ...
status Specifies if you want the CD tray to be open or closed:Spécifie si vous voulez ouvrir ou fermer le lecteur CD :
    $CDTRAY_OPEN ("open") - to be open
    $CDTRAY_CLOSED ("closed") - to be closed on cd tray (not laptop-style)

Constants are defined in AutoItConstants.au3

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 if drive is locked via CD burning software or if the drive letter is not a CD drive.Retourne 0 si le lecteur est verrouillé par logiciel de gravure ou si la lettre indiquée n'est pas celle d'un lecteur CD.

RemarksRemarques

CDTray() works as expected with virtual CD drives such as DAEMON Tools.CDTray() fonctionne comme attendu avec des lecteurs CD virtuels tels que des utilitaires de type DAEMON.
CDTray() does not work on non-local/mapped CD drives; CDTray() must be run from the computer whose drive it affects.CDTray() ne fonctionne pas avec des lecteurs CD non locaux ou mappés.
CDTray("X:", "close") returns 1 on laptop-style cd trays that can only be closed manually.

ExampleExemple

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

; Get List of all CDRom's
Local $aCDRom = DriveGetDrive($DT_CDROM)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "", "No CDRom found.")
    Exit
EndIf

; Open the CD tray of the first CDRom
CDTray($aCDRom[1], $CDTRAY_OPEN)
Sleep(2000)
CDTray($aCDRom[1], $CDTRAY_CLOSED) ; does not work on laptop-style cd trays that can only be closed manually.
