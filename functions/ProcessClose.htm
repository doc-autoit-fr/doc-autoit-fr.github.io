ProcessClose

Terminates a named process.Termine un processus identifié.

ProcessClose ( "process" )

ParametersParamètres

process The name or PID of the process to terminate.Le nom ou le PID du processus à terminer.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 and sets the @error flag to non-zero.Retourne 0 et @error <> 0.
@error:@error : 1 = OpenProcess failed1 = L'ouverture du processus à échoué.
2 = AdjustTokenPrivileges Failed2 = Vous n'avez pas les privilèges requis.
3 = TerminateProcess Failed3 = La fermeture du processus à échoué.
4 = Cannot verify if process exists4 = Ne peut vérifier l’existence du processus
@extended: sets to Windows API GetLastError()Contient Windows API GetLastError()

RemarksRemarques

Process names are executables without the full path, e.g., "notepad.exe" or "winword.exe"

If multiple processes have the same name, the one with the highest PID is terminated--regardless of how recently the process was spawned.Si plusieurs processus ont le même nom, celui avec le plus grand PID est terminé, selon le processus le plus récemment engendré.

PID is the unique number which identifies a Process.PID est le numéro unique qui identifie un processus. A PID can be obtained through the ProcessExists() or Run() commands.Un PID peut être obtenu par les commandes ProcessExists() or Run().

The process is polled approximately every 250 milliseconds.Le processus est interrogé environ toutes les 250 millisecondes.

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Local $iPID = Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    WinWait("[CLASS:Notepad]", "", 10)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Close the Notepad process using the PID returned by Run.Ferme le processus Notepad en utilisant le PID renvoyé par Run.
    ProcessClose($iPID)
EndFunc   ;==>Example
