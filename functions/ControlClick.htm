ControlClick

Sends a mouse click command to a given control.Envoie un clic de souris sur un contrôle spécifié.

ControlClick ( "title", "text", controlID [, button = "left" [, clicks = 1 [, x [, y]]]] )

ParametersParamètres

title The title/hWnd/class of the window to access.Le titre/handle/class de la fenêtre à atteindre. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to access.Le texte de la fenêtre à accéder. See Text special definition.Consultez Définition spéciale de Texte .
controlID The control to interact with.Le contrôle avec lequel interagir. See Controls.Consultez Controls.
button [optional][optionnel] The button to click, "left", "right", "middle", "main", "menu", "primary", "secondary".Le bouton à cliquer :"left", "right", "middle", "main", "menu", "primary", "secondary". Default is the left button.Bouton gauche ("left") par défaut.
clicks [optional][optionnel] The number of times to click the mouse.Le nombre de clics de souris. Default is 1.Par défaut c'est 1.
x [optional][optionnel] The x position to click within the control.La position x du clic sur le contrôle. Default is center.Centré par défaut.
y [optional][optionnel] The y position to click within the control.La position y du clic sur le contrôle. Default is center.Centré par défaut.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0.Retourne 0.

RemarksRemarques

Some controls will resist clicking unless they are the active window.Certains contrôles peuvent résister aux clics tant qu'ils ne sont pas dans la fenêtre active. Use the WinActivate() function to force the control's window to the top before using ControlClick().Utilisez la fonction WinActivate() pour mettre la fenêtre des contrôles au premier plan avant d'utiliser ControlClick().
Using 2 for the number of clicks will send a double-click message to the control - this can even be used to launch programs from an explorer control!La valeur 2 pour le nombre de clics permettra d'effectuer un double-clic sur le contrôle - Cela peut également servir à lancer des programmes depuis une fenêtre de l'explorateur !

If the user has swapped the left and right mouse buttons in the control panel, then the behaviour of the buttons is different.Si l'utilisateur a permuté les boutons gauche et droite de la souris dans le panneau de configuration, alors le comportement de la fonction est différent. "Left" and "right" always click those buttons, whether the buttons are swapped or not."Left" et "right" cliquent respectivement sur les boutons gauche et droite, qu'ils soient inversés ou non. The "primary" or "main" button will be the main click, whether or not the buttons are swapped.Le bouton "primary" ou "main" sera le clic principal, que les boutons soient inversés ou non. The "secondary" or "menu" buttons will usually bring up the context menu, whether the buttons are swapped or not.Le bouton "secondary" ou "menu" fera généralement apparaître le menu contextuel, que les boutons soiennt inversés ou non.

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Send a mouse click to the edit control of Notepad using the handle returned by WinWait.Envoie un clic de souris vers le contrôle d'édition de bloc-notes à l'aide du handle renvoyé par WinWait.
    ControlClick($hWnd, "", "Edit1")

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
