DriveSpaceTotal

Returns the total disk space of a path in Megabytes.Retourne l'espace disque total d'un lecteur en Méga-octet.

DriveSpaceTotal ( "path" )

ParametersParamètre

path Path of drive to receive information from.Le chemin sur lequel agir.

Return ValueRemarque

Success: the diskspace in Megabytes as a float number.Retourne l'espace total du lecteur en Méga-octets comme nombre flottant.
Failure: sets the @error flag to 1.Retourne 0 et @error = 1.

Remarks<i0>

DriveSpaceTotal() may even work when a complete directory path (that exists) is given.DriveSpaceTotal() peut fonctionner même si le chemin donné est celui d'un répertoire existant sur le lecteur. However, a file path won't work.Un chemin vers un fichier ne fonctionnera pas.
