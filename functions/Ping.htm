Ping

Pings a host and returns the roundtrip-time.Ping un hôte et retourne le temps de réponse.

Ping ( "address/hostname" [, timeout = 4000] )

ParametersParamètres

address/hostname Can be i.e. "www.autoitscript.com" or "87.106.244.38".
timeout [optional][optionnel] Is the time to wait for an answer in milliseconds (default is 4000).Temps d'attente pour une réponse en millisecondes (4000 par défaut).

Return ValueValeur de retour

Success:Succès : the roundtrip-time in milliseconds ( greater than 0 ).Retourne le temps de réponse en millisecondes (supérieur à 0).
Failure:Échec : 0 if host is not pingable or other network errors occurred and sets the @error flag to non-zero.Retourne 0 si l'hôte n'est pas joignable ou autres erreurs réseau et définit @error <> 0.(Voir remarques).
@error:@error : 1 = Host is offline1 = Hôte déconnecté
2 = Host is unreachable2 = Hhôte est inaccessible
3 = Bad destination3 = Mauvaise destination
4 = Other errors4 = Autres erreurs

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Ping the AutoIt website with a timeout of 250ms.Ping, le site AutoIt avec un délai d'attente de 250 ms.
    Local $iPing = Ping("autoitscript.com", 250)

    If $iPing Then ; If a value greater than 0 was returned then display the following message.Si une valeur supérieure à 0 est ensuite retournée, affiche le message suivant.
        MsgBox($MB_SYSTEMMODAL, "", "The roundtrip-time took: " & $iPing & "ms.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred with @error value of: " & @error)
    EndIf
EndFunc   ;==>Example
