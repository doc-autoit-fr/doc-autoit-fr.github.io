BlockInput

Disable/enable the mouse and keyboard.Désactive/Active la souris et le clavier.

BlockInput ( flag )

ParametersParamètre

flag     $BI_DISABLE (1) = Disable user input
    $BI_ENABLE (0) = Enable user input

Constants are defined in "AutoItConstants.au3".

Return ValueRemarque

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0.Retourne 0. Already Enable or #RequireAdmin not used.Déjà activé ou #requireAdmin non utilisé.

Remarks<i0>

The table below shows how BlockInput() behavior depends on the Windows version; however, pressing Ctrl+Alt+Del on any platform will re-enable input due to a Windows API feature.Le tableau ci-dessous montre le comportement de BlockInput() en fonction de la version de Windows; toutefois, quel que soit la plateforme, l'appui de Ctrl+Alt+Del réactive les entrées du fait des caractéristiques de l'API de Windows.

Operating System "BlockInput()" ResultsRésultat de BlockInput()
Windows XP User input is blocked and AutoIt can simulate mouse and keyboard input.L'entrée utilisateur est bloquée et AutoIt peut simuler les entrées de la souris et du clavier. See remarks below for an issue with Windows XP SP1.Consultez les remarques ci-dessous pour un problème avec Windows XP SP1.
Windows Vista and above User input is blocked and AutoIt can simulate mouse and keyboard if #RequireAdmin is used.L'entrée utilisateur est bloquée et AutoIt peut simuler les entrées de la souris et du clavier si #RequireAdmin est utilisé.

A Windows hotfix was released for Windows XP SP1 that contained a bug that prevents the Alt keystroke from being sent (via the Send() function) while BlockInput() is active.Un correctif Windows a été réalisé pour un bug sous Windows XP SP1 qui empêche l'envoi de l'appui de la touche Alt (envoyé par la fonction Send()) tant que BlockInput() est actif. The Windows bug is fixed in Windows XP SP2 and newer.Ce bug a été corrigé dans Windows XP SP2 et supérieur.

BlockInput() only affects user-input.BlockInput() affecte seulement les entrées de type utilisateur. Input from functions like Send() or MouseMove() still work.Les entrées provenant des fonctions comme Send() ou MouseMove() continuent à fonctionner.

Example<a0>StringInStr()</a0>, <a1>StringRegExpReplace()</a1>

#RequireAdmin

#include <AutoItConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Disable user input from the mouse and keyboard.Désactive les entrées utilisateur de la souris et du clavier.
    BlockInput($BI_DISABLE)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Send the 'F5' key to the edit control of Notepad to display the date and time.Envoie la touche 'F5' au contrôle d'édition de Bloc-notes pour afficher la date et l'heure. The handle returned by WinWait is used for the "title" parameter of ControlSend.Le handle renvoyé par WinWait est utilisé pour le \"titre\" paramètre de ControlSend.
    ControlSend($hWnd, "", "Edit1", "{F5}")

    ; Enable user input from the mouse and keyboard.Active les entrées utilisateur à partir de la souris et du clavier.
    BlockInput($BI_ENABLE)

    ; Wait for 2 seconds.Attend 2 secondes.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)

    ; Now a screen will pop up and ask to save the changes, the classname of the window is calledMaintenant, une fenêtre s'ouvre et demande d'enregistrer les modifications, le nom de classe de la fenêtre est appelé
    ; "#32770" and simulating the "TAB" key to move to the second button in which the "ENTER" is simulated to not "save the file""# 32770" et la simulation de la touche "TAB" pour passer au deuxième bouton dans lequel le "ENTER" est simulé pour ne pas enregistrer le fichier
    WinWaitActive("[CLASS:#32770]")
    Sleep(500)
    Send("{TAB}{ENTER}")
EndFunc   ;==>Example
