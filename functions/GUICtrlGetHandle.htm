GUICtrlGetHandle

Returns the handle for a control and some special (item) handles (Menu, ContextMenu, TreeViewItem).Retourne le handle d'un contrôle et certains handles spéciaux (Menu, ContextMenu, TreeViewItem).

GUICtrlGetHandle ( controlID )

ParametersParamètres

controlID Control identifier as returned by a GUICtrlCreate...() function.Identificateur de contrôle tel que retourné par une fonctions GUICtrlCreate...().

Return ValueValeur de retour

Success:Succès : the handle of the given control ID.Retourne le handle de l'ID du contrôle donné.
Failure:Échec : 0.

RemarksRemarques

The following controls will not return a handle:Les contrôles suivants ne retourneront pas de handle: GUICtrlCreateDummy(), GUICtrlCreateGraphic(), GUICtrlCreateObj(), GUICtrlCreateListViewItem() and GUICtrlCreateTabItem().GUICtrlCreateDummy(), GUICtrlCreateGraphic(), GUICtrlCreateObj(), GUICtrlCreateListViewItem() et GUICtrlCreateTabItem() .

ListViewItems and TabItems are managed through indexes.ListViewItems et TabItems sont gérés par des indices. To get the index of these items use DllCall() and DllStructCreate().Pour obtenir l'indice de ces items utilisez DllCall() et DllStructCreate() .

Func Example()
    GUICreate("My GUI", 300, 200)

    Local $idFileMenu = GUICtrlCreateMenu("&File")
    GUICtrlCreateMenuItem("&Open", $idFileMenu)
    GUICtrlCreateMenuItem("&Save", $idFileMenu)
    GUICtrlCreateMenuItem("", $idFileMenu)

    Local $idOptionsMenu = GUICtrlCreateMenu("O&ptions", $idFileMenu)
    GUICtrlCreateMenuItem("View", $idOptionsMenu)
    GUICtrlCreateMenuItem("", $idOptionsMenu)
    GUICtrlCreateMenuItem("Tools", $idOptionsMenu)

    GUICtrlCreateMenuItem("", $idFileMenu)
    Local $idExitItem = GUICtrlCreateMenuItem("&Exit", $idFileMenu)

    Local $idHelpMenu = GUICtrlCreateMenu("&?")
    Local $idAboutItem = GUICtrlCreateMenuItem("&About", $idHelpMenu)

    Local $idEndBtn = GUICtrlCreateButton("End", 110, 140, 70, 20)

    SetMenuColor($idFileMenu, 0xEEBB99) ; BGR color valueValeur de couleur RGB
    SetMenuColor($idOptionsMenu, 0x66BB99); BGR color valueValeur de couleur RGB
    SetMenuColor($idHelpMenu, 0x99BBEE) ; BGR color valueValeur de couleur RGB

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        $idMsg = GUIGetMsg()

        Switch $idMsg
            Case $idExitItem, $idEndBtn, $GUI_EVENT_CLOSE
                ExitLoop

            Case $idAboutItem
                MsgBox($MB_SYSTEMMODAL, "About...", "Colored menu sample")
        EndSwitch
    WEnd
EndFunc   ;==>Example

; Apply the color to the menuApplique la couleur au menu
Func SetMenuColor($nMenuID, $nColor)
    Local Const $MIM_APPLYTOSUBMENUS = 0x80000000
    Local Const $MIM_BACKGROUND = 0x00000002

    Local $hMenu = GUICtrlGetHandle($nMenuID)

    Local $hBrush = DllCall("gdi32.dll", "hwnd", "CreateSolidBrush", "int", $nColor)
    $hBrush = $hBrush[0]

    Local $tMenuInfo = DllStructCreate("dword;dword;dword;uint;dword;dword;ptr")
    DllStructSetData($tMenuInfo, 1, DllStructGetSize($tMenuInfo))
    DllStructSetData($tMenuInfo, 2, BitOR($MIM_APPLYTOSUBMENUS, $MIM_BACKGROUND))
    DllStructSetData($tMenuInfo, 5, $hBrush)

    DllCall("user32.dll", "int", "SetMenuInfo", "hwnd", $hMenu, "struct*", $tMenuInfo)

    ; release Struct not really needed as it is a locallibére Struct (pas vraiment nécessaire car elle est Local)
    $tMenuInfo = 0
EndFunc   ;==>SetMenuColor
