DllStructGetPtr

Returns the pointer to the struct or an element in the struct.Retourne le pointeur vers la structure ou un élément dans la structure.

DllStructGetPtr ( Struct [, Element] )

ParametersParamètres

Struct The struct returned by DllStructCreate().La structure renvoyée par DllStructCreate().
Element [optional] The element of the struct whose pointer you need, starting at 1 or the element name as defined in DllStructCreate().L'élément de la structure qui est le pointeur dont vous avez besoin, à partir de 1 ou du nom de l’élément tel que défini dans DllStructCreate().

Return ValueValeur de retour

Success: The pointer to the struct.Retourne le pointeur vers la structure.
Failure: 0.Retourne 0.
@error:@error : 1 = Struct not a valid struct returned by DllStructCreate().1 = La structure n'est pas une structure correcte retournée par DllStructCreate().
2 = Element out of range.2 = La valeur de l’élément est hors limites.

RemarksRemarques

Used in DllCall().Utilisé dans DllCall().

RelatedLiens

DllCall, DllStructCreate

Example 1Exemple 1

#include <MsgBoxConstants.au3>
#include <StructureConstants.au3>

Example()

Func Example()
    ; Assign a Local variable the handle of the current active window.Assigne une variable locale avec le handle de la fenêtre active.
    Local $hWnd = WinGetHandle("") ; Same as:Même que : "[active]".« [ actif ] ».

    ; Assign a Local variable the window's rectangle (array).Assigne une variable locale avec le rectangle de la fenêtre (tableau).
    Local $aWndPos = WinGetPos($hWnd)

    ; Assign a Local variable the structure created with the tagRECT definition.Assigne une variable locale avec la structure créée avec la définition de tagRECT.
    Local $tRECT = DllStructCreate($tagRECT)
    Local $iError = 0

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), Default, "Error in DllStructCreate, Code: " & $iError)
        Return False
    EndIf

    ; Make the DllCall of the GetWindowRect function.Fait le DllCall de la fonction GetWindowRect.
    DllCall("user32.dll", "int", "GetWindowRect", _
            "hwnd", $hWnd, _
            "ptr", DllStructGetPtr($tRECT))

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), Default, "Error in DllCall, Code: " & $iError)
        Return False
    EndIf

    ; Note:Note : The 2nd parameter of the GetWindowRect function requires a pointer,
    ; the result returned by the DllStructCreate is NOT a pointer, but using 'struct*' allows to pass a structure as a ptr.

    ; Assign Local variables the returned rectangle.Affecte des variables locales avec le rectangle retourné.
    Local $iLeft = DllStructGetData($tRECT, "Leftgauche") ; Or 1 instead of "Left".Ou 1 à la place de «gauche».
    Local $iTop = DllStructGetData($tRECT, 2) ; Or "Top" instead of 2.Ou \"Top\" au lieu de 2.
    Local $iRight = DllStructGetData($tRECT, 3) ; Or "Right" instead of 3.Ou \" droit \" au lieu de 3.
    Local $iBottom = DllStructGetData($tRECT, "Bottom") ; Or 4 instead of "Bottom".Ou 4 au lieu de \"Bas\".

    ; Release the resources used by the structure.Libére les ressources utilisées par la structure.
    $tRECT = 0

    ; Display the results of WinGetPos and the returned rectangle.Affiche les résultats de WinGetPos et le rectangle retourné.
    MsgBox($MB_SYSTEMMODAL, "", "WinGetPos(): (" & $aWndPos[0] & ", " & $aWndPos[1] & ") " & _
            "(" & $aWndPos[2] + $aWndPos[0] & ", " & $aWndPos[3] + $aWndPos[1] & ")" & @CRLF & _
            "GetWindowRect(): (" & $iLeft & ", " & $iTop & ") (" & $iRight & ", " & $iBottom & ")")
EndFunc   ;==>Example

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local variable a structure with the definition of an int.Assigne une variable locale d'une structure à la définition d'un int.
    Local $tSTRUCT1 = DllStructCreate("int")
    Local $iError = 0

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), Default, "Error in DllCall, Code: " & $iError)
        Return False
    EndIf

    ; Assign a Local variable the tSTRUCT1 structure except that the elements will be got according to the new definition.Assigne une variable locale de la structure tSTRUCT1 sauf que les éléments seront obtenu selon la nouvelle définition.
    Local $tSTRUCT2 = DllStructCreate("uint", DllStructGetPtr($tSTRUCT1, 1))

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), Default, "Error in DllCall, Code: " & $iError)
        Return False
    EndIf

    Local $tSTRUCT3 = DllStructCreate("float", DllStructGetPtr($tSTRUCT1, 1))

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), Default, "Error in DllCall, Code: " & $iError)
        Return False
    EndIf

    ; Notes: -The structures points to the same structure (they have the same pointer);
    ; The first element got with the:Le premier élément a avec le :
    ; *tSTRUCT1 struct will be an int.* tSTRUCT1 structure sera un int.
    ; *tSTRUCT2 struct will be an unsigned int (the element is casted from int to uint).* tSTRUCT2 structure sera un unsigned int (l'élément est coulé de int en uint).
    ; *tSTRUCT3 struct will be a float (the element is casted from int to float).* tSTRUCT3 structure sera un flotteur (l'élément est coulé de int à flotter).
    ;
    ; -The "reinterpret_cast" casting operator is used.-Le Opérateur de casting \" de reinterpret_cast \" est utilisé.

    ; Set the data of the first element (int) in the $tSTRUCT1.Définir les données du premier élément (int) dans le tSTRUCT1 $.
    DllStructSetData($tSTRUCT1, 1, -1)

    ; Display the different data types of the same data.Affiche les différents types de données des mêmes données.
    MsgBox($MB_SYSTEMMODAL, "", _
            "int: " & DllStructGetData($tSTRUCT1, 1) & @CRLF & _
            "uint: " & DllStructGetData($tSTRUCT2, 1) & @CRLF & _
            "float: " & DllStructGetData($tSTRUCT3, 1))

    ; Release the resources used by the structures.Libérer les ressources utilisées par les structures.
    $tSTRUCT1 = 0
    $tSTRUCT2 = 0
    $tSTRUCT3 = 0
EndFunc   ;==>Example
