WinClose

Closes a window.Ferme une fenêtre.

WinClose ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to close.Le titre/hWnd/class de la fenêtre à fermer. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the window to close.Le texte de la fenêtre à fermer. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0 if the window is not found.Retourne 0 si la fenêtre n'existe pas.

RemarksRemarques

This function sends a close message to a window, the result depends on the window (it may ask to save data, etc.).Cette fonction envoie un message de fermeture à une fenêtre, le résultat dépend de la fenêtre (elle peut demander d'enregistrer des données, etc...). To force a window to close, use the WinKill() function.Pour forcer une fenêtre à se fermer, utilisez la fonction WinKill(). If multiple windows match the criteria, the window that was most recently active is closed.Si plusieurs fenêtres correspondent aux critères, la fenêtre active la plus récente est fermée.

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    WinWait("[CLASS:Notepad]", "", 10)

    ; Wait for 2 seconds to display the Notepad window.Attend 2 secondes pour afficher la fenêtre du Bloc-notes.
    Sleep(2000)

    ; Close the Notepad window using the classname of Notepad.Ferme la fenêtre du Bloc-notes à l'aide du nom de classe du Bloc-notes.
    WinClose("[CLASS:Notepad]")
EndFunc   ;==>Example
