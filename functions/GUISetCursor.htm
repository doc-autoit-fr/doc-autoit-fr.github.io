GUISetCursor

Sets the mouse cursor icon for a GUI window.Définit l'icône du curseur de la souris pour une fenêtre GUI.

GUISetCursor ( [cursorID [, override = 0 [, winhandle]]] )

ParametersParamètres

cursorID [optional][optionnel] Cursor Id (See Remarks).Id du curseur (Voir Remarques).
override [optional][optionnel] Force the requested mouse cursor even when over controls (see below).Force le curseur souris requis, même lorsqu'il se trouve sur d'autres contrôles (voir ci-dessous ).
    $GUI_CURSOR_NOOVERRIDE (0) = (default) Don't override a control's default mouse cursor.
    $GUI_CURSOR_OVERRIDE (1) = override control's default mouse cursor.

Constants are defined in GUIConstantsEx.au3.
winhandle [optional][optionnel] Windows handle as returned by GUICreate() (default is the previously used window).Handle Windows retourné par GUICreate() (par défaut, la fenêtre précédemment utilisée ).

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

If the cursorID is invalid the standard arrow will be displayed.Si l'ID du curseur est invalide la flèche standard sera utilisée.

Usually when you move the mouse cursor over an edit control or other control the mouse cursor changes shape.Habituellement, lorsque vous déplacez le curseur souris sur un contrôle Edit ou un autre contrôle le curseur souris change de forme. The "override" option allows you to force the requested mouse cursor to be shown at all times.L'option "override" vous permet de forcer le curseur souris demandé à être utilisé tout le temps. Note:Note : If you have changed a controls mouse cursor with GUICtrlSetCursor() then this control mouse cursor will always be shown.Si vous avez modifié le curseur souris d'un contrôle avec GUICtrlSetCursor(), ce curseur sera toujours affichée.

For a list of valid cursor IDs see MouseGetCursor().Pour une liste d'IDs de curseur valides voir MouseGetCursor().
CursorId = 16 will hide the mouse cursor.CursorID = 16 permet de cacher le curseur souris.

Example()

Func Example()
    HotKeySet("{ESC}", "Increment")

    GUICreate("Press ESC to Increment", 400, 400, 0, 0)

    GUISetState(@SW_SHOW)

    While GUIGetMsg() <> $GUI_EVENT_CLOSE
        If $g_iNewIDC <> $g_iIDC Then
            $g_iIDC = $g_iNewIDC
            GUISetCursor($g_iIDC)
        EndIf
        ToolTip("GUI Cursor #" & $g_iIDC & " (" & $g_aArray[$g_iIDC] & ")")
    WEnd

    GUIDelete()
EndFunc   ;==>Example

Func Increment()
    $g_iNewIDC = $g_iIDC + 1
    If $g_iNewIDC > 16 Then $g_iNewIDC = 0
EndFunc   ;==>Increment
