BitAND

Performs a bitwise AND operation.Exécute un 'ET' logique bit à bit.

BitAND ( value1, value2 [, value n] )

ParametersParamètre

value1 The first number.Le premier nombre.
value2 The second number.Le deuxième nombre.
value n [optional][optional] False - i, v, x ... The nth number - up to 255 values can be specified.Le nième nombre - Jusqu'à 255 valeurs peuvent être spécifiées.

Return ValueRemarque

Returns the value of the parameters bitwise-AND'ed together.Retourne le "ET" logique entre les valeurs entrées en paramètres.
Bit operations are performed as 32-bit integers.Les opérations sont effectuées comme entiers 32 bits.

Remarks<i0>

Remember hex notation can be used for numbers.La notation hexadécimale peut être utilisée comme arguments.
BitAND() returns 1 in each bit position where all input arguments have a 1 in the corresponding position and 0 in all others.BitAND() retourne 1 à chaque position de bit où tous les arguments donnés ont un 1 en position correspondante et 0 dans les autres cas.

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Note:Note : "b" is the symbol for byte."b" est le symbole de l'octet.

    ; Assign a Local variable the bitwise AND operation of 1 and 0.Assigne une variable 'Local' avec le résultat de l'opération AND de 1 et 0.
    Local $iBitAND1 = BitAND(1, 0)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitAND1)

    ; Assign a Local variable the bitwise AND operation of 1 and 1.Assigne une variable 'Local' avec le résultat de l'opération AND de 1 et 1.
    Local $iBitAND2 = BitAND(1, 1)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitAND2)

    ; Assign a Local variable the bitwise AND operation of 13 (1101b) and 7 (0111b).Assigne une variable 'Local' avec opération AND de 13 (1101b) et 7 (0111b).
    Local $iBitAND3 = BitAND(13, 7) ; 1101b AND 0111b = 0101b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitAND3)

    ; Assign a Local variable the bitwise AND operation of 2 (0010b), 3 (0011b) and 6 (0110b).Assigne une variable 'Local' avec le résultat de l'opération AND de 2 (0010b), 3 (0011b) et 6 (0110b).
    Local $iBitAND4 = BitAND(2, 3, 6) ; 0010b AND 0011b AND 0110b = 0010b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitAND4)
EndFunc   ;==>Example
