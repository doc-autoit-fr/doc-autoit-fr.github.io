TCPConnect

Create a socket connected to an existing server.Crée un socket connecté à un serveur existant.

TCPConnect ( IPAddr, port )

ParametersParamètres

IPAddr Internet Protocol dotted address(IpV4) as "192.162.1.1".L'adresse IPv4 séparée par des points tel que :"192.162.1.1".
port port on which the created socket will be connected.Port sur lequel le socket créé sera connecté.

Return ValueValeur de retour

Success:Succès : the main socket identifier.Retourne l'identifiant principale du socket.
Failure:Échec : -1 or 0 and sets the @error flag to non-zero.Retourne -1 ou 0, et @error <> 0.
@error:@error : -2 not connected.
1 IPAddr is incorrect.1 = IPAddr incorrect.
2 port is incorrect.2 = port incorrect.
10060 Connection timed out.
    A connection attempt failed because the connected party did not properly respond after a period of time,
    or the established connection failed because the connected host has failed to respond.
    See connect timeout example.
Windows API WSAGetError return value (see MSDN).Valeur retournée par l'API Windows WSAGetError (Consultez MSDN).

RemarksRemarques

This function is used by a client to communicate with the server.Aucune.

#include <MsgBoxConstants.au3>

; I am the client, start me after the server!Je suis le client, démarrez-moi après le serveur! (Start first the TCPAccept example script).(Démarrez d'abord l'exemple du script TCPAccept).

Example()

Func Example()
    TCPStartup() ; Start the TCP service.Démarre le service TCP.

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

    ; Assign Local variables the loopback IP Address and the Port.Assigne des variables locales avec l'adresse IP et le port.
    Local $sIPAddress = "127.0.0.1" ; This IP Address only works for testing on your own computer.Cette adresse IP ne fonctionne que pour les tests sur votre propre ordinateur.
    Local $iPort = 65432 ; Port used for the connection.Port utilisé pour la connexion.

    ; Assign a Local variable the socket and connect to a Listening socket with the IP Address and Port specified.Assigne une variable locale avec le socket et se connecte à un socket d'écoute avec l'adresse IP et le port spécifié.
    Local $iSocket = TCPConnect($sIPAddress, $iPort)

    ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
    If @error Then
        ; The server is probably offline/port is not opened on the server.Le serveur est probablement hors ligne ou le port n'est pas ouvert sur le serveur.
        Local $iError = @error
        MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Could not connect, Error code: " & $iError)
        Return False
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Connection successful")
    EndIf

    ; Close the socket.Ferme le socket.
    TCPCloseSocket($iSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Close the TCP service.Ferme le service TCP.
EndFunc   ;==>OnAutoItExit

TCPConnect with timeout

#include <MsgBoxConstants.au3>

; I am the client, start me after the server no more than 10sec ! (The server script is the TCPAccept example script).

Example()

Func Example()
    TCPStartup() ; Start the TCP service.Démarre le service TCP.

    ; Register OnAutoItExit to be called when the script is closed.Inscrit OnAutoItExit qui sera appelée lorsque le script se fermera.
    OnAutoItExitRegister("OnAutoItExit")

    ; Assign Local variables the loopback IP Address and the Port.Assigne des variables locales avec l'adresse IP et le port.
    Local $sIPAddress = "127.0.0.1" ; This IP Address only works for testing on your own computer.Cette adresse IP ne fonctionne que pour les tests sur votre propre ordinateur.
    Local $iPort = 65432 ; Port used for the connection.Port utilisé pour la connexion.

    Opt("TCPTimeout", 1000)
    Local $nMaxTimeout = 10 ; script will abort if no server available after 10 secondes

    Local $iSocket, $iError

    While 1
        ; Assign a Local variable the socket and connect to a Listening socket with the IP Address and Port specified.Assigne une variable locale avec le socket et se connecte à un socket d'écoute avec l'adresse IP et le port spécifié.
        $iSocket = TCPConnect($sIPAddress, $iPort)

        ; If an error occurred display the error code and return False.Si une erreur s'est produite, affiche le code d'erreur et retourne.
        If @error = 10060 Then
            ; Timeout occurs try again
            $nMaxTimeout -= 1
            If $nMaxTimeout < 0 Then
                MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Could not connect, after " & 10 - $nMaxTimeout & " TimeOut")
                Return False
            EndIf
            ContinueLoop
        ElseIf @error Then
            $iError = @error
            ; The server is probably offline/port is not opened on the server.Le serveur est probablement hors ligne ou le port n'est pas ouvert sur le serveur.
            MsgBox(BitOR($MB_SYSTEMMODAL, $MB_ICONHAND), "", "Could not connect, Error code: " & $iError)
            Return False
        Else
            MsgBox($MB_SYSTEMMODAL, "", "Connection successful after " & 10 - $nMaxTimeout & " TimeOut")
            ExitLoop
        EndIf

    WEnd

    ; Close the socket.Ferme le socket.
    TCPCloseSocket($iSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Close the TCP service.Ferme le service TCP.
EndFunc   ;==>OnAutoItExit
