BitXOR

Performs a bitwise exclusive OR (XOR) operation.Exécute un "OU EXCLUSIF" logique bit à bit.

BitXOR ( value1, value2 [, value n] )

ParametersParamètres

value1 The first number.Le premier nombre.
value2 The second number.Le deuxième nombre.
value n [optional][optionnel] The nth number - up to 255 values can be specified.Le nième nombre - Jusqu'à 255 valeurs peuvent être spécifiées.

Return ValueValeur de retour

Returns the value of the parameters bitwise-XOR'ed together.Retourne le "OU EXCLUSIF" bit à bit des valeurs entrées en paramètres.
Bit operations are performed as 32-bit integers.Les opérations sont effectuées comme entiers 32 bits.

RemarksRemarques

Remember hex notation can be used for numbers.La notation hexadécimale peut être utilisée comme arguments.
BitXOR() returns 1 in a bit position if there are an odd number of 1's in the corresponding bit position in all the input arguments, and 0 otherwise.BitXOR() retourne 1 pour chaque position de bit où on a un nombre impair de 1 en position correspondante dans tous les arguments, et 0 sinon.

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Note:Note : "b" is the symbol for byte."b" est le symbole de l'octet.

    ; Assign a Local variable the bitwise XOR operation of 1 and 0.Assigne une variable locale du XOR fonctionnement de 1 et 0.
    Local $iBitXOR1 = BitXOR(1, 0) ; 0001b XOR 0000b = 0001b0001b 0000b XOR = 0001b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitXOR1)

    ; Assign a Local variable the bitwise XOR operation of 1 and 1.Assigne une variable locale du XOR fonctionnement de 1 et 1.
    Local $iBitXOR2 = BitXOR(1, 1) ; 0001b XOR 0001b = 0000b0001b 0001b XOR = 0000b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitXOR2)

    ; Assign a Local variable the bitwise XOR operation of 13 (1101b) and 7 (0111b).Assigne une variable locale de l'opération au niveau du bit XOR de 13 (1101b) et 7 (0111b).
    Local $iBitXOR3 = BitXOR(13, 7) ; 1101b XOR 0111b = 1010b1101b 0111b XOR = 1010B

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitXOR3)

    ; Assign a Local variable the bitwise XOR operation of 2 (0010b), 3 (0011b) and 6 (0110b).Assigne une variable locale de l'opération au niveau du bit XOR de deux (0010b), 3 (??0011b) et 6 (0110b).
    Local $iBitXOR4 = BitXOR(2, 3, 6) ; 0010b XOR 0011b XOR 0110b = 0111b0010b 0011b XOR XOR 0110b = 0111b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitXOR4)
EndFunc   ;==>Example
