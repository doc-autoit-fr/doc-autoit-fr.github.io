IniWrite

Writes a value to a standard format .ini file.Écrit une valeur dans un fichier ini de format standard.

IniWrite ( "filename", "section", "key", "value" )

ParametersParamètres

filename The filename of the .ini file.Le nom du fichier ini.
section The section name in the .ini file.Le nom de la section dans le fichier ini.
key The key name in the .ini file.Le nom de la clé dans le fichier ini.
value The value to write/change.La valeur à écrire/changer.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if file is read-only.Retourne 0 si le fichier est en lecture seule.

RemarksRemarques

A standard ini file looks like:Un fichier ini standard ressemble à :
[SectionName][NomdelaSection]
Key=Valueclé=Valeur

If the file does not already exist, it is created so long as the directory exists.Si le fichier n'existe pas, il est créé, autant que le répertoire existe. Keys and/or sections are added to the end and are not sorted in any way.Les clés et/ou les sections sont ajoutés à la fin et ne sont pas triés en aucune façon.

If you would like to wrap quotes around an ini value, then you need to double up the quotation marks as the first set of quotation marks are stripped.Lorsque vous écrivez une valeur qui est entre guillemets, les guillemets sont supprimés.Pour pouvoir écrire des guillemets dans la valeur, vous devez mettre des doubles guillemets entouré par des simple guillemets. For example:Par exemple : ""This is a test"" will produce "This is a test" when using IniRead() or such-like.""Ceci est un test"" va produire "Ceci est un test" lors de l'utilisation de IniRead() ou semblable.

Leading and trailing whitespace is stripped.Les espaces de début et de fin sont supprimés. In order to preserve the whitespace, the string must be quoted.Pour conserver les espaces, la chaîne de caractère doit être entre guillemets. For example, "    this is a test" will preserve the whitespace but per above, the quotation marks are stripped.

Multi-line values are not possible.Une valeur sur plusieurs lignes n'est pas possible.

If you want to use an ini file with Unicode encoding, first create an .ini file by using the FileOpen() function with the mode parameter set to a Unicode parameter.Si vous desirez utiliser un fichier ini avec un encodage Unicode, premièrement créez un fichier .ini en utilisant la fonction FileOpen() avec le paramètre de mode positionné sur Unicode.

ExampleExemple

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Write the value of 'AutoIt' to the key 'Title' and in the section labelled 'General'.Ecrit la valeur de 'AutoIt' à la clé "Title" et dans la section "General".
    IniWrite($sFilePath, "General", "Title", "AutoIt")

    ; Read the INI file for the value of 'Title' in the section labelled 'General'.Lit le fichier INI pour la valeur de 'Title' dans la section 'General'.
    Local $sRead = IniRead($sFilePath, "General", "Title", "Default Value")

    ; Display the value returned by IniRead.Affiche la valeur retournée par IniRead.
    MsgBox($MB_SYSTEMMODAL, "", "The value of 'Title' in the section labelled 'General' is: " & $sRead)

    ; Delete the key labelled 'Title'.Supprime la clé 'Title'.
    IniDelete($sFilePath, "General", "Title")

    ; Read the INI file for the value of 'Title' in the section labelled 'General'.Lit le fichier INI pour la valeur de 'Title' dans la section 'General'.
    $sRead = IniRead($sFilePath, "General", "Title", "Default Value")

    ; Display the value returned by IniRead.Affiche la valeur retournée par IniRead. Since there is no key stored the value will be the 'Default Value' passed to IniRead.Comme il n'y a pas de clé stocké la valeur sera la 'Valeur par défaut' passée à IniRead.
    MsgBox($MB_SYSTEMMODAL, "", "The value of 'Title' in the section labelled 'General' is: " & $sRead)

    ; Delete the INI file.Supprime le fichier INI.
    FileDelete($sFilePath)
EndFunc   ;==>Example
