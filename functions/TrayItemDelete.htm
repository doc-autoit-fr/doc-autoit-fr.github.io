TrayItemDelete

Deletes a menu/item control from the tray menu.Supprime un contrôle menu ou item dans le menu de la zone de notification.

TrayItemDelete ( controlID )

ParametersParamètres

controlID The control identifier (controlID) as returned by a TrayCreateItem or TrayCreateMenu function.L'identifiant du contrôle (controlID) tel que retourné par la fonction TrayCreateItem ou TrayCreateMenu fonction.

Return ValueValeur de retour

Success:Succès : 1.
Failure:Échec : 0.Retourne 0.

RelatedLiens

TrayCreateItem, TrayCreateMenu

Example()

Func Example()
    Local $idDelete = TrayCreateItem("Delete")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

            Case $idDelete
                ; Display a message box to ask whether or not to delete the 'Delete' item.Affiche une boîte de message pour demander si oui ou non il faut supprimer l'item 'Delete'. If the return value of MsgBox is equal to $IDYES then delete the item.Si la valeur de retour de MsgBox est égale à $IDYES alors supprime l'item.
                If MsgBox(BitOR($MB_YESNO, $MB_SYSTEMMODAL), "", "Do you want to delete the 'Delete' tray menu item?") = $IDYES Then
                    TrayItemDelete($idDelete) ; Delete the tray menu item.Supprimer l'élément de menu de la barre.
                EndIf

            Case $idExit ; Exit the loop.Sort de la boucle.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
