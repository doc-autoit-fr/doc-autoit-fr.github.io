Shutdown

Shuts down the system.Arrête le système.

Shutdown ( code )

ParametersParamètres

code A combination of shutdown codes.Une combinaison des codes d'arrêt. See "remarks".Voir "Remarques".

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 and sets the @error flag to Windows API GetLastError().Retourne 0 et affecte @error avec l'API Windows GetLastError().

RemarksRemarques

The shutdown code is a combination of the following values:Le code d'arrêt est une combinaison des valeurs suivantes.
    $SD_LOGOFF (0) = Logoff
    $SD_SHUTDOWN (1) = Shutdown
    $SD_REBOOT (2) = Reboot
    $SD_FORCE (4) = Force
    $SD_POWERDOWN (8) = Power down
    $SD_FORCEHUNG (16) = Force if hung
    $SD_STANDBY (32) = Standby
    $SD_HIBERNATE (64) = Hibernate

Constants are defined in AutoItConstants.au3.

Required values should be BitOR()'ed together.Les valeurs requises peuvent être combinées avec BitOR(). To shutdown and power down, for example, the code would be BitOR($SD_SHUTDOWN, $SD_POWERDOWN).Pour l'arrêt et la mise hors tension, par exemple, le code serait BitOR($SD_SHUTDOWN, $SD_POWERDOWN).

Standby or Hibernate are ignored if other codes are set.La Veille ou la Veille prolongée sont ignorées si d'autres codes sont définis.
