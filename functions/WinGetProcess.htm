WinGetProcess

Retrieves the Process ID (PID) associated with a window.Récupère l'identifiant du processus (PID) associé à une fenêtre.

WinGetProcess ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to get the PID.Le titre/hWnd/class de la fenêtre à traiter. See Title special definition.Consultez Définiton spéciale de titre.
text [optional][optionnel][optionnel] The text of the window to get the PID.Le texte de la fenêtre à traiter. Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : a process ID (PID).Retourne l'identifiant du processus (PID).
Failure:Échec : -1 if the window is not found.Retourne -1 si la fenêtre n'est pas trouvée.

RelatedLiens

ProcessList, ProcessWait, ProcessWaitClose

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Retrieve the PID of Notepad using the window handle returned by WinWait.Récupére le PID du Bloc-notes en utilisant Le handle de fenêtre retourné par WinWait.
    Local $iPID = WinGetProcess($hWnd)

    ; Display the PID of the window.Affiche le PID de la fenêtre.
    MsgBox($MB_SYSTEMMODAL, "", "The PID is: " & $iPID)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
