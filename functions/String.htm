String

Returns the string representation of an expression.Retourne la représentation chaîne d'une expression.

String ( expression )

ParametersParamètres

expression An expression to convert into a string.Une expression à convertir en chaîne.

Return ValueValeur de retour

Returns a string.Retourne une chaîne.

RemarksRemarques

Maximum string length is 2147483647 characters (but keep in mind that no line in an AutoIt script can exceed 4095 characters.)La longueur maximum d'une chaîne est 2 147 483 647 caractères (notez qu'aucune ligne ne peut dépasser 4095 caractères dans un script AutoIt.)

ExampleExemple

#include <MsgBoxConstants.au3>

Local $sString = String(10) ; The expression passed to String will return a string representation, currently 10 is a number.L'expression passée à String renverra une représentation de chaîne, ici "10".
MsgBox($MB_SYSTEMMODAL, "", "$sString contains the value: " & $sString & " and IsString returns: " & IsString($sString))
