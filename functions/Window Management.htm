Window functions ReferenceWindow - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
WinActivate Activates (gives focus to) a window.Active (donne le focus) à une fenêtre.
WinActive Checks to see if a specified window exists and is currently active.Vérifie si une fenêtre spécifiée existe et si elle est active.
WinClose Closes a window.Ferme une fenêtre.
WinExists Checks to see if a specified window exists.Vérifie si la fenêtre indiquée existe.
WinFlash Flashes a window in the taskbar.Fait clignoter une fenêtre dans la barre des tâches.
WinGetCaretPos Returns the coordinates of the caret in the foreground window.Retourne les coordonnées du curseur dans la fenêtre du premier plan.
WinGetClassList Retrieves the classes from a window.Récupère les classes d'une fenêtre.
WinGetClientSize Retrieves the size of a given window's client area.Récupère la taille de la zone client d'une fenêtre.
WinGetHandle Retrieves the internal handle of a window.Récupère le handle interne d'une fenêtre.
WinGetPos Retrieves the position and size of a given window.Récupère la position et la taille de la fenêtre indiquée.
WinGetProcess Retrieves the Process ID (PID) associated with a window.Récupère l'identifiant du processus (PID) associé à une fenêtre.
WinGetState Retrieves the state of a given window.Récupère l'état d'une fenêtre donnée.
WinGetText Retrieves the text from a window.Récupère le texte d'une fenêtre.
WinGetTitle Retrieves the full title from a window.Récupère le titre complet d'une fenêtre.
WinKill Forces a window to close.Force la fermeture d'une fenêtre.
WinList Retrieves a list of windows.Récupère une liste de fenêtres.
WinMenuSelectItem Invokes a menu item of a window.Sélectionne un élément du menu d'une fenêtre.
WinMinimizeAll Minimizes all windows.Minimise toutes les fenêtres.
WinMinimizeAllUndo Undoes a previous WinMinimizeAll function.Annule un précédent WinMinimizeAll.
WinMove Moves and/or resizes a window.Déplace et/ou redimensionne une fenêtre.
WinSetOnTop Change a window's "Always On Top" attribute.Change l'attribut "Always On Top" d'une fenêtre.
WinSetState Shows, hides, minimizes, maximizes, or restores a window.Montre, cache, minimise, maximise ou restaure une fenêtre.
WinSetTitle Changes the title of a window.Change le titre d'une fenêtre.
WinSetTrans Sets the transparency of a window.Définit la transparence d'une fenêtre.
WinWait Pauses execution of the script until the requested window exists.Met le script en pause jusqu'à ce que la fenêtre indiquée existe.
WinWaitActive Pauses execution of the script until the requested window is active.Met le script en pause jusqu'à ce que la fenêtre indiquée soit active.
WinWaitClose Pauses execution of the script until the requested window does not exist.Met le script en pause jusqu'à ce que la fenêtre indiquée n'existe plus.
WinWaitNotActive Pauses execution of the script until the requested window is not active.Met le script en pause jusqu'à ce que la fenêtre indiquée ne soit plus active.
