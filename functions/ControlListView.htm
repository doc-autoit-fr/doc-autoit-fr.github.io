Function ReferenceRéférence de fonction

ControlListView

Sends a command to a ListView32 control.Envoie une commande à un contrôle ListView32.

ControlListView ( "title", "text", controlID, "command" [, option1 [, option2]] )

ParametersParamètres

title The title/hWnd/class of the window to access.Le titre/handle/class de la fenêtre à atteindre. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to access.Le texte de la fenêtre à accéder. See Text special definition.Consultez Définition spéciale de Texte .
controlID The control to interact with.Le contrôle avec lequel interagir. See Controls.Consultez Controls.
command The command to send to the control (see below).La commande à envoyer au contrôle (Voir ci-dessous).
option1 [optional][optionnel] Additional parameter required by some commands.Paramètre additionnel requis par certaines commandes.
option2 [optional][optionnel] Additional parameter required by some commands.Paramètre additionnel requis par certaines commandes.

Return ValueValeur de retour

Return depends on command as table below shows.Dépend de la commande comme le montre le tableau ci-dessous. In case of an error (such as an invalid command or window/control could not be found) then @error is set to 1.En cas d'erreur (telle qu'une commande ou fenêtre/contrôle invalide), @error = 1.

Command, Option1, Option2 OperationOpération
"DeSelect", From [, To] Deselects one or more items.Dé-sélectionne un ou plusieurs éléments.
"FindItem", "string to find" [, SubItem] Returns the item index of the string.Retourne l'index de la chaîne recherchée. Returns -1 if the string is not found.Retourne -1 si la chaîne n'est pas trouvée.
"GetItemCount" Returns the number of list items.Retourne le nombre d'éléments de la liste.
"GetSelected" [, option] Returns a string containing the item index of selected items.Retourne une chaîne contenant l'index des éléments sélectionnés. If option=0 (default) only the first selected item is returned.Si option=0 (défaut) seul l'index du premier élément sélectionné est retourné. If option=1 then all the selected items are returned delimited by |, e.g:Si option=1 alors tout les index sélectionnés sont retournés séparés par le caractère "|", par exemple: "0|3|4|10". If no items are selected a blank "" string is returned.Si aucun éléments n'est sélectionné, une chaine vide "" est retournée.
"GetSelectedCount" Returns the number of items that are selected.Retourne le nombre d'éléments qui sont sélectionnés.
"GetSubItemCount" Returns the number of subitems.Retourne le nombre de sous-éléments.
"GetText", Item, SubItem Returns the text of a given item/subitem.Retourne le texte d'un élément/sous élément donné.
"IsSelected", Item Returns 1 if the item is selected, otherwise returns 0.Retourne 1 si l'élément est sélectionné, sinon retourne 0.
"Select", From [, To] Selects one or more items.Sélectionne un ou plusieurs éléments.
"SelectAll" Selects all items.Sélectionne tous les éléments.
"SelectClear" Clears the selection of all items.Efface la sélection de tous les éléments.
"SelectInvert" Inverts the current selection.Inverse la sélection en cours.
"ViewChange", "view" Changes the current view.Change le type de vue en cours. Valid views are "list", "details", "smallicons", "largeicons".Les types de vues possibles sont :"list", "details", "smallicons", "largeicons".


In a "Details" view of a ListView32 control, the "item" can be thought of as the "row" and the "subitem" as the "column".Dans une vue "détaillée" d'un contrôle de type ListView32, un élément ("item") correspond à une ligne ("row") et un sous-élément ("subitem") à une colonne ("column").

RemarksRemarques

Some commands may fail when using a 32-bit AutoIt process to read from a 64-bit process.Certaines commandes peuvent échouer lors de l'utilisation d'un processus AutoIt 32-Bits pour lire un processus 64-bits. Likewise commands may fail when using a 64-bit AutoIt process to read from a 32-bit process.De même des commandes peuvent échouer lors de l'utilisation d'un processus AutoIt 64-Bits pour lire un processus 32-bits.
