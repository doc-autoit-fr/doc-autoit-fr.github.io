StringMid

Extracts a number of characters from a string.Extrait un nombre donné de caractères d'une chaîne.

StringMid ( "string", start [, count = -1] )

ParametersParamètres

string The string to evaluate.La chaîne à traiter.
start The character position to start.La position du caractère de départ. (1 = first character)(1 = premier caractère)
count [optional][optionnel][optionnel] The number of characters to extract.Le nombre de caractères à extraire. By default the entire remainder of the string.Par défaut tout le reste de la chaîne.

Return ValueValeur de retour

Returns the extracted string.Retourne la chaîne extraite.

RemarksRemarques

If start is out-of-bounds, an empty string is returned.Si start est hors limites, une chaîne vide est retournée. If start is valid but count is out-of-bounds, the entire remainder of the string is returned.Si start est valide, mais <e1>count</e1> hors limites, le reste de la chaîne est renvoyée.

ExampleExemple

#include <MsgBoxConstants.au3>

; Retrieve 5 characters from the 10th position in the string.Récupére 5 caractères à partir de la 10e position dans la chaîne.
Local $sString = StringMid("This is a sentence with whitespace.", 10, 5)
MsgBox($MB_SYSTEMMODAL, "", $sString)
