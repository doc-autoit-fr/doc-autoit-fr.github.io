IniRenameSection

Renames a section in a standard format .ini file.Renomme une section dans un fichier ini de format standard.

IniRenameSection ( "filename", "section", "new section" [, flag = 0] )

ParametersParamètres

filename The filename of the .ini file.Le nom du fichier ini.
section The section name in the .ini file.Le nom de la section dans le fichier ini.
new section The new section name.Le nouveau nom de la section.
flag [optional][optionnel][optionnel]
    $FC_NOOVERWRITE(0) = (default) Fail if "new section" already exists.
    $FC_OVERWRITE(1) = Overwrite "new section". This will erase any existing keys in "new section".

Constants are defined in FileConstants.au3.

Return ValueValeur de retour

Success:Succès : Non-zero.Retourne une valeur différente de zéro.
Failure:Échec : 0 and may sets the @error flag to non-zero, if renaming failed because the section already exists (only when flag = 0).Retourne 0 et @error <> 0, si le renommage a échoué parce que la section existe déjà (seulement quand flag = 0).

RemarksRemarques

A standard ini file looks like:Un fichier ini standard ressemble à :
[SectionName][NomdelaSection]
Key=Valueclé=Valeur

ExampleExemple

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath that will be read/written to.Créezune constante de portée locale avec le chemin du fichier qui sera lu/écrit.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Create an INI section structure as a string.Crée une structure de section INI sous forme de chaîne.
    Local $sSection = "Title=AutoIt" & @LF & "Version=" & @AutoItVersion & @LF & "OS=" & @OSVersion

    ; Write the string to the sections labelled 'General', 'Version' and 'Other'.Écrit la chaîne dans le sections intitulées 'General', 'Version' et 'Other'.
    IniWriteSection($sFilePath, "General", $sSection)
    IniWriteSection($sFilePath, "Version", $sSection)
    IniWriteSection($sFilePath, "Other", $sSection)

    ; Rename the section labelled 'General' to 'System'.Renomme la section 'General' en 'System'.
    IniRenameSection($sFilePath, "General", "System")

    ; Read the INI section names.Lit les noms des sections INI. This will return a 1 dimensional array.Ce qui renvoie un tableau à 1 dimension.
    Local $aArray = IniReadSectionNames($sFilePath)

    ; Check if an error occurred.Vérifie si une erreur s'est produite.
    If Not @error Then
        ; Enumerate through the array displaying the section names.Énumére le tableau où figurent les noms des sections.
        For $i = 1 To $aArray[0]
            MsgBox($MB_SYSTEMMODAL, "", "Section: " & $aArray[$i])
        Next
    EndIf

    ; Delete the INI file.Supprime le fichier INI.
    FileDelete($sFilePath)
EndFunc   ;==>Example
