FileReadToArray

Reads the specified file into an array.Lit le fichier spécifié et construit un tableau avec les lignes.

FileReadToArray ( "filehandle/filename" )

ParametersParamètres

filehandle/filename The handle of a file, as returned by a previous call to FileOpen().Le handle d'un fichier, renvoyé par l'utilisation de la commande FileOpen(). Alternatively you may use a string filename as the first parameter.Alternativement le nom du fichier.

Return ValueValeur de retour

Success:Succès : a 1 dimension array containing one line of text per element and @extended set to the number of lines read.
Failure:Échec : sets the @error flag to non_zero.Retourne @error <> 0.
@error:@error : 1 = Error opening specified file1 = Erreur à l'ouverture du fichier
2 = Empty file2 = Fichier vide

RemarksRemarques

Use @extended or UBound() to determine the number of lines read.

RelatedLiens

FileRead, FileReadLine

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Read the current script file into an array using the filepath.Lit le fichier du script courant dans un tableau en utilisant le chemin du fichier.
    Local $aArray = FileReadToArray(@ScriptFullPath)
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "There was an error reading the file. @error:@error : " & @error) ; An error occurred reading the current script file.Une erreur s'est produite lecture du fichier de script courant.
    Else
        For $i = 0 To UBound($aArray) - 1 ; Loop through the array.Boucle dans le tableau.
            MsgBox($MB_SYSTEMMODAL, "", $aArray[$i]) ; Display the contents of the array.Affiche le contenu du tableau.
        Next
    EndIf
EndFunc   ;==>Example
