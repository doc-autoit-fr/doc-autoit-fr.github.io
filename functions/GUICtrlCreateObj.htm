GUICtrlCreateObj

Creates an ActiveX control in the GUI.Crée un contrôle ActiveX dans la GUI.

GUICtrlCreateObj ( ObjectVar, left, top [, width [, height]] )

ParametersParamètre

ObjectVar A variable pointing to a previously opened objectUne variable qui pointe sur un objet précédemment ouvert
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional][optional] False - i, v, x ... The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional][optional] False - i, v, x ... The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).

Return ValueRemarque

Success: the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure: 0.Retourne 0.

Remarks<i0>

This function attempts to embed an 'ActiveX Control' or a 'Document Object' inside the GUI.Cette fonction tente d'intégrer un 'contrôle ActiveX' ou un 'Document Object' dans une GUI.
Not every control can be embedded.Pas tous les contrôles peuvent être intégrés. They must at least support an 'IDispatch' interface.Ils doivent supporter au moins une interface 'IDispatch'.

See the Obj/COM Reference for more information about Objects.Consultez Référence Obj/COM pour plus d'informations sur les objets.

'Document Objects' will only be visible if the Windows style $WS_CLIPCHILDREN has been used in GUICreate().'Document Objects' ne sera visible que si le style Windows $WS_CLIPCHILDREN Windows a été utilisé dans GUICreate().

The GUI functions GUICtrlRead() and GUICtrlSet have no effect on this control.Les fonctions GUI GUICtrlRead() et GUICtrlSet n'ont aucun effet sur ce contrôle. The object can only be controlled using 'methods' or 'properties' on the $ObjectVar.L'objet ne peut être contrôlée qu'en utilisant des 'méthodes' ou 'propriétés' de l'objet $ObjectVar.

#include <GUIConstantsEx.au3>

Example()

; Simple example:Exemple simple: Embedding an Internet Explorer Object inside an AutoIt GUIIncorporation d'un objet Internet Explorer dans une GUI AutoIt
; ; See also: http://msdn.microsoft.com/workshop/browser/webbrowser/reference/objects/internetexplorer.aspConsultez aussi : http://msdn.microsoft.com/workshop/browser/webbrowser/reference/objects/internetexplorer.asp
Func Example()
    Local $idButton_Back, $idButton_Forward
    Local $idButton_Home, $idButton_Stop, $iMsg

    Local $oIE = ObjCreate("Shell.Explorer.2")

    ; Create a simple GUI for our outputCrée une GUI simple pour notre résultat
    GUICreate("Embedded Web control Test", 640, 580, (@DesktopWidth - 640) / 2, (@DesktopHeight - 580) / 2, BitOR($WS_OVERLAPPEDWINDOW, $WS_CLIPSIBLINGS, $WS_CLIPCHILDREN))
    GUICtrlCreateObj($oIE, 10, 40, 600, 360)
    $idButton_Back = GUICtrlCreateButton("Back", 10, 420, 100, 30)
    $idButton_Forward = GUICtrlCreateButton("Forward", 120, 420, 100, 30)
    $idButton_Home = GUICtrlCreateButton("Home", 230, 420, 100, 30)
    $idButton_Stop = GUICtrlCreateButton("Stop", 330, 420, 100, 30)

    GUISetState(@SW_SHOW) ;Show GUIAffiche l'interface utilisateur

    $oIE.navigate("http://www.autoitscript.com")

    ; Loop until the user exits.Création d'un objet chemin
    While 1
        $iMsg = GUIGetMsg()

        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                ExitLoop
            Case $iMsg = $idButton_Home
                $oIE.navigate("http://www.autoitscript.com")
            Case $iMsg = $idButton_Back
                $oIE.GoBack
            Case $iMsg = $idButton_Forward
                $oIE.GoForward
            Case $iMsg = $idButton_Stop
                $oIE.Stop
        EndSelect

    WEnd

    GUIDelete()
EndFunc   ;==>Example
