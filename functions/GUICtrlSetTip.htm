GUICtrlSetTip

Sets the tip text associated with a control.Définit le texte de l'info-bulle associée à un contrôle.

GUICtrlSetTip ( controlID, tiptext [, "title" [, icon [, options]]] )

ParametersParamètres

controlID The control identifier (controlID) as returned by a GUICtrlCreate...() function, or -1 for the last created control.L'identifiant du contrôle (controlID) tel que renvoyé par une fonction GUICtrlCreate...(), ou -1 pour désigner le dernier contrôle créé.
tiptext Tip text that will be displayed when the mouse is hovered over the control.Texte de la bulle qui s'affiche lorsque la souris survole le contrôle.
title [optional] The title for the tooltip.
icon [optional] Pre-defined icon to show next to the title: requires a title.
    $TIP_NOICON (0) = No icon
    $TIP_INFOICON (1) = Info icon
    $TIP_WARNINGICON (2) = Warning icon
    $TIP_ERRORICON (3) = Error Icon

Constants are defined in "AutoItConstants.au3".
options [optional] Sets different options for how the tooltip will be displayed (Can be added together):Définit différentes options déterminant la façon avec laquelle l'info-bulle sera affichée (Peuvent être additionnées ensemble):
    $TIP_BALLOON (1) = Display as Balloon Tip.
    $TIP_CENTER (2) = Center the tip horizontally along the control.

Constants are defined in "AutoItConstants.au3".

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0.Retourne 0.

RemarksRemarques

This tip text is displayed in a tooltip rectangular area.Le texte de l'Info-bulle est affiché dans une zone rectangulaire.
To skip an optional parameter, leaving its default value intact, use the Default keyword..Pour sauter un paramètre optionnel, laissez sa valeur par défaut intacte, utilisez le mot-clé Default.
You may use @CR or @LF to create multi-line tooltips.Vous pouvez utiliser @CR ou @LF pour créer des info-bulles multi-lignes.
To display an icon, you must specify a non-empty title.Pour afficher une icône, vous devez indiquer un titre non vide. The icon appears on the same row as the title and thus requires a title to be present.L'icône apparaît sur la même ligne que le titre et nécessite donc qu'un titre soit présent.

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI control tip") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUICtrlCreateLabel("my label", 10, 20)
    GUICtrlSetTip(-1, "tip of my label")

    GUISetState(@SW_SHOW)

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
