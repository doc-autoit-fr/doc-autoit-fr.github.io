IsBinary

Checks if a variable or expression is a binary type.Vérifie si une variable ou une expression est du type binaire.

IsBinary ( expression )

ParametersParamètres

expression The variable or expression to check.La variable ou l'expression à analyser.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if expression is not binary type.Retourne 0 si l'expression n'est pas de type binaire.

RemarksRemarques

See language datatypes for a detailed description.Consultez Types de données du langage pour une description détaillée.

RelatedLiens

ExampleExemple

#include <MsgBoxConstants.au3>

Local $dBinary = Binary("0x00204060")
Local $sString = "0x00204060"

MsgBox($MB_SYSTEMMODAL, "", "Returns 1 as the variable is binary: " & IsBinary($dBinary))
MsgBox($MB_SYSTEMMODAL, "", "Returns 0 as the variable is a string: " & IsBinary($sString))
