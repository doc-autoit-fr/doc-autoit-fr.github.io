ObjCreate

Creates a reference to a COM object from the given classname.Crée une référence à un objet COM à partir d'un nom de classe donné.

ObjCreate ( "classname" [, "servername" [, "username" [, "password"]]] )

ParametersParamètres

classname The class of the object in the following format:Le nom de la classe de l'objet dans le format suivant : "appname.objectype"
It can also be a string representation of the CLSID.Il peut également être une représentation de chaîne du CLSID.
servername [optional] name of a remote computer from which the object must be obtained.nom d'un ordinateur distant à partir duquel l'objet doit être obtenue.
username [optional] name of a usercode on the remote computer.code utilisateur sur l'ordinateur distant.
You have to enter this in the format "computer\usercode" or "domain\usercode".Vous devez entrer ce code dans le format "ordinateur\usercode" ou "domaine\usercode".
password [optional] password for the usercode on the remote computer.mot de passe pour le code utilisateur sur l'ordinateur distant.

Return ValueValeur de retour

Success: an object.Retourne un objet.
Failure: sets the @error flag to non-zero.met @error <> 0.

RemarksRemarques

Use ObjCreate() if you want to have a new instance of the referring application.Utilisez ObjCreate() si vous voulez avoir une nouvelle instance de l'application référencé.
If you want to connect to an existing process, use ObjGet() instead.Si vous souhaitez vous connecter à un processus existant, utilisez à la place ObjGet().

Keep in mind that not all computers have the same set of Objects.Gardez à l'esprit que tous les ordinateurs ont le même ensemble d'Objets. So always check for errors after calling ObjCreate().Donc testez toujours les erreurs après un appel de ObjCreate(). To check errors use ObjEvent().Pour tester les erreurs utilisez ObjEvent() .

The following requirements apply if you want to access objects on remote computers:Les exigences suivantes s'appliquent si vous voulez accéder à des objets sur des ordinateurs distants :


  • The user running the script must have the appropriate permissions.L'utilisateur qui exécute le script doit avoir des droits appropriés.

  • The Object on the remote computer must support DCOM (Distributed COM)L'Objet sur l'ordinateur distant doit supporter DCOM (Distributed COM)

  • The remote computer must have 'Remote Registry Service' and 'File and Printer sharing' services running.L'ordinateur distant doit avoir les services 'Remote Registry Service' et 'File and Printer sharing' en cours d'exécution.



See the Obj/COM Reference for more information about Objects.Consultez Référence Obj/COM pour plus d'informations sur les objets.

ExampleExemple

Example 1Exemple 1

#include <MsgBoxConstants.au3>

; Counting the number of open shell windowsCompte le nombre de fenêtres shell ouvertes

Local $oShell = ObjCreate("shell.application") ; Get the Windows Shell ObjectObtient l'objet Shell Windows
Local $oShellWindows = $oShell.Windows() ; Get the collection of open shell WindowsObtient la collection de Windows shell ouvertes

If IsObj($oShellWindows) Then
    Local $sString = "" ; String for displaying purposesChaîne à des fins d'affichage

    For $oWnd In $oShellWindows ; Count all existing shell windowsCompte toutes les fenêtres shell existantes
        $sString &= $oWnd.LocationName & @CRLF
    Next

    MsgBox($MB_SYSTEMMODAL, "", "Shell Windows:" & @CRLF & "You have the following shell windows:" & @CRLF & @CRLF & $sString)
EndIf

Example 2Exemple 2

#include <MsgBoxConstants.au3>

; Open the MediaPlayer on a REMOTE computerOuvre MediaPlayer sur un ordinateur distant

Local $oRemoteMedia = ObjCreate("MediaPlayer.MediaPlayer.1", "name-of-remote-computer")

If Not @error Then
    MsgBox($MB_SYSTEMMODAL, "Remote ObjCreate Test", "ObjCreate() of a remote Mediaplayer Object successful !")
    $oRemoteMedia.Open(@WindowsDir & "\media\tada.wav") ; Play sound if file is presentJoue un son si le fichier est présent
Else
    MsgBox($MB_SYSTEMMODAL, "Remote ObjCreate Test", "Failed to open remote Object. Error code: " & Hex(@error, 8))
EndIf
