WinSetTitle

Changes the title of a window.Change le titre d'une fenêtre.

WinSetTitle ( "title", "text", "newtitle" )

ParametersParamètres

title The title/hWnd/class of the window to change.Le titre/hWnd/class de la fenêtre à changer. See Title special definition.Consultez Définiton spéciale de titre.
text The text of the window to change.Le texte de la fenêtre à changer. See Text special definition.Consultez Définition spéciale de Texte .
newtitle The new title to give to the window.Le nouveau titre à donner à la fenêtre.

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if the window is not found.Retourne 0 si la fenêtre n'existe pas.

RemarksRemarques

If multiple windows match the criteria the title of most recently active window is changed.Si plusieurs fenêtres correspondent aux critères, le titre de la fenêtre la plus récente est changé.

ExampleExemple

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Set the title of the Notepad window using the handle returned by WinWait.Définit le titre de la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinSetTitle($hWnd, "", "New Notepad Title - AutoIt")

    ; Wait for 2 seconds to display the Notepad window and the new title.Attend 2 secondes pour afficher la fenêtre du Bloc-notes et le nouveau titre.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
