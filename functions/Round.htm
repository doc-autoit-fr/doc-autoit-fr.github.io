Round

Returns a number rounded to a specified number of decimal places.Arrondi un décimal avec un nombre spécifié de décimales.

Round ( expression [, decimalplaces] )

ParametersParamètres

expression Any valid numeric expression.Toute expression numérique valide.
decimalplaces [optional][optionnel] Number indicating how many places to the right of the decimal are included in the rounding. If omitted, Round returns an integer.

Return ValueValeur de retour

Returns rounded number.Retourne le nombre arrondis.

RemarksRemarques

The decimalplaces parameter can be negative which allows you to round to the ones, tens, hundred, etc. place. Note that up to fifteen digits of a number are displayed, and note that decimalplaces will never pad a number with trailing zeros.Notez qu'au plus quinze chiffres d'un nombre sont affichés, et notez que decimalplaces ne comblera jamais les décimales avec des zéros.

Example 1Exemple 1

#include <MsgBoxConstants.au3>

Local $iRound1 = Round(-1.582, 1) ; Returns -1.6.Retourne -1.6.
Local $iRound2 = Round(3.1415, 9) ; No change is made.Aucun changement n'est effectué.
Local $iRound3 = Round(123.5, -1) ; Returns 120Retourne 120

MsgBox($MB_SYSTEMMODAL, "", "The following values were rounded: " & @CRLF & _
        $iRound1 & @CRLF & $iRound2 & @CRLF & $iRound3)

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local variable an Array which will contain the rounded numbers.Assigne une variable locale avec un tableau qui contiendra les nombres arrondis.
    Local $aRound[5]

    $aRound[0] = Round(-1.582, 1) ; Returns -1.6.Retourne -1.6.
    $aRound[1] = Round(3.1415, 9) ; Returns 3.1415Retourne 3.1415
    $aRound[2] = Round(123.5, -1) ; Returns 120.Retourne 120.
    $aRound[3] = Round(123.5) ; Returns 124.Retourne 124.
    $aRound[4] = Round(50) ; Returns 50.Retourne 50.

    ; Display the results.Affiche les résultats.
    For $i = 0 To UBound($aRound) - 1
        MsgBox($MB_SYSTEMMODAL, "", "Round" & $i & ": " & $aRound[$i] & @CRLF)
    Next
EndFunc   ;==>Example
