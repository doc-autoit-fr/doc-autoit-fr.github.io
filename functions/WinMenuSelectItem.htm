RemarksRemarques

You should note that underlined menu items actually contain a & character to indicate the underlining.Notez que les éléments de menu souligné contiennent un caractère & afin d'indiquer le soulignement. Thus, the menu item File would actually require the text "&File", and Convert would require "Con&vert" You can access menu items up to six levels deep; and the window can be inactive, minimized, and/or even hidden.Ainsi, l'élément du menu Fichier devra contenir l'élément "&Fichier", et Convert devra contenir l’élément "Con&vert".Vous pouvez accéder aux éléments du menu jusqu'à six niveaux de profondeur; et la fenêtre peut être inactive, minimisée et/ou cachée.

WinMenuSelectItem() will only work on standard menus.WinMenuSelectItem() ne fonctionne que sur des menus standard. Unfortunately, many menus in use today are actually custom written or toolbars "pretending" to be menus.Malheureusement, aujourd'hui, de nombreux menus ou barres d'outils sont personnalisés afin de "prétendre" d'être des menus. This is true for most Microsoft applications.Ceci est vrai pour la plupart des applications Microsoft.
