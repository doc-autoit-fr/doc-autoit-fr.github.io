DllClose

Closes a previously opened DLL.Ferme une DLL précédemment ouverte.

DllClose ( dllhandle )

ParametersParamètres

dllhandle The handle of a dll, as returned by a previous call to DllOpen().Le handle d'une dll, renvoyé par un précédent appel à DllOpen().

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

Upon termination, AutoIt automatically closes any dlls it opened but calling DllClose() is still a good idea.Lors de la fermeture du script, AutoIt ferme automatiquement toutes les dlls qu'il a ouvert, mais appeler DllClose() est toujours une bonne idée.

RelatedLiens

DllCall, DllOpen

ExampleExemple

Local $hDLL = DllOpen("user32.dll")
DllCall($hDLL, "int", "MessageBox", "hwnd", 0, "str", "Some text", "str", "Some title", "int", 0)
DllClose($hDLL)
