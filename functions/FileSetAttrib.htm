FileSetAttrib

Sets the attributes of one or more files/directories.Définit les attributs d'un ou plusieurs fichiers.

FileSetAttrib ( "file pattern", "+-RASHNOT" [, recurse = 0] )

ParametersParamètres

file pattern The path of the file(s) to set, e.g.Le chemin des Fichier(s) à modifier, ex : C:\*.au3, C:\Dir. (* and ?( caractères spéciaux * et ? wildcards accepted - See Remarks)acceptés - Voir Remarques)
+-RASHNOT Attribute(s) to set/clear.Attribut(s) à définir/effacer. e.g.Ex.: "+A", "+RA-SH"
recurse [optional]
    $FT_NONRECURSIVE (0) - no recursion (Default)
    $FT_RECURSIVE (1) - directories are recursed into.

Constants are defined in FileConstants.au3.

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if encountered any errors.Retourne 0 si des erreurs sont apparues.

RemarksRemarques

See FileFindFirstFile() for a discussion about wildcards.Consultez FileFindFirstFile() pour des précisions sur les caractères spéciaux.

The file pattern cannot contain spaces!Le motif du fichier ne peut pas contenir d'espaces!
The attributes that can be modified with the function are + or -:Les attributs qui peuvent être modifiés avec la fonction sont + ou -:
    "R" = READONLY
    "A" = ARCHIVE
    "S" = SYSTEM
    "H" = HIDDEN
    "N" = NORMAL
    "O" = OFFLINE
    "T" = TEMPORARY

(Note that you cannot set the compressed/directory attributes with this function.)(Notez que vous ne pouvez pas définir les attributs compressé et répertoire avec cette fonction.)

ExampleExemple

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>

If MsgBox(BitOR($MB_YESNO, $MB_SYSTEMMODAL), "", "Note:Note : This is an example and shouldn't be run under normal circumstances. " & @CRLF & @CRLF & "Do you wish to run the example?") = $IDYES Then
    ; Mark all .au3 files in current directory as read-only and system.Marque tous les fichiers .au3 dans le répertoire courant comme en lecture seule et système.
    If Not FileSetAttrib(@ScriptDir & "\*.au3", "+RS") Then
        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "Problem setting attributes.")
    EndIf

    ; Mark all .bmp files in @HomeDrive and sub-directories writable and archived.Marque tous les fichiers .bmp dans @HomeDrive et dans les sous-répertoires en écriture libre et archivés.
    If Not FileSetAttrib(@HomeDrive & "\*.bmp", "-R+A", $FT_RECURSIVE) Then
        MsgBox($MB_SYSTEMMODAL, "ErrorErreur", "Problem setting attributes.")
    EndIf
EndIf
