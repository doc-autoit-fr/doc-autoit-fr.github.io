SplashImageOn

Creates a customizable image popup window.Crée une fenêtre d'image popup personnalisable.

SplashImageOn ( "title", "file" [, width [, height [, x pos [, y pos [, opt]]]]] )

ParametersParamètres

title Title for splash window.Titre de la fenêtre.
file full path\filename of image (BMP, GIF, or JPG)Chemin d'accès complet\nom du fichier image (BMP, GIF, ou JPG)
width [optional][optionnel][optionnel] width of window in pixels.largeur de la fenêtre en pixels. (default is 500)(500 par défaut)
height [optional][optionnel][optionnel] height of window in pixels.hauteur de la fenêtre en pixels. (default is 400)(400 par défaut)
x pos [optional][optionnel][optionnel] position from left (in pixels) of splash window.position de la fenêtre splash à partir de la gauche (en pixels). (default is centered)(centrée par défaut)
y pos [optional][optionnel][optionnel] position from top (in pixels) of splash window.position de la fenêtre splash à partir du haut (en pixels). (default is centered)(centrée par défaut)
opt [optional][optionnel][optionnel] Default is 'always on top/with title'Par défaut: toujours Au Premier Plan et avec Titre
Add up the following options you want:Ajoute les options que vous souhaitez.
    $DLG_NOTITLE (1) = borderless, titleless window
    $DLG_NOTONTOP (2) = Without "always on top" attribute
    $DLG_MOVEABLE (16) = Window can be moved

Constants are defined in AutoItConstants.au3.

Return ValueValeur de retour

None.Aucune.

RemarksRemarques

To skip an optional parameter, leaving its default value intact, use a value of -1.Pour sauter un paramètre optionnel, en laissant sa valeur par défaut intacte, utilisez la valeur -1.

The image is scaled to specified width and height and should be a Bitmap, GIF, or JPEG image.L'image est redimensionnée à la largeur et la hauteur spécifiée et devrait être une image Bitmap, GIF ou JPEG. Icons (ICOs) will not display, though the window still appears.Les icônes (ICO) ne seront pas affichés, malgré l'apparition de la fenêtre. PNG images throw an error message of failing to load, and so do invalid files/paths.Les images PNG provoquent un message d'erreur de chargement, tout comme les fichiers/chemins invalides.

Images are not embedded in the executable unless you use FileInstall()Les images ne sont pas incorporées dans l'exécutable à moins que vous n'utilisiez FileInstall().

Only one SplashImage/Text window is allowed at one time; so if you wish to cycle through multiple images/text, simply call SplashImageOn()/SplashTextOn() again with the new information.Une seule fenêtre splashImage/splashtext est permise en même temps, donc si vous souhaitez faire défiler plusieurs images/textes, utilisez simplement SplashImageOn()/SplashTextOn() à nouveau avec les nouvelles informations.

Splash with opt=1 cannot be moved and cannot be activated by click.Un splash avec opt=1 ne peut être déplacé et ne peut être activé par un clic.

RelatedLiens

SplashOff, SplashTextOn

ExampleExemple

Local $sDestination = "..\GUI\mslogo.jpg"

SplashImageOn("Splash Screen", $sDestination, 250, 50)
Sleep(3000)
SplashOff()
