File, Directory and Disk functions ReferenceFile, Directory and Disk - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
ConsoleRead Read from the STDIN stream of the AutoIt script process.Lit à partir du flux STDIN du processus du script AutoIt.
ConsoleWrite Writes data to the STDOUT stream.Écrit des données dans le flux STDOUT. Some text editors can read this stream as can other programs which may be expecting data on this stream.Certains éditeurs de texte peuvent lire ce flux comme d'autres programmes peuvent attendre les données de ce flux.
ConsoleWriteError Writes data to the STDERR stream.Écrit des données dans le flux STDERR. Some text editors can read this stream as can other programs which may be expecting data on this stream.Certains éditeurs de texte peuvent lire ce flux comme d'autres programmes peuvent attendre les données de ce flux.
DirCopy Copies a directory and all sub-directories and files (Similar to xcopy).Copie un répertoire et tous ses sous-répertoires et fichiers (similaire à XCopy).
DirCreate Creates a directory/folder.Créé un répertoire/dossier.
DirGetSize Returns the size in bytes of a given directory.Retourne la taille en octets d'un répertoire donné.
DirMove Moves a directory and all sub-directories and files.Déplace un répertoire et tous les sous-répertoires et fichiers qu'il contient.
DirRemove Deletes a directory/folder.Supprime un répertoire/dossier.
DriveGetDrive Returns an array containing the enumerated drives.Retourne un tableau contenant les lecteurs énumérés.
DriveGetFileSystem Returns File System Type of a drive.Retourne le type de système de fichiers d'un disque.
DriveGetLabel Returns Volume Label of a drive, if it has one.Retourne le nom de volume d'un lecteur s'il en a un.
DriveGetSerial Returns Serial Number of a drive.Retourne le numéro de série d'un lecteur.
DriveGetType Returns drive type.Retourne le type de lecteur.
DriveMapAdd Maps a network drive.Mappe un lecteur réseau.
DriveMapDel Disconnects a network drive.Déconnecte un lecteur réseau.
DriveMapGet Retrieves the details of a mapped drive.Récupère les informations d'un disque mappé.
DriveSetLabel Sets the Volume Label of a drive.Définit le nom de volume d'un lecteur.
DriveSpaceFree Returns the free disk space of a path in Megabytes.Retourne l'espace disque libre d'un lecteur en Méga-octet.
DriveSpaceTotal Returns the total disk space of a path in Megabytes.Retourne l'espace disque total d'un lecteur en Méga-octet.
DriveStatus Returns the status of the drive as a string.Retourne le status d'un disque sous forme de chaîne.
FileChangeDir Changes the current working directory.Change le répertoire de travail actuel.
FileClose Closes a previously opened file.Ferme un fichier texte précédemment ouvert.
FileCopy Copies one or more files.Copie un ou plusieurs fichiers.
FileCreateNTFSLink Creates an NTFS hardlink to a file or a directory.Crée un lien dur NTFS vers un fichier ou un répertoire.
FileCreateShortcut Creates a shortcut (.lnk) to a file.Crée un raccourci (.lnk) vers un fichier.
FileDelete Delete one or more files.Supprime un ou plusieurs fichiers.
FileExists Checks if a file or directory exists.Vérifie si un fichier ou un dossier existe.
FileFindFirstFile Creates a search handle, defined by a path and file mask.Retourne un "handle" de recherche, selon la chaîne de caractères de recherche de fichiers.
FileFindNextFile Returns the next filename defined by the search handle.Retourne un nom de fichier en fonction d'un précédent appel à FileFindFirstFile.
FileFlush Flushes the file's buffer to disk.Vide la mémoire tampon du fichier sur le disque.
FileGetAttrib Returns a code string representing a file's attributes.Retourne une chaîne de code représentant les attributs d'un fichier.
FileGetEncoding Determines the text encoding used in a file.Détermine l'encodage du texte utilisé dans un fichier.
FileGetLongName Returns the long path+name of the path+name passed.Retourne le chemin+nom long du chemin+nom donné.
FileGetPos Retrieves the current file position.Retrouve la position dans le fichier courant.
FileGetShortcut Retrieves details about a shortcut.Retrouve les détails d'un raccourci.
FileGetShortName Returns the 8.3 short path+name of the path+name passed.Retourne le chemin+nom au format court 8.3 du chemin+nom donné.
FileGetSize Returns the size of a file in bytes.Retourne la taille d'un fichier en octets.
FileGetTime Returns the time and date information for a file.Retourne les informations de date et heure d'un fichier.
FileGetVersion Returns version information stored in a file.
FileInstall Include and install a file with the compiled script.Inclut et installe un fichier avec le script compilé.
FileMove Moves one or more files.Déplace un ou plusieurs fichiers.
FileOpen Opens a file for reading or writing.Ouvre un fichier texte pour lecture ou écriture.
FileOpenDialog Initiates a Open File Dialog.Ouvre un dialogue permettant à l'utilisateur de choisir un fichier.
FileRead Read in a number of characters from a previously opened file.Lit un nombre donné de caractères dans un fichier texte précédemment ouvert.
FileReadLine Read in a line of text from a previously opened text file.Lit une ligne de texte dans un fichier texte précédemment ouvert.
FileReadToArray Reads the specified file into an array.Lit le fichier spécifié et construit un tableau avec les lignes.
FileRecycle Sends a file or directory to the recycle bin.Envoie un fichier ou un répertoire à la corbeille.
FileRecycleEmpty Empties the recycle bin.Vide la corbeille.
FileSaveDialog Initiates a Save File Dialog.Ouvre un dialogue permettant à l'utilisateur d'enregistrer un fichier.
FileSelectFolder Initiates a Browse For Folder dialog.Lance une boîte de dialogue pour choisir un dossier.
FileSetAttrib Sets the attributes of one or more files/directories.Définit les attributs d'un ou plusieurs fichiers.
FileSetEnd Sets the end of the file at the current file position.
FileSetPos Sets the current file position.Définit la position actuelle dans un fichier.
FileSetTime Sets the timestamp of one of more files.Attribue le timestamp (l'horodatage) d'un ou plusieurs fichiers/dossiers.
FileWrite Write text/data to the end of a previously opened file.Ajoute un texte/des données à la fin d'un fichier précédemment ouvert.
FileWriteLine Append a line of text to the end of a previously opened text file.Ajoute une ligne de texte à la fin d'un fichier texte précédemment ouvert.
IniDelete Deletes a value from a standard format .ini file.Supprime une valeur à partir d'un fichier .ini au format standard.
IniRead Reads a value from a standard format .ini file.Lit une valeur d'un fichier .ini de format standard.
IniReadSection Reads all key/value pairs from a section in a standard format .ini file.Lit toutes les paires clé/valeur d'une section d'un fichier .ini au format standard.
IniReadSectionNames Reads all sections in a standard format .ini file.Lit toutes les sections dans un fichier .ini de format standard.
IniRenameSection Renames a section in a standard format .ini file.Renomme une section dans un fichier ini de format standard.
IniWrite Writes a value to a standard format .ini file.Écrit une valeur dans un fichier ini de format standard.
IniWriteSection Writes a section to a standard format .ini file.Écrit une section dans un fichier ini de format standard.
