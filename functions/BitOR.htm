BitOR

Performs a bitwise OR operation.Exécute un "OU" logique bit à bit.

BitOR ( value1, value2 [, value n] )

ParametersParamètres

value1 The first number.Le premier nombre.
value2 The second number.Le deuxième nombre.
value n [optional][optionnel] The nth number - up to 255 values can be specified.Le nième nombre - Jusqu'à 255 valeurs peuvent être spécifiées.

Return ValueValeur de retour

Returns the two parameters bitwise-OR'ed together.Retourne le "OU" logique entre les valeurs entrées en paramètres.
Bit operations are performed as 32-bit integers.Les opérations sont effectuées comme entiers 32 bits.

RemarksRemarques

Remember hex notation can be used for numbers.La notation hexadécimale peut être utilisée comme arguments.
BitOR() returns 0 in each bit position where all input arguments have a 0 in the corresponding position and 1 wherever there is at least one 1-bit.BitOR() retourne 0 pour chaque position de bit où tous les arguments ont un 0 en position correspondante et 1 partout où il y à au moins un bitt à 1.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Note:Note : "b" is the symbol for byte."b" est le symbole de l'octet.

    ; Assign a Local variable the bitwise OR operation of 1 and 0.Assigne une variable locale le opération OU de 1 et 0.
    Local $iBitOR1 = BitOR(1, 0) ; 0001b OR 0000b = 0001b0001b 0000b OU = 0001b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR1)

    ; Assign a Local variable the bitwise OR operation of 1 and 1.Assigne une variable locale le opération OU de 1 et 1.
    Local $iBitOR2 = BitOR(1, 1) ; 0001b OR 0001b = 0001b0001b 0001b OU = 0001b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR2)

    ; Assign a Local variable the bitwise OR operation of 13 (1101b) and 7 (0111b).Assigne une variable locale le opération OU de 13 (1101b) et 7 (0111b).
    Local $iBitOR3 = BitOR(13, 7) ; 1101b OR 0111b = 1111b1101b 0111b OU = 1111b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR3)

    ; Assign a Local variable the bitwise OR operation of 2 (0010b), 3 (0011b) and 6 (0110b).Assigne une variable locale le opération OU de 2 (0010b), 3 (??0011b) et 6 (0110b).
    Local $iBitOR4 = BitOR(2, 3, 6) ; 0010b OR 0011b OR 0110b = 0111b0010b 0011b OU OU = 0110b 0111b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR4)
EndFunc   ;==>Example
