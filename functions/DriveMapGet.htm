DriveMapGet

Retrieves the details of a mapped drive.Récupère les informations d'un disque mappé.

DriveMapGet ( "device" )

ParametersParamètres

device The device (drive or printer) letter to query, e.g. "O:" or "LPT1:"

Return ValueValeur de retour

Success:Succès : a details of the mapping, e.g. \\server\share
Failure:Échec : an empty string "" and sets the @error flag to 1.Retourne une chaîne vide "" et @error = 1.

RelatedLiens

DriveMapAdd, DriveMapDel

ExampleExemple

#include <MsgBoxConstants.au3>

; Map X drive to \\myserver\stuff using current userMappe le lecteur X sur \\myserver\stuff en utilisant l'utilisateur courant
DriveMapAdd("X:", "\\myserver\stuff")

; Get details of the mappingObtient des renseignements sur le mappage
MsgBox($MB_SYSTEMMODAL, "", "Drive X: is mapped to: " & DriveMapGet("X:"))
