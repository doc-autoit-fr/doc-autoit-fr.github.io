String functions Reference

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
StringAddCR Takes a string and prefixes all linefeed characters ( Chr(10) ) with a carriage return character ( Chr(13) ).Ajoute @CR devant tous les caractèrs @LF d'une chaîne.
StringCompare Compares two strings with options.Compare deux chaînes avec, ou pas, sensibilité à la casse.
StringFormat Returns a formatted string (similar to the C sprintf() function).Retourne une chaîne formatée (similaire à sprintf() en C).
StringFromASCIIArray Converts an array of ASCII codes to a string.Convertit un tableau de codes ASCII en une chaîne de caractères.
StringInStr Checks if a string contains a given substring.Vérifie si une chaîne contient une sous-chaîne donnée.
StringIsAlNum Checks if a string contains only alphanumeric characters.Vérifie si une chaîne contient uniquement des caractères alphanumériques.
StringIsAlpha Checks if a string contains only alphabetic characters.Vérifie si la chaîne contient uniquement des caractères alphabétiques.
StringIsASCII Checks if a string contains only ASCII characters in the range 0x00 - 0x7f (0 - 127).Vérifie si la chaîne contient uniquement des caractères ASCII compris entre 0x00 et 0x7f (0 - 127).
StringIsDigit Checks if a string contains only digit (0-9) characters.Vérifie si une chaîne contient uniquement des chiffres (0-9).
StringIsFloat Checks if a string is a floating point number.Vérifie si une chaîne contient est un nombre à virgule flottante.
StringIsInt Checks if a string is an integer.Vérifie si une chaîne contient un nombre entier.
StringIsLower Checks if a string contains only lowercase characters.Vérifie si une chaîne contient uniquement des lettres minuscules.
StringIsSpace Checks if a string contains only whitespace characters.
StringIsUpper Checks if a string contains only uppercase characters.Vérifie si une chaîne contient uniquement des lettres majuscules.
StringIsXDigit Checks if a string contains only hexadecimal digit (0-9, A-F) characters.Vérifie si une chaîne contient uniquement des caractères hexadécimaux (0-9, A-F).
StringLeft Returns a number of characters from the left-hand side of a string.Retourne un nombre de caractères à partir de la gauche d'une chaîne.
StringLen Returns the number of characters in a string.Retourne le nombre de caractères d'une chaîne.
StringLower Converts a string to lowercase.Convertit une chaîne en minuscules.
StringMid Extracts a number of characters from a string.Extrait un nombre donné de caractères d'une chaîne.
StringRegExp Check if a string fits a given regular expression pattern.Vérifie si une chaîne de caractères correspond à un modèle d'expression régulière.
StringRegExpReplace Replace text in a string based on regular expressions.Remplace du texte, dans une chaîne, selon une expression régulière donnée.
StringReplace Replaces substrings in a string.Remplace des sous-chaînes dans une chaîne.
StringReverse Reverses the contents of the specified string.Inverse le contenu de la chaîne spécifiée.
StringRight Returns a number of characters from the right-hand side of a string.Retourne un nombre donné de caractères à partir de la droite d'une chaîne.
StringSplit Splits up a string into substrings depending on the given delimiters.Découpe une chaîne en tableau de sous-chaînes selon des délimiteurs donnés.
StringStripCR Removes all carriage return values ( Chr(13) ) from a string.Supprime tous les retours à la ligne ( Chr(13) ) dans une chaîne.
StringStripWS Strips the white space in a string.Enlève les 'espaces blancs' dans une chaîne.
StringToASCIIArray Converts a string to an array containing the ASCII code of each character.Convertit une chaîne en un tableau contenant le code ASCII de chaque caractère.
StringTrimLeft Trims a number of characters from the left hand side of a string.Coupe un nombre de caractères depuis la gauche de la chaîne.
StringTrimRight Trims a number of characters from the right hand side of a string.Coupe un nombre de caractères depuis la droite de la chaîne.
StringUpper Converts a string to uppercase.Convertit une chaîne en majuscules.
