SendKeepActive

Attempts to keep a specified window active during Send().Tente de garder une fenêtre spécifiée active pendant Send().

SendKeepActive ( "title" [, "text"] )

ParametersParamètres

title The title/hWnd/class of the window to activate.Le titre/hWnd/class de la fenêtre à activer. See Title special definition.Consultez Définiton spéciale de titre. Use a blank title to disable the function.Utilisez un titre vide pour désactiver la fonction.
text [optional][optionnel] The text of the window.Le texte de la fenêtre. . Default is an empty string.La valeur par défaut est une chaîne vide. See Text special definition.Consultez Définition spéciale de Texte .

Return ValueValeur de retour

Success:Succès : 1.Retourne 1 si la fenêtre existe.
Failure:Échec : 0 if window is not found.Retourne 0 si la fenêtre n'est pas trouvée.

RemarksRemarques

Using SendKeepActive("") attempts to reset the active window in between each simulated keystroke from Send().Utilisation SendKeepActive("") tente de réinitialiser la fenêtre active entre chaque simulation de touche avec Send() .

Example()

Func Example()
    ; Run NotepadExécute le Bloc-notes
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.Attend 10 secondes pour que la fenêtre du bloc-notes fenêtre apparaisse.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Keep the Notepad window active when using the Send function.Garde la fenêtre du Bloc-notes active lorsque vous utilisez la fonction Send().
    SendKeepActive("[CLASS:Notepad]")

    ; Simulate entering a string of text.Simule l'entrée d'une chaîne de texte. If you try to change to a different window other than Notepad, then the Notepad window willSi vous essayez de passer à une autre fenêtre que le Bloc-notes, alors la fenêtre du Bloc-notes
    ; be brought to focus again.récupèrera le focus de nouveau.
    For $i = 1 To 10
        Sleep(500)
        Send("notepad - ")
    Next

    ; Disable the Notepad window being active when using the Send function.Désactive la fenêtre du Bloc-notes qui est active lorsque vous utilisez la fonction Send().
    SendKeepActive("")

    ; Close the Notepad window using the handle returned by WinWait.Ferme la fenêtre du Bloc-notes en utilisant le handle renvoyé par WinWait.
    WinClose($hWnd)

    ; Now a screen will pop up and ask to save the changes, the classname of the window is calledMaintenant, une fenêtre s'ouvre et demande d'enregistrer les modifications, le nom de classe de la fenêtre est appelé
    ; "#32770" and simulating the "TAB" key to move to the second button in which the "ENTER" is simulated to not "save the file""# 32770" et la simulation de la touche "TAB" pour passer au deuxième bouton dans lequel le "ENTER" est simulé pour ne pas enregistrer le fichier
    WinWaitActive("[CLASS:#32770]")
    Sleep(500)
    Send("{TAB}{ENTER}")
EndFunc   ;==>Example
