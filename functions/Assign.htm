Référence Fonction

Assign

Assigns a variable by name with the data.Assigne des données à une variable définie par son nom.

Assign ( "varname", "data" [, flag = 0] )

ParametersParamètres

varname The name of the variable you wish to assign.Nom de la variable pour laquelle vous souhaitez faire l'assignation. Cannot be an array element and must only contain letters, digits and underscore characters (as per normal variable naming conventions).Ne peut pas être l'élément d'un tableau et doit contenir seulement des lettres, des chiffres et le caractère de soulignement (mêmes conventions que pour les noms de variable normale).
data The data you wish to assign to the variable.Les données que vous souhaitez attribuer à la variable.
flag [optional] controls the way that variables are assigned (add required options together):Permet de contrôler la manière dont les variables sont crées.Peut être une combinaison des éléments suivants :
    $ASSIGN_CREATE (0) = (default) Create variable if required
    $ASSIGN_FORCELOCAL (1) = Force creation in local scope
    $ASSIGN_FORCEGLOBAL (2) = Force creation in global scope
    $ASSIGN_EXISTFAIL (4) = Fail if variable does not already exist

Constants are defined in AutoItConstants.au3

Return ValueValeur de retour

Success: 1.Retourne 1 si la fenêtre existe.
Failure: 0 if unable to create/assign the variable.Retourne 0 si incapacité de créer/assigner la variable.

RemarksRemarques

If there is a need to use Assign() to create/write to a variable, then in most situations, Eval() should be used to read the variable and IsDeclared() should be used to check that the variable exists.Si vous avez besoin d'utiliser Assign() pour créer ou écrire dans une variable, alors dans la plupart des situations Eval() devra être utilisée pour lire la variable , et IsDeclared() sera utilisée pour vérifier que la variable existe.

RelatedLiens

Eval, Execute, IsDeclared

ExampleExemple

#include <MsgBoxConstants.au3>

; Assign the variable string sString with data.Affecte la variable chaîne sChaine de données.
Assign("sString", "This is a string which is declared using the function Assign")

; Find the value of the variable string sString and assign to the variable $sEvalString.Trouve la valeur de la chaîne sChaine et l'affecte à la variable $sEvalString.
Local $sEvalString = Eval("sString")

; Display the value of $sEvalString.Affiche la valeur de $sEvalString. This should be the same value as $sString.Cela devrait être la même valeur que $sChaine.
MsgBox($MB_SYSTEMMODAL, "", $sEvalString)
