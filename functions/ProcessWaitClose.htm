ProcessWaitClose

Pauses script execution until a given process does not exist.Met en pause l'exécution du script jusqu'à ce qu'un processus donné n'existe plus.

ProcessWaitClose ( "process" [, timeout = 0] )

ParametersParamètres

process The name or PID of the process to check.Le nom ou le PID du processus à vérifier.
timeout [optional][optionnel][optionnel] Specifies how long to wait (in seconds).Spécifie la durée d'attente (en secondes ). Default is to wait indefinitely.Par défaut attendre indéfiniment.

Return ValueValeur de retour

Success:Succès : 1 and sets @extended to the exit code of the process.Retourne 1 et définit @extended avec le code de sortie du processus.
Failure:Échec : 0 if the wait timed out.Retourne 0 si l'attente a expiré. On invalid PID the @error flag is set to non-zero and @extended is set to 0xCCCCCCCC.Si le PID est invalide @error n'est pas nul - et @extended = 0xCCCCCCCC.

RemarksRemarques

Process names are executables without the full path, e.g., "notepad.exe" or "winword.exe"

PID is the unique number which identifies a Process.PID est le numéro unique qui identifie un processus. A PID can be obtained through the ProcessExists() or Run() commands.Un PID peut être obtenu par les commandes ProcessExists() or Run().

The process is polled approximately every 250 milliseconds.Le processus est interrogé environ toutes les 250 millisecondes.

The return is 1 if the process was not existing.Le retour est 1 si le processus n'existait pas.

RelatedLiens

ExampleExemple

; Wait until no instance of Notepad exists.Attend jusqu'à ce que aucune instance de notepad existe.
ProcessWaitClose("notepad.exe")

; Wait until this particular instance of Notepad is closed by the user.Attend jusqu'à ce que cet instance particulière de Notepad soit fermé par l'utilisateur.
Local $iPID = Run("notepad.exe")
ProcessWaitClose($iPID)
