BitShift

Performs a bit shifting operation.Exécute un décalage de bits.

BitShift ( value, shift )

ParametersParamètres

value The number to be shifted.La valeur à décaler.
shift Number of bits to shift to the right (negative numbers shift left).Le nombre de bits à décaler vers la droite (les nombres négatifs décalent à gauche).

Return ValueValeur de retour

Returns the value shifted by the required number of bits.Retourne la valeur décalée du nombre bits demandés.
Bit operations are performed as 32-bit integers.Les opérations sont effectuées comme entiers 32 bits.

RemarksRemarques

Remember hex notation can be used for numbers.La notation hexadécimale peut être utilisée comme arguments.
Right shifts are equivalent to halving; left shifts to doubling.Les décalages à droite sont équivalents à des successions de divisions par deux; les décalages à gauche à des multiplications par deux.

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Note:Note : "b" is the symbol for byte."b" est le symbole de l'octet.

    ; Assign a Local variable the bitwise left-shift operation of 2.Assigne une variable locale de l'opération au niveau du bit de décalage à gauche de 2.
    Local $iBitShift1 = BitShift(2, -1) ; 2 = 0010b left-shifted once -> 4 = 0100b2 = 0010b décalée à gauche une fois - > 4 = 0100b

    ; Note:Note : It is equivalent to do this:Il est équivalent à faire : BitRotate(2, 1)BitRotate (2, 1)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift1)

    ; Assign a Local variable the bitwise right-shift operation of 1.Assigne une variable locale l'opération de décalage à droite au niveau du bit de 1.
    Local $iBitShift2 = BitShift(1, 1) ; 1 = 0001b right-shifted once -> 0 = 0000b1 = 0001b décalé à droite une fois - > 0 = 0000b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift2)

    ; Assign a Local variable the bitwise left-shift operation of 14.Assigne une variable locale de l'opération au niveau du bit de décalage à gauche de 14 ans.
    Local $iBitShift3 = BitShift(14, -2) ; 14 = 1110b left-shifted twice -> 56 = 0011 1000b14 = 1110b gauche décalé deux fois -> 56 = 0011 1000b

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift3)

    ; Assign a Local variable the bitwiseAssigne une variable locale bit à bit
    Local $iBitShift4 = BitShift(1, -31)
    ;1 = 0001b right-shifted 31 times -> -2147483648 (32 bits) = 1000 0000 0000 0000 0000 0000 0000 0000b1 = 0001b décalé à droite 31 fois -> -2147483648 (32 bits) = 1000 0000 0000 0000 0000 0000 0000 0000b

    ;Note: in 2's-complement notation, the 32nd digit from the right has a negative sign.Remarque : dans la notation complément à 2, le chiffre 32e de la droite a un signe négatif.

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift4)
EndFunc   ;==>Example
