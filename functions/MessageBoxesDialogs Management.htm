Message Boxes and Dialogs functions ReferenceMessage Boxes and Dialogs - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

 

FunctionFonction Description
InputBox Displays an input box to ask the user to enter a string.Affiche une boîte de saisie pour demander à l'utilisateur de saisir une chaîne.
MsgBox Displays a simple message box with optional timeout.Affiche une simple boîte de message, avec temporisation optionnelle.
ProgressOff Turns Progress window off.Ferme une fenêtre de progression.
ProgressOn Creates a customizable progress bar window.Crée une fenêtre de barre de progression personnalisable.
ProgressSet Sets the position and/or text of a previously created Progress bar window.Définit la position et/ou le texte d'une fenêtre de barre de progression précédemment créé.
SplashImageOn Creates a customizable image popup window.Crée une fenêtre d'image popup personnalisable.
SplashOff Turns SplashText or SplashImage off.Désactive SplashText ou SplashImage.
SplashTextOn Creates a customizable text popup window.Crée une fenêtre de texte popup personnalisable.
ToolTip Creates a tooltip anywhere on the screen.Crée une info-bulle n'importe où sur l'écran.
