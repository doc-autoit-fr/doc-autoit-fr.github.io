RegRead

Reads a value from the registry.Lit une valeur dans le Registre.

RegRead ( "keyname", "valuename" )

ParametersParamètres

keyname The registry key to read.La clé de registre à lire.
valuename The value to read.La valeur à lire.

Return ValueValeur de retour

Success:Succès : the requested registry value.Retourne la valeur de Registre demandé. @extended is set to the type of the value $REG_... .@extended contient le type de la valeur $REG_.... These types are defined in the "Constants.au3" include file.Ces types sont définis dans le fichier 'include' "Constants.au3".
Failure:Échec : sets the @error flag to non-zero.met @error <> 0.
@error:@error : 1 = unable to open requested key1 = impossible d'ouvrir la clé demandée
 2 = unable to open requested main key
 3 = unable to remote connect to the registry
-1 = unable to open requested value-1 = impossible d'ouvrir la valeur demandée
-2 = value type not supported-2 = type de valeur non pris en charge

RemarksRemarques

A registry key must start with "HKEY_LOCAL_MACHINE" ("HKLM") or "HKEY_USERS" ("HKU") or "HKEY_CURRENT_USER" ("HKCU") or "HKEY_CLASSES_ROOT" ("HKCR") or "HKEY_CURRENT_CONFIG" ("HKCC").Une clé de registre doit commencer par "HKEY_LOCAL_MACHINE" ("HKLM") ou "HKEY_USERS" ("HKU") ou "HKEY_CURRENT_USER" ("HKCU") ou "HKEY_CLASSES_ROOT" ("HKCR") ou "HKEY_CURRENT_CONFIG" ("HKCC") .

When running on 64-bit Windows if you want to read a value specific to the 64-bit environment you have to suffix the HK... with 64 i.e.Lors de l'exécution sur Windows 64 bits si vous souhaitez lire une valeur spécifique à l'environnement 64 bits, vous devez suffixer HK... HKLM64.avec 64 c'est à dire HKLM64.

AutoIt supports registry keys of type REG_BINARY, REG_SZ, REG_MULTI_SZ, REG_EXPAND_SZ, REG_QWORD, and REG_DWORD.AutoIt supporte les clés de registre de type REG_BINARY, REG_SZ, REG_MULTI_SZ, REG_EXPAND_SZ, REG_QWORD, et REG_DWORD.

To access the (Default) value use "" (an empty string) for the valuename.Pour accéder à la valeur par (par défaut) utilisez "" (une chaîne vide) pour le nom de la valeur.

When reading a REG_BINARY key the result is a binary datatype (in previous versions it was a string of hex characters).Lors de la lecture d'une clé REG_BINARY le résultat est un type de données binaire (dans les versions précédentes, il était une chaîne de caractères hexadécimaux ).

When reading a REG_MULTI_SZ key the multiple entries are separated by @LF - use with StringSplit(..., @LF) to get each entry.Lors de la lecture d'une clé de REG_MULTI_SZ les multiples entrées sont séparées par @LF - à utiliser avec StringSplit (..., @LF) pour obtenir chaque entrée.

It is possible to access remote registries by using a keyname in the form "\\computername\keyname".Il est possible d'accéder à des registres à distance en utilisant des noms de valeur sous la forme "\\nom_ordinateur\nom_de_la_clé". To use this feature you must have the correct access rights.Pour utiliser cette fonctionnalité vous devez avoir les droits d'accès corrects.
