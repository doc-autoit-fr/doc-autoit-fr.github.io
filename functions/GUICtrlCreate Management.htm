GUI Control creation functions ReferenceGUI Control creation - Référence Fonctions

Below is a complete list of the functions available in AutoIt.Voici une liste complète des fonctions disponibles dans AutoIt. Click on a function name for a detailed description.Cliquez sur le nom d'une fonction pour une description détaillée.

See Gui Constants include files if you need to use the related controls Constants .Voir Fichiers 'include' des constantes GUI si vous avez besoin d'utiliser des constantes relatives aux contrôles.

 

FunctionFonction Description
GUICtrlCreateAvi Creates an AVI video control for the GUI.Crée un contrôle vidéo AVI dans une GUI.
GUICtrlCreateButton Creates a Button control for the GUI.Crée un contrôle Button (bouton) dans la GUI.
GUICtrlCreateCheckbox Creates a Checkbox control for the GUI.Crée un contrôle Chekbox (cases à cocher) dans une GUI.
GUICtrlCreateCombo Creates a ComboBox control for the GUI.Crée un Contrôle ComboBox dans une GUI.
GUICtrlCreateContextMenu Creates a context menu for a control or entire GUI window.Crée un menu contextuel pour un contrôle ou pour toute la fenêtre GUI.
GUICtrlCreateDate Creates a date control for the GUI.Crée un contrôle date dans une GUI.
GUICtrlCreateDummy Creates a Dummy control for the GUI.Crée un contrôle fictif dans la GUI.
GUICtrlCreateEdit Creates an Edit control for the GUI.Crée un contrôle Edit pour l'interface graphique.
GUICtrlCreateGraphic Creates a Graphic control for the GUI.Crée un contrôle graphique dans une GUI.
GUICtrlCreateGroup Creates a Group control for the GUI.Crée un contrôle de groupe dans une GUI.
GUICtrlCreateIcon Creates an Icon control for the GUI.Crée un contrôle Icône dans une GUI.
GUICtrlCreateInput Creates an Input control for the GUI.Crée un contrôle Input dans une GUI.
GUICtrlCreateLabel Creates a static Label control for the GUI.Crée un contrôle Label dans une GUI.
GUICtrlCreateList Creates a List control for the GUI.Crée un contrôle liste dans une GUI.
GUICtrlCreateListView Creates a ListView control for the GUI.Crée un contrôle ListView dans une GUI.
GUICtrlCreateListViewItem Creates a ListView item.Crée un item de ListView.
GUICtrlCreateMenu Creates a Menu control for the GUI.Crée un contrôle Menu dans une GUI.
GUICtrlCreateMenuItem Creates a MenuItem control for the GUI.Crée un contrôle MenuItem dans une GUI.
GUICtrlCreateMonthCal Creates a month calendar control for the GUI.Crée un contrôle de Calendrier Mensuel dans une GUI.
GUICtrlCreateObj Creates an ActiveX control in the GUI.Crée un contrôle ActiveX dans la GUI.
GUICtrlCreatePic Creates a Picture control for the GUI.Crée un contrôle Picture (image) dans une GUI.
GUICtrlCreateProgress Creates a Progress control for the GUI.Crée un contrôle Progress dans une GUI.
GUICtrlCreateRadio Creates a Radio button control for the GUI.Crée un contrôle RadioButton dans une GUI.
GUICtrlCreateSlider Creates a Slider control for the GUI.Crée un contrôle Slider dans une GUI.
GUICtrlCreateTab Creates a Tab control for the GUI.Crée un contrôle Tab (avec onglet) dans une GUI.
GUICtrlCreateTabItem Creates a TabItem control within an existing tab control in the GUI.Crée un contrôle TabItem dans un contrôle Tab existant dans une GUI.
GUICtrlCreateTreeView Creates a TreeView control for the GUI.Crée un contrôle TreeView (Vue arborescente) dans une GUI.
GUICtrlCreateTreeViewItem Creates a TreeViewItem control for the GUI.Crée un contrôle TreeViewItem dans le contrôle TreeView d'une GUI.
GUICtrlCreateUpdown Creates an UpDown control for the GUI.Crée un contrôle UpDown dans une GUI.
GUICtrlDelete Deletes a control.Supprime un contrôle.
