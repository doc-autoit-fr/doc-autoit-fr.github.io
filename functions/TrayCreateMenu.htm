TrayCreateMenu

Creates a menu control for the tray menu.Crée un contrôle menu pour le menu de la zone de notification.

TrayCreateMenu ( "sub/menutext" [, menuID = -1 [, menuentry = -1]] )

ParametersParamètres

sub/menutext The sub/menu text.Le texte menu ou du sous-menu.
menuID [optional][optionnel] If defined, allows you to create a submenu in the referenced menu.Si défini, vous permet de créer un sous-menu dans le menu référencé. Default -1 (refers to first level menu).Par défaut, -1 (se réfère au premier niveau du menu).
menuentry [optional][optionnel] Allows you to define the entry number to be created.Vous permet de définir le nombre d'entrées à créer. The entries are numbered starting at 0.Les entrées sont numérotées à partir de 0. Default -1 (at the bottom).Par défaut -1 (en bas).

Return ValueValeur de retour

Success:Succès : the identifier (controlID) of the new tray menu.Retourne l'identifiant (controlID) du nouveau menu de la zone de notification.
Failure:Échec : 0.Retourne 0.

RelatedLiens

TrayGetMsg, TrayItemDelete, TrayItemSetState, TrayItemSetText

#NoTrayIcon

These are options 1 and 2 for TrayMenuMode.Ce sont les options 1 et 2 de TrayMenuMode.

Example()

Func Example()
    Local $iSettings = TrayCreateMenu("Settings") ; Create a tray menu sub menu with two sub items.Crée un sous-menu du menu de la zone de notification avec deux sous-items.
    Local $iDisplay = TrayCreateItem("Display", $iSettings)
    Local $iPrinter = TrayCreateItem("Printer", $iSettings)
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Create a separator line.Crée une ligne de séparation.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Show the tray menu.Affiche le menu de la barre.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Display a message box about the AutoIt version and installation path of the AutoIt executable.Affiche une boîte de message à propos de la version et du chemin d'installation de l'exécutable AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", 0, -1) - 1)) ; Find the folder of a full path.Trouve le dossier d'un chemin d'accès complet.

            Case $iDisplay, $iPrinter
                MsgBox($MB_SYSTEMMODAL, "", "A sub menu item was selected from the tray menu.")

            Case $idExit ; Exit the loop.Sort de la boucle.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
