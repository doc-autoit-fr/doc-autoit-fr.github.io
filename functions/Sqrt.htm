Référence Fonction

Sqrt

Calculates the square-root of a number.Calcule la racine carrée d'un nombre.

Sqrt ( expression )

ParametersParamètres

expression Any nonnegative expression to get the square-root of.Toute expression positive ou nulle dont on veut obtenir la racine carrée.

Return ValueValeur de retour

Success: the square-root.Retourne la racine carrée.
Failure: -1.#IND if parameter is negative.Retourne -1.#IND si le paramètre est strictement négatif.

RemarksRemarques

To compute an nth root, use the power operator: x ^ (1/n)Pour calculer une racine n-ième, utilisez l'opérateur de puissance: x ^ (1/n)
MsgBox(4096,"The cube root of 27 is", 27 ^ (1/3) )MsgBox(4096, "La racine cubique de 27 est ", 27 ^ (1 / 3))

ExampleExemple

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Assign a Local variable the square-root of 2.Assigne une variable locale avec la racine carrée de 2.
    Local $fSqrt1 = Sqrt(2)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fSqrt1)

    ; Assign a Local variable the square-root of 9.Assigne une variable locale avec la racine carrée de 9.
    Local $fSqrt2 = Sqrt(9)

    ; Display the result.Affiche le résultat.
    MsgBox($MB_SYSTEMMODAL, "", $fSqrt2)
EndFunc   ;==>Example
