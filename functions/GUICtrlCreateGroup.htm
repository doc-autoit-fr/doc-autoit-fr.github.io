GUICtrlCreateGroup

Creates a Group control for the GUI.Crée un contrôle de groupe dans une GUI.

GUICtrlCreateGroup ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

ParametersParamètres

text The text of the control.Le texte de la commande.
left The left side of the control.La distance à gauche à la GUI. If -1 is used then left will be computed according to GUICoordMode.Si vous utilisez -1, elle sera calculée en accord avec GUICoordMode.
top The top of the control.La distance en haut à la GUI. If -1 is used then top will be computed according to GUICoordMode.Si -1 est utilisé, elle sera calculée en accord avec GUICoordMode .
width [optional] The width of the control (default is the previously used width).La largeur du contrôle (par défaut, la largeur utilisée précédemment).
height [optional] The height of the control (default is the previously used height).La hauteur du contrôle (par défaut, la hauteur précédemment utilisé ).
style [optional] Defines the style of the control.Définit le style de la commande. See GUI Control Styles Appendix.Voir l'Appendice Styles des contrôles GUI.
    default ( -1) : none.
    forced styles : $WS_GROUP, $BS_GROUPBOX.
exStyle [optional] Defines the extended style of the control.Définit le style étendu de la commande. See Extended Style Table.Consultez en Appendice le Tableau des Styles Etendus .

Return ValueValeur de retour

Success: the identifier (controlID) of the new control.Retourne l'identifiant ( controlID ) du nouveau contrôle.
Failure: 0.Retourne 0.

RemarksRemarques

A group control is the thin line you see around controls (usually only Radio button) that visually groups them together.Un groupe de contrôle est la mince ligne que vous voyez autour des contrôles (généralement le bouton radio uniquement) qui les regroupe de manière visuelle. Only one Radio button within a Group can be selected at once.Un seul bouton radio dans un groupe peut être sélectionné à la fois.
If you want to have multiple groups without the visible line then you must use GUIStartGroup() to group your Radio buttons.Si vous voulez avoir plusieurs groupes sans la ligne visible, alors vous devez utiliser GUIStartGroup() pour regrouper vos boutons radio.

To use the values specified above you must #include <ButtonConstants.au3> in your script.

Default resizing is $GUI_DOCKAUTO size and position will occur.Pour redimensionner par défaut, utiliez $GUI_DOCKAUTO.

ExampleExemple

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI group") ; will create a dialog box that when displayed is centeredCrée une boîte de dialogue qui s'affichera centrée

    GUICtrlCreateGroup("Group 1", 190, 60, 90, 140)
    GUICtrlCreateRadio("Radio 1", 210, 90, 50, 20)
    GUICtrlCreateRadio("Radio 2", 210, 110, 60, 50)
    GUICtrlCreateGroup("", -99, -99, 1, 1) ;close groupFerme le groupe

    GUISetState(@SW_SHOW) ; will display an empty dialog boxAffiche une boîte de dialogue vide

    ; Loop until the user exits.Boucle jusqu'à ce que l'utilisateur quitte.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
